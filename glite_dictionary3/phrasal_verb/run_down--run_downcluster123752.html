to run down

verb
trace or investigate thoroughly
Pronunciation: / rʌn daʊn / (AmE), / rʌn daʊn / (BrE)
to trace or investigate something thoroughly, often to discover the facts or source

Examples

to run down the source
It took weeks to run down the origin of the mysterious email.
Journalists worked to run down the source of the leaked documents.
The police are trying to run down every lead related to the case.

  • synonym – track down: to search for and locate someone or something
