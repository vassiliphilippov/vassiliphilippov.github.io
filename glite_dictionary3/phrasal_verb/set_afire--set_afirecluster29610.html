to set afire

verb
to cause to start burning
Pronunciation: / sɛt əˈfaɪɚ / (AmE), / sɛt əˈfaɪə / (BrE)
to cause something to start burning; to set on fire

Examples

to set afire a building
A careless spark could easily set afire this pile of dry leaves.
He accidentally set afire the corner of his notebook with a candle.
The protestors set afire an abandoned car during the demonstration.

Related Terms

  • antonym – extinguish: to put out a fire; cause (something burning) to stop burning
  • synonym – ignite: to catch fire or cause to catch fire
