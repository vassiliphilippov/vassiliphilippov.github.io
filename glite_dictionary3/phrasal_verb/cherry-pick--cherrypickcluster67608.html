to cherry-pick

verb
apply a single commit between branches (software)
Pronunciation: / \u02c8t\u0283\u025bri p\u026ak / (AmE), / \u02c8t\u0283\u025bri p\u026ak / (BrE)
(in software development) to select a specific commit or set of changes from one branch and apply it to another branch without merging all other changes.

Examples

to cherry-pick a commit
The maintainer cherry-picked a critical commit to the hotfix branch today.
Before release, she decided to cherry-pick bug fixes rather than merge the entire feature branch.
While reviewing the pull request, he was cherry-picking only commits that fixed failing tests.

Related Terms

  • synonym – backport: to apply a change or fix from a newer version of software to an older version or branch
