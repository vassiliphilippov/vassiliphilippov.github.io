to pull taut

verb
stretch something tightly by pulling
Pronunciation: / pʊl tɔt / (AmE), / pʊl tɔːt / (BrE)
to stretch something tightly by pulling it so that there is no slack

Examples

to pull taut a rope
If you pull taut the cord, the blinds will stay up securely.
She asked him to pull taut the string before tying the knot.
Using both hands, he pulled taut the bandage around her arm.
