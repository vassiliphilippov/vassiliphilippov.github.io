to run off

verb
decide by a runoff (competition)
Pronunciation: / rʌn ɔf / (AmE), / rʌn ɒf / (BrE)
to decide the outcome of a contest or competition by holding a runoff between top candidates or teams

Examples

run off a final match
If both candidates receive equal votes, they will run off for the seat next week.
Since the teams were tied, they had to run off the final match to determine the winner.
The election was too close to call, so officials decided to run off a second round.

Related Terms

  • synonym – runoff: an additional contest or election to decide a tie or close result
