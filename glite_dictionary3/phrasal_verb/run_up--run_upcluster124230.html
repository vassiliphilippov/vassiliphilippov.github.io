to run up

verb
accumulate (debts, expenses, scores) rapidly
Pronunciation: / rʌn ʌp / (AmE), / rʌn ʌp / (BrE)
to accumulate or increase debts, bills, or expenses quickly

Examples

to run up debts
He managed to run up a huge credit card debt during college.
If you run up too many bills, it will be hard to pay them later.
The team ran up the score in the second half of the game.

If you run up too much credit card debt, it becomes hard to pay off.
She didn't mean to run up so many expenses during the trip.
If you use your credit card for every purchase, you can run up a large amount of debt.
She managed to run up a huge phone bill in just one week.
The company began to run up significant expenses as the project moved forward.
