to add a shim

verb
apply a software compatibility shim
Pronunciation: / æd ə ʃɪm / (AmE), / æd ə ʃɪm / (BrE)
(in computing) to insert or apply a small piece of software (a compatibility shim) to allow older code, drivers, or components to work with a newer system, typically as a temporary workaround.

Examples

to add a shim for software
We had to add a shim to the build to support the legacy driver.
After upgrading the OS, the team added a shim so the old plugin would still run.
Their release notes warned they were adding a shim as a temporary compatibility workaround.

Related Terms

  • synonym – compatibility layer: software that allows programs written for one environment to run in another
  • synonym – shim: a thin piece of material or a small compatibility layer used to adjust or adapt components
  • synonym – workaround: a temporary fix or method used to bypass a problem in software or systems
