to run down

verb
examine or review quickly
Pronunciation: / rʌn daʊn / (AmE), / rʌn daʊn / (BrE)

Examples

to run down the list

Let me run down the main points before we start the meeting.
She quickly ran down the list of candidates to find any familiar names.
