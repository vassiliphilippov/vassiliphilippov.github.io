to set aside

verb
annul/revoke a legal decision
Pronunciation: / sɛt əˈsaɪd / (AmE), / sɛt əˈsaɪd / (BrE)
to officially cancel, annul, or suspend something, especially a law, rule, or decision

Examples

to set aside a decision
If the appeal is successful, the court may set aside the conviction.
The judge decided to set aside the earlier ruling due to new evidence.
The lawyer filed a motion to set aside the verdict on procedural grounds.
In times of crisis, governments may set aside certain regulations to act swiftly.
The company agreed to set aside the current policy while it's under review.
