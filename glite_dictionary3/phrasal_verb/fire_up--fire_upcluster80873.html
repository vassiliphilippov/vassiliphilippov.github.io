to fire up

verb
start a device or program
Pronunciation: / faɪɚ ʌp / (AmE), / faɪə ʌp / (BrE)
to start or switch on a machine, device, computer, or software application; to boot or launch it.

Examples

to fire up the computer
Before the meeting, she decided to fire up her laptop to review the slides.
They fired up the test server at midnight to run the automated deployment scripts.
If the app crashes, try firing up the program again after clearing the cache.

Related Terms

  • synonym – boot: to load and begin running a computer's operating system
  • synonym – launch: to begin running an application or to make something available for use
  • synonym – start: to cause something such as a machine or process to begin operating
