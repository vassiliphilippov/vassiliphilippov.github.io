to run on

verb
continue uninterrupted
Pronunciation: / rʌn ɑn / (AmE), / rʌn ɒn / (BrE)
to continue for longer than expected or without interruption

Examples

run on for hours
His lectures often run on longer than scheduled.
She let the engine run on for a few minutes before turning it off.
The meeting ran on for hours without any resolution.
