to bring up

verb
start (an operating system/process)
Pronunciation: / brɪŋ ʌp / (AmE), / brɪŋ ʌp / (BrE)
(computing) To cause an operating system to load and start the initial processes.

Examples

bring up the server
After installing the update, you need to bring up the server again.
If the computer crashes, try to bring up the operating system from the recovery disk.
The technician will bring up the system after routine maintenance.
