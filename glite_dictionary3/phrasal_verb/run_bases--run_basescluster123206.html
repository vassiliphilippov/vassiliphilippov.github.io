to run bases

verb
to advance around baseball bases
Pronunciation: / rʌn ˈbeɪsɪz / (AmE), / rʌn ˈbeɪsɪz / (BrE)
(in baseball) to advance by running around all four bases, usually after hitting the ball.

Examples

to run bases after hitting
After hitting the home run, he ran bases as the crowd erupted in cheers.
Coaches teach players how to run bases quickly and efficiently.
She was tagged out while trying to run bases aggressively.
