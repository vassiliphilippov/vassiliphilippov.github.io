to set down

verb
to write down or record officially
Pronunciation: / sɛt daʊn / (AmE), / sɛt daʊn / (BrE)
to put something into writing or record it officially, such as a statement, rule, or composition.

Examples

to set down rules
He carefully set down every word she spoke so he wouldn't forget.
The committee set down its recommendations in the final report.
The laws were set down centuries ago but are still in effect today.

Related Terms

  • synonym – note: to write down or record something briefly
  • synonym – record: to set down in writing or another permanent form
