to fork off

verb
create a separate software repository
Pronunciation: / fɔrk ɑf / (AmE), / fɔːk ɒf / (BrE)
(computing) to create a separate copy of a software repository or project that diverges from the original, typically to develop features, experiment, or maintain an independent version.

Examples

to fork off a project
After disagreements, the developers agreed to fork off the project and work separately.
She forked off a copy of the repository to experiment with a new UI design.
Continuous integration failed when someone tried to fork off the package without updating dependencies.

Related Terms

  • synonym – branch: to create a separate line of development within a version-control system
  • synonym – fork: (in software) to copy a repository so development can proceed independently
