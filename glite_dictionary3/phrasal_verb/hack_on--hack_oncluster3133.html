to hack on

verb
to fix code piecemeal until it works
Pronunciation: / hæk ɑn / (AmE), / hæk ɒn / (BrE)
to fix or modify a computer program in an improvised, piecemeal way until it works

Examples

to hack on a project
He stayed up all night to hack on the old game until it finally ran.
Sometimes you just have to hack on the code until the errors are gone.
They spent the weekend hacking on the app to add a few quick fixes.

Related Terms

  • synonym – debug: to identify and remove errors from computer software
  • synonym – patch: to repair code or software by applying updates or minor fixes
