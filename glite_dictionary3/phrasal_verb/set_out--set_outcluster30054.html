to set out

verb
begin a journey or leave for somewhere
Pronunciation: / sɛt aʊt / (AmE), / sɛt aʊt / (BrE)
to begin a journey, project, or activity; to start doing something

to set out on a trip

The team set out to solve the complex problem with a clear strategy.
They set out on their hike before sunrise to avoid the afternoon heat.
She set out to write her first novel with great enthusiasm.
The explorers set out in search of new lands.
We set out at dawn to climb the mountain before noon.
