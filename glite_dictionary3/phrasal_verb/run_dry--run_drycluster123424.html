to run dry

verb
become empty or depleted
Pronunciation: / rʌn draɪ / (AmE), / rʌn draɪ / (BrE)
to become empty of a supply or resource; to be exhausted or depleted.

Examples

run dry of ideas
After hours of searching for inspiration, my ideas finally ran dry.
Sometimes, even the most talented writers run dry creatively.
The well will run dry if there is no rain this season.

Related Terms

  • synonym – dry up: to become dry or to be depleted completely, especially of a resource
  • synonym – run out: to be used up so that nothing is left
