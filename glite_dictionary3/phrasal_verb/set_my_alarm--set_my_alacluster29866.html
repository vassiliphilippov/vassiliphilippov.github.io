to set my alarm

verb
program an alarm clock or device
Pronunciation: / sɛt maɪ əˈlɑrm / (AmE), / sɛt maɪ əˈlɑːm / (BrE)
to program an alarm clock or device to go off at a specific time, usually to wake oneself up

Examples

to set my alarm for 7am
Before going to bed, she carefully set my alarm to make sure she’d wake up on time.
I always set my alarm for 6:30 so I don't miss the bus.
Last night I forgot to set my alarm and overslept.
