passing note

noun
nonharmonic transition note (music)
Pronunciation: / ˈpæsɪŋ noʊt / (AmE), / ˈpɑːsɪŋ nəʊt / (BrE)
A note that is not part of the main harmony, used to smoothly connect two harmonic notes in a melody.

Examples

passing note in a melody
In classical music, a passing note often connects two harmonic tones by stepwise motion.
The melody features a passing note between the main chords, adding interest to the tune.
