backup file

noun
computer file for data backup
Pronunciation: / ˈbæk.ʌp faɪl / (AmE), / ˈbæk.ʌp faɪl / (BrE)
A computer file that is used to store a copy of data for the purpose of backup or recovery.

Examples

save a backup file
Before installing the update, make sure you create a backup file of your documents.
He accidentally deleted the original, but thankfully the backup file was available on the server.
The IT department restored the lost data from the backup file created last week.
