set point

noun
final point needed to win a set (tennis)
Pronunciation: / ˈsɛt pɔɪnt / (AmE), / ˈsɛt pɔɪnt / (BrE)
(in tennis) a situation in which one player needs only one more point to win the set.

Examples

set point in tennis
After an intense rally, she finally won the set point and took the first set.
He saved three set points before eventually losing the set.
Serving at set point, he delivered an ace to clinch it.
