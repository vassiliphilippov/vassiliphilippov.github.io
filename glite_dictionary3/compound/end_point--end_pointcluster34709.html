end point

noun
network or API interface (URL)
Pronunciation: / endp6ant / (AmE), / endp6ant / (BrE)
(in computing) a specific network address, URL, or API interface where a service receives or sends data, often used as the target for requests.

Examples

API end point
The IoT device sends its data to the cloud via a secure end point.
Developers updated the end point to accept JSON requests from third-party apps.
Before testing, configure the authentication for every API end point to prevent unauthorized access.

Related Terms

  • synonym – API endpoint: a specific address within an API where a particular resource or service can be accessed
  • synonym – URL: the address used to access a resource on the web; often used to refer to endpoints
