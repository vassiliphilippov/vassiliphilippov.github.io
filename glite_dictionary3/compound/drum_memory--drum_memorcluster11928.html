drum memory

noun
early magnetic computer storage device
Pronunciation: / drʌm ˈmɛməri / (AmE), / drʌm ˈmɛməri / (BrE)
an early form of magnetic data storage device consisting of a rotating metal cylinder coated with magnetic material, used in computers from the 1930s to the 1950s

Examples

drum memory in computers
Drum memory became obsolete when smaller and faster storage technologies were developed.
Technicians had to carefully align the heads on the drum memory to ensure accurate reading and writing.
The first computers in the laboratory used drum memory to store their programs and data.
