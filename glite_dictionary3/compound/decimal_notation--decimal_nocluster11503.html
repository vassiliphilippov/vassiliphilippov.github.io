decimal notation

noun
system for writing numbers using digits 0–9
Pronunciation: / ˈdɛsɪməl noʊˈteɪʃən / (AmE), / ˈdɛsɪməl nəʊˈteɪʃən / (BrE)
a system of writing numbers that uses ten digits, 0 to 9, to represent all numbers

Examples

decimal notation for numbers
Computers often use binary, but people are more familiar with decimal notation.
In the metric system, measurements are usually written in decimal notation for clarity.
The number 245 in decimal notation represents two hundred forty-five.

Related Terms

  • antonym – binary notation: system of representing numbers using only 0 and 1
  • antonym – hexadecimal notation: number system using base 16, digits 0–9 and letters A–F
