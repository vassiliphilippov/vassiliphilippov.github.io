scratch race

noun
race where all start equally, no handicaps
Pronunciation: / skrætʃ reɪs / (AmE), / skrætʃ reɪs / (BrE)
a race in which all competitors start from the same point with no handicaps, competing on equal terms

Examples

scratch race start together
In a scratch race, no one receives a head start or a handicap.
She won her first scratch race last weekend, beating more experienced competitors.
The cycling championship featured a scratch race where every rider started side by side.
