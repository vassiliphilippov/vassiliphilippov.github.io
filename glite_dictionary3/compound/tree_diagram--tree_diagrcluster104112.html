tree diagram

noun
branching diagram showing relationships or outcomes
Pronunciation: / tri ˈdaɪ.əˌɡræm / (AmE), / triː ˈdaɪ.əˌɡræm / (BrE)
a diagram in which lines branch out from a single starting point, showing relationships or possible outcomes in a hierarchical structure

Examples

tree diagram of choices
The family’s ancestry is neatly organized in a tree diagram format.
The linguistics professor drew a tree diagram on the board to illustrate sentence structure.
You can use a tree diagram to list all the possible outcomes of this experiment.
