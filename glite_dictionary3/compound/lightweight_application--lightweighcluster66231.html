lightweight application

noun
software using minimal resources
Pronunciation: / ˈlaɪtˌweɪt ˌæpləˈkeɪʃən / (AmE), / ˈlaɪtweɪt ˌaplɪˈkeɪʃən / (BrE)
A software program that requires minimal system resources to run.

Examples

lightweight application for quick tasks
A lightweight application is ideal for quick tasks without consuming much memory.
Our team decided to develop a lightweight application that could be used on older laptops.
Users appreciate a lightweight application because it runs smoothly even on smartphones.
