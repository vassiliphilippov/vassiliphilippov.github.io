system program

noun
program that manages a computer system
Pronunciation: / ˈsɪstəm ˈproʊɡræm / (AmE), / ˈsɪstəm ˈprəʊɡram / (BrE)
A computer program such as an operating system, compiler, or utility that manages or controls the functions of a computer system.

Examples

system program controls computer
A compiler is one example of a system program that translates source code into executable files.
Many modern devices include various system programs to manage hardware components.
The operating system is a type of system program essential for running a computer.
