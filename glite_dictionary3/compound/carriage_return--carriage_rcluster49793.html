carriage return

noun
action starting a new line (printing/computing)
Pronunciation: / ˈkærɪdʒ rɪˈtɜrn / (AmE), / ˈkærɪdʒ rɪˈtɜːn / (BrE)
an action or control character that causes the start of a new line, moving the cursor or printhead to the beginning of the line

Examples

press the carriage return key
After typing each line, you need to press the carriage return key to start a new one.
Many programming languages interpret a carriage return as the end of a line.
The old printer made a loud noise every time the carriage return was triggered.

Related Terms

  • synonym – line feed: the operation that moves the cursor or printhead down one line
