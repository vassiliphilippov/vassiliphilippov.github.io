file allocation table

noun
system for tracking storage on computer disks
Pronunciation: / ˈfaɪl ˌæləkˈeɪʃən ˈteɪbəl / (AmE), / ˈfaɪl ˌæləˈkeɪʃ(ə)n ˈteɪbl / (BrE)
a part of a computer disk that keeps track of where files and unusable areas are located on the disk
