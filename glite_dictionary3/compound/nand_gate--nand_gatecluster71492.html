NAND gate

noun
digital logic gate; NOT-AND function
Pronunciation: / ˈnænd ɡeɪt / (AmE), / ˈnænd ɡeɪt / (BrE)
a logic gate that outputs true only if at least one of its inputs is false; the inverse of an AND gate.

Examples

NAND gate circuit design
A single NAND gate can be used to implement any logical operation in theory.
The NAND gate is a fundamental building block in digital electronics.
You can combine two NAND gates to create a basic flip-flop circuit.
