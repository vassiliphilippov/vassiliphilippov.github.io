broaching machine

noun
machine tool using a broach to remove material
Pronunciation: / ˈbroʊ.tʃɪŋ məˈʃin / (AmE), / ˈbrəʊ.tʃɪŋ məˈʃiːn / (BrE)
a machine tool that uses a toothed tool, called a broach, to remove material from a workpiece by pushing or pulling the broach through it

Examples

broaching machine cuts metal
A broaching machine can create complex profiles in a single pass.
Operators must be trained before using the broaching machine for precision parts.
The factory installed a new broaching machine to speed up production.
