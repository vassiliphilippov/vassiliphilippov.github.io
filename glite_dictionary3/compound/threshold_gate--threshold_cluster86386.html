threshold gate

noun
logic gate with output based on threshold sum
Pronunciation: / ˈθrɛʃˌhoʊld ɡeɪt / (AmE), / ˈθrɛʃhəʊld ɡeɪt / (BrE)
a logic circuit element that produces an output only when the weighted sum of its inputs reaches or exceeds a certain threshold value
