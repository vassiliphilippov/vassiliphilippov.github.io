object code

noun
machine code produced by a compiler
Pronunciation: / ˈɑbdʒɛkt koʊd / (AmE), / ˈɒbdʒɛkt kəʊd / (BrE)
machine code generated by a compiler that is ready to be executed on a specific computer

Examples

object code generated by compiler
After compiling the program, the object code was loaded into memory for execution.
The linker combines multiple object code files into a final executable.
You cannot easily read object code because it consists of machine instructions.
