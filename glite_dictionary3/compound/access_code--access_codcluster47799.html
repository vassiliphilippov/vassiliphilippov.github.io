access code

noun
code for gaining entry or use
Pronunciation: / ˈæksɛs koʊd / (AmE), / ˈæksɛs kəʊd / (BrE)
a sequence of characters or digits used to gain entry or use of a secure system, device, or service

Examples

enter access code

She forgot her laptop's access code and had to contact IT support for help.
To unlock the building's main door, you need to enter your access code on the keypad.
