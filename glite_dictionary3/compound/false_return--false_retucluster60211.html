false return

noun
incorrect or fraudulent tax return
Pronunciation: / fɔls rɪˈtɝn / (AmE), / fɔːls rɪˈtɜːn / (BrE)
an incorrect or deliberately deceptive statement submitted as a tax return, especially with the intent to evade taxes or mislead authorities

Examples

file a false return
Filing a false return can result in serious legal penalties, including fines and imprisonment.
It's important to ensure accuracy to avoid accusations of making a false return.
The accountant was charged with submitting a false return to the tax authorities.
