editor program

noun
software for editing data or files
Pronunciation: / ˈɛdɪtər ˈproʊɡræm / (AmE), / ˈɛdɪtə ˈprəʊɡræm / (BrE)
A computer program designed to modify, rearrange, or delete data, such as text or code files.

Examples

Many writers rely on an editor program to check and revise their manuscripts before publishing.
