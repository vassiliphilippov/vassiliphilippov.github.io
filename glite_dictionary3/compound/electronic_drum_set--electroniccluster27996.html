electronic drum set

noun
set of drums with electronically generated sounds
Pronunciation: / ɪˌlɛkˈtrɑnɪk drʌm sɛt / (AmE), / ɪˌlɛkˈtrɒnɪk drʌm set / (BrE)
A set of electronic drums, including pads and sound modules, played like a traditional drum kit but producing electronically generated or sampled sounds.

Examples

practice on electronic drum set
He prefers to practice quietly on his electronic drum set at home.
Many beginners start with an electronic drum set because it allows for silent practice with headphones.
The band's drummer used an electronic drum set to create unique sounds during the performance.

Related Terms

  • antonym – acoustic drum set: a drum kit using traditional acoustic drums instead of electronic sounds
  • synonym – drum kit: a collection of drums and cymbals set up to be played by a single player
