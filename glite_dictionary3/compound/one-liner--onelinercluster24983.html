one-liner

noun
single-line code snippet
Pronunciation: / ˈwʌnˌlaɪnər / (AmE), / ˈwʌnˌlaɪnə / (BrE)
A single line of computer code or a short command that performs a complete task or expresses a small, useful operation.

Examples

Python one-liner
The tutorial showed a clever one-liner that sorted the list in just a single function call.
During the code review, she suggested replacing several lines with a short one-liner.
Many scripting languages let you write powerful one-liners to process text quickly on the command line.

Related Terms

  • synonym – command: an instruction given to a computer or program, often entered at a prompt
  • synonym – snippet: a small piece of computer code
