ignition system

noun
engine mechanism for igniting fuel
Pronunciation: / ɪɡˈnɪʃən ˌsɪstəm / (AmE), / ɪɡˈnɪʃ(ə)n ˌsɪstəm / (BrE)
The set of components in an internal combustion engine that produces the spark or heat needed to ignite the fuel-air mixture.

Examples

car ignition system
He replaced the old ignition system with a newer, more efficient model.
If the ignition system fails, the engine will not start properly.
Modern cars use electronic ignition systems to improve reliability and performance.
