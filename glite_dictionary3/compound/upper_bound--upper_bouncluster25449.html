upper bound

noun
maximum value of a set (math)
Pronunciation: / ˈʌpɚ baʊnd / (AmE), / ˈʌpə baʊnd / (BrE)
(mathematics) a number or value that is greater than or equal to every element in a given set.

Examples

upper bound of a set
Every finite set of real numbers has an upper bound.
If no upper bound exists, the set is said to be unbounded.
The function has an upper bound of 10 on this interval.

Related Terms

  • antonym – lower bound: a number less than or equal to every element in a set (math)
