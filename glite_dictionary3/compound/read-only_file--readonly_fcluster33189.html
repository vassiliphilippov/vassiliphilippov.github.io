read-only file

noun
file that cannot be altered
Pronunciation: / rid ˈoʊnli faɪl / (AmE), / riːd ˈəʊnli faɪl / (BrE)
(computing) a file that can be viewed or read but cannot be modified or deleted.

Examples

read-only file cannot change
If the document is saved as a read-only file, you will need permission to edit it.
Many system files are set to read-only file status to prevent accidental changes.
When you try to save the changes, the software warns that the read-only file cannot be modified.

  • synonym – write-protected: cannot be modified or written to
