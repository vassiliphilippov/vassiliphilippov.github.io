swap file

noun
file used for virtual memory
Pronunciation: / swɑp faɪl / (AmE), / swɒp faɪl / (BrE)
A file on a hard disk that is used as virtual memory to supplement a computer's physical memory (RAM).

Examples

swap file size
If your computer is slow, you may want to increase the size of the swap file.
Linux manages its virtual memory by using a swap file or a swap partition.
The operating system automatically creates a swap file when it runs low on physical memory.

Related Terms

  • synonym – page file: a similar file used in Windows systems for virtual memory
