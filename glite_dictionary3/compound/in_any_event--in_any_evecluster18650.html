in any event

adverb
regardless; anyway; in any case
Pronunciation: / ɪn ˈɛni ɪˈvɛnt / (AmE), / ɪn ˈɛni ɪˈvɛnt / (BrE)
used to say that something is true or will happen regardless of what has happened or will happen; anyway; in any case

Examples

in any event it happens
In any event, we will need to submit the report by Friday.
I don't know if the package arrived; in any event, I'll check with the mailroom.
She might be late, but in any event, the meeting will start on time.
The project has challenges; in any event, our team is prepared.
We hoped for good weather, but in any event, the festival will proceed as planned.
You might be late, but in any event, call me when you arrive.
