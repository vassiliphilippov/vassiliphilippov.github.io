chess set

noun
set of board and pieces for chess
Pronunciation: / tʃɛs sɛt / (AmE), / tʃɛs sɛt / (BrE)
A set consisting of a chessboard and the 32 pieces required to play chess.

Examples

classic chess set
He bought a beautiful wooden chess set as a gift for his father.
She carefully arranged the pieces on the chess set before the match began.
The school tournament provided a chess set for each pair of players.
