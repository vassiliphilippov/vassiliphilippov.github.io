prompt box

noun
text input for AI/chatbot prompts
Pronunciation: / pr51d0mpt b51ks / (AmE), / pr52mpt b52ks / (BrE)
(in computing) a text-input field in a graphical user interface where a user types a prompt, question, or instruction for an AI assistant, chatbot, or other interactive system.

Examples

type in the prompt box
She typed a detailed instruction in the prompt box of the AI chat app.
The website designer labeled the prompt box to guide users in submitting clear requests.
During testing, the assistant failed to respond when the user left the prompt box empty.
