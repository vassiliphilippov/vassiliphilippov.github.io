background processing

noun
execution of tasks in the background (computing)
Pronunciation: / ˈbækˌɡraʊnd ˌproʊˈsɛsɪŋ / (AmE), / ˈbækˌɡraʊnd ˌprəʊˈsɛsɪŋ / (BrE)
the execution of computer programs or tasks in the background, using the processor when higher-priority tasks are idle

Examples

background processing runs tasks
Setting up background processing allows routine maintenance tasks to run automatically.
The software supports background processing to keep the main application responsive.
With background processing, large files can be uploaded without interrupting your work.
