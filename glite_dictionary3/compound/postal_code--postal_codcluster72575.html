postal code

noun
address code for sorting mail
Pronunciation: / ˈpoʊstl ˌkoʊd / (AmE), / ˈpəʊstl ˌkəʊd / (BrE)
A code consisting of letters and numbers added to a postal address to help in the sorting and delivery of mail.
