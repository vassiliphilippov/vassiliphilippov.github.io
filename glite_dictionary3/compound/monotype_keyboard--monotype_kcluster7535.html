monotype keyboard

noun
typesetting keyboard for Monotype machines
Pronunciation: / ˈmɑːnoʊtaɪp ˈkiːbɔːrd / (AmE), / ˈmɒnətaɪp ˈkiːbɔːd / (BrE)
A specialized keyboard used in typesetting to prepare punched paper tapes (ribbons) that control a Monotype machine for casting individual pieces of type.

Examples

monotype keyboard operator
Operators used a monotype keyboard to create punched tape for casting letters.
