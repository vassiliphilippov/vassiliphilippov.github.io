false return

noun
boolean return value indicating failure
Pronunciation: / fɔls rɪˈtɜrn / (AmE), / fɔːls rɪˈtɜːn / (BrE)
(computing) a function's return value of the boolean false, used to indicate failure, a negative result, or that a requested operation was not performed.

Examples

function gives false return
When the API call fails, the function gives a false return that the caller must check.
You should not treat a false return as an error without inspecting the error log first.
Unit tests caught a false return from the validation routine when input was missing required fields.

Related Terms

  • synonym – return value: the value produced by a function or procedure when it completes, often used to convey results or status
