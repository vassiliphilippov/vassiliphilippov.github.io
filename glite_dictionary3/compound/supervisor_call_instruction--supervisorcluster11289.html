supervisor call instruction

noun
instruction interrupting a program to call the supervisor
Pronunciation: / səˈpɚvaɪzɚ kɔl ɪnˌstrʌkʃən / (AmE), / səˈpɜːvaɪzə kɔːl ɪnˌstrʌkʃən / (BrE)
a machine instruction that interrupts the currently executing program and transfers control to the system supervisor, often used to request operating system services

Examples

supervisor call instruction triggers system
A supervisor call instruction is issued to switch from user mode to supervisor mode.
Developers use the supervisor call instruction to communicate with hardware at a privileged level.
The operating system responds to a supervisor call instruction by executing the requested service.
