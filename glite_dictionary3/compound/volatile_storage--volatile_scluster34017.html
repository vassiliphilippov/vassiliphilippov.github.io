volatile storage

noun
computer memory erased when power off
Pronunciation: / ˈvɑːlətəl ˈstɔːrɪdʒ / (AmE), / ˈvɒl.ə.taɪl ˈstɔː.rɪdʒ / (BrE)
Computer memory that is erased when power is turned off, such as RAM.

Examples

volatile storage in computers
Because flash memory is not volatile storage, it retains information even if the device is turned off.
RAM is a type of volatile storage that loses its data when the computer is powered down.
When selecting components, it's important to distinguish between volatile storage and permanent memory like hard drives.

Related Terms

  • antonym – non-volatile storage: computer storage that retains data when power is off
