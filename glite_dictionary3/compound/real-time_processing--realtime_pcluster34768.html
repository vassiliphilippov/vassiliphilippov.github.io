real-time processing

noun
data processing with no perceptible delay
Pronunciation: / ˌriːəl taɪm ˈprɑːsesɪŋ / (AmE), / ˌrɪəl taɪm ˈprɒsɛsɪŋ / (BrE)
The processing of data quickly enough to keep pace with an external process, without perceptible delay.

Examples

real-time processing of data
Medical equipment often relies on real-time processing to monitor a patient's vital signs.
The new software supports real-time processing of video streams.
With real-time processing, transactions can be validated instantly.

Related Terms

  • antonym – batch processing: data processed in groups rather than continuously in real time
