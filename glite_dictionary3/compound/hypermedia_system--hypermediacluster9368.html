hypermedia system

noun
interactive system linking multimedia elements
Pronunciation: / ˌhaɪpərˈmiːdiə ˈsɪstəm / (AmE), / ˌhaɪpəˈmiːdiə ˈsɪstəm / (BrE)
A computer-based system in which various forms of media, such as text, images, and video, are interconnected and can be navigated interactively.

Examples

interactive hypermedia system
Researchers developed a hypermedia system that links relevant case studies and reference materials for medical students.
The educational website was designed as a hypermedia system to let students explore topics through videos, diagrams, and texts.
With a modern hypermedia system, users can easily jump between related information using interactive links.
