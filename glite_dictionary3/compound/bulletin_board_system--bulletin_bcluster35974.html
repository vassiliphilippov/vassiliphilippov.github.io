bulletin board system

noun
computer system for message sharing
Pronunciation: / ˈbʊlətən bɔrd ˈsɪstəm / (AmE), / ˈbʊlətɪn bɔːd ˈsɪstəm / (BrE)
a computer system that runs software allowing users to exchange messages and access shared information, especially prominent before the widespread use of the internet

Examples

old bulletin board system
Although largely replaced by the modern internet, a few bulletin board systems are still active today.
In the 1980s, many people connected to a bulletin board system using a modem and telephone line.
She posted a question about computer games on her favorite bulletin board system.
