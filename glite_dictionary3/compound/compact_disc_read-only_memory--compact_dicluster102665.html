compact disc read-only memory

noun
computer storage disc (data not rewritable)
Pronunciation: / kəmˈpækt dɪsk ˌriːd ˈoʊnli ˈmɛməri / (AmE), / kəmˈpækt dɪsk ˌriːd ˈəʊnli ˈmɛməri / (BrE)
A type of compact disc used with computers for storing large amounts of digital data that cannot be altered by the user.

Examples

compact disc read-only memory drive
Because it is a compact disc read-only memory, users cannot modify or delete its contents.
Many old computer games were sold on compact disc read-only memory in the 1990s.
The manual instructed users to insert the compact disc read-only memory into the drive to install the software.
