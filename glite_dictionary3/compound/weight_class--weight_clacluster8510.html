weight class

noun
category grouping competitors by weight
Pronunciation: / ˈweɪt ˌklæs / (AmE), / ˈweɪt ˌklɑːs / (BrE)
A category in sports, especially combat sports, that groups competitors by body weight to ensure fair competition.

Examples

weight class in boxing
In boxing, each weight class has its own champion.
Mixed martial arts has several weight classes to ensure fair fights.
The athlete had to lose a few pounds to qualify for a lower weight class.
