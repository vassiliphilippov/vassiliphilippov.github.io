push-down queue

noun
last-in, first-out data queue
Pronunciation: / ˈpʊʃ daʊn kjuː / (AmE), / ˈpʊʃ daʊn kjuː / (BrE)
A queue in which the last item to be inserted is the first to be removed (last in, first out).

Examples

push-down queue in programming
A push-down queue is often implemented using a stack in programming.
The programmer used a push-down queue to reverse the order of the elements.
Understanding how a push-down queue works is important for solving certain algorithm problems.

Related Terms

  • antonym – queue: a line or sequence of people or items awaiting their turn; in computing, first-in, first-out
  • synonym – stack: a data structure in which elements are added and removed from the same end (LIFO)
