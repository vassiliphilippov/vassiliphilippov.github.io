hexadecimal digit

noun
digit 0–9 or A–F in base 16
Pronunciation: / ˌhɛk.səˈdɛs.ɪ.məl ˈdɪdʒ.ɪt / (AmE), / ˌhɛk.səˈdɛs.ɪ.məl ˈdɪdʒ.ɪt / (BrE)
a digit or character used to represent numbers in base 16, ranging from 0 to 9 and A to F

Examples

hexadecimal digit F
Each hexadecimal digit corresponds to four binary bits in computer memory.
The color code is made up of six hexadecimal digits to represent RGB values.
When writing memory addresses, programmers often use hexadecimal digits instead of decimal numbers.
