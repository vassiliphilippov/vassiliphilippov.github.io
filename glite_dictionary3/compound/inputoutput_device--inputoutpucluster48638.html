input/output device

noun
hardware for input and output (computing)
Pronunciation: / ˈɪnpʊt ˈaʊtpʊt dɪˌvaɪs / (AmE), / ˈɪnpʊt ˈaʊtpʊt dɪˌvaɪs / (BrE)
A piece of computer hardware that is capable of both receiving input from and sending output to a computer system.

Examples

input/output device connects computer
For maximum flexibility, the laboratory purchased several new input/output devices compatible with their computer systems.
Printers with built-in scanners are considered input/output devices because they can both print and scan documents.
The touchscreen is an input/output device that allows users to both enter commands and receive visual feedback.

Related Terms

  • degree lower – input device: hardware for entering data into a computer
  • degree lower – output device: hardware for displaying or transmitting data from a computer
  • synonym – peripheral: an external device connected to a computer
