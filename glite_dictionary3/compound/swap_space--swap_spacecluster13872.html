swap space

noun
disk space used as virtual memory
Pronunciation: / swɑp speɪs / (AmE), / swɒp speɪs / (BrE)
the part of a computer's hard drive reserved for use as virtual memory when physical memory is full

Examples

increase swap space
Adding more swap space can help your system handle larger workloads.
The server ran out of swap space and started killing processes to free up memory.
When physical RAM is exhausted, the operating system uses swap space to keep programs running.

Related Terms

  • synonym – virtual memory: computer memory that appears to be main memory but is actually simulated by disk storage
