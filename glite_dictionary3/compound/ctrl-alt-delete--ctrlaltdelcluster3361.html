Ctrl-Alt-Delete

noun
keyboard command for restarting or interrupting a computer
Pronunciation: / kənˌtroʊl ɔlt dɪˈliːt / (AmE), / kənˌtrəʊl ɔːlt dɪˈliːt / (BrE)
a keyboard command using the Control, Alt, and Delete keys together, commonly used to interrupt or restart a computer or to access system functions like closing unresponsive programs

Examples

press Ctrl-Alt-Delete keys
If an application stops responding, try using Ctrl-Alt-Delete to force it to close.
My computer froze, so I pressed Ctrl-Alt-Delete to open the task manager.
She taught her grandfather how to use Ctrl-Alt-Delete when his PC became unresponsive.
