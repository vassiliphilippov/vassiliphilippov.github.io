Universal Product Code

noun
barcode system for retail products
Pronunciation: / ˌjuːnɪˈvɝːsəl ˈprɑːdʌkt koʊd / (AmE), / ˌjuːnɪˈvɜːsəl ˈprɒdʌkt kəʊd / (BrE)
a machine-readable code consisting of a series of vertical bars of varying widths, printed on product packages to identify the item and facilitate price and inventory control in retail stores

Examples

Universal Product Code on packaging
Each cereal box is marked with a Universal Product Code for easy inventory tracking.
The cashier scanned the Universal Product Code to quickly ring up the groceries.
Without a Universal Product Code, a product cannot be scanned at most supermarkets.

Related Terms

  • synonym – barcode: a machine-readable code in the form of numbers and a pattern of parallel lines
