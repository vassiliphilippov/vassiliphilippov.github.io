and so forth

adverb
used to show a list continues similarly
Pronunciation: / ænd soʊ ˈfɔrθ / (AmE), / ənd səʊ ˈfɔːθ / (BrE)
used to indicate that a list continues in a similar way, without giving all the examples

Examples

pencils, paper, and so forth
The children need pencils, paper, and so forth for the art class.
We discussed the budget, scheduling, and so forth during the meeting.
You must bring identification, proof of address, and so forth to complete the registration.

Related Terms

  • synonym – etcetera: used to show that more similar items are included but not named
