object ball

noun
the ball to be struck by the cue ball in billiards
Pronunciation: / ˈɑb.dʒɛkt bɔl / (AmE), / ˈɒb.dʒɪkt bɔːl / (BrE)
In billiards and similar cue sports, the ball that is intended to be struck first by the cue ball.

Examples

object ball in billiards
