relocatable program

noun
computer program that runs from different memory locations
Pronunciation: / ˌriːˈloʊkəteɪtəbl ˈproʊɡræm / (AmE), / ˌriːˈləʊkəteɪtəbl ˈprəʊɡram / (BrE)
a computer program that can be loaded into and run from different memory locations without modification
