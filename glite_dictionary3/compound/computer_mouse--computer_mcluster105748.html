computer mouse

noun
device for controlling a computer cursor
Pronunciation: / kəmˈpjuːt̬ɚ maʊs / (AmE), / kəmˈpjuːtə maʊs / (BrE)
A hand-operated electronic device used to control the movement of a cursor on a computer screen, typically by moving it across a flat surface.

Examples

use a computer mouse
Modern offices often provide both a computer mouse and a touchpad for each workstation.
My computer mouse stopped working, so I had to use the keyboard shortcuts instead.
She used the computer mouse to quickly navigate through the program's menus.
