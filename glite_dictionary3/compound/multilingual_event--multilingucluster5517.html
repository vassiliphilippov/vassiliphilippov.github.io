multilingual event

noun
an event involving multiple languages
Pronunciation: / ˌmʌltiˈlɪŋɡwəl ɪˈvɛnt / (AmE), / ˌmʌltɪˈlɪŋɡwɪl ɪˈvɛnt / (BrE)
An event where two or more languages are used or required for communication.

Examples

multilingual event planning
Organizing a multilingual event requires careful planning to accommodate different linguistic groups.
The multilingual event celebrated both cultural and language diversity in the community.
The international conference was a multilingual event with translation services provided for all attendees.
