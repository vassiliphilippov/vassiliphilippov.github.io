and so on

adverb
et cetera; continuing in the same way
Pronunciation: / ənd ˈsoʊ ɑn / (AmE), / ənd ˈsəʊ ɒn / (BrE)
used to indicate that a list continues in a similar manner; and so forth.

Examples

hiking, biking, and so on
She enjoys hiking, biking, swimming, and so on.
The kit includes buttons, thread, needles, and so on.
We discussed marketing strategies, target audiences, and so on.
