support vector network

noun
machine learning classifier using hyperplanes
Pronunciation: / səˈpɔrt ˈvɛktər ˈnɛtwɜrk / (AmE), / səˈpɔːt ˈvɛktə ˈnɛtwɜːk / (BrE)
