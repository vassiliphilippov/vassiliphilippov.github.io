logic bomb

noun
hidden code that activates a harmful function
Pronunciation: / ˈlɑːdʒɪk bɑm / (AmE), / ˈlɒdʒɪk bɒm / (BrE)
A piece of programming code secretly inserted into a software system that is designed to set off a malicious function when certain conditions are met.

Examples

logic bomb in software
If a logic bomb is triggered, it can cause massive disruption to essential services.
Security experts are trained to detect and neutralize logic bombs before they can do any harm.
The company suffered data loss after a disgruntled employee planted a logic bomb in their system.

Related Terms

  • synonym – Trojan horse: A program designed to breach the security of a computer system while ostensibly performing some innocuous function.
  • synonym – virus: A piece of code that can copy itself and typically has a detrimental effect, such as corrupting the system or destroying data.
  • synonym – worm: A self-replicating malware that spreads across computers, usually causing harm.
