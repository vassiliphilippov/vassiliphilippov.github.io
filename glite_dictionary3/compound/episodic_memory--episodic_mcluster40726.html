episodic memory

noun
computational memory for past events

(in artificial intelligence and cognitive science) a computational memory system that records, stores, and retrieves representations of past events (episodes) to support learning, planning, or decision-making.

Examples

robot episodic memory
The robot's episodic memory allowed it to recall yesterday's tool location to complete the task.
In the simulator, the agent updated its episodic memory after each trial to improve planning.
Researchers gave the neural network an explicit episodic memory to store past experiences for decision-making.
