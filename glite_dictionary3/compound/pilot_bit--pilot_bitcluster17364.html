pilot bit

noun
small drill bit that guides a larger one
Pronunciation: / ˈpaɪlət bɪt / (AmE), / ˈpaɪlət bɪt / (BrE)
a small drill bit used to create a starter hole to guide and position a larger drill bit accurately

Examples

pilot bit guides drilling
Before using the larger auger, a pilot bit is often employed for accuracy.
The pilot bit ensures that the main drill stays on target when starting a hole.
You should always check that your pilot bit is sharp before beginning the job.
