I/O device

noun
hardware for computer input/output
Pronunciation: / aɪ oʊ dɪˈvaɪs / (AmE), / aɪ əʊ dɪˈvaɪs / (BrE)
A piece of equipment, such as a keyboard, mouse, printer, or monitor, that is used to input data into or output data from a computer.

Examples

I/O device like a keyboard
A keyboard is considered an I/O device because it allows input to the computer.
Engineers tested the new I/O device before integrating it with existing hardware.
Every computer system relies on at least one I/O device to interact with the user.

Related Terms

  • synonym – peripheral: A device attached to a computer, such as a printer or scanner
