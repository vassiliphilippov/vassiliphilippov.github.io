interactive multimedia system

noun
a multimedia system with interactive, linked content
Pronunciation: / ˌɪntɚˈæktɪv ˌmʌltiˈmiːdiə ˈsɪstəm / (AmE), / ˌɪntərˈæktɪv ˌmʌltɪˈmiːdɪə ˈsɪstəm / (BrE)
a multimedia system that allows users to interact with and navigate between connected items of information, often combining text, audio, video, and graphics

Examples

use an interactive multimedia system for learning
Students can learn at their own pace using an interactive multimedia system that adapts to their progress.
The company developed an interactive multimedia system to train its employees more effectively.
The museum installed a state-of-the-art interactive multimedia system for its new exhibit.
