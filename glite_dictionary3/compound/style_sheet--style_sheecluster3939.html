style sheet

noun
file of rules defining webpage appearance
Pronunciation: / c8sta6al cc83it / (AmE), / c8sta6al cc83id0t / (BrE)
(computing) a file that contains rules describing the presentation and layout of a document, especially a web page (for example, a CSS file).

Examples

style sheet for web design
Web developers maintain a style sheet to ensure consistent layouts across pages.
She edited the site's style sheet to change fonts and button colors quickly.
If the browser ignores the style sheet, the page will display with default fonts and spacing.
