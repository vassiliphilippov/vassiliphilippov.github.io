rat-tail file

noun
thin, round, tapering file tool
Pronunciation: / ˈræt teɪl faɪl / (AmE), / ˈræt teɪl faɪl / (BrE)
a thin, round file with a tapering shape, resembling a rat's tail, used for filing circular openings or curves

Examples

use a rat-tail file on metal
A rat-tail file is perfect for enlarging small, round holes in metal.
He used a rat-tail file to smooth out the inside of the pipe.
The jeweler selected a rat-tail file to shape the intricate curves of the silver ring.
