blow moulding machine

noun
machine for shaping hollow plastic parts
Pronunciation: / bloʊ ˈmoʊldɪŋ məˈʃin / (AmE), / bləʊ ˈməʊldɪŋ məˈʃiːn / (BrE)
A machine used to form hollow plastic parts by shaping heated plastic using air pressure inside a mold.

Examples

blow moulding machine shapes bottles
Technicians performed regular maintenance on the blow moulding machine to ensure optimal performance.
The blow moulding machine can produce containers of various sizes by adjusting the mold.
The factory invested in a new blow moulding machine to improve its production of plastic bottles.
