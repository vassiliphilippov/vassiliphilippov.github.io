reference grid

noun
grid for locating points by coordinates
Pronunciation: / ˈrɛf.ɚ.əns ɡrɪd / (AmE), / ˈrɛf.ər.əns ɡrɪd / (BrE)
a pattern of intersecting horizontal and vertical lines on an image, map, or diagram, used to identify locations by coordinates

Examples

reference grid on a map
Coordinates on the reference grid correspond to specific points on the aerial photograph.
Engineers use a reference grid when designing building layouts for accurate positioning.
The map included a reference grid to help users find specific locations quickly.
