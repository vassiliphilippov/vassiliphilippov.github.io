contingent probability

noun
probability of an event given other events
Pronunciation: / kənˈtɪndʒənt prəˈbæbɪlɪti / (AmE), / kənˈtɪndʒənt prəˈbabɪlɪti / (BrE)
the probability that a particular event will occur if one or more specified other events have already occurred

Examples

contingent probability given an event
The contingent probability of drawing an ace from a deck increases if it's known that a king has already been removed.
To calculate the contingent probability of rain given cloudy skies, we use the appropriate formula from probability theory.
Understanding contingent probability is essential for analyzing dependent events in complex systems.
