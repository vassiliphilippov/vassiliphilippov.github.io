Highway Code

noun
official road rules (UK)
Pronunciation: / ˈhaɪ.weɪ koʊd / (AmE), / ˈhaɪ.weɪ kəʊd / (BrE)
The official set of rules and guidelines regulating the use of public roads, especially in the United Kingdom.

Examples

read the Highway Code booklet
All drivers in the UK are expected to know the Highway Code before they take their driving test.
Cyclists must also follow the rules set out in the Highway Code.
The Highway Code is updated periodically to improve road safety.
