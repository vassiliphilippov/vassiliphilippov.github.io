field-sequential color TV

noun
early color TV using sequential color fields
Pronunciation: / ˈfiːld səˈkwɛn.ʃəl ˈkʌlər tiː ˈviː / (AmE), / ˈfiːld sɪˈkwen.ʃəl ˈkʌlə ˌtiː ˈviː / (BrE)
an early type of color television in which each frame is broadcast in sequence using separate fields for the three primary colors, creating a full-color image when combined rapidly by the eye.
