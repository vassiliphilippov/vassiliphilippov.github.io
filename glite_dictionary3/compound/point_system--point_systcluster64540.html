point system

noun
system awarding points by rules
Pronunciation: / pɔɪnt ˈsɪstəm / (AmE), / pɔɪnt ˈsɪstəm / (BrE)
a method of evaluation in which points are awarded or deducted according to set rules or criteria

Examples

point system for rewards
Our school uses a point system to determine honor roll eligibility.
The driver's license point system penalizes repeat traffic offenders.
The tournament adopted a new point system to rank the teams fairly.
