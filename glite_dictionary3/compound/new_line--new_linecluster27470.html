new line

noun
operation or character to start text on next line
Pronunciation: / nuː ˈlaɪn / (AmE), / njuː ˈlaɪn / (BrE)
an operation or character in computing or printing that causes subsequent text to begin on the next line

Examples

insert a new line
Be sure to start each address on a new line when filling out the form.
In many programming languages, '\n' represents a new line character.
Pressing the Enter key will insert a new line in the text editor.

Related Terms

  • synonym – line break: the point where one line of text ends and the next begins
