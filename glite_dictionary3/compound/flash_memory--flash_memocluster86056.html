flash memory

noun
nonvolatile electronic storage memory
Pronunciation: / flæʃ ˈmɛməri / (AmE), / flæʃ ˈmɛməri / (BrE)
a type of nonvolatile electronic storage that can be electrically erased and reprogrammed

Examples

flash memory chip
Many smartphones use flash memory chips because they are reliable and compact.
The camera saves all your pictures onto its flash memory for quick access.
Upgrading the flash memory can improve a device's storage capacity.

Related Terms

  • synonym – EEPROM: electrically erasable programmable read-only memory
  • antonym – RAM: random access memory; fast, volatile memory used for temporary data
