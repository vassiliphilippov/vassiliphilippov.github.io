unary operation

noun
operation with a single operand
Pronunciation: / ˈjuːˌnɛri ˌɑpəˈreɪʃən / (AmE), / ˈjuːnəri ˌɒpəˈreɪʃən / (BrE)
an operation in mathematics or computing that acts on exactly one input or operand

Examples

unary operation on a number
Applying the absolute value function to a number is a unary operation.
In programming, the increment is a unary operation that adds one to a variable.
The negative sign is a classic example of a unary operation in arithmetic.

Related Terms

  • antonym – binary operation: an operation with exactly two operands
