board game

noun
game played on a board
Pronunciation: / bɔrd ɡeɪm / (AmE), / bɔːd ɡeɪm / (BrE)
A game that is played by moving pieces on a specially designed board according to set rules.

Examples

classic board game
Chess is perhaps the most famous board game in the world.
He collects vintage board games from the 1960s.
We spent the evening playing a board game with friends.
