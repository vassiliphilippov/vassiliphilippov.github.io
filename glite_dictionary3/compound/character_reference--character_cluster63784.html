character reference

noun
HTML/markup text entity
Pronunciation: / ˈkærəktər ˈrɛfərəns / (AmE), / ˈkærəktə ˈrɛfərəns / (BrE)
In computing (especially HTML and other markup languages), a sequence of characters used to represent a particular character or code point in text, for example named entities ( , é) or numeric references (é).
