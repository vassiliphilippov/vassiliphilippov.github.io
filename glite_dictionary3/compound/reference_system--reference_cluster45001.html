reference system

noun
system for defining positions with coordinates
Pronunciation: / ˈrɛfərəns ˌsɪstəm / (AmE), / ˈrɛfərəns ˌsɪstəm / (BrE)
A system that defines positions or locations using coordinates, often used in mathematics, navigation, and astronomy.

Examples

celestial reference system
Astronomers use a celestial reference system to map the positions of stars and planets.
In geometry, a Cartesian reference system is often used to plot points in space.
The GPS relies on a global reference system to provide accurate locations.
