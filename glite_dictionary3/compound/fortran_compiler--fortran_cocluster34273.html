Fortran compiler

noun
program translating FORTRAN code
Pronunciation: / ˈfɔrˌtræn kəmˈpaɪlər / (AmE), / ˈfɔːˌtræn kəmˈpaɪlə / (BrE)
a program that translates source code written in the FORTRAN programming language into machine code or another programming language

Examples

install a Fortran compiler
Installing a modern Fortran compiler can help run legacy scientific applications efficiently.
She spent hours debugging because her Fortran compiler produced a cryptic error message.
The old laboratory still relies on a Fortran compiler to process its simulation code.

Related Terms

  • synonym – compiler: a program that translates source code into executable code
  • antonym – interpreter: a program that executes instructions directly, translating each statement into a sequence of actions
