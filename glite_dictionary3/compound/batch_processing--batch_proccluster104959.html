batch processing

noun
sequential computer task execution
Pronunciation: / ˈbætʃ ˌproʊsɛsɪŋ / (AmE), / ˈbætʃ ˌprəʊsɛsɪŋ / (BrE)
the automatic sequential execution of a series of computer programs or jobs without manual intervention

Examples

batch processing computer jobs
In the early days of computing, batch processing was the primary mode of running programs.
Most payroll systems use batch processing to handle thousands of transactions overnight.
The IT team scheduled batch processing jobs to minimize disruption during business hours.

Related Terms

  • synonym – job: a unit of work for a computer, often run during batch processing
  • antonym – real-time processing: immediate processing of data as it is input or received
