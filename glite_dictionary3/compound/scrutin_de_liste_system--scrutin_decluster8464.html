scrutin de liste system

noun
electoral system using party lists
Pronunciation: / skruːˌtæ̃ də ˈliːst ˈsɪstəm / (AmE), / skruːˌtæ̃ də ˈliːst ˈsɪstəm / (BrE)
an electoral system based on proportional representation, in which voters select among party lists and the number elected from each list is determined by the percentage of votes each receives.

Examples

scrutin de liste system in elections
Debates about electoral reform often mention the advantages of the scrutin de liste system over majoritarian voting.
Electors in the scrutin de liste system cast their votes for party lists rather than individual candidates.
The country adopted the scrutin de liste system in order to better reflect the diversity of its political parties.

Related Terms

  • synonym – proportional representation: an electoral system in which parties gain seats in proportion to the number of votes cast for them
