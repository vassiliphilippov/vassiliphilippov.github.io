area code

noun
telephone code for a geographic region
Pronunciation: / ˈɛriə koʊd / (AmE), / ˈeəriə kəʊd / (BrE)
a sequence of usually three digits assigned to a specific geographic region, used as part of a telephone number in North America and some other countries

Examples

area code for a city
Since moving, I've had to get a new area code for my phone.
The area code helps to identify where a telephone number is located.
To call New York City, dial the area code 212 before the local number.
