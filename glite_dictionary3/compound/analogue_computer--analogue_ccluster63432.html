analogue computer

noun
computer using variable quantities
Pronunciation: / ˈænəˌlɔg kəmˈpjuːtər / (AmE), / ˈanəlɒɡ kəmˈpjuːtə / (BrE)
a computer that represents and processes information using continuously variable physical quantities, such as electrical voltages or mechanical positions, rather than discrete values

Examples

old analogue computer model
A flight simulator often uses an analogue computer to model real-world conditions.
Before digital technology became widespread, engineers relied on the analogue computer for complex calculations.
The analogue computer processes data by continuously changing voltages rather than using binary code.

Related Terms

  • antonym – digital computer: computer using discrete values, typically binary
