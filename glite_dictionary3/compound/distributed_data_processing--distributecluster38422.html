distributed data processing

noun
data processing across multiple locations
Pronunciation: / dɪˈstrɪb.ju.t̬ɪd ˈdeɪ.t̬ə ˈprɑː.ses.ɪŋ / (AmE), / dɪˈstrɪb.ju.tɪd ˈdeɪ.tə ˈprɒs.ɪs.ɪŋ / (BrE)
A form of data processing in which parts of the processing are performed at different locations, linked by communication networks.

Examples

distributed data processing across computers
Many modern applications rely on distributed data processing to handle large volumes of information in real time.
Security protocols are especially important in distributed data processing environments.
To improve efficiency, the company switched to distributed data processing across its international branches.

Related Terms

  • antonym – centralized data processing: data processing performed entirely in a single location
