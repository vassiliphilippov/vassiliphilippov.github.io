script language

noun
interpreted programming language for automation
Pronunciation: / skrɪpt ˈlæŋɡwɪdʒ / (AmE), / skrɪpt ˈlaŋɡwɪdʒ / (BrE)
A programming language designed to write scripts, which are programs that are interpreted and can automate the execution of tasks.

JavaScript is a popular script language used to add interactivity to web pages.
Learning a script language can make routine computer tasks much easier to manage.
Many system administrators write automation tools in a script language like Python.
