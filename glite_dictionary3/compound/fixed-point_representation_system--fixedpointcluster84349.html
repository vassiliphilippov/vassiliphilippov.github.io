fixed-point representation system

noun
numerical system with fixed decimal point
Pronunciation: / ˈfɪkst pɔɪnt ˌrɛprɪzɛnˈteɪʃən ˌsɪstəm / (AmE), / ˈfɪkst pɔɪnt ˌrɛprɪzɛnˈteɪʃən ˌsɪstəm / (BrE)
a system for representing numbers in which the position of the decimal point is fixed by convention, as opposed to floating-point systems

Examples

fixed-point representation system for numbers
Calculations in a fixed-point representation system can be faster on certain hardware.
Embedded systems often use fixed-point representation systems instead of floating-point for efficiency.
Engineers must carefully choose the scaling factor in a fixed-point representation system.
