push-down stack

noun
last-in, first-out data structure
Pronunciation: / ˈpʊʃ daʊn stæk / (AmE), / ˈpʊʃ daʊn stæk / (BrE)
a list or data structure in which the next item to be removed is the item most recently stored (last in, first out or LIFO)

Examples

push-down stack stores data
Recursion often relies on a push-down stack for storing intermediate results.
The computer program used a push-down stack to keep track of function calls.
When data is added to a push-down stack, it becomes the first item to be removed.
