shell stitch

noun
a crochet stitch forming a fan shape
Pronunciation: / ʃɛl stɪtʃ / (AmE), / ʃɛl stɪtʃ / (BrE)
A crochet stitch in which several stitches are worked into the same base, creating a fan-shaped pattern, also known as a fan stitch.

Examples

shell stitch pattern
Beginners often practice the shell stitch because it creates a pretty, textured effect.
She used the shell stitch to add a decorative edge to the blanket.
The pattern calls for alternating rows of shell stitch and single crochet.
