read/write memory

noun
computer memory for reading and writing data
Pronunciation: / ˌriːd ˈraɪt ˈmɛməri / (AmE), / ˌriːd ˈraɪt ˈmɛməri / (BrE)
computer memory that allows both reading and writing of data, commonly used for temporary data storage while a computer is running

Examples

read/write memory stores data
If the read/write memory is full, the system may slow down significantly.
Many devices include several gigabytes of read/write memory to improve performance.
Modern computers rely on read/write memory to store applications while they are running.
