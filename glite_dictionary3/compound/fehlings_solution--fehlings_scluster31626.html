Fehling's solution

noun
chemical test solution for sugars
Pronunciation: / ˈfeɪlɪŋz səˈluʃən / (AmE), / ˈfiːlɪŋz səˈluːʃ(ə)n / (BrE)
a chemical solution containing copper sulfate, potassium tartrate, and sodium hydroxide, used to test for the presence of reducing sugars

Examples

Fehling's solution detects sugar
In the laboratory, students learned how to use Fehling's solution to detect reducing sugars.
The doctor added Fehling's solution to the test tube to check the patient's urine for sugar.
When glucose is present, Fehling's solution changes color from blue to reddish-orange.
