dirt file

noun
collection of damaging information
Pronunciation: / dɝt faɪl / (AmE), / dɜːt faɪl / (BrE)
A collection of negative or damaging information about a person, group, or organization, often compiled to be used as leverage or for exposure.

Examples

secret dirt file
Political rivals are always looking for any dirt file that could undermine their opponents.
Rumors spread quickly after a dirt file about the celebrity leaked online.
The journalist kept a dirt file on several influential business leaders.
