device driver

noun
program enabling communication with hardware
Pronunciation: / dɪˈvaɪs ˌdraɪvər / (AmE), / dɪˈvaɪs ˌdraɪvə / (BrE)
a program that allows a computer to communicate with a specific hardware device, such as a printer or keyboard

Examples

install a device driver
After installing a new graphics card, he had to update the device driver.
Some operating systems automatically detect and install the correct device driver for your hardware.
The printer stopped working because its device driver was missing.
