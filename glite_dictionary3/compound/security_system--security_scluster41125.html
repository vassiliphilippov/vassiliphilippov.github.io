security system

noun
computing system controlling network access
Pronunciation: / sɪˈkjʊr.ə.ti ˈsɪs.təm / (AmE), / sɪˈkjʊə.rɪ.ti ˈsɪs.təm / (BrE)
A computing system or set of measures designed to enforce boundaries and control access between computer networks.

Examples

computer security system
A strong security system is essential for protecting sensitive information in cloud computing environments.
Before connecting new devices, IT staff checked that the security system was properly configured for remote access.
The company upgraded its security system to prevent hackers from breaching its network.

  • synonym – firewall: a device or software that blocks unauthorized access to a network
