Tuscan order

noun
Roman architectural order with plain columns
Pronunciation: / ˈtʌskən ˈɔrdər / (AmE), / ˈtʌskən ˈɔːdə / (BrE)
a Roman architectural order resembling the Doric order but with an unfluted column shaft and a simpler base and capital.

Examples

Tuscan order column style
Many Renaissance architects favored the Tuscan order for its simplicity and strength.
The columns in the new museum were built in the Tuscan order for a clean and sturdy appearance.
Unlike the Doric, the Tuscan order features a plain, unfluted shaft and an unadorned capital.
