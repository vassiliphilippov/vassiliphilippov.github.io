active application

noun
currently running computer program in the foreground
Pronunciation: / ˈæktɪv ˌæplɪˈkeɪʃən / (AmE), / ˈæktɪv ˌæplɪˈkeɪʃən / (BrE)
A computer program that is currently running and displayed in the foreground, allowing user interaction.

Examples

switch to the active application
If you click on another window, that program becomes the active application.
Only one active application at a time will receive keyboard input on your computer.
The toolbar changes depending on the active application you’re using.

Related Terms

  • antonym – background application: an application running but not currently in the foreground
