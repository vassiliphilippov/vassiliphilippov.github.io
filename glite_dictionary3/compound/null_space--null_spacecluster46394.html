null space

noun
mathematical subspace with only the zero element
Pronunciation: / ˈnʌl speɪs / (AmE), / ˈnʌl speɪs / (BrE)
A space that contains no elements or points; in mathematics, commonly a subspace in which a given linear transformation maps all vectors to the zero vector.

Examples

null space of a matrix
Any vector in the null space will be transformed into the zero vector by the matrix.
In linear algebra, understanding the null space is essential for solving systems of equations.
The null space of a matrix contains all solutions to the homogeneous equation.

Related Terms

  • synonym – kernel: the set of elements mapped to zero by a function or transformation
