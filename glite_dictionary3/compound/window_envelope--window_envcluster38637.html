window envelope

noun
envelope with transparent address window
Pronunciation: / ˈwɪndoʊ ˈɛnˌvəˌloʊp / (AmE), / ˈwɪndəʊ ˈɛn.və.ləʊp / (BrE)
An envelope with a transparent panel that displays the address written on the enclosed document.

Examples

window envelope for bills
Companies often send bills in a window envelope to avoid rewriting the recipient's address.
I bought a box of window envelopes for mailing invoices more efficiently.
Please place the letter inside the window envelope so the address shows through.
