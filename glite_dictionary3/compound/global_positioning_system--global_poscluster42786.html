Global Positioning System

noun
satellite-based navigation system
Pronunciation: / ˌɡloʊbəl pəˈzɪʃənɪŋ ˈsɪstəm / (AmE), / ˌɡləʊbəl pəˈzɪʃənɪŋ ˈsɪstəm / (BrE)
A navigational system based on satellites and computers that determines the exact position of a receiver on Earth by calculating signal time differences from satellites.

Examples

Global Positioning System navigation device
Many smartphones use the Global Positioning System to provide accurate map directions.
Scientists rely on the Global Positioning System for precise measurements of tectonic plate movement.
The Global Positioning System allows hikers to track their location even in remote areas.
