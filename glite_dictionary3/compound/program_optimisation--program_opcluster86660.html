program optimisation

noun
making software run more efficiently
Pronunciation: / ˈproʊɡræm ˌɑptɪmaɪˈzeɪʃən / (AmE), / ˈprəʊɡræm ˌɒptɪmaɪˈzeɪʃən / (BrE)
