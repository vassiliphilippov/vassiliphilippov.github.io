disk overhead

noun
extra disk space for managing data
Pronunciation: / dɪsk ˈoʊvərˌhɛd / (AmE), / dɪsk ˈəʊvəˌhɛd / (BrE)
The amount of disk space used for storing information that is not actual user data but is required for managing the location and timing of data on a disk.

Examples

disk overhead in storage
Reducing disk overhead can free up more space for actual data storage.
The disk overhead on encrypted drives is often higher due to added security information.
When you format a new partition, some disk overhead is reserved for the file system.

Related Terms

  • synonym – data overhead: additional data storage or processing needed for control or management purposes
