parity check

noun
error-detection method in computing
Pronunciation: / ˈpæɹəti tʃɛk / (AmE), / ˈpærɪti tʃɛk / (BrE)
a method of detecting errors in data transmission or storage by ensuring that the number of set bits, or 1s, in a set of data is even or odd, as previously agreed upon.

Examples

parity check on data
Implementing a parity check adds a basic layer of protection against simple data errors.
Most computer memory uses a parity check to detect errors in stored data.
The parity check failed, indicating that some bits were corrupted during transmission.

Related Terms

  • synonym – checksum: a value used to verify the integrity of data
