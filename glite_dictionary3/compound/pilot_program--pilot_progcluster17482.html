pilot program

noun
trial project or test initiative
Pronunciation: / ˈpaɪlət ˌproʊɡræm /; / ˈpaɪlət ˈproʊɡræm / (AmE), / ˈpaɪlət ˌprəʊɡræm /; / ˈpaɪlət ˈprəʊɡram / (BrE)
