ZIP code

noun
US postal address code
Pronunciation: / zɪp koʊd / (AmE), / zɪp kəʊd / (BrE)
A group of numbers or letters and numbers used as part of a postal address in the United States to help sort mail efficiently.

Examples

ZIP code on an envelope
Be sure to include your ZIP code on the application form.
The package was delayed because the ZIP code was incorrect.
You can find restaurants in your area by entering your ZIP code online.

Related Terms

  • synonym – postal code: a general term for codes used in postal addresses to aid mail sorting
  • synonym – postcode: a code of letters and/or numbers used for sorting mail, especially in the UK
