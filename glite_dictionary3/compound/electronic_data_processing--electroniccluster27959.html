electronic data processing

noun
automatic data processing using electronic means
Pronunciation: / ɪˌlɛkˈtrɑnɪk ˌdeɪtə ˈprɑsɛsɪŋ / (AmE), / ɪˌlɛkˈtrɒnɪk ˌdeɪtə ˈprɒsɛsɪŋ / (BrE)
the automatic processing of data using electronic equipment, especially computers, rather than mechanical means such as tabulating cards or punched tapes

Examples

electronic data processing system
Early electronic data processing revolutionized the way businesses managed large amounts of information.

Training in electronic data processing is essential for modern IT professionals.
