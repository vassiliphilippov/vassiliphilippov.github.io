dress code

noun
rules about required clothing
Pronunciation: / drɛs koʊd / (AmE), / drɛs kəʊd / (BrE)
a set of rules specifying what clothing is required or allowed in a particular place or situation

Examples

dress code at work
Many restaurants have a strict dress code for their dining areas.
Our company dress code requires employees to wear formal business attire.
The school introduced a new dress code to prevent students from wearing jeans.
