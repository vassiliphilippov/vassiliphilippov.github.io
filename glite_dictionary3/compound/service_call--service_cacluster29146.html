service call

noun
computing request to a service
Pronunciation: / ˈsɜːvɪs kɔːl / (AmE), / ˈsɜːvɪs kɔːl / (BrE)
(in computing) a request sent by a client or application to a software service asking it to perform a specific operation, typically made over a network (for example, an API call or RPC).

Examples

send a service call
The mobile app sends a service call to the server to fetch the user's profile data.
During testing, several service calls failed because the endpoint was misconfigured and returned errors.
To speed startup, the system batches multiple service calls into a single network request.

Related Terms

  • synonym – API call: a request made to an application programming interface to perform an operation or retrieve data
  • synonym – network request: a message sent over a network asking a remote system to perform an action or return data
