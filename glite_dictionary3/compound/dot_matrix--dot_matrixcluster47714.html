dot matrix

noun
rectangular grid of dots for characters/images
Pronunciation: / dɑt ˈmeɪ.trɪks / (AmE), / dɒt ˈmeɪ.trɪks / (BrE)
A rectangular grid of dots used to form characters, symbols, or images, especially in printing and display technology.

Examples

dot matrix printer
Engineers designed the display using a dot matrix to show simple graphics.
Many digital clocks display numbers using a seven-by-five dot matrix.
The old printer used a dot matrix to generate text on each page.
