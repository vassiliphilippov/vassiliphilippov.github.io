extended time scale

noun
simulation time scale with factor >1
Pronunciation: / ɪkˈstɛndɪd taɪm skeɪl / (AmE), / ɪkˈstɛndɪd taɪm skeɪl / (BrE)
(in simulation) a time scale used in data processing where the time-scale factor is greater than one, causing events to be simulated over a longer period than real time.

Examples

on an extended time scale in simulation
Analyzing the system on an extended time scale revealed rare, slow-developing phenomena.
By using an extended time scale, researchers could model processes that take years in real life.
The simulation was run on an extended time scale to observe long-term trends.
