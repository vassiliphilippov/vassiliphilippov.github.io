rock python

noun
large African python species
Pronunciation: / rɑk ˈpaɪθɑn / (AmE), / rɒk ˈpʌɪθən / (BrE)
a very large non-venomous python native to tropical and southern Africa, known for its size and strength

Examples

giant rock python
A rock python was spotted near the river, causing excitement among the villagers.
The rock python is Africa's largest snake and can grow to over six meters long.
Unlike the venomous puff adder, the rock python kills its prey by constriction.
