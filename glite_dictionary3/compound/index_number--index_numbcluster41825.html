index number

noun
statistical value showing relative change
Pronunciation: / ˈɪnˌdɛks ˈnʌmbər / (AmE), / ˈɪndɛks ˈnʌmbə / (BrE)
a statistical value, usually expressed as a number or ratio, that measures changes in a variable or group of related variables over time or between places

Examples

index number for prices
An index number makes it easier to compare economic data from different periods.
The consumer price index number reflects changes in the cost of living over the years.
The government publishes an index number every month to track inflation.
