serial processing

noun
processing tasks sequentially in order
Pronunciation: / ˈsɪriəl ˈprɑsɛsɪŋ / (AmE), / ˈsɪəriəl ˈprɒsɛsɪŋ / (BrE)
processing information or tasks in a sequence, where each step is completed before the next begins, often using a single or sequential set of units or stages.

Examples

serial processing of tasks
The laboratory experiment compared the speed of serial processing with that of parallel processing.
Traditional computers relied mostly on serial processing to solve problems step by step.
Unlike serial processing, parallel processing can handle multiple tasks at once.

  • antonym – parallel processing: processing multiple tasks or items simultaneously
