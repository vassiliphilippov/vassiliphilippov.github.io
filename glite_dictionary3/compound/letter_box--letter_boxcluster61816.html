letter box

noun
public box for mailing letters
Pronunciation: / ˈlɛt̬ɚ ˌbɑks / (AmE), / ˈlet.ə ˌbɒks / (BrE)
a public box in which letters and packages are placed for collection by the postal service

Examples

public letter box

Make sure your letter is in the letter box before the last collection.
The letter box on the corner is painted bright red.
