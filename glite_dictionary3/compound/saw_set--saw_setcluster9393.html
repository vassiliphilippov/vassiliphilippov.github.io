saw set

noun
tool for setting saw teeth
Pronunciation: / sɔ ˈsɛt / (AmE), / sɔː ˈsɛt / (BrE)
a tool used for bending each alternate tooth of a saw blade outward to set the teeth at the correct angle

Examples

saw set tool
A properly tuned saw set can greatly improve cutting efficiency.
After sharpening the blade, she reached for the saw set to finish the job.
He used a saw set to adjust the teeth of his handsaw.
