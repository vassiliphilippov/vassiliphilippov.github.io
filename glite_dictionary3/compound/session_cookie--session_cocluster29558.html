session cookie

noun
temporary internet cookie deleted when browser closes
Pronunciation: / ˈsɛʃən ˈkʊki / (AmE), / ˈseʃən ˈkʊki / (BrE)
A temporary cookie stored in a web browser that is deleted when the browser is closed, typically used to manage user sessions.

Examples

session cookie expires on close
A session cookie is deleted as soon as you end your browsing session.
If you disable session cookies, some website features may not work properly.
Some websites use a session cookie to keep you logged in until you close your browser.

Related Terms

  • antonym – persistent cookie: a cookie stored on a device for a set period and not deleted when the browser closes
