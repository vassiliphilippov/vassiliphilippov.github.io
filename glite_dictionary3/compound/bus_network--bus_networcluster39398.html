bus network

noun
network topology with devices sharing one communication line
Pronunciation: / bʌs ˈnɛtˌwɝk / (AmE), / bʌs ˈnɛtˌwɜːk / (BrE)
A network topology in which devices are connected to a single central communication line (the bus), allowing data to be transmitted between nodes in a linear arrangement.

Examples

bus network connects computers
A bus network can be susceptible to disruptions if the main cable is damaged.
Many early local area networks were based on a bus network design.
The old office used a bus network that allowed all computers to share a single cable.

Related Terms

  • antonym – ring network: Network topology where each node connects to two others, forming a ring
  • antonym – star network: Network topology with all nodes connected to a central hub
