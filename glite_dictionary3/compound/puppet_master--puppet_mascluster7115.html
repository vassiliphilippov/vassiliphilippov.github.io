puppet master

noun
central server in a Puppet deployment
Pronunciation: / ˈpʌpɪt ˌmæstər / (AmE), / ˈpʌpɪt ˌmɑːstə / (BrE)
(in information technology, especially system administration) the central server or controller in a Puppet deployment that stores configuration manifests and communicates with agent nodes to enforce system state.

Examples

puppet master server
During the outage, the puppet master failed to connect with dozens of agents across the network.
Our team updated the manifests on the puppet master before rolling changes to production servers.
The documentation explains how to migrate a legacy puppet master to the newer Puppet Server architecture.

Related Terms

  • synonym – Puppet Server: the modern server implementation that performs the same central controller role in Puppet deployments
  • synonym – controller: a system or service that coordinates and manages the behavior of other devices or software
