type genus

noun
genus giving its name to a family (biology)
Pronunciation: / taɪp ˈdʒiː.nəs / (AmE), / taɪp ˈdʒiː.nəs / (BrE)
In biology, the genus from which the name of a family or subfamily is formed, though not necessarily the most representative genus.

Examples

type genus of a family
Homo is the type genus for the family Hominidae.
In zoological studies, the type genus provides the root for the family's scientific name.
The paleontologist explained which fossil belonged to the type genus of that group.
