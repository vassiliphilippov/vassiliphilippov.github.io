choropleth map

noun
map using colors/patterns to show data values
Pronunciation: / ˈkɔːrəˌplɛθ mæp / (AmE), / ˈkɒrəˌplɛθ mæp / (BrE)
a map in which areas are shaded or patterned in proportion to the value of a variable being measured, such as population density or income

Examples

choropleth map showing population
A choropleth map can make it easy to compare unemployment rates across different regions.
The population density of each state was depicted on the choropleth map with varying shades of blue.
The project included creating a choropleth map to visualize election results at the county level.
