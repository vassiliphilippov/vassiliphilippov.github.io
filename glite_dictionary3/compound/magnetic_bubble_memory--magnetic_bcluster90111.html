magnetic bubble memory

noun
obsolete magnetic data storage device
Pronunciation: / məɡˈnɛtɪk ˈbʌbəl ˈmɛməri / (AmE), / maɡˈnɛtɪk ˈbʌb(ə)l ˈmɛm(ə)ri / (BrE)
a type of nonvolatile data storage device that stores information as tiny magnetized areas, called bubbles, on a thin film of magnetic material; now largely obsolete in computers

Examples

magnetic bubble memory device
During the 1970s, magnetic bubble memory was considered a promising technology for computer data storage.
Engineers replaced the outdated magnetic bubble memory with more efficient hard disk drives.
