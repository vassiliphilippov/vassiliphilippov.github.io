onto function

noun
mathematics: function covering every output value
Pronunciation: / ˈɑnˌtu ˈfʌŋkʃən / (AmE), / ˈɒn.tuː ˈfʌŋk.ʃən / (BrE)
a function in which every element of the codomain is the image of at least one element of the domain; a surjective function

Examples

onto function in math
A onto function guarantees that the codomain has no unused elements.
Every onto function from a set with two elements to itself is also one-to-one.
To prove the mapping is an onto function, we must show that every output is achievable.

Related Terms

  • degree higher – bijective function: a function that is both injective and surjective
  • antonym – injection: a function that is one-to-one but not necessarily onto
  • synonym – surjection: a function that is surjective (onto)
