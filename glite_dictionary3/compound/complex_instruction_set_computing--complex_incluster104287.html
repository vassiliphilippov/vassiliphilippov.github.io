complex instruction set computing

noun
computer CPU architecture with many complex instructions
Pronunciation: / kəmˈplɛks ɪnˈstrʌkʃən sɛt kəmˈpjuːtɪŋ / (AmE), / kəmˈplɛks ɪnˈstrʌkʃən sɛt kəmˈpjuːtɪŋ / (BrE)
(in computing) a type of computer architecture in which the CPU is designed to execute a large number of complex instructions directly in hardware.

Examples

complex instruction set computing architecture
Mainframes traditionally used complex instruction set computing to improve computational flexibility.
Modern processors based on complex instruction set computing can execute sophisticated commands directly in hardware.
Unlike RISC, complex instruction set computing aims to make each instruction more powerful.

Related Terms

  • antonym – reduced instruction set computing: computer architecture with fewer, simpler instructions (RISC)
