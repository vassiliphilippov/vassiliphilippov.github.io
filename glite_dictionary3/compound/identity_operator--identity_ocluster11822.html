identity operator

noun
operator leaving element unchanged
Pronunciation: / aɪˈdɛntɪti ˈɑpəˌreɪtər / (AmE), / aɪˈdɛntɪti ˈɒpəreɪtə / (BrE)
an operator in mathematics or logic that leaves an element unchanged when applied to it

Examples

identity operator on a vector
Applying the identity operator to any vector returns the same vector.
In many programming languages, the identity operator is used to compare objects for sameness.
The identity operator acts as a neutral element in certain algebraic structures.
