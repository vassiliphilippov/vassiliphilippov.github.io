cascading menu

noun
secondary menu in computer interfaces
Pronunciation: / kæsˈkeɪdɪŋ ˈmɛnju / (AmE), / kæsˈkeɪdɪŋ ˈmɛnjuː / (BrE)
a secondary menu that appears when the cursor is held over an item on a primary menu, typically to show additional options

Examples

open a cascading menu
Hovering over 'File' opens a cascading menu with more file management options.
Many modern interfaces employ the cascading menu to organize functions efficiently.
The application allows users to access advanced features through a cascading menu.

Related Terms

  • synonym – submenu: a menu that is subordinate to the main menu
