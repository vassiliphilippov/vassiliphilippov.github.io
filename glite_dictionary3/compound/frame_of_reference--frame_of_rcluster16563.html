frame of reference

noun
coordinate system for measuring position
Pronunciation: / freɪm əv ˈrɛfɚəns / (AmE), / freɪm əv ˈrɛfərəns / (BrE)
a system of coordinates used to specify the position of objects and measure physical phenomena, often relative to an observer or a context

Examples

physics frame of reference
Different observers may choose different frames of reference to analyze the same event.
In classical mechanics, a frame of reference is essential for describing motion.
The position of the object can only be determined relative to a frame of reference.

  • synonym – coordinate system: a system for specifying positions with coordinates
