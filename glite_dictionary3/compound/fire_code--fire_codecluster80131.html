fire code

noun
regulations for fire prevention and safety
Pronunciation: / ˈfaɪɚ ˌkoʊd / (AmE), / ˈfaɪə ˌkəʊd / (BrE)
A set of standards established and enforced by authorities to prevent fires and ensure safety, covering requirements such as fire escapes and alarms.

Examples

fire code violations
After the inspection, the restaurant was cited for several violations of the fire code.
Firefighters praised the apartment complex for its adherence to the city's fire code.
The new school was built to comply with the latest fire code requirements.
