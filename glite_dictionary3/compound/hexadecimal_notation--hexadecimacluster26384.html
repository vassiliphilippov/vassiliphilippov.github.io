hexadecimal notation

noun
number system using sixteen symbols
Pronunciation: / ˌhɛksəˈdɛsəməl noʊˈteɪʃən / (AmE), / ˌhɛksəˈdɛsɪməl nəʊˈteɪʃ(ə)n / (BrE)
a number system that uses sixteen distinct symbols, often the digits 0–9 and the letters A–F, to represent values

Examples

hexadecimal notation for colors
Colors in web design are usually specified using hexadecimal notation, such as #FF5733.
In computer science, hexadecimal notation is often used because it is more concise than binary.
Programmers convert binary values to hexadecimal notation to make them easier to read.
