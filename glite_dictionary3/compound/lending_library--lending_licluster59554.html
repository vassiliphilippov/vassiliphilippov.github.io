lending library

noun
library that lends books for home use
Pronunciation: / ˈlɛndɪŋ ˌlaɪbrɛri / (AmE), / ˈlɛndɪŋ ˌlaɪbrəri / (BrE)
A library that allows books to be borrowed and taken outside the building for a set period.

Examples

lending library membership card
Our city opened a new lending library in the community center last year.
She joined the lending library to borrow novels and biographies every month.
Unlike a reference library, a lending library allows you to take books home.
