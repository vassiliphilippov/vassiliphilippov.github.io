remote-access data processing

noun
distributed data processing via remote access
Pronunciation: / rɪˈmoʊt ˈæksɛs ˈdeɪtə ˌprɑsɛsɪŋ / (AmE), / rɪˈməʊt ˈæksɛs ˈdeɪtə ˌprɒsɛsɪŋ / (BrE)
A method of data processing in which some operations are carried out at different locations, connected by communication networks or transmission facilities.

Examples

remote-access data processing over networks
Remote-access data processing allows companies to centralize their databases while employees work from various locations.
Many modern organizations rely on remote-access data processing to support their distributed workforces.
With remote-access data processing, users can retrieve data from the main server through secure network connections.
