classification system

noun
method for organizing into categories
Pronunciation: / ˌklæsəfɪˈkeɪʃən ˈsɪstəm / (AmE), / ˌklɑːsɪfɪˈkeɪʃən ˈsɪstəm / (BrE)
A method or set of rules for organizing or arranging things into categories or groups based on shared characteristics.

Examples

classification system for animals
Her research focused on improving the classification system for mental health disorders.
Scientists developed a new classification system for identifying marine species.
The classification system used by the library helps patrons find books more easily.
