run batted in

noun
baseball statistic for scoring a run
Pronunciation: / rʌn ˌbætɪd ˈɪn / (AmE), / rʌn ˌbætɪd ˈɪn / (BrE)
(in baseball) a run scored as a result of a batter's action, credited to that batter as a statistic.

Examples

run batted in record
Each run batted in contributes to the team's overall score.
He set a new record for runs batted in during the championship season.
She led the league in runs batted in last year.
