master file

noun
authoritative computer file; permanent record
Pronunciation: / ˈmæstər faɪl / (AmE), / ˈmɑːstə faɪl / (BrE)
(Computing) a computer file that serves as the main authoritative record, typically remaining relatively permanent and used for reference or control.

Examples

master file stores records
After each transaction, the system automatically updates the master file to reflect the latest information.
It's critical to back up the master file regularly to prevent data loss.
The master file contains all the original data the program needs to run reports accurately.

Related Terms

  • degree lower – data file: a computer file that stores data, typically less permanent or authoritative
