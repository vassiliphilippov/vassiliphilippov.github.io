rank and file

noun
ordinary members (not leaders) of a group
Pronunciation: / ræŋk ənd faɪl /; / ˌræŋk ən ˈfaɪl / (AmE), / ræŋk ənd faɪl /; / ˌræŋk ən ˈfaɪl / (BrE)
the ordinary members of an organization or group, as opposed to its leaders

Examples

the rank and file workers
Changes in management rarely affect the rank and file as much as the higher-ups.
The rank and file were not consulted before the new policies were announced.
The union's rank and file showed strong support for the strike.
She has always fought for the rights of the rank and file within the union.
The company's rank and file were dissatisfied with the new management policies.
Unlike the executives, the rank and file employees did not receive bonuses this year.
