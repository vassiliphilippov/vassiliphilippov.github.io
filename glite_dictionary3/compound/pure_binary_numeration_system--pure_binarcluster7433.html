pure binary numeration system

noun
numerical system with base two using binary digits
Pronunciation: / pjʊr ˈbaɪˌnɛri nuˈmɛrˌeɪʃən ˈsɪstəm / (AmE), / pjʊə ˈbɪnəri njuːˈmɛrɪˌeɪʃən ˈsɪstəm / (BrE)
A positional numerical system that uses only binary digits (0 and 1) and has a base (radix) of two.
