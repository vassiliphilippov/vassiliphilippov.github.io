push-down list

noun
LIFO (last-in, first-out) data structure
Pronunciation: / ˈpʊʃ daʊn lɪst / (AmE), / ˈpʊʃ daʊn lɪst / (BrE)
a list or data structure in which the most recently added item is the first to be removed (last in, first out)

Examples

push-down list in memory
In computer science, a push-down list is often referred to as a stack.
The compiler uses a push-down list to keep track of function calls and returns.
To reverse the order of the elements, insert them into a push-down list and then remove them one by one.

  • synonym – stack: a data structure in which the last item added is the first removed
