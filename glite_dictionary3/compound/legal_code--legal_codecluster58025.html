legal code

noun
system of laws for a state or nation
Pronunciation: / ˈliɡəl koʊd / (AmE), / ˈliːɡəl kəʊd / (BrE)
A collection or system of laws officially adopted by a state or nation.

Examples

legal code of a country
Changes to the legal code can have a significant impact on society.
Law students must study the legal code in detail for their exams.
The country's legal code was updated to address new technologies.
