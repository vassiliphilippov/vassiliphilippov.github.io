trash bin

noun
virtual folder for deleted files
Pronunciation: / træʃ bɪn / (AmE), / træʃ bɪn / (BrE)
A virtual folder in an operating system or application that stores deleted files temporarily, allowing them to be restored or permanently removed later.

Examples

computer trash bin
I restored the file from the trash bin just before permanently deleting it.
The system warns that emptying the trash bin will permanently remove all files in that folder.
Her email's trash bin kept a copy of the message for thirty days before auto-deletion.

Related Terms

  • synonym – recycle bin: Windows term for the virtual folder that holds deleted files and allows restoration.
  • synonym – trash: Short form (often macOS) for the virtual folder that temporarily stores deleted items.
