web browser

noun
software for viewing web pages
Pronunciation: / wɛb ˈbraʊzɚ / (AmE), / wɛb ˈbraʊzə / (BrE)
a software application used to locate, retrieve, and display content on the World Wide Web, including web pages written in HTML.
