data formatting

noun
organizing data according to specific rules
Pronunciation: / ˈdeɪtə ˌfɔrmætɪŋ / (AmE), / ˈdeɪtə ˌfɔːmætɪŋ / (BrE)
the process of organizing data according to a predefined structure or set of specifications, especially for computer processing or storage

Examples

data formatting for files
Mistakes in data formatting can lead to errors when importing files into a database.
Proper data formatting is essential for ensuring compatibility between different computer systems.
The software provides several data formatting options for exporting reports.
