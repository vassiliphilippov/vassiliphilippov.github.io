indirect request

noun
implied expression of a request
Pronunciation: / ˌɪn.dəˈrɛkt rɪˈkwɛst / (AmE), / ˌɪn.dɪˈrɛkt rɪˈkwɛst / (BrE)
a statement or question that implies a request without stating it directly

Examples

make an indirect request
His sigh was interpreted as an indirect request for a break.
She made an indirect request for help by mentioning how tired she was.
The teacher used an indirect request when she asked if anyone knew what time it was.
