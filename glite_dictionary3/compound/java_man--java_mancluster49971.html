Java man

noun
early human fossil discovered in Java
Pronunciation: / ˈdʒɑː.və mæn / (AmE), / ˈdʒɑː.və mæn / (BrE)
an early human (Homo erectus) whose fossil remains were discovered in Java, Indonesia, and formerly known as Pithecanthropus erectus.

Examples

Java man fossils
Fossils of Java man were first uncovered by Eugène Dubois in the 1890s.
Many anthropologists study Java man to learn about early human migration out of Africa.
The discovery of Java man changed scientists' understanding of human evolution.

Related Terms

  • synonym – Homo erectus: an extinct species of human, to which Java man belongs
