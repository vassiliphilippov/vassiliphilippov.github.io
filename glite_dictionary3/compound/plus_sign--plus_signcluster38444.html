plus sign

noun
UI icon for add/create
Pronunciation: / plʌs saɪn / (AmE), / plʌs saɪn / (BrE)
A graphical symbol (usually a +) used in software interfaces as a button or icon to add or create a new item.

Examples

tap plus sign
In the app, tap the plus sign at the bottom to create a new note.
The designer chose a green plus sign icon to clearly indicate the add action.
When managing contacts, press the plus sign next to a group to add a member.

Related Terms

  • synonym – add button: a control in a user interface used to add or create an item
  • antonym – minus sign: the symbol (−) often used to remove or collapse items; opposite function in many interfaces
