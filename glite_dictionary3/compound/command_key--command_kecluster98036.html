command key

noun
a keyboard key for shortcuts and commands
Pronunciation: / kəˈmænd ki / (AmE), / kəˈmɑːnd kiː / (BrE)
a key on a computer keyboard, especially on Apple computers, that is used with other keys to perform specific commands or shortcuts

Examples

press the command key
The command key is unique to Apple keyboards.
To copy text, hold down the command key and press C.
You can use the command key to access many shortcuts in macOS.

Related Terms

  • synonym – control key: a keyboard key used with others to perform commands
