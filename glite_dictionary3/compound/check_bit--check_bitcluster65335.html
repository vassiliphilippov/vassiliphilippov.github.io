check bit

noun
bit for error detection in computing
Pronunciation: / ˈtʃɛk bɪt / (AmE), / ˈtʃɛk bɪt / (BrE)
a bit added to a group of binary digits to help detect errors in data transmission or storage by ensuring the total number of 1's meets a specific parity condition

Examples

check bit for errors
A single check bit can be used to detect errors in each byte of transmitted data.
If the check bit does not match the expected parity, the system signals an error.
The most common use of a check bit is in memory storage devices to ensure data integrity.

Related Terms

  • synonym – parity bit: a bit used specifically to enforce parity for error detection
