dead set on

adjective
firmly determined to do something
Pronunciation: / dɛd sɛt ɑn / (AmE), / dɛd sɛt ɒn / (BrE)
determined or firmly resolved to achieve or do something

Examples

The company is dead set on expanding into new markets despite economic uncertainty.
