to go native

verb
choose native platform development
Pronunciation: / ɡoʊ ˈneɪtɪv / (AmE), / ɡəʊ ˈneɪtɪv / (BrE)
(computing, informal) to develop software specifically for a particular platform using that platform's native language, APIs, or compiled code instead of using cross-platform or interpreted frameworks.

Examples

to go native with software
The team decided to go native on Android to improve graphics performance and battery life.
After testing the cross-platform build, she recommended going native to access low-level device features.
They went native with separate iOS and Android apps to meet platform-specific UX expectations.

Related Terms

  • antonym – cross-platform development: developing software to run on multiple platforms from a single codebase, often using abstraction layers or frameworks
  • synonym – native development: the practice of writing software specifically for a single platform using its native tools and APIs
