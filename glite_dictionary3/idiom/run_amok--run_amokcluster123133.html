to run amok

verb
to behave wildly and uncontrollably
Pronunciation: / rʌn əˈmʌk / (AmE), / rʌn əˈmʌk / (BrE)
to behave in a wild, frenzied, or uncontrolled manner, often causing chaos or destruction

Examples

run amok in the streets
After the teacher left the room, the students began to run amok and shout loudly.
During the festival, excitement made the children run amok around the streets.
If rumors are allowed to run amok, they can damage reputations quickly.
