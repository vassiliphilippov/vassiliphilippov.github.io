running on Mickeysoft

x
humorous phrase for unreliable software
Pronunciation: / ˈrʌnɪŋ ɑn ˈmɪkiˌsɑft / (AmE), / ˈrʌnɪŋ ɒn ˈmɪkiˌsɒft / (BrE)
(humorous, often derogatory) Used to describe a computer or system that is operating with unreliable or poorly performing software, especially by likening it to 'Microsoft' software in a satirical manner.

Examples

running on Mickeysoft after errors
After the third error message this morning, he said his phone must be running on Mickeysoft.
Don't trust that server—it's always been running on Mickeysoft and never stable.
Ever since the update, my old laptop feels like it's running on Mickeysoft, crashing every hour.

  • synonym – crash-prone: likely to stop working suddenly (of software)
