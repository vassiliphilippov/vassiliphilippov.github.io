end run around

noun
an evasive action to avoid a problem
Pronunciation: / ˈɛnd ˈrʌn əˈraʊnd / (AmE), / ˈɛnd ˈrʌn əˈraʊnd / (BrE)
an indirect or evasive action designed to avoid a problem, rule, or obstacle

Examples

pull an end run around rules
By coordinating with another department, she pulled off an end run around the budget restrictions.
He made an end run around his manager to get approval from the director.
The company tried to make an end run around the new regulations by exploiting a loophole.

Related Terms

  • synonym – circumvent: to find a way around an obstacle or rule
  • antonym – confront: to face a problem directly rather than avoid it
