to relativise

verb
convert absolute path to relative (computing)
Pronunciation: / \u02c8r\u025bl\u0259\u02cct\u026ava\u026az / (AmE), / \u02c8r\u025bl\u0259t\u026ava\u026az / (BrE)
(in computing) to convert an absolute file path or URL into a path that is relative to a specified base path, so the reference works within a different location or environment.

Examples

to relativise a file path
The build tool will relativise absolute paths to make the project portable across machines.
She relativised the image URL against the site root before deploying the website.
When packaging the app, developers often avoid hard-coded paths by relativising file references.

Related Terms

  • synonym – convert: to change something from one form or format to another (general technical synonym).
  • synonym – relativize: American spelling; same technical meaning of converting an absolute path to a relative one.
