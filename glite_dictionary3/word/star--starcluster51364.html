star

noun
network topology with components connected to a hub
Pronunciation: / stɑr / (AmE), / stɑː / (BrE)
A network topology in which all components are individually connected to a central hub or switch.

Examples

star network layout
If the hub fails in a star topology, the entire network is affected.
In a star network, each computer is directly linked to a central hub.
Many modern offices use a star configuration for their local area networks.

Related Terms

  • antonym – bus: a network topology where each node is connected to a single communication line
  • antonym – ring: a network topology where each component is connected to two others, forming a closed loop
