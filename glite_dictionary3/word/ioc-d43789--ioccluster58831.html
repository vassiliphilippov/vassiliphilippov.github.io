IOC

noun
cybersecurity indicator of compromise
Pronunciation: / a6a o8a c8si / (AmE), / a6a 598a c8sid0 / (BrE)
(in cybersecurity) an abbreviation for 'indicator of compromise': a forensic artifact (for example, a file hash, IP address, domain, or URL) that suggests a system has been breached or is malicious.

Examples

IOC alert
The security analyst found a suspicious file hash and reported it as a new IOC to colleagues.
Threat intelligence platforms automatically share new IOCs so teams can block malicious infrastructure quickly.
Before blocking, analysts verify each IOC to reduce false positives and avoid disrupting normal traffic.
