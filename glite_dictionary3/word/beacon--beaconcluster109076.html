beacon

noun
digital tracking image or script
Pronunciation: / c8bid0k59n / (AmE), / c8bid0k59n / (BrE)
a tiny, often invisible image or piece of code embedded in an email or webpage that reports when the content is viewed, used to collect analytics or track opens and visits.

Examples

email beacon tracker
The marketing email included a hidden beacon to track which recipients opened the message.
Web developers use a tiny beacon that requests a server image to measure page views precisely.
Privacy advocates warned users that some free sites embed a beacon to monitor browsing behavior.

Related Terms

  • synonym – tracking pixel: a single-pixel image embedded in content to track when it is loaded by a user's browser
  • synonym – web beacon: a small graphic on a web page that notifies a server when the page is viewed
