mode

noun
most frequent value in a data set
Pronunciation: / moʊd / (AmE), / məʊd / (BrE)
the value that appears most frequently in a set of data

Examples

mode of numbers
If you list the numbers 1, 2, 2, 3, and 4, the mode is 2.
In statistics, the mode provides information about the most common observation in a data set.
The mode of the survey responses was 'yes', meaning most people agreed.

Related Terms

  • synonym – mean: the average value of a set of numbers
  • synonym – median: the middle value in a set of ordered data
