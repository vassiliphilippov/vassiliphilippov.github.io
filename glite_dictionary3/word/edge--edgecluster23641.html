edge

noun
local/near-device computing (edge computing)
Pronunciation: / ɛdʒ / (AmE), / ɛdʒ / (BrE)
(in computing) the part of a network or computing architecture that is close to data sources or end devices, where data is processed locally to reduce latency, save bandwidth, or improve responsiveness.

Examples

edge computing devices
Many IoT deployments rely on edge computing to reduce latency.
The company moved data processing to the edge to speed up real-time analytics.
Deploying models at the edge saved bandwidth and improved response times for devices.

Related Terms

  • antonym – cloud computing: centralized computing services provided by remote servers; often contrasted with edge computing
  • synonym – fog computing: a related decentralized computing model that extends cloud services to the network edge and intermediate nodes
