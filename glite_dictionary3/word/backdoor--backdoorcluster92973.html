backdoor

noun
secret access to a computer system
Pronunciation: / ˈbækˌdɔɹ /; / ˈbækdɔr / (AmE), / ˈbækˌdɔː /; / ˈbækdɔː / (BrE)
a hidden or secret way to access a computer system or software, often used to bypass security

Examples

install a backdoor program
Security experts warn that any backdoor can be exploited by malicious actors.
The hackers gained entry through a backdoor left in the server’s operating system.
The software developer denied intentionally installing a backdoor in the program.
The research team discovered a backdoor in the image classifier triggered by a subtle sticker.
Attackers planted a hidden backdoor that caused the chatbot to leak private conversation snippets.
Before deployment, the engineers scanned the model for any unusual triggers or embedded backdoor.

  • synonym – trojan: a type of malicious program that hides its true intent and can include hidden triggers or payloads
