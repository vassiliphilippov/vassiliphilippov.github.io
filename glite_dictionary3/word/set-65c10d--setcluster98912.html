set

noun
scenery and props for a play or film
Pronunciation: / sɛt / (AmE), / sɛt / (BrE)
the scenery and other objects arranged on a stage or film set to represent a particular location for a dramatic production

Examples

the set on stage
After the show, the crew quickly dismantled the set for the next performance.
The set for the new musical was both elaborate and realistic.
The director wanted the set to reflect the atmosphere of 1920s Paris.
