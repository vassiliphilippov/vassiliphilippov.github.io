decorator

noun
software wrapper that modifies behavior

(in programming) a construct that wraps or augments a function, method, or object to modify or extend its behavior without changing the original code.

Examples

Python decorator
In Python, a decorator wraps a function to add behavior like logging or permission checks.
The team used multiple decorators to register each view with authentication and caching layers.
When debugging, she removed a decorator to see the original function's return values clearly.
