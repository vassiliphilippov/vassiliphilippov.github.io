normaliser

noun
software/hardware data-normalization tool
Pronunciation: / ˈnɔr.mə.laɪ.zər / (AmE), / ˈnɔː.mə.laɪ.zə / (BrE)
(in computing and data processing) a software or hardware component that standardizes or scales input values, features, or channels so they conform to a chosen format or numeric range.

Examples

data normaliser tool
The data pipeline uses a normaliser to scale each feature before training the neural network.
Our audio engineer connected the new normaliser to prevent sudden volume spikes during recording sessions.
After ingesting logs, the ETL job applies a normaliser to standardize timestamps and numeric fields.

Related Terms

  • synonym – normalizer: American spelling variant used for the same technical component.
  • synonym – scaler: a function or tool that rescales numeric features, especially in machine learning preprocessing.
