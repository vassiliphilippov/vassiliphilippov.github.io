microcode

noun
coded instructions in a computer processor
Pronunciation: / ˈmaɪkroʊˌkoʊd / (AmE), / ˈmaɪkrəʊˌkəʊd / (BrE)
coded instructions stored in the memory of a computer's processor that control its basic operations.

Examples

microcode update
Engineers analyzed the microcode to improve the CPU's efficiency.
Modern processors rely on complex microcode to execute advanced instructions.
The processor's microcode can be updated to fix certain hardware bugs.
