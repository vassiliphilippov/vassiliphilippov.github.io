save-all

noun
software command to save all files
Pronunciation: / ˈseɪvˌɔl / (AmE), / ˈseɪvˌɔːl / (BrE)
a command or function in software that saves all currently open documents or files at once.

Examples

save-all command
Before compiling, he clicked the save-all command to ensure every file was up to date.
The IDE shows a blue icon for save-all so developers don't lose unsaved changes.
She assigned a keyboard shortcut to save-all and now saves all tabs in one keystroke.

Related Terms

  • synonym – save: to record or write changes to a document or file so they are not lost
