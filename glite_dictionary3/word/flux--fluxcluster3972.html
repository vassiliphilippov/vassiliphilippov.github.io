flux

noun
software architecture for unidirectional data flow
Pronunciation: / fl8cks / (AmE), / fl8cks / (BrE)
(in software engineering) a design pattern for front-end applications that enforces unidirectional data flow to manage state and updates predictably.

flux architecture pattern
Developers adopted the flux architecture to manage unidirectional data flow in their web apps.
