to pickle

verb
serialize an object (computing)
Pronunciation: / ˈpɪkəl / (AmE), / ˈpɪkəl / (BrE)
(in computing, especially Python) to convert an in-memory object into a byte stream so it can be stored or transmitted and later reconstructed.

Examples

to pickle data
The server pickled the session data before sending it to the backup service.
Developers often pickle complex objects to store them between program runs.
To speed testing, she started pickling sample responses and reloading them locally.

Related Terms

  • synonym – dump: to write a serialized representation of an object to a file or stream
  • synonym – serialize: to convert data structures or objects into a format that can be stored or transmitted
