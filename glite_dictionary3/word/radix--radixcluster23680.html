radix

noun
base of a numeration system
Pronunciation: / ˈreɪ.dɪks / (AmE), / ˈreɪ.dɪks / (BrE)
the base number of a numeration system, indicating the value of each digit position (e.g. 10 in the decimal system, 2 in binary)

Examples

radix of a number system
In the binary system, the radix is 2, so each digit represents a power of two.
The hexadecimal system uses a radix of 16 to represent numbers.
When converting numbers, it's important to specify the radix to avoid confusion.
