to cross-index

verb
create references between index entries
Pronunciation: / krɔs ˈɪndɛks / (AmE), / krɒs ˈɪndɛks / (BrE)
to create or include references in an index that direct the reader from one entry to others related to it

Examples

to cross-index related topics
The database allows users to cross-index entries for comprehensive searches.
These medical records are cross-indexed for quick retrieval based on multiple criteria.
To make the catalog more useful, the librarian decided to cross-index all the new subjects.

  • synonym – index: to list or catalog systematically
