bearer

noun
credential (bearer token) for access
Pronunciation: / \u02c8b\u025br\u0259r / (AmE), / \u02c8be\u0259r\u0259 / (BrE)
(in computing, security) a credential (often called a bearer token) presented to a service to prove access rights; whoever possesses the credential can use it.

Examples

bearer token access
The API accepts a bearer token in the Authorization header to authenticate requests.
Developers warned that storing a bearer token insecurely could let attackers access user accounts.
During testing she used several bearer tokens to simulate different permission levels for the app.

Related Terms

  • synonym – credential: a piece of information used to prove identity or grant access
  • synonym – token: a string or piece of data used to authenticate or authorize access to a service
