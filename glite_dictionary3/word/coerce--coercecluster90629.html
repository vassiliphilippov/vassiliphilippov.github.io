to coerce

verb
convert data types (computing)
Pronunciation: / koʊˈɜrs / (AmE), / kəʊˈɜːs / (BrE)
(in computing) to convert a value from one data type to another, often automatically or implicitly performed by a programming language or runtime.

Examples

to coerce a value
JavaScript will coerce '5' to the number 5 when using loose equality (==).
TypeScript prevents the compiler from coercing types implicitly without an explicit cast.
The function coerced the input into a float to ensure consistent arithmetic results.

Related Terms

  • synonym – cast: to explicitly change a value from one data type to another in programming
  • synonym – convert: to change something into a different form; in computing, to change data from one type to another
