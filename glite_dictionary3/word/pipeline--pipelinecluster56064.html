pipeline

noun
sequence of processing stages; workflow
Pronunciation: / c8pa6apccla6an / (AmE), / c8pa6ap.la6an / (BrE)
A set of connected stages or processes through which data, tasks, or materials move and are transformed; a workflow or processing system.

Examples

data pipeline
The company's data pipeline transforms raw logs into actionable insights in real time.
We added a caching layer to the analytics pipeline to reduce processing latency during peak hours.
The hiring pipeline moves candidates from application to interview and finally to offer decisions.
