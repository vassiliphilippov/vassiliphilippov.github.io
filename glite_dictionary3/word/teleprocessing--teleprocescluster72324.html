teleprocessing

noun
data processing across connected locations
Pronunciation: / ˌtɛləˌprɑsɛsɪŋ / (AmE), / ˌtɛlɪˌprəʊsɛsɪŋ / (BrE)
Data processing in which some tasks are carried out in different locations connected by transmission facilities, often via telecommunications.

Examples

teleprocessing system network
Many banks use teleprocessing to handle transactions from their nationwide branches.
The company upgraded its teleprocessing system to improve communication between remote offices.
With teleprocessing, data can be entered in one location and processed in another.
