console

noun
control panel for technical systems
Pronunciation: / ˈkɑnˌsoʊl / (AmE), / ˈkɒn.səʊl / (BrE)
a panel or unit with controls, displays, and input devices used for monitoring and operating a system, especially a computer or technical system

Examples

console control panel
Each game system comes with its own console for the user to interact with.
She quickly typed commands into the main console to override the system error.
The technician sat at the console monitoring the progress of the experiment.

Related Terms

  • synonym – control panel: an interface with controls for operating a machine or system
  • synonym – dashboard: an information display panel for monitoring systems
