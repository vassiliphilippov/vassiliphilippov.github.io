to set

verb
adapt (text) for musical or performance use
Pronunciation: / sɛt / (AmE), / sɛt / (BrE)
to adapt or arrange something for performance or display (such as music, text, or hair)

Examples

to set a poem to music
Many poets have had their works set to classical compositions over the centuries.
The composer decided to set Shakespeare's sonnet to music for the choir.
The playwright set the ancient story as a modern musical.
She spent an hour having her hair set at the salon.
The chef set the table beautifully before the guests arrived.
The florist set the flowers in a delicate arrangement for the event.
She carefully set the title in bold type at the top of each page.
The designer is setting the text in italics for emphasis.
The printer will set the manuscript in a clear, readable font.

Related Terms

  • synonym – arrange: to adapt or put together music for performance
  • synonym – dress: to style or arrange (especially hair) carefully
  • synonym – typeset: arrange text (as by a typesetter) for printing or publishing
