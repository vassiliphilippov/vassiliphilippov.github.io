FTP

noun
file transfer network protocol
Pronunciation: / ˌɛf ˌti ˈpi / (AmE), / ˌɛf ˌtiː ˈpiː / (BrE)
a standard network protocol used to transfer files between computers over a network

Examples

FTP server setup
I set up an FTP server to share documents with remote colleagues.
Many companies block FTP because it is not secure by default.
You can use FTP to upload your website files to the server.

Related Terms

  • synonym – SFTP: a secure version of FTP using SSH for encryption
