to Latinize

verb
write in the Latin alphabet
Pronunciation: / ˈlæt.əˌnaɪz / (AmE), / ˈlæt.ɪˌnaɪz / (BrE)
to write or represent (a word, text, or language) using the Latin alphabet instead of another script

Examples

to Latinize a name
Historians often Latinize names from ancient Greek manuscripts for easier study.
It is common to Latinize foreign brand names for international markets.
The Japanese word was Latinized so that English speakers could read it.

Related Terms

  • synonym – romanize: to write using the Roman (Latin) alphabet
