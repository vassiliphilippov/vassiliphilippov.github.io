image

noun
digital copy of a storage device
Pronunciation: / ˈɪmɪdʒ / (AmE), / ˈɪmɪdʒ / (BrE)
A digital file that contains a complete copy of the contents and structure of a storage medium (such as a CD, DVD, or hard drive), used for backup, distribution, or mounting (for example, an ISO or DMG file).

Examples

ISO image file
They downloaded an ISO image of Linux to install the distribution.
Before upgrading, she created a full image of the hard drive for safety.
The technician mounted the image and examined its files for missing data.

Related Terms

  • synonym – ISO image: a disk image in ISO 9660 format commonly used for optical discs
  • synonym – disk image: a complete copy of a disk's contents and structure stored in a single file
