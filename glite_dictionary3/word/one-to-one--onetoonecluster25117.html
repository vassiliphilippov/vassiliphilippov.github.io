one-to-one

adjective
correspondence between two sets, each element paired uniquely
Pronunciation: / ˌwʌn tə ˈwʌn / (AmE), / ˌwʌn tə ˈwʌn / (BrE)
(of a relationship) involving the association of each member of one set with exactly one member of another set, and vice versa.

Examples

one-to-one relationship
A one-to-one mapping ensures that each input has a unique output.
The teacher provided one-to-one support to help the student succeed.
There is a one-to-one correspondence between the letters of this code and numbers.

Related Terms

  • synonym – bijective: relating to a function that is both one-to-one and onto
  • antonym – many-to-one: a relation where multiple elements from one set may correspond to a single element in another set
