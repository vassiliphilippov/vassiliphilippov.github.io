mesh

noun
number of openings in a screen per inch
Pronunciation: / mɛʃ / (AmE), / mɛʃ / (BrE)
a measure of fineness indicating the number of openings per linear inch in a screen, used to specify the size of particles that can pass through

Examples

fine mesh size
A finer mesh produces smaller particle sizes in the final product.
The manufacturer specifies a 60 mesh for filtering impurities.
The powder must pass through a mesh of 200 to be suitable for this use.
