query

noun
request for data (computing)
Pronunciation: / c8kw6ari / (AmE), / c8kw6a59ri / (BrE)
(computing) a formal request for information from a database, search engine, or information system, often expressed in a query language or as search terms.

Examples

run a query
The analyst ran a complex SQL query to extract sales data for the quarter.
She refined her search query to get more relevant articles from the database.
When the server timed out, the application retried the failed query after a short delay.

Related Terms

  • synonym – request: an act of asking for something (in computing, a call for data or service)
  • synonym – search query: a string of words or parameters submitted to a search engine or database to find information
