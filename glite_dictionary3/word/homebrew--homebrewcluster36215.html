homebrew

noun
hobbyist-made software or hardware
Pronunciation: / ˈhoʊmˌbru / (AmE), / ˈhəʊmˌbruː / (BrE)
Software or hardware created by hobbyists for consumer electronics or game consoles, typically unofficial and shared by enthusiasts.

Examples

homebrew game software
He downloaded a new homebrew app for his handheld game console last night.
The developer posted several homebrew plugins that add features to the smart TV.
At the meetup, enthusiasts swapped tips for creating homebrew hardware for vintage consoles.
