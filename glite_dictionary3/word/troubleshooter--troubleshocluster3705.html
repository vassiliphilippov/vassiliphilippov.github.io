troubleshooter

noun
automated diagnostic tool
Pronunciation: / c8tr8cblcc83ud0t59r / (AmE), / c8tr8cblcc83ud0t59 / (BrE)
a computer program or automated tool that detects, diagnoses, and sometimes fixes problems in hardware, software, or networks.

Examples

run the troubleshooter
The operating system's troubleshooter scanned the network and fixed the printer's connectivity problem.
Before calling IT, she ran the Windows troubleshooter that diagnosed the audio driver error automatically.
Many routers include a web-based troubleshooter to check connection settings and resolve simple faults.

Related Terms

  • synonym – diagnostic tool: a program or utility used to check and report the state of hardware or software
  • synonym – troubleshooting utility: a software utility designed to find and often repair common system problems
