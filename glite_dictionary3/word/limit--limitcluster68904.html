limit

noun
value a function approaches in mathematics
Pronunciation: / ˈlɪmɪt / (AmE), / ˈlɪmɪt / (BrE)
the value that a function or sequence approaches as the input or index approaches some value, often infinity, in mathematics

Examples

limit of a function
As x increases, the limit of the function approaches 5.
In calculus, determining the limit of a sequence is an essential skill.
