hash mark

noun
the '#' symbol; number sign/hashtag
Pronunciation: / hæʃ mɑrk / (AmE), / hæʃ mɑːk / (BrE)
the typographic symbol '#' used to mark numbers, create hashtags on social media, or represent the hash/number sign in computing and telephony.

Examples

hash mark on a keyboard
She added the hash mark before the keyword to create a trending hashtag on the platform.
On the old phone keypad, users pressed the hash mark after entering the access code.
Programmers use the hash mark to start a comment line in some scripting languages.

Related Terms

  • synonym – hash: the '#' character; an alternative name for the symbol used in computing and social media
  • synonym – number sign: the symbol '#' used to indicate numbers or 'No.'
  • synonym – pound sign: a regional name (especially in US English) for the symbol '#' used on phones and in informal writing
