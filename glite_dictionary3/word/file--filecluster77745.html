to file

verb
to place or organize documents for record-keeping
Pronunciation: / faɪl / (AmE), / faɪl / (BrE)
to place documents or records in an organized container or system so that they can be found easily later

Examples

to file papers
Could you file these invoices in the cabinet?
Please file the enrollment forms in alphabetical order.
She carefully filed all the receipts for future reference.

Related Terms

  • synonym – archive: to store documents or records in a long-term storage system
  • antonym – retrieve: to get back a record that has been filed
