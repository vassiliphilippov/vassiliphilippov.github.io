to address

verb
write an address on mail
Pronunciation: / əˈdrɛs, æˈdrɛs / (AmE), / əˈdrɛs, æˈdrɛs / (BrE)
to write or print the address of the recipient on (an envelope, package, etc.)

Examples

to address an envelope
Please address the envelope clearly so the letter reaches the right person.
She forgot to address the package before mailing it.
You must address each invitation to the correct guest.
