to interpolate

verb
estimate a value within known values
Pronunciation: / ɪnˈtɜːr.pə.leɪt / (AmE), / ɪnˈtɜː.pə.leɪt / (BrE)
To estimate or calculate a value within two known values in a sequence of values.

Examples

to interpolate a value
Given two temperature readings, she decided to interpolate the value at noon.
The software can automatically interpolate intermediate frames to create smoother animations.
We can interpolate the missing data points using the values from before and after the gap.

Related Terms

  • antonym – extrapolate: to estimate a value outside the known range
