luau

noun
Roblox scripting language
Pronunciation: / ˈlu.aʊ / (AmE), / ˈluː.aʊ / (BrE)
A scripting language developed by Roblox, used to write game logic and scripts for Roblox games; a dialect/variant of the Lua language.

Examples

luau scripting language
Developers write game logic in luau to run on Roblox.
She learned luau to add interactive features and improve player experience in her studio's games.
The tutorial explained how luau differs from Lua and why Roblox modified performance and type features.
