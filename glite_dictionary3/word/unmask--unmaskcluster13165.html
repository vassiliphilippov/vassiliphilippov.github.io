to unmask

verb
reveal characters in a masked input
Pronunciation: / ʌnˈmæsk / (AmE), / ʌnˈmɑːsk / (BrE)
(in computing, especially user interfaces) to disable character masking in an input field so the actual characters (for example a password) are shown.

Examples

to unmask a password
She clicked the eye icon to unmask her password in the login field.
The developer unmasked the input during testing so teammates could verify the authentication flow.
Accidentally unmasking credentials in a screenshot led to a serious security incident last month.

Related Terms

  • antonym – mask: to hide or replace characters (for example in a password field) so they are not visible
  • synonym – reveal: to make visible or show something that was hidden
