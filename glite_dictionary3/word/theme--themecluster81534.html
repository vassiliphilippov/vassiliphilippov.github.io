theme

noun
visual design package for software
Pronunciation: / b8im / (AmE), / b8id0m / (BrE)
a packaged set of visual design elements for software or a website—such as colors, fonts, icons, and layout—that can be applied to change its appearance.

Examples

website theme design
The WordPress theme controls colors, fonts, and layout for the entire website.
She downloaded a dark theme to reduce eye strain when using the app at night.
Many phones let you buy new themes from the store to change icons and wallpapers.

Related Terms

  • synonym – skin: a user interface appearance package that changes the look of software
  • synonym – template: a predesigned layout used to create consistent pages or documents
