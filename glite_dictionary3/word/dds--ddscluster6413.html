DDS

noun
real-time publish–subscribe middleware
Pronunciation: / di di ɛs / (AmE), / diː diː ɛs / (BrE)
An acronym for Data Distribution Service, a middleware standard for real-time, data-centric publish–subscribe communication in distributed systems.

Examples

DDS protocol
The robotics system relies on DDS for real-time, reliable data sharing between modules.
During testing the team used DDS to coordinate sensor data across multiple autonomous vehicles.
Engineers chose DDS because it provides fine-grained quality-of-service controls for time-critical messages.

Related Terms

  • synonym – middleware: software that connects separate components or applications to enable communication and data exchange
  • synonym – publish–subscribe: a messaging pattern where senders publish messages to topics and receivers subscribe to receive them
