fluoroscope

noun
X-ray machine with fluorescent viewing screen
Pronunciation: / ˈflʊrəˌskoʊp / (AmE), / ˈflʊərəˌskəʊp / (BrE)
an X-ray machine that uses a fluorescent screen to allow real-time observation of internal structures

Examples

fluoroscope exam room
Older departments still have a functioning fluoroscope for certain diagnostic tests.
The doctor used a fluoroscope to guide the placement of the catheter.
With the help of a fluoroscope, surgeons can observe live X-ray images during procedures.
