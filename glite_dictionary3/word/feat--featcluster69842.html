feat

noun
software feature (informal)
Pronunciation: / fit / (AmE), / fiːt / (BrE)
(informal, computing) a feature or new capability in software or hardware, especially as labeled in commit messages, changelogs, or release notes.

Examples

new feat added
The release notes list a new feat that allows users to export reports in CSV format.
Check the commit labeled feat for the UI changes on GitHub before merging.
In the changelog, feat marks functional improvements rather than simple bug fixes.
