wanderer

noun
internet program that gathers resources
Pronunciation: / ˈwɑːndərər / (AmE), / ˈwɒndərə / (BrE)
A computer program that automatically travels across the internet to locate publicly accessible resources for inclusion in a searchable database.

Examples

a wanderer scans websites
A wanderer continuously scans the internet for new sites to include in its database.
The early wanderer programs laid the foundation for today's complex search engines.
The system relies on a wanderer to update its index of web pages automatically.

Related Terms

  • synonym – spider: Computer program that browses the web to collect data
  • synonym – web crawler: An automated program that scans the web to index content for search engines
