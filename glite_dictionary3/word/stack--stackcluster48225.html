stack

noun
set of software components
Pronunciation: / stæk / (AmE), / stæk / (BrE)
a set of software technologies (programming languages, frameworks, libraries, databases, and tools) used together to develop, run, and maintain an application.

Examples

software stack choice
The startup's stack includes React, Node.js, PostgreSQL, and Redis for rapid development.
Before hiring, the company asked about my experience with their existing stack and deployment workflow.
Developers often choose a stack that balances performance, team skills, and future maintenance needs.

Related Terms

  • synonym – software stack: a layered set of software components that work together to provide application functionality
  • synonym – tech stack: the collection of technologies and tools used together to build and run software
