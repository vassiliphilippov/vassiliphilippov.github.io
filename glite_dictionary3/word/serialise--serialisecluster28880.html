to serialise

verb
convert data to transportable format
Pronunciation: / c8s61ri59ccla6az / (AmE), / c8s6159ri59la6az / (BrE)
(in computing) to convert a data structure or object into a linear, transportable format (for example JSON, XML, or a byte stream) so it can be stored or sent and later reconstructed.

Examples

to serialise data
The program can serialise the user profile into a JSON string for storage.
After saving, the service serialised the session data so it could be sent to another server.
They're serialising complex objects into a binary stream for faster network transmission today.

Related Terms

  • antonym – deserialize: to convert serialized data back into a data structure or object (the reverse process).
  • synonym – serialize: American spelling of the same computing process: convert data into a linear format.
