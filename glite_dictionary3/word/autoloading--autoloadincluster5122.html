(in computing) the automatic loading of code modules, classes, or components on first reference, so they are loaded only when needed.

Examples

autoloading code modules
Frameworks increasingly rely on autoloading to load components as needed.
Thanks to autoloading, the application starts faster because modules load only when used.
In Ruby on Rails, autoloading reduces memory usage by deferring class loading until runtime.

Related Terms

  • synonym – dynamic loading: the process of loading code or libraries into a program at runtime
  • synonym – lazy loading: a programming technique that delays initialization of resources until they are needed
  • synonym – on-demand loading: loading resources only when a program requires them rather than at startup
