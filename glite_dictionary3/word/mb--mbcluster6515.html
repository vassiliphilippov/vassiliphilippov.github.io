MB

noun
abbreviation for motherboard
Pronunciation: / ɛm bi / (AmE), / ɛm bi / (BrE)
An abbreviation for motherboard: the main printed circuit board in a computer that holds the CPU, memory, and expansion slots.

Examples

MB failure
We upgraded the PC, but the MB failed, so the technician replaced the motherboard.
The new gaming build uses an X570 MB to support multiple high-speed NVMe SSDs.
Before installing the CPU, make sure the MB BIOS is updated to the latest version.
