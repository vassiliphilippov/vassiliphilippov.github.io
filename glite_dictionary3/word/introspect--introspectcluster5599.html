to introspect

verb
examine program/object structure at runtime

(in computing) to examine the internal state, structure, or metadata of a program, object, class, or module at runtime, often to discover properties, methods, or types.

Examples

to introspect program code
During debugging, the tool can introspect objects to list available methods and fields quickly.
The runtime was introspecting class metadata to bind methods dynamically when the program started.
Automated tests often introspect modules to collect docstrings and generate API documentation during builds.

Related Terms

  • synonym – reflection: a programming capability that allows a program to examine and sometimes modify its own structure and behavior at runtime
