RAF

noun
Fujifilm RAW image file format
Pronunciation: / ɑːr eɪ ɛf / (AmE), / ɑːr eɪ ɛf / (BrE)
A Fujifilm RAW image file format (file extension .raf) used by certain digital cameras to store unprocessed sensor data for later processing.

Examples

RAF image file
She imported the RAF file into the editor to adjust the white balance.
The photographer converted several RAF images to DNG before sharing them with the retoucher.
Many raw processors now offer profiles specifically for Fujifilm RAF files to improve color rendering.

Related Terms

  • synonym – RAW (image file): an unprocessed image file containing sensor data, used as a high-quality source for editing
