call

noun
programming instruction to execute a routine
Pronunciation: / kɔl / (AmE), / kɔːl / (BrE)
(computing) an instruction in a program that causes the execution of a specific routine or procedure.

Examples

a function call
Stack overflow occurs if too many nested calls are made without returning.
The subroutine was triggered by a call in the main method.
When the program reaches a call, it temporarily stops its main flow to execute a separate function.

Related Terms

  • synonym – function call: an instance of invoking a function in a program
  • antonym – return: instruction to return control to a caller in programming
