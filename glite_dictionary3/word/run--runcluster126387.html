to run

verb
extend or vary within limits
Pronunciation: / rʌn / (AmE), / rʌn / (BrE)
to exist, continue, or be present in a persistent or consistent way (such as a trait, tradition, or idea); to extend or continue in space, time, or scope

Examples

prices run high
Gasoline prices run from $3 to $5 per gallon depending on the location.
Participants' ages run from 18 to 65 in this survey.
The questions on the test run from very easy to quite difficult.
My family tends to run on the tall side.
The meetings run long when there are many topics to cover.
These shoes usually run small, so you might want to order a size up.
A strong sense of humor runs in his family, with every generation loving a good joke.
Generosity seems to run in their community, as many residents devote time to charity.
That medical condition runs in several branches of our extended family.
