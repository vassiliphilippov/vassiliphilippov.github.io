to beautify

verb
reformat code for readability
Pronunciation: / ˈbjuːtəˌfaɪ / (AmE), / ˈbjuːtɪˌfaɪ / (BrE)
(in computing) to automatically reformat source code or structured data so it is more readable and follows a consistent style.

Examples

to beautify code
Before merging, she used a linter to beautify the JavaScript files for consistency.
The CI pipeline automatically beautified committed code to match the project's formatting rules.
He ran a tool that beautifies JSON logs so the data is easier to inspect.

Related Terms

  • synonym – format: to arrange text or data according to rules so it appears in a particular style
  • synonym – prettify: to make code or data more readable by adding appropriate spacing and line breaks
