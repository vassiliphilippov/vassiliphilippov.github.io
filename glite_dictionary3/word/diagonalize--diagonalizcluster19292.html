to diagonalize

verb
transform a matrix into a diagonal matrix
Pronunciation: / daɪˈæɡənəˌlaɪz / (AmE), / daɪˈaɡənəlaɪz / (BrE)
(of a matrix) to transform into a diagonal matrix, usually by finding a suitable basis in which the matrix has zeros off the main diagonal.

Examples

to diagonalize a matrix
If a matrix can be diagonalized, many computations become much easier.
They diagonalized the covariance matrix to simplify the calculations.
To solve the system efficiently, we need to diagonalize the matrix.

  • degree higher – diagonal matrix: a matrix where all entries outside the main diagonal are zero
