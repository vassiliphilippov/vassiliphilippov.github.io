breadcrumb

noun
website navigation trail
Pronunciation: / ˈbrɛdˌkrʌm / (AmE), / ˈbrɛd.krʌm / (BrE)
(in computing and web design) a navigational aid consisting of a horizontal list of links that shows the user's location within a website or app, often allowing easy backtracking.

Examples

breadcrumb navigation bar
The website displays a clear breadcrumb at the top so visitors can see where they are.
She followed the breadcrumb links to return to the product category after a long search.
Good design shows a clickable breadcrumb trail that helps users navigate deep website hierarchies easily.

Related Terms

  • synonym – breadcrumb trail: a sequence of links showing the user's path or position within a site
  • synonym – navigation trail: a path or sequence that helps users find their location in a site or app
