bootstrap

noun
strap for pulling on a boot
Pronunciation: / ˈbuːtˌstræp / (AmE), / ˈbuːtstræp / (BrE)
a looped strap sewn to the top of a boot, used to help pull the boot on

Examples

bootstrap loop on boot
He grabbed the bootstrap to help slide his boot onto his foot.
Some modern boots no longer feature a bootstrap at the top.
The old leather bootstrap was worn, making it hard to pull the boot on.
