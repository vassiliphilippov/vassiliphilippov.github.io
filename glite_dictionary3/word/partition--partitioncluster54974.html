partition

noun
section of a hard disk (computing)
Pronunciation: / pɑːrˈtɪʃən / (AmE), / pɑːˈtɪʃən / (BrE)
(computing) a section of a computer's hard disk that is reserved for a specific operating system or application and is accessed as a single unit.

Examples

partition on a disk
Before installing the new operating system, make sure you back up the partition where your files are stored.
If your hard disk is running out of space, you can resize the existing partition using a disk management tool.
The technician created a separate partition for the system files to keep them organized.

  • synonym – volume: a storage unit on a computer that appears as a single drive or partition
