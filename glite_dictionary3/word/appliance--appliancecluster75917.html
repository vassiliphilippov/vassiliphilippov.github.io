appliance

noun
prepackaged software image or virtual device
Pronunciation: / əˈplaɪəns / (AmE), / əˈplaɪəns / (BrE)
(computing) A prepackaged, self-contained software image or virtual device that includes an application and the minimal operating environment needed to run it on hardware, a virtual machine, or in the cloud.

Examples

virtual appliance
The company deployed a virtual appliance to handle secure web traffic in the cloud.
Instead of installing many components, they bought a preconfigured appliance that ran on a virtual machine.
Administrators update the router appliance image before deploying it across their production cloud servers.

Related Terms

  • synonym – software appliance: a packaged software product combined with a minimal operating system to run as an appliance
  • synonym – virtual appliance: a synonym often used for prepackaged software images that run on virtual machines
