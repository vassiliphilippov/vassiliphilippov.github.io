TCP

noun
internet protocol for reliable data transmission
Pronunciation: / ti si pi / (AmE), / tiː siː piː / (BrE)
a set of rules (Transmission Control Protocol) used to ensure reliable transmission of data over the Internet and other computer networks

Examples

TCP connection setup
Engineers often compare TCP with UDP when discussing network protocols.
If the connection fails, TCP will attempt to retransmit the lost data.
The server communicates with clients using TCP to ensure all data packets are received correctly.

Related Terms

  • antonym – UDP: an alternative network protocol offering faster but less reliable data transfer
