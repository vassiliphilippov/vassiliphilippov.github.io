unknown

noun
TypeScript type requiring checks
Pronunciation: / ʌnˈnoʊn / (AmE), / ʌnˈnəʊn / (BrE)
(in programming, especially TypeScript) a data type that can hold any value but prevents use until the programmer performs explicit type checks or conversions.

Examples

unknown type safety
In TypeScript the compiler treats a value typed as unknown more safely than 'any'.
Before using a variable declared as unknown, you must perform a type check or assertion.
The library returns JSON as unknown, so callers verify fields before accessing them.

Related Terms
