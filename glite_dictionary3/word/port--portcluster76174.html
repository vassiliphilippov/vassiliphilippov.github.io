to port

verb
adapt software for a new platform
Pronunciation: / pɔːrt / (AmE), / pɔːt / (BrE)
to adapt or modify (software) so that it can be used on a different computer, operating system, or platform.

Examples

to port software
After several weeks of work, the team successfully ported the software to the new hardware architecture.
The developer managed to port the game from Windows to macOS in just two months.
We need to port the application so it runs smoothly on mobile devices.
