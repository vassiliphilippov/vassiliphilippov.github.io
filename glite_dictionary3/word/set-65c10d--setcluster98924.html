set

adjective
fixed or determined; established
Pronunciation: / sɛt / (AmE), / sɛt / (BrE)
(of a date, time, rule, etc.) decided, established, or determined, especially by authority.

Examples

set deadline
All applications must be submitted before the set deadline.
The set procedures must be followed in case of an emergency.
The meeting is at a set time every Monday.

Related Terms

  • synonym – fixed: firmly set, not changing
