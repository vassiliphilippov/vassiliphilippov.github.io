to throw

verb
raise an exception (programming)
Pronunciation: / θroʊ / (AmE), / θrəʊ / (BrE)
(in programming) to raise or signal an exception or error during program execution so that it can be handled or reported.

Examples

to throw an exception
When the input is invalid, the function will throw an exception and stop execution.
During testing, her module threw a null-pointer error every time the user left the field blank.
The new library keeps throwing specific exceptions so developers can identify errors quickly.

Related Terms

  • antonym – catch: to receive and handle an exception raised by code
  • synonym – raise: to cause something to occur or exist (often used in programming to mean 'produce an exception')
  • synonym – signal: to send a notification or indicator (for example, to indicate an error condition)
