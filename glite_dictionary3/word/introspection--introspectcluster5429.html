introspection

noun
API or schema discovery

(in computing) the ability or process by which a system, API, or toolkit exposes or discovers its own structure, types, fields, or capabilities, often by responding to metadata queries.

Examples

API introspection tools
GraphQL's introspection allows clients to discover available types and fields.
During testing, the developer enabled introspection so tools could generate API documentation automatically.
Some production servers disable introspection to reduce information exposure to untrusted clients.
