matrix

noun
rectangular array in mathematics
Pronunciation: / ˈmeɪ.trɪks / (AmE), / ˈmeɪ.trɪks / (BrE)
(Mathematics) A rectangular arrangement of numbers, symbols, or expressions in rows and columns that is treated as a single entity and manipulated according to specific rules.

Examples

matrix of numbers
Each matrix in the system represents a set of equations for the network.
The matrix was filled with random numbers for the simulation.
When multiplying two matrices, the number of columns in the first must match the rows in the second.
