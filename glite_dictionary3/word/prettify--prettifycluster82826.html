to prettify

verb
format code or data for readability
Pronunciation: / \u02c8pr\u025bti\u02ccfa\u026a / (AmE), / \u02c8pr\u025bt\u026afa\u026a / (BrE)
(in computing) to reformat code, JSON, or other serialized data to improve readability by adding consistent indentation, line breaks, and spacing.

Examples

to prettify code
Developers often prettify JSON outputs to make them easier to read during debugging sessions.
She prettified the exported CSV, adding line breaks and indentation so the data looked tidy.
Their build script automatically prettifies source files before committing to ensure consistent style.
