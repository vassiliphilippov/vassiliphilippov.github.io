nil

noun
programming null value
Pronunciation: / nɪl / (AmE), / nɪl / (BrE)
(in programming) a literal or value used in some programming languages to represent the absence of a value, a null or empty value.

Examples

nil value
In Ruby, the method returned nil when no matching record was found.
Assigning nil to the variable cleared its value before the next computation step.
The configuration file used nil for empty fields, indicating no value should be stored.

Related Terms

  • synonym – NULL: the SQL (and C-family) representation of a missing or undefined value in databases and some languages.
  • synonym – None: the Python literal representing the absence of a value (language-specific equivalent of nil).
  • synonym – null: a programming term for a special value meaning 'no value' or 'undefined'.
