prompt

noun
computer command symbol/message
Pronunciation: / prɑmpt / (AmE), / prɒmpt / (BrE)
(computing) a symbol or message displayed on a computer screen indicating that the system is ready to accept input.

Examples

computer prompt symbol
After logging in, the shell presents a prompt indicating readiness for input.
The system displays a blinking prompt on the terminal window.
When the user sees the prompt, they can type their next command.
