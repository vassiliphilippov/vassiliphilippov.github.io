emu

noun
software emulator (informal)
Pronunciation: / c8id0mud0 / (AmE), / c8id0mjud0 / (BrE)
(informal, computing) a software program that imitates another computer system or video game console so software or games designed for the original system can run on a different platform.

Examples

game emu software
I used an emu to play classic SNES games on my laptop.
The retro community recommended a lightweight emu that runs smoothly on low-end hardware.
Before testing, she configured the emu with BIOS files and controller mappings for accuracy.

Related Terms

  • synonym – emulator: software that imitates the functions of another computer system or device
  • synonym – simulator: a program that simulates the behavior of a system; often used similarly to an emulator in some contexts
