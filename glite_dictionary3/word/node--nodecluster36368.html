node

noun
point of zero displacement in a wave
Pronunciation: / noʊd / (AmE), / nəʊd / (BrE)
(Physics) A point of minimum or zero displacement in a standing wave, where destructive interference causes no movement.

Examples

wave node
A node is formed at each end of a fixed string due to complete destructive interference.
In a standing wave, each node remains stationary while the antinodes oscillate.
The lowest point on the vibrating string is known as a node.
