meltdown

noun
CPU hardware security flaw
Pronunciation: / \u02c8m\u025bltda\u028an / (AmE), / \u02c8m\u025bltda\u028an / (BrE)
(in computing, especially cybersecurity) a hardware security vulnerability in many modern processors that can allow ordinary (unprivileged) programs to read protected kernel memory, breaking process isolation.

Examples

the meltdown bug
Security researchers warned that the meltdown vulnerability allowed unprivileged programs to read protected memory.
After the report, companies patched systems to protect against meltdown and similar processor flaws.
Many users installed updates quickly when news of meltdown affected cloud services and laptops.

Related Terms

  • synonym – hardware bug: a flaw in a computer's physical components or microarchitecture that causes incorrect or insecure behaviour
  • synonym – security vulnerability: a weakness in a system that can be exploited to cause harm or gain unauthorized access
