routine

noun
set of instructions in a computer program
Pronunciation: / ruˈtin / (AmE), / ruːˈtiːn / (BrE)
a sequence of programmed instructions that performs a specific task within a larger computer program

Examples

computer routine
Each routine in the program is responsible for a specific operation.
Experienced programmers often reuse the same routine to perform common tasks like sorting data.
The new software update included a routine for encrypting user data.

Related Terms

  • synonym – function: a named section of a program that performs a specific task
  • synonym – procedure: a set of coded instructions that perform a specific task, often used in programming
  • synonym – subroutine: a routine that is called by another routine within a program
