annotation

noun
metadata tag in source code
Pronunciation: / ˌænəˈteɪʃən / (AmE), / ˌænəˈteɪʃən / (BrE)
(in computing) a metadata tag attached to source-code elements (such as classes, methods, or fields) that provides information to tools, frameworks, or compilers and may guide processing or configuration.

Examples

annotation in code
The framework reads the annotation on the class to configure dependency injection.
You should add an annotation above the method to indicate it handles HTTP POST requests.
During compilation, the tool used the annotation to generate extra validation code automatically.

Related Terms

  • synonym – attribute: a language-specific term (for example, C#) for a construct that attaches metadata to program elements, similar to an annotation.
  • synonym – metadata: data that provides information about other data; descriptive information attached to data objects.
