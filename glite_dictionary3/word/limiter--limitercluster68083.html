limiter

noun
rate-limiting mechanism

a software or hardware mechanism that restricts the rate at which events (for example, API calls or network packets) may occur, used to prevent overload, ensure fair use, or block abuse.

Examples

API limiter tool
Many services implement a limiter to cap API calls per minute and prevent abuse.
The network engineer added limiters to control packet bursts and stabilize traffic during peak hours.
You can configure a rate limiter in the gateway to reject excessive requests automatically.

Related Terms
