mapper

noun
software component that maps data
Pronunciation: / ˈmæpər / (AmE), / ˈmæpə / (BrE)
(in computing) a software component or module that converts or maps data between different models or representations, for example turning database rows into programming objects or transforming one data format into another.

Examples

data mapper software
The ORM uses a mapper to convert database rows into domain objects.
Our team wrote a custom mapper to transform API JSON into internal product models.
When migrating, they tested each mapper to ensure no data was lost during conversion.

Related Terms

  • synonym – adapter: a component that converts data or interfaces between different formats
  • synonym – object-relational mapper: a library that maps database tables to programming objects
