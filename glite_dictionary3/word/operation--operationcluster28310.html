operation

noun
computing action defined by a rule
Pronunciation: / ˌɑː.pəˈreɪ.ʃən / (AmE), / ˌɒp.əˈreɪ.ʃən / (BrE)
An action or process in computing, defined by a rule, that produces a specific result (especially as a single instruction within a computer program).

Examples

computer operation
Sorting the data requires several sequential operations to be completed.
The processor executes each operation in less than a nanosecond.
This command initiates an operation that stores the user's input in memory.

Related Terms

  • synonym – instruction: A command given to a computer processor to perform a specific operation
  • synonym – process: A series of actions or steps taken to achieve a particular end in computing
