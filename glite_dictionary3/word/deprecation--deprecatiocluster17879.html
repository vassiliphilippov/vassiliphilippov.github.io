deprecation

noun
marking a software feature obsolete
Pronunciation: / ˌdɛprəˈkeɪʃən / (AmE), / ˌdɛprəˈkeɪʃən / (BrE)
In computing, the act or state of marking a software feature, API, or function as obsolete and discouraged for use, typically with a warning and planned removal in a future release.

Examples

deprecation warning
The API's deprecation warning alerts developers that the function will be removed in a future release.
After the library's deprecation, maintainers provided migration guides and compatibility shims for users.
Browsers show console warnings when a deprecation affects a web API used by many sites.

Related Terms

  • synonym – deprecate: to indicate that a software feature or API should no longer be used and may be removed in the future
  • synonym – phase out: to discontinue something gradually, often replacing it with a newer alternative
