to configure

verb
set up for a specific purpose
Pronunciation: / kənˈfɪɡjɚ / (AmE), / kənˈfɪɡjə / (BrE)
to arrange or set up something, especially a device or system, for a specific purpose or to meet particular requirements

Examples

to configure the settings
Before using the software, you must configure the settings to match your preferences.
Engineers will configure the system so it can handle increased demand.
It took me an hour to configure the network for all the new devices.
