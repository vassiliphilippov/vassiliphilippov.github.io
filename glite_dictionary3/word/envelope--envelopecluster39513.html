envelope

noun
cryptographic wrapper for keys or data
Pronunciation: / ˈɛn.və.loʊp / (AmE), / ˈɛn.və.ləʊp / (BrE)
(in cryptography and cloud security) a container that securely wraps encryption keys or encrypted payloads, typically used in 'envelope encryption' to protect data.

Examples

encryption envelope key
The cloud provider uses envelope encryption to protect customer data at rest across multiple regions.
During the audit they found an envelope containing wrapped keys stored alongside the encrypted database backup.
Developers returned the decrypted payload after verifying the envelope signature and wrapping policy in code.

Related Terms

  • synonym – encryption: the process of converting data into a code to prevent unauthorized access
  • synonym – key wrapping: a technique for securely encrypting cryptographic keys so they can be stored or transported
