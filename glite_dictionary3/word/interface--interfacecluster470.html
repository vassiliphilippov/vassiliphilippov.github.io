interface

noun
computing: means of user-system interaction
Pronunciation: / ˈɪn.t̬ɚˌfeɪs / (AmE), / ˈɪn.təˌfeɪs / (BrE)
(computing) a program or system that enables a user to interact with a computer, device, or software, often through a graphical or command-line display.

Examples

user interface design
Developers aim to design an interface that is both attractive and easy to use.
The software update introduced a new interface that made navigation more intuitive.
You can customize the interface to better suit your workflow preferences.

Related Terms

  • degree lower – command line interface: text-based user interface for entering commands
  • degree higher – graphical user interface: a user interface that uses graphics and icons
  • synonym – user interface: the space where user and machine interact; another term for this concept
