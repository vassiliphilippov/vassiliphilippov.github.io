to programme

verb
write code for computer software (British)
Pronunciation: / ˈproʊɡræm / (AmE), / ˈprəʊɡræm / (BrE)
(British) To write code for a computer program or to develop computer software.

Examples

to programme a computer
He is currently programming an app for the school's website.
She learned to programme in several languages at university.
They programmed the new software to handle financial calculations.
