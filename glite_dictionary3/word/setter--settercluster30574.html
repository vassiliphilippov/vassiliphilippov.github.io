setter

noun
method that assigns an object's property
Pronunciation: / ˈsɛtər / (AmE), / ˈsetə / (BrE)
(in computing, especially object-oriented programming) a method that assigns a value to an object's property (attribute), typically paired with a getter (accessor).

Examples

object setter method
The framework automatically called the setter when the form field was updated by the user.
During code review she suggested renaming the setter to make its purpose clearer and safer.
Unit tests verified that the setter rejected invalid input and preserved object invariants.

Related Terms

  • antonym – getter: a method that returns the value of an object's property; commonly paired with a setter
  • synonym – mutator: a method that changes or updates an object's state or property; often synonymous with setter
