commit

noun
version-control snapshot
Pronunciation: / kəˈmɪt / (AmE), / kəˈmɪt / (BrE)
In version-control systems, a recorded snapshot of changes saved to a repository, typically including the author, timestamp, and a message describing the changes.

Examples

make a commit message
She wrote a clear message for the commit before pushing her changes to GitHub.
The CI server ran tests automatically whenever a new commit arrived on the main branch.
They reverted the bad commit to restore the repository to its previous stable state.

Related Terms

  • synonym – changeset: a collection of changes recorded together in a version-control system
  • synonym – revision: a particular version of files recorded in version control
  • synonym – snapshot: a recorded state of a repository or file system at a particular moment
