to thrash

verb
(computing) excessively swap data, not compute
Pronunciation: / θræʃ / (AmE), / θræʃ / (BrE)
(of a computer system) to spend excessive time moving data between memory and storage rather than performing useful computational work, often due to insufficient memory.

Examples

computer thrashes memory
If too many programs are running, the computer will thrash and slow down considerably.
The server started thrashing after the database query overloaded the memory.
The system administrator noticed that the machine was thrashing due to a lack of available RAM.

Related Terms

  • synonym – swap: (computing) to exchange data between memory and storage
