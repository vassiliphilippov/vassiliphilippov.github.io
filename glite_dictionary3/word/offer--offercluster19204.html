to offer

verb
attempt to enqueue (conditional)
Pronunciation: / ˈɔfər / (AmE), / ˈɒfə / (BrE)
(in computing, especially data structures and APIs) to attempt to insert an element into a queue or bounded buffer, succeeding only if there is available capacity; typically returns a success/failure result.

Examples

to offer an item
The queue's API offers a method to enqueue items when capacity allows.
When the buffer was full, the system offered the packet but returned a failure flag.
In our test, queue.offer(item) returned false because the bounded buffer was already full.

Related Terms

  • synonym – enqueue: to add an element to the end of a queue
  • synonym – insert: to put an item into a data structure or collection
