wiper

noun
secure data‑erasing tool
Pronunciation: / ˈwaɪpər / (AmE), / ˈwaɪpə / (BrE)
(in computing) a program or device that securely deletes, overwrites, or destroys data on storage media so the data cannot be recovered.

Examples

data wiper program
The IT team used a wiper to sanitize the hard drives before disposal.
A notorious malware called NotPetya acted as a wiper, destroying files across corporate networks overnight.
Before donating the laptop, she ran a certified wiper to ensure personal documents could not be recovered.

Related Terms

  • synonym – data sanitization: processes that irreversibly remove or overwrite data to prevent recovery
  • synonym – secure erase: a method or command to overwrite storage so deleted data cannot be retrieved
