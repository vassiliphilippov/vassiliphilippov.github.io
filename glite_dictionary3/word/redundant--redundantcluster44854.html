redundant

adjective
having backup components for reliability
Pronunciation: / rɪˈdʌndənt / (AmE), / rɪˈdʌndənt / (BrE)
(of a system or component) duplicated or having backup elements so that function continues if one part fails; providing fault tolerance.

Examples

redundant backup system
The data center uses a redundant power supply to ensure uninterrupted service during failures.
Flight systems are designed with redundant controls so pilots can rely on backups if one fails.
We added a redundant network connection to maintain access while the main link undergoes maintenance.

Related Terms

  • synonym – backup: a duplicate system or component kept to replace the primary one if it fails
  • synonym – fault-tolerant: designed to continue operating correctly even if some components fail
