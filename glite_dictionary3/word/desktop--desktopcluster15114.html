desktop

noun
background area of a computer screen
Pronunciation: / ˈdɛskˌtɑp /; / ˈdɛsktɑp / (AmE), / ˈdɛsktɒp / (BrE)
the main screen area on a computer where icons, windows, and menus appear; also, a virtual version of this interface accessed remotely

Examples

computer desktop
