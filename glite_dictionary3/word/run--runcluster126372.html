to run

verb
sail with the wind behind (nautical)
Pronunciation: / rʌn / (AmE), / rʌn / (BrE)
to move quickly on foot, especially by moving both feet off the ground alternately; to go somewhere quickly (including racing, escaping, or going for a run)

Examples

to run fast
As the storm approached, the captain decided to run before the wind to reach a safe harbor.

The yacht began to run before the wind, gaining speed across the open sea.
She saw the wild dog approach and began to run for safety.
The protesters had to run when the police arrived unexpectedly.
When the alarm sounded, the thieves decided to run before anyone could catch them.
He decided to run a marathon for charity.
She trained every day and could run five miles without stopping.
The children ran three laps around the playground during gym class.
