torrent

noun

A computer file (or small descriptor file) used to distribute and download content via the BitTorrent peer-to-peer protocol; often refers to the set of files available from a torrent swarm.

Examples

torrent file download
He downloaded a torrent of indie films via BitTorrent last night.
The university blocked the torrent site to stop students sharing copyrighted material.
She opened the small torrent file to begin downloading the game from peers.

Related Terms

  • synonym – BitTorrent file: a file format used by the BitTorrent protocol to describe and share files across peers
  • synonym – peer-to-peer file: a file distributed directly between users over a peer-to-peer network
