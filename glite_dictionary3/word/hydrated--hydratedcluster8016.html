hydrated

adjective
made interactive by client JavaScript
Pronunciation: / ˈhaɪdreɪtɪd / (AmE), / ˈhaɪdreɪtɪd / (BrE)
(in web development) (of server-rendered HTML or a component) enhanced by client-side JavaScript so it is interactive.

Examples

hydrated web page
After the JavaScript ran, the previously static page was fully hydrated and responded to clicks.
We deploy server-rendered routes so their HTML arrives already hydrated for faster perceived performance.
Developers check whether the component is hydrated before attaching event listeners in production.

Related Terms

  • synonym – hydrate: to attach client-side JavaScript to server-rendered HTML so it becomes interactive
