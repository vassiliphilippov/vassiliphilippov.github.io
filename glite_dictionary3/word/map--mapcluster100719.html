map

noun
mathematical function relating sets
Pronunciation: / mæp / (AmE), / mæp / (BrE)
(mathematics) a function or rule that associates each element in one set with exactly one element in another set.

Examples

mathematical map
A bijective map ensures every element in the first set pairs with a unique element in the second set.
In this proof, we define a map from the set of real numbers to the set of positive numbers.
The function acts as a map linking integers to their corresponding squares.

Related Terms

  • synonym – function: a relation from a domain to a range in which each element of the domain is assigned to exactly one element of the range
