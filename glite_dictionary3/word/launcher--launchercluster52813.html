launcher

noun
software interface that launches apps
Pronunciation: / ˈlɔntʃər / (AmE), / ˈlɔːntʃə / (BrE)
a software interface on a computer or mobile device that provides icons, shortcuts, or gestures to start and organise applications and sometimes group or suggest them.

Examples

app launcher
He customized his Android launcher to group apps by color and display only essential shortcuts.
The new desktop launcher shows suggested apps and recent documents in a single column.
When the launcher crashed, she could still open apps from the system tray as a temporary workaround.

Related Terms

  • synonym – app launcher: software that starts and organises applications; commonly used synonym for a mobile or desktop launcher.
  • synonym – start menu: a system interface element that gives access to installed programs and settings; often serves the same purpose as a launcher.
