descendant

noun
node below another in a tree (computing)
Pronunciation: / dɪˈsɛndənt / (AmE), / dɪˈsɛndənt / (BrE)
(computing) a node or element in a tree or directed graph that is reachable from another node by following one or more child (downward) links; a node that is below another in the structure.

Examples

descendant node structure
In the DOM, a descendant node is any element reachable by following child links from a parent.
When querying the tree, the algorithm returned all descendants of the selected vertex in breadth-first order.
To update styles, add a class selector that targets any descendant elements inside the container.
