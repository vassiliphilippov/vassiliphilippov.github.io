tombstone

noun
deletion marker in databases
Pronunciation: / c8tud0mccsto8an / (AmE), / c8tud0mst598an / (BrE)
(in computing) a marker placed on a record or object to indicate it has been deleted, retained temporarily so that replicas and consistency protocols can process the deletion until the data is permanently removed.

Examples

tombstone in a database
During replication, the system stored a tombstone so other nodes would know the record was deleted.
After compaction, the database permanently removed the old data and cleared the tombstones.
Investigators found that many rows still had a tombstone preventing their immediate restoration.

Related Terms

  • synonym – deletion marker: a marker indicating that a record has been deleted and should be ignored until removal
  • synonym – soft delete: a deletion approach where records are marked as deleted but not immediately removed from storage
