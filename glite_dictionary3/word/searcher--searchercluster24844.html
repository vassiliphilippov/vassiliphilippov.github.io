searcher

noun
software agent that searches data
Pronunciation: / c8s5dd0t8359r / (AmE), / c8s5cd0t8359 / (BrE)
(computing) a software program or automated agent that searches databases, logs, or the internet to locate and retrieve information.

Examples

computer searcher
The security team deployed a searcher to scan system logs for unusual login attempts.
When the database grew, they added parallel searchers to speed up query responses during peak hours.
The research lab wrote a custom searcher to index historical documents and return ranked matches.
