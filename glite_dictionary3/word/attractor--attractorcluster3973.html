attractor

noun
physics: state a system evolves toward
Pronunciation: / əˈtræktər / (AmE), / əˈtræktə / (BrE)
(physics) A set of states in a dynamical system towards which the system tends to evolve, regardless of the starting conditions.

Examples

strange attractor in physics
In chaos theory, a strange attractor can create complex, unpredictable behavior within deterministic systems.
Scientists simulated the weather patterns to identify the underlying attractor in the model.
The system eventually settles into a stable attractor regardless of its initial state.
