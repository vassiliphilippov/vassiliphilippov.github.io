ibid.

adverb
same source as previous (citation)
Pronunciation: / ˈɪbɪd / (AmE), / ˈɪbɪd / (BrE)
used to indicate that a reference is to the same source as the previous one

Examples

ibid. in a citation
All subsequent references use ibid. instead of repeating the full citation.
In her bibliography, she frequently writes ibid. to save space.
The second note simply reads ibid., referring to the same book as the first.
