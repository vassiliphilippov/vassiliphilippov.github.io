resolving

noun
DNS name-to-IP translation
Pronunciation: / rɪˈzɑlvɪŋ / (AmE), / rɪˈzɒlvɪŋ / (BrE)
(in computer networking) the process of translating a domain name or hostname into its numerical IP address, typically performed by the Domain Name System (DNS) or a local resolver.

Examples

resolving a website address
When the site failed to load, the browser's resolving of the domain returned no IP address.
Slow resolving by the DNS server caused long delays before web pages appeared in my browser.
Network engineers logged repeated resolving failures while troubleshooting the customer's intermittent connectivity issue.

Related Terms

  • synonym – DNS resolution: the process by which DNS translates domain names into IP addresses
  • synonym – name resolution: the general process of resolving hostnames to IP addresses in a network
