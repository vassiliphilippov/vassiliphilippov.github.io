printable

adjective
suitable for 3D printing
Pronunciation: / ˈprɪn.tə.bəl / (AmE), / ˈprɪn.tə.bəl / (BrE)
(of a digital model or design) able to be produced using a 3D printer; suitable for 3D printing without major modification.

Examples

printable 3D model
The designer uploaded a printable STL file so hobbyists could make their own figure at home.
Before selling, she checked that the model was printable on common consumer printers without supports.
The repository shows which designs are printable and provides recommended settings for each printer.

Related Terms

  • synonym – 3D-printable: able to be produced using 3D printing (synonymous term often used with files and models).
