gherkin

noun
BDD domain-specific language
Pronunciation: / \u02c8\u0261\u025d\u02d0.k\u026an / (AmE), / \u02c8\u0261\u025c\u02d0.k\u026an / (BrE)
(in software development, especially behavior-driven development) a simple, human-readable domain-specific language for writing executable specifications and test scenarios, typically using Given/When/Then steps.

Examples

gherkin test language
Developers write features in gherkin to drive automated acceptance tests across the team.
She translated the client's requirements into clear gherkin so the testers could implement scenarios.
During the workshop, participants practiced writing gherkin scenarios for common login failures.

Related Terms

  • synonym – Cucumber (tool): an automation tool for behavior-driven development that uses the Gherkin language to define features
  • synonym – domain-specific language: a programming or specification language tailored to a particular application domain
