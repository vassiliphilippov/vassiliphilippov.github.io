visit

noun
web analytics: single site access/session
Pronunciation: / ˈvɪzɪt / (AmE), / ˈvɪzɪt / (BrE)
(in web analytics) a single instance of a user accessing a website or web page, counted by analytics systems (often equivalent to a session).

Examples

website visit count
The analytics dashboard recorded a sudden visit to the site from a new region.
The marketing team noticed that each email generated more visits than clicks on the landing page.
During the promotion, daily visits doubled, pushing server load beyond normal capacity.

Related Terms

  • synonym – pageview: a single load or view of a web page; a pageview counts individual pages while a visit counts the user's overall access.
  • synonym – session: a period of user activity on a website, often used interchangeably with a visit in analytics.
