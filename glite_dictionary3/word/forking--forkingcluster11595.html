forking

noun
creating a separate software project from a repository
Pronunciation: / c8f54rk6a4b / (AmE), / c8f54d0k6a4b / (BrE)
(in software development) the act of creating a separate project by copying an existing repository so it can be developed independently.

Examples

forking a repository
The forking of the repository allowed developers to experiment without affecting the original project's codebase.
Company policy forbids forking core tools because it creates multiple incompatible versions across teams.
After the disagreement, she suggested forking the project and maintaining an independent release cycle.

Related Terms

  • synonym – branch: a separate line of development within a repository, usually kept under the same project
  • synonym – fork: a copy of a software project's repository created so it can be modified independently
