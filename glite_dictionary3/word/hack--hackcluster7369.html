to hack

verb
improvise fixes for computer programs
Pronunciation: / hæk / (AmE), / hæk / (BrE)
to fix or modify a computer program in a quick or makeshift way, often to get it working temporarily rather than elegantly or efficiently

Examples

to hack computer code
If you just hack the code like this, it might work but won't be easy to maintain.
She had to hack the script to get it running before the deadline.
We've been hacking together fixes all night to try and keep the server online.

  • synonym – patch: to make a quick fix to a program
