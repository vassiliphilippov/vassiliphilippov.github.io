to unwrap

verb
extract value from a wrapped data type (programming)
Pronunciation: / \u028cn\u02c8r\u00e6p / (AmE), / \u028cn\u02c8rap / (BrE)
(in computing) to obtain or extract the value contained inside a wrapped data type (for example Option/Maybe/Result), usually removing the container and producing the inner value or an error if no value is present.

Examples

to unwrap a value
In Rust, you must unwrap the Option to access its value.
Rather than calling unwrap, handle the None case to avoid runtime panics in production code.
The unit tests failed because the code attempted to unwrap a Result that contained an error.

Related Terms

  • synonym – extract: to remove or obtain something from a larger whole; obtain the inner value from a container
  • synonym – retrieve: to get back or obtain (data or a value) from storage or a container
