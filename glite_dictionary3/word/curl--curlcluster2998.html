curl

noun
command-line URL/data transfer tool
Pronunciation: / k5dl / (AmE), / k5cd0l / (BrE)
(computing) a command-line program and tool for transferring data to or from servers using URLs and a variety of protocols (for example HTTP, FTP).

Examples

curl command
I used curl to fetch the JSON response from the API and inspect the headers.
You can run curl from the terminal to download a file or check a URL quickly.
Her deployment script calls curl to send POST requests with the required JSON payload.
