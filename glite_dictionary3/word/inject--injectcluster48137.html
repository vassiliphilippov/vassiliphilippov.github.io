to inject

verb
insert malicious code into a system
Pronunciation: / ɪnˈdʒɛkt / (AmE), / ɪnˈdʒɛkt / (BrE)
(computing, security) to insert or embed malicious data, code, or commands into a program, query, or system input in order to exploit a vulnerability or change program behaviour.

Examples

to inject malicious code
Hackers often inject SQL code into web forms to bypass authentication and steal data.
During the breach, attackers injected malicious scripts that redirected users to fake login pages.
The security team detected attempts at injecting commands through the upload field during the penetration test.

Related Terms

  • synonym – exploit: to take advantage of a vulnerability in software or a system, often to carry out an attack
  • synonym – insert: to put data or code into something (e.g., a program or document)
