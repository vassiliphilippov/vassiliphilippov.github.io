hook

noun
HTTP callback; webhook
Pronunciation: / hʊk / (AmE), / hʊk / (BrE)
(in computing) a registered callback endpoint or mechanism that an application calls when a specified event occurs; in web contexts this often means an HTTP request (a webhook) sent to a user-specified URL.

Examples

web hook event
The payment system posts an event and the partner's hook receives a JSON payload for processing.
Developers added a hook to the signup flow so external services get notified immediately.
You must secure the hook URL with a secret token to prevent unauthorized requests.

Related Terms

  • synonym – callback: a routine or endpoint provided so a system can call it when a particular event happens
  • synonym – event handler: code that runs in response to an event in a system
  • synonym – webhook: an HTTP callback: a web-based hook that posts data to a configured URL when events occur
