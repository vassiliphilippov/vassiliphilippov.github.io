reference

noun
identifier for stored information (computing)
Pronunciation: / ˈrɛf.ɚ.əns / (AmE), / ˈrɛf.ər.əns / (BrE)
(Computer science) A code or data item that identifies where a particular piece of information is stored in memory or a database.

Examples

reference code
A database reference is required to retrieve user records efficiently.
In many programming languages, a reference allows you to access data stored elsewhere in memory.
The function returns a reference to the object instead of a copy.
