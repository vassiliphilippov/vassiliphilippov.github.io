snapshot

noun
point-in-time digital copy
Pronunciation: / ˈsnæpˌʃɑt / (AmE), / ˈsnæpʃɒt / (BrE)
A point-in-time copy of a digital system's state (for example, a virtual machine, filesystem, or database) used to preserve data for backup, rollback, or replication.

Examples

create a snapshot backup
The database administrator created a snapshot of the production database before applying the patch.
Before updating the server, they stored several snapshots so they could roll back quickly if needed.
Using filesystem snapshots reduced downtime during the migration by allowing fast recoveries.

Related Terms

  • synonym – backup: a copy of data kept to restore the original after loss or corruption
  • synonym – checkpoint: a saved state used to resume or restore a process or system
