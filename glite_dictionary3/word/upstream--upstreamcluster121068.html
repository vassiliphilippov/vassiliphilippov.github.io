upstream

adjective
from the main repository
Pronunciation: / 8cpcstrid0m / (AmE), / 8cpcstrid0m / (BrE)
(computing) relating to the main source repository or to changes that originate there and may be propagated to forks or downstream projects.

Examples

upstream software changes
The team monitored upstream changes to decide whether to merge them into their fork.
We fixed several bugs when an upstream commit introduced a regression in the library.
Their package automatically pulls upstream updates every morning to stay current.

Related Terms

  • antonym – downstream: relating to projects that receive or apply changes from an upstream project; opposite in relation.
  • synonym – merge: to combine changes from one branch or repository into another; often used with upstream changes.
