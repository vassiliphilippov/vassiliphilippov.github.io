comet

noun
server-push web technique (long-lived connection)
Pronunciation: / c8k51d0m59t / (AmE), / c8k52m6at / (BrE)
(computing, web development) a technique for delivering server-to-client updates by keeping an HTTP connection open (or repeatedly long-polling) so the server can push data to the client in real time.

Examples

comet technology
Many early chat apps used comet techniques to deliver messages in real time.
The developer implemented comet long-polling because WebSocket support was unavailable on the server.
Although outdated, comet remains useful for simple real-time updates where simplicity matters.

Related Terms

  • synonym – long polling: a technique in which the client makes a request that the server holds open until it has new data to send
  • synonym – server push: a method where a server sends updates to clients without an explicit request each time
