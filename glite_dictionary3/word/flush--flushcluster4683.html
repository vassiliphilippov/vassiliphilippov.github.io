to flush

verb
force buffered data to be written or cleared
Pronunciation: / flʌʃ / (AmE), / flʌʃ / (BrE)
(computing) to force the immediate transfer, writing, or removal of data held in a temporary buffer or cache so that it is saved to or cleared from permanent storage.

Examples

to flush the cache
The database server will flush its cache to disk immediately after the transaction completes.
To avoid data loss, the system periodically flushes write buffers so changes are actually saved to storage.
After editing hosts, remember to flush the DNS cache so your computer uses the updated records.

Related Terms

  • synonym – clear: to remove stored or temporary data (for example, clearing a cache)
  • synonym – synchronize: to write buffered or cached data to permanent storage so it matches the source state
