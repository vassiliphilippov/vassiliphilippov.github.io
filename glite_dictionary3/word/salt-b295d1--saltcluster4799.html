salt

noun
random value added before hashing
Pronunciation: / sɔlt / (AmE), / sɒlt / (BrE)
(in computing, cryptography) a random value added to data (often a password) before hashing to make precomputed attacks and rainbow-table lookups ineffective.

Examples

password salt value
Systems store password hashes with a unique salt per user to improve security.
The developer accidentally reused the same salt, making multiple accounts vulnerable to attack.
Modern frameworks generate random salts automatically when creating password hashes for new users.
