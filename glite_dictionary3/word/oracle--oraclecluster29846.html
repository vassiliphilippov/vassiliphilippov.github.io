oracle

noun
blockchain data provider
Pronunciation: / ˈɔrəkəl / (AmE), / ˈɒrəkəl / (BrE)
(in computing, especially blockchain) a service or system that provides external real-world data (for example, prices, weather, events) to smart contracts or decentralized applications so they can act on information from outside the blockchain.

Examples

blockchain oracle service
Smart contracts rely on an oracle to fetch price data before executing trades.
The developer connected the decentralized exchange to several oracles for more reliable rates.
Researchers warned that a compromised oracle could feed false information and break contracts.

Related Terms

  • synonym – data feed: an automated supply of data (for example, price or weather updates) to a software system
  • synonym – data provider: a service or organization that supplies data to applications or systems
