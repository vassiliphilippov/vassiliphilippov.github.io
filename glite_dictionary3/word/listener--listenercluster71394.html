(in computing) a program or component that waits for and responds to events or incoming connections, such as network requests or user actions.

Examples

network listener
The server started a TCP listener on port 8080 to accept incoming connections from clients.
Our event-driven app registers a file-change listener that triggers a sync whenever the directory updates.
During testing the team noticed two listeners were competing for the same socket and caused errors.
