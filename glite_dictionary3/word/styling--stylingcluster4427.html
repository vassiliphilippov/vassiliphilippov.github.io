styling

noun
visual design for interfaces (CSS)
Pronunciation: / ˈstaɪ.lɪŋ / (AmE), / ˈstaɪ.lɪŋ / (BrE)
(in web and software design) the process of applying visual appearance rules (for example, CSS, themes, or component styles) to user interfaces to control layout, colours, fonts, and other visual properties.

Examples

website styling
The front-end developer spent the afternoon improving the site's styling to match the new brand guidelines.
Beginner tutorials cover layout, colours, and responsive styling so pages look good on all devices.
They debated whether the component's default styling should be inlined or provided through a theme file.

Related Terms

  • synonym – CSS: a stylesheet language commonly used to implement styling for web pages and components.
  • synonym – theming: creating or applying a consistent set of visual rules and assets (a theme) across an interface.
