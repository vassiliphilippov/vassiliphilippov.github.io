router

noun
software component that maps paths to views
Pronunciation: / c8ra8at59r / (AmE), / c8rud0t59 / (BrE)
(in software development) a component that directs navigation by mapping URLs or client-side paths to views, screens, or request handlers, often allowing navigation without a full page reload.

Examples

app router
The app's router defines which view appears when users change the URL in their browser.
During development, she updated the router to load components lazily and speed up navigation.
The router in our React app maps pathnames to screen components in single-page applications.

Related Terms

  • synonym – route: to direct along a particular path; in computing, to map a URL or path to a handler or view
