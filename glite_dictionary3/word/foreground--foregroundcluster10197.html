foreground

noun
active window or application (computing)
Pronunciation: / ˈfɔrˌɡraʊnd / (AmE), / ˈfɔːɡraʊnd / (BrE)
(computing) The window or application currently active and in focus on a computer screen.

Examples

the foreground window
If you click on the icon, that program will move to the foreground.
Make sure the browser is in the foreground before you start the meeting.
The chat window popped up in the foreground, covering my document.

Related Terms

  • antonym – background: (computing) the state or place where an application is running but not currently active or visible
