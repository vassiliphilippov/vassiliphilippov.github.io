adjective
highly detailed; granular

Examples

fine-grained data control
The dataset offers fine-grained categories enabling highly precise customer segmentation.
The security team implemented fine-grained permissions to control access across different user roles.
Machine learning models benefit from fine-grained labels that capture subtle differences between classes.
