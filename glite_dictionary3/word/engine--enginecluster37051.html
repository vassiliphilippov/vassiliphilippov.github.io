engine

noun
software system providing core functions
Pronunciation: / ˈen.dʒɪn / (AmE), / ˈen.dʒɪn / (BrE)
a software system or component that provides the core functions needed to run, render, or control a digital application or service (for example, a game engine or rendering engine).

Examples

game engine
The new engine improves physics and collision detection in the mobile game.
Developers chose a different rendering engine to achieve realistic lighting and reflections.
Search engines use complex software to find and rank relevant web pages for queries.

Related Terms

  • synonym – framework: a reusable set of software components that provides a structure for developing applications
  • synonym – platform: a base software environment that supports the running of applications or services
