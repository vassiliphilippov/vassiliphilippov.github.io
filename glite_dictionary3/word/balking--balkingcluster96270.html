balking

noun
operation that returns early when preconditions fail
Pronunciation: / ˈbɔlkɪŋ / (AmE), / ˈbɔːkɪŋ / (BrE)
(computing) the behaviour of an operation or design pattern in which a method immediately returns without taking action when required preconditions are not met.

Examples

balking pattern skips
The API uses balking to skip work if the input is invalid.
After detecting a race condition, the service implemented balking to avoid unsafe state changes.
Developers write unit tests to ensure a method's balking behaviour returns immediately on bad input.

Related Terms

  • synonym – fail-fast: a design approach where a system or component stops normal operation and reports an error immediately when it detects a problem
  • synonym – guard clause: a programming practice of checking conditions early and returning to avoid deeper nesting
