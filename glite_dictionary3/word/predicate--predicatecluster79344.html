predicate

noun
boolean test function (computing)
Pronunciation: / ˈprɛdɪkət / (AmE), / ˈprɛdɪkət / (BrE)
In computer science, a function or expression that evaluates input and returns a Boolean value (true or false), used to test conditions, filter data, or control program flow.

Examples

predicate function
In functional programming, a predicate returns true or false to decide which items to keep.
Database queries use predicates to filter rows that meet specific conditions efficiently.
The unit test included a predicate that verified whether the input string was empty.

Related Terms

  • synonym – boolean function: a function that returns either true or false for given inputs; essentially equivalent to a predicate in computing.
  • synonym – predicate (logic): the logical notion of what is asserted about a subject; the computing sense is historically derived from this idea.
