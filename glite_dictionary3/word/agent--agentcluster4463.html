agent

noun
person or thing causing an effect
Pronunciation: / ˈeɪdʒənt / (AmE), / ˈeɪdʒənt / (BrE)
a person, thing, or substance that produces a particular effect or is responsible for a specific result, especially in science or medicine

Examples

cleaning agent
Enzymes function as catalysts and agents in chemical reactions within the body.
He investigated the unknown agent that was causing the mysterious illness.
The disinfectant acts as an agent in killing harmful bacteria.
Doctors used a contrast agent to improve the clarity of the MRI scan.
The cleaning agent removed even the toughest stains from the floor.
This chemical agent is found in many household disinfectants.

  • synonym – cause: something that produces a result or effect
  • antonym – effect: a result or outcome produced by a cause
