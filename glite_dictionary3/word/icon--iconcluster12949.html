icon

noun
graphic symbol representing a program or file
Pronunciation: / ˈaɪkɑn / (AmE), / ˈaɪkɒn / (BrE)
a small graphic symbol on a computer screen that represents a program, function, file, or concept, used to trigger actions in a graphical user interface

Examples

desktop icon
Each file type is represented by a unique icon in the folder window.
She created a shortcut icon for quick access to her favorite program.
To open the application, double-click the icon on your desktop.
