fact

noun
numeric measure in a fact table
Pronunciation: / fækt / (AmE), / fækt / (BrE)
(in data warehousing) a numeric measurement or quantitative value recorded in a fact table that represents a business event or transaction and is used for analysis.

Examples

fact table data
The data warehouse's fact table stores monthly_sales as numeric measures for analysis.
When designing the schema, include clear keys so each fact links correctly to its dimensions.
Analysts found the fact for order_total missing, so they recalculated values from transaction logs.

  • synonym – metric: a standard of measurement used to evaluate or monitor performance
