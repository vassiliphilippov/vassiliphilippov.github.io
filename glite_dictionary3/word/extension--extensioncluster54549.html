extension

noun
computer filename suffix
Pronunciation: / ɪkˈstɛnʃən / (AmE), / ɪkˈstɛnʃən / (BrE)
the set of letters, usually following a period at the end of a computer filename, indicating the file type or format

Examples

file extension name
Make sure to include the correct extension when saving your file so it opens properly.
Many image files have the .jpg extension to identify them as JPEG images.
The document's extension was changed from .txt to .pdf to indicate its new format.

Related Terms

  • synonym – suffix: a letter or group of letters added at the end of a word or filename
