to step

verb
execute one instruction (computing)
Pronunciation: / stɛp / (AmE), / stɛp / (BrE)
(computing) to cause a computer or program to execute a single command or instruction, usually for debugging purposes.

Examples

to step through code
She likes to step the application line by line to understand its logic.
The debugger allows programmers to step each instruction and monitor variable changes.
You can step through the code to find exactly where the error occurs.

Related Terms

  • synonym – debug: find and fix errors in a computer program
