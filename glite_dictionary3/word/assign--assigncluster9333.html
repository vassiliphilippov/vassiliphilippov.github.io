to assign

verb
give a value to a variable
Pronunciation: / əˈsaɪn / (AmE), / əˈsaɪn / (BrE)
(in programming) to give or store a value in a variable or identifier so the program can use that value.

Examples

to assign a value
When you assign 5 to count in Python, the variable will hold that number.
The instructor asked students to assign a unique ID to each submitted file.
She assigned the function's return value to the variable named total.

Related Terms

  • synonym – bind: to connect or link a value to a variable, name, or parameter in code.
  • synonym – set: to put a value into a variable or place (often used interchangeably with assign in programming).
