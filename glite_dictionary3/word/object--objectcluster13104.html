object

noun
computing entity with data and methods
Pronunciation: / ˈɑbˌdʒɛkt /; / ˈɑb.dʒɛkt / (AmE), / ˈɒbdʒɪkt /; / ˈɒb.dʒɛkt / (BrE)
a discrete entity in computing, such as a unit in programming or a unit of data in storage

Examples

data object
By modifying the object, you can update the user's information dynamically.
Each object in the program can send messages to other objects.
The object stores both a name and an age as its attributes.
The backup service stores each document as an object in cloud storage.
To retrieve the photo, she requested the object by its unique identifier from S3.
Large datasets are often stored as objects, each with metadata that aids searching and versioning.

Related Terms

  • synonym – blob: a binary large object — a raw block of data often stored in databases or cloud storage; similar concept in some systems.
  • synonym – class: a blueprint for creating objects in programming
  • synonym – file: a named container for data on a storage system; often analogous to an object though systems differ.
  • synonym – instance: an individualized realization of an object or class
