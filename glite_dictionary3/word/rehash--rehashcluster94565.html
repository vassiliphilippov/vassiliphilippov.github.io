to rehash

verb
recompute hash value (computing)
Pronunciation: / riˈhæʃ / (AmE), / ˌriˈhæʃ / (BrE)
(computing) to recompute a hash value for data or to rebuild a hash table, often to use a stronger algorithm or to redistribute entries.

Examples

to rehash a password
The security team will rehash all user passwords using a stronger hashing algorithm this weekend.
After the migration, the database administrator rehashed the table keys to distribute entries across new servers.
When the load increased, they started rehashing entries more often to reduce collisions in the hash table.

Related Terms

  • synonym – hash function: a function that maps data of arbitrary size to fixed-size values used in hashing and cryptography
  • synonym – recompute: to compute something again, often to update or verify a result
