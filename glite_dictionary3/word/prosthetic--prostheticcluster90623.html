prosthetic

adjective
metaphorical: memory or cognitive augmentation
Pronunciation: / prɑˈsθɛtɪk / (AmE), / prɒsˈθɛtɪk / (BrE)
(in cultural or media studies) describing an artificial or technology‑mediated extension or replacement of memory or cognitive faculties; relating to memories formed, shaped, or provided by media or devices.

Examples

prosthetic memory
Landsberg coined prosthetic memory to describe media's power to shape collective memory.
Documentaries often create prosthetic memories by using archival footage to simulate direct experience.
Some theorists argue social media functions as a prosthetic instrument of cultural recall and identity.

Related Terms

  • synonym – augmentation: an increase or improvement that enhances a function or ability
