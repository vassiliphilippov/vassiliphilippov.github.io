mapping

noun
mathematics: function or association between sets
Pronunciation: / ˈmæpɪŋ / (AmE), / ˈmæpɪŋ / (BrE)
(Mathematics) A function or relation that associates each element of a given set (the domain) with an element of another set (the range).

Examples

mathematical mapping function
A continuous mapping may have a wide range of applications in analysis.
Any linear transformation is a type of mapping between vector spaces.
