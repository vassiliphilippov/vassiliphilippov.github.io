overcloud

noun
top-level cloud deployment (OpenStack)
Pronunciation: / oʊvərˈklaʊd / (AmE), / əʊvəˈklaʊd / (BrE)
(in cloud computing, esp. OpenStack) the top-level cloud deployment that provides tenant-facing services and virtual resources; typically deployed on top of an administrative 'undercloud'.

Examples

OpenStack overcloud deployment
The OpenStack team deployed an overcloud to provide virtual machines for multiple tenants.
During testing they noticed the overcloud failed to scale under heavy network traffic and load.
Operators use configuration files to customize each overcloud when provisioning resources for customers.

Related Terms

  • synonym – cloud: a network of remote servers providing computing services such as storage and virtual machines
  • antonym – undercloud: the lower-level administrative cloud used to deploy and manage the overcloud in OpenStack
