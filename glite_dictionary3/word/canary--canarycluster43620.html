canary

noun
small test release in software
Pronunciation: / kəˈnɛri / (AmE), / kəˈneəri / (BrE)
(in software development) a small, limited release of a new version deployed to a subset of users to test for problems before a wider rollout.

Examples

canary software release
Before the wide launch, the engineers ran a canary to monitor error rates and performance.
They noticed a bug in the canary and rolled back the change for affected users immediately.
Our release process includes a canary serving ten percent of traffic for twenty-four hours before scaling.

Related Terms

  • synonym – canary release: a small-scale deployment of a new software version to a limited user group for testing
  • antonym – full rollout: deployment of a new software version to all users at once
  • synonym – staged rollout: a process of releasing software updates gradually to increasing groups of users
