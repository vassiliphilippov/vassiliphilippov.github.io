identifier

noun
symbol or code used for identification
Pronunciation: / aɪˈdɛntəˌfaɪər / (AmE), / aɪˈdɛntɪfaɪə / (BrE)
a symbol, word, or set of characters that uniquely identifies a person, object, or entity

Examples

unique identifier
Each employee is assigned a unique identifier for accessing the computer system.
In programming, a variable name acts as an identifier for data storage.
The barcode on the package serves as an identifier for tracking shipments.
