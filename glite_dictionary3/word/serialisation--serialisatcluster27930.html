serialisation

noun
converting objects for storage or transmission
Pronunciation: / ccs6ari59la6ac8ze6a8359n / (AmE), / ccs6a59ri59la6ac8ze6a8359n / (BrE)
(in computing) the process of converting an in‑memory object or data structure into a linear format suitable for storage or transmission, so it can later be reconstructed.

Examples

serialisation of data
The library supports serialisation of complex objects to a portable binary format for network transfer.
A bug in the code made the serialisation produce corrupted files that failed to reload correctly.
They benchmarked JSON and Protobuf serialisation speed when designing the API for mobile clients.

Related Terms

  • antonym – deserialization: the reverse process of reconstructing objects or data structures from a stored or transmitted representation.
  • synonym – serialization: American English spelling of the same computing process.
