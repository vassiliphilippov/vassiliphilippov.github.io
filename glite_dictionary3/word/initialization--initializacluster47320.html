initialization

noun
setting up a storage device or system to start
Pronunciation: / ɪˌnɪʃəlaɪˈzeɪʃən / (AmE), / ɪˌnɪʃɪlaɪˈzeɪʃən / (BrE)
(Computing) The process of preparing a computer storage device, such as formatting sectors on a hard disk drive, and setting a starting position so that it can be accessed by the operating system.

Examples

disk initialization
After initialization, the operating system can recognize and use the disk.
Before using the new hard drive, an initialization is required to format its sectors.
The initialization process ensures that all data storage locations are accessible.

Related Terms

  • synonym – formatting: preparing a storage medium, especially by creating a file system
  • synonym – reset: restoring a system to an original or starting condition
