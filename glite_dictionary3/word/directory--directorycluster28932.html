directory

noun
computing: list of computer files and folders
Pronunciation: / dəˈrɛktəri, daɪˈrɛktəri / (AmE), / dɪˈrɛktəri, daɪˈrɛktəri / (BrE)
(computing) a file or folder on a computer that contains a list of files and subdirectories.

Examples

directory of files
