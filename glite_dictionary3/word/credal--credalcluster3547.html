credal

adjective
relating to credal sets/ imprecise probability
Pronunciation: / ˈkriː.dəl / (AmE), / ˈkriː.dəl / (BrE)
(in statistics and imprecise probability) relating to a credal set or models that represent uncertainty by a range of plausible probability distributions rather than a single distribution.

Examples

credal set
The researchers built a credal network to represent uncertain relationships between variables in the model.
Using a credal set allows statisticians to express several plausible probability distributions simultaneously.
Decision makers preferred the credal approach because it captured ambiguity in the available evidence.

Related Terms

  • synonym – credal set: a set of probability distributions representing several plausible beliefs about uncertain events
  • synonym – imprecise probability: a framework for representing uncertainty using sets of probability distributions rather than a single one
