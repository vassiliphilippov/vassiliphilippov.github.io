syllabic

adjective
consisting of or using a syllabary
Pronunciation: / sɪˈlæbɪk / (AmE), / sɪˈlæbɪk / (BrE)
consisting of or relating to a writing system in which each symbol represents a syllable rather than an individual sound; using a syllabary.

Examples

syllabic writing system
Canadian Aboriginal scripts include syllabic characters that represent complete syllables.
In linguistics, a syllabic script differs from an alphabetic script, as each sign corresponds to a syllable.
The Cherokee language employs a syllabic writing system developed in the early 19th century.

Related Terms

  • antonym – alphabetic: using or relating to an alphabet, where each symbol represents an individual sound
