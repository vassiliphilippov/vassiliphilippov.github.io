default

noun
automatically selected option
Pronunciation: / dɪˈfɔlt / (AmE), / dɪˈfɔːlt / (BrE)
an option or setting that is automatically used by a system or device unless the user chooses a different one

Examples

default settings option
If you don't choose a font, the program uses the default setting.
She reset the device to its factory defaults before selling it.
The default browser on the new phone is quite fast.

Related Terms

  • antonym – preference: a greater liking for one alternative over others
