worker

noun
computing process or thread that performs tasks
Pronunciation: / ˈwɜrkər / (AmE), / ˈwɜːkə / (BrE)
(in computing) a program process, thread, or node that carries out tasks or units of work, often in parallel or asynchronously and managed by a scheduler or job queue.

Examples

worker process
The server uses several worker processes to execute tasks in parallel.
Each worker polls the job queue, performs the required computation, then reports completion.
They scaled the application by adding more workers, reducing processing time for background jobs.

Related Terms

  • synonym – process: a running instance of a program that can execute tasks or operations in an operating system
  • synonym – thread: a smaller sequence of programmed instructions within a process that can run concurrently
