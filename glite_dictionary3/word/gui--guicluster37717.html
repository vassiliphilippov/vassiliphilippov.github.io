GUI

noun
graphical user interface for computers/devices
Pronunciation: / dʒi ju aɪ / (AmE), / dʒiː juː aɪ / (BrE)
a user interface that allows interaction with electronic devices using graphical elements such as windows, icons, and menus, typically operated by a mouse and keyboard

Examples

easy GUI navigation
Some advanced users still prefer the command line, but most rely on a GUI for daily tasks.
Switching from a text-based system to a GUI improved user productivity considerably.
The new software uses a modern GUI that is easy for beginners to navigate.

Related Terms

  • antonym – CLI: command-line interface; a text-based way to interact with computers
  • synonym – graphical user interface: a user interface using graphics; expanded form of 'GUI'
