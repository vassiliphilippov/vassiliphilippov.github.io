programme

noun
set of instructions for a computer
Pronunciation: / ˈproʊɡræm / (AmE), / ˈprəʊɡræm / (BrE)
a set of coded instructions that a computer can understand and carry out to perform a specific task or function

Examples

computer programme code
He spent the afternoon writing a programme to sort large databases.
The company released an updated programme that fixes several security issues.
Without the correct programme, the computer cannot process the data.
