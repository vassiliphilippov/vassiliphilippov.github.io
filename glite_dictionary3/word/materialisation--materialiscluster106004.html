materialisation

noun
storing query/view results (computing)
Pronunciation: / məˌtɪəriəlaɪˈzeɪʃən / (AmE), / məˌtɪəriəlaɪˈzeɪʃən / (BrE)
(in computing) the process of storing a virtual query, view, or computed result as a physical object (for example, a table) so it can be accessed more quickly.

Examples

materialisation of a table
The database team improved performance by materialisation of the query results into a cached table.
To speed reporting, they scheduled nightly materialisation of the view so analysts had instant access.
Cloud data pipelines often include materialisation steps that store computed results for reuse.

Related Terms

  • synonym – cache: a stored copy of data kept for faster access; similar in purpose to materialisation
  • synonym – materialized view: a stored database object that contains the results of a query, used to speed up queries
