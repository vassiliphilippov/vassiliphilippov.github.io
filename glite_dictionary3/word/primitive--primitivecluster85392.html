primitive

noun
function from which another is derived (math)
Pronunciation: / ˈprɪm.ə.tɪv / (AmE), / ˈprɪm.ɪ.tɪv / (BrE)
A mathematical function or expression from which another is derived, commonly by differentiation; an antiderivative.

Examples

a primitive function
In calculus, finding a primitive is the inverse process of differentiation.
Physicists often use the primitive of a velocity function to determine displacement.
The primitive of a continuous function can be found using integration.

Related Terms

  • synonym – antiderivative: a function whose derivative is a given function
