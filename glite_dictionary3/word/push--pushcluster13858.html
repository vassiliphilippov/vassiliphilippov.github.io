push

noun
server-initiated delivery model
Pronunciation: / p\u028a\u0283 / (AmE), / p\u028a\u0283 / (BrE)
(in computing) a method or system in which a server or service initiates delivery of updates, notifications, or data to clients without waiting for client requests.

Examples

push notification system
Mobile apps use a push model to deliver notifications and updates to users.
Switching to push reduced polling traffic and made the interface feel more responsive.
Their cloud platform supports both pull and push for delivering live data streams.

Related Terms

  • antonym – pull: a model where the client requests data from the server (contrasts with push)
  • synonym – send: the act of transmitting data or messages to another device or user
