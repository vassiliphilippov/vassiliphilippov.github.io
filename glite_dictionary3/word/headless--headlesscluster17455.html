headless

adjective
without GUI or display (computing)
Pronunciation: / ˈhɛdləs / (AmE), / ˈhɛdləs / (BrE)
(computing) Operating without a graphical user interface or local display, controlled programmatically, from the command line, or over a network.

Examples

headless browser
Developers run automated tests using a headless browser to avoid GUI overhead.
The server hosts a headless application that accepts API requests without rendering any user interface.
You can install a headless Raspberry Pi and manage it entirely over SSH from your laptop.

Related Terms

  • antonym – graphical user interface (GUI): a visual interface with windows, icons, and menus that allows users to interact with software
