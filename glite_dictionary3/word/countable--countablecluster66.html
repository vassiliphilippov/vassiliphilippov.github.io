countable

adjective
able to be counted
Pronunciation: / ˈkaʊntəbəl / (AmE), / ˈkaʊntəbl / (BrE)
able to be counted; existing in a quantity that can be individually enumerated.

Examples

countable items

Some nouns in English are countable, like 'apple' or 'book'.
The loan was given only against countable assets as collateral.
