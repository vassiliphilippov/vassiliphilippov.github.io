wheel

noun
Python built-package format (.whl)
Pronunciation: / wil / (AmE), / wiːl / (BrE)
(in computing, especially Python packaging) a built distribution format for Python packages, provided as a binary file with the .whl extension that can be installed without building from source.

Examples

Python wheel file
The installer prefers a wheel file over a source tarball for faster installs.
You can install the package from the wheel instead of compiling its extensions locally.
The CI build produced a universal wheel compatible with Python 3.8 and 3.9.

Related Terms

  • antonym – source distribution: a package format that contains source code and must be built on installation; contrasted with a wheel
  • synonym – wheel file: a single .whl file containing a built Python package ready for installation
