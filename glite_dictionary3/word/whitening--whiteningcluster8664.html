whitening

noun
data preprocessing to decorrelate and scale features
Pronunciation: / ˈwaɪt.nɪŋ / (AmE), / ˈwaɪt.nɪŋ / (BrE)
(in statistics and machine learning) a preprocessing transformation that decorrelates features and rescales them so the data have identity covariance (uncorrelated, unit-variance components).

Examples

data whitening step
We applied whitening to the dataset before running the clustering algorithm.
A common approach is PCA whitening, which reduces correlations then rescales the principal components.
Before training the neural network she used whitening to speed up convergence.

Related Terms

  • synonym – decorrelation: the process of making variables uncorrelated, a key effect of whitening transforms.
  • synonym – sphering: a transformation that makes a dataset's covariance matrix the identity matrix (another name for whitening).
