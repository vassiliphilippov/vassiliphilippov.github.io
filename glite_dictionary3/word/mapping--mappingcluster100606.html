mapping

noun
computing: linking data fields between schemas
Pronunciation: / ˈmæpɪŋ / (AmE), / ˈmæpɪŋ / (BrE)
In computing, the process of defining correspondences between fields or elements in different databases or schemas so that data can be transformed, migrated, or integrated.

Examples

data mapping fields
During the migration, the team created a detailed mapping of customer fields to the new CRM schema.
Automated tools suggested several mappings, but developers reviewed each one to avoid data loss.
A clear mapping between source and target tables prevented errors in the nightly ETL job.

Related Terms

  • synonym – data transformation: the process of converting data from one format or structure into another
  • synonym – schema mapping: defining how elements in one database schema correspond to elements in another
