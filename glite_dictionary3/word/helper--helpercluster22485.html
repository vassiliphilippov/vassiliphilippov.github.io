helper

noun
small supporting software component
Pronunciation: / ˈhɛlpər / (AmE), / ˈhɛlpə / (BrE)
a small software component (such as a function or module) that provides auxiliary or commonly used functionality to support other code.

Examples

helper function
The helper function formats dates and calculates time zones used across the application.
Developers created several helpers to validate input and simplify form submission logic.
When debugging, she imported a small helper module that generated mock data for tests.

Related Terms

  • synonym – module: a self-contained piece of software that provides related functionality
  • synonym – utility function: a small function that performs a common, reusable task for other code
