trait

noun
programming unit of reusable methods
Pronunciation: / treɪt / (AmE), / treɪt / (BrE)
(in programming) a reusable unit of methods and behavior that can be mixed into classes or types to add functionality without relying on classical inheritance.

Examples

programming trait
In Scala, a trait defines methods and fields that classes can reuse without full inheritance.
She used multiple traits to compose functionality across different classes in the project.
When adding logging, they created a lightweight trait that provided the necessary helper methods.

Related Terms

  • synonym – mixin: a set of methods or behavior that can be included in a class to provide additional functionality
