bounce

noun
single-page session (web analytics)
Pronunciation: / ba8ans / (AmE), / ba8ans / (BrE)
(in web analytics) a single-page session on a website in which the visitor leaves without viewing any other pages or taking a tracked action.

Examples

website bounce rate
The web analytics report shows a high bounce rate for the homepage last month.
Most users left after viewing a single article, so that visit counts as a bounce.
Lowering page load time helped reduce the number of bounce events across the site.

Related Terms

  • related – bounce rate: the percentage of sessions that are bounces (single-page visits)
  • synonym – single-page session: a website visit in which the user views only one page before leaving
