ROM

noun
digital image of cartridge/software
Pronunciation: / ɑr oʊ ɛm / (AmE), / ɑːr əʊ ɛm / (BrE)
A digital file that contains a copy of the data from a read-only memory device (for example, a game cartridge), typically used with emulators.

Examples

ROM file
Retro gamers often download ROM images to play classic games on emulators.
She kept a backup of the original ROM after dumping the cartridge to her computer.
Distributing commercial ROM files is illegal in many countries despite emulator availability.

Related Terms

  • synonym – ROM image: a file containing an extracted copy of a ROM's contents, used with emulators
  • synonym – disk image: a file that is an exact copy of a storage device's contents; similar concept for storage images
