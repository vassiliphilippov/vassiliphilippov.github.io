to dispose

verb
release/free program resources (computing)
Pronunciation: / dɪˈspoʊz / (AmE), / dɪˈspəʊz / (BrE)
(in computing) to release or free resources used by a program (such as memory, file handles, or network connections), typically by calling a specific method or function so the resources can be reused.

Examples

to dispose resources safely
When a connection closes, the library automatically disposes any associated file handles to avoid errors.
After running the tests, she disposed the temporary buffers to free up system memory.
Developers are careful about disposing long-lived objects, especially those holding network sockets.

Related Terms

  • synonym – deallocate: to remove an allocation of memory or resources so they become available again
  • synonym – release: to let go of or free a resource so it can be used by others
