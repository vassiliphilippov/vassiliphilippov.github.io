menu

noun
list of options on a computer screen
Pronunciation: / ˈmɛnju / (AmE), / ˈmɛnjuː / (BrE)
(computing) a list of commands or options displayed on a computer screen from which a user can choose.

Examples

menu of options
A pop-up menu appeared when I right-clicked the desktop.
The File menu contains commands for opening and saving documents.
You can access the settings through the drop-down menu at the top of the screen.

Related Terms

  • synonym – toolbar: a strip of icons or buttons for performing actions in a software application
