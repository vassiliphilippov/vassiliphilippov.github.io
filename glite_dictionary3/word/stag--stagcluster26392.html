stag

noun
staging environment (IT)
Pronunciation: / stæɡ / (AmE), / stæɡ / (BrE)
(in information technology, informal) a staging or pre-production environment used to test software, deployments, or changes before they go live.

Examples

stag environment
Before the release, we deployed the update to stag to run the full integration tests.
The QA team found a bug on stag that didn't appear in development but failed in production.
We maintain separate databases for stag to mirror production without risking live user data.

Related Terms

  • synonym – pre-production: the phase or environment where final testing occurs before production release
  • antonym – production: the live environment where software is available to end users; the opposite of staging
  • synonym – staging environment: an environment that mirrors production used to test deployments before release
