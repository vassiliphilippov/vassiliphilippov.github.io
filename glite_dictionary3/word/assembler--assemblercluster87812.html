assembler

noun
program converting assembly to machine code
Pronunciation: / əˈsɛmbələr / (AmE), / əˈsɛmblə / (BrE)
a computer program that translates assembly language into machine code.

Examples

assembler software tool
High-level languages use a compiler, but assembly code needs an assembler.
She downloaded an assembler to help build her embedded application.
The assembler converts the written assembly instructions into executable code.

Related Terms

  • synonym – compiler: a program that translates high-level code into machine code
  • antonym – interpreter: a program that executes code line by line instead of compiling it
