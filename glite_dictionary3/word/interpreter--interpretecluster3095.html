interpreter

noun
program that executes code line by line
Pronunciation: / ɪnˈtɝːprətɚ / (AmE), / ɪnˈtɜːprɪtə / (BrE)
(Computing) a computer program that translates and executes each line of source code one at a time, rather than compiling the entire program first.

Examples

interpreter runs code
An interpreter executes the source code line by line rather than translating it all at once.
If you are learning Python, you can run your scripts directly in the interpreter.
Many scripting languages, such as Ruby, rely on an interpreter rather than a compiler.

Related Terms

  • antonym – compiler: a program that translates source code into machine code before execution
