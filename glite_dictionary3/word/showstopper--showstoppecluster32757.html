showstopper

noun
critical blocking defect
Pronunciation: / ˈʃoʊˌstɑpər / (AmE), / ˈʃəʊˌstɒpə / (BrE)
(in software development and project management) a critical defect, bug, or issue that blocks a product release or prevents progress until it is fixed.

Examples

showstopper bug in software
The startup's latest build couldn't ship due to a showstopper bug during testing.
During the meeting QA listed three showstoppers that must be fixed before the product ships.
The project manager called the crash a showstopper and delayed the release until it was solved.

Related Terms

  • synonym – blocking bug: a software bug that prevents a system from functioning or a release from proceeding
  • synonym – release blocker: an issue that must be resolved before software can be released to users
