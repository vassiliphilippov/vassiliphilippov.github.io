more

noun
UI button revealing additional content
Pronunciation: / mɔr / (AmE), / mɔː / (BrE)
A button, menu item, or label in a software interface that the user selects to reveal additional content, options, or settings.

Examples

More button
Tap the More option to view the full article and photos.
The toolbar's More menu hides advanced formatting tools to keep the interface simple.
If you don't see the setting, open More at the bottom of the screen to expand options.
