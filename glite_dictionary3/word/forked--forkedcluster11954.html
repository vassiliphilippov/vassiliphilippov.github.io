forked

adjective
created by forking (software)
Pronunciation: / fɔrkt / (AmE), / fɔːkt / (BrE)
(in computing) created by copying a project's source code or repository so development can continue independently, resulting in divergence from the original.

Examples

forked repository
The forked repository contains experimental features not present in the original.
After disagreements, the community continued development on the forked project independently.
GitHub shows that the popular package now has several forked copies with unique changes.

Related Terms

  • synonym – fork: a copy of a software repository made to begin separate development
