ingestion

noun
importing data into systems
Pronunciation: / ɪnˈdʒɛs.tʃən / (AmE), / ɪnˈdʒɛs.tʃən / (BrE)
(in computing) the process of importing data from external sources into a data store or processing pipeline, either in real time or in batches.

Examples

ingestion of data
The real-time ingestion pipeline streams sensor data into the analytics platform every few seconds.
After the merger, engineers improved batch ingestion to handle larger CSV files reliably overnight.
Cloud services charge for data ingestion based on volume, not processing, which surprised the team.

Related Terms

  • synonym – ETL: the process of extracting, transforming, and loading data into storage or analytics systems
  • synonym – data import: transferring or loading data from external sources into a system
