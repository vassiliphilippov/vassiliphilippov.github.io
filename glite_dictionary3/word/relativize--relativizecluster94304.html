to relativize

verb
make file paths/URLs relative (computing)
Pronunciation: / c8r5bl59cct6ava6az / (AmE), / c8r5bl59t6ava6az / (BrE)
(in computing) to convert an absolute file path or URL into a relative path with respect to a specified base location, so it can be used portably or within a different directory structure.

Examples

to relativize a file path
The build tool can relativize file paths to the project root for portability.
When deploying the site, the script relativized image URLs so links worked on any server.
You should avoid relativizing paths that must remain absolute for system security.

Related Terms

  • synonym – convert: to change something into a different form; here, change an absolute path into a relative one
  • antonym – resolve: to turn a relative path into an absolute path (opposite operation)
