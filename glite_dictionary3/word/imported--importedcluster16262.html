imported

adjective
brought into a program via import
Pronunciation: / ɪmˈpɔrtɪd / (AmE), / ɪmˈpɔːtɪd / (BrE)
(in computing) brought into a program, project, or runtime environment by an import operation; used of modules, libraries, code, or datasets that originate elsewhere.

Examples

imported data
The project failed because the imported library had incompatible dependencies with the runtime environment.
She cleaned and transformed the imported dataset before feeding it into the machine-learning pipeline.
During deployment, the team discovered the imported module was missing a crucial configuration file.

Related Terms

  • antonym – built-in: provided as part of the system or language rather than added from an external source; opposite of imported in computing contexts.
  • synonym – import (computing): to bring code, modules, or data into a program or environment using an import statement or operation.
