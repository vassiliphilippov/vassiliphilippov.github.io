BIN

noun
bank identification number used by financial institutions
Pronunciation: / bɪn / (AmE), / bɪn / (BrE)
an identification number assigned to a financial institution, typically used to identify the issuing bank in a transaction; the first part of the code indicates location, and the second identifies the bank itself

Examples

BIN code number
Each credit card has a unique BIN that identifies its issuing bank.
Merchants use the BIN to determine which bank processed the customer's payment.
Regulations require every new financial institution to receive a new BIN from the central authority.
