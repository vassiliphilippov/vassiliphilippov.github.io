ephemeron

noun
short-lived key–value structure (computing)
Pronunciation: / ɪˈfɛm.əˌrɑn / (AmE), / ɪˈfɛm.ər.ɔn / (BrE)
(in computing) a data structure used in garbage-collected systems to hold a key–value association that does not by itself keep the key alive; when the key becomes unreachable, the association can be removed so the value may be reclaimed.

Examples

an ephemeron object
The runtime uses an ephemeron to cache metadata that should disappear when the key is collected.
Implementers chose an ephemeron so the value would not prevent unused objects from being garbage-collected.
When the key becomes unreachable, the language's ephemeron allows both key and value to be reclaimed.

Related Terms

  • synonym – weak map: a map/dictionary that holds keys weakly so entries can be removed when keys are unreachable
  • synonym – weak reference: a reference that does not prevent an object from being garbage-collected
