cipher

1. NOUN

/ ˈsaɪfər / (AmE), / ˈsaɪfə / (BrE)
a message written in a secret or disguised code

Examples

Mythological concepts may by all means still be used, but they can be used responsibly only as "symbols "or "ciphers ", that is, only if they are also constantly interpreted in non mythological (or existential) terms.

Related Terms

  • Synonym: code - a system of words, letters, figures, or symbols used to represent others, especially for secrecy
  • Antonym: decipher - to convert a coded message into normal language

Etymology

From Arabic *ṣifr* (zero, empty).

2. VERB

/ ˈsaɪfər / (AmE), / ˈsʌɪfə / (BrE)
to convert ordinary language into a secret or coded form

Examples

Print it in real big letters, an' I can cipher it out later ".

Synset Examples

We should encode the message for security reasons

Related Terms

  • Synonym: encode - to convert information into code
  • Antonym: decipher - to convert code into ordinary language
  • Antonym: decode - to translate coded material into ordinary language

Etymology

From Arabic 'ṣifr' (empty, zero), via Medieval Latin 'cifra', referring originally to zero or null value, later to secret writing.
