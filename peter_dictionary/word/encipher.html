encipher

1. VERB

/ ɪnˈsaɪfɚ / (AmE), / ɪnˈsaɪfə / (BrE)
to convert (a message or text) into a coded form, especially for secrecy; encrypt

Examples

In the work of every artist, I suppose, there may be found one or more moments which strike the student as absolutely decisive, ultimately emblematic of what it is all about; not less strikingly so for being mysterious, as though some deeply hidden constatation of thoughts were enciphered in a single image, a single moment.

Synset Examples

We should encode the message for security reasons

Related Terms

  • Antonym: decipher - convert coded text back to ordinary language
  • Synonym: encrypt - convert data into a coded form (especially digital)
  • Synonym: code - to convert a message into a coded form

Etymology

From prefix 'en-' (to make) + 'cipher' (a secret code or zero), influenced by Middle French 'chiffrer'.
