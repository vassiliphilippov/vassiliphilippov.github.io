extremum

1. NOUN

/ ɪkˈstriːməm / (AmE), / ɪkˈstriːməm / (BrE)
the point at which a function or quantity reaches its maximum or minimum value

Examples

Furthermore, one can find a neighborhood of Q in which the difference function is monotone, for since it is analytic it can have only a finite number of extrema in any interval.

Related Terms

  • Degree Lower: minimum - the lowest point or value in a mathematical context
  • Degree Higher: maximum - the highest point or value in a mathematical context

Etymology

From Latin 'extremum', neuter of 'extremus' (outermost, utmost).
