base hit

1. NOUN

/ beɪs hɪt / (AmE), / beɪs hɪt / (BrE)
(Baseball) A successful hit that enables the batter to reach first base or a subsequent base safely.

Examples

If it's true that contented cows give more milk, why should n't happy ball players produce more base hits?

Related Terms

  • Degree Lower: single - a base hit in which the batter reaches first base
  • Degree Higher: double - a base hit in which the batter reaches second base
  • Degree Higher: triple - a base hit in which the batter reaches third base
  • Degree Higher: home run - a base hit in which the batter circles all bases and scores
