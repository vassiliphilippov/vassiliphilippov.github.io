inverse function

1. NOUN

/ ɪnˈvɝs ˈfʌŋkʃən / (AmE), / ɪnˈvɜːs ˈfʌŋkʃən / (BrE)
A function that reverses the effect of another function, so that applying one after the other returns the original value.

Examples

The difficulty of analysis of any subsystem in the phonology is an inverse function of the size - smaller systems are more troublesome - for any given degree of morphophonemic complexity.

Related Terms

  • Synonym: function - a relation or expression involving one or more variables

Etymology

Formed from Latin 'inversus' (turned upside down) and 'function'.
