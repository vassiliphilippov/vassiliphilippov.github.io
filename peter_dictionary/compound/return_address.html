return address

1. NOUN

/ rɪˈtɝn ædˌrɛs / (AmE), / rɪˈtɜːn əˌdrɛs / (BrE)
the address of the sender on a letter or parcel, indicating where it should be sent back if delivery fails

Examples

The check had been mailed from Chicago, the envelope bore no return address, and the check was not signed.
