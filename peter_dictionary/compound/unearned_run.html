unearned run

1. NOUN

/ ʌnˈɝnd rʌn / (AmE), / ʌnˈɜːnd rʌn / (BrE)
(in baseball) a run that scores as a result of a defensive error or passed ball, and not as the result of normal play.

Examples

Two errors by New York Yankee shortstop Tony Kubek in the eleventh inning donated four unearned runs and a 5 - to 2 victory to the Chicago White Sox today.

Related Terms

  • Antonym: earned run - a run that scores without benefit of a defensive error
