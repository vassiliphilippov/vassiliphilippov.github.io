set decoration

1. NOUN

/ sɛt ˌdɛk.əˈreɪ.ʃən / (AmE), / sɛt ˌdɛk.əˈreɪ.ʃən / (BrE)
An object or ornament used to decorate a stage or film set as part of the overall visual design in a theatrical or movie production.

Examples

"Designs, plans and furnishes interiors of houses, commercial and institutional structures, hotels, clubs, ships, theaters, as well as set decorations for motion picture arts and television.

Related Terms

  • Synonym: set dressing - decorative objects placed on a set for visual effect
