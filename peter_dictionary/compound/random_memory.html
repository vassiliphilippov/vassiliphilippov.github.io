random memory

1. NOUN

/ ˈrændəm ˈmɛməri / (AmE), / ˈrændəm ˈmɛməri / (BrE)
A type of computer memory chip, also known as random-access memory (RAM), where data can be read from or written to in any order and all storage locations are equally accessible by programs while the computer is running.

Related Terms

  • Synonym: RAM - abbreviation for random-access memory
  • Antonym: read-only memory - computer memory where data can only be read, not written

Etymology

Shortened from 'random-access memory,' so called because access to storage locations can occur in any order, as opposed to sequential storage.
