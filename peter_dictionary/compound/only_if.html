only if

1. SCONJ

/ ˈoʊnli ɪf / (AmE), / ˈəʊnli ɪf / (BrE)
used to indicate that something occurs or is true in no circumstances except the one specified; never except when.

Examples

The proper correlation of the art with the academic can be achieved only if this standard is observed.

Synset Examples

call me only if your cold gets worse

Related Terms

  • Synonym: if - used to introduce a conditional clause
