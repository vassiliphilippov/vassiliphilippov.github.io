earned run

1. NOUN

/ ˈɝnd ˈrʌn / (AmE), / ˈɜːnd ˈrʌn / (BrE)
(Baseball) A run that is scored without the benefit of an error or a passed ball by the fielding team.

Examples

Broglio's 4 - 0 won-lost record and 1.24 earned-run mark against Pittsburgh a year ago; 2.

Related Terms

  • Antonym: unearned run - a run in baseball that results from a fielding error or passed ball
