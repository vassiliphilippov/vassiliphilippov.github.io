pool table

1. NOUN

/ pul ˈteɪ.bəl / (AmE), / puːl ˈteɪ.bəl / (BrE)
A heavy, rectangular table with a smooth, cloth-covered surface and six pockets, used for playing the game of pool.

Examples

If you're gonna ask for something, ask for a pool table.

Related Terms

  • Synonym: billiard table - table used for various billiards games, such as pool or snooker
  • Degree Higher: snooker table - table specifically designed for playing snooker
