random-access memory

1. NOUN

/ ˈrændəm ˈæksɛs ˈmɛməri / (AmE), / ˈrændəm ˈæksɛs ˈmɛməri / (BrE)
A type of computer memory (RAM) that allows information to be read from or written to any address in roughly the same amount of time, commonly used to temporarily store data for active programs and processes.

Examples

The formula for computing random addresses from the form of each occurrence will not give a distinct address for each distinct form.

Related Terms

  • Synonym: RAM - Abbreviation for random-access memory
  • Antonym: read-only memory - Type of computer memory that can only be read and not written to

Etymology

Formed from 'random access' (indicating direct access to any memory location) and 'memory'; widely abbreviated as RAM.
