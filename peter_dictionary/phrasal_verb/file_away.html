file away

1. VERB

/ faɪl əˈweɪ / (AmE), / faɪl əˈweɪ / (BrE)
To put documents or information into a file or other place for storage and later reference.

Examples

She instinctively sheered away from personal discussions but she filed the comment away for further meditation.

Synset Examples

File these bills, please

Related Terms

  • Synonym: store away - to put something in a safe or secret place for future use
