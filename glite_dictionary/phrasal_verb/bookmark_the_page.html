to bookmark the page

1. VERB

/ ˈbʊkˌmɑrk ðə peɪdʒ / (AmE), / ˈbʊkˌmɑːk ðə peɪdʒ / (BrE)
to save the address of a webpage in a web browser so that you can find it again easily later
save a webpage for later access

Examples

I always bookmark the page when I find an interesting article online.
Make sure to bookmark the page so you can return to it easily later.
She quickly bookmarked the page before closing her browser.

Synset Examples

Don't forget to bookmark the page before you close the book.

Related Terms

  • Synonym: save a webpage - to keep the URL of a webpage for later

Etymology

From 'bookmark' (originally a physical marker in a book), applied to digital contexts in the late 20th century.
