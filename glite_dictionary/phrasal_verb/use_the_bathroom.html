to use the bathroom

1. VERB

/ juz ðə ˈbæθˌrum / (AmE), / juːz ðə ˈbɑːθruːm / (BrE)
to go to the toilet; to urinate or defecate, especially expressed in a polite or indirect way.
go to the toilet

Examples

If you need to use the bathroom, please let me know before class starts.
She quietly excused herself to use the bathroom during dinner.
He taught his toddler how to use the bathroom on her own.

Synset Examples

Before we leave, do you need to use the bathroom?
At the rest stop, I asked, do you need to use the bathroom?
Excuse me, I need to use the bathroom quickly.
I’ll be back; I need to use the bathroom.

Related Terms

  • Synonym: go to the toilet - to go to the place where one relieves oneself
