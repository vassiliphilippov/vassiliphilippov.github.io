to set alight

1. VERB

/ sɛt əˈlaɪt / (AmE), / sɛt əˈlaɪt / (BrE)
to cause something to start burning; to ignite or make catch fire
to ignite or cause to catch fire

Examples

The rioters set alight several cars during the protest.
He accidentally set alight the barbeque while trying to start it.
A careless match set alight the curtains in the living room.

Synset Examples

The firecracker set alight the dry grass.
He accidentally set alight the pile of leaves.

Related Terms

  • Synonym: to ignite - to set on fire or cause to burn
  • Antonym: to extinguish - to put out a fire; to cause to stop burning
