to go for a run

1. VERB

/ ɡoʊ fər ə rʌn / (AmE), / ɡəʊ fə ə rʌn / (BrE)
to run for exercise or enjoyment, often as part of a routine or leisure activity
run for exercise or leisure

Examples

He likes to go for a run every morning before work.
Whenever I need to clear my head, I go for a run in the park.
She was going for a run when she came up with the idea.

Synset Examples

I'm going to go for a run before breakfast.

Related Terms

  • Synonym: jog - to run at a steady, gentle pace, especially as a form of exercise
  • Antonym: walk - to move at a regular pace by lifting and setting down each foot in turn
