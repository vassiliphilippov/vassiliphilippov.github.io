to start without

1. VERB

/ stɑrt wɪθˈaʊt / (AmE), / stɑːt wɪðˈaʊt / (BrE)
to begin an activity, event, or meeting before a particular person or group has arrived or is present
begin before someone is present

Examples

Please don't wait for me—just start without me if I'm running late.
They decided to start without John when he failed to show up on time.
If everyone else is ready, we should start without them so we keep to the schedule.

Synset Examples

If I'm not there, start without me and enjoy.
