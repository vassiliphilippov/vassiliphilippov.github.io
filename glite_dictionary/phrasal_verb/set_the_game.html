to set the game

1. VERB

/ sɛt ðə ɡeɪm / (AmE), / sɛt ðə ɡeɪm / (BrE)
to arrange or schedule a game or sporting event for a specific time or date
arrange or schedule a game

Examples

Let's set the game for Friday evening so everyone can attend.
The coach will set the game as soon as the venue is confirmed.
After much discussion, they finally set the game for next Saturday.

Synset Examples

The game is set; it's on this Saturday.

Related Terms

  • Synonym: schedule - to arrange for something to take place at a particular time
