to zip a file

1. VERB

/ zɪp ə faɪl / (AmE), / zɪp ə faɪl / (BrE)
to compress a file or group of files into a single archive using the Zip format, typically to reduce file size or for easier transfer
compress files into a Zip archive

Examples

Please zip a file before you attach it to the email.
She always zips a file when submitting her assignments online to save space.
Learning how to zip a file is basic computer literacy today.

Synset Examples

I will Zip my file to save space.
Make sure to Zip your files before sending them.

Related Terms

  • Synonym: compress - to reduce the size of a file or data
  • Antonym: unzip a file - to extract files from a Zip archive

Etymology

From the verb 'zip', meaning to compress with the Zip format, itself from the proprietary Zip software format developed in 1989.
