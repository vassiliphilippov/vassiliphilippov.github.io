to upgrade the RAM

1. VERB

/ ʌpˈɡreɪd ðə ræm / (AmE), / ʌpˈɡreɪd ðə ræm / (BrE)
to increase the memory capacity of a computer by installing additional or better RAM modules.
increase a computer's memory capacity

Examples

If you upgrade the RAM, you might be able to run more programs simultaneously.
Many gamers choose to upgrade the RAM for better graphics performance.
She decided to upgrade the RAM in her desktop to extend its useful life.

Synset Examples

Upgrading the ram significantly improved the laptop's performance.

Related Terms

  • Synonym: upgrade - raise something to a higher standard, typically by adding or replacing components

Etymology

From 'upgrade' (to raise to a higher standard) and the acronym 'RAM' (Random Access Memory).
