to boot a computer

1. VERB

/ buːt ə kəmˈpjutɚ / (AmE), / buːt ə kəmˈpjuːtə / (BrE)
to start up a computer by loading its operating system
start up a computer

Examples

He forgot to boot a computer before connecting to the internet.
Every morning, she boots a computer to check her emails.
Make sure you boot a computer before running the update.

Synset Examples

I need to boot my computer before starting the work.
He had to boot the old laptop several times to get it working.
Don't forget to boot the system before the presentation.

Related Terms

  • Degree Higher: restart a computer - to shut down and start up a computer again

Etymology

From the figurative use of 'boot' as a shortening of 'bootstrap', referencing the computer term 'bootstrap' (to start a system up from scratch).
