to run one's name past

1. VERB

/ ˈrʌn wʌnz ˈneɪm pæst / (AmE), / ˈrʌn wʌnz ˈneɪm pɑːst / (BrE)
to submit or present one's name to someone for their consideration, approval, or review
submit one's name for consideration

Examples

Before applying, you should run your name past the committee to ensure eligibility.
I decided to run my name past the director before moving forward with the project.
She plans to run her name past the hiring manager for the open position.

Synset Examples
