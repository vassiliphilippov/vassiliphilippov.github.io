to spin up

1. VERB

/ spɪn ʌp / (AmE), / spɪn ʌp / (BrE)
to start or initialize a computer system, service, or process, especially in a technical context
start or initialize a system or service

Examples

The IT team will spin up a new server to handle the increased traffic.
Before testing, you need to spin up the development environment.
It only takes a few minutes to spin up a virtual machine in the cloud.

Synset Examples

We need to spin up the server before Monday.
The team will spin up a new project next week.

Related Terms

  • Synonym: to initialize - set up or start a system or process
  • Antonym: to shut down - to turn off or stop a system or service
