to set aflame

1. VERB

/ sɛt əˈfleɪm / (AmE), / sɛt əˈfleɪm / (BrE)
to cause something to catch fire or start burning
cause to catch fire

Examples

The protestors threatened to set aflame several cars parked outside the building.
An electrical fault caused the old house to be set aflame within minutes.
She watched in horror as the pile of papers was set aflame by a careless match.

Synset Examples

Lightning set the forest afire

Related Terms

  • Synonym: to ignite - to set something on fire; to catch fire
  • Antonym: to extinguish - to put an end to a fire; to put out
