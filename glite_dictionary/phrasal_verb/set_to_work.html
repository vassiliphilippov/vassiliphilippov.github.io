to set to work

1. VERB

/ sɛt tə wɜrk / (AmE), / sɛt tə wɜːk / (BrE)
to begin working on a task or job immediately and with energy
to start working on something immediately

Examples

As soon as the delivery arrived, the team set to work unpacking the boxes.
The moment she entered the kitchen, she set to work preparing dinner for everyone.
Despite feeling tired, he set to work on his project without any delay.

Synset Examples

Once he arrived, he set to work right away.
She set to work organizing the files immediately.

Related Terms

  • Synonym: to get to work - to start working, often used similarly
