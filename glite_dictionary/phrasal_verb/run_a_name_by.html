to run a name by

1. VERB

/ rʌn ə neɪm baɪ / (AmE), / rʌn ə neɪm baɪ / (BrE)
to ask for approval or feedback on a name from someone, especially before making a final decision
seek approval for a name

Examples

Before submitting the application, she decided to run the name by her supervisor.
I’ll run your name by the selection committee and let you know what they think.
Could you run the name by legal before we print the invitations?

Synset Examples

He had to run his name by the committee first.
Make sure to run your name by HR for the event.

Related Terms

  • Synonym: to run by - to present something (such as an idea) for approval or feedback
