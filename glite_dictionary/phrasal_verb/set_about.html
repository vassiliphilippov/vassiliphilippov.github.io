to set about

1. VERB

/ sɛt əˈbaʊt / (AmE), / sɛt əˈbaʊt / (BrE)
to start doing or dealing with something, especially in a determined or organized way
start doing or dealing with something

Examples

After receiving the assignment, she immediately set about researching the topic in detail.
The builders set about repairing the old farmhouse as soon as they arrived.
Once he understood the problem, he set about finding the best solution.

Synset Examples

approach a task
go about a difficult problem
approach a new project

Related Terms

  • Synonym: to go about - to begin or continue with a task or activity
  • Synonym: tackle - to make a strong effort to deal with a problem or task
