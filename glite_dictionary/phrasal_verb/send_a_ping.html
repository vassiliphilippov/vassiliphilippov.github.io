to send a ping

1. VERB

/ sɛnd ə pɪŋ / (AmE), / sɛnd ə pɪŋ / (BrE)
to send a message or signal, especially electronically, to check if someone or something is responsive or available
send a message or signal to check connection

Examples

Let me send a ping to the server to see if it's up.
You can send a ping to your friend to check if he's online.
Before the meeting starts, I'll send a ping to confirm everyone received the invite.

Synset Examples

I sent a ping to check if he's online.

Related Terms

  • Synonym: ping - an electronic signal or message sent to test connectivity; also the action of doing so

Etymology

From the technical term 'ping', originally referring to a network utility that tests connectivity by sending electronic signals; derived from sonar terminology mimicking the sound 'ping'.
