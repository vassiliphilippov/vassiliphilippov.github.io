to speak in code

1. VERB

/ spiːk ɪn koʊd / (AmE), / spiːk ɪn kəʊd / (BrE)
to communicate using words, phrases, or symbols intended to be understood only by a specific group, often to keep information secret
communicate using secret or specialized language
