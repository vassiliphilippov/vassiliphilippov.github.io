to set a target

1. VERB

/ sɛt ə ˈtɑrɡɪt / (AmE), / sɛt ə ˈtɑːɡɪt / (BrE)
to decide on a specific goal or objective to be achieved
establish a goal or objective

Examples

The manager asked each employee to set a target for the upcoming quarter.
It's important to set a target before starting any major project.
They set a target of raising $10,000 for the charity event.

Synset Examples

She set a target to finish her project by Friday.

Related Terms

  • Synonym: set a goal - decide on a specific aim to achieve
  • Degree Higher: achieve a target - successfully reach a previously established goal
