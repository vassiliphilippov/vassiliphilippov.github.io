to file off

1. VERB

/ faɪl ɑf / (AmE), / faɪl ɒf / (BrE)
to remove something from a surface by rubbing it with a file
remove by rubbing with a file

Examples

It is illegal to file off the serial number from any product.
She tried to file off the old paint before repainting the door.
Over time, he had to file off the rough edges to make the part fit.

Synset Examples

It's illegal to file off the serial numbers on products.
He was caught trying to file off the serial numbers.

Related Terms

  • Synonym: to rub off - to remove by rubbing
