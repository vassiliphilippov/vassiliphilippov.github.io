to set up a mobile chicane

1. VERB

/ sɛt ʌp ə ˈmoʊbəl ʃɪˈkeɪn / (AmE), / sɛt ʌp ə ˈməʊbaɪl ʃɪˈkeɪn / (BrE)
to create a temporary barrier or diversion on a road to manage the flow of traffic, often using movable objects.
create temporary road diversion for traffic management

Examples

During the roadworks, they set up a mobile chicane to slow down the vehicles.
To improve pedestrian safety, the city decided to set up a mobile chicane on Main Street.
After the accident, workers quickly set up a mobile chicane to redirect traffic away from the scene.

Synset Examples

They set up a mobile chicane to manage traffic.

Etymology

From French 'chicane' (obstacle), originally in fencing and racing.
