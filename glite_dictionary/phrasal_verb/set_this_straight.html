to set this straight

1. VERB

/ sɛt ðɪs streɪt / (AmE), / sɛt ðɪs streɪt / (BrE)
to correct a misunderstanding or make something clear
correct a misunderstanding

Examples

Let me set this straight—no one is blaming you for the mistake.
Before rumors spread, she wanted to set this straight with everyone involved.
He quickly set this straight by explaining exactly what happened.

Synset Examples

I need to set this straight before it gets worse.

Related Terms

  • Synonym: to clear up - to make something less confused or more understandable
  • Antonym: to confuse - to make unclear or difficult to understand
