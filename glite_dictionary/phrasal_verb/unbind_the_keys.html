to unbind the keys

1. VERB

/ ʌnˈbaɪnd ðə kiz / (AmE), / ʌnˈbaɪnd ðə kiːz / (BrE)
to release or remove assigned functions from keys in a software or hardware system, often to allow reassignment or disable actions
remove functions assigned to keys in a system

Examples

To customize your shortcuts, you should unbind the keys you no longer need.
The instructions say to unbind the keys before assigning new commands.
After an update, I had to unbind the keys to fix a conflict with the default setup.

Synset Examples

You need to unbind the keys before you can switch modes.

Related Terms

  • Antonym: bind the keys - assign functions to keys in a system

Etymology

From "unbind" (to release or set free) + "key" in the sense of a keyboard button; common in computing contexts.
