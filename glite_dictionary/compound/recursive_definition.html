recursive definition

1. NOUN

/ rɪˈkɝːsɪv ˌdɛfɪˈnɪʃən / (AmE), / rɪˈkɜːsɪv ˌdɛfɪˈnɪʃən / (BrE)
a definition in which a function or sequence is defined in terms of itself, allowing its values to be calculated step by step
mathematical definition using recursion
medical

Examples

A recursive definition allows us to describe infinite sequences concisely using only a few equations.
The Fibonacci sequence is commonly defined using a recursive definition.
Understanding the recursive definition helped students calculate higher-order terms in the sequence.

Related Terms

  • Synonym: inductive definition - a definition specifying how to build up all members of a set from the basic cases

Etymology

From Latin 'recurrere' (to run back), via English 'recursion' + 'definition'.
