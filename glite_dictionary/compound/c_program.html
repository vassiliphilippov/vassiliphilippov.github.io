C program

1. NOUN

/ siː ˈproʊɡræm / (AmE), / siː ˈprəʊɡram / (BrE)
a computer program written using the C programming language
computer program written in C

Examples

He wrote a simple C program to sort a list of numbers.
The C program failed to compile due to a missing semicolon.
Learning to debug a C program can be challenging for beginners.

Related Terms

  • Synonym: C - a general-purpose programming language

Etymology

C refers to the C programming language, created in the early 1970s.
