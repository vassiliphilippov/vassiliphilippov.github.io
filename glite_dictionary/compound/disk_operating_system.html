disk operating system

1. NOUN

/ dɪsk ˈɑpəˌreɪtɪŋ ˌsɪstəm / (AmE), / dɪsk ˈɒpəˌreɪtɪŋ ˌsɪstəm / (BrE)
an operating system that is stored on and run from a disk, commonly used in microcomputers to manage files and hardware.
operating system on a disk

Examples

In the early 1980s, most personal computers ran a disk operating system like MS-DOS.
To access files on the floppy drive, you needed to start the disk operating system.
The introduction of the disk operating system greatly simplified data storage and retrieval for users.

Related Terms

  • Synonym: DOS - acronym for disk operating system
