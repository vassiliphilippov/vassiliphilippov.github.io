server issue

1. NOUN

/ ˈsɜrvɚ ˈɪʃu / (AmE), / ˈsɜːvə ˈɪʃuː / (BrE)
a problem or malfunction affecting a computer server, often causing disruptions in services or applications
problem with a computer server

Examples

Several customers reported server issues after last night's upgrade.
Unexpected server issues delayed the launch of the new website.
The IT department is working to resolve persistent server issues affecting email access.

Synset Examples

EverCrack is notorious for causing server issues.

Related Terms

  • Synonym: server problem - an issue or trouble with a server
  • Synonym: network issue - a problem affecting a computer network
