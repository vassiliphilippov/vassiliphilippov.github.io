pen mouse

1. NOUN

/ pɛn maʊs / (AmE), / pɛn maʊs / (BrE)
A wireless computer mouse designed in the shape of a pen, allowing for precision input similar to a writing instrument.
wireless computer mouse shaped like a pen

Examples

The graphic designer prefers using a pen mouse for detailed illustration work.
I bought a pen mouse to make it easier to write notes on my tablet.
The pen mouse connects to the laptop via Bluetooth for wireless convenience.

Related Terms

  • Synonym: stylus - a pen-shaped device used to input commands to a touchscreen
