mouse button

1. NOUN

/ maʊs ˈbʌtən / (AmE), / maʊs ˈbʌtn̩ / (BrE)
a push button found on a computer mouse used to perform functions such as selecting and activating items on a screen
button on a computer mouse

Examples

To select a file, click the left mouse button once.
Some games use the right mouse button for special actions.
If your mouse button sticks, try cleaning it carefully.

Related Terms

  • Synonym: mouse (computer) - a hand-held device used to control a cursor on a computer screen
  • Antonym: touchpad - a touch-sensitive input device as an alternative to a mouse
