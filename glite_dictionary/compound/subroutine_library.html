subroutine library

1. NOUN

/ ˈsʌb.ruˌtin ˈlaɪ.brɛr.i / (AmE), / ˈsʌb.ruˌtiːn ˈlaɪ.brəri / (BrE)
(computing) a collection of standard programs and subroutines stored together for immediate use in software development.
collection of ready-to-use subroutines (computing)

Examples

The development team integrated a subroutine library to speed up coding and simplify maintenance.
Many programming languages provide a standard subroutine library for common tasks.
You can extend the subroutine library with your own custom routines.

Related Terms

  • Synonym: library - a collection of software routines or resources for use in programs
