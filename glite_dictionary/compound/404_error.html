404 error

1. NOUN

/ fɔːr oʊ fɔːr ˈɛrər / (AmE), / fɔːr oʊ fɔːr ˈɛrər / (BrE)
an HTTP error message indicating that the requested web page cannot be found on the server
web page not found error

Examples

When I clicked the old bookmark, a 404 error appeared instead of the page I expected.
The developer fixed the broken link that was causing the 404 error on the homepage.
If you mistype the URL, you might see a 404 error message.

Synset Examples

The website returned a 404 error when I tried to access it.

Related Terms

  • Degree Higher: server error - an error message indicating a problem with the web server
  • Degree Lower: 403 error - an HTTP status code indicating forbidden access to a resource

Etymology

From the HTTP status code 404, meaning 'Not Found', used in the early specifications of the World Wide Web.
