tv set

1. NOUN

/ ˈtiː ˈviː sɛt / (AmE), / ˈtiː ˈviː set / (BrE)
an electronic device that receives television signals and displays them on a screen
device for receiving and displaying TV signals

Examples

We bought a new tv set for the living room last weekend.
The old tv set finally broke after years of loyal service.
She turned off the tv set before going to bed.

Synset Examples

the British call a tv set a ‘telly’

Related Terms

  • Synonym: television - device for viewing television broadcasts
  • Synonym: telly - (British, informal) television set

Etymology

Shortened from 'television set', with 'TV' as an abbreviation for 'television'.
