system command

1. NOUN

/ ˈsɪstəm kəˈmænd / (AmE), / ˈsɪstəm kəˈmɑːnd / (BrE)
An instruction typed or issued by a computer user, typically outside of a program, that requests the operating system to perform an action.
instruction to the computer's operating system

Examples

To shut down the server, enter the appropriate system command in the terminal.
Accidentally deleting files can happen if you mistype a system command.
Learning to use system commands efficiently can make system administration much easier.

Related Terms

  • Synonym: shell command - a command entered at a command-line interface to be executed by a shell
