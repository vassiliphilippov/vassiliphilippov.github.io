telephone extension

1. NOUN

/ ˈtɛləˌfoʊn ɪkˈstɛnʃən / (AmE), / ˈtɛlɪfəʊn ɪkˈstɛnʃən / (BrE)
an additional telephone set connected to the same telephone line as the main telephone, allowing calls to be made or received from another location
additional telephone set on same line

Examples

We installed a telephone extension in the kitchen so we could answer calls while cooking.
The office manager requested a new telephone extension for her assistant's desk.
She answered the call from the telephone extension upstairs.
