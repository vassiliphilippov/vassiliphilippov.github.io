alphabetic script

1. NOUN

/ ˌæl.fəˈbɛ.tɪk ˌskrɪpt / (AmE), / ˌæl.fəˈbɛt.ɪk ˌskrɪpt / (BrE)
a writing system that uses individual letters or characters to represent sounds, forming words and sentences
writing system using an alphabet

Examples

The Greek language was one of the first to be written using an alphabetic script.
Many modern languages, including English and Spanish, use an alphabetic script.
Archaeologists discovered an alphabetic script carved into ancient tablets.

Related Terms

  • Antonym: syllabary - a writing system in which each symbol represents a syllable
  • Synonym: alphabet - a set of letters or symbols in a fixed order used to represent the basic sounds of a language

Etymology

From 'alphabetic' (relating to an alphabet) and 'script' (writing system). 'Alphabetic' from Greek 'alphabētos'.
