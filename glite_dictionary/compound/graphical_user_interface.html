graphical user interface

1. NOUN

/ ˈɡræfɪkəl ˈjuːzər ˈɪntərfeɪs / (AmE), / ˈɡræfɪkəl ˈjuːzə ˈɪntəfeɪs / (BrE)
A computer interface that allows users to interact with electronic devices through graphical icons and visual indicators, rather than text-based commands.
visual computer interface with icons and menus

Examples

Many modern applications use a graphical user interface to make interaction easier for non-technical users.
Before the widespread use of the graphical user interface, most computers were operated through text-based commands.
The development of the graphical user interface revolutionized the way people work with computers.

Related Terms

  • Synonym: GUI - abbreviation for graphical user interface
