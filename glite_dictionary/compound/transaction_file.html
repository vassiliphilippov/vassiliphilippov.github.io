transaction file

1. NOUN

/ trænˈzæk.ʃən faɪl / (AmE), / trænˈzæk.ʃən faɪl / (BrE)
(Computing) A computer file that contains temporary or current data relating to a specific data processing task, typically updated or replaced during processing cycles.
temporary data file in computing

Examples

The system updates the transaction file every time a new order is processed.
After auditing, all records in the transaction file are moved to the master file.
A corrupted transaction file can lead to data inconsistencies in the database.
