crank handle

1. NOUN

/ kræŋk ˈhændl̩ / (AmE), / kræŋk ˈhændl / (BrE)
a handle attached to a crank used to start an engine or operate machinery by turning
handle for starting an engine or machine

Examples

He used the crank handle to start the old tractor.
The generator came with a crank handle in case the battery failed.
You need to attach the crank handle before you can operate the machine manually.

Related Terms

  • Synonym: crank - a handle that is turned to start an engine or operate a machine
