logic element

1. NOUN

/ ˈlɑːdʒɪk ˈɛləmənt / (AmE), / ˈlɒdʒɪk ˈɛlɪmənt / (BrE)
an electronic device that performs a basic logic operation, such as AND, OR, or NOT, in a digital circuit
elementary logic circuit component

Examples

Modern computers rely on millions of logic elements to execute operations quickly.
A simple circuit might use a single logic element such as an AND gate.
Engineers tested the new logic element to ensure it met the required specifications.

Related Terms

  • Synonym: logic gate - a device implementing a Boolean function; core type of logic element
