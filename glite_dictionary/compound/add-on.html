add-on

1. NOUN

/ ˈæd ɑn / (AmE), / ˈæd ɒn / (BrE)
A software module that extends or enhances the functionality of an existing application.
software extension or enhancement

Examples

You can install an add-on to block ads in your browser.
This photo editor offers many add-ons for enhanced functionality.
After installing the language pack add-on, the program supported French.

Related Terms

  • Synonym: plugin - A software component that adds a specific feature to an existing computer program
  • Synonym: extension - A computer program that adds features to another program
Source: WordNet
WordNet ID: oewn-06578832-n
Synset POS: n

2. NOUN

/ ˈæd ɑn / (AmE), / ˈæd ɒn / (BrE)
A supplementary component or feature that can be added to enhance or extend the functionality of something, especially software or equipment.
supplementary feature or component

Examples

You can install an add-on to give the browser more features.
The car's sunroof was available only as an add-on.
Many video games offer optional add-ons to enhance gameplay.

Related Terms

  • Synonym: accessory - an additional item that is useful but not essential
  • Synonym: extension - a part that is added to something to enlarge or prolong it
  • Antonym: core - the central or most important part of something
Source: WordNet
WordNet ID: oewn-02674051-n
Synset POS: n
