set phrase

1. NOUN

/ sɛt freɪz / (AmE), / sɛt freɪz / (BrE)
a fixed expression whose meaning cannot be deduced from the individual words it contains
fixed expression with non-literal meaning

Examples

The teacher explained that 'kick the bucket' is a set phrase meaning 'to die'.
Many set phrases add color and richness to the English language.
Learning the meanings of set phrases is essential for mastering idiomatic English.

Related Terms

  • Synonym: idiom - an expression whose meaning is not predictable from its parts
