unstratified language

1. NOUN

/ ʌnˈstrætɪˌfaɪd ˈlæŋɡwɪdʒ / (AmE), / ʌnˈstrætɪfaɪd ˈlaŋɡwɪdʒ / (BrE)
a programming language that can be used as its own metalanguage, similar to natural languages.
programming language usable as its own metalanguage

Examples

LISP is often cited as an example of an unstratified language due to its ability for meta-programming.
In an unstratified language, the boundaries between code and its description are blurred.
Researchers discuss whether natural languages function as unstratified languages in certain contexts.

Related Terms

  • Antonym: stratified language - a programming language that cannot be used as its own metalanguage

Etymology

From prefix 'un-' (not) + 'stratified' (having layers) + 'language', indicating a language without separate layers of object and metalanguage.
