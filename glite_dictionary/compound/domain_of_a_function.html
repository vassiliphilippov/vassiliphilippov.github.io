domain of a function

1. NOUN

/ doʊˈmeɪn əv ə ˈfʌŋkʃən / (AmE), / dəʊˈmeɪn əv ə ˈfʌŋkʃən / (BrE)
the set of all possible input values (independent variables) for which a mathematical function is defined
set of permissible inputs for a function

Examples

The domain of a function is crucial for determining which values you can input.
To graph this equation properly, you have to identify the domain of a function first.
For the function f(x) = 1/x, the domain of a function excludes zero.

Related Terms

  • Antonym: range of a function - the set of output values a function can produce

Etymology

From Latin 'dominium' (ownership, property); 'function' from Latin 'functio' (performance, execution).
