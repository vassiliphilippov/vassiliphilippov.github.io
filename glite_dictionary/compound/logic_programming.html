logic programming

1. NOUN

/ ˈlɑdʒɪk ˈproʊɡræmɪŋ / (AmE), / ˈlɒdʒɪk ˈprəʊɡræmɪŋ / (BrE)
a style of computer programming in which program statements express facts and rules about problems within a system of formal logic
computer programming using formal logic rules

Examples

Logic programming is widely used in artificial intelligence and computational linguistics.
The course introduced students to logic programming with practical exercises using Prolog.
Although less common than imperative paradigms, logic programming offers unique advantages for certain applications.

Related Terms

  • Antonym: functional programming - a computer programming paradigm based on mathematical functions
  • Synonym: Prolog - a programming language based on logic programming

Etymology

From 'logic' (Greek logikē, reasoning) + 'programming', reflecting the use of formal logic in programming.
