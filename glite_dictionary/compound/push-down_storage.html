push-down storage

1. NOUN

/ ˈpʊʃ daʊn ˈstɔɹ.ɪdʒ / (AmE), / ˈpʊʃ daʊn ˈstɔː.rɪdʒ / (BrE)
a storage device or data structure in which the most recently stored item is the first to be retrieved (last in, first out)
LIFO storage device or structure

Examples

The push-down storage system ensures the latest data entered is the first to be accessed.
A push-down storage is often implemented using a stack in computer science.
When using push-down storage, older records remain until the more recent ones are removed.

Related Terms

  • Synonym: stack - a data structure that stores items in a last-in, first-out manner
