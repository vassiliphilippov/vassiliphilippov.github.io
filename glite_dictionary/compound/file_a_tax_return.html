to file a tax return

1. VERB

/ faɪl ə tæks rɪˈtɝn / (AmE), / faɪl ə tæks rɪˈtɜːn / (BrE)
to officially submit a completed form reporting income and calculating taxes owed, as required by law
submit completed income tax documents

Examples

You must file a tax return if you earned income this year.
Many people use software to file a tax return electronically.
If you don't file a tax return on time, you may be charged a penalty.

Synset Examples

We need to file our tax returns by April.

Related Terms

  • Synonym: submit a tax return - to send in completed tax documents to the authorities
  • Synonym: file taxes - to complete and send in one's tax documents
