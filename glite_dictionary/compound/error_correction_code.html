error correction code

1. NOUN

/ ˈɛrɚ kəˈrɛkʃən koʊd / (AmE), / ˈɛrə kəˈrɛkʃən kəʊd / (BrE)
a system of encoding data that adds extra parity or check bits to detect and correct errors during transmission or storage
code for detecting and correcting transmission errors

Examples

Many modern communication systems use error correction codes to reduce data loss during transmission.
Implementing an error correction code can significantly improve the reliability of digital storage devices.
Engineers selected a robust error correction code to ensure data integrity in satellite communications.

Related Terms

  • Synonym: parity bit - a bit added for error detection
  • Degree Lower: error detection code - a code that detects but may not correct errors
