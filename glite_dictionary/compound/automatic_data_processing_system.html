automatic data processing system

1. NOUN

/ ˌɔtəˌmætɪk ˌdeɪtə ˈprɑsɛsɪŋ ˈsɪstəm / (AmE), / ˌɔːtəˌmætɪk ˌdeɪtə ˈprɒsɛsɪŋ ˈsɪstəm / (BrE)
a computer system consisting of one or more computers and related software, designed for the automatic storage, processing, and retrieval of data
computer system for data processing

Examples

The company installed a new automatic data processing system to improve efficiency and manage large volumes of information.
By using an automatic data processing system, the organization streamlined its accounting operations considerably.
An automatic data processing system allows multiple departments to access shared data securely.

Related Terms

  • Synonym: computer system - a set of integrated computers and software
