computer database

1. NOUN

/ kəmˈpjuːtɚ ˈdeɪtəˌbeɪs / (AmE), / kəmˈpjuːtə ˈdeɪtəˌbeɪs / (BrE)
A collection of data organized and stored electronically that can be accessed and managed by computers.
digitally stored data accessible by computers

Examples

The company's computer database stores information about millions of customers.
A secure computer database is essential for protecting sensitive medical records.
He updated the computer database to include the latest financial reports.

Related Terms

  • Synonym: database - a structured set of data held in a computer
