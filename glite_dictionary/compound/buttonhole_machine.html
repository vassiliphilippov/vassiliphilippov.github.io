buttonhole machine

1. NOUN

/ ˈbʌtnˌhoʊl məˈʃin / (AmE), / ˈbʌtnˌhəʊl məˈʃiːn / (BrE)
a sewing machine designed to make buttonholes, often with the option of using gimp thread for reinforcement
sewing machine for making buttonholes

Examples

The tailor used a buttonhole machine to create neat and uniform openings on the shirt cuffs.
Modern buttonhole machines can sew different shapes and sizes, making garment production more efficient.
She adjusted the settings on the buttonhole machine to ensure each buttonhole was reinforced with gimp thread.
