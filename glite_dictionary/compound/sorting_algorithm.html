sorting algorithm

1. NOUN

/ ˈsɔrtɪŋ ˌælɡəˌrɪðəm / (AmE), / ˈsɔːtɪŋ ˌælɡərɪðəm / (BrE)
a method or set of rules used by a computer to arrange items in a particular order, especially in a list or array
algorithm for arranging items in order

Examples

The quicksort is a popular sorting algorithm for large datasets.
When learning to program, students often study different sorting algorithms such as bubble sort and merge sort.
Choosing the right sorting algorithm can greatly improve software performance.

Related Terms

  • Synonym: search algorithm - an algorithm for finding an item in a list or data structure
  • Synonym: sorting - the process of arranging items in order

Etymology

Formed from 'sort' and 'algorithm', with 'algorithm' from Arabic 'al-Khwarizmi', a mathematician's name.
