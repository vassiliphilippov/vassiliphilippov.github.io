null set

1. NOUN

/ nʌl sɛt / (AmE), / nʌl sɛt / (BrE)
a set that contains no elements; an empty set
a set with no elements (math)

Examples

In set theory, the null set is represented by the symbol ∅.
There are no prime numbers between 8 and 10, so that set is a null set.
If a group has no members who speak French, the group of French speakers is the null set.

Related Terms

  • Synonym: empty set - a set with no elements

Etymology

From Latin 'nullus' (none) + English 'set' (collection).
