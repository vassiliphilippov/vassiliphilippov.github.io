LISP compiler

1. NOUN

/ lɪsp kəmˈpaɪlər / (AmE), / lɪsp kəmˈpaɪlə / (BrE)
A computer program that translates programs written in the LISP programming language into executable machine code or another lower-level language.
compiler for the LISP programming language

Examples

The new LISP compiler significantly improved the performance of our AI application.
Many universities teach students how to build a simple LISP compiler as part of their curriculum.
Updating the LISP compiler allowed legacy code to run on modern hardware.

Related Terms

  • Synonym: compiler - a program that translates code into executable form

Etymology

LISP from 'LISt Processing'; compiler from Latin 'compilare' (to compile).
