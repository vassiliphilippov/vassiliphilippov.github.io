software system

1. NOUN

/ ˈsɔftˌwɛɹ ˈsɪstəm / (AmE), / ˈsɒftweə ˈsɪstəm / (BrE)
a collection of related programs, procedures, and documentation designed to operate together on a computer or network
set of related computer programs and procedures

Examples

Our company is developing a new software system to manage customer data more efficiently.
The hospital's software system ensures that medical records are kept secure and up to date.
Learning to operate the new software system only took the staff a few hours.

Synset Examples

the market for software is expected to expand
