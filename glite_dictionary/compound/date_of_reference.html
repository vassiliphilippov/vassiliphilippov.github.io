date of reference

1. NOUN

/ ˈdeɪt əv ˈrɛf(ə)rəns / (AmE), / ˈdeɪt əv ˈrɛf(ə)rəns / (BrE)
an arbitrarily fixed date used as the standard point in time relative to which astronomical information, such as the coordinates of a celestial body, is recorded
standard date used as astronomical reference

Examples

The star catalog listed all positions for the date of reference January 1, 2000.
Researchers used the same date of reference to compare orbital parameters accurately.
Before updating the charts, astronomers must note the date of reference for all recorded data.

Related Terms

  • Synonym: epoch - a reference point in time used for astronomical calculations
