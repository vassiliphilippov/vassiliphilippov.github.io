sexadecimal notation

1. NOUN

/ ˌsɛksəˈdɛsɪməl noʊˈteɪʃən / (AmE), / ˌsɛksəˈdɛsɪməl nəʊˈteɪʃən / (BrE)
a numeral system that uses sixteen distinct characters or symbols to represent numbers; also known as the base-16 system
base-16 numeral system

Examples

In some computing applications, sexadecimal notation is used to simplify the representation of binary numbers.
The color code #FF5733 is written in sexadecimal notation.
He found it easier to convert binary directly into sexadecimal notation for programming tasks.

Related Terms

  • Synonym: hexadecimal notation - base-16 numeral system using digits 0-9 and letters A-F
  • Antonym: decimal notation - base-10 numeral system
  • Degree Lower: octal notation - base-8 numeral system
  • Degree Lower: duodecimal notation - base-12 numeral system

Etymology

From Latin 'sex' (six) and 'decem' (ten), referring to sixteen.
