reproductive memory

1. NOUN

/ ˌriːprəˈdʌktɪv ˈmɛməri / (AmE), / ˌriːprəˈdʌktɪv ˈmɛməri / (BrE)
memory that recalls information by reproducing the original stimulus input as it was received, rather than reconstructing it.
memory reproducing original input

Examples

Eyewitness testimony often assumes reproductive memory, expecting precise recall of events.
In contrast to reconstructive memory, reproductive memory aims to produce an exact replica of earlier experiences.
Research suggests that reproductive memory is less common than previously believed, as memories are often reconstructed.

Related Terms

  • Antonym: constructive memory - memory involving reconstruction of events or information

Etymology

From Latin 're-' (again) + 'producere' (to bring forth) and 'memory' from Latin 'memoria'.
