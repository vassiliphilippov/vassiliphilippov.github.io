target language

1. NOUN

/ ˈtɑrɡɪt ˌlæŋɡwɪdʒ / (AmE), / ˈtɑːɡɪt ˌlaŋɡwɪdʒ / (BrE)
the language into which a text or speech is translated from another language
language into which something is translated

Examples

Fluency in the target language is essential for accurate translation.
While translating a novel, she carefully considered cultural differences in the target language.
The students practiced speaking in the target language during their language immersion program.

Related Terms

  • Antonym: source language - the language from which a text or speech is translated
Source: WordNet
WordNet ID: oewn-06294720-n
Synset POS: n

2. NOUN

/ ˈtɑrɡɪt ˌlæŋɡwɪdʒ / (AmE), / ˈtɑːɡɪt ˌlæŋɡwɪdʒ / (BrE)
a computer language into which a program or code written in another language is translated
computer language code is translated into

Examples

The compiler converts the instructions from the source language into the target language.
When building cross-platform applications, you must specify the desired target language.
Different programming tasks may require translating code into a different target language.

Related Terms

  • Antonym: source language - a computer language from which a program is translated
