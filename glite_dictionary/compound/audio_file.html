audio file

1. NOUN

/ ˈɔdiˌoʊ faɪl / (AmE), / ˈɔːdiəʊ faɪl / (BrE)
a digital file that stores recorded sound in formats such as MP3, AAC, or CDDA
a computer file containing digitized sound

Examples

She downloaded the audio file to listen to the interview on her phone.
Please send me the audio file as an email attachment.
The podcast is available as an audio file on our website.

Related Terms

  • Synonym: sound file - a computer file containing sound or music
  • Antonym: video file - a computer file that stores video content
