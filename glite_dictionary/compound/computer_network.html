computer network

1. NOUN

/ kəmˈpjuːtər ˈnɛtwɜrk / (AmE), / kəmˈpjuːtə ˈnɛtwɜːk / (BrE)
A group of interconnected computers that can communicate and share resources with each other.
group of interconnected computers

Examples

The company upgraded its computer network to improve communication between departments.
All devices in a computer network can access shared files and printers.
A virus can spread quickly throughout a computer network if security is weak.

Related Terms

  • Synonym: network - an interconnected system
