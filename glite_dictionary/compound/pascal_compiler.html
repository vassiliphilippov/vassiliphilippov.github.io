Pascal compiler

1. NOUN

/ ˈpæskəl kəmˈpaɪlər / (AmE), / ˈpæskæl kəmˈpaɪlə / (BrE)
a computer program that translates code written in the Pascal programming language into machine code or another programming language
software for compiling Pascal code

Examples

The university developed its own Pascal compiler for teaching introductory programming courses.
Before running the program, you need to process your files with a Pascal compiler.
She contributed to the open-source Pascal compiler, improving its support for modern features.

Related Terms

  • Synonym: Pascal - a high-level programming language developed in the 1970s
  • Synonym: compiler - a program that translates code from a high-level programming language to machine code

Etymology

Pascal is named after the mathematician Blaise Pascal; compiler from Latin 'compilare' (to compile, collect).
