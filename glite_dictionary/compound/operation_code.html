operation code

1. NOUN

/ ˌɑːpəˈreɪʃən koʊd / (AmE), / ˌɒpəˈreɪʃən kəʊd / (BrE)
A part of a computer instruction that specifies the operation to be performed by the computer.
part of a computer instruction indicating the operation

Examples

The operation code determines which instruction the CPU will execute.
Each machine language instruction starts with an operation code followed by its operands.
A mismatch in the operation code can cause the program to crash.

Related Terms

  • Synonym: opcode - short form for 'operation code' in computing
