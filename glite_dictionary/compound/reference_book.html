reference book

1. NOUN

/ ˈrɛfərəns bʊk / (AmE), / ˈrɛfərəns bʊk / (BrE)
a book, such as a dictionary or encyclopedia, that is consulted for authoritative information or facts rather than read from beginning to end
book used for authoritative facts

Examples

I keep a reference book on my desk for quick access to important information.
The library has an extensive collection of reference books available for research.
A dictionary is a type of reference book you might use to check spelling or meaning.

Synset Examples

he contributed articles to the basic reference work on that topic

Related Terms

  • Synonym: reference work - a book intended to be consulted for information, not read cover to cover
