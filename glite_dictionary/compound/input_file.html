input file

1. NOUN

/ ˈɪnˌpʊt faɪl / (AmE), / ˈɪnpʊt faɪl / (BrE)
(computing) a computer file that contains data to be processed by a device or program.
file used as input for a program

Examples

Make sure the input file is in the correct folder before running the program.
The application reads data from the input file and generates a report.
If the input file is missing, the software will display an error message.

Related Terms

  • Antonym: output file - a computer file that receives data produced by a device or program
