data file

1. NOUN

/ ˈdeɪtə faɪl / (AmE), / ˈdeɪtə faɪl / (BrE)
A file containing a set of related records, either stored electronically or written, often used for storing information in computing.
file of related information or records

Examples

The IT department regularly backs up each data file to prevent loss.
After downloading the data file, you can import it into the program for analysis.
The researcher saved each participant’s information in a separate data file.

Related Terms

  • Degree Higher: database - an organized collection of data, usually electronically stored
  • Degree Higher: file - a collection of data or documents stored together
