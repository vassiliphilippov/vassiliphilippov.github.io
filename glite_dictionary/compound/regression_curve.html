regression curve

1. NOUN

/ rɪˈɡrɛʃən kɝv / (AmE), / rɪˈɡrɛʃən kɜːv / (BrE)
a smooth curve fitted to paired data in regression analysis to show the relationship between variables; for linear regression, this is a straight line
curve representing fitted trend in data analysis
medical

Examples

The regression curve revealed a clear upward trend in the data.
By fitting a regression curve to the scatter plot, the scientists identified the underlying pattern.
A simple linear regression curve is often represented as a straight line in statistics.

Related Terms

  • Synonym: trend line - a line indicating the general course or tendency of data points
  • Synonym: regression analysis - a statistical process for estimating relationships among variables
  • Degree Lower: scatter plot - a graph showing values for typically two variables for a set of data

Etymology

Formed from 'regression' (from Latin regressio 'a return') and 'curve' (from Latin curva 'a bend').
