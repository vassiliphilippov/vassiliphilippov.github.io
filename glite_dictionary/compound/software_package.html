software package

1. NOUN

/ ˈsɔf.twɛr ˈpæk.ɪdʒ / (AmE), / ˈsɒf.tweə ˈpæk.ɪdʒ / (BrE)
a set of related programs, usually sold together, that perform a specific function or set of functions on a computer
a set of related computer programs sold together

Examples

The company released a new software package that simplifies accounting tasks for small businesses.
Before purchasing the software package, make sure it is compatible with your operating system.
Many businesses rely on a standard software package for word processing and spreadsheets.

Related Terms

  • Synonym: application - a computer program designed for a particular purpose
  • Synonym: software suite - a collection of related computer programs sold as a unit

/ ˈsɒft.wɛr ˈpæk.ɪdʒ / (AmE), / ˈsɒft.weə ˈpæk.ɪdʒ / (BrE)
A collection of computer programs and related data or documentation delivered as a single unit to perform a specific function or set of functions on a computer system.
collection of programs and data for computers

Examples

The company released a new software package that streamlines data analysis for small businesses.
Installing an antivirus software package is essential to protect your computer from malware.
This software package includes both the core program and several useful add-ons.
