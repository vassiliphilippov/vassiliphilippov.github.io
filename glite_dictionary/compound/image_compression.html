image compression

1. NOUN

/ ˈɪmɪdʒ kəmˈprɛʃən / (AmE), / ˈɪmɪdʒ kəmˈprɛʃən / (BrE)
the process of reducing the file size of digital images for storage or transmission, typically by removing redundant or unnecessary data
reducing file size of digital images

Examples

Modern cameras often use image compression to save space on memory cards.
JPEG is a common format that relies on image compression techniques.
Without image compression, transferring large photo files over the internet would take much longer.

Related Terms

  • Synonym: data compression - the process of reducing the size of data files
