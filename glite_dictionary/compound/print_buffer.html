print buffer

1. NOUN

/ prɪnt ˈbʌfɚ / (AmE), / prɪnt ˈbʌfə / (BrE)
a memory area or device that temporarily stores data to be sent to a printer when it is ready
temporary storage for printing data

Examples

The computer sent the document to the print buffer before it started printing.
A large print buffer can help improve the speed of network printing.
When the print buffer is full, additional print jobs must wait until space is available.

Related Terms

  • Synonym: buffer - a memory area for temporarily storing data
