algorithmic rule

1. NOUN

/ ˌælɡəˈrɪðmɪk ruːl / (AmE), / ˌælɡəˈrɪðmɪk ruːl / (BrE)
a precise rule or set of rules that defines a step-by-step procedure for solving a problem, especially in mathematics or computing
step-by-step problem-solving rule

Examples

The computer program follows an algorithmic rule to process each data entry efficiently.
Teachers often use an algorithmic rule to show students how to solve quadratic equations.
In chess, one basic algorithmic rule determines the legal movements of each piece.

Related Terms

  • Synonym: algorithm - a process or set of rules to be followed in calculations or problem-solving

Etymology

Algorithmic from 'algorithm' (from the name of Persian mathematician al-Khwarizmi) combined with 'rule'.
