UNIX operating system

1. NOUN

/ ˈjuːnɪks ˌɑːpəˌreɪtɪŋ ˌsɪstəm / (AmE), / ˈjuːnɪks ˌɒpəˌreɪtɪŋ ˌsɪstəm / (BrE)
A powerful, multiuser operating system originally developed in the 1970s, widely used on servers and workstations.
multiuser computer operating system

Examples

The UNIX operating system is known for its stability and multitasking capabilities.
Many web servers around the world run on a UNIX operating system.
He learned to write shell scripts using the UNIX operating system at university.

Related Terms

  • Synonym: Linux - A family of open-source operating systems similar to UNIX

Etymology

UNIX: coined as a pun on Multics, the name of an earlier operating system; 'operating system' from Latin 'operari' (to work).
