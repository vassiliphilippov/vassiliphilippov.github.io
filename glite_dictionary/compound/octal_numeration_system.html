octal numeration system

1. NOUN

/ ˈɑktəl nuˌmɛrˈeɪʃən ˌsɪstəm / (AmE), / ˈɒktəl njʊˈmɜːrˈeɪʃən ˌsɪstəm / (BrE)
A numeral system that uses base eight, employing the digits 0 to 7.
numeral system with base eight

Examples

In some early computers, the octal numeration system was commonly used for programming machine instructions.
You can convert binary numbers to the octal numeration system for easier reading.
The octal numeration system is less common today but still appears in certain technical fields.

Related Terms

  • Synonym: binary numeration system - numeral system with base two
  • Synonym: decimal numeration system - numeral system with base ten
  • Synonym: hexadecimal numeration system - numeral system with base sixteen

Etymology

From 'octal' (from Latin 'octo', meaning eight) + 'numeration system'.
