free software

1. NOUN

/ fri ˈsɔftˌwɛr / (AmE), / friː ˈsɒftˌwɛə / (BrE)
software that can be used, modified, and distributed by anyone without restriction, typically with source code available
software that is free to use and modify

Examples

Many developers choose to use free software to avoid licensing fees.
The team contributed code to a popular free software project.
Using free software encourages collaboration and innovation in the tech community.

Synset Examples

The GNUtard community is passionate about free software.

Related Terms

  • Synonym: open-source software - software with source code available for modification and sharing

Etymology

From 'free' (meaning 'without cost' or 'liberty') and 'software'; became prominent in the 1980s with the Free Software Foundation.
