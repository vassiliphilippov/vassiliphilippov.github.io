parity bit

1. NOUN

/ ˈpɛrəti bɪt / (AmE), / ˈparɪti bɪt / (BrE)
A bit added to a group of binary digits to ensure that the total number of 1s is either even (even parity) or odd (odd parity), used for error detection in data transmission.
bit for data error detection

Examples

The system uses a parity bit to detect any errors that might occur during data transmission.
If the parity bit does not match the expected value, an error is signaled.
Adding a parity bit is one of the simplest methods for error checking in computer networks.

Related Terms

  • Synonym: checksum - a value used to verify the integrity of data
  • Synonym: parity - the quality of being even or odd (in computing, even/odd parity for error checking)

Etymology

From 'parity' (equality, correspondence), in this context referring to odd or even count, plus 'bit' (binary digit).
