cat box

1. NOUN

/ ˈkæt ˌbɑks / (AmE), / ˈkæt ˌbɒks / (BrE)
a container filled with litter for cats to use as a toilet indoors
container for cat excrement

Examples

She changes the cat box every morning to keep it clean.
If you have a new kitten, show it where the cat box is located.
The apartment smelled better after he emptied the cat box.

Related Terms

  • Synonym: litter box - a container with litter for a cat to use as a toilet
