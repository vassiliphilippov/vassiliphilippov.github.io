reduced instruction set computing

1. NOUN

/ rɪˈduːst ɪnˈstrʌk.ʃən sɛt kəmˈpjuː.tɪŋ / (AmE), / rɪˈdjuːst ɪnˈstrʌk.ʃən sɛt kəmˈpjuː.tɪŋ / (BrE)
(computer science) a type of computer architecture characterized by a relatively small and simple set of instructions that the processor can carry out.
computer architecture with a small instruction set

Examples

The new processor uses reduced instruction set computing to achieve faster speeds and greater efficiency.
Many modern smartphones rely on reduced instruction set computing architecture for better performance and lower power consumption.
Engineers discussed the benefits of reduced instruction set computing during the technology conference.

Related Terms

  • Synonym: RISC - abbreviation of reduced instruction set computing
  • Antonym: complex instruction set computing - computer architecture with a large and complex set of instructions
