markup language

1. NOUN

/ ˈmɑrkˌʌp ˈlæŋɡwɪdʒ / (AmE), / ˈmɑːkʌp ˈlaŋɡwɪdʒ / (BrE)
A computer language consisting of a set of symbols and rules used to annotate a document's structure, layout, or formatting.
computer language for document annotation

Examples

HTML is a widely used markup language for creating web pages.
Learning a markup language can help you format documents more efficiently.
XML is another popular markup language used for data storage and transfer.

Related Terms

  • Synonym: markup - symbols or codes inserted in text to indicate structure or formatting
  • Antonym: programming language - an artificial language for programming computers

Etymology

From 'markup' (editing annotations) and 'language' (system of communication).
