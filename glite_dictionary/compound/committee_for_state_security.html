Committee for State Security

1. PROPN

/ kəˈmɪɾi ˌfɔɹ steɪt sɪˈkjʊɹɪɾi / (AmE), / kəˈmɪti fə ˈsteɪt sɪˈkjʊərɪti / (BrE)
the former primary security and intelligence agency of the Soviet Union, responsible for state security, intelligence, and secret police operations
Soviet Union's former main security agency

Examples

The Committee for State Security played a central role in Soviet intelligence gathering during the Cold War.
After the Soviet Union dissolved, the Committee for State Security was replaced by a new agency.
Many former officers of the Committee for State Security went on to have prominent roles in Russian politics.

Related Terms

  • Synonym: KGB - acronym for Committee for State Security, Soviet security service
