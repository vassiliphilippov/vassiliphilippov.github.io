blunt file

1. NOUN

/ blʌnt faɪl / (AmE), / blʌnt faɪl / (BrE)
a type of metal file with parallel edges, used for smoothing or shaping materials
metal file with parallel edges

Examples

The machinist selected a blunt file to smooth the edges of the metal plate.
Using a blunt file, she carefully shaped the piece of wood.
You can achieve a flat surface with a blunt file thanks to its parallel sides.
