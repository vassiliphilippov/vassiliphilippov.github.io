wireless telegraph

1. NOUN

/ ˈwaɪərləs təˈlɛɡræf / (AmE), / ˈwaɪələs tɪˈlɛɡrɑːf / (BrE)
the transmission of telegraphic messages by radio signals, typically using Morse code
telegraphic messages sent by radio waves

Examples

The ship sent out an SOS message using its wireless telegraph.
Before telephones, news was sometimes transmitted via wireless telegraph across long distances.
The development of the wireless telegraph revolutionized maritime communication.

Related Terms

  • Synonym: radio telegraphy - telegraphic communication using radio waves

Etymology

wireless (without wires) + telegraph (from Greek 'tele' far + 'graphein' to write)
