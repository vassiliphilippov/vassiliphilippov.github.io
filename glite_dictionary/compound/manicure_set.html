manicure set

1. NOUN

/ ˈmænɪˌkjʊr sɛt / (AmE), / ˈmanɪkjʊə sɛt / (BrE)
a set of tools used for the care and grooming of fingernails and hands
set of nail care tools

Examples

She keeps her manicure set in a small velvet pouch in her drawer.
The manicure set includes nail clippers, a file, and cuticle scissors.
Before the party, he used his manicure set to tidy up his nails.
