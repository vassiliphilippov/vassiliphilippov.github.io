wood file

1. NOUN

/ wʊd faɪl / (AmE), / wʊd faɪl / (BrE)
a coarse file with sharp, pointed projections, used for shaping or smoothing wood surfaces
coarse tool for shaping wood

Examples

He used a wood file to smooth the edges of the wooden shelf.
After sawing the plank, a wood file helped to remove any splinters.
Every carpenter's toolbox contains at least one wood file for shaping projects.

Related Terms

  • Synonym: rasp - a coarse file with rough, sharp teeth, often used for shaping wood
