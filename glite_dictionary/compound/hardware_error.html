hardware error

1. NOUN

/ ˈhɑrdˌwɛr ˈɛrɚ / (AmE), / ˈhɑːdˌwɛə ˈɛrə / (BrE)
An error resulting from a malfunction of a physical component of a computer.
malfunction of computer hardware

Examples

The server crashed because of a hardware error in the memory module.
After running diagnostics, we found that a hardware error was causing the blue screen.
A sudden power surge led to a hardware error in several workstations.

Related Terms

  • Antonym: software error - an error resulting from a problem with the computer's software
