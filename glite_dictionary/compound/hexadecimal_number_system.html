hexadecimal number system

1. NOUN

/ ˌhɛksəˈdɛsɪm(ə)l ˈnʌmbər ˌsɪstəm / (AmE), / ˌhɛksəˈdɛsɪməl ˈnʌmbə ˌsɪstəm / (BrE)
A positional numeral system with a base of sixteen, using the digits 0–9 and the letters A–F to represent values.
number system with base 16

Examples

In computer science, the hexadecimal number system is often used to represent memory addresses.
Colors in web design are frequently specified using the hexadecimal number system.
Each digit in the hexadecimal number system represents four binary digits, or bits.

Related Terms

  • Degree Lower: binary number system - numeral system with base 2, uses only digits 0 and 1
  • Degree Lower: decimal number system - numeral system with base 10, uses digits 0 to 9

Etymology

From Greek 'hex' (six) + 'deka' (ten) + Latin 'numerus' (number) + Latin 'systema' (system).
