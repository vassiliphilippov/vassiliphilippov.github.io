two-line great primer

1. NOUN

/ ˈtuː laɪn ɡreɪt ˈpraɪmər / (AmE), / ˈtuː laɪn ɡreɪt ˈprɪmə / (BrE)
A size of printing type measuring 36 points, slightly smaller than meridian.
36-point printing type size

Examples

The old book was printed using two-line great primer for its main headings.
Collectors of antique type specimens often seek rare two-line great primer blocks.
When setting the title page, the printer chose two-line great primer for emphasis.

Related Terms

  • Degree Higher: meridian - a size of printing type just larger than two-line great primer
  • Degree Lower: great primer - a printing type size that is half the size of two-line great primer

Etymology

From the practice in printing where 'great primer' refers to a type size, and 'two-line' indicates a type twice the size of the basic unit.
