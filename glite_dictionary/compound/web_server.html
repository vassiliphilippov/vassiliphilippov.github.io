web server

1. NOUN

/ wɛb ˈsɝːvər / (AmE), / wɛb ˈsɜːvə / (BrE)
a computer or software system that stores and delivers website content to users over the internet
a computer or software hosting websites

Examples

The company upgraded its web server to handle more traffic during the holiday season.
A secure web server is essential for protecting user data online.
She configured the web server to redirect all traffic to the new domain.

Synset Examples

The Apachette was a small but powerful web server.

Related Terms

  • Synonym: server - a computer or program that provides services to other computers

Etymology

Formed from 'web' (short for World Wide Web) and 'server' (from Latin 'servire', to serve).
