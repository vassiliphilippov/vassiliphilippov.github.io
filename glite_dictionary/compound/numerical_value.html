numerical value

1. NOUN

/ nuˈmɛr.ɪ.kəl ˈvæl.ju / (AmE), / njuːˈmɛr.ɪ.kəl ˈvæl.juː / (BrE)
the value of a number expressed quantitatively, regardless of its sign
a number's value regardless of sign

Examples

Please write down the numerical value of each measurement, regardless of whether it is positive or negative.
The calculator displayed the numerical value of the complex equation.
You must consider only the numerical value when rounding decimals.

Related Terms

  • Synonym: absolute value - the non-negative value of a real number without regard to its sign
