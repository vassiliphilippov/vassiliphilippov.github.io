chance variable

1. NOUN

/ tʃæns ˈværiəbl̩ / (AmE), / tʃɑːns ˈvɛəriəbl̩ / (BrE)
A variable quantity whose possible values are determined by chance, often used in statistics and probability theory.
variable quantity with random outcome

Examples

In this experiment, the outcome is determined by a chance variable.
Each chance variable represents a possible result of the process.
Statisticians use a chance variable to model random events.

Related Terms

  • Synonym: random variable - a variable whose values result from a random phenomenon
