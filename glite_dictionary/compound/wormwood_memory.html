wormwood memory

1. NOUN

/ ˈwɜrmwʊd ˈmɛməri / (AmE), / ˈwɜːmwʊd ˈmɛməri / (BrE)
A memory that is bitter or painful to recall.
bitter or painful memory

Examples

She tried to bury the wormwood memory of her failed relationship.
Each time he walked by the old house, a wormwood memory surfaced in his mind.
Writing about the experience helped her overcome the wormwood memory of that difficult night.

Synset Examples

She spoke of wormwood memories from her troubled past.

Related Terms

  • Synonym: painful memory - a memory that causes emotional pain or discomfort
  • Synonym: haunting memory - a memory that continues to disturb or trouble someone

Etymology

From 'wormwood', a bitter-tasting plant, used metaphorically for bitterness or sorrow.
