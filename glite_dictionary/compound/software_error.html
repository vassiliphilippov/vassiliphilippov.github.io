software error

1. NOUN

/ ˈsɑːf.wɛr ˈɛr.ər / (AmE), / ˈsɒft.wɛər ˈɛr.ər / (BrE)
An error in a computer program caused by faulty or incorrect code, leading to unintended or incorrect operation.
an error caused by faulty software code

Examples

The update failed to install because of a software error in the new version.
After hours of debugging, she finally located the software error causing the crash.
Many companies release patches to fix software errors discovered by users.

Related Terms

  • Synonym: bug - an error or flaw in a computer program
  • Synonym: glitch - a minor malfunction or error in software or hardware
  • Antonym: hardware error - an error resulting from a malfunction in computer hardware
