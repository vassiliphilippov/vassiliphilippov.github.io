reusable routine

1. NOUN

/ riˈjuːzəbl̩ ruːˈtiːn / (AmE), / riˈjuːzəbl ruːˈtiːn / (BrE)
a routine in computing that can be loaded once and used repeatedly without rewriting or reloading
routine used multiple times in computing

Examples

Writing a reusable routine saves time and reduces errors in large software projects.
The library contains several reusable routines for data processing.
Developers should document each reusable routine to make integration easier.

Related Terms

  • Synonym: subroutine - a set of instructions designed to perform a frequently used operation
  • Synonym: function - a named section of a program that performs a specific task
