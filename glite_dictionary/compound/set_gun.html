set gun

1. NOUN

/ sɛt ɡʌn / (AmE), / sɛt ɡʌn / (BrE)
A device consisting of a gun rigged to fire automatically when a tripwire or similar trigger is disturbed, typically intended as a trap for intruders.
gun triggered by a tripwire or trap

Examples

The soldiers discovered a set gun hidden near the entrance to the camp.
During the raid, one of the intruders triggered a set gun that had been carefully concealed.
Laws in many places explicitly prohibit the use of a set gun for property defense.
