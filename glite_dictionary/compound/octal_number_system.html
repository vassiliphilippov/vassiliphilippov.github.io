octal number system

1. NOUN

/ ˈɑktəl ˈnʌmbɚ ˌsɪstəm / (AmE), / ˈɒktəl ˈnʌmbə ˌsɪstəm / (BrE)
A system of numeration using base eight, with digits from 0 to 7.
numeral system with base 8

Examples

Early computers often used the octal number system to simplify binary data representation.
In the octal number system, the number nine does not exist.
Learning the octal number system can be helpful for understanding low-level programming.

Related Terms

  • Antonym: decimal number system - numeration system with base 10
  • Antonym: binary number system - system of numeration with base 2
  • Antonym: hexadecimal number system - system of numeration with base 16

Etymology

From Latin 'octo' (eight); related to numeral systems terminology.
