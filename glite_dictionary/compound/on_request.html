on request

1. ADV

/ ɑn rɪˈkwɛst / (AmE), / ɒn rɪˈkwɛst / (BrE)
if someone asks for it; upon being requested.
if or when requested

Examples

The brochure is available on request from the front desk.
Special dietary options can be provided on request for guests with allergies.
Further details will be supplied on request.

Synset Examples

advice was free for the asking

Related Terms

  • Synonym: upon request - if or when requested
