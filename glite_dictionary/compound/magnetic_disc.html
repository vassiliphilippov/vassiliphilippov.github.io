magnetic disc

1. NOUN

/ mæɡˈnɛtɪk dɪsk / (AmE), / mæɡˈnɛtɪk dɪsk / (BrE)
A memory device consisting of a flat disk coated with magnetic material, used to store information in computers.
computer memory device with magnetic coating

Examples

Early computers used magnetic discs to store operating systems and application files.
The data on the magnetic disc can be retrieved even after the power is turned off.
Engineers examined the magnetic disc for signs of physical damage after the system crashed.

Related Terms

  • Synonym: hard disk - a magnetic disk used as a computer data storage device
  • Synonym: floppy disk - a type of magnetic disk used for data storage
  • Antonym: optical disc - a data storage device that uses lasers to read and write data
