Java olives

1. NOUN

/ ˈdʒɑːvə ˈɑlɪvz / (AmE), / ˈdʒɑːvə ˈɒlɪvz / (BrE)
a large tree of the Old World tropics with foul-smelling orange-red flowers and red pods containing oil-rich seeds, sometimes used as food
tropical tree with edible oil-rich seeds

Examples

The seeds of Java olives are sometimes harvested and pressed for their oil.
In some regions, Java olives are valued for their resilience in tropical climates.
Researchers study the nutritional properties of Java olives for potential food applications.
