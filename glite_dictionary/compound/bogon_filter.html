bogon filter

1. NOUN

/ ˈboʊɡɒn ˈfɪltɚ / (AmE), / ˈbəʊɡɒn ˈfɪltə / (BrE)
a network filter designed to block traffic from invalid or unassigned IP addresses
network filter for invalid IPs

Examples

The company's new bogon filter helped prevent attacks from unallocated IP ranges.
After activating the bogon filter, we saw a significant drop in suspicious network traffic.
IT staff regularly update the bogon filter to maintain the network's security.

Synset Examples

Her IT skills helped set up the bogon filter quickly.
We enabled the bogon filter to enhance network security.
The router's bogon filter blocked unwanted traffic.

Related Terms

  • Synonym: firewall - a system that controls incoming and outgoing network traffic
  • Synonym: access control list - a list of rules used to control network traffic

Etymology

From 'bogon', a network slang term for bogus or invalid IP addresses, and 'filter.'
