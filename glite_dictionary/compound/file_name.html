file name

1. NOUN

/ ˈfaɪl neɪm / (AmE), / ˈfaɪl neɪm / (BrE)
the name assigned to a computer file to identify it, often including an extension that indicates its type
name assigned to a computer file

Examples

Always save your work frequently and choose a descriptive file name for each document.
The system wouldn't let me use special characters in the file name.
She accidentally deleted the wrong file because the file name was too similar to another.

Related Terms

  • Synonym: filename - alternative spelling of 'file name'

2. NOUN

/ ˈfaɪl neɪm / (AmE), / ˈfaɪl neɪm / (BrE)
(computing) a name assigned to a computer file to distinguish it from others, often including an extension indicating its type.
name assigned to a computer file

Examples

Please save the document under a new file name to avoid overwriting your work.
Make sure your file name doesn't contain any special characters.
The program will prompt you for a file name when exporting the image.

Related Terms

  • Synonym: filename - alternative, especially single-word spelling of 'file name'
