anonymous file transfer protocol

1. NOUN

/ əˈnɑnɪməs faɪl trænsˈfɚ ˈproʊtəkɑl / (AmE), / əˈnɒnɪməs faɪl ˈtrænsfə ˈprəʊtəkɒl / (BrE)
a method for making files available on a computer network that allows users to access and download files without providing a password, typically by logging in as 'guest' or 'anonymous'
public file-sharing network protocol

Examples

Many open source projects used to distribute their code via anonymous file transfer protocol servers.
Researchers often downloaded datasets using anonymous file transfer protocol before newer cloud solutions became popular.
The company limited access to confidential files and only allowed software updates through anonymous file transfer protocol sites.
