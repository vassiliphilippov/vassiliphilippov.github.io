return ticket

1. NOUN

/ rɪˈtɝn ˈtɪkɪt / (AmE), / rɪˈtɜːn ˈtɪkɪt / (BrE)
a ticket that allows you to travel to a place and back, usually using the same route
ticket for both outward and return journeys
british

Examples

I bought a return ticket to London for the weekend.
It's usually cheaper to purchase a return ticket rather than two singles.
Do you want a single or a return ticket?

Related Terms

  • Synonym: round-trip ticket - a ticket that allows travel to a place and back, chiefly US English
