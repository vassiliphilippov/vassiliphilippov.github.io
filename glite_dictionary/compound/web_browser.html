web browser

1. NOUN

/ wɛb ˈbraʊzɚ / (AmE), / wɛb ˈbraʊzə / (BrE)
a software application used to locate, retrieve, and display content on the World Wide Web, including web pages written in HTML.
software for viewing web pages

Examples

I prefer to use a different web browser because it loads pages faster.
The first web browser was created in the early 1990s.
Make sure your web browser is updated to protect against security threats.

Related Terms

  • Synonym: browser - a software application for accessing web pages
