non-dedicated file server

1. NOUN

/ nɑn ˌdɛdəˈkeɪtɪd faɪl ˈsɝvər / (AmE), / nɒn ˌdɛdɪˈkeɪtɪd faɪl ˈsɜːvə / (BrE)
a computer that functions as both a file server and a workstation at the same time, instead of being used exclusively for file serving
computer used as both file server and workstation

Examples

A non-dedicated file server allows users to work locally while sharing files over the network.
In small offices, a non-dedicated file server can reduce hardware costs by doubling as a workstation.
Unlike a dedicated model, a non-dedicated file server doesn't require a separate machine solely for file sharing.

Related Terms

  • Antonym: dedicated file server - a computer used solely for file serving, not for local work
