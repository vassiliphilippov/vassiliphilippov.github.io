job control

1. NOUN

/ dʒɑb kənˈtroʊl / (AmE), / dʒɒb kənˈtrəʊl / (BrE)
a computer program responsible for preparing each job to be run, typically in a batch processing system.
program preparing jobs to run (computing)

Examples

The job control program ensures that each batch job starts and finishes correctly.
After the job control routine was updated, the system became more efficient.
If there's an error in job control, scheduled tasks might not execute as expected.
