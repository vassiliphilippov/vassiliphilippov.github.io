floating-point operation

1. NOUN

/ ˈfloʊtɪŋ pɔɪnt ˌɑpəˈreɪʃən / (AmE), / ˈfləʊtɪŋ pɔɪnt ˌɒpəˈreɪʃən / (BrE)
an arithmetic calculation carried out using floating-point numbers, often by a computer
arithmetic operation with floating-point numbers

Examples

A modern graphics card can execute billions of floating-point operations per second.
Accurate floating-point operations are essential for scientific computing applications.
Optimizing floating-point operations can greatly improve a program's performance.

Synset Examples

this computer can perform a million flops per second

Related Terms

  • Antonym: fixed-point operation - arithmetic operation performed using fixed-point numbers
  • Synonym: floating-point number - a number represented in floating-point notation
