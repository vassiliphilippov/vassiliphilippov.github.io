number system

1. NOUN

/ ˈnʌmbɚ ˌsɪstəm / (AmE), / ˈnʌmbə ˌsɪstəm / (BrE)
a system or notation for representing numbers, such as the decimal or binary system
system for representing numbers

Examples

The decimal number system uses the digits 0 through 9 to represent values.
In computing, the binary number system is essential for processing information.
Roman numerals are an example of an ancient number system.

Related Terms

  • Synonym: numeral system - a synonym for 'number system'; any system for representing numbers
