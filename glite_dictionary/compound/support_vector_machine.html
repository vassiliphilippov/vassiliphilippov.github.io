support vector machine

1. NOUN

/ səˈpɔrt ˈvɛktɚ məˈʃin / (AmE), / səˈpɔːt ˈvɛktə məˈʃiːn / (BrE)
a supervised learning algorithm used for classification and regression tasks, which works by finding the optimal separating hyperplane in a high-dimensional space
machine learning classification algorithm

Examples

The support vector machine outperformed other models in the accuracy test.
Many researchers use a support vector machine for image recognition tasks.
She implemented a support vector machine to classify medical data.

Synset Examples

The SVM algorithm has been widely applied in the biological and other sciences.

Related Terms

  • Synonym: SVM - abbreviation for support vector machine

Etymology

From the terms 'support vector' (referring to the vectors that define the hyperplane) and 'machine' (denoting an algorithmic process) in machine learning.
