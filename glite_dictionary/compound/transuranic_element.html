transuranic element

1. NOUN

/ ˌtrænz.jʊˈrænɪk ˈɛl.ə.mənt / (AmE), / ˌtranz.jʊˈrænɪk ˈɛl.ɪ.mənt / (BrE)
any chemical element with an atomic number greater than 92 (the atomic number of uranium); all such elements are radioactive
element with atomic number greater than 92

Examples

Plutonium is a well-known transuranic element used as fuel in nuclear reactors.
Scientists are continuing to synthesize new transuranic elements in particle accelerators.
All transuranic elements are radioactive and do not occur naturally on Earth.

Related Terms

  • Degree Lower: uranium - a radioactive chemical element with atomic number 92

Etymology

From Latin trans- (across, beyond) + uranium (the element uranium).
