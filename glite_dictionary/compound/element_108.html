element 108

a radioactive transuranic chemical element with atomic number 108, known as hassium
radioactive element, atomic number 108

Examples

Scientists first synthesized element 108 in 1984 at the GSI Helmholtz Centre for Heavy Ion Research.
Due to its instability, element 108 has only been produced in small amounts in laboratories.
The official name for element 108 is hassium, derived from the German state of Hesse.
