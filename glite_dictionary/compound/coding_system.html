coding system

1. NOUN

/ ˈkoʊdɪŋ ˌsɪstəm / (AmE), / ˈkəʊdɪŋ ˌsɪstəm / (BrE)
a set of rules or signals used to represent letters, numbers, or data for transmitting messages or information
system using signals or codes to represent data

Examples

The hospital uses a coding system to keep patient information secure.
Morse code is a famous coding system used for long-distance communication.
An efficient coding system is essential in digital data transmission.

Related Terms

  • Synonym: code - a system of words, letters, figures, or symbols used to represent others, especially for secrecy
