cheat code

1. NOUN

/ ˈtʃit ˌkoʊd / (AmE), / ˈtʃiːt ˌkəʊd / (BrE)
a sequence of commands or inputs used to gain an unfair advantage or unlock special features in a video game
special commands for advantages in games

Examples

He entered a cheat code to unlock all the characters in the game.
Some players rely on a cheat code when levels seem impossible to beat.
The magazine published several new cheat codes for the latest video games.

Synset Examples

He entered a cheat code for unlimited lives.
Using a cheat code makes the game too easy.

Related Terms

  • Synonym: exploit - an act of taking advantage of a flaw in software, especially in games
