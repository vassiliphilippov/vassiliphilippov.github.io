living-room set

1. NOUN

/ ˈlɪvɪŋ ruːm sɛt / (AmE), / ˈlɪvɪŋ rʊm sɛt / (BrE)
a matching set of furniture, typically including a sofa and chairs, designed for use in a living room
matching living room furniture set

Examples

We bought a new living-room set for our apartment last weekend.
The living-room set includes a sofa, two armchairs, and a coffee table.
Their old living-room set was worn out, so they decided to replace it.
