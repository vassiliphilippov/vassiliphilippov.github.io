operating system

1. NOUN

/ ˈɑpəˌreɪtɪŋ ˈsɪstəm / (AmE), / ˈɒpəreɪtɪŋ ˈsɪstəm / (BrE)
A set of software that manages computer hardware and software resources and provides common services for computer programs.
software managing computer operations

Examples

The new phone comes with an updated operating system that offers better security features.
Linux is a popular open-source operating system used by developers around the world.
Your computer may run slower if the operating system is outdated or corrupt.

Related Terms

  • Synonym: software - programs and data used by computers
  • Antonym: hardware - the physical components of a computer
