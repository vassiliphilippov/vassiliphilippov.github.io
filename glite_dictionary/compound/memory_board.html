memory board

1. NOUN

/ ˈmɛməri bɔrd / (AmE), / ˈmɛməri bɔːd / (BrE)
a circuit board containing electronic memory used for storing data or programs in a computer or electronic device
circuit board with electronic memory

Examples

After upgrading the memory board, my computer ran much faster.
Technicians replaced the faulty memory board to restore the server's performance.
Each memory board in the system can hold up to 16 gigabytes of RAM.

Synset Examples

a memory and the CPU form the central part of a computer to which peripherals are attached

Related Terms

  • Synonym: RAM - random access memory used for short-term data storage in computers
