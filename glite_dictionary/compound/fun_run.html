fun run

1. NOUN

/ ˈfʌn ˌrʌn / (AmE), / ˈfʌn ˌrʌn / (BrE)
an organized footrace held mainly for enjoyment or to raise money for charity, rather than for serious competition
non-competitive charity race

Examples

Thousands of people participated in the annual fun run to support local charities.
I signed up for a fun run with my coworkers next weekend.
The fun run attracted families and children of all ages.

Related Terms

  • Synonym: race - a competition of speed, as in running
