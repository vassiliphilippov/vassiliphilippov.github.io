eyelet buttonhole machine

1. NOUN

/ ˈaɪ.lət ˈbʌt.nˌhoʊl məˈʃin / (AmE), / ˈaɪ.lət ˈbʌt.nəʊl məˈʃiːn / (BrE)
a sewing machine designed to make buttonholes, including round (eyelet) types, with or without reinforced gimp thread
sewing machine for making buttonholes

Examples

The factory installed a new eyelet buttonhole machine to speed up shirt production.
Tailors prefer using an eyelet buttonhole machine for durable and neat buttonholes.
Modern eyelet buttonhole machines can sew a variety of buttonhole shapes efficiently.

  • Synonym: buttonhole machine - a sewing machine for making buttonholes
