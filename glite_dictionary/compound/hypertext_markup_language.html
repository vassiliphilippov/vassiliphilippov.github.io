hypertext markup language

1. NOUN

/ ˈhaɪ.pɚ.tɛkst ˈmɑːrk.ʌp ˈlæŋ.ɡwɪdʒ / (AmE), / ˈhaɪ.pə.tɛkst ˈmɑːk.ʌp ˈlæŋ.ɡwɪdʒ / (BrE)
the standard system of tags and rules used for creating and structuring pages on the World Wide Web
standard web page markup language

Examples

All modern websites are built using hypertext markup language as the foundation for their pages.
She learned to create her first webpage by studying hypertext markup language in an online course.
The browser interprets hypertext markup language to display images, links, and styled text.

Related Terms

  • Synonym: HTML - abbreviation for hypertext markup language

Etymology

From 'hypertext' + 'markup language'; first developed by Tim Berners-Lee in 1990.
