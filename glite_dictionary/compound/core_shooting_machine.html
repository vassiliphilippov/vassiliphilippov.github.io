core shooting machine

1. NOUN

/ kɔr ˈʃuːtɪŋ məˈʃin / (AmE), / kɔː ˈʃuːtɪŋ məˈʃiːn / (BrE)
A machine used in foundries for making sand cores by shooting core sand into a core box under pressure.
machine for making sand cores in foundries

Examples

The foundry installed a new core shooting machine to increase production efficiency.
A skilled operator is required to run the core shooting machine safely.
The maintenance team performed routine checks on the core shooting machine last week.
