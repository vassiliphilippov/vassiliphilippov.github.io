serial operation

1. NOUN

/ ˈsɪriəl ˌɑpəˈreɪʃən / (AmE), / ˈsɪəriəl ˌɒpəˈreɪʃən / (BrE)
the sequential execution of processes or tasks, with each operation starting only after the previous one has finished
sequential task execution

Examples

This computer supports both serial operation and parallel processing, depending on the task requirements.
In a serial operation, each step must finish before the next one can begin.
The manufacturing line used serial operation to assemble products in a set order.

Related Terms

  • Antonym: parallel operation - the simultaneous execution of multiple operations
