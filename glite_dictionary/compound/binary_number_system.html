binary number system

1. NOUN

/ ˈbaɪˌnɛri ˈnʌmbər ˈsɪstəm / (AmE), / ˈbaɪnəri ˈnʌmbə ˈsɪstəm / (BrE)
A system of numeration using only the digits 0 and 1, with a base of two.
numeration system using base two (0 and 1)

Examples

Computers process information using the binary number system, which relies on only two digits: 0 and 1.
Early digital circuits were designed to operate on the principles of the binary number system.
Learning how to convert decimal numbers to the binary number system is an essential skill in computer science.

Related Terms

  • Antonym: decimal system - a system of numeration with base ten

Etymology

From Latin 'binary' (consisting of two) + 'number system'.
