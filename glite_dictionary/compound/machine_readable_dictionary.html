machine readable dictionary

1. NOUN

/ məˈʃin ˌridəbəl dɪkˈʃəˌnɛri / (AmE), / məˈʃiːn ˌriːdəbl dɪkˈʃən(ə)ri / (BrE)
A dictionary in a format that can be read and processed by a computer, typically organized alphabetically.
dictionary in computer-readable format

Examples

Many language researchers rely on a machine readable dictionary to analyze linguistic patterns automatically.
The company developed a machine readable dictionary to improve its text processing software.
A machine readable dictionary stores lexical information in a structured digital format for computational use.
