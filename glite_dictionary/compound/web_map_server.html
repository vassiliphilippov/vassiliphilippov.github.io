Web Map Server

1. NOUN

/ wɛb mæp ˈsɝːvɚ / (AmE), / wɛb mæp ˈsɜːvə / (BrE)
A computer program or service that generates maps dynamically from spatially referenced data and makes them available over the internet.
program/service generating web-based maps

Examples

Many online mapping applications rely on a Web Map Server to deliver up-to-date geographic information.
The organization uses an open-source Web Map Server to support interactive mapping on their website.
A Web Map Server can combine multiple datasets to create customized map views for different users.

Related Terms

  • Synonym: map server - service that generates maps from data for web access
