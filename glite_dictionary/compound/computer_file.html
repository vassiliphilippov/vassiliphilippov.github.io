computer file

1. NOUN

/ kəmˈpjuːtər faɪl / (AmE), / kəmˈpjuːtə faɪl / (BrE)
a collection of data, text, or other information stored on a computer and identified by a unique name
digital data stored on a computer

Examples

I accidentally deleted the computer file containing my project report.
Before you shut down the system, make sure to save each computer file.
He transferred the computer file from his laptop to the office network.

Related Terms

  • Synonym: file - a collection of information stored electronically or on paper
