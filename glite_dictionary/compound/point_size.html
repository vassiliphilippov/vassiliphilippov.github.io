point size

1. NOUN

/ pɔɪnt saɪz / (AmE), / pɔɪnt sʌɪz / (BrE)
the height of characters in a typeface, measured in points from the lowest descender to the highest ascender
typographic measurement of font height

Examples

Be sure to select a point size that is easy to read for your audience.
The designer increased the point size to make the text more prominent on the flyer.
Changing the point size of a font can dramatically affect the appearance of a document.

Synset Examples

When a point size of a font is specified, the font is scaled so that its em square has a side length of that particular length in points.

Related Terms

  • Synonym: font size - the overall size of a set of printed characters
