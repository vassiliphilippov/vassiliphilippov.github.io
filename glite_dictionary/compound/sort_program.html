sort program

1. NOUN

/ sɔrt ˈproʊɡræm / (AmE), / sɔːt ˈprəʊɡræm / (BrE)
a computer utility that arranges data items in a specific order, typically alphabetically or numerically
utility that sorts data

Examples

The sort program quickly organized the customer records alphabetically.
Many operating systems include a built-in sort program for handling text files.
You can use a sort program to arrange data in ascending or descending order.

Related Terms

  • Synonym: sort - to arrange items in a particular sequence
