algebraic language

1. NOUN

/ ˌældʒəˈbreɪɪk ˈlæŋɡwɪdʒ / (AmE), / ˌaldʒɪˈbreɪɪk ˈlaŋɡwɪdʒ / (BrE)
a programming or algorithmic language in which instructions are written using expressions similar to those found in algebra
programming language resembling algebraic expressions

Examples

FORTRAN is an example of an algebraic language used for scientific computing.
In the 1960s, developers preferred algebraic languages for mathematical programming.
Programs written in an algebraic language are more readable for people with a background in mathematics.

Related Terms

  • Synonym: programming language - a formal language for giving instructions to a computer
  • Antonym: assembly language - a low-level programming language closely related to machine code

Etymology

Formed from 'algebraic' (from Arabic 'al-jabr', reunion of broken parts) and 'language', denoting a formal system resembling algebra.
