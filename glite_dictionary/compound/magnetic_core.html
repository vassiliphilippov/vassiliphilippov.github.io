magnetic core

1. NOUN

/ məɡˈnɛtɪk kɔr / (AmE), / maɡˈnɛtɪk kɔː / (BrE)
(computer science) a small ferrite ring formerly used in computer memory to store one bit of data, replaced by semiconductor memories.
tiny ferrite memory element in early computers

Examples

The computer museum has an old memory board filled with magnetic cores.
Before integrated circuits, magnetic core memory was standard in early mainframe computers.
Technicians used to repair magnetic core modules by hand, threading tiny wires through each ring.

Synset Examples

each core has three wires passing through it, providing the means to select and detect the contents of each bit
