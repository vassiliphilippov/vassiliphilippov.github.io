reusable program

1. NOUN

/ riˈjuːzəbl ˈproʊɡræm / (AmE), / riˈjuːzəbl ˈprəʊɡram / (BrE)
a computer program designed to be loaded once and executed multiple times without reloading
program that can be repeatedly executed

Examples

The team developed a reusable program to automate the data analysis process for multiple projects.
By using a reusable program, the company saved time on routine tasks.
Our software library includes several reusable programs for common engineering calculations.

Related Terms

  • Synonym: reusable - able to be used again
