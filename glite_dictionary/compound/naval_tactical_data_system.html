naval tactical data system

1. NOUN

/ ˈneɪvəl ˈtæk.tɪ.kəl ˈdeɪ.t̬ə ˈsɪs.təm / (AmE), / ˈneɪ.vəl ˈtæk.tɪ.kəl ˈdeɪ.tə ˈsɪs.təm / (BrE)
A shipboard system used for collecting, processing, and displaying tactical information to assist naval operations.
shipboard tactical data collection system

Examples

The naval tactical data system allowed the crew to track enemy ships in real time.
Upgrading the naval tactical data system significantly improved the ship's situational awareness.
Effective communication between vessels depends on a well-functioning naval tactical data system.
