dining room set

1. NOUN

/ ˈdaɪnɪŋ ruːm sɛt / (AmE), / ˈdaɪnɪŋ ruːm set / (BrE)
a matching set of furniture designed for use in a dining room, typically including a table and chairs
set of dining room furniture

Examples

We bought a beautiful dining room set for the new house.
She polished the antique dining room set before the guests arrived.
The store offers a wide selection of modern dining room sets at affordable prices.

Related Terms

  • Synonym: dining set - set of furniture for dining, often used interchangeably
