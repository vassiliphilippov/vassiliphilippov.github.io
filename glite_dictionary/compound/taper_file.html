taper file

1. NOUN

/ ˈteɪpɚ faɪl / (AmE), / ˈteɪpə faɪl / (BrE)
a hand tool with edges that gradually narrow, used for shaping and smoothing materials
a file with narrowing edges

Examples

The machinist used a taper file to smooth the inside edge of the metal pipe.
Always select the right size of taper file for fine, precise work.
My toolkit includes both flat files and a taper file for different tasks.

Related Terms

  • Synonym: file - a hand tool used to smooth or shape materials
