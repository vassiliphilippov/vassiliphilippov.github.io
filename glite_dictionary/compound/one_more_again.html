one more again

1. X

/ wʌn mɔr əˈɡɛn / (AmE), / wʌn mɔːr əˈɡɛn / (BrE)
(informal) Used to request that something be repeated or tried another time.
request to repeat or try again (informal)
informal

Examples

Can you say that one more again so I can write it down correctly?
She asked him to show her how to do it one more again.
If you didn't understand, just read the instructions one more again.

Synset Examples

Let's try that one more again.
Can you explain that one more again?
