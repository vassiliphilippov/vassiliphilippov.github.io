checking program

1. NOUN

/ ˈtʃɛkɪŋ ˌproʊɡræm / (AmE), / ˈtʃɛkɪŋ ˌprəʊɡræm / (BrE)
A computer program that examines other programs for syntax errors.
computer program for checking syntax errors

Examples

The developer used a checking program to identify errors in her code before compiling.
A good checking program can save hours of debugging by pointing out mistakes early.
Most modern IDEs include a built-in checking program for real-time syntax checking.

Related Terms

  • Synonym: syntax checker - a program that checks code for syntactic correctness
