compiling program

1. NOUN

/ kəmˈpaɪlɪŋ ˈproʊɡræm / (AmE), / kəmˈpaɪlɪŋ ˈprəʊɡræm / (BrE)
(computing) a computer program that translates code from a high-level programming language into assembly language or machine code.
program that translates computer code

Examples

The compiling program checks for errors in your code before producing an executable file.
She downloaded a new compiling program to help her learn C++.
Each compiling program is designed to support specific programming languages.

Related Terms

  • Synonym: compiler - program that translates source code into machine code
