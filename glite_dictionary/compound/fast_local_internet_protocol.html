fast local internet protocol

1. NOUN

/ fæst ˈloʊkəl ˈɪntərˌnɛt proʊˈtɔkəl / (AmE), / fɑːst ˈləʊkəl ˈɪntənɛt prəˈtəʊkəl / (BrE)
a network protocol suite designed to provide secure, transparent communication and network management, typically known as FLIP
network protocol suite (FLIP) for secure communication

Examples

The fast local internet protocol was designed with security and transparency as top priorities.
Many distributed systems have experimented with using fast local internet protocol for group communication.
Proper implementation of fast local internet protocol can enhance network management and secure message delivery.

Synset Examples

FLIP is a connectionless protocol designed to support transparency, group communication, secure communication and easy network management.

Etymology

Acronym for fast local internet protocol, developed for distributed operating systems.
