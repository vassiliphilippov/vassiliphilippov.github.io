anonymous FTP

1. NOUN

/ əˈnɑːnɪməs ˌɛf ˌti ˈpi / (AmE), / əˈnɒnɪməs ˌef tiː ˈpiː / (BrE)
a method of accessing files on an FTP server without needing a personalized login, typically by logging in as 'guest' or 'anonymous'
file access on FTP server without login

Examples

Many software developers use anonymous FTP to distribute their latest programs.
The university library provides anonymous FTP access to its digital archives.
In the early days of the Internet, anonymous FTP was a primary method for sharing files.

Related Terms

  • Synonym: FTP - File Transfer Protocol, a standard for transferring files over a network

Etymology

From 'anonymous' (without a name) and 'FTP', acronym for File Transfer Protocol.
