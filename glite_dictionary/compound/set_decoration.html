set decoration

1. NOUN

/ sɛt ˌdɛkəˈreɪʃən / (AmE), / sɛt ˌdɛkəˈreɪʃən / (BrE)
An object or arrangement used to decorate the set of a theatrical or film production, contributing to the visual environment.
decoration used in stage or film set

Examples

The set decoration gave the living room in the play a warm and realistic feel.
She was responsible for all the set decoration in the period drama.
Awards for outstanding set decoration are often given at major film ceremonies.

Related Terms

  • Synonym: scenery - the painted backdrops and physical constructions on a stage set
  • Synonym: prop - an object used by actors during a performance
