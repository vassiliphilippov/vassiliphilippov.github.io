typewriter font

1. NOUN

/ ˈtaɪpˌraɪtər fɑnt / (AmE), / ˈtaɪpˌraɪtə fɒnt / (BrE)
A typeface in which each character occupies the same amount of horizontal space, similar to the output of a traditional typewriter.
monospaced typeface, like a typewriter

Examples

The report must be typed in a typewriter font for consistency.
Many computer programmers prefer to write code using a typewriter font.
The document looked dated because it was printed in a typewriter font.

Related Terms

  • Synonym: monospaced font - font in which all characters have the same width
