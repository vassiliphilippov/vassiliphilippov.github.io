dump routine

1. NOUN

/ dʌmp ruːˈtin / (AmE), / dʌmp ruːˈtiːn / (BrE)
a computer program or subroutine that transfers data from internal memory to an external storage medium, such as a file or disk.
routine transferring data to external storage

Examples

The system administrator ran a dump routine to save the memory contents to a file for analysis.
Before shutting down the server, please execute the dump routine to preserve the current session data.
The dump routine ensures that critical logs are written to disk before any potential hardware failure.

Related Terms

  • Synonym: backup routine - program that creates a copy of data for backup
  • Antonym: restore routine - routine used to retrieve stored data back into memory

Etymology

From 'dump' (verb, meaning to transfer data) + 'routine' (sequence of instructions in computing).
