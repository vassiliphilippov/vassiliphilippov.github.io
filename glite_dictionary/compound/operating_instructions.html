operating instructions

1. NOUN

/ ˈɑːpəˌreɪtɪŋ ɪnˈstrʌkʃənz / (AmE), / ˈɒpəˌreɪtɪŋ ɪnˈstrʌkʃənz / (BrE)
a document that provides information on how to install or use a machine, device, or system
guide on using a device or system

Examples

Please read the operating instructions carefully before setting up your new printer.
The operating instructions are included in the box along with the warranty card.
If you lose the operating instructions, you can download a copy from the manufacturer's website.

Related Terms

  • Synonym: manual - a book or booklet giving instructions, especially on how to use something
  • Synonym: user guide - a guide with instructions for users of a particular product or system
