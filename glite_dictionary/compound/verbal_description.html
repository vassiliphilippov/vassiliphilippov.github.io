verbal description

1. NOUN

/ ˈvɜrbəl dɪˌskrɪpʃən / (AmE), / ˈvɜːbəl dɪˈskrɪpʃən / (BrE)
a statement that represents or explains something using words rather than pictures or symbols
statement explaining something in words

Examples

The artist provided a verbal description of her painting for visitors who could not see it.
A clear verbal description can help others understand complex scientific concepts.
He struggled to give a verbal description of the strange sound he had heard.

Related Terms

  • Antonym: visual description - description made using images or visual representation
