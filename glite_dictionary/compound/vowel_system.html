vowel system

1. NOUN

/ ˈvaʊəl ˌsɪstəm / (AmE), / ˈvaʊəl ˌsɪstəm / (BrE)
the set and organization of vowel sounds used in a particular language
set of vowel sounds in a language

Examples

Arabic has a relatively simple vowel system compared to English.
The complexity of the French vowel system often challenges English speakers.
Phonologists study how each language organizes its vowel system.

Related Terms

  • Antonym: consonant system - the set and organization of consonant sounds used in a particular language
