allocation unit

1. NOUN

/ ˌæləˈkeɪʃən ˈjuːnɪt / (AmE), / ˌaləˈkeɪʃ(ə)n ˈjuːnɪt / (BrE)
a group of sectors on a storage disk reserved for the use of a single file, representing the smallest amount of disk space that can be allocated to a file
group of disk sectors reserved for a file

Examples

Each allocation unit on this hard drive is 4 kilobytes in size.
When a file is saved, it occupies one or more allocation units depending on its size.
Changing the allocation unit size can impact disk performance and storage efficiency.

Related Terms

  • Synonym: cluster - group of sectors managed as a unit on a disk
