Boolean operation

1. NOUN

/ ˈbuːliən ˌɑpəˈreɪʃən / (AmE), / ˈbuːliən ˌɒpəˈreɪʃən / (BrE)
an operation that follows the rules of Boolean algebra, where each input and the result are either true or false
operation using Boolean algebra rules

Examples

A basic Boolean operation like AND returns true only if both inputs are true.
Computer searches often use Boolean operations to narrow or broaden the results.
You can combine multiple conditions with a Boolean operation such as OR or NOT.

Related Terms

  • Synonym: logical operation - an operation based on logical values (true/false)
  • Antonym: arithmetic operation - an operation such as addition or multiplication on numbers

Etymology

Named after George Boole (1815–1864), English mathematician, whose algebra is the foundation of Boolean logic.
