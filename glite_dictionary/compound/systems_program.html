systems program

1. NOUN

/ ˈsɪstəmz ˌproʊɡræm / (AmE), / ˈsɪstəmz ˌprəʊɡram / (BrE)
A computer program, such as an operating system, compiler, or utility, that manages or controls various aspects of a computer's operation.
program managing computer operations

Examples

The operating system is a type of systems program that manages hardware resources.
Installing a new systems program often requires administrator privileges.
Compilers are one example of a systems program essential for software development.

Related Terms

  • Antonym: application program - A program designed to carry out a specific task for the user
