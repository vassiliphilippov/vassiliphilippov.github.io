information processing system

1. NOUN

/ ˌɪn.fɚˈmeɪ.ʃən ˈproʊ.sɛs.ɪŋ ˈsɪs.təm / (AmE), / ˌɪn.fəˈmeɪ.ʃən ˈprəʊ.ses.ɪŋ ˈsɪs.təm / (BrE)
a machine or device that performs calculations and processes data automatically, especially a computer or similar electronic system
machine for automatic data processing

Examples

The hospital upgraded its information processing system to improve patient record management.
Our company relies heavily on an efficient information processing system for daily operations.
Advancements in technology have made the information processing system faster and more reliable.

Related Terms

  • Synonym: computer - an electronic device for storing and processing data
