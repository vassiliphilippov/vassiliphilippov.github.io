input data

1. NOUN

/ ˈɪnˌpʊt ˈdeɪtə / (AmE), / ˈɪnpʊt ˈdeɪtə / (BrE)
data or information supplied to a device or program, especially as a file used for processing by a computer.
data supplied to a program or device

Examples

The program processes input data from various sources to generate reports.
Always verify your input data for accuracy before running the simulation.
A bug in the code caused the input data to be read incorrectly.

Related Terms

  • Antonym: output data - data produced by a device or program as a result of processing input data
