tabular array

1. NOUN

/ ˈtæbjələr əˈreɪ / (AmE), / ˈtabjʊlə əˈreɪ / (BrE)
an arrangement of data in rows and columns, especially for easy reference or analysis
data set arranged in rows and columns

Examples

The spreadsheet displayed the results as a tabular array for clarity.
All student grades were recorded in a tabular array on the last page of the report.
The scientist organized her findings into a tabular array for the journal article.
