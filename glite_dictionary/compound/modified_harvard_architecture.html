modified Harvard architecture

1. NOUN

/ ˈmɑːdəˌfaɪd ˈhɑrvərd ˌɑrkəˈtɛktʃɚ / (AmE), / ˈmɒdɪˌfaɪd ˈhɑːvəd ˌɑːkɪˈtɛktʃə / (BrE)
A variation of the Harvard computer architecture in which instruction memory can be accessed as data memory.
variation of Harvard computer architecture

Examples

Most embedded processors now use a modified Harvard architecture for increased efficiency.
The modified Harvard architecture allows programs to write to instruction memory, enabling flexible updates.
When studying microcontroller designs, understanding the modified Harvard architecture is essential for memory access planning.

Synset Examples

Most modern computers that are documented as Harvard architecture are, in fact, Modified Harvard architecture.

Related Terms

  • Degree Lower: Harvard architecture - An architecture with physically separate storage and signal pathways for instructions and data.

Etymology

Named after the Harvard Mark I computer, with 'modified' indicating its adaptation.
