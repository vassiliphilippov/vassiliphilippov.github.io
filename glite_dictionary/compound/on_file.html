on file

1. ADP

/ ɑn faɪl / (AmE), / ɒn faɪl / (BrE)
kept or recorded in an official or organized collection of documents or records
stored in official records

Examples

The university keeps all student transcripts on file for several years.
Please ensure your updated contact information is on file with the office.
The application will not be processed until your medical records are on file.

Synset Examples

The HR department keeps all employee records on file.
Make sure the contract is on file before signing.
I need to verify your address is on file with us.
