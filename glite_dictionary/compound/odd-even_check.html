odd-even check

1. NOUN

/ ɑd ˈivɪn tʃɛk / (AmE), / ɒd ˈiːvən tʃɛk / (BrE)
a method of detecting errors in computer data by checking whether the number of set bits is odd or even
error-checking method in computing

Examples

The system uses an odd-even check to detect corrupted transmissions in the data stream.
When developing new hardware, engineers implemented an odd-even check for added reliability.
If an odd-even check fails, the device will request the data to be sent again.

Related Terms

  • Synonym: parity check - a test that checks the parity (odd or even) of bits for error detection
