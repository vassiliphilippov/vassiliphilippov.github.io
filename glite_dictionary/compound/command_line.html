command line

1. NOUN

/ kəˈmænd laɪn / (AmE), / kəˈmɑːnd laɪn / (BrE)
a way of interacting with a computer by typing text commands to run programs or perform tasks, rather than using a graphical interface
text-based computer interface

Examples

To install the software, open the command line and enter the instructions provided.
Experienced users often prefer the command line for its efficiency and control.
The system administrator fixed the issue directly from the command line.

Related Terms

  • Synonym: terminal - computer interface used to input commands, often via command line
  • Antonym: graphical user interface - interface allowing users to interact with computers through graphics and icons
