random memory

1. NOUN

/ ˈrændəm ˈmɛməri / (AmE), / ˈrændəm ˈmɛməri / (BrE)
a type of computer memory that allows data to be accessed or stored in any order, used for temporary storage while a computer is running
computer memory for temporary data storage

Examples

The computer's random memory determines how many programs can run at once.
Upgrading your system's random memory can significantly improve performance.
All unsaved data is lost when the power goes off because it's stored in random memory.

Related Terms

  • Synonym: RAM - random access memory; computer memory that can be accessed randomly

Etymology

Derived from the concept of 'random access', referring to the ability to access any memory location directly, not sequentially.
