tool kit

1. NOUN

/ ˈtuːl kɪt / (AmE), / ˈtuːl kɪt / (BrE)
a set of tools kept together for use, especially by a carpenter or mechanic
set of tools kept together

Examples

Every carpenter should have a reliable tool kit for everyday projects.
She bought a new tool kit to assemble the furniture herself.
The mechanic opened his tool kit to fix the leaking pipe.

Related Terms

  • Synonym: toolbox - container for keeping tools
