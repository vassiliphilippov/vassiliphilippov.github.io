uniform resource locator

1. NOUN

/ ˈjuː.nə.fɔrm rɪˈsɔrs ləˈkeɪ.tər / (AmE), / ˈjuː.nɪ.fɔːm rɪˈsɔːs ləˈkeɪ.tə / (BrE)
the address used to identify and locate a resource on the World Wide Web, commonly abbreviated to 'URL'.
web address (URL)

Examples

Please enter the uniform resource locator of the website you want to visit.
Each webpage can be found using a unique uniform resource locator.
He sent me the uniform resource locator for the registration page via email.

Related Terms

  • Synonym: URL - abbreviation of uniform resource locator
  • Synonym: web address - common term for the address of a web page

Etymology

Formed from 'uniform' + 'resource' + 'locator', coined in the early 1990s in computing.
