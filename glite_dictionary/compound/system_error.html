system error

1. NOUN

/ ˈsɪstəm ˌɛrər / (AmE), / ˈsɪstəm ˌɛrər / (BrE)
an error message or condition indicating that an instruction is not recognized by an operating system, or that procedural rules have been violated
error condition in computing

Examples

The computer displayed a system error when it tried to process the corrupted file.
After the software update, users reported receiving frequent system errors.
If you see a system error, try restarting your computer to resolve the issue.

Related Terms

  • Synonym: bug - an error or flaw in computer software or hardware
  • Synonym: crash - a sudden failure of a computer or system
