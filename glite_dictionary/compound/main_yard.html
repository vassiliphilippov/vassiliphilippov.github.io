main yard

1. NOUN

/ meɪn jɑrd / (AmE), / meɪn jɑːd / (BrE)
the horizontal spar on a square-rigged ship from which the mainsail is set
nautical: spar for a mainsail

Examples

The crew climbed the rigging to secure the main yard before the storm arrived.
During the restoration, the main yard was carefully replaced with a replica.
The captain ordered the main yard to be braced to catch more wind.
