relational database management system

1. NOUN

/ rɪˈleɪʃənəl ˈdeɪtəˌbeɪs ˈmænɪdʒmənt ˌsɪstəm / (AmE), / rɪˈleɪʃənəl ˈdeɪtəˌbeɪs ˈmænɪdʒmənt ˌsɪstəm / (BrE)
a type of database management system designed to store, manage, and retrieve data organized in a relational database, using tables and structured query language (SQL)
system managing data in relational databases

Examples

The company upgraded its software to a modern relational database management system to handle more complex queries.
Most enterprise applications rely on a relational database management system like Oracle or MySQL for data storage.
Learning how to use a relational database management system is essential for many careers in IT.

Related Terms

  • Synonym: database management system - software for storing and retrieving large amounts of data

Etymology

Term formed in the 1970s from 'relation' (referring to mathematical relations) and 'database management system'.
