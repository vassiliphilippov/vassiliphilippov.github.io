matrix addition

1. NOUN

/ ˈmeɪtrɪks əˈdɪʃən / (AmE), / ˈmeɪtrɪks əˈdɪʃən / (BrE)
the process of adding two or more matrices by adding their corresponding elements
adding matrices by combining corresponding elements

Examples

To find the sum of two matrices, you use matrix addition.
Matrix addition is only possible when both matrices have the same dimensions.
In linear algebra, matrix addition is one of the fundamental operations studied.

Related Terms

  • Antonym: matrix multiplication - the multiplication of matrices
