field-sequential color television

1. NOUN

/ ˈfiːld səˈkwɛnʃəl ˈkʌlər ˌtɛləˈvɪʒən / (AmE), / ˈfiːld sɪˈkwɛnʃəl ˈkʌlə ˌtɛlɪˈvɪʒən / (BrE)
an early type of color television in which images are displayed by rapidly scanning successive fields in the three primary colors, creating the impression of full color when viewed by the eye.
early color television using sequential color fields

Examples

The museum displayed an original field-sequential color television prototype from the 1940s.
CBS developed the field-sequential color television system before the standardization of the NTSC format.
Technicians explained how field-sequential color television relied on spinning color filters to create images.
