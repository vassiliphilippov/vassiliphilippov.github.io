tickler file

1. NOUN

/ ˈtɪk.lɚ faɪl / (AmE), / ˈtɪk.lə faɪl / (BrE)
A file or organized collection of reminders, memoranda, or notices arranged to prompt action at specific times.
file of reminders for future tasks

Examples

She set up a tickler file to keep track of her clients' appointments and deadlines.
Each bill was placed in the tickler file so it could be paid on time each month.
The office relies on a tickler file system to ensure that all important paperwork is handled promptly.

Etymology

From 'tickler' (something that stimulates or prompts) + 'file,' originally office slang from the early 20th century.
