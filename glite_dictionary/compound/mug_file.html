mug file

1. NOUN

/ mʌɡ faɪl / (AmE), / mʌɡ faɪl / (BrE)
a file or collection of mug shots kept by the police for identifying criminals
police file of mug shots

Examples

The detective searched the mug file for a match to the suspect.
Before computers, officers maintained a physical mug file at the station.
She recognized the thief's face from the mug file shown to her by police.

Related Terms

  • Synonym: mug shot - a photograph of a person's face, especially one made for police records
