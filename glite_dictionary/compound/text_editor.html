text editor

1. NOUN

/ ˈtɛkst ˌɛd.ɪ.tɚ / (AmE), / ˈtɛkst ˌɛd.ɪ.tə / (BrE)
A computer program used for creating, viewing, and editing plain text files.
computer program for editing text files

Examples

Many programmers use a text editor to write and modify source code.
She prefers working in a simple text editor instead of a word processor.
The text editor allows users to search for and replace words easily.

Related Terms

  • Degree Higher: word processor - a program for creating and editing documents with formatting features
