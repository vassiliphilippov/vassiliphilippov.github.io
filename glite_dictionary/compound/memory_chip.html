memory chip

1. NOUN

/ ˈmɛməri tʃɪp / (AmE), / ˈmɛməri tʃɪp / (BrE)
A small integrated circuit that stores data and can be used as computer memory, such as RAM.
microchip for computer memory

Examples

The computer runs faster after I installed a new memory chip.
Some laptops allow you to upgrade by adding extra memory chips.
A faulty memory chip can cause data corruption and system crashes.

Related Terms

  • Synonym: RAM - Random Access Memory, a type of computer memory
