control system

1. NOUN

/ kənˈtroʊl ˌsɪstəm / (AmE), / kənˈtrəʊl ˌsɪstəm / (BrE)
A system that manages, commands, directs, or regulates the behavior or operation of another system.
system that regulates another system

Examples

The automated control system improved the efficiency of the power plant.
A modern car’s control system ensures that the engine runs smoothly under all conditions.
Engineers designed a new control system to monitor and adjust the assembly line in real time.
