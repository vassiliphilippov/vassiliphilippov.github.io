programming language

1. NOUN

/ ˈproʊɡræmɪŋ ˈlæŋɡwɪdʒ / (AmE), / ˈprəʊɡramɪŋ ˈlaŋɡwɪdʒ / (BrE)
a formal language designed to communicate instructions to a computer and produce various kinds of output
formal language for programming computers

Examples

Python is a programming language that is popular for its simplicity and readability.
Learning a programming language can open many career opportunities in technology.
Each programming language has its own syntax and use cases.

Related Terms

  • Synonym: computer language - language designed for communicating with computers
Source: WordNet

2. NOUN

/ ˈproʊɡræmɪŋ ˌlæŋɡwɪdʒ / (AmE), / ˈprəʊɡramɪŋ ˌlaŋɡwɪdʒ / (BrE)
a language with a set of instructions and rules designed for writing programs to be executed by a computer
a language used to write computer programs

Examples

The Python programming language is popular for its simplicity and readability.
Students learned to write code in a new programming language this semester.
Each programming language has its own syntax and unique features.

Related Terms

  • Synonym: computer language - another term for a programming language used to communicate instructions to a computer
