computer menu

1. NOUN

/ kəmˈpjuːtɚ ˈmɛnjuː / (AmE), / kəmˈpjuːtə ˈmɛnjuː / (BrE)
a list of commands or options presented on a computer screen that a user can choose from to perform specific actions
on-screen list of computer commands or options

Examples

To print your document, select 'Print' from the computer menu.
The new software features a more intuitive computer menu for easier navigation.
Click the 'File' option in the computer menu to save your changes.

Related Terms

  • Synonym: menu - a list of options displayed to a user by a computer application
