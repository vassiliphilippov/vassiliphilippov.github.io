periodic table

1. NOUN

/ ˌpɪriˈɑdɪk ˈteɪbəl / (AmE), / ˌpɪəriˈɒdɪk ˈteɪbl̩ / (BrE)
A chart showing all known chemical elements arranged by atomic number, grouping elements with similar properties together.
chart of chemical elements by atomic number

Examples

Students in chemistry class memorized the periodic table to help identify each element.
Dmitri Mendeleev is credited with creating the first version of the periodic table in the 19th century.
Each row of the periodic table is called a period and has its own unique properties.

Etymology

From 'periodic', relating to periods or cycles, and 'table', from Latin tabula (a board, plank, or chart). The name reflects its arrangement by periodic trends.
