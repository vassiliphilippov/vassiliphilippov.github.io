sexadecimal number system

1. NOUN

/ ˌsɛksəˈdɛsɪməl ˈnʌmbɚ ˌsɪstəm / (AmE), / ˌsɛksəˈdɛsɪməl ˈnʌmbə ˌsɪstəm / (BrE)
a positional numeral system with a base of sixteen, using sixteen distinct symbols (0–9 and A–F) to represent values
numeral system with base 16

Examples

Some programmers still refer to the sexadecimal number system, though 'hexadecimal' is much more common today.
Counting in the sexadecimal number system requires memorizing digits up to F.
The sexadecimal number system is often used in computing because it aligns neatly with binary code.

Related Terms

  • Synonym: hexadecimal number system - numeral system with base 16, commonly known as hexadecimal

Etymology

From Latin 'sex' (six) and 'decem' (ten), modelled after 'hexadecimal.'
