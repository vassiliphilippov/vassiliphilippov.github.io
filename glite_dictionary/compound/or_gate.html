OR gate

1. NOUN

/ ɔr ɡeɪt / (AmE), / ɔː ɡeɪt / (BrE)
A logic circuit in computing or electronics that produces an output signal when any of its input signals are present.
logic circuit; outputs if any input is present

Examples

The OR gate allows the circuit to trigger if either switch is pressed.
Combining several OR gates can help simplify complex digital logic designs.
Unlike the AND gate, the OR gate only needs one input to be high for the output to be high.

Related Terms

  • Antonym: AND gate - a logic circuit that outputs only when all inputs are present
  • Degree Higher: NAND gate - a logic circuit producing output unless all inputs are present

Etymology

From 'OR', denoting the logical operation meaning 'one or more', plus 'gate', meaning a device that controls the flow of signals.
