object language

1. NOUN

/ ˈɑbdʒɛkt ˌlæŋɡwɪdʒ / (AmE), / ˈɒbdʒɛkt ˌlæŋɡwɪdʒ / (BrE)
A computer language into which a program or code written in another language is translated.
language that code is translated into

Examples

The compiler translates the source code into the object language for execution.
Understanding both the source and object language is crucial for debugging compiled programs.
Different object languages can be supported by a versatile translation tool.

Related Terms

  • Antonym: source language - the language from which code or data is translated
Source: WordNet

2. NOUN

/ ˈɑbdʒɛkt ˌlæŋɡwɪdʒ / (AmE), / ˈɒbdʒɛkt ˌlæŋɡwɪdʒ / (BrE)
the language into which a text is translated from another language
language translated into

Examples

The English edition of the novel serves as the object language for readers around the world.
When translating from French to German, German is the object language.
A skilled translator must understand both the source language and the object language well.

Related Terms

  • Antonym: source language - the language from which a text is translated

Etymology

Formed from 'object' (as the target or goal) and 'language'; modeled after technical usage in linguistics.
