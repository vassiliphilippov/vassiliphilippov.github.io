target program

1. NOUN

/ ˈtɑrɡət ˈproʊɡræm / (AmE), / ˈtɑːɡɪt ˈprəʊɡræm / (BrE)
a fully compiled or assembled program that is ready to be loaded and run on a computer
compiled program ready to run

Examples

The compiler generates a target program that can be executed on the specified hardware.
After debugging the source code, we produced the final target program for deployment.
A target program must be compatible with the operating system to function correctly.

Related Terms

  • Synonym: executable - a file that contains a program ready to run
