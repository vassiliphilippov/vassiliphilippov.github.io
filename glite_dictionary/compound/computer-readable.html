computer-readable

1. ADJ

/ kəmˈpjuːtɚ ˈriːdəbəl / (AmE), / kəmˈpjuːtə ˈriːdəbəl / (BrE)
(of data or a document) in a format that can be directly processed by a computer without further conversion.
able to be processed by a computer

Examples

The report was saved as a computer-readable file for easy sharing.
All applications must submit their data in computer-readable form.
The transcript is available in a computer-readable format on the website.

Related Terms

  • Synonym: machine-readable - in a format that can be read automatically by a machine
