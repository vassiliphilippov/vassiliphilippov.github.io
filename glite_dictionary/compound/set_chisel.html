set chisel

1. NOUN

/ sɛt ˈtʃɪzəl / (AmE), / sɛt ˈtʃɪzəl / (BrE)
a narrow steel chisel used for cutting stone or bricks
tool for cutting stone or bricks

Examples

The mason reached for his set chisel to make precise cuts in the granite block.
When working with bricks, a set chisel provides better control than a wider chisel.
You should always wear safety goggles when using a set chisel to chip stone.

Related Terms

  • Synonym: chisel - a tool with a sharp cutting edge at the end of a metal blade, used for shaping wood, stone, or metal
