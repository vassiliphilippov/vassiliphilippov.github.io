binary system

1. NOUN

/ ˈbaɪnɛri ˈsɪstəm / (AmE), / ˈbaɪnəri ˈsɪstəm / (BrE)
A method of counting or expressing numbers using only the digits 0 and 1, with a base of two.
number system using only 0 and 1

Examples

All information in a computer is ultimately represented in the binary system.
The binary system uses only two symbols: zero and one.
Learning the binary system is essential for understanding digital technology.

Related Terms

  • Antonym: decimal system - number system with base ten, using digits 0 through 9

Etymology

From Latin 'binarius' (consisting of two) + English 'system'.
