command line interface

1. NOUN

/ kəˈmænd laɪn ˈɪntərˌfeɪs / (AmE), / kəˈmɑːnd laɪn ˈɪntəˌfeɪs / (BrE)
A method of interacting with a computer in which the user types text commands instead of using menus or icons.
text-based computer user interface

Examples

Developers often prefer using the command line interface for tasks that require automation and scripting.
You can access the system’s settings by typing commands in the command line interface.
Some operating systems only provide a command line interface for user interaction.

Related Terms

  • Antonym: graphical user interface - a user interface that includes graphical elements like windows and icons
