security system

1. NOUN

/ sɪˈkjʊr.ə.ti ˈsɪs.təm / (AmE), / sɪˈkjʊə.rɪ.ti ˈsɪs.təm / (BrE)
A computing system or set of measures designed to enforce boundaries and control access between computer networks.
computing system controlling network access

Examples

The company upgraded its security system to prevent hackers from breaching its network.
Before connecting new devices, IT staff checked that the security system was properly configured for remote access.
A strong security system is essential for protecting sensitive information in cloud computing environments.

Related Terms

  • Synonym: firewall - a device or software that blocks unauthorized access to a network
Source: WordNet
WordNet ID: oewn-04173172-n
Synset POS: n

2. NOUN

/ sɪˈkjʊrəti ˌsɪstəm / (AmE), / sɪˈkjʊərəti ˌsɪstəm / (BrE)
A set of electronic devices or equipment designed to detect unauthorized entry or activities and alert occupants or authorities, typically by setting off an alarm.
electronic alarm against unauthorized entry

Examples

The security system was triggered when the window was opened late at night.
Many homeowners install a security system to protect their property and belongings.
Our company upgraded its security system to include cameras and motion sensors.

Related Terms

  • Synonym: alarm system - electronic device for warning of intrusion or fire
Source: WordNet
WordNet ID: oewn-04173006-n
Synset POS: n
