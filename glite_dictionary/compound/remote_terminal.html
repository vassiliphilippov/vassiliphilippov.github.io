remote terminal

1. NOUN

/ rɪˈmoʊt ˈtɜrmɪnəl / (AmE), / rɪˈməʊt ˈtɜːmɪnəl / (BrE)
a device that allows a user to interact with a computer from a distant location via a data connection
terminal connected to a computer from afar

Examples

The engineer accessed the server using a remote terminal from his office.
Many universities provide students with remote terminals to access campus computers remotely.
Security measures must be in place when connecting through a remote terminal.

Related Terms

  • Synonym: terminal - a device for entering data into or receiving data from a computer
  • Antonym: local terminal - a terminal directly connected to the computer
