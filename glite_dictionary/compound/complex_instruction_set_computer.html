complex instruction set computer

1. NOUN

/ ˈkɑmˌplɛks ɪnˈstrʌkʃən sɛt kəmˈpjutɚ / (AmE), / ˈkɒm.pleks ɪnˈstrʌk.ʃən set kəmˈpjuː.tə / (BrE)
a type of computer architecture in which the CPU is designed to execute a large set of specialized instructions
computer with many built-in CPU instructions

Examples

The complex instruction set computer architecture was widely used in early desktop and mainframe systems.
Many modern processors have roots in the complex instruction set computer design philosophy.
The differences between a complex instruction set computer and a reduced instruction set computer affect compiler design.

Related Terms

  • Antonym: reduced instruction set computer - a computer architecture with a minimal set of instructions
