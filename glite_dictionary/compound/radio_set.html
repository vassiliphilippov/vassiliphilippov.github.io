radio set

1. NOUN

/ ˈreɪdioʊ sɛt / (AmE), / ˈreɪdiəʊ sɛt / (BrE)
an electronic device that receives, demodulates, and amplifies radio signals for listening.
radio signal receiving device

Examples

My grandfather still uses a vintage radio set to listen to news broadcasts every morning.
They found an old radio set in the attic, covered in dust but still working.
During the storm, the radio set provided vital updates when the power went out.

Related Terms

  • Synonym: radio - electronic device that receives radio signals
