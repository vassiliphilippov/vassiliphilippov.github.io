code flag

1. NOUN

/ koʊd flæɡ / (AmE), / kəʊd flaɡ / (BrE)
a flag used as part of an international code of signals for communication between ships
international maritime signal flag

Examples

The ship hoisted the code flag to indicate a change in course.
During naval exercises, various code flags were displayed to signal maneuvers.
Each code flag in the set represents a different letter or message.

Related Terms

  • Synonym: signal flag - a flag used to convey signals or information at sea
