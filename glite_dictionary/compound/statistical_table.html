statistical table

1. NOUN

/ stəˈtɪstɪkəl ˈteɪbəl / (AmE), / stəˈtɪstɪkəl ˈteɪbəl / (BrE)
A table that displays numerical data in an organized way for statistical analysis or reference.
table presenting statistical data

Examples

The report included a detailed statistical table summarizing the survey results.
Researchers often use a statistical table to compare variables over multiple years.
She referenced the statistical table during her presentation to illustrate key trends.

Related Terms

  • Synonym: data table - a table organizing information, often numerical
