machine translation

1. NOUN

/ məˈʃin trænzˈleɪʃən / (AmE), / məˈʃiːn trænzˈleɪʃən / (BrE)
the process of using computers to automatically translate text or speech from one language into another
automatic computer-based language translation

Examples

Recent advances in machine translation have made it easier to communicate across languages.
Many websites now offer machine translation for foreign language content.
While helpful, machine translation sometimes struggles with idiomatic expressions.

Related Terms

  • Degree Lower: computer-assisted translation - translation aided by computer software, but not fully automated
