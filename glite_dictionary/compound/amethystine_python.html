amethystine python

1. NOUN

/ ˌæməˌθɪstiːn ˈpaɪθɑn / (AmE), / ˌamɪˈθɪstiːn ˈpʌɪθən / (BrE)
a large python species native to Australia and New Guinea, noted for its iridescent, amethyst-colored sheen
large python species with amethyst sheen

Examples

The amethystine python is one of the largest snake species found in Australia.
Herpetologists admire the amethystine python for its striking, shimmering scales.
You can spot an amethystine python basking in the sun near rainforests.

Etymology

From 'amethystine' (resembling amethyst, a purple quartz) + 'python' (a type of large snake); the name refers to its coloration.
