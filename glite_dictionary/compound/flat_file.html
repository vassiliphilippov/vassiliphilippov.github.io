flat file

1. NOUN

/ flæt faɪl / (AmE), / flæt faɪl / (BrE)
a hand tool with two flat surfaces used for smoothing or shaping wood, metal, or other materials
hand tool with two flat surfaces

Examples

He used a flat file to smooth the edges of the metal sheet.
A flat file is essential for sharpening tools in a workshop.
After cutting the wood, she reached for a flat file to refine its surface.

Related Terms

  • Synonym: file - a hand tool with a roughened surface for smoothing or shaping materials
