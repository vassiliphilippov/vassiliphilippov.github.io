executive program

1. NOUN

/ ɪɡˈzɛkjətɪv ˈproʊɡræm / (AmE), / ɪɡˈzɛkjʊtɪv ˈprəʊɡræm / (BrE)
A computer program that controls the execution of other programs, typically as part of an operating system or similar environment.
program controlling other programs

Examples

The executive program manages the scheduling of multiple user applications.
A malfunction in the executive program can disrupt the operation of the entire computer system.
Updating the executive program is essential for improving system security.

Related Terms

  • Synonym: operating system - system software that manages computer hardware and software resources
