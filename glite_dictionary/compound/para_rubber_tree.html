Para rubber tree

1. NOUN

/ ˈpærə ˈrʌbər tri / (AmE), / ˈpɑːrə ˈrʌbə triː / (BrE)
A deciduous tree native to South America, especially the Amazon, that produces milky latex used as the main source of commercial natural rubber.
tree yielding commercial natural rubber

Examples

The Para rubber tree is cultivated in plantations throughout Southeast Asia.
Natural latex harvested from the Para rubber tree is essential for producing many rubber goods.
The Para rubber tree yields a milky sap that solidifies into natural rubber.

Related Terms

  • Synonym: Hevea brasiliensis - scientific name for the Para rubber tree
  • Synonym: rubber tree - another common name for trees yielding latex for rubber

Etymology

From 'Para', a region and river in Brazil, combined with 'rubber tree'.
