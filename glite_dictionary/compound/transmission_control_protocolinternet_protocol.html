Transmission Control Protocol/Internet Protocol

1. NOUN

/ trænzˈmɪʃən kənˈtroʊl ˈproʊtəkɑl ˈɪntərˌnɛt prəˈtoʊkɑl / (AmE), / trænzˈmɪʃən kənˈtrəʊl ˈprəʊtəkɒl ˈɪntənɛt prəˈtəʊkɒl / (BrE)
a set of networking protocols, including TCP and IP, developed to allow computers on different networks to communicate over the Internet
core internet communication protocols

Examples

Most modern networks use Transmission Control Protocol/Internet Protocol as their communication standard.
Setting up a secure server requires proper configuration of Transmission Control Protocol/Internet Protocol parameters.
The development of Transmission Control Protocol/Internet Protocol revolutionized global communication and data sharing.

Related Terms

  • Synonym: TCP/IP - abbreviation for Transmission Control Protocol/Internet Protocol

Etymology

From the names of the two main protocols: Transmission Control Protocol (TCP) and Internet Protocol (IP), developed in the 1970s.
