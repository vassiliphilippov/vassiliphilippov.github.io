machine tool

1. NOUN

/ məˈʃin tuːl / (AmE), / məˈʃiːn tʊːl / (BrE)
A power-driven machine used for cutting, shaping, or finishing metals or other materials.
power-driven cutting or shaping machine

Examples

The factory invested in a new machine tool to improve their production efficiency.
Using a precision machine tool, technicians shaped the steel components to exact specifications.
Advances in machine tool technology have transformed modern manufacturing processes.

Related Terms

  • Synonym: lathe - a type of machine tool that rotates the workpiece
  • Synonym: milling machine - a machine tool for shaping metal by rotating cutters
