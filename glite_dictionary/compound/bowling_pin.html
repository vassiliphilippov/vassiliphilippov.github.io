bowling pin

1. NOUN

/ ˈboʊlɪŋ pɪn / (AmE), / ˈbəʊlɪŋ pɪn / (BrE)
a heavy, club-shaped wooden object used as the target in the game of bowling, typically arranged in triangular groups of ten
club-shaped target in bowling

Examples

He knocked down every bowling pin on his first try.
Each bowling pin is carefully set up before a new game starts.
The child struggled to carry a single bowling pin because it was heavier than expected.

Related Terms

  • Synonym: pin - a slender, usually pointed object used as a fastener or marker
