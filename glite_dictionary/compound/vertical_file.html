vertical file

1. NOUN

/ ˈvɜrtɪkəl faɪl / (AmE), / ˈvɜːtɪkəl faɪl / (BrE)
a filing cabinet or storage system in which documents or records are stored upright on one edge, typically in folders.
filing cabinet with upright folders

Examples

She pulled the report from the vertical file beside her desk.
Most offices store client records in a vertical file for easy access.
The new assistant organized the invoices in the vertical file by date.

Related Terms

  • Antonym: lateral file - a filing cabinet where records are stored horizontally or side-to-side
