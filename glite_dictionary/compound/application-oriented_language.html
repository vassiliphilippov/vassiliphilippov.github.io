application-oriented language

1. NOUN

/ ˌæplɪˌkeɪʃən ɔːriˌɛntɪd ˈlæŋɡwɪdʒ / (AmE), / ˌæplɪˌkeɪʃən ɔːriˌɛntɪd ˈlæŋɡwɪdʒ / (BrE)
a programming language designed so that its statements closely resemble terminology used by its intended users or application domain
programming language resembling user terminology

Examples

The new application-oriented language made it easier for engineers to write code using familiar terms.
Financial analysts appreciated that the application-oriented language allowed them to automate reports without learning traditional programming syntax.
The development team opted for an application-oriented language tailored to the needs of scientists in the lab.

Related Terms

  • Synonym: domain-specific language - a computer language specialized to a particular application domain
