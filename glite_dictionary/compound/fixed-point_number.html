fixed-point number

1. NOUN

/ ˈfɪkst pɔɪnt ˈnʌm.bɚ / (AmE), / ˈfɪkst pɔɪnt ˈnʌm.bə / (BrE)
a number expressed using fixed-point notation, in which a specific number of digits is allotted to the integer and fractional parts
number in fixed-point notation

Examples

In embedded systems, calculations are often performed using fixed-point numbers for efficiency.
A fixed-point number can represent decimal values but with limited precision.
The software requires input in the form of fixed-point numbers instead of floating-point numbers.

Related Terms

  • Antonym: floating-point number - a number represented in floating-point notation
