computer code

1. NOUN

/ kəmˈpjuːtər koʊd / (AmE), / kəmˈpjuːtə kəʊd / (BrE)
the set of written instructions or symbols used to program a computer
instructions or symbols in a computer program

Examples

He spent several hours writing computer code for the new application.
The company hired a specialist to review the existing computer code for errors.
Learning how to read computer code can open many career opportunities.

Related Terms

  • Synonym: source code - the original form of computer code written by a programmer
  • Degree Lower: machine code - a computer code written in machine language directly understood by the computer
