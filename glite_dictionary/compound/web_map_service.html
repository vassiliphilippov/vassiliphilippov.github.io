Web Map Service

1. NOUN

/ wɛb mæp ˈsɝːvɪs / (AmE), / wɛb mæp ˈsɜːvɪs / (BrE)
a computer protocol or program that dynamically provides maps of spatially referenced data over the internet, typically using geographic information systems (GIS) standards
protocol for serving dynamic geographic maps online

Examples

Many geographic applications rely on a Web Map Service to display interactive maps to users.
The team implemented a Web Map Service to share real-time weather data online.
By integrating a Web Map Service, the company improved access to geospatial information for developers.

Related Terms

  • Synonym: WMS - abbreviation for Web Map Service in geographic information systems

Etymology

Formed from 'web' (World Wide Web), 'map', and 'service', commonly used in GIS contexts.
