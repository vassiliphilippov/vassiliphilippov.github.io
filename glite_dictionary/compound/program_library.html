program library

1. NOUN

/ ˈproʊɡræm ˈlaɪˌbrɛri / (AmE), / ˈprəʊɡræm ˈlaɪbrəri / (BrE)
(computing) a collection of standard programs and subroutines, stored together and made available for immediate use.
computing: collection of reusable programs/subroutines

Examples

The developer imported a program library to simplify the implementation of common functions.
Most modern software systems rely heavily on at least one reusable program library.
By updating the program library, the team fixed several bugs across multiple applications.

Related Terms

  • Synonym: library - a collection of programs and subroutines available for use in computing
