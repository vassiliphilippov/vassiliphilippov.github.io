memory device

1. NOUN

/ ˈmɛməri dɪˌvaɪs / (AmE), / ˈmɛməri dɪˌvaɪs / (BrE)
a piece of hardware or equipment used to store data or information for later retrieval
device for storing information

Examples

The USB flash drive is a popular memory device for transferring files between computers.
Many modern phones rely on a solid-state memory device to store photos and apps.
You can upgrade your computer's performance by installing a faster memory device.

Related Terms

  • Synonym: storage device - a device for storing digital data
