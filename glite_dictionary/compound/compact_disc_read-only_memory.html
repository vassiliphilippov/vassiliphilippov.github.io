compact disc read-only memory

1. NOUN

/ kəmˈpækt dɪsk ˌriːd ˈoʊnli ˈmɛməri / (AmE), / kəmˈpækt dɪsk ˌriːd ˈəʊnli ˈmɛməri / (BrE)
A type of compact disc used with computers for storing large amounts of digital data that cannot be altered by the user.
computer storage disc (data not rewritable)

Examples

Many old computer games were sold on compact disc read-only memory in the 1990s.
The manual instructed users to insert the compact disc read-only memory into the drive to install the software.
Because it is a compact disc read-only memory, users cannot modify or delete its contents.

Related Terms

  • Synonym: CD-ROM - abbreviation for compact disc read-only memory
  • Degree Higher: DVD-ROM - digital versatile disc read-only memory, a similar storage format
  • Degree Lower: compact disc - an optical disc used to store audio, video, or data

Etymology

From 'compact disc' (abbreviated CD) and 'read-only memory' (abbreviated ROM); 'compact disc' from Latin 'compactus' (joined together), 'read-only memory' coined in computing in the 20th century.
