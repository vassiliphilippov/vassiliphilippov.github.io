parallel interface

1. NOUN

/ ˈpærəˌlɛl ˈɪn.təˌfeɪs / (AmE), / ˈpærəˌlɛl ˈɪn.təˌfeɪs / (BrE)
a type of interface between a computer and a peripheral device, such as a printer, that transmits multiple bits of data simultaneously across separate wires
interface transmitting multiple bits simultaneously

Examples

Most older printers connect to the computer using a parallel interface instead of a USB port.
The parallel interface allows data to be transferred much faster than a serial connection in certain applications.
Troubleshooting a parallel interface often involves checking the cable and port for physical damage.

Related Terms

  • Antonym: serial interface - an interface sending data one bit at a time
