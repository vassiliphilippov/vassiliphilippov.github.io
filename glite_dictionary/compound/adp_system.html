ADP system

1. NOUN

/ ˌeɪ ˌdiː ˈpiː ˈsɪstəm / (AmE), / ˌeɪ ˌdiː ˈpiː ˈsɪstəm / (BrE)
a system of one or more computers and related software used for automated data processing, sharing common storage
automated data processing system

Examples

The company upgraded its ADP system to handle increased data volumes.
All employee records are managed through the ADP system for better efficiency.
Outdated ADP systems can pose security risks to sensitive information.

Related Terms

  • Synonym: computer system - a complete, functional computer, including hardware, software, and peripheral devices

Etymology

Abbreviation of Automated Data Processing + system.
