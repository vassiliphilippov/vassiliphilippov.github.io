instruction manual

1. NOUN

/ ɪnˈstrʌkʃən ˈmænjuəl / (AmE), / ɪnˈstrʌkʃən ˈmanjʊəl / (BrE)
a booklet or document that provides detailed information on how to install, operate, or use a technical device or system
booklet with operating instructions

Examples

Before assembling the furniture, she carefully read the instruction manual.
The instruction manual is available in several languages for international customers.
If you lose the instruction manual, you can usually download a copy from the manufacturer's website.

Related Terms

  • Synonym: user guide - a document with instructions for using a product or system
  • Synonym: handbook - a concise reference book providing instructions or information
