additional RAM

1. NOUN

/ əˈdɪʃənəl ræm / (AmE), / əˈdɪʃənəl ram / (BrE)
extra random-access memory added to a computer to improve its performance
extra random-access memory for a computer

Examples

He installed additional RAM to make his computer run faster.
Many modern laptops allow users to add additional RAM for better multitasking.
If your applications are slow, upgrading with additional RAM can help improve performance.

Synset Examples

He installed additional ram for better gaming experience.

Related Terms

  • Synonym: RAM - random-access memory, computer memory that can be accessed randomly

Etymology

RAM is an acronym for random-access memory.
