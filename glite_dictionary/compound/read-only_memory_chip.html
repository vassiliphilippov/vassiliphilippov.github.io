read-only memory chip

1. NOUN

/ ˌriːd ˈoʊnli ˈmɛməri tʃɪp / (AmE), / ˌriːd ˈəʊnli ˈmɛməri tʃɪp / (BrE)
a microchip that stores data which can only be read and not altered, typically used to hold firmware or permanent instructions for a computer or device
microchip storing data that can't be changed

Examples

The computer's BIOS is stored on a read-only memory chip to prevent accidental modification.
Replacing a defective read-only memory chip restored the device to working order.
Many embedded systems rely on a read-only memory chip for essential control software.

Related Terms

  • Synonym: ROM - abbreviation for read-only memory, often used for such chips
