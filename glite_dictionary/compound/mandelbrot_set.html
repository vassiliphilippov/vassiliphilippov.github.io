Mandelbrot set

1. NOUN

/ ˈmæn.dəlˌbrɑt sɛt / (AmE), / ˈman.dəlˌbrɒt sɛt / (BrE)
a set of complex numbers that forms a fractal, with a boundary exhibiting intricate patterns when plotted in the complex plane, significant in mathematics and chaos theory
mathematical fractal set in the complex plane

Examples

The Mandelbrot set is famous for its stunningly intricate fractal patterns.
In my mathematics class, we studied the properties of the Mandelbrot set using computer models.
Programs can visually render the Mandelbrot set by iterating a simple equation over complex numbers.

Etymology

Named after Benoit B. Mandelbrot (1924–2010), a French-American mathematician who studied and popularized fractals.
