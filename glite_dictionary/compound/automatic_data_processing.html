automatic data processing

1. NOUN

/ ˌɔtəˈmætɪk ˈdeɪtə ˌproʊˈsɛsɪŋ / (AmE), / ˌɔːtəˈmætɪk ˈdeɪtə ˌprəʊˈsɛsɪŋ / (BrE)
the use of computers to systematically process data without manual intervention.
computerized data processing

Examples

Many businesses rely on automatic data processing to handle payroll and record-keeping efficiently.
The development of automatic data processing revolutionized how organizations manage large volumes of information.
With advances in automatic data processing, routine administrative tasks can be completed much faster than before.

Related Terms

  • Synonym: data processing - the handling and manipulation of data, often by computer
