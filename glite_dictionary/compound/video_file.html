video file

1. NOUN

/ ˈvɪdioʊ faɪl / (AmE), / ˈvɪdiəʊ faɪl / (BrE)
a computer file that contains digitized video, usually in a specific format such as MP4 or AVI
a computer file with digitized video

Examples

She attached the video file to the email so her colleagues could watch the presentation.
Make sure the video file is compatible with your media player before playing it.
The video file was too large to upload without compressing it first.

Synset Examples

A program (or hardware) which can decode video or audio is called a codec; playing or encoding a video file will sometimes require the user to install a codec library corresponding to the type of video and audio coding used in the file.

Related Terms

  • Synonym: audio file - a computer file containing digitized sound
