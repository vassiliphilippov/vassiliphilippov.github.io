identity matrix

1. NOUN

/ aɪˈdɛntəti ˈmeɪtrɪks / (AmE), / aɪˈdɛntɪti ˈmætrɪks / (BrE)
A square matrix with ones on the diagonal and zeros elsewhere, serving as the multiplicative identity in matrix operations.
matrix with ones on the diagonal

Examples

Multiplying any matrix by the identity matrix leaves the original matrix unchanged.
Every square matrix has a corresponding identity matrix of the same size.
The identity matrix plays a crucial role in solving systems of linear equations.

Related Terms

  • Degree Lower: diagonal matrix - a matrix where all non-diagonal elements are zero
  • Degree Lower: zero matrix - a matrix in which all elements are zero

Etymology

Formed from 'identity' (from Latin 'identitas', sameness) and 'matrix' (Latin 'matrix': womb, source), referring to the property of not changing other matrices when multiplied.
