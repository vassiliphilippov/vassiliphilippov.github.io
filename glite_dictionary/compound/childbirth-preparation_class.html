childbirth-preparation class

1. NOUN

/ ˈtʃaɪldˌbɝθ prɪˈpɛrˌeɪʃən klæs / (AmE), / ˈtʃaɪldˌbɜːθ prɪˈpɛərəʃən klɑːs / (BrE)
a class for pregnant women that teaches breathing, concentration, and exercise techniques to use during labor
prenatal class teaching labor techniques

Examples

Sara enrolled in a childbirth-preparation class to learn breathing techniques for labor.
Many hospitals offer a childbirth-preparation class for expectant parents before delivery.
During the childbirth-preparation class, participants practiced relaxation exercises to manage pain.
