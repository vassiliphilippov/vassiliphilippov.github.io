schema definition

1. NOUN

/ ˈskiː.mə ˌdɛf.ɪˈnɪʃ.ən / (AmE), / ˈskiː.mə ˌdɛf.ɪˈnɪʃ.ən / (BrE)
a structured specification that defines the organization, structure, and constraints of information in a database or data system
specification of data structure or organization

Examples

The new database required a detailed schema definition to ensure proper data validation.
Developers updated the schema definition to accommodate additional user information fields.
A clear schema definition makes integrating different systems much more manageable.

Synset Examples

The project was streamlined thanks to superstruct's schema definitions.

Related Terms

  • Synonym: schema - a structured framework or plan; in computing, a formal description of a database's structure

Etymology

schema from Greek 'skhēma' (form, shape); definition from Latin 'definitio' (a bounding, a setting of limits).
