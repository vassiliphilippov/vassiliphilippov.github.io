secret code

1. NOUN

/ ˈsiːkrət koʊd / (AmE), / ˈsiːkrɪt kəʊd / (BrE)
a system of words, letters, symbols, or signals used to conceal the meaning of a message from all but those who are initiated
system for hiding a message's meaning

Examples

The spy wrote his report in a secret code to protect the information.
Students used a secret code to pass notes without teachers understanding them.
During the war, armies often communicated through secret code to prevent enemies from intercepting their plans.

Related Terms

  • Synonym: cipher - a system of secret writing
  • Synonym: cryptogram - a text written in code
  • Antonym: plain text - ordinary, unencoded writing
