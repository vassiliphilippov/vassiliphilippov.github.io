computer language

1. NOUN

/ kəmˈpjuːtər ˈlæŋɡwɪdʒ / (AmE), / kəmˈpjuːtə ˈlæŋɡwɪdʒ / (BrE)
a programming language designed for use on computers, used to control and communicate with computer systems
programming language for computers

Examples

Python is a versatile computer language widely used in web development and data analysis.
Learning a new computer language can open up many career opportunities in technology.
Different types of computer language are suited to various programming tasks and platforms.
