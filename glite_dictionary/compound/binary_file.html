binary file

1. NOUN

/ ˈbaɪ.nɛr.i faɪl / (AmE), / ˈbʌɪ.nər.i faɪl / (BrE)
a computer file that contains data in a format readable only by specific software or hardware, not as plain text
file containing non-text, machine-readable data

Examples

Be careful not to open a binary file in a text editor, as it may display as unreadable symbols.
The application can convert a binary file into a format you can use in Excel.
Transferring a binary file over the wrong protocol can corrupt its data.

Related Terms

  • Antonym: text file - a file containing only plain, human-readable text
