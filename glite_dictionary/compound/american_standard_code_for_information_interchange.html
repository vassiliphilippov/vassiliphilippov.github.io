American Standard Code for Information Interchange

1. NOUN

/ əˌmɛrɪkən ˈstændɚd ˌkoʊd fər ˌɪnfərˈmeɪʃn ˌɪntɚˈtʃeɪndʒ / (AmE), / əˌmɛrɪkən ˈstændəd ˌkəʊd fə ˌɪnfəˈmeɪʃən ˌɪntəˈtʃeɪndʒ / (BrE)
a standard 7-bit character code for the electronic exchange of information between computers, representing letters, digits, and control characters
standard binary computer character code (ASCII)

Examples

The American Standard Code for Information Interchange is essential for ensuring compatibility between computer systems.
Most text files on early computers used the American Standard Code for Information Interchange to represent characters.
He researched how the American Standard Code for Information Interchange encodes characters using only seven bits.

Related Terms

  • Synonym: ASCII - abbreviation for American Standard Code for Information Interchange, representing the same standard

Etymology

Formed from the full name, first published in 1963; commonly known as 'ASCII', from initials of its name.
