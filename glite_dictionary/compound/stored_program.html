stored program

1. NOUN

/ stɔrd ˈproʊɡræm / (AmE), / stɔːd ˈprəʊɡræm / (BrE)
A program that is held in a computer's memory so that it can be accessed and executed by the machine.
program stored in a computer's memory

Examples

The stored program concept revolutionized the way computers could be programmed and used.
In modern computers, any stored program can be modified or replaced as needed.
A stored program allows a computer to execute instructions directly from memory.

Related Terms

  • Synonym: stored-program computer - a computer that uses a stored program
  • Antonym: hardwired program - a program built directly into hardware rather than memory
