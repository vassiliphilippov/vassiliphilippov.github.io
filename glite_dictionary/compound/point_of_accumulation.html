point of accumulation

1. NOUN

/ pɔɪnt əv əˌkjuːmjəˈleɪʃən / (AmE), / pɔɪnt əv əˌkjuːmjʊˈleɪʃən / (BrE)
A value toward which a function approaches as the variable tends to infinity, especially in mathematical analysis.
mathematics: limiting value a function approaches

Examples

As x increases without bound, the point of accumulation of the function becomes evident.
Mathematicians use the term point of accumulation to describe a function's limiting behavior.
For this sequence, the only point of accumulation is zero.

Related Terms

  • Synonym: limit - the value a function approaches as input approaches some value
