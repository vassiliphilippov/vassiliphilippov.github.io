output file

1. NOUN

/ ˈaʊtpʊt faɪl / (AmE), / ˈaʊtpʊt faɪl / (BrE)
A computer file that stores data produced as the result of a process, device, or program.
file holding results from a program or process

Examples

The application saves the processed data in an output file located in your documents folder.
After running the simulation, make sure to check the output file for results and errors.
You can specify the name of the output file using a command-line option.

Related Terms

  • Antonym: input file - a file used to provide data to a device or program
