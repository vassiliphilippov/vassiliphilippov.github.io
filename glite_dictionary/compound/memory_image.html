memory image

1. NOUN

/ ˈmɛm.ɚi ˈɪm.ɪdʒ / (AmE), / ˈmɛm.əri ˈɪm.ɪdʒ / (BrE)
a mental picture recalled from past experience or memory
mental image from memory

Examples

The memory image of her childhood home was so vivid it felt real.
He tried to reconstruct the accident from his memory image of that day.
A therapist may help patients revisit difficult events by exploring their memory images.

Related Terms

  • Synonym: mental image - an image or picture formed in the mind
