input device

1. NOUN

/ ˈɪnˌpʊt dɪˈvaɪs / (AmE), / ˈɪn.pʊt dɪˈvaɪs / (BrE)
a device used to enter data or instructions into a computer or other electronic system
device for entering data into a computer

Examples

The keyboard is a common input device used to enter information into a computer.
Without an input device, it is impossible to communicate commands to most electronic systems.
A touchscreen is an input device that allows users to interact directly with what is displayed.

Related Terms

  • Antonym: output device - a device that receives data from a computer and presents it outside the system
  • Synonym: keyboard - commonly used input device for typing data
