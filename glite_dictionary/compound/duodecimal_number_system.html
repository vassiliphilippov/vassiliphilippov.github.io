duodecimal number system

1. NOUN

/ ˌduː.oʊˈdɛs.ə.məl ˈnʌm.bər ˌsɪs.təm / (AmE), / ˌdjuː.əˈdɛs.ɪ.məl ˈnʌm.bər ˌsɪs.təm / (BrE)
A numeral system based on twelve as its base, using twelve distinct digits.
numeral system with base twelve

Examples

The ancient Babylonians are known for their use of the duodecimal number system in mathematics.
Some argue that switching to a duodecimal number system would simplify certain calculations.
In a duodecimal number system, the digits run from 0 to 11 instead of 0 to 9.

Related Terms

  • Antonym: decimal system - numeral system with base ten
  • Degree Higher: sexagesimal system - numeral system with base sixty
  • Degree Lower: octal system - numeral system with base eight

Etymology

From Latin 'duodecim' meaning 'twelve' and English 'number system'.
