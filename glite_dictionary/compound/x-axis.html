x-axis

1. NOUN

/ ˈɛks ˌæksɪs / (AmE), / ˈɛks ˌaksɪs / (BrE)
the horizontal axis in a plane coordinate system, usually representing the independent variable
horizontal axis in a coordinate system

Examples

On the graph, time is represented along the x-axis.
To find the intersection, plot the points where the line crosses the x-axis.
The data is distributed evenly across the x-axis in this chart.

Related Terms

  • Antonym: y-axis - the vertical axis in a plane coordinate system

Etymology

From x (referring to the convention of using 'x' for the horizontal axis in mathematics) + axis.
