index register

1. NOUN

/ ˈɪnˌdɛks ˈrɛdʒɪstər / (AmE), / ˈɪndɛks ˈrɛdʒɪstə / (BrE)
(in computing) a register in a computer's CPU whose contents can be used to modify the address of an operand during program execution.
CPU register for address calculation

Examples

The index register was used to speed up memory access during the program's execution.
By changing the value stored in the index register, the loop efficiently processed each array element.
Modern processors often have more than one index register to handle complex addressing modes.

Related Terms

  • Synonym: register - a small amount of fast storage in a computer processor

Etymology

From 'index,' meaning indicator, and 'register,' meaning storage location in computing.
