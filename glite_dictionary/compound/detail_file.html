detail file

1. NOUN

/ ˈdiːteɪl faɪl / (AmE), / ˈdiːteɪl faɪl / (BrE)
a computer file containing data specific to a particular data processing task, often holding temporary or transient information
transient data file for a processing task

Examples

The backup system creates a separate detail file for each batch operation.
After processing, all detail files are deleted to free up memory.
The report generator reads data from the detail file before compiling the summary.

Related Terms

  • Antonym: master file - a principal, permanent data file in data processing
