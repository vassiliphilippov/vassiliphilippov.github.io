earned run

1. NOUN

/ ˈɝːnd ˌrʌn / (AmE), / ˈɜːnd ˌrʌn / (BrE)
a run in baseball that is scored without the aid of a fielding error or passed ball by the opposing team
baseball run not resulting from an error

Examples

The pitcher's earned run average dropped after his last two games.
He allowed only one earned run over seven innings, showing great control.
If not for the costly error, that would have been an earned run against him.

Related Terms

  • Antonym: unearned run - a run that scores due to a fielding error or passed ball
