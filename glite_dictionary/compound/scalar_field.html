scalar field

1. NOUN

/ ˈskeɪ.lɚ fiːld / (AmE), / ˈskeɪ.lə fiːld / (BrE)
A quantity defined at each point in space (or spacetime) that assigns a single scalar value to every point, used especially in mathematics and physics.
mathematical function assigning values to points in space

Examples

A temperature map is a classic example of a scalar field in physics.
The gravitational potential in space can be described by a scalar field.
In quantum field theory, the Higgs boson is associated with a fundamental scalar field.

Related Terms

  • Antonym: vector field - a quantity that assigns a vector to each point in space

Etymology

From 'scalar' (quantity described by a single value) and 'field' (mathematical function assigning values to each point in space).
