octal notation

1. NOUN

/ ˈɑk.təl noʊˈteɪ.ʃən / (AmE), / ˈɒk.təl nəʊˈteɪ.ʃən / (BrE)
A numerical system of notation that uses eight distinct digits, typically 0 to 7, to represent values.
numerical system using digits 0 to 7

Examples

In computer programming, file permissions are often expressed in octal notation.
The number 10 in octal notation is equal to 8 in decimal notation.
Early digital computers sometimes used octal notation for memory addressing.

Related Terms

  • Synonym: binary notation - numerical system using only digits 0 and 1
  • Synonym: decimal notation - numerical system using digits 0 to 9
  • Synonym: hexadecimal notation - numerical system using digits 0-9 and letters A-F

Etymology

From Latin 'octo' (eight) and 'notation'.
