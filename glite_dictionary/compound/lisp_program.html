LISP program

1. NOUN

/ lɪsp ˈproʊɡræm / (AmE), / lɪsp ˈprəʊɡram / (BrE)
A computer program written in the LISP programming language.
program written in LISP

Examples

He wrote a LISP program to automate the data analysis tasks.
The university teaches students how to create a basic LISP program in the introductory AI course.
Debugging a LISP program requires understanding both the syntax and the logic.

Related Terms

  • Synonym: program - a set of coded instructions that a computer can execute

Etymology

LISP: from 'LISt Processing', a computer programming language developed in 1958; program: from Greek 'programma', meaning 'proclamation, agenda'.
