code yellow

1. NOUN

/ koʊd ˈjɛloʊ / (AmE), / kəʊd ˈjɛləʊ / (BrE)
an alert or warning indicating a specific emergency situation, such as severe weather, a missing person, or another institutional emergency
emergency alert or warning level

Examples

The hospital declared a code yellow after detecting a chemical spill in the laboratory.
Schools announced a code yellow due to the approaching thunderstorm.
During last year's hurricane, the city was under a code yellow for several hours.

Synset Examples

During the drill, we practiced responses to a code yellow situation.
The weather service issued a code yellow for severe storms.
The school issued a code yellow alert for a missing student.

Related Terms

  • Degree Higher: code red - an emergency alert indicating a more severe or life-threatening situation
  • Synonym: code blue - an emergency alert, typically for a medical emergency such as cardiac arrest
