program line

1. NOUN

/ ˈproʊˌɡræm laɪn / (AmE), / ˈprəʊˌɡræm laɪn / (BrE)
(computing) a single line of code in a computer program.
a line of code in a program

Examples

Each program line must end with a semicolon in this language.
She spent hours debugging a single program line that caused the error.
Good programmers write comments beside each program line to explain its purpose.

Related Terms

  • Synonym: statement - an instruction in a programming language
