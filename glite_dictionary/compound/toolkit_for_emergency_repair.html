toolkit for emergency repair

1. NOUN

/ ˈtuːlˌkɪt fər ɪˈmɝːdʒənsi rɪˈpɛr / (AmE), / ˈtuːlˌkɪt fə ɪˈmɜːdʒənsi rɪˈpeə / (BrE)
a set of tools assembled for fixing urgent or unexpected problems
set of tools for emergency fixes
