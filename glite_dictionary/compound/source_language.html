source language

1. NOUN

/ ˈsɔrs ˌlæŋɡwɪdʒ / (AmE), / ˈsɔːs ˌlæŋɡwɪdʒ / (BrE)
a language from which a text, speech, or information is translated into another language
language to be translated from

Examples

The translator must have fluency in both the source language and the target language.
In this course, English will be treated as the source language for all translation exercises.
Understanding the cultural context of the source language is crucial for accurate translation.

Related Terms

  • Antonym: target language - the language into which a text is translated
