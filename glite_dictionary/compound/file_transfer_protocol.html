File Transfer Protocol

1. NOUN

/ faɪl ˈtræns.fɚ ˈproʊ.təˌkɑl / (AmE), / faɪl ˈtræns.fə ˈprəʊ.tə.kɒl / (BrE)
a standard network protocol used to transfer files from one computer to another over a network such as the internet
network protocol for transferring files

Examples

Many web developers use File Transfer Protocol to upload site files to a web server.
You need a client program that supports File Transfer Protocol to download large files safely.
Although File Transfer Protocol is convenient, it is not considered very secure for sensitive data.

Related Terms

  • Synonym: FTP - common abbreviation for File Transfer Protocol
  • Degree Higher: SFTP - Secure File Transfer Protocol, a more secure alternative to FTP
  • Synonym: HTTP - Hypertext Transfer Protocol, another protocol used for transferring data on the web

Etymology

Composed of 'file', 'transfer', and 'protocol'; widely known as FTP since the 1970s in computer science.
