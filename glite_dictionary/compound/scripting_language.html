scripting language

1. NOUN

/ ˈskrɪptɪŋ ˌlæŋɡwɪdʒ / (AmE), / ˈskrɪptɪŋ ˌlaŋɡwɪdʒ / (BrE)
a programming language designed to automate tasks by running scripts, which are interpreted rather than compiled
interpreted programming language for automation

Examples

Many web developers use a scripting language to automate tasks on websites.
Python is often considered both a general-purpose language and a scripting language.
The IT team wrote a scripting language program to handle data backups automatically.

Synset Examples

HTML is a scripting language, and it's the underlying language used on the web.

Related Terms

  • Synonym: programming language - a language used to write computer programs
