self-adapting program

1. NOUN

/ ˌsɛlf əˈdæptɪŋ ˈproʊɡræm / (AmE), / ˌsɛlf əˈdæptɪŋ ˈprəʊɡram / (BrE)
a computer program that can modify its performance or operation in response to changes in its environment
a program that adjusts to its environment

Examples

The new security system uses a self-adapting program to recognize unusual behavior patterns.
A self-adapting program can automatically update its parameters based on user input and device status.
Researchers are developing self-adapting programs to improve performance in unpredictable operating conditions.

Related Terms

  • Synonym: adaptive software - software that adapts its behavior to user or environmental changes
