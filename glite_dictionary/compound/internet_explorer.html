Internet Explorer

1. NOUN

/ ˌɪntɚnɛt ɪkˈsplɔrɚ / (AmE), / ˌɪntənet ɪkˈsplɔːrə / (BrE)
A web browser developed by Microsoft and released as part of the Microsoft Windows line of operating systems.
web browser by Microsoft

Examples

Many companies still use Internet Explorer for certain legacy applications.
Microsoft discontinued support for Internet Explorer in 2022.
You might experience issues if you try to access modern websites with Internet Explorer.

Related Terms

  • Synonym: browser - a program used to access the Internet
