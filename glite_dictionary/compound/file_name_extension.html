file name extension

1. NOUN

/ faɪl neɪm ɪkˈstɛnʃən / (AmE), / faɪl neɪm ɪkˈstɛnʃən / (BrE)
a group of characters, usually starting with a period, that follows a file name to indicate its format or type, commonly used in computing
suffix indicating file type (computing)

Examples

You need to add the correct file name extension to save the document as a PDF.
The system couldn't recognize the file name extension and refused to open the file.
Many viruses masquerade as familiar file name extensions to trick users into opening them.

Synset Examples

most applications provide extensions for the files they create
most BASIC files use the filename extension .BAS

Related Terms

  • Synonym: file extension - short form for file name extension
