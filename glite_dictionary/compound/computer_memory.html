computer memory

1. NOUN

/ kəmˈpjuːtɚ ˈmɛməri / (AmE), / kəmˈpjuːtə ˈmɛməri / (BrE)
an electronic device that stores data and instructions for use by a computer's processor
electronic device for storing computer data

Examples

The speed of computer memory can significantly affect the performance of your system.
Upgrading the computer memory allowed us to run more applications at once.
Modern laptops often use solid-state computer memory for faster data access.
