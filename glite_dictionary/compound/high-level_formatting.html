high-level formatting

1. NOUN

/ ˌhaɪ ˈlɛvəl ˈfɔrmætɪŋ / (AmE), / ˌhaɪ ˈlɛvəl ˈfɔːmætɪŋ / (BrE)
(computing) the process of setting up the root directory, file allocation tables, and other basic structures on a disk, making it ready for use by an operating system.
setup of disk file structures in computing
medical

Examples

It is essential to perform high-level formatting before using a new hard drive.
Mistakes during high-level formatting can render a disk unusable.
Unlike low-level formatting, high-level formatting prepares the disk's file system and directory structure.
