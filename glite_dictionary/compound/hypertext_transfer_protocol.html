Hypertext Transfer Protocol

1. NOUN

/ ˈhaɪ.pɚˌtɛkst ˈtræn.sfɚ prəˈtɑː.kɑl / (AmE), / ˈhaɪ.pəˌtɛkst ˈtræns.fə prəˈtəʊ.kɒl / (BrE)
a protocol used for transmitting hypertext (such as web pages) between web servers and browsers, utilizing TCP as the underlying transport
the main protocol for transferring web pages

Examples

Most websites rely on the Hypertext Transfer Protocol to deliver content to users' browsers.
The Hypertext Transfer Protocol forms the foundation for data communication on the World Wide Web.
Developers often use a secure version of the Hypertext Transfer Protocol, known as HTTPS, for privacy.

Related Terms

  • Synonym: HTTP - abbreviation for Hypertext Transfer Protocol

Etymology

From 'hypertext' + 'transfer' + 'protocol', first coined with the development of the World Wide Web in the late 20th century.
