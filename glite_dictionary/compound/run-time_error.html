run-time error

1. NOUN

/ ˈrʌn taɪm ˈɛr.ɚ / (AmE), / ˈrʌn taɪm ˈɛr.ər / (BrE)
An error that occurs during the execution of a program, typically due to invalid operations or logic mistakes that are not detectable until the program runs.
error occurring while a program runs

Examples

The program crashed due to a run-time error that occurred when it tried to divide by zero.
To prevent a run-time error, always check that variables have valid values before using them.
Uncaught exceptions can result in a run-time error that stops the execution of your software.

Related Terms

  • Antonym: syntax error - a grammatical mistake in the code, detected before running the program
  • Synonym: logic error - an error in a program's logic that produces incorrect results but does not prevent the program from running

Etymology

From 'run-time' (the period when a program is executing) and 'error' (a mistake or fault).
