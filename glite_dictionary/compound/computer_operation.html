computer operation

1. NOUN

/ kəmˈpjuːtər ˌɑːpəˈreɪʃən / (AmE), / kəmˈpjuːtə ˌɒpəˈreɪʃən / (BrE)
an elementary action that a computer is designed and built to execute, such as arithmetic, logic, or control functions
basic action performed by a computer

Examples

Every computer operation must be completed before the next instruction can be processed.
Addition and subtraction are typical examples of a computer operation in basic computing.
The speed of a processor determines how many computer operations it can perform per second.

Related Terms

  • Synonym: machine instruction - a basic command understood by a computer processor
