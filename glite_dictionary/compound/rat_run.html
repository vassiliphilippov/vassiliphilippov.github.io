rat run

1. NOUN

/ ræt ˌrʌn / (AmE), / ræt ˌrʌn / (BrE)
A narrow, secret, or frequently used route or path, originally referring to those made by rats but also used more generally.
narrow or secret pathway

Examples

The old alley behind the market became a well-used rat run for delivery bikes.
Children discovered a hidden rat run through the hedge to reach the park more quickly.
Investigators identified a rat run between abandoned buildings used by intruders.

Synset Examples

The rat run was evident after the building was abandoned.
After the storm, there's a rat run through the streets.
We saw a rat run when we opened the cupboard.

Etymology

From 'rat' (the animal) + 'run' (route frequented by an animal or person).
