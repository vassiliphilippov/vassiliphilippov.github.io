continuous integration

1. NOUN

/ kənˈtɪnjuəs ˌɪntəˈɡreɪʃən / (AmE), / kənˈtɪnjuːəs ˌɪntɪˈɡreɪʃən / (BrE)
the practice in software development of frequently merging code changes into a shared repository, often followed by automated testing and builds
frequent code merging in software development

Examples

Our engineering team uses continuous integration to catch bugs early in the development cycle.
Automated testing is a key component of continuous integration workflows.
Implementing continuous integration improved the team's productivity and code quality.

Synset Examples

The team relies on Jenkins for continuous integration.

Related Terms

  • Synonym: deployment - the process of delivering and installing software in a production environment
