systems software

1. NOUN

/ ˈsɪstəmz ˌsɔf(t)ˌwɛr / (AmE), / ˈsɪstəmz ˌsɒf(t)ˌwɛə / (BrE)
Software designed to control and manage the hardware and basic functions of a computer, including operating systems, compilers, and utility programs.
software that manages computer hardware and core operations

Examples

Modern computers require systems software like operating systems to function efficiently.
The developer specialized in creating systems software for medical devices.
Without systems software, users cannot interact directly with the computer's hardware.

Related Terms

  • Antonym: application software - software designed for end-users to perform specific tasks
