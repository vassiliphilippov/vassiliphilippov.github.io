bits per second

1. NOUN

/ bɪts pɚ ˈsekənd / (AmE), / bɪts pə ˈsekənd / (BrE)
A unit measuring the speed at which data is transmitted, equal to the number of bits transferred every second.
unit of data transfer rate

Examples

The internet connection has a speed of 100,000 bits per second.
Older modems operated at only 56,000 bits per second.
Data is transmitted between devices in millions of bits per second on modern networks.

Related Terms

  • Synonym: baud - unit of symbol rate in data transmission
  • Synonym: bytes per second - unit of data transfer rate measured in bytes
