reduced instruction set computer

1. NOUN

/ rɪˈdjuːst ɪnˈstrʌkʃən sɛt kəmˈpjuːtər / (AmE), / rɪˈdjuːst ɪnˈstrʌkʃən sɛt kəmˈpjuːtə / (BrE)
a type of computer architecture that uses a relatively small and simple set of instructions for processing data, designed to increase efficiency and performance
computer with a simplified instruction set

Examples

Apple used a reduced instruction set computer architecture in developing its early desktop computers.
A reduced instruction set computer is designed to execute operations more efficiently than traditional architectures.
The transition to reduced instruction set computer processors improved performance in many modern devices.

Related Terms

  • Synonym: RISC - abbreviation for reduced instruction set computer
  • Antonym: complex instruction set computer - computer with a larger and more complex set of instructions

Etymology

Formed from 'reduced' + 'instruction set' + 'computer', describing a computer with a smaller set of machine instructions.
