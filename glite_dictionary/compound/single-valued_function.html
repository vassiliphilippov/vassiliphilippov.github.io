single-valued function

1. NOUN

/ ˈsɪŋɡəlˌvæl.jud ˈfʌŋk.ʃən / (AmE), / ˈsɪŋ.ɡl̩ˌvæl.juːd ˈfʌŋk.ʃən / (BrE)
(mathematics) a function in which each element of the domain is associated with exactly one element in the range.
function assigning one output per input (math)

Examples

Every continuous real function is a single-valued function.
In mathematics, a single-valued function assigns only one output to each input.
The concept of single-valued function is fundamental in calculus and analysis.

Related Terms

  • Synonym: function - mathematical relation assigning exactly one output for each input
  • Antonym: multi-valued function - a relation where one input can correspond to multiple outputs in math
