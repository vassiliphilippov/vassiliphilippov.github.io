small computer system interface

1. NOUN

/ smɔl kəmˈpjuːt̬ɚ ˈsɪstəm ˈɪntɚˌfeɪs / (AmE), / smɔːl kəmˈpjuːtə ˈsɪstəm ˈɪntəˌfeɪs / (BrE)
a standard interface for connecting computers and peripheral devices, used to enable communication between them
standard interface for connecting computers and devices

Examples

The external hard drive connects to the server via a small computer system interface port.
Early desktop computers often used a small computer system interface to link to printers and scanners.
She needed to install a small computer system interface card to support her new backup device.

Related Terms

  • Synonym: SCSI - abbreviation of small computer system interface

Etymology

Acronym SCSI, from the full phrase 'Small Computer System Interface', coined in the early 1980s in computer engineering.
