binary numeration system

1. NOUN

/ ˈbaɪnɛri nuˌmɛrɛɪʃən ˈsɪstəm / (AmE), / ˈbaɪnəri njuːməˌreɪʃən ˈsɪstəm / (BrE)
A number system that uses only two digits, 0 and 1, with a base (radix) of two.
number system using 0 and 1; base two

Examples

Computers process data using the binary numeration system, which relies solely on the digits 0 and 1.
Learning to convert decimal numbers to the binary numeration system is essential for understanding digital electronics.
The binary numeration system underpins all modern computer operations and data storage.

Related Terms

  • Synonym: binary system - number system using base two, only digits 0 and 1
  • Antonym: decimal numeration system - number system using base ten, digits 0 to 9

Etymology

From Latin 'bini' (two together) and Latin 'numeratio' (counting).
