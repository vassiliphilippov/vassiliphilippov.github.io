monadic operation

1. NOUN

/ moʊˈnædɪk ˌɑːpəˈreɪʃən / (AmE), / məˈnadɪk ˌɒpəˈreɪʃən / (BrE)
An operation that takes exactly one operand as its input, commonly used in mathematics, logic, or computing.
operation with one operand

Examples

The mathematical function f(x) = x^2 is a monadic operation because it requires only one input.
In some programming languages, the negation operator is implemented as a monadic operation.
Learning about monadic operations helps students understand more complex binary and ternary operations.

Related Terms

  • Synonym: unary operation - an operation with only one operand

Etymology

From Greek 'monas' (unit, alone) and Latin 'operatio' (work, action).
