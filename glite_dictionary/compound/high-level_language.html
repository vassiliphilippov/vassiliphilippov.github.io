high-level language

1. NOUN

/ ˌhaɪ ˈlɛvəl ˈlæŋɡwɪdʒ / (AmE), / ˌhaɪ ˈlɛvəl ˈlaŋɡwɪdʒ / (BrE)
a programming language designed to be easy for humans to use, requiring little detailed knowledge of the computer's hardware
programming language abstracted from hardware details

Examples

Python is a popular high-level language used in web development and data science.
Because it's easier to read and write, a high-level language is often recommended for beginners.
The compiler translates code written in a high-level language into machine instructions the computer can execute.

Related Terms

  • Antonym: low-level language - a programming language that provides little or no abstraction from a computer's instruction set architecture
