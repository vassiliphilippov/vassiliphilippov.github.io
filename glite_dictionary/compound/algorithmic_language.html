algorithmic language

1. NOUN

/ ˌælɡəˈrɪðmɪk ˈlæŋɡwɪdʒ / (AmE), / ˌælɡəˈrɪðmɪk ˈlaŋɡwɪdʒ / (BrE)
an artificial language created to express algorithms, particularly for programming and computation
programming language for algorithms

Examples

Learning an algorithmic language is essential for anyone who wants to become a computer programmer.
FORTRAN was one of the earliest algorithmic languages developed for scientific calculations.
Many computer science courses start by teaching students an algorithmic language like Python or Java.

Related Terms

  • Synonym: programming language - formal language for instructing computers

Etymology

From 'algorithm' (from Arabic 'al-Khwarizmi', name of a Persian mathematician) + 'language'.
