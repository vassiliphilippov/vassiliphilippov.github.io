random-access memory

1. NOUN

/ ˈrændəm ˌæksɛs ˈmɛməri / (AmE), / ˈrændəm ˌæksɛs ˈmɛməri / (BrE)
A type of computer memory that allows data to be read or written in any order, used by programs while the computer is operating.
type of computer memory (RAM)

Examples

The computer has 16 gigabytes of random-access memory for handling multiple applications at once.
When you turn off your computer, all the data stored in random-access memory is lost.
Upgrading your random-access memory can significantly improve system performance.

Related Terms

  • Synonym: RAM - abbreviation for random-access memory

Etymology

Formed from 'random access' (referring to non-sequential data access) and 'memory'.
