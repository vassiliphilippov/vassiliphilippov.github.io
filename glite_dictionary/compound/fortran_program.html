FORTRAN program

1. NOUN

/ ˈfɔr.træn ˌproʊ.ɡræm / (AmE), / ˈfɔː.træn ˌprəʊ.ɡræm / (BrE)
a computer program written in the FORTRAN programming language
a program written in FORTRAN

Examples

The scientist developed a complex FORTRAN program to process the experimental data.
Many legacy systems still rely on old FORTRAN programs for their computations.
She learned how to debug a FORTRAN program during her computer science course.

Related Terms

  • Synonym: COBOL program - a program written in COBOL
  • Synonym: Pascal program - a program written in Pascal

Etymology

FORTRAN: from 'Formula Translation', a programming language developed in the 1950s.
