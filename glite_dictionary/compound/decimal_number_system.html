decimal number system

1. NOUN

/ ˈdɛsɪməl ˈnʌmbɚ ˌsɪstəm / (AmE), / ˈdɛsɪməl ˈnʌmbə ˌsɪstəm / (BrE)
A numeration system based on powers of ten, using digits 0 to 9.
number system using base ten

Examples

The decimal number system is used universally in everyday life for counting and calculations.
When learning about computers, students often compare the decimal number system with binary and hexadecimal systems.
The invention of the decimal number system greatly simplified arithmetic operations.

Related Terms

  • Antonym: binary number system - a numeration system using base two, digits 0 and 1
  • Antonym: octal number system - a numeration system using base eight, digits 0–7
  • Antonym: hexadecimal number system - a numeration system using base sixteen, digits 0–9 and A–F
