common run

1. NOUN

/ ˈkɑmən rʌn / (AmE), / ˈkɒmən rʌn / (BrE)
a route or path that is frequently used by people, vehicles, or animals
a frequently used route

Examples

We followed the common run through the woods to reach the village.
During rush hour, the common run to work is always crowded.
Hikers should avoid the common run if they want more solitude.

Synset Examples
