library program

1. NOUN

/ ˈlaɪˌbrɛri ˈproʊɡræm / (AmE), / ˈlaɪ.brə.ri ˈprəʊ.ɡræm / (BrE)
a computer program that is stored within a program library and usually designed to be used by other programs
a program stored in a program library

Examples

The developer reused a library program to speed up the project's completion.
Each library program handles a specific task, such as sorting or searching data.
You must link the library program before calling its functions in your application.

Related Terms

  • Degree Higher: library - a collection of programs or routines for computer use
  • Synonym: subroutine - a set of instructions in a computer program; a routine designed to perform a frequently used operation
