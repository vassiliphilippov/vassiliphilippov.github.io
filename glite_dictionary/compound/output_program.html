output program

1. NOUN

/ ˈaʊtpʊt ˌproʊɡræm / (AmE), / ˈaʊtpʊt ˌprəʊɡræm / (BrE)
a utility program that manages or organizes the results produced by a computer
utility that manages computer output

Examples

The output program formats the data before it is sent to the printer.
She installed an output program to convert text files to PDFs automatically.
Some systems require a specialized output program to handle different report types.

Related Terms

  • Synonym: utility program - software designed to perform a specific task, usually related to managing computer resources
