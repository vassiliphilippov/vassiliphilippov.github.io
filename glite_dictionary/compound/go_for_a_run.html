to go for a run

1. VERB

/ ˈɡoʊ fəɹ ə ˈɹʌn / (AmE), / ˈɡəʊ fəɹ ə ˈɹʌn / (BrE)
to take part in running for exercise or pleasure, especially as a regular activity
take part in running as exercise

Examples

I like going for a run in the park before breakfast.
Whenever he's stressed, he finds that going for a run helps clear his mind.
They usually go for a run together every Saturday morning.

Synset Examples

She loves going for runs at dawn when it's quiet.

Related Terms

  • Synonym: jog - to run at a slow and steady pace for exercise
