Dewey decimal classification

1. NOUN

/ ˈduːi ˈdɛsɪməl ˌklæsɪfɪˈkeɪʃən / (AmE), / ˈdjuːi ˈdɛsɪməl ˌklɑːsɪfɪˈkeɪʃən / (BrE)
a system for organizing books and other materials in libraries by dividing nonfiction subjects into numbered categories, using a three-digit number and decimals for more specific topics
library classification system using numbers

Examples

The library uses the Dewey decimal classification to arrange its nonfiction books.
Students learn to find resources faster by understanding the Dewey decimal classification system.
Many public libraries around the world organize their collections using the Dewey decimal classification.

Etymology

Named after Melvil Dewey (1851–1931), American librarian, who developed the system.
