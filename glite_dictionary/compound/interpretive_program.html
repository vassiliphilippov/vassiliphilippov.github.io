interpretive program

1. NOUN

/ ɪnˈtɜːrprətɪv ˈproʊɡræm / (AmE), / ɪnˈtɜːprətɪv ˈprəʊɡræm / (BrE)
(computing) a computer program that translates and executes source code statements one line at a time, rather than compiling the entire code before execution.
computer program that executes code line by line

Examples

The software was designed as an interpretive program to facilitate rapid testing of code.
Unlike compiled languages, an interpretive program executes instructions directly, which can simplify debugging.
Basic is often run using an interpretive program instead of being compiled.

Related Terms

  • Synonym: interpreter - a program that translates and runs code line by line
  • Antonym: compiler - a program that translates source code into executable code as a whole
