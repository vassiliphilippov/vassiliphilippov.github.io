object-oriented programming language

1. NOUN

/ ˌɑbdʒɛkt ˈɔːriˌɛntɪd ˈproʊɡræmɪŋ ˌlæŋɡwɪdʒ / (AmE), / ˌɒbdʒɛkt ˈɔːriˌɛntɪd ˈprəʊɡræmɪŋ ˌlæŋɡwɪdʒ / (BrE)
a programming language that enables the association of data structures with the procedures (methods) that operate on them, supporting concepts such as objects, classes, inheritance, and encapsulation
programming language supporting objects and classes

Examples

Java is a popular object-oriented programming language used to build enterprise applications.
Learning an object-oriented programming language can make it easier to write modular and maintainable code.
Python, an object-oriented programming language, allows developers to define their own classes and objects.

Synset Examples

C++ is an object-oriented programming language that is an extension of C

Related Terms

  • Synonym: object-oriented programming - a programming paradigm based on objects containing data and methods

2. NOUN

/ ˈɑb.d͡ʒɛkt ˌɔːr.i.ɛn.tɪd ˈproʊ.ɡræm.ɪŋ ˈlæŋ.ɡwɪdʒ / (AmE), / ˈɒb.dʒɛkt ˌɔː.ri.ɛn.tɪd ˈprəʊ.ɡræm.ɪŋ ˈlæŋ.ɡwɪdʒ / (BrE)
a programming language in which each data structure is associated with a set of procedures, allowing the use of objects that encapsulate data and behavior
programming language supporting objects and procedures

Examples

Java is a popular object-oriented programming language used for building a wide range of applications.
Python was not originally an object-oriented programming language, but now supports object-oriented features extensively.
Smalltalk is considered the pioneering object-oriented programming language that influenced many languages today.

Synset Examples

C++ is an object-oriented programming language that is an extension of C

Related Terms

  • Synonym: object-oriented - relating to a programming paradigm based on objects
  • Antonym: procedural programming language - a language focusing on sequences of procedures or routines

Etymology

From 'object-oriented' (a paradigm using objects) + 'programming language'.
