postal code

1. NOUN

/ ˈpoʊstl ˌkoʊd / (AmE), / ˈpəʊstl ˌkəʊd / (BrE)
A code consisting of letters and numbers added to a postal address to help in the sorting and delivery of mail.
address code for sorting mail

Examples

Please include your postal code when filling out the form.
The package was delayed due to an incorrect postal code on the label.
In Canada, the postal code format includes both letters and numbers.

Related Terms

  • Synonym: zip code - postal code system used in the United States
  • Synonym: postcode - term for postal code in the United Kingdom and some other countries
