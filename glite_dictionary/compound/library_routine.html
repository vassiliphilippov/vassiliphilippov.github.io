library routine

1. NOUN

/ ˈlaɪˌbrɛri ruːˈtiːn / (AmE), / ˈlaɪ.brəri ruːˈtiːn / (BrE)
a reusable program or subroutine that is stored in a program library for use in many different applications
reusable program subroutine

Examples

The developer called a library routine to perform the sorting operation efficiently.
Many library routines are carefully tested and documented to ensure reliability.
Using a standard library routine can save time and reduce programming errors.

Related Terms

  • Synonym: subroutine - a sequence of program instructions that performs a specific task, packaged as a unit
  • Synonym: function - a named section of a code that performs a specific task
