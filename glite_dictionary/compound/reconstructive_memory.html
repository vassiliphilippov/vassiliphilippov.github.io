reconstructive memory

1. NOUN

/ ˌriː.kənˈstrʌk.tɪv ˈmɛm.ə.ri / (AmE), / ˌriː.kənˈstrʌk.tɪv ˈmem.əri / (BrE)
a type of memory retrieval in which abstract features or general themes are stored and then used to construct specific details during recall, rather than the exact original experience being reproduced
memory recall where details are reconstructed

Examples

Studies in psychology show that reconstructive memory often leads to people recalling events inaccurately.
Eyewitness testimonies can be unreliable because reconstructive memory fills in missing information with assumptions or suggestions.
The theory of reconstructive memory explains why two people might remember the same incident differently.

Related Terms

  • Synonym: constructive memory - memory concept focusing on building or creating recall from fragments

Etymology

From 'reconstructive' (from re- + Latin 'construere' to build) and 'memory' (from Latin 'memoria', remembrance).
