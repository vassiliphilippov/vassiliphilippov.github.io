data input device

1. NOUN

/ ˈdeɪtə ˈɪnpʊt dɪˌvaɪs / (AmE), / ˈdeɪtə ˈɪnpʊt dɪˌvʌɪs / (BrE)
a hardware device used to enter data into a computer or similar system
hardware for entering data into a computer

Examples

A keyboard is the most common data input device used in offices around the world.
The technician replaced the faulty data input device with a new touchscreen panel.
Modern smartphones combine several data input devices such as cameras and microphones in a single unit.

Related Terms

  • Synonym: input device - any device used to provide data and control signals to an information processing system
  • Antonym: output device - a device that receives data from a computer and presents it in a readable or usable form
