computer address

1. NOUN

/ kəmˈpjuːtɚ əˈdrɛs / (AmE), / kəmˈpjuːtə əˈdrɛs / (BrE)
a code or set of characters that specifies the location of data or a device in a computer system
identifier for location of data in computing

Examples

Each file on the server can be reached using its unique computer address.
The program crashed when it tried to access an invalid computer address.
Modern networks rely on accurate computer addresses to transfer data quickly and securely.

Related Terms

  • Synonym: memory address - a specific location in computer memory identified by a unique number or code
  • Degree Lower: IP address - a unique string of numbers separated by periods that identifies each computer using the Internet Protocol to communicate over a network
