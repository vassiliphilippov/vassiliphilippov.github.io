unary numeral system

1. NOUN

/ ˈjuːˌnɛri ˈnuː.mɚ.əl ˈsɪs.təm / (AmE), / ˈjuː.nəri ˈnjuː.mə.rəl ˈsɪs.təm / (BrE)
A numeral system in which each natural number is represented by repeating a single symbol that stands for one, as many times as the value of the number.
numeral system using repeated single symbols

Examples

In the unary numeral system, the number five is represented as '11111'.
The unary numeral system is simple but inefficient for large numbers.
Ancient tally marks are an example of the unary numeral system in use.

Related Terms

  • Degree Higher: binary numeral system - numeral system using base 2, digits 0 and 1
  • Degree Higher: decimal numeral system - numeral system using base 10, digits 0 to 9

Etymology

From Latin 'unarius' (consisting of one), used in combination with 'numeral system'.
