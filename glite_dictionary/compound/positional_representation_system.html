positional representation system

1. NOUN

/ pəˈzɪʃənəl ˌrɛprɪˈzɛnˌteɪʃən ˈsɪstəm / (AmE), / pəˈzɪʃənəl ˌrɛprɪˈzɛnˌteɪʃən ˈsɪstəm / (BrE)
A numeration system in which the value of each digit is determined by its position within the number.
numeration system where digit position determines value

Examples

The decimal system is the most commonly used positional representation system in everyday life.
In a positional representation system, the same digit can represent different values depending on where it appears.
The binary positional representation system is fundamental to modern computing.

Related Terms

  • Synonym: numeral system - a writing system for expressing numbers
