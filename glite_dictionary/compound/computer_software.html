computer software

1. NOUN

/ kəmˈpjuːtər ˈsɔːf(t)wɛr / (AmE), / kəmˈpjuːtə ˈsɒf(t)wɛə / (BrE)
The programs, procedures, and related documentation that control or assist the operation of a computer system, stored electronically.
programs and data for computers

Examples

Many businesses rely on computer software to automate their operations.
The latest computer software update fixed several security issues.
Learning new computer software can improve your productivity.

Synset Examples

the market for software is expected to expand

Related Terms

  • Synonym: software - programs and instructions used by a computer
  • Antonym: hardware - the physical components of a computer system
