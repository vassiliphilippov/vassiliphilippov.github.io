object program

1. NOUN

/ ˈɑbdʒɛkt ˈproʊˌɡræm / (AmE), / ˈɒbdʒɛkt ˈprəʊɡræm / (BrE)
a computer program that has been fully compiled or assembled and is ready to be loaded and executed by a computer
compiled program ready for execution

Examples

After compiling the source code, the computer produced an object program ready for execution.
The object program must be loaded into memory before it can run.
Errors during linking may prevent an object program from functioning correctly.

Related Terms

  • Synonym: object code - the output of a compiler or assembler, not yet linked
