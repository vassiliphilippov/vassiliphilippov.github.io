translating program

1. NOUN

/ ˈtrænzˌleɪtɪŋ ˈproʊɡræm / (AmE), / ˈtrænzˌleɪtɪŋ ˈprəʊɡræm / (BrE)
A computer program that converts code written in one programming language into another.
program that translates programming languages

Examples

The new translating program efficiently converts Python code into C++.
Developers often use a translating program to migrate software to a different platform.
She designed a translating program that supports multiple source and target languages.

Related Terms

  • Synonym: compiler - a program that translates source code into machine code
  • Synonym: interpreter - a program that executes instructions written in a programming language
