hierarchical data structure

1. NOUN

/ haɪˈɝːˌɑrkɪkəl ˈdeɪtə ˈstrʌktʃɚ / (AmE), / haɪˈɒːrˌɑːkɪkəl ˈdeɪtə ˈstrʌktʃə / (BrE)
a system of organizing data with multiple levels, where each level is subordinate to the one above, often resembling a tree
multi-level data organizing system

Examples

The company's new database uses a hierarchical data structure to store employee records efficiently.
File systems often rely on a hierarchical data structure with folders and subfolders.
Understanding a hierarchical data structure is essential for designing scalable software architectures.

Related Terms

  • Synonym: tree structure - type of data structure with a hierarchical organization resembling a tree
