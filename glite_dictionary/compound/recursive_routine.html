recursive routine

1. NOUN

/ rɪˈkɝːsɪv ruːˈtiːn / (AmE), / rɪˈkɜːsɪv ruːˈtiːn / (BrE)
a function or procedure in a computer program that calls itself in its definition or execution
computer routine that calls itself

Examples

The factorial function is often implemented using a recursive routine.
If the base case is not defined, a recursive routine can result in infinite loops.
Many programming languages support the creation of a recursive routine for solving complex problems.

Related Terms

  • Antonym: iteration - repetition of a process in a loop, alternative to recursion
  • Synonym: recursion - the process of a function calling itself

Etymology

From Latin 'recurrere' (to run back) via 'recursion' + 'routine' (regular course of procedure).
