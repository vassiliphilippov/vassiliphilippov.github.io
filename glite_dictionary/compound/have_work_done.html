to have work done

1. VERB

/ hæv wɝːk dʌn / (AmE), / hæv wɜːk dʌn / (BrE)
to arrange for a specific task to be completed by someone else, especially by hiring a professional or service
arrange for a task to be completed by another

Examples

I'm planning to have work done on the kitchen this month.
She had to have work done on her house after the storm.
We usually have work done by professionals to ensure it's done well.

Synset Examples

I need to have work done on my car.
They will have work done on the landscaping next week.
