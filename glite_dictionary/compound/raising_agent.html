raising agent

1. NOUN

/ ˈreɪ.zɪŋ ˌeɪ.dʒənt / (AmE), / ˈreɪ.zɪŋ ˌeɪ.dʒənt / (BrE)
A substance added to dough or batter to make it rise by producing gas, resulting in a lighter and softer texture.
substance causing dough to rise

Examples

Always check the label to see which raising agent is used in your baking powder.
Soda bread is made without yeast, using baking soda as the raising agent.
Yeast is the traditional raising agent for making homemade bread.

Synset Examples

I know I can use chemical leaveners, but I think that using yeast as a leavening agent is more healthy.

Related Terms

  • Synonym: leavening agent - substance used to make baked goods rise
