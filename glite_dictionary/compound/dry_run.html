dry run

1. NOUN

/ ˈdraɪ ˌrʌn / (AmE), / ˈdraɪ ˌrʌn / (BrE)
a practice session or rehearsal to prepare for an actual event or performance
practice session before an event

Examples

The crew held a dry run of the wedding ceremony to make sure everything would go smoothly.
Before launching the new system, the team did a dry run to test for any issues.
The actors gathered for a final dry run the night before the play's opening.

Synset Examples

he missed too many rehearsals
a rehearsal will be held the day before the wedding

Related Terms

  • Synonym: rehearsal - a session of practice before a performance

2. NOUN

/ ˈdraɪ ˌrʌn / (AmE), / ˈdraɪ ˌrʌn / (BrE)
an instance of rehearsal or practice for a procedure or event, to ensure readiness or avoid mistakes
practice or rehearsal for a procedure

Examples

Before the product launch, the team conducted a dry run to identify possible issues.
Let's do a dry run of the concert so we catch any technical problems.
The firefighters held a dry run to prepare for emergency situations.

Synset Examples

He suggested a dry run to avoid mistakes during the live event.
The team planned a dry run to ensure everything worked smoothly.
Let's do a dry run of the presentation before the big day.

Related Terms

  • Synonym: rehearsal - a practice or trial performance of a play or other work
  • Synonym: trial run - a test or rehearsal of a procedure or process

Etymology

Originally from the idea of running a process or machine without water (i.e., a test run).
