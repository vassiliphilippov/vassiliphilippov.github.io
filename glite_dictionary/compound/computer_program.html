computer program

1. NOUN

/ kəmˈpjutər ˈproʊɡræm / (AmE), / kəmˈpjuːtə ˈprəʊɡræm / (BrE)
A set of instructions written to be executed by a computer to perform a specific task or solve a problem.
set of instructions for a computer

Examples

She wrote a computer program to organize the data automatically.
Installing a new computer program can sometimes require administrator access.
Every computer program must be thoroughly tested before it is released to users.

Synset Examples

the program required several hundred lines of code

Related Terms

  • Synonym: software - programs and other operating information used by a computer
