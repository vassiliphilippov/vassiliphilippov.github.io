real time

1. NOUN

/ ˈriːəl taɪm / (AmE), / ˈrɪəl taɪm / (BrE)
the actual time during which a process or event occurs, especially when information is updated instantly as events happen
actual time an event occurs

Examples

The software allows you to track your expenses in real time.
Stock market data is streamed in real time for active traders.
We monitored the patient's heart rate in real time during the experiment.

Synset Examples

information is updated in real time

Related Terms

  • Antonym: delayed - not happening immediately; opposite of real time

2. NOUN

/ ˈriːl ˌtaɪm / (AmE), / ˈriːəl ˌtaɪm / (BrE)
the actual time during which an event or process takes place, especially as measured by a computer system; the immediate processing of data without noticeable delay
actual time in which a process occurs, especially in computing

Examples

The new software processes transactions in real time, allowing customers to see updates instantly.
Weather forecasts are now updated in real time using satellite data.
Streaming services aim to deliver media in real time with minimal buffering.

Related Terms

  • Antonym: delay - the period of waiting before an event or process is completed
