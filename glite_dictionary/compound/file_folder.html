file folder

1. NOUN

/ ˈfaɪl ˌfoʊldər / (AmE), / ˈfaɪl ˌfəʊldə / (BrE)
a folded piece of stiff paper or thin cardboard used to hold and organize loose papers, typically in a filing cabinet or drawer
folder for organizing papers

Examples

She put all the important documents into a file folder before storing them in the cabinet.
Each student received a file folder to keep their assignments organized throughout the semester.
Please label the file folder with the client's name for easy identification.

Related Terms

  • Synonym: folder - a folded cover or large envelope for holding papers
