sanitary code

1. NOUN

/ ˈsænəˌtɛri koʊd / (AmE), / ˈsænɪtəri kəʊd / (BrE)
a set of official regulations established by authorities to ensure public health standards, especially concerning sanitation, plumbing, and hygiene
official public health regulations
legal

Examples

Local businesses must comply with the sanitary code to ensure customer safety.
The sanitary code requires regular inspections of plumbing and waste systems.
After several violations of the sanitary code, the restaurant was temporarily closed.

Related Terms

  • Synonym: health code - regulations relating to public health standards
  • Synonym: building code - rules governing the construction and occupancy of buildings
