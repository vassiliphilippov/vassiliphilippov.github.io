split run

1. NOUN

/ splɪt rʌn / (AmE), / splɪt rʌn / (BrE)
a print run of a newspaper or magazine in which some articles or advertisements are changed to create different editions for distribution in different regions or to different audiences
a print run with content changes for different editions

Examples

The magazine used a split run to tailor advertisements for readers in Europe and North America.
A split run allows publishers to target specific content to different demographic groups.
By producing a split run, the newspaper could offer localized news coverage for each city.
