IO device

1. NOUN

/ aɪ oʊ dɪˌvaɪs / (AmE), / aɪ əʊ dɪˌvaɪs / (BrE)
A piece of equipment that allows data to be input into or output from a computer or similar device.
device for input/output of data

Examples

The keyboard is a common IO device used to enter commands into a computer.
You must connect an IO device for data transfer to work.
Technicians tested each IO device before installing them on the network.

Related Terms

  • Synonym: input device - a device used to send data to a computer
  • Synonym: output device - a device used to receive data from a computer

Etymology

IO is an abbreviation for 'input/output', used in computing terminology.
