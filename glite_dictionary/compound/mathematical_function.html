mathematical function

1. NOUN

/ məˌθæmətɪkəl ˈfʌŋkʃən / (AmE), / məˌθæməˈtɪkəl ˈfʌŋkʃən / (BrE)
a relation in which each element of a given set (the domain) is associated with exactly one element of another set (the range)
specific mapping in mathematics

Examples

The sine function is a classic example of a mathematical function.
Every mathematical function has a domain and a range.
Defining a mathematical function precisely is crucial in calculus.

Related Terms

  • Synonym: function - a relation in mathematics associating one value to another
