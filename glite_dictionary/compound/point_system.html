point system

1. NOUN

/ pɔɪnt ˈsɪstəm / (AmE), / pɔɪnt ˈsɪstəm / (BrE)
A system for classifying the sizes of type or print using multiples of a standardized unit called the point.
system for standardizing type sizes in points

Examples

The point system ensures that fonts are produced in consistent and precise sizes.
Many modern publishers use the point system to measure both text and spacing.
Before the point system, type sizes often varied from one foundry to another.

2. NOUN

/ pɔɪnt ˈsɪstəm / (AmE), / pɔɪnt ˈsɪstəm / (BrE)
a system of writing or printing that uses patterns of raised dots, often read by touch, for communication, especially by people who are visually impaired
system using raised dots for tactile reading

Examples

The point system allows blind individuals to read books independently by feeling the patterns of dots.
Braille is the most widely used point system for communication among the visually impaired.
She learned to read with the point system after losing her sight in childhood.

Related Terms

  • Synonym: Braille - a tactile writing system using raised dots, used by the blind and visually impaired

3. NOUN

/ pɔɪnt ˈsɪstəm / (AmE), / pɔɪnt ˈsɪstəm / (BrE)
a method of evaluation in which points are awarded or deducted according to set rules or criteria
system awarding points by rules

Examples

The driver's license point system penalizes repeat traffic offenders.
Our school uses a point system to determine honor roll eligibility.
The tournament adopted a new point system to rank the teams fairly.
