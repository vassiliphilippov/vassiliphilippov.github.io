source program

1. NOUN

/ sɔrs ˈproʊɡræm / (AmE), / sɔːs ˈprəʊɡræm / (BrE)
A program written in a human-readable programming language, which is then translated into machine code for execution by a computer.
program written in a programming language before translation

Examples

The compiler checks the source program for errors before generating executable code.
Modifying the source program allows developers to fix bugs or add new features.
Her assignment was to write a source program in Python that sorts a list of numbers.

Related Terms

  • Antonym: object program - machine-code version of a program generated from a source program
