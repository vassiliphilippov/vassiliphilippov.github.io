user interface

1. NOUN

/ ˈjuːzər ˈɪntərˌfeɪs / (AmE), / ˈjuːzə ˈɪntəfeɪs / (BrE)
in computing, the means by which a user interacts with and controls a computer program or system, usually through visual displays and controls
means for interacting with a computer or program

Examples

The new software features a more intuitive user interface designed for beginners.
Designers spent months improving the user interface to enhance usability.
A poorly designed user interface can make even simple tasks frustrating.

Related Terms

  • Degree Higher: graphical user interface - a type of user interface using graphical elements
