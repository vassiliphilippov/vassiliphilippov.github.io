inverse function

1. NOUN

/ ˈɪnˌvɝs ˈfʌŋkʃən / (AmE), / ˈɪnvɜːs ˈfʌŋkʃən / (BrE)
a function that reverses the effect of another function, so that applying one after the other returns the original value; if f(x) = y, then its inverse function g satisfies g(y) = x
function that undoes another function

Examples

The logarithm is the inverse function of the exponential function.
If f(x) = 2x + 3, then its inverse function undoes this operation.
Solving equations often involves finding the inverse function of a given function.

Related Terms

  • Synonym: function - a mathematical relationship in which each input has a single output

Etymology

From Latin 'inversus' (turned upside down, reversed) + 'function'.
