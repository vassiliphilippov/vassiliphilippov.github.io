mesh topology

1. NOUN

/ mɛʃ təˈpɑlədʒi / (AmE), / mɛʃ təˈpɒlədʒi / (BrE)
A network layout in which each component is directly connected to every other component.
network design with direct connections between all nodes

Examples

A mesh topology ensures that every device in the network can communicate directly with every other device.
Many modern wireless networks use a mesh topology to increase reliability and coverage.
In a mesh topology, the failure of a single connection does not disrupt the entire network.

Related Terms

  • Antonym: star topology - network design with a central connection point
  • Antonym: bus topology - network design with all devices connected to a single backbone
  • Antonym: ring topology - network configuration where each device is connected in a closed loop

Etymology

From 'mesh' (interlaced structure) + 'topology' (study of spatial relations); used in computing since the late 20th century.
