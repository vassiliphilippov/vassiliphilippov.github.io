pit run

1. NOUN

/ pɪt ˌrʌn / (AmE), / pɪt ˌrʌn / (BrE)
gravel found in natural deposits, typically unprocessed and as taken directly from pits or quarries
naturally occurring, unprocessed gravel

Examples

The contractor chose to use pit run for the base layer of the road.
Because pit run contains a mix of stone sizes, it is ideal for some foundation work.
We ordered several truckloads of pit run to fill the site before construction began.

Related Terms

  • Synonym: gravel - small stones used in construction or landscaping
