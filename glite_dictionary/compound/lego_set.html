Lego set

1. NOUN

/ ˈlɛɡoʊ sɛt / (AmE), / ˈlɛɡəʊ sɛt / (BrE)
A child's construction toy consisting of interlocking plastic bricks and other parts, manufactured by the LEGO Group, used to build models.
plastic construction toy set

Examples

He spent the afternoon assembling a new Lego set he received for his birthday.
The instructions included with each Lego set make building complex models possible for children.
Many adults enjoy collecting and displaying rare Lego sets.

Related Terms

  • Synonym: building set - a set of parts for constructing models

Etymology

From the Danish company name LEGO, from 'leg godt' meaning 'play well.'
