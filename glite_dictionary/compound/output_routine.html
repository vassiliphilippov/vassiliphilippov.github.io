output routine

1. NOUN

/ ˈaʊtˌpʊt ruːˈtin / (AmE), / ˈaʊtpʊt ruːˈtiːn / (BrE)
a sequence of instructions or a program routine that controls the operation of an output device, such as a printer or display screen, in a computer system.
routine for managing output devices

Examples

The output routine converts processed data into a format suitable for the printer.
If the output routine fails, the monitor may not display any results.
Programmers often optimize the output routine to improve overall speed.

Related Terms

  • Antonym: input routine - a routine that controls an input device
