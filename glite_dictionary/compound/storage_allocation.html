storage allocation

1. NOUN

/ ˈstɔrɪdʒ ˌæləˈkeɪʃən / (AmE), / ˈstɔːrɪdʒ ˌæləˈkeɪʃən / (BrE)
(computing) the process of assigning specific areas of memory or disk space to data or instructions within a computer system.
assignment of memory or disk space in computing

Examples

Proper storage allocation ensures efficient use of system resources and better performance.
A bug in the storage allocation routine caused the program to crash unexpectedly.
The operating system handles storage allocation for files saved on the hard drive.

Related Terms

  • Synonym: memory allocation - the assignment of memory blocks to programs or data
