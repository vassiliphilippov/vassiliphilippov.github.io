via ferrata set

1. NOUN

/ viə fərˈɑːtə sɛt / (AmE), / viːə fəˈrætə sɛt / (BrE)
A set of climbing equipment, typically consisting of a lanyard and two carabiners, used for safety on via ferrata routes.
climbing safety equipment for via ferrata

Examples

Before starting the climb, make sure your via ferrata set is properly attached and secure.
Many rental shops in the Alps provide helmets and a via ferrata set for beginners.
A faulty via ferrata set can be extremely dangerous during a steep ascent.

Synset Examples

After a fatal via ferrata accident in August 2012 where both elastic lanyards on the energy-absorbing systems (EAS) in a via ferrata set failed, the UIAA (International Mountaineering and Climbing Federation) worked with manufacturers to identify and recall several models of EAS systems.

Etymology

From Italian 'via ferrata' (iron road), referring to protected climbing routes; 'set' as in a group of equipment.
