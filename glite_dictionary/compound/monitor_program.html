monitor program

1. NOUN

/ ˈmɑː.nɪ.tɚ ˌproʊ.ɡræm / (AmE), / ˈmɒn.ɪ.tə ˌprəʊ.ɡræm / (BrE)
a computer program that observes, regulates, controls, or verifies the operations of a data-processing system
computer program that oversees operations

Examples

The system relies on a monitor program to detect and report hardware failures.
A monitor program can automatically restart services if they crash unexpectedly.
Engineers developed a new monitor program to ensure data integrity during processing.

Related Terms

  • Synonym: supervisor program - a program that controls the execution of other programs
  • Antonym: application program - a program designed to perform a specific function for the user
