curvilinear regression

1. NOUN

/ ˌkɝːvəˈlɪniɚ rɪˈɡrɛʃən / (AmE), / ˌkɜːvɪˈlɪniə rɪˈɡrɛʃən / (BrE)
a type of regression analysis in which the relationship between variables is represented by a nonlinear equation (such as quadratic or higher-order polynomials) instead of a straight line
regression using nonlinear equations

Examples

The researchers used curvilinear regression to better model the complex growth patterns in the data.
Unlike linear models, curvilinear regression can account for acceleration or deceleration in relationships between variables.
When plotted, the results of a curvilinear regression often show a curve rather than a straight line.

Related Terms

  • Antonym: linear regression - regression analysis using a straight-line equation
  • Synonym: polynomial regression - regression analysis using polynomial functions

Etymology

From Latin 'curvus' (curved) + 'linea' (line) and 'regressio' (return, retreat).
