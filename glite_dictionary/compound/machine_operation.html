machine operation

1. NOUN

/ məˈʃin ˌɑpəˈreɪʃən / (AmE), / məˈʃiːn ˌɒpəˈreɪʃən / (BrE)
A basic operation that a computer or machine is specifically designed to perform as part of its instruction set.
basic computer or machine-level operation

Examples

The CPU executes each machine operation in its instruction cycle.
Learning about each machine operation is essential for understanding low-level programming.
This error occurs when an unsupported machine operation is called by the software.

Related Terms

  • Synonym: instruction - A command given to a computer to perform a specific operation
  • Synonym: operation code - A part of a computer instruction that specifies the operation to be performed
