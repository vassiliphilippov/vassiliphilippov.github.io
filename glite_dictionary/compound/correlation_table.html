correlation table

1. NOUN

/ ˌkɔːr.əˈleɪ.ʃən ˈteɪ.bəl / (AmE), / ˌkɒr.əˈleɪ.ʃən ˈteɪ.bəl / (BrE)
a table displaying the relationship between two variables, with row and column headings representing possible values and each cell showing the frequency or association between pairs of values
table showing relationship between two variables

Examples

The researcher created a correlation table to analyze how test scores related to study hours.
A correlation table helps identify if there is any association between two variables in a dataset.
In the report, the correlation table showed that higher income was linked to greater life satisfaction.

Related Terms

  • Synonym: contingency table - a table used in statistics to show the frequency distribution of variables
