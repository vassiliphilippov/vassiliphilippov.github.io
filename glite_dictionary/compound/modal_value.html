modal value

1. NOUN

/ ˈmoʊdəl ˈvæljuː / (AmE), / ˈməʊdəl ˈvaljuː / (BrE)
The value that appears most frequently in a set of data or a probability distribution.
most frequent value in data

Examples

The modal value of this data set is 7, since it occurs more often than any other number.
In a survey of favorite colors, the modal value was blue.
If a distribution is bimodal, it means it has more than one modal value.

Related Terms

  • Synonym: mode - the value that occurs most frequently in a data set

Etymology

From 'modal', relating to the mode (most frequent value), and 'value'. 'Mode' from French 'mode', from Latin 'modus' (measure, manner).
