blow molding machine

1. NOUN

/ bloʊ ˈmoʊldɪŋ məˈʃin / (AmE), / bləʊ ˈməʊldɪŋ məˈʃiːn / (BrE)
a machine used in the manufacturing process to form hollow plastic objects by inflating heated plastic into a mold.
machine for forming hollow plastic items

Examples

The factory purchased a new blow molding machine to increase bottle production.
Operators must be trained to safely use the blow molding machine in the plastics plant.
The blow molding machine requires regular maintenance to avoid production delays.

Synset Examples

The injection blow molding machine is based on an extruder barrel and screw assembly which melts the polymer.
