pingpong table

1. NOUN

/ ˈpɪŋˌpɔŋ ˌteɪbəl / (AmE), / ˈpɪŋˌpɒŋ ˌteɪbəl / (BrE)
A table designed for playing table tennis, typically with a smooth, rectangular surface and a net across the middle.
table for playing table tennis

Examples

The children spent the afternoon playing doubles on the pingpong table.
He set up the pingpong table in the garage to practice his serve.
We need to clear the pingpong table before dinner guests arrive.

Related Terms

  • Synonym: table tennis table - a table used in the game of table tennis

Etymology

From 'ping-pong', an early trademark for table tennis, plus 'table'.
