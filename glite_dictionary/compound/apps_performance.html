app's performance

1. NOUN

/ æps pərˈfɔrməns / (AmE), / æps pəˈfɔːməns / (BrE)
how well a software application functions, especially in terms of speed, stability, and effectiveness
effectiveness of a software application

Examples

The team released an update to improve the app's performance on older devices.
Several users complained about the app's performance after the latest software update.
She ran diagnostics to identify what was affecting the app's performance.

Synset Examples

The programmer shared a hack that improved the app's performance.

Related Terms

  • Synonym: application performance - performance of a software program or application
