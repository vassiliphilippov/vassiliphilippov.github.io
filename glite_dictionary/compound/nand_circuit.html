NAND circuit

1. NOUN

/ nænd ˈsɜːrkɪt / (AmE), / nænd ˈsɜːkɪt / (BrE)
an electronic logic gate that produces an output which is the opposite of the output of an AND gate; only outputs false when all inputs are true
logic gate producing inverted AND output

Examples

The output of a NAND circuit goes low only when both its inputs are high.
Modern digital systems often use NAND circuits because they are easy to implement.
By combining several NAND circuits, complex logic functions can be constructed.

Related Terms

  • Antonym: AND gate - a logic gate that outputs true only if all inputs are true
  • Degree Higher: NOR gate - a logic gate that outputs true only if all inputs are false
  • Degree Lower: OR gate - a logic gate that outputs true if at least one input is true

Etymology

Abbreviation of 'Not AND', from electronic logic terminology.
