application program

1. NOUN

/ ˌæplɪˈkeɪʃən ˈproʊɡræm / (AmE), / ˌap.lɪˈkeɪ.ʃən ˈprəʊ.ɡræm / (BrE)
a computer program designed to help the user perform specific tasks, such as word processing or managing data
computer program for user tasks

Examples

I installed a new application program to help organize my photos.
Many application programs include user-friendly interfaces for beginners.
He specializes in developing custom application programs for small businesses.

Synset Examples

he has tried several different word processing applications

Related Terms

  • Synonym: application - a piece of software designed for a specific task
  • Synonym: program - a set of instructions that a computer executes
