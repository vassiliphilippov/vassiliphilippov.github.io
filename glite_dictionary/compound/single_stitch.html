single stitch

1. NOUN

/ ˈsɪŋɡəl stɪtʃ / (AmE), / ˈsɪŋɡəl stɪtʃ / (BrE)
A basic crochet stitch made by inserting the hook into the work, pulling up a loop, and drawing yarn through both loops on the hook.
basic crochet stitch

Examples

The pattern calls for a row of single stitch to create a simple edge.
She learned how to do a single stitch before trying more complex crochet techniques.
Make sure each single stitch is even for a neat appearance.

Related Terms

  • Degree Higher: double crochet - another type of crochet stitch
