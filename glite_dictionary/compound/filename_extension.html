filename extension

1. NOUN

/ ˈfaɪlˌneɪm ɪkˈstɛntʃən / (AmE), / ˈfaɪlneɪm ɪkˈstɛnʃən / (BrE)
a set of characters, usually beginning with a period, added to the end of a computer filename to indicate its type or format
suffix indicating a computer file's type

Examples

You can recognize a Word document by its filename extension .docx.
It's important to use the correct filename extension when saving images.
The program only opens files with the .csv filename extension.

Synset Examples

most applications provide extensions for the files they create
most BASIC files use the filename extension .BAS

Related Terms

  • Synonym: file extension - a suffix at the end of a filename, used to indicate its format
