light pen

1. NOUN

/ ˈlaɪt ˌpɛn / (AmE), / ˈlaɪt ˌpɛn / (BrE)
A handheld device used to detect light from a computer display, allowing the user to select or draw directly on the screen.
computer input device for detecting display light

Examples

Early computer artists used a light pen to draw images directly onto the screen.
The engineer demonstrated how the light pen could select menu items from the monitor.
Although now rare, the light pen was once a common accessory for certain graphics terminals.

Related Terms

  • Synonym: stylus - a pen-shaped device used for touchscreens or graphics tablets
