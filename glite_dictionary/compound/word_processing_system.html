word processing system

1. NOUN

/ wɝːd ˈprɑː.sɛs.ɪŋ ˈsɪs.təm / (AmE), / wɜːd ˈprɒ.sɛs.ɪŋ ˈsɪs.təm / (BrE)
a computer application that provides tools for writing, editing, formatting, and printing text documents
software for creating and editing text documents

Examples

Most offices rely on a word processing system to produce professional reports and letters.
She upgraded her word processing system to access more advanced editing features.
Learning to use a word processing system is essential for many modern workplaces.

Related Terms

  • Synonym: word processor - a computer program for creating and editing text documents
