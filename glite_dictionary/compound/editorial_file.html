editorial file

1. NOUN

/ ˌɛdɪˈtɔriəl faɪl / (AmE), / ˌɛdɪˈtɔːriəl faɪl / (BrE)
a file containing documents or printouts prepared and ready for publication
file of printouts ready for publishing

Examples

The journalist submitted her editorial file to the publishing department before the deadline.
Please review the editorial file to ensure all articles are formatted correctly for print.
The editorial file contains the final versions of the stories ready for layout.
