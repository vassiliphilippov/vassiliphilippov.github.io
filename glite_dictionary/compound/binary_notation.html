binary notation

1. NOUN

/ ˈbaɪnɛri noʊˈteɪʃən / (AmE), / ˈbaɪnəri nəʊˈteɪʃən / (BrE)
a method of representing numbers or information using only two different symbols, typically 0 and 1
system using 0 and 1 to represent data

Examples

Computers process information using binary notation, which relies on only the digits 0 and 1.
The number 5 in binary notation is represented as 101.
He learned how to convert decimal numbers into binary notation for his computer science class.

Related Terms

  • Synonym: binary system - a system of numerical notation that uses base 2
  • Antonym: decimal notation - a notation that uses ten characters (0–9) to represent numbers

Etymology

From Latin 'binarius' (consisting of two) and Latin 'notatio' (a marking, notation).
