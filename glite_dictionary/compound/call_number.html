call number

1. NOUN

/ kɔl ˈnʌmbɚ / (AmE), / kɔːl ˈnʌmbə / (BrE)
a code of letters and numbers assigned to a book in a library to show its shelf location
library code showing book's location

Examples

The call number is printed on the spine of every library book.
If you know the call number, you can easily find the book in the library.
She wrote down the call number to locate the reference text for her research.

Related Terms

  • Synonym: shelf mark - another term for a code indicating a book's library location
