simplex algorithm

1. NOUN

/ ˈsɪmplɛks ˈælɡəˌrɪðəm / (AmE), / ˈsɪmplɛks ˈælɡərɪðəm / (BrE)
a method for solving linear programming problems by successively moving to adjacent vertices of the feasible set to find the optimum value of the objective function
method for solving linear programming problems

Examples

The simplex algorithm is widely used in operations research to solve large-scale optimization problems.
After setting up the constraints, she implemented the simplex algorithm to find the maximum profit.
Learning how the simplex algorithm works is critical for students studying linear programming.

Synset Examples

The simplex algorithm begins at a starting vertex and moves along the edges of the polytope until it reaches the vertex of the optimum solution.

Related Terms

  • Synonym: linear programming - the mathematical technique for optimizing a linear objective function subject to linear constraints

Etymology

From 'simplex' (simple or single) and 'algorithm', the latter from the name of the Persian mathematician al-Khwarizmi.
