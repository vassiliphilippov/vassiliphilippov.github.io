computer-oriented language

1. NOUN

/ kəmˌpjuːtər ɔːr.iˌɛn.t̬ɪd ˈlæŋ.ɡwɪdʒ / (AmE), / kəmˌpjuː.tər ɔːr.iˌen.tɪd ˈlæŋ.ɡwɪdʒ / (BrE)
a programming language designed to be used with a specific class of computers, often tailored to their architecture or operating system.
a programming language for a specific computer type

Examples

The new system was programmed using a computer-oriented language created for that particular hardware.
Each computer-oriented language typically has features optimized for its target machine.
COBOL began as a computer-oriented language for business data processing.

Related Terms

  • Synonym: assembly language - a low-level programming language for a computer or processor
  • Synonym: machine language - the lowest-level programming language, consisting of binary code
  • Antonym: high-level language - a programming language with strong abstraction from computer details
