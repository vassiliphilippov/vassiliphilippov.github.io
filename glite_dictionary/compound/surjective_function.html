surjective function

1. NOUN

/ sɚˈdʒɪktɪv ˈfʌŋkʃən / (AmE), / sɜːˈdʒɪktɪv ˈfʌŋkʃən / (BrE)
A function in mathematics in which every element of the target set is mapped to by at least one element of the domain; an onto mapping.
mathematics: function that is onto
medical

Examples

A surjective function ensures that every output value is associated with at least one input value.
In this exercise, you must determine whether the given mapping is a surjective function.
The concept of a surjective function is important in determining whether an equation has solutions in every case.

Synset Examples

In fact, if we define a ρ-kernel as the kernel of any surjection of the two graphs of interest, then there are ρ-kernels whose maximum excess does not vanish.

Related Terms

  • Antonym: injective function - a function where each element of the target set is mapped to by at most one element of the domain (one-to-one function)
  • Degree Higher: bijective function - a function that is both injective and surjective (one-to-one correspondence)

Etymology

From Latin 'sur-' (over, above) + 'jacere' (to throw), meaning 'thrown over'; used in mathematics to mean mapping over the whole set.
