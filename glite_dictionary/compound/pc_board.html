PC board

1. NOUN

/ ˌpiː ˈsiː bɔrd / (AmE), / ˌpiː ˈsiː bɔːd / (BrE)
a removable circuit board that can be inserted into a slot in a personal computer's motherboard to add functionality or features
removable circuit board for a personal computer

Examples

He installed a new PC board to improve the computer's graphics capabilities.
The technician removed the faulty PC board from the system for replacement.
You can add more memory by plugging another PC board into the available slot.

Related Terms

  • Synonym: expansion card - a removable circuit board that adds functionality to a computer
