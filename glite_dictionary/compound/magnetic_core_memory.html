magnetic core memory

1. NOUN

/ məɡˈnɛtɪk kɔr ˈmɛməri / (AmE), / mæɡˈnɛtɪk kɔː ˈmɛməri / (BrE)
A type of computer memory consisting of an array of tiny magnetic cores, used for storing data, which was widely used before the advent of semiconductor memory.
early computer memory using magnetic cores

Examples

Early computers relied on magnetic core memory to store both data and instructions.
The invention of magnetic core memory marked a major breakthrough in computer technology during the 1950s.
Technicians frequently repaired the delicate wires woven through the magnetic core memory modules.

Related Terms

  • Degree Higher: semiconductor memory - computer memory using semiconductor components
