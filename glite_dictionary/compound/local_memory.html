local memory

1. NOUN

/ ˈloʊkəl ˈmɛməri / (AmE), / ˈləʊkəl ˈmɛməri / (BrE)
The memory located within or directly accessible by a single CPU, program, or function, used for quick data access as opposed to shared or remote memory.
memory directly accessible by a single CPU or program

Examples

Each processor is responsible for managing its own local memory efficiently.
Variables stored in local memory are not accessible by other functions.
Upgrading the local memory can significantly improve a computer's performance.

Synset Examples

A processor can access its own local memory faster than non-local memory (memory which is local to another processor or shared between processors).

Related Terms

  • Antonym: shared memory - memory that can be accessed by multiple CPUs or processes
  • Antonym: remote memory - memory not directly connected to a CPU or function
