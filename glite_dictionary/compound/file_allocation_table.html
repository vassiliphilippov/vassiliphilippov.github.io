file allocation table

1. NOUN

/ ˈfaɪl ˌæləkˈeɪʃən ˈteɪbəl / (AmE), / ˈfaɪl ˌæləˈkeɪʃ(ə)n ˈteɪbl / (BrE)
a part of a computer disk that keeps track of where files and unusable areas are located on the disk
system for tracking storage on computer disks

Examples

The file allocation table is essential for organizing and retrieving data on your hard drive.
If the file allocation table is corrupted, you may lose access to important information.
Older versions of Windows rely heavily on the file allocation table for file management.
