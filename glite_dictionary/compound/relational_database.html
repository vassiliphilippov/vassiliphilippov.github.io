relational database

1. NOUN

/ rɪˈleɪʃənəl ˈdeɪtəˌbeɪs / (AmE), / rɪˈleɪʃənəl ˈdeɪtəˌbeɪs / (BrE)
A database in which data is organized into tables with defined relationships between them, allowing for easy access and management of associated information.
database organized as related tables

Examples

Many companies use a relational database to efficiently store and retrieve customer information.
Learning how to design a relational database is essential for aspiring data engineers.
A relational database allows users to link tables such as customers and orders through key fields.

Synset Examples

in a relational database the data are organized as a number of differently sized tables

Related Terms

  • Synonym: database - a structured set of data held in a computer

Etymology

From 'relation' and 'database'; introduced in the 1970s by computer scientist Edgar F. Codd.
