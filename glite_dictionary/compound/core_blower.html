core blower

1. NOUN

/ kɔr ˈbloʊɚ / (AmE), / kɔː ˈbləʊə / (BrE)
a machine that uses compressed air to blow and pack sand into a core box, used in foundry work to shape sand cores for molds
machine for packing sand cores in foundry work

Examples

The foundry recently installed a new core blower to increase production efficiency.
Using a core blower ensures that the sand cores are tightly packed and uniform.
The technician operated the core blower to prepare molds for the casting process.

Related Terms

  • Synonym: core box - a box-shaped mold used to form sand cores for casting
