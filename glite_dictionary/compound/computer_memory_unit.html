computer memory unit

1. NOUN

/ kəmˈpjuːtər ˈmɛməri ˈjuːnɪt / (AmE), / kəmˈpjuːtə ˈmɛməri ˈjuːnɪt / (BrE)
a standard unit used to measure the amount of computer memory, such as a byte or kilobyte
a unit for measuring computer memory

Examples

The file size exceeded the standard computer memory unit used by this system.
A kilobyte is a common computer memory unit representing 1024 bytes.
Engineers often refer to the computer memory unit when comparing system requirements.

Related Terms

  • Synonym: byte - a unit of digital information, usually consisting of eight bits
