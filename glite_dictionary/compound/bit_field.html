bit field

1. NOUN

/ bɪt fiːld / (AmE), / bɪt fiːld / (BrE)
A sequence of binary digits within a computer memory or data structure, used to represent data compactly or to store separate values in adjacent bits.
sequence of binary digits (in computing)

Examples

Each bit field in the data structure controls different device settings.
You can set the permissions using a bit field in the file header.
The size of a bit field determines how much information it can store.

Related Terms

  • Synonym: bit - a unit of information in computing, either 0 or 1
