duodecimal digit

1. NOUN

/ ˌduːoʊˈdɛsɪməl ˈdɪdʒɪt / (AmE), / ˌdjuːəʊˈdɛsɪməl ˈdɪdʒɪt / (BrE)
a digit from 0 to 11 used in the duodecimal (base-12) numeral system
a digit used in base-12 notation

Examples

In the duodecimal system, the highest duodecimal digit is typically represented as 'B'.
Each duodecimal digit can be mapped to a decimal value ranging from 0 to 11.
Converting from decimal to duodecimal requires understanding what each duodecimal digit represents.

Etymology

From Latin 'duodecimus' (twelfth) + 'digitus' (finger, digit).
