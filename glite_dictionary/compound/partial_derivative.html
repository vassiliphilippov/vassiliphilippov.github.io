partial derivative

1. NOUN

/ ˈpɑːrʃəl dɪˈrɪvətɪv / (AmE), / ˈpɑːʃəl dɪˈrɪvətɪv / (BrE)
the derivative of a function of two or more variables with respect to one variable, keeping the other variables constant
derivative with respect to one variable of a multivariable function

Examples

The partial derivative of f(x, y) with respect to x is written as ∂f/∂x.
You need to calculate the partial derivative to understand how the function changes with only one variable.
In multivariable calculus, the partial derivative is used to analyze how a surface bends in different directions.

Related Terms

  • Synonym: derivative - the rate at which a function changes at any given point

Etymology

From Latin 'pars' (part) and medieval Latin 'derivativus' (pertaining to derivation).
