sorting program

1. NOUN

/ ˈsɔrtɪŋ ˌproʊɡræm / (AmE), / ˈsɔːtɪŋ ˌprəʊɡræm / (BrE)
a utility program used to arrange data items in a specified order, typically alphabetically or numerically
utility program for ordering data

Examples

The company uses a sorting program to organize large lists of customer names.
Before analyzing the dataset, a sorting program arranged all the entries in order.
He designed a new sorting program that can handle millions of records efficiently.

Related Terms

  • Synonym: sort - arrange systematically in groups or in a particular sequence
