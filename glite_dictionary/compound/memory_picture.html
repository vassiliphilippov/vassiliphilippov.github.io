memory picture

1. NOUN

/ ˈmɛməri ˈpɪktʃɚ / (AmE), / ˈmɛməri ˈpɪktʃə / (BrE)
a mental image recalled from memory that is similar to a visual perception
mental image resembling visual perception

Examples

Whenever she thought of her childhood home, a vivid memory picture came to mind.
The artist was able to recreate scenes from memory using detailed memory pictures.
After the trip, he often recalled a memory picture of the mountain sunrise.

Related Terms

  • Synonym: mental image - an image that occurs in the mind
