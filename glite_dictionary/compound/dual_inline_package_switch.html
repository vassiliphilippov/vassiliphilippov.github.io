dual inline package switch

1. NOUN

/ ˈduːəl ˈɪnˌlaɪn ˈpækɪdʒ swɪtʃ / (AmE), / ˈdjuːəl ˈɪnˌlaɪn ˈpækɪdʒ swɪtʃ / (BrE)
A small on-off switch mounted in computer hardware, used instead of jumpers to configure settings for the user.
hardware switch for configuring computers

Examples

The motherboard uses a dual inline package switch to control hardware settings.
Before starting up the device, check the dual inline package switch positions to ensure proper configuration.
Many older computers required users to manually adjust a dual inline package switch rather than using software settings.

Related Terms

  • Synonym: DIP switch - an abbreviation for dual inline package switch, meaning the same hardware component

Etymology

From 'dual inline package' (DIP), referring to a type of electronic component packaging, combined with 'switch'.
