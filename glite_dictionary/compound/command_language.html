command language

1. NOUN

/ kəˈmænd ˌlæŋɡwɪdʒ / (AmE), / kəˈmɑːnd ˌlæŋɡwɪdʒ / (BrE)
a programming or input language consisting of specific commands or procedural instructions that direct a computer to perform particular functions
programming language with procedural commands

Examples

To automate routine tasks, the developer wrote scripts in the command language provided by the system.
Learning a command language can help users control complex applications with precision.
The tool accepts instructions only in its native command language, requiring users to learn its syntax.

Related Terms

  • Synonym: programming language - formal language used to communicate instructions to a computer
