amended return

1. NOUN

/ əˈmɛndɪd rɪˈtɜrn / (AmE), / əˈmɛndɪd rɪˈtɜːn / (BrE)
a tax return submitted to correct or update information provided in an previously filed return
corrected tax return
legal

Examples

After discovering a mistake in last year’s taxes, she filed an amended return with the IRS.
The accountant recommended submitting an amended return to correct the reported income error.
Taxpayers have up to three years to send in an amended return if they need to make changes.

Related Terms

  • Synonym: tax return - a form filed with a tax authority to report income, expenses, and other pertinent tax information
