macro instruction

1. NOUN

/ ˈmækroʊ ɪnˌstrʌkʃən / (AmE), / ˈmakrəʊ ɪnˌstrʌkʃən / (BrE)
A single computer instruction that expands automatically into a set of machine language instructions.
computer instruction that expands into several instructions

Examples

The programmer used a macro instruction to automate the repetitive code sequence.
Compilers generally replace a macro instruction with the corresponding series of machine operations.
By defining a macro instruction, you can simplify complex tasks in assembly language programming.

Related Terms

  • Synonym: macro - a single instruction that expands into a set of instructions in computer programming

Etymology

From 'macro', meaning large or long, and 'instruction', in reference to programming usage since mid-20th century.
