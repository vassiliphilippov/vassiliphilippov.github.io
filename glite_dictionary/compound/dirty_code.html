dirty code

1. NOUN

/ ˈdɝːti koʊd / (AmE), / ˈdɜːti kəʊd / (BrE)
programming code that is poorly structured, difficult to read, or full of bad practices
messy or poorly structured programming code

Examples

The team spent several days cleaning up the dirty code left by their predecessors.
If you keep writing dirty code, it will be much harder to maintain the project.
Automated tools can help identify sections of dirty code for refactoring.

Synset Examples

We need to refactor this dirty code to improve performance.
He regretted writing dirty code that nobody could understand.
The software crashed due to the dirty code left by developers.

Related Terms

  • Antonym: clean code - programming code that is well-organized, readable, and free of bad practices
