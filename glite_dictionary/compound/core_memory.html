core memory

1. NOUN

/ kɔr ˈmɛməri / (AmE), / kɔː ˈmɛməri / (BrE)
A type of computer memory that uses an array of magnetic cores to store data, formerly widespread but now replaced by semiconductor memory.
historical magnetic computer memory

Examples

Many early computers relied on core memory for data storage before alternatives became available.
The engineer was able to repair the malfunctioning core memory module by replacing several damaged cores.
Despite being obsolete, core memory is still appreciated for its durability and non-volatility.

Related Terms

  • Degree Higher: semiconductor memory - computer memory using semiconductor technology
  • Synonym: magnetic core - the tiny ferrite ring used in core memory

Etymology

Refers to the small magnetic cores used as storage elements in the memory array.
