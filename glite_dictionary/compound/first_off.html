first off

1. ADV

/ fɝst ɔf / (AmE), / fɜːst ɒf / (BrE)
before anything else; as the first point or matter to be considered
before anything else; as a first point
informal

Examples

First off, I want to thank everyone for coming tonight.
First off, let me clarify the purpose of this meeting before we begin.
First off, we need to address the most urgent issue.

Synset Examples

first we must consider the garter snake

Related Terms

  • Synonym: firstly - used to introduce the first point in a series
  • Synonym: first of all - before anything else; as the very first point

2. ADV

/ ˈfɝst ɔf / (AmE), / ˈfɜːst ɒf / (BrE)
used to introduce the first point or topic in a discussion; to begin with; initially.
to begin with; at the start
informal

Examples

First off, we need to identify the main issues this project will face.
I want to thank everyone for coming, and first off, let me say how grateful I am.
First off, let's review the agenda before we start the meeting.

Synset Examples

First off, we should establish our goals for this project.
First off, I want to thank you all for your hard work.
First off, let’s talk about the budget constraints.

Related Terms

  • Synonym: firstly - introducing the first point in a list or argument
