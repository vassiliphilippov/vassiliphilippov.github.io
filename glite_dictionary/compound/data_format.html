data format

1. NOUN

/ ˈdeɪtə ˈfɔːrmæt / (AmE), / ˈdeɪtə ˈfɔːmæt / (BrE)
a particular way of organizing information according to defined specifications, especially for computer storage or processing
organization of information for computer use

Examples

The software can import files in any data format, making it highly versatile.
Make sure you save your document in the correct data format before sharing it.
Each sensor output is transmitted using a standardized data format for compatibility.

Related Terms

  • Synonym: file format - the structure of a file for storing data
  • Degree Lower: data structure - an arrangement for storing and organizing data
