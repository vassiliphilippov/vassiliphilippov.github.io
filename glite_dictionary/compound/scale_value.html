scale value

1. NOUN

/ skeɪl ˈvælju / (AmE), / skeɪl ˈvæljuː / (BrE)
a value or number on a particular scale used for measurement or comparison
value on a measurement scale

Examples

Each data point was assigned a scale value corresponding to its position on the graph.
The pH meter displays the scale value of the solution's acidity.
Temperature readings must be converted to a standard scale value for comparison.
