dining-room set

1. NOUN

/ ˈdaɪnɪŋ ˌruːm sɛt / (AmE), / ˈdaɪnɪŋ ˌruːm sɛt / (BrE)
a matching set of furniture designed for use in a dining room, typically including a dining table and chairs
matching dining room furniture set

Examples

They purchased a new dining-room set for their home after moving in.
The antique dining-room set features carved wood and elegant upholstery.
She spent hours polishing the dining-room set before the guests arrived.
