duodecimal system

1. NOUN

/ duː.oʊˈdɛs.ɪ.məl ˈsɪs.təm / (AmE), / ˌdjuː.əʊˈdɛs.ɪ.məl ˈsɪs.təm / (BrE)
A numeral system based on twelve as its base, using twelve distinct digits to represent numbers.
numeral system with base twelve

Examples

Some ancient cultures used the duodecimal system for counting and calculations.
The duodecimal system requires twelve different digits instead of the usual ten.
Interest in the duodecimal system persists among mathematicians studying alternative numeral bases.

Related Terms

  • Degree Lower: decimal system - numeral system with base ten
  • Degree Higher: hexadecimal system - numeral system with base sixteen

Etymology

From Latin 'duodecim' meaning 'twelve' + 'systema' meaning 'system'.
