hypertext mark-up language

1. NOUN

/ ˈhaɪ.pɚ.tɛkst ˈmɑːrk.ʌp ˈlæŋ.ɡwɪdʒ / (AmE), / ˈhaɪ.pə.tɛkst ˈmɑːk.ʌp ˈlæŋ.ɡwɪdʒ / (BrE)
a standard system for tagging text files to achieve font, colour, graphic, and hyperlink effects on web pages; commonly known as HTML
system for tagging web documents (HTML)

Examples

Web developers use hypertext mark-up language to structure and display content on websites.
Learning hypertext mark-up language is essential for anyone interested in web design.
Most modern websites are built using hypertext mark-up language in combination with other technologies.

Related Terms

  • Synonym: HTML - abbreviation for hypertext mark-up language, used to build web pages

Etymology

From 'hypertext' + 'markup language'; referring to its function in marking up hypertext documents for the web.
