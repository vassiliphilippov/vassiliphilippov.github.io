uninominal voting system

1. NOUN

/ ˌjuː.nɪˈnɑː.mɪn.əl ˈvoʊ.tɪŋ ˌsɪs.təm / (AmE), / ˌjuː.nɪˈnɒ.mɪn.əl ˈvəʊ.tɪŋ ˌsɪs.təm / (BrE)
a voting system in which only one representative is elected from each electoral district
system with one representative per district

Examples

Many countries use a uninominal voting system to elect their lawmakers.
Critics argue that the uninominal voting system can distort the representation of smaller parties.
A shift from proportional representation to a uninominal voting system is a significant electoral reform.

Related Terms

  • Antonym: proportional representation - electoral system in which parties gain seats in proportion to the number of votes cast for them

Etymology

From French uninominal (from Latin unus 'one' + nomen 'name').

2. NOUN

/ ˌjuːnɪˈnɑːmɪnəl ˈvoʊtɪŋ ˈsɪstəm / (AmE), / ˌjuːnɪˈnɒmɪnəl ˈvəʊtɪŋ ˈsɪstəm / (BrE)
an electoral system in which only one representative is elected from each district or constituency
voting system with one representative per district

Examples

In a uninominal voting system, only one candidate is elected per electoral district.
Many countries use a uninominal voting system to ensure direct representation.
Critics argue that a uninominal voting system can disadvantage smaller political parties.

Related Terms

  • Synonym: single-member district system - an electoral system where each district elects a single representative
  • Antonym: proportional representation - an electoral system where seats are allocated in proportion to votes

Etymology

From Latin 'uni-' (one) + 'nomen' (name), referring to the selection of a single representative.
