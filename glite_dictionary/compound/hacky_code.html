hacky code

1. NOUN

/ ˈhæki ˌkoʊd / (AmE), / ˈhæki ˌkəʊd / (BrE)
(informal) Computer code that is poorly written, inelegant, or created as a quick and temporary solution rather than following good programming practices.
inelegant or poorly written computer code
informal

Examples

The app crashed frequently due to all the hacky code left in after the last update.
He regretted shipping the project with so much hacky code just to meet the deadline.
It's often tempting to write hacky code to fix bugs quickly, but it can cause more trouble later.

Synset Examples

They dismissed the hacky code as unprofessional.

Related Terms

  • Synonym: spaghetti code - computer code with a complex and tangled structure
  • Antonym: clean code - computer code that is well-organized and easy to understand

Etymology

From 'hacky', an informal derivative of 'hack' (in programming, a quick or inelegant solution), plus 'code'.
