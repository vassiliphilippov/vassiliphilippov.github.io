command prompt

1. NOUN

/ kəˈmænd prɑmpt / (AmE), / kəˈmɑːnd prɒmpt / (BrE)
(computing) a symbol or line on a computer screen indicating that the computer is ready to receive a command.
symbol showing a computer is ready for a command

Examples

When the command prompt appears, you can type your instructions.
He waited for the command prompt before entering the next line of code.
If you don't see the command prompt, the computer might still be processing your last request.

Related Terms

  • Synonym: prompt - the symbol or message on a computer inviting user input
