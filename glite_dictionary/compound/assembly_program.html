assembly program

1. NOUN

/ əˈsɛmbli ˌproʊɡræm / (AmE), / əˈsɛmbli ˌprəʊɡram / (BrE)
a computer program that translates assembly language into machine code
translates assembly language to machine code

Examples

She learned how to use an assembly program in her computer architecture class.
Many microcontrollers require a specific assembly program to generate the correct binary output.
