memory cache

1. NOUN

/ ˈmɛməri kæʃ / (AmE), / ˈmɛməri kæʃ / (BrE)
(Computing) a section of RAM reserved as a specialized buffer store, continually updated to optimize data transfers between system components with differing speeds.
specialized buffer in computer memory for improved data transfer

Examples

The processor relies on the memory cache to quickly access frequently used data.
Upgrading your computer's memory cache can improve overall system performance.
The memory cache temporarily stores information to speed up data retrieval.

Related Terms

  • Synonym: cache - a store of data in computing that allows for faster access
