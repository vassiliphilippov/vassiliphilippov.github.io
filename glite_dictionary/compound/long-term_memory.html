long-term memory

1. NOUN

/ lɔŋ tɝm ˈmɛməri / (AmE), / lɒŋ tɜːm ˈmɛməri / (BrE)
the part of the mind that stores information for an extended period, often indefinitely
memory storing information long-term

Examples

As we age, it's common for long-term memory to stay intact while short-term memory fades.
He relied on his long-term memory to recall the names of classmates from decades earlier.
A head injury can sometimes affect long-term memory, making it difficult to remember events from the past.

Related Terms

  • Antonym: short-term memory - memory that holds information temporarily, for a few seconds or minutes
