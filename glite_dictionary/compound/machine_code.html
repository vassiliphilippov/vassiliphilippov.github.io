machine code

1. NOUN

/ məˈʃin koʊd / (AmE), / məˈʃiːn kəʊd / (BrE)
the set of instructions in binary or numerical format that a computer's central processing unit can execute directly, without further translation
binary instructions executed directly by a computer

Examples

The software was converted into machine code before being loaded onto the microcontroller.
Only the CPU can directly interpret and execute machine code.
Writing programs in machine code is difficult and rarely done by most programmers today.

Related Terms

  • Degree Lower: assembly language - a low-level programming language that is closely related to machine code
  • Degree Higher: high-level language - a programming language that uses human-readable instructions, farther removed from machine code
