international Morse code

1. NOUN

/ ˌɪntərˈnæʃənəl mɔrs koʊd / (AmE), / ˌɪntəˈnæʃ(ə)nəl mɔːs kəʊd / (BrE)
a telegraph code in which letters and numbers are represented by standardized sequences of dots and dashes (short and long signals)
telegraph code using dots and dashes

Examples

Radio operators communicated secretly using international Morse code during the war.
Learning international Morse code can be a useful skill in amateur radio.
The distress signal SOS is transmitted in international Morse code as three dots, three dashes, and three dots.

Related Terms

  • Synonym: Morse code - telegraph code invented by Samuel Morse using dots and dashes

Etymology

From the name of Samuel Morse, American inventor, and 'international' indicating the globally agreed standard from 1865.
