random access memory

1. NOUN

/ ˈrændəm ˈæksɛs ˈmɛməri / (AmE), / ˈrændəm ˈæksɛs ˈmɛməri / (BrE)
a type of computer memory that can be accessed randomly; used by programs to temporarily store and retrieve data while a computer is running
computer memory for temporary data storage

Examples

My laptop has eight gigabytes of random access memory to handle multiple applications at once.
Upgrading your random access memory can significantly improve computer performance.
The engineer explained how random access memory temporarily stores data while the machine is running.

Related Terms

  • Synonym: RAM - abbreviation for random access memory
  • Antonym: read-only memory - computer memory that can only be read and not altered

Etymology

From 'random access' (ability to access data in any order) + 'memory'; technical usage established in the 1960s.
