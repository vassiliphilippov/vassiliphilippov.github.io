software program

1. NOUN

/ ˈsɔft.wɛɹ ˈproʊ.ɡræm / (AmE), / ˈsɒft.wɛə ˈprəʊ.ɡræm / (BrE)
a set of written instructions, procedures, or rules, along with related documentation, that direct the operation of a computer system; stored in a computer's memory
instructions and documentation for computer operation

Examples

The new software program made it much easier to edit photos on his computer.
Before installing the software program, make sure your system meets the minimum requirements.
Our team developed a software program to help automate accounting tasks.

Synset Examples

the market for software is expected to expand

Related Terms

  • Synonym: application - a program designed for a specific function on a computer
  • Synonym: software - the programs and other operating information used by a computer
  • Antonym: hardware - the physical components of a computer system as opposed to programs
