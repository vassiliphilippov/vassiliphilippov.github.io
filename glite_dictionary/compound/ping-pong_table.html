ping-pong table

1. NOUN

/ ˈpɪŋ pɔŋ ˌteɪbəl / (AmE), / ˈpɪŋ pɒŋ ˌteɪbəl / (BrE)
a table specifically designed for playing table tennis, usually with a smooth, rectangular surface and a dividing net
table for playing table tennis

Examples

The school just bought a new ping-pong table for the recreation room.
He challenged me to a match at the ping-pong table after lunch.
They folded up the ping-pong table to make space for the party.

Related Terms

  • Synonym: table tennis table - a table used for playing table tennis

Etymology

Ping-pong from a trademark for the game; table as in the common noun.
