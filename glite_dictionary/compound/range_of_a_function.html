range of a function

1. NOUN

/ reɪndʒ əv ə ˈfʌŋkʃən / (AmE), / reɪndʒ ɒv ə ˈfʌŋkʃən / (BrE)
(Mathematics) The set of all possible values that a function's dependent variable can take, given its domain.
set of values a function can take

Examples

The range of a function is the set of all possible output values it can produce.
If the range of a function is restricted, some outputs may never be achieved.
For f(x) = sin(x), the range of a function is all real numbers between -1 and 1, inclusive.

Synset Examples

the image of f(x) = x^2 is the set of all non-negative real numbers if the domain of the function is the set of all real numbers

Related Terms

  • Antonym: domain of a function - set of all possible input values of a function
