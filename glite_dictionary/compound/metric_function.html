metric function

1. NOUN

/ ˈmɛtrɪk ˈfʌŋkʃən / (AmE), / ˈmɛtrɪk ˈfʌŋkʃən / (BrE)
a mathematical function that defines the distance between any two points in a given space, especially in topology
function defining distance in a space

Examples

In topology, the metric function helps to determine whether two points are close together or far apart.
A space is called a metric space if there exists a metric function that satisfies certain conditions.
The metric function must satisfy the triangle inequality to correctly define distances.

Related Terms

  • Synonym: metric - a function defining a distance in mathematics

Etymology

From metric (relating to measurement) + function (mathematical relation).
