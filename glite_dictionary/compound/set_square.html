set square

1. NOUN

/ sɛt skwɛr / (AmE), / sɛt skwɛː / (BrE)
A tool used in technical drawing and carpentry, typically consisting of a straightedge with an adjustable or fixed head, used for drawing and checking angles.
tool for drawing or measuring angles

Examples

Please use the set square to check if the edges are at right angles.
She carried her set square and ruler to the drafting table before starting her design.
A good quality set square is essential for accurate technical drawings.

Related Terms

  • Synonym: try square - a carpenter's tool for marking and measuring right angles
