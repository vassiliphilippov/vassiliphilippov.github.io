exponential return

1. NOUN

/ ˌɛkspoʊˈnɛnʃəl rɪˈtɜrn / (AmE), / ˌɛkspəˈnɛnʃəl rɪˈtɜːn / (BrE)
A rate of decrease or increase that follows an exponential function, typically observed in statistics, mathematics, or finance.
change following an exponential function

Examples

The concept of exponential return is crucial in understanding how some investments lose value rapidly over time.
Researchers modeled the population decrease using an exponential return to account for the fast decline.
Unlike simple interest, an exponential return shows a decrease that accelerates as time progresses.

Related Terms

  • Antonym: exponential growth - an increase that follows an exponential function
  • Antonym: linear return - a return that changes at a constant rate

Etymology

From 'exponential', relating to the mathematical exponent, and 'return', meaning a gain, loss, or change in value.
