screen saver

1. NOUN

/ ˈskriːn ˌseɪvər / (AmE), / ˈskriːn ˌseɪvə / (BrE)
A program that displays moving images or patterns on a computer screen when it is not in use for a set period, originally designed to prevent screen damage.
program that displays images when a computer is idle

Examples

I set a colorful screen saver to appear on my computer after ten minutes of inactivity.
Many modern devices no longer require a screen saver because of advances in display technology.
The old computer played a bouncing ball screen saver whenever it was idle.

Synset Examples

screen savers prevent the damage that occurs when the same areas of light and dark are displayed too long

Related Terms

  • Synonym: screensaver - alternate spelling of 'screen saver'
