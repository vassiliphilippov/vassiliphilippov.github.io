disk file

1. NOUN

/ dɪsk faɪl / (AmE), / dɪsk faɪl / (BrE)
(computer science) a computer file stored on a magnetic disk and identified by a unique label.
computer file stored on a disk
medical

Examples

The software saves your document as a disk file for later retrieval.
Always back up important data to a separate disk file to prevent loss.
You can transfer a disk file from one computer to another using a USB drive.

Related Terms

  • Synonym: file - a collection of data stored on a computer
  • Degree Higher: disk - a flat, circular device for storing data magnetically
