field-sequential color television system

1. NOUN

/ ˈfiːld səˈkwɛnʃəl ˈkʌlər ˈtɛləˌvɪʒən ˈsɪstəm / (AmE), / ˈfiːld sɪˈkwɛnʃəl ˈkʌlə ˈtɛlɪˌvɪʒən ˈsɪstəm / (BrE)
an early method of displaying color television in which images are shown in quick succession using the three primary colors, creating the perception of full color to the viewer.
early color TV method using sequential fields

Examples

The field-sequential color television system was eventually replaced by more practical technologies for broadcasting color TV.
John has a vintage set that uses a field-sequential color television system from the early 1950s.
Researchers studied the limitations of the field-sequential color television system when evaluating early color broadcast standards.
