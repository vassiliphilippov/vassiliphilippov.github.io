check character

1. NOUN

/ ˈtʃɛk ˌkærɪktɚ / (AmE), / ˈtʃɛk ˌkærɪktə / (BrE)
A character added at the end of a block of data in transmission to verify its accuracy by enabling error detection.
character for detecting data transmission errors

Examples

The software verifies the integrity of each message by analyzing the check character at the end of the string.
A transmission error was detected when the check character did not match the expected value.
Barcodes often include a check character to ensure digits have been scanned correctly.

Related Terms

  • Synonym: checksum - a computed value used to check the integrity of data
