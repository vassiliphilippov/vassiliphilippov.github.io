set piece

1. NOUN

/ sɛt pis / (AmE), / sɛt piːs / (BrE)
an individual item of scenery designed to stand alone as part of a stage setting
theatrical scenery item

Examples

The crew quickly moved the set piece into position for the next scene.
A large painted tree served as the main set piece in the forest setting.
Each set piece had to be constructed to be both sturdy and lightweight.
