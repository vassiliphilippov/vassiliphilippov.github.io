insufficient RAM

1. ADJ

/ ɪn.səˈfɪʃ.ənt ɹæm / (AmE), / ɪn.səˈfɪʃ.ənt ræm / (BrE)
not having enough random access memory (RAM) in a computer to efficiently run tasks or programs
not enough computer memory (RAM)

Examples

The program crashed because there was insufficient RAM to load all the files.
If your laptop is slow, it may be due to insufficient RAM.
Upgrading your system can help solve issues caused by insufficient RAM.

Synset Examples

The computer started to lag due to insufficient ram.

Related Terms

  • Synonym: low memory - having little available memory in a computing device
  • Antonym: adequate RAM - having enough random access memory for computing tasks

Etymology

RAM is an acronym for 'random access memory', a technical term from computing.
