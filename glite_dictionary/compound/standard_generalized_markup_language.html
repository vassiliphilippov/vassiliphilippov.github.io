standard generalized markup language

1. NOUN

/ ˈstæn.dɚd ˌdʒɛn.ɚ.əˌlaɪzd ˈmɑr.kʌp ˌlæŋ.ɡwɪdʒ / (AmE), / ˈstæn.dəd ˌdʒɛn.ə.rəˌlaɪzd ˈmɑː.kʌp ˌlæŋ.ɡwɪdʒ / (BrE)
(in computer science) a standardized system for defining the structure and formatting of documents using a set of rules for markup tags.
document markup language standard (computing)

Examples

The standard generalized markup language was influential in the development of web technologies.
Learning about standard generalized markup language helped me understand how document formats work.
Many document management systems were originally based on the standard generalized markup language.

Related Terms

  • Synonym: SGML - abbreviation for standard generalized markup language
  • Degree Lower: HTML - HyperText Markup Language, an application of SGML used for web pages
  • Degree Higher: XML - Extensible Markup Language, a simplified subset of SGML for data interchange

Etymology

Composed of 'standard', 'generalized', and 'markup language', the latter from 'markup' (printer's instructions) and 'language'. Used since the 1980s in computing.
