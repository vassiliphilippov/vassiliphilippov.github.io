object-oriented programming

1. NOUN

/ ˈɑb.dʒɛkt ˌɔːr.i.ɛn.tɪd ˈproʊˌɡræm.ɪŋ / (AmE), / ˈɒb.dʒɛkt ˌɔː.ri.ɛn.tɪd ˈprəʊ.ɡræm.ɪŋ / (BrE)
a method of programming that organizes software design around data, or objects, rather than functions and logic
a programming method using objects

Examples

Many modern languages, such as Java and Python, support object-oriented programming natively.
Understanding object-oriented programming can help developers structure their code more efficiently.
She learned about object-oriented programming concepts like classes and inheritance in her computer science class.

Related Terms

  • Antonym: procedural programming - a programming paradigm based on procedures or routines

Etymology

Combines 'object-oriented' (from 'object', meaning a data structure, and 'oriented', meaning directed toward) and 'programming'.

2. NOUN

/ ˌɑbdʒɛkt ˌɔːriˈɛntɪd ˈproʊɡræmɪŋ / (AmE), / ˌɒbdʒɪkt ˌɔːriˈɛntɪd ˈprəʊɡræmɪŋ / (BrE)
a style of computer programming in which concepts are represented as 'objects' that have state and behavior, allowing for code reuse and modularity
style of computer programming with objects

Examples

Learning object-oriented programming can help you design flexible and reusable software systems.
Many modern languages, such as Java and Python, support object-oriented programming by default.
The course covers both procedural and object-oriented programming techniques in detail.

Related Terms

  • Antonym: procedural programming - a programming paradigm based on procedures or routines
  • Synonym: object-oriented - relating to or based on object-oriented programming

Etymology

From 'object', 'oriented' (meaning directed toward), and 'programming'.

Source: WordNet
WordNet ID: oewn-00931247-n
Synset POS: n
