string line

1. NOUN

/ ˈstrɪŋ ˌlaɪn / (AmE), / ˈstrɪŋ ˌlaɪn / (BrE)
an imaginary or marked line across a billiard table behind which the cue balls are placed at the start of a game
line on billiard table for starting play

Examples

The referee placed the balls behind the string line before the match began.
After a foul, the player had to place the cue ball behind the string line.
You must ensure both cue balls are behind the string line at the game's start.
