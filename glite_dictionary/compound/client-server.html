client-server

1. ADJ

/ ˈklaɪənt ˈsɝːvɚ / (AmE), / ˈklaɪənt ˈsɜːvə / (BrE)
Relating to a computer system in which a central server provides resources or services to multiple networked client computers.
computing architecture with central server and clients

Examples

Many modern applications use the client-server model for efficient data processing and storage.
A typical client-server network involves several personal computers connected to a central server.
The new software supports both client-server and peer-to-peer architectures for flexibility.

Related Terms

  • Antonym: peer-to-peer - networking model where computers communicate directly without a server

Etymology

Formed from 'client' (user of a service) and 'server' (provider of a service) in computing terminology.
