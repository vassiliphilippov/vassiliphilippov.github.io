CPU board

1. NOUN

/ siː piː juː bɔrd / (AmE), / siː pɪː juː bɔːd / (BrE)
the main circuit board in a computer, usually containing the central processing unit and other key components
main circuit board of a computer

Examples

The technician replaced the faulty CPU board to get the computer running again.
Upgrading the CPU board can significantly improve your system's performance.
Make sure you handle the CPU board carefully to avoid static electricity damage.

Related Terms

  • Synonym: motherboard - the main printed circuit board in a computer

Etymology

Abbreviation 'CPU' for 'central processing unit', combined with 'board'; from 20th-century computing terminology.
