query language

1. NOUN

/ ˈkwɪri ˌlæŋɡwɪdʒ / (AmE), / ˈkwɪəri ˌlaŋɡwɪdʒ / (BrE)
a computer language used to make queries or retrieve information from databases
computer language for database queries

Examples

The most popular query language for relational databases is SQL.
Learning a query language can help you retrieve specific information from large datasets.
He spent the afternoon writing complex statements in a query language to analyze the data.

Related Terms

  • Synonym: SQL - structured query language, widely used for managing databases
  • Synonym: programming language - a language used to write instructions for computers

Etymology

From 'query' (a request for information) + 'language'.
