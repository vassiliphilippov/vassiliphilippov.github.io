malevolent program

1. NOUN

/ məˈlɛvələnt ˈproʊɡræm / (AmE), / məˈlɛvələnt ˈprəʊɡræm / (BrE)
a computer program created to cause harm or perform unwanted actions on a system
harmful computer program

Examples

The IT department detected a malevolent program that was stealing user data from their computers.
Antivirus software is designed to protect against any malevolent program that tries to access the system.
Clicking on suspicious email links can install a malevolent program without your knowledge.

Related Terms

  • Synonym: malware - software intended to damage or disable computers
  • Synonym: virus - a type of malicious computer program
  • Antonym: benign program - a computer program causing no harm or unwanted effect
