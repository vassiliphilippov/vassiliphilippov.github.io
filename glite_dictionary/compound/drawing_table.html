drawing table

1. NOUN

/ ˈdrɔɪɪŋ ˌteɪbəl / (AmE), / ˈdrɔːɪŋ ˌteɪbəl / (BrE)
A table with a surface that can be tilted, used especially for drawing or drafting.
table with tilting top for drawing

Examples

She adjusted the height of her drawing table before starting work on the new design.
Many architects prefer using a drawing table because of its adjustable surface.
The art classroom was equipped with a drawing table for each student.

Related Terms

  • Synonym: drafting table - A table with an adjustable top, used for technical drawing or drafting
