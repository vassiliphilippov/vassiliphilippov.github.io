closed loop

1. NOUN

/ kloʊzd luːp / (AmE), / kləʊzd luːp / (BrE)
A control system in which output information is fed back into the system to adjust its operation, keeping the desired state.
control system with feedback

Examples

A closed loop system automatically adjusts the temperature based on real-time sensor feedback.
Unlike open loop systems, a closed loop constantly corrects its output to match the target result.
Engineers recommend using a closed loop for processes that require precise regulation.

Related Terms

  • Antonym: open loop - a control system without a feedback mechanism
