machine-oriented language

1. NOUN

/ məˈʃiːn ɔːrˌiːɛntɪd ˈlæŋɡwɪdʒ / (AmE), / məˈʃiːn ɔːrɪˌɛntɪd ˈlaŋɡwɪdʒ / (BrE)
a programming language designed for use with a specific class of computers, allowing close control over hardware operations
programming language specific to certain computer classes

Examples

Many early computers required programmers to use a machine-oriented language to interact directly with hardware.
Writing in a machine-oriented language can optimize performance but reduces portability across different systems.
The company transitioned from a machine-oriented language to one that could be used on multiple platforms.

Related Terms

  • Synonym: assembly language - a low-level programming language closely related to machine code
  • Antonym: machine-independent language - a programming language that can be run on different types of computers
