diagnostic program

1. NOUN

/ daɪəgˈnɑstɪk ˈproʊɡræm / (AmE), / daɪəɡˈnɒstɪk ˈprəʊɡræm / (BrE)
A software program designed to detect and explain faults in computer hardware or software systems.
software for detecting system faults

Examples

The technician used a diagnostic program to identify the fault in the server hardware.
Running the diagnostic program revealed several errors in the computer's operating system.
Before repairing the software, she launched a diagnostic program to pinpoint the issue.

Related Terms

  • Synonym: utility program - A program that performs a specific task, usually related to managing system resources

Etymology

From 'diagnostic' (from Greek 'diagnōstikos', able to distinguish) and 'program' (from Greek 'programma', a written public notice).
