laundry detergent

1. NOUN

/ ˈlɔn.dri dɪˈtɝ.dʒənt / (AmE), / ˈlɔːn.dri dɪˈtɜː.dʒənt / (BrE)
a cleaning agent in powder or liquid form designed for washing clothes in a washing machine
cleaning agent for washing clothes

Examples

I need to buy more laundry detergent before I run out.
She prefers to use a fragrance-free laundry detergent for sensitive skin.
Please add the laundry detergent to the washing machine before starting the cycle.
