text file

1. NOUN

/ tɛkst faɪl / (AmE), / tɛkst faɪl / (BrE)
A computer file that contains text, usually in plain, human-readable form, often using ASCII or Unicode encoding.
file containing plain text

Examples

Please send the report as a text file rather than a PDF.
She edited the text file using a basic word processor.
The program saves your notes automatically in a text file on your computer.

Related Terms

  • Antonym: binary file - a file that contains data in a format not intended for direct human reading
