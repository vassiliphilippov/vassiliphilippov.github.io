Morse code

1. NOUN

/ mɔrs koʊd / (AmE), / mɔːs kəʊd / (BrE)
an alphabet and numeral encoding system using sequences of dots and dashes, developed for telegraphic communication
system of dots and dashes for telegraphy

Examples

During the emergency, the sailors used Morse code to send a distress signal.
She learned to communicate in Morse code as part of her training.
The flashing light was actually a message in Morse code to warn of danger.

Related Terms

  • Synonym: telegraph code - any code used to transmit messages via telegraph

Etymology

Named after Samuel F. B. Morse, the American inventor.
