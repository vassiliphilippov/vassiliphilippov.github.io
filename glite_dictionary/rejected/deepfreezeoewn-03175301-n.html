electric refrigerator (trade name Deepfreeze) in which food is frozen and stored for long periods of time

Reason

'Deepfreeze' as described here refers to a trademarked brand name for a type of electric freezer. While 'deep freeze' is sometimes used generically for a large freezer, 'Deepfreeze' as a standalone dictionary headword referring only to the specific trade name or as a generic noun for 'electric refrigerator' is not standard in modern dictionaries. The concept is also outdated and brand-specific, so it does not meet inclusion criteria for a general-purpose English dictionary.
