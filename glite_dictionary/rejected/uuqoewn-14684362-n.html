Uuq

Source: WordNet
WordNet ID: oewn-14684362-n
Synset POS: n
Internal ID: Uuq|oewn-14684362-n
a radioactive transuranic element, with atomic number 114

Reason

The concept 'Uuq' refers to a temporary systematic element name (Ununquadium) which is no longer in official usage; the element with atomic number 114 is now known as 'flerovium.' 'Uuq' would not be included in a contemporary English dictionary as a main entry, as it is obsolete nomenclature only used as a placeholder before the element was officially named.
