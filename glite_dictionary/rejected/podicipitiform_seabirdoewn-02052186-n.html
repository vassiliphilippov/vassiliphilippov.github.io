Reason

The term 'podicipitiform seabird' is a technical descriptor combining the order 'Podicipitiformes' (grebes) and a generic 'seabird' label. This phrase is not used as a headword or lexicalized term in standard English dictionaries (such as Oxford), nor is it a common noun, compound, or set phrase. Instead, it describes a broad taxonomic group in scientific usage, and would not appear as an entry distinct from the recognized name 'grebe' (the actual bird group). Therefore, it should not be included as a dictionary headword.
