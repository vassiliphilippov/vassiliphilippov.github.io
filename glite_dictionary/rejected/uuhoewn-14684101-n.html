Uuh

Source: WordNet
WordNet ID: oewn-14684101-n
Synset POS: n
Internal ID: Uuh|oewn-14684101-n
a radioactive transuranic element, with atomic number 116

Reason

'Uuh' refers to the temporary systematic element name ununhexium (element 116) which has since been given the official name 'livermorium.' The entry 'Uuh' as a noun for a chemical element has been made obsolete and would not normally appear as a headword in current dictionaries.
