Cartesian product

Source: WordNet
WordNet ID: oewn-08015510-n
Synset POS: n
Internal ID: Cartesian product|oewn-08015510-n
the set of elements common to two or more sets

Synset Examples

the set of red hats is the intersection of the set of hats and the set of red things

Reason

The concept described (the set of elements common to two or more sets) is actually 'intersection', not 'Cartesian product'. The synset and sense do not match the headword; 'Cartesian product' refers to the set of all ordered pairs from two sets, which is a distinct mathematical concept. Therefore, this concept should not be included under 'Cartesian product'.
