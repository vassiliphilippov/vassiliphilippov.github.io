more than half

Synset Examples

argued for the better part of an hour

Reason

This sense of 'better' as 'more than half' is only used in set phrases like 'the better part of', which differs from the usual adjective or adverb senses. 'Better' on its own is not used with this meaning; only the fixed phrase 'the better part of' would merit a separate entry. Therefore, this concept, defined as 'more than half' for the word 'better' alone, should be excluded.
