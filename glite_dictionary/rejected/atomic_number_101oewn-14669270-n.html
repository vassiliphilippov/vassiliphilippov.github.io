atomic number 101

Source: WordNet
WordNet ID: oewn-14669270-n
Synset POS: n
Internal ID: atomic number 101|oewn-14669270-n
a radioactive transuranic element synthesized by bombarding einsteinium with alpha particles (Md is the current symbol for mendelevium but Mv was formerly the symbol)

Reason

The concept 'atomic number 101' refers to the property of 'being the chemical element with atomic number 101,' which is a generic scientific designation and not a headword. Instead, the actual entry would be 'mendelevium,' the element's name. Dictionaries do not have entries for atomic numbers themselves in this fashion.
