Reason

The concept 'atomic number 97' refers generically to the position of an element in the periodic table, not to the name of the element itself. Dictionaries like Oxford typically include the proper name of the element ('berkelium'), not the numeric form, unless the numeric term is widely used in general English. Therefore, this concept should be excluded.
