Reason

This sense refers only to the generic synset 'n' (noun), but the example is simply referring to 'wine' as a color comparison, not a distinct color lexeme. It is not a separate dictionary entry but rather an example of how 'wine' (the beverage) is used in metaphorical comparison for color. The dictionary entry for 'wine' as an alcoholic beverage covers this implicitly. Oxford Dictionary would not list this as a separate sense or entry for 'wine.'
