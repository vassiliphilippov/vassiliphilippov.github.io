ROM

1. NOUN

/ ɑr oʊ ɛm / (AmE), / ɑːr əʊ ɛm / (BrE)
(computing) Memory in a computer that can be read but not written to, which retains its contents even when the computer is turned off.
read-only computer memory
american

Examples

The BIOS is stored in ROM so it is not erased when the computer is powered off.
Older video game consoles used cartridges containing ROM chips to store games.
You can't modify data on ROM since it's not writable like RAM.

Related Terms

  • Antonym: RAM - computer memory that allows both reading and writing

Etymology

Acronym of 'Read-Only Memory'.
