debugger

1. NOUN

/ dɪˈbʌɡər / (AmE), / dɪˈbʌɡə / (BrE)
A computer program used to detect, locate, and correct errors in other computer programs.
program for finding programming errors

Examples

The software engineer used a debugger to track down a critical bug in the code.
Modern integrated development environments typically include a built-in debugger for convenience.
Learning how to set breakpoints in the debugger can significantly speed up the troubleshooting process.

Related Terms

  • Synonym: to debug - to identify and remove errors from computer hardware or software

Etymology

From 'debug' (to remove errors) + '-er' (agent noun suffix).
