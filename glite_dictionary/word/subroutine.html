subroutine

1. NOUN

/ ˈsʌb.ruˌtin / (AmE), / ˈsʌb.ruː.tiːn / (BrE)
a sequence of instructions within a computer program designed to perform a specific task, which can be used repeatedly as part of the larger program
sequence of program steps for a specific task

Examples

The main program calls the subroutine whenever it needs to sort the data.
Each subroutine is responsible for handling a specific type of calculation.
Carefully commenting your subroutine can make your code easier to maintain and understand.

Related Terms

  • Synonym: function - a named section of a program that returns a value
  • Synonym: procedure - a section of code in a program, often not returning a value

Etymology

From Latin 'sub-' (under) + 'routine', influenced by computing terminology in the mid 20th century.
