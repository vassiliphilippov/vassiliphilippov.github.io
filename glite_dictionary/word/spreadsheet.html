spreadsheet

1. NOUN

/ ˈsprɛdˌʃit / (AmE), / ˈsprɛdʃiːt / (BrE)
a computer program that allows users to organize, analyze, and store data in tabular form, often used for financial calculations and data management
computer program for tabular data management

Examples

The financial team reviewed the spreadsheet to track this month's expenses.
She entered the data into a spreadsheet to create budget forecasts for the project.
Learning how to use a spreadsheet program is essential for most office jobs.

Related Terms

  • Synonym: worksheet - a single page or file within a spreadsheet program

Etymology

Formed from 'spread' + 'sheet', originally referring to paper sheets laid out for calculations, later applied to computer programs.
