RISC

1. NOUN

/ rɪsk / (AmE), / rɪsk / (BrE)
(in computer science) a type of computer architecture that uses a relatively small set of simple instructions for fast processing.
computer architecture with a small instruction set

Examples

Many modern processors are based on the RISC architecture for improved efficiency.
The transition from CISC to RISC designs marked a significant shift in computer engineering.
University courses often include a section on RISC principles in their computer architecture modules.

Related Terms

  • Antonym: CISC - computer architecture with a larger, more complex instruction set

Etymology

Acronym for Reduced Instruction Set Computer.
