RAM

1. NOUN

/ ræm / (AmE), / ræm / (BrE)
Random Access Memory; a type of computer memory used for temporary storage, allowing data to be accessed and changed quickly while the computer is running.
main temporary computer memory

Examples

My computer is running slowly because it doesn't have enough RAM.
Upgrading the RAM can improve your system's performance significantly.
The program will not work properly unless the machine has at least 8GB of RAM.

Related Terms

  • Synonym: memory - computer's storage for data and programs

Etymology

Acronym of Random Access Memory.
