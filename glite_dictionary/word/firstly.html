firstly

1. ADV

/ ˈfɝst.li / (AmE), / ˈfɜːst.li / (BrE)
Used to introduce the first point or reason in a list of items or arguments.
introducing the first point

Examples

Firstly, we need to address the most urgent problem before discussing the others.
Firstly, let me thank everyone for attending this meeting.
Firstly, you should read the instructions carefully before beginning the test.

Synset Examples

first we must consider the garter snake

Related Terms

  • Degree Higher: secondly - used to introduce the second point in a list
  • Synonym: first - before anything else (as an adverb); in the first place
