ASCII

1. NOUN

/ ˈæski / (AmE), / ˈæski / (BrE)
(in computing) a character encoding standard in which each letter, number, or symbol is represented by a string of seven binary digits; used for information exchange between computers.
computing character encoding standard

Examples

Many early email systems used ASCII to represent text.
Please make sure your file is saved in ASCII format for compatibility.
The ASCII table assigns numeric values to letters and symbols.

Etymology

Acronym from American Standard Code for Information Interchange.
