unary

1. NOUN

/ ˈjuːˌnɛri / (AmE), / ˈjuː.nəri / (BrE)
A numeral system, also called base-1, in which a single symbol is repeated as many times as the value of the number being represented.
base-1 numeral system

Examples

The unary numeral system represents seven as seven tally marks.
Some primitive counting methods use a unary system, where each stick or mark represents one item.
Counting in unary is inefficient for large numbers, but simple for small amounts.

Related Terms

  • Degree Higher: binary - a base-2 numeral system using only the digits 0 and 1

Etymology

From Latin 'unarius' (consisting of one), from 'unus' (one).

2. ADJ

/ ˈjuːˌnɛri / (AmE), / ˈjuː.nəri / (BrE)
Consisting of or involving a single element, component, or operand, especially in mathematics or logic.
involving a single element or operand

Examples

A unary operator takes only one argument in a programming language.
Negation is a typical example of a unary operation in logic.
The unary minus changes the sign of a number.

Synset Examples

in a unary operation in a mathematical system one element is used to yield a single result

Related Terms

  • Antonym: binary - involving two elements or components

Etymology

From Latin 'unarius', from 'unus' meaning 'one'.

WordNet ID: oewn-02223236-s
Synset POS: s
