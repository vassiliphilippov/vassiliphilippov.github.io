function

1. NOUN

/ ˈfʌŋkʃən / (AmE), / ˈfʌŋkʃən / (BrE)
(Mathematics) A relation in which each element of a given set (the domain) is paired with exactly one element of another set (the range).
mathematical relation with single-valued outputs

Examples

The square root function is defined only for non-negative numbers.
Every value in the domain must correspond to one output in a function.
In calculus, the derivative measures how a function changes as its input changes.

Related Terms

  • Synonym: mapping - mathematical relation pairing elements from one set to another

Etymology

From Latin functio (performance, execution), from fungi (to perform).
