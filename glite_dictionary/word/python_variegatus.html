Python variegatus

1. NOUN

/ ˈpaɪθɑn ˌvær.iˈɡeɪ.təs / (AmE), / ˈpaɪθɒn ˌvær.iˈɡeɪ.təs / (BrE)
an Australian python species distinguished by a variegated pattern on its back; also known as the variegated python
Australian python species with patterned back

Examples

The Python variegatus can be found in diverse habitats across northern Australia.
Herpetologists are studying the unique coloration of the Python variegatus.
A Python variegatus was spotted basking on a rocky outcrop during the expedition.

Etymology

From Latin 'variegatus' (variegated), referring to its varied color pattern; 'Python' is the genus name from Greek mythology.
