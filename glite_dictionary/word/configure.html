to configure

1. VERB

/ kənˈfɪɡjɚ / (AmE), / kənˈfɪɡjə / (BrE)
to arrange or set up something, especially a device or system, for a specific purpose or to meet particular requirements
set up for a specific purpose

Examples

Before using the software, you must configure the settings to match your preferences.
It took me an hour to configure the network for all the new devices.
Engineers will configure the system so it can handle increased demand.

Synset Examples

configure my new computer
configure a plane for a combat mission

Related Terms

  • Synonym: to set up - to arrange or prepare something so that it is ready for use
  • Synonym: to install - to put in place or set up for use or service

Etymology

From Latin configurare (to shape, fashion, form), from con- (together) + figurare (to form, shape)
