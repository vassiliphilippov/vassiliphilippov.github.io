scratchpad

1. NOUN

/ ˈskrætʃˌpæd / (AmE), / ˈskrætʃpad / (BrE)
(computing) a high-speed internal memory used for temporary storage of data or instructions.
temporary high-speed computer memory

Examples

The processor uses a scratchpad to store frequently accessed variables for faster computation.
Some embedded systems employ a dedicated scratchpad instead of conventional RAM.
Programmers optimize performance by utilizing the available scratchpad memory.

Related Terms

  • Synonym: cache - a hardware or software component that stores data so future requests are served faster
