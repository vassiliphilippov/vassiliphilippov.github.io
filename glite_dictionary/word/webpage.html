webpage

1. NOUN

/ ˈwɛbˌpeɪdʒ / (AmE), / ˈwɛb.peɪdʒ / (BrE)
A document on the World Wide Web that can be viewed by anyone using a web browser.
document on the World Wide Web

Examples

The company's webpage provides information about its products and services.
She updated the webpage to reflect the latest news.
Each webpage on the site has a unique address called a URL.

Related Terms

  • Synonym: website - a set of related web pages under a single domain name

Etymology

Formed from 'web' (the World Wide Web) + 'page' (a single sheet in a book).
