scheduler

1. NOUN

/ ˈskɛdʒuˌlɚ / (AmE), / ˈʃɛdjuːlə / (BrE)
a computer program or component that arranges jobs or tasks to be executed by a computer in an appropriate order
computer component/program that sequences jobs

Examples

The operating system uses a scheduler to manage multiple processes efficiently.
When the scheduler fails, jobs may not run in the correct sequence.
Each core has its own scheduler to allocate tasks dynamically.

Related Terms

  • Synonym: queue manager - a system that manages job queues in computing
  • Synonym: dispatcher - a component that assigns jobs or resources in computing
