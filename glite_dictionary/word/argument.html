argument

1. NOUN

/ ˈɑːrɡjəmənt / (AmE), / ˈɑːɡjʊmənt / (BrE)
A variable or quantity in a mathematical or logical function or expression whose value determines the outcome of another variable (the dependent variable).
variable in math or logic determining outcome

Examples

In the function g(t) = t^2 + 1, the argument is the variable t.
The value of the argument determines the output of the sine function.
When programming, you often need to specify an argument for a function to run properly.

Synset Examples

if f(x)=y, x is the independent variable

Etymology

From Latin argumentum (evidence, proof, reason), from arguere (to make clear, prove).
