enumerable

1. ADJ

/ ɪˈnuːmərəbəl / (AmE), / ɪˈnjuːmərəbəl / (BrE)
able to be counted; capable of being enumerated.
able to be counted

Examples

The collection is enumerable and contains exactly one hundred artifacts.
Only enumerable risks were included in the insurance policy assessment.
In mathematics, a set is enumerable if its elements can be matched one-to-one with the natural numbers.

Synset Examples

countable sins
numerable assets

Related Terms

  • Synonym: countable - able to be counted; not infinite
  • Antonym: uncountable - not able to be counted

Etymology

From Latin 'enumerare' (to count out), via Medieval Latin 'enumerabilis'.
