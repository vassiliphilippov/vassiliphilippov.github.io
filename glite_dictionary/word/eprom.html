EPROM

1. NOUN

/ ˈiːˌprɑm / (AmE), / ˈiːˌprɒm / (BrE)
a type of computer memory chip that can be erased using ultraviolet light and reprogrammed with new data
erasable programmable read-only memory chip

Examples

The engineer replaced the faulty EPROM to fix the device's memory issues.
You must use ultraviolet light to erase the data stored in an EPROM.
Many older computers rely on EPROMs to store their firmware.

Related Terms

  • Degree Higher: EEPROM - a memory chip that can be electrically erased and reprogrammed
  • Degree Lower: ROM - a memory chip that cannot be erased or rewritten

Etymology

Acronym from Erasable Programmable Read-Only Memory.
