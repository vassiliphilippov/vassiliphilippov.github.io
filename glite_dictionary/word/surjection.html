surjection

1. NOUN

/ sərˈdʒɛkʃən / (AmE), / səˈdʒɛkʃ(ə)n / (BrE)
A mathematical function in which every element of the codomain is the image of at least one element of the domain; an onto mapping.
function mapping onto its codomain

Examples

A function f: A → B is a surjection if every element of B is mapped to by some element of A.
The professor illustrated a surjection using a diagram with arrows covering all elements in the codomain.
In mathematics, proving that a mapping is a surjection can be essential for solving certain equations.

Synset Examples

In fact, if we define a ρ-kernel as the kernel of any surjection of the two graphs of interest, then there are ρ-kernels whose maximum excess does not vanish.

Related Terms

  • Antonym: injection - a function mapping each element of the domain to a unique element of the codomain (one-to-one)
  • Degree Higher: bijection - a function that is both an injection and a surjection

Etymology

From French 'sur-' (over) + 'jection' (from 'projection').
