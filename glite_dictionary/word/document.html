document

1. NOUN

/ ˈdɑː.kjə.mənt / (AmE), / ˈdɒk.jʊ.mənt / (BrE)
A computer file containing text, and possibly formatting instructions, typically using ASCII or similar character encoding.
computer file containing text

Examples

She attached the document to her email before sending it to the client.
The software can open any text document saved in a common format.
After editing the document, remember to save your changes to avoid losing your work.

Related Terms

  • Synonym: file - a collection of data or information stored on a computer
Source: WordNet

/ ˈdɑː.kjə.mənt / (AmE), / ˈdɒk.jʊ.mənt / (BrE)
an official or written paper that provides information or evidence
