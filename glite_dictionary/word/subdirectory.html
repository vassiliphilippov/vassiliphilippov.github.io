subdirectory

1. NOUN

/ ˌsʌb.dɪˈrɛk.tə.ri / (AmE), / ˌsʌb.dɪˈrɛk.tər.i / (BrE)
A directory within another directory on a computer, used to organize files and folders hierarchically.
directory within another directory (computing)

Examples

Each project has its own subdirectory to keep files organized.
The images are stored in a subdirectory called 'photos'.
You must create a new subdirectory before installing the software.

Related Terms

  • Degree Higher: directory - a file system structure that contains references to other files or directories

Etymology

From 'sub-' meaning 'under' and 'directory' (from Latin 'dirigere', to direct).
