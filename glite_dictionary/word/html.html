HTML

1. NOUN

/ ˌeɪtʃ ˌtiː ˌɛm ˈɛl / (AmE), / ˌeɪtʃ ˌtiː ˌɛm ˈɛl / (BrE)
a markup language consisting of a set of tags and rules used to create and structure documents on the World Wide Web
markup language for web pages

Examples

Web developers use HTML to design and structure web pages.
The document was written entirely in HTML for compatibility with most browsers.
Learning HTML is a fundamental step for anyone interested in web development.

Related Terms

  • Synonym: XML - a flexible markup language used for data storage and transfer

Etymology

Abbreviation of HyperText Markup Language.
