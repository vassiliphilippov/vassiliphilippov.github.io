Windows

1. PROPN

/ ˈwɪndoʊz / (AmE), / ˈwɪndəʊz / (BrE)
A trademarked operating system for personal computers with a graphical user interface, produced by Microsoft.
Microsoft operating system

Examples

Many offices still use Windows as their primary operating system.
The latest version of Windows offers improved security features.
He installed Windows on his new laptop after removing the previous software.

Etymology

Trade name introduced by Microsoft in 1985, referring to the software's use of on-screen 'windows' for running applications.
