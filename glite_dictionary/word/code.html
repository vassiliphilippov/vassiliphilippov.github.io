code

1. NOUN

/ koʊd / (AmE), / kəʊd / (BrE)
a set of letters, numbers, or symbols given to represent, classify, or identify something
identifying set of letters, numbers, or symbols

Examples

Every employee must enter their unique code to access the building.
The product has a bar code on its packaging for easy scanning.
He memorized the security code to open the safe.

Related Terms

  • Synonym: password - a secret word or phrase used to gain access to something
  • Synonym: cipher - a secret or disguised way of writing; a code

Etymology

From Latin 'codex' (book, code).

Source: WordNet
WordNet ID: oewn-89822606-n
Synset POS: n

2. NOUN

/ koʊd / (AmE), / kəʊd / (BrE)
the symbolic representation of instructions or data in a computer program
instructions in a computer program

Examples

She spent the evening writing code for her new mobile app.
The company reviews all code before releasing software updates.
Learning how to read code can be challenging at first.

Related Terms

  • Synonym: program - a sequence of instructions for a computer

Etymology

From Latin codex, codicis (book, code of laws).

Source: WordNet
WordNet ID: oewn-06367301-n

3. NOUN

/ koʊd / (AmE), / kəʊd / (BrE)
A system of words, letters, symbols, or signals used to represent information for the purpose of secrecy or conciseness in communication.
system of symbols for secret or brief communication

Examples

Military officers often send messages in code to prevent enemies from understanding them.
Can you crack the secret code hidden in this letter?
Communicating in code allowed them to keep their plans confidential.

Related Terms

  • Synonym: cipher - a secret or disguised way of writing; a code

Etymology

From Latin 'codex' (book, written tablet).

Source: WordNet
WordNet ID: oewn-06365341-n
Synset POS: n

4. NOUN

/ koʊd / (AmE), / kəʊd / (BrE)
a set of rules, principles, or laws, especially ones that are written and formally accepted, for regulating behavior or procedures
set of written rules or laws

Examples

The school follows a strict code of conduct that all students must respect.
French civil law is organized under the Napoleonic Code.
Journalists are expected to adhere to a professional code of ethics.

Related Terms

  • Synonym: statute - a written law passed by a legislative body
  • Synonym: regulation - a rule or directive made and maintained by an authority
  • Synonym: law - the system of rules recognized by a community or country

Etymology

From Latin codex (book, law code), later code in Old French, then Middle English.

Source: WordNet

5. VERB

/ koʊd / (AmE), / kəʊd / (BrE)
to attach numbers, symbols, or other codes to something in order to identify or classify it
to attach identification codes

Examples

Please code all the files according to their importance.
The lab technician coded the samples to ensure confidentiality.
After coding each item, we placed them in separate bins.

Synset Examples

Code the pieces with numbers so that you can identify them later

Related Terms

  • Synonym: label - to attach a label to something for identification
  • Antonym: to decipher - to interpret or decode a code or message

Etymology

From French *code* (system of laws, code), from Latin *codex* (book).

Source: WordNet
WordNet ID: oewn-01592644-v
Synset POS: v

6. VERB

/ koʊd / (AmE), / kəʊd / (BrE)
to convert information or a message into a system of symbols, numbers, or signs, often for secrecy or computer processing
convert into symbols or code

Examples

All the important messages were coded before being sent to headquarters.
She learned how to code sensitive data to protect users' privacy.
The instructions were coded so that only authorized staff could understand them.

Synset Examples

We should encode the message for security reasons

Related Terms

  • Synonym: to encode - convert information into code, especially for secrecy or transmission
  • Antonym: to decode - convert coded information back into ordinary language

Etymology

From French *code* (system of laws), from Latin *codex* (book of laws).

Source: WordNet
WordNet ID: oewn-00996121-v
Synset POS: v
