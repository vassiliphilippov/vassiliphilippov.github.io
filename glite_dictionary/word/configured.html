configured

1. ADJ

/ kənˈfɪɡjɚd / (AmE), / kənˈfɪɡjəd / (BrE)
arranged or set up in a particular form or configuration
arranged in a specific way

Examples

The network was configured to maximize security and efficiency.
Each device must be configured properly before being added to the system.
The software comes pre-configured for basic use but allows for customization.

Synset Examples

a magnet is surrounded by a configured field

Related Terms

  • Synonym: arranged - put in a particular order or position
  • Antonym: disorganized - not arranged or structured

Etymology

From the verb 'configure', which is from Latin 'configurare' (to shape, fashion).
