usable

1. ADJ

/ ˈjuːzəbəl / (AmE), / ˈjuːzəbəl / (BrE)
able to be used for a particular purpose; fit or suitable for use
able to be used

Examples

The equipment was still usable after minor repairs.
We collected several usable pieces of furniture from the donation drive.
Only a few of the suggestions were actually usable in our project.

Synset Examples

usable byproducts

Related Terms

  • Synonym: useful - able to be used in a helpful or effective way
  • Antonym: unusable - not able to be used for its intended purpose

Etymology

From 'use' + '-able', formed in English.

able to be used for a particular purpose; fit or suitable for use.
able to be used

Examples

3. ADJ

/ ˈjuzəbəl / (AmE), / ˈjuːzəbəl / (BrE)
able to be used for a particular purpose because it is in good enough condition or suitable
able to be used; fit for use

Examples

The old bicycle was surprisingly usable after a few repairs.
Please ensure all equipment is usable before the event begins.
Most of the donated computers are still usable despite their age.

Synset Examples

the toaster was still functional even after being dropped
the lawnmower is a bit rusty but still usable
an operational aircraft

Related Terms

  • Antonym: unusable - not able to be used; not fit for use
  • Synonym: functional - in working order; capable of operating

Etymology

From use + -able, meaning capable of being used.

Source: WordNet
