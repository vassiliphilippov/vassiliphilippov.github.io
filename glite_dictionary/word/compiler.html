compiler

1. NOUN

/ kəmˈpaɪlɚ / (AmE), / kəmˈpaɪlə / (BrE)
(computing) a program that translates instructions written in a high-level programming language into machine code or an intermediate form, such as assembly language.
program translating code from one language to another

Examples

The C++ compiler generated an executable file from the source code.
Installing the correct compiler is essential for building the software on your computer.
She learned how to optimize code to improve compiler performance.

Related Terms

  • Antonym: interpreter - a program that directly executes instructions without compiling them first

Etymology

From 'compile' (to assemble or collect) + '-er' (agent noun suffix).

Source: WordNet
WordNet ID: oewn-06585776-n
Synset POS: n

2. NOUN

/ kəmˈpaɪlər / (AmE), / kəmˈpaɪlə / (BrE)
a person who collects and arranges information from different sources, often for use in a book, list, or report
person who compiles information

Examples

She served as the compiler of a comprehensive reference guide to world literature.
The compiler meticulously gathered statistics from numerous sources.
A good compiler credits all contributors clearly in the introduction.

Related Terms

  • Synonym: editor - person who prepares written material for publication
  • Antonym: author - person who writes a book or article

Etymology

From Latin compilator (collector, plagiarist), from compilare (to compile) via Old French compiler.

Source: WordNet
WordNet ID: oewn-09966436-n
Synset POS: n
