syllabary

1. NOUN

/ ˈsɪl.əˌbɛr.i / (AmE), / ˈsɪl.ə.bəri / (BrE)
a set of written symbols, each representing a syllable, that together make up a writing system
writing system based on syllables

Examples

The Japanese script katakana is a well-known syllabary used for foreign words.
Unlike an alphabet, a syllabary assigns a unique character for each syllable in the language.
The Cherokee writing system is a syllabary developed in the early nineteenth century.

Related Terms

  • Antonym: alphabet - a writing system where each character represents a phoneme

Etymology

From New Latin syllabārium, from Latin syllaba (syllable).
