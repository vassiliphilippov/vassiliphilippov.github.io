stringly-typed

1. ADJ

/ ˈstrɪŋli taɪpt / (AmE), / ˈstrɪŋli taɪpt / (BrE)
(of a programming language or code) characterized by excessive or inappropriate use of strings to represent data of various types, often leading to type-related issues and bugs.
prone to using strings excessively in programming

Examples

The legacy system is notorious for being stringly-typed, which makes code maintenance difficult.
Some developers avoid stringly-typed programming to reduce errors caused by incorrect data types.
Using JSON with dynamic keys can sometimes lead to stringly-typed code.

Synset Examples

Developers struggled with the stringly-typed variables in the code.
JavaScript is often criticized for being stringly-typed.

Related Terms

  • Antonym: strongly typed - a programming language enforcing strict type rules
  • Degree Lower: weakly typed - a programming language allowing flexible or loose type rules

Etymology

Blend of 'string' and 'strongly typed', humorously altered in programming jargon.
