Java

1. NOUN

/ ˈdʒɑː.və / (AmE), / ˈdʒɑː.və / (BrE)
a high-level, platform-independent, object-oriented programming language widely used for building computer applications
programming language

Examples

She decided to learn Java to enhance her software development skills.
Many enterprise applications are built using Java for its reliability and portability.
The engineering course covered basic Java syntax and object-oriented programming concepts.

Related Terms

  • Synonym: Python - a high-level programming language known for its readability
  • Synonym: C++ - a general-purpose programming language with object-oriented and generic features

Etymology

Named after Java coffee, as coffee was consumed in large amounts by its creators.

2. PROPN

/ ˈdʒɑvə / (AmE), / ˈdʒɑːvə / (BrE)
an island in Indonesia, lying to the south of Borneo and noted as one of the most densely populated regions in the world
Indonesian island south of Borneo

Examples

The island of Java is home to Indonesia’s capital, Jakarta.
Many tourists visit Java to experience its unique blend of culture and nature.
Java is renowned for its rich history and bustling cities.

Etymology

From Javanese 'Jawa'.
