cryptogram

1. NOUN

/ ˈkrɪptoʊˌɡræm / (AmE), / ˈkrɪptəˌɡræm / (BrE)
a piece of writing that is in code or cipher and intended to be deciphered
a message written in code or cipher

Examples

The detective stared at the mysterious cryptogram, searching for a hidden message within the symbols.
She enjoys solving the daily newspaper cryptogram before breakfast.
A complex cryptogram was sent to the team, challenging them to break the code.

Related Terms

  • Synonym: cipher - a secret or disguised way of writing; a code
  • Antonym: to decipher - to convert a text written in code into normal language

Etymology

From Greek kryptos (hidden) + gramma (letter, something written).
