HTTP

1. NOUN

/ ˌeɪtʃ ˌtiː ˌtiː ˈpiː / (AmE), / ˌeɪtʃ ˌtiː ˌtiː ˈpiː / (BrE)
a protocol used for transferring hypertext requests and information on the World Wide Web
web protocol for transferring information

Examples

Most websites use HTTP to transfer data between your browser and their servers.
The address bar displayed 'http://' at the start of every URL in the early days of the web.
Learning about HTTP is essential for anyone interested in web development.

Related Terms

  • Degree Higher: HTTPS - secure version of HTTP

Etymology

Abbreviation of HyperText Transfer Protocol.
