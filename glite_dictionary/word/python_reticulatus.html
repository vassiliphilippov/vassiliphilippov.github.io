Python reticulatus

1. NOUN

/ ˈpaɪθɑn rɛˌtɪkjəˈleɪtəs / (AmE), / ˈpaɪθən rɪˌtɪkjʊˈleɪtəs / (BrE)
A species of python native to Southeast Asia and the East Indies, recognized as the largest snake in the world.
largest species of python; reticulated python

Examples

The Python reticulatus can reach lengths of over six meters in the wild.
Researchers tracked a Python reticulatus as it moved through the mangroves of Indonesia.
I read an article about a record-breaking Python reticulatus found in Malaysia.

Related Terms

  • Synonym: reticulated python - common name for Python reticulatus, the largest python species

Etymology

From Latin: 'python' (a kind of serpent) and 'reticulatus' (net-like, reticulated); refers to the snake's net-like pattern.
