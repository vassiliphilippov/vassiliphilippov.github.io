run-time

1. NOUN

/ ˈrʌn taɪm / (AmE), / ˈrʌn taɪm / (BrE)
The time at which a computer program or multimedia application is being executed.
when a program is executing

Examples

Memory errors may occur at run-time if the code is not properly tested.
The run-time of the application can vary depending on the device.
Developers often debug programs to reduce run-time errors.

Related Terms

  • Synonym: execution - the act of carrying out a program's instructions

Etymology

Formed from 'run' + 'time', reflecting the time while a program runs.

2. NOUN

/ ˈrʌnˌtaɪm / (AmE), / ˈrʌnˌtʌɪm / (BrE)
the period of time taken for a computer program to execute or be in operation
the duration a program runs

Examples

The run-time of the new algorithm was significantly less than that of the previous version.
Developers worked to minimize the run-time to improve user experience.
The application crashed midway through its run-time due to a memory leak.

Related Terms

  • Synonym: execution time - the time during which a program is running on a computer

Etymology

Formed from 'run' (to operate) and 'time,' commonly used in computing since the mid-20th century.

Source: WordNet
WordNet ID: oewn-15287171-n
Synset POS: n
