octal

1. ADJ

/ ˈɑk.t̬l̩ / (AmE), / ˈɒk.təl / (BrE)
relating to or using a number system with eight as its base
relating to the base-8 number system

Examples

File permissions in Unix are often specified using octal values.
The computer program displays numbers in octal instead of decimal.
Each octal digit represents three binary digits.

Synset Examples

an octal digit

Related Terms

  • Antonym: decimal - relating to the base-10 number system
  • Antonym: binary - relating to the base-2 number system
  • Antonym: hexadecimal - relating to the base-16 number system

Etymology

From Latin octo (eight) + -al (forming adjectives).
