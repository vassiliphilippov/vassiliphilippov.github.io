OS

1. NOUN

/ oʊ ɛs / (AmE), / əʊ ɛs / (BrE)
An abbreviation for 'operating system'; the software that manages computer hardware and provides common services for computer programs.
abbreviation for operating system in computing
american

Examples

The new version of the OS supports multiple user accounts.
You need to update your OS to install the latest software.
Which OS do you prefer for your laptop, Windows or macOS?

Related Terms

  • Synonym: operating system - software managing computer hardware and software resources

Etymology

Abbreviation of 'operating system'.
