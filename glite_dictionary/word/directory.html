directory

1. NOUN

/ dəˈrɛktəri / (AmE), / dɪˈrɛktəri / (BrE)
a book or digital listing containing an alphabetical list of names and addresses, often of people or organizations
alphabetical list of names and addresses

Examples

You can find her phone number in the directory under her last name.
The company updated its employee directory to include email addresses.
A hospital directory is available at the information desk for visitors.

Related Terms

  • Synonym: listing - a list or catalog of items
  • Synonym: index - an alphabetical listing of names and topics with page references

Etymology

From Latin 'directorium' (guide, directory), meaning 'that which guides or directs'.

2. NOUN

/ dəˈrɛktəri, daɪˈrɛktəri / (AmE), / dɪˈrɛktəri, daɪˈrɛktəri / (BrE)
(computing) a file or folder on a computer that contains a list of files and subdirectories.
computing: list of computer files and folders

Examples

The software let me create a new directory to store all my project files.
Please move the document to the main directory before uploading it.
He searched the directory for a specific file he had saved last week.

Related Terms

  • Synonym: folder - a virtual container within a computer's file system, used for organizing files
