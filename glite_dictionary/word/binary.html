binary

1. ADJ

/ ˈbaɪnɛri / (AmE), / ˈbʌɪnəri / (BrE)
relating to a number system that has 2 as its base, using only the digits 0 and 1
relating to the base-2 number system

Examples

In computer science, all data is represented in binary form, using combinations of 0s and 1s.
The binary number system is essential for digital electronics and computing.
She converted the decimal value to its binary equivalent for the assignment.

Synset Examples

a binary digit

Related Terms

  • Antonym: decimal - relating to the base-10 number system

Etymology

From Latin 'binarius' (consisting of two).

Source: WordNet
WordNet ID: oewn-02675405-a
Synset POS: a

2. NOUN

/ ˈbaɪˌnɛri / (AmE), / ˈbʌɪnəri / (BrE)
An astronomical system consisting of two stars that revolve around each other under their mutual gravitation.
system of two stars orbiting each other

Examples

The astronomers spent years observing the binary to determine the masses of both stars.
A binary can sometimes appear as a single point of light through a telescope.
New data have revealed that the binary is gradually losing energy and the stars are moving closer together.

Related Terms

  • Synonym: binary star - a system of two stars orbiting each other

Etymology

From Latin 'binarius' (consisting of two).

Source: WordNet
WordNet ID: oewn-09243977-n
Synset POS: n

4. NOUN

/ ˈbaɪˌnɛri / (AmE), / ˈbɪnəri / (BrE)
A compiled program or executable file that can be run on a particular operating system; not compatible with other operating systems without modification.
compiled computer program or executable file

Examples

Due to differences in architecture, this binary won't run on your Mac.
The developer released new binaries for Windows and Linux systems.
Before distributing software, you must compile the source code into a binary.

Synset Examples

the same source code can be compiled to produce different binaries for different operating systems

Related Terms

  • Synonym: executable - A file containing a program that can be executed on a computer

Etymology

From binary, originally relating to the binary numeral system (0 and 1) used in computers, extended to mean compiled files that are in binary form.

Source: WordNet
WordNet ID: oewn-06583139-n
Synset POS: n
