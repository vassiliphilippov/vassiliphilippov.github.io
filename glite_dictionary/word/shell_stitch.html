to shell stitch

1. VERB

/ ʃɛl stɪtʃ / (AmE), / ʃɛl stɪtʃ / (BrE)
to create by using the shell stitch technique in crochet or similar crafts
make using the shell stitch technique

Examples

She decided to shell stitch a border around the baby blanket for a decorative finish.
After learning how to shell stitch, he made his first scarf using this technique.
The instructor demonstrated how to shell stitch rows to create a lacy pattern in the shawl.

Related Terms

  • Synonym: crochet - to make fabric using a hooked needle

Etymology

From 'shell' (referring to the shell-like appearance of the stitch) + 'stitch.' Used in crafts terminology.
