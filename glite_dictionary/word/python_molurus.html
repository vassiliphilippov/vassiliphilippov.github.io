Python molurus

1. PROPN

/ ˈpaɪθɑn moʊˈlʊərəs / (AmE), / ˈpaɪθən məʊˈlɔːrəs / (BrE)
a very large nonvenomous python species found in southern and Southeast Asia, also known as the Indian python
a large python species of South and Southeast Asia

Examples

The Python molurus can reach lengths exceeding six meters in the wild.
Conservationists are concerned about the declining numbers of Python molurus in certain regions.
A mature Python molurus is capable of constricting large prey such as deer or wild boar.

Related Terms

  • Synonym: Indian python - common name for Python molurus

Etymology

From Latinized Greek: 'Python' (mythical serpent) and specific epithet 'molurus', from Greek 'mōlourós' (deadly).
