Python

1. NOUN

/ ˈpaɪ.θɑn / (AmE), / ˈpaɪ.θən / (BrE)
a high-level, general-purpose programming language, developed by Guido van Rossum, known for its readability and versatility
programming language developed by Guido van Rossum

Examples

Many developers prefer Python for its simple syntax and powerful libraries.
She built a web application using Python in just a few days.
The computer science course teaches Python as the first programming language.

Related Terms

  • Synonym: programming language - a computer language used to write software programs

Etymology

Named after the British comedy group Monty Python, a favorite of the language's creator.

2. PROPN

/ ˈpaɪθɑn / (AmE), / ˈpaɪθɒn / (BrE)
(In Greek mythology) a giant serpent or dragon that was slain by the god Apollo at Delphi.
mythical serpent slain by Apollo

Examples

According to legend, Apollo defeated the monstrous Python at Delphi, claiming the site as his own.
The oracle at Delphi was said to be located where the Python once lived.
In Greek myths, Python is often depicted as a guardian serpent killed by Apollo.

Etymology

From Greek Πύθων (Pythōn), the name of the mythical serpent.

3. NOUN

/ ˈpaɪ.θɑn / (AmE), / ˈpaɪ.θən / (BrE)
A genus of non-venomous constrictor snakes native to Africa and Asia, commonly known as pythons.
genus of large constrictor snakes

Examples

The Python is one of the largest snakes in the world, capable of reaching lengths over six meters.
Several species within the Python genus are popular among exotic pet enthusiasts.
Scientists recently discovered a new Python species in Southeast Asia.

Related Terms

  • Synonym: boa - large non-venomous constrictor snake of the family Boidae

Etymology

From Ancient Greek Πύθων (Púthōn), a mythical serpent slain by Apollo.
