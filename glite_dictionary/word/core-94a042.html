core

1. NOUN

/ kɔɹ / (AmE), / kɔː / (BrE)
(computer science) a small ferrite ring formerly used in early computers as a memory element to store a single bit of data, now replaced by semiconductor memory.
magnetic memory element in early computers

Examples

Each memory core stores a single bit of information in early computer systems.
Technicians had to manually thread wires through every core in the assembly process.
The development of semiconductor memory made the magnetic core obsolete.

Synset Examples

each core has three wires passing through it, providing the means to select and detect the contents of each bit

Related Terms

  • Synonym: core memory - magnetic memory using ferrite cores to store data
