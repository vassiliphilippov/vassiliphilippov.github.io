to initialise

1. VERB

/ ɪˈnɪʃəˌlaɪz / (AmE), / ɪˈnɪʃəlaɪz / (BrE)
to assign an initial value to a variable or data structure in a computer program
set an initial value in programming

Examples

Before using the array, you must initialise all its elements to zero.
The program will not run correctly if you forget to initialise the variables.
He spent the morning initialising the new database to prepare it for testing.

Related Terms

  • Synonym: to initialize - assign an initial value to a computer program (US spelling)

Etymology

From Latin 'initium' (beginning) + English suffix '-ise'.

Source: WordNet
WordNet ID: oewn-00702662-v
Synset POS: v

2. VERB

/ ɪˈnɪʃəˌlaɪz / (AmE), / ɪˈnɪʃəlaɪz / (BrE)
to prepare or format (a disk) by dividing it into marked sectors so that it can store data
prepare a disk for data storage

Examples

You should always initialise a new disk before trying to save files on it.
The technician initialised the hard drive to eliminate previous data partitions.
After initialising the storage device, it was ready for software installation.

Synset Examples

Please format this disk before entering data!

Related Terms

  • Synonym: format - to prepare a storage medium for use, typically by dividing it into sectors

Etymology

From Latin 'initium' (beginning) + '-ise', via French 'initialiser', referring to making something ready at the start.

Source: WordNet
WordNet ID: oewn-00564629-v
Synset POS: v
