monolingual

1. NOUN

/ ˌmoʊnəˈlɪŋɡwəl / (AmE), / ˌməʊnəˈlɪŋɡwəl / (BrE)
a person who speaks only one language
person speaking only one language

Examples

Many monolingual Americans travel abroad with only English as their means of communication.
Being monolingual can sometimes limit career opportunities in a globalized world.
The conference attracted both monolingual and bilingual participants from around the globe.

Related Terms

  • Antonym: bilingual - a person who speaks two languages fluently

Etymology

From mono- (one) + Latin lingua (language) via French 'monolingue'.

Source: WordNet
WordNet ID: oewn-10348654-n
Synset POS: n

2. ADJ

/ ˌmoʊnəˈlɪŋɡwəl / (AmE), / ˌmɒnəˈlɪŋɡwəl / (BrE)
using or able to use only one language; relating to or expressed in only one language.
using or knowing only one language

Examples

Although he lives in a multicultural city, he remains strictly monolingual.
This is a monolingual edition, presented only in English.
Many monolingual children start learning a second language at school.

Synset Examples

monolingual speakers
a monolingual dictionary

Related Terms

  • Antonym: bilingual - knowing or using two languages
  • Antonym: multilingual - knowing or using several languages

Etymology

From mono- (one) + Latin lingua (language).

Source: WordNet
WordNet ID: oewn-01549594-a
Synset POS: a
