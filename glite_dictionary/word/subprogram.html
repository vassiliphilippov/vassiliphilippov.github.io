subprogram

1. NOUN

/ ˈsʌbˌproʊɡræm / (AmE), / ˈsʌbˌprəʊɡræm / (BrE)
A sequence of programmed instructions that performs a specific task as part of a larger computer program.
part of a computer program with a specific task

Examples

The main program calls each subprogram as needed to complete its tasks.
Modifying a single subprogram can sometimes fix multiple bugs in the software.
Each subprogram in the library is designed to handle a specific type of calculation.

Related Terms

  • Synonym: function - a named section of a program that performs a specific task
  • Synonym: procedure - a set of coded instructions in a computer program
  • Antonym: main program - the primary body of a computer program, calling subprograms as needed

Etymology

Formed from 'sub-' (meaning 'under' or 'part of') and 'program'.
