FTP

1. NOUN

/ ˌɛf ˌti ˈpi / (AmE), / ˌɛf ˌtiː ˈpiː / (BrE)
a standard network protocol used to transfer files between computers over a network
file transfer network protocol

Examples

You can use FTP to upload your website files to the server.
Many companies block FTP because it is not secure by default.
I set up an FTP server to share documents with remote colleagues.

Related Terms

  • Synonym: SFTP - a secure version of FTP using SSH for encryption

Etymology

Abbreviation of 'File Transfer Protocol'.

Source: WordNet
WordNet ID: oewn-06678115-n
Synset POS: n

2. VERB

/ ˌɛf ti ˈpi / (AmE), / ˌɛf tiː ˈpiː / (BrE)
to transfer files between computers using the File Transfer Protocol.
transfer files using File Transfer Protocol

Examples

She will FTP the latest reports to the server tonight.
I need to FTP these images to our clients by noon.
The files were FTP'd successfully from our office computer to the data center.

Synset Examples

You can FTP these data

Related Terms

  • Synonym: to upload - to transfer data to a larger or remote computer system
  • Synonym: to download - to transfer data from a remote system to a local computer

Etymology

Acronym of 'File Transfer Protocol.'

Source: WordNet
WordNet ID: oewn-02238134-v
Synset POS: v
