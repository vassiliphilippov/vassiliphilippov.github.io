substring

1. NOUN

/ ˈsʌbˌstrɪŋ / (AmE), / ˈsʌbˌstrɪŋ / (BrE)
a sequence of characters that forms part of a longer string of text or data
part of a longer string of text

Examples

You can extract a substring from the original string using this function.
The password must not contain any substring from the user's name.
Searching for a substring is a common operation in text processing.

Related Terms

  • Degree Higher: string - a sequence of characters, as in computing

Etymology

Formed from 'sub-' (meaning under or part) and 'string' (a sequence of characters).
