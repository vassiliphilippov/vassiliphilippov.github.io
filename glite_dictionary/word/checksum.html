checksum

1. NOUN

/ ˈtʃɛkˌsʌm / (AmE), / ˈtʃɛkˌsʌm / (BrE)
a numerical value used to verify the integrity of data, calculated from the sum or algorithmic processing of its digits or bits, often for detecting errors in transmission or storage
data integrity verification value

Examples

Every file downloaded from the server comes with a checksum for verification.
You can use a checksum to detect errors introduced during the transfer of data.
Make sure the checksum matches before installing the software to ensure it hasn’t been corrupted.

Related Terms

  • Synonym: parity bit - a single bit added to binary data to detect errors

Etymology

From 'check' + 'sum', referring to the calculation made to check data accuracy.
