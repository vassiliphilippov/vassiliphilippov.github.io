undecidable

1. ADJ

/ ˌʌn.dɪˈsaɪ.də.bəl / (AmE), / ˌʌn.dɪˈsaɪ.də.bəl / (BrE)
(of a proposition or problem) not capable of being proved either true or false using the rules and axioms of a given logical system.
not provable true or false in a logical system

Examples

In mathematics, the halting problem is a classic example of an undecidable problem.
Some statements in set theory are undecidable and cannot be resolved using its axioms.
The question of whether every even number greater than two is the sum of two primes is currently undecidable in standard arithmetic.

Related Terms

  • Antonym: decidable - able to be proved true or false in a logical system

Etymology

From un- (not) + decidable, after decide, from Latin decidere (to decide, determine).
