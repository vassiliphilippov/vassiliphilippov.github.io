to initialize

1. VERB

/ ɪˈnɪʃəˌlaɪz / (AmE), / ɪˈnɪʃəlaɪz / (BrE)
to set the initial value of a variable, object, or other element in computing; to prepare something for use by assigning a starting value
set an initial value (computing)

Examples

Before running the program, you need to initialize all variables.
The system failed to initialize the hardware properly.
She wrote a function to initialize the database with default values.

Related Terms

  • Synonym: reset - to set something again to its initial state
  • Antonym: to terminate - to bring to an end; in computing, to stop a process

Etymology

From 'initial' + '-ize', modeled after French 'initialiser'.

Source: WordNet
WordNet ID: oewn-00702662-v
Synset POS: v

2. VERB

/ ɪˈnɪʃəˌlaɪz / (AmE), / ɪˈnɪʃəlaɪz / (BrE)
(computing) to prepare a disk or other storage device for use by dividing it into marked sectors for data storage; to format.
prepare or format a disk for data storage

Examples

Before you can use the new hard drive, you need to initialize it using the disk utility.
The technician asked if I wanted to initialize the USB device, which would erase all existing data.
Please remember to back up your files before you initialize the disk, as this process cannot be undone.

Synset Examples

Please format this disk before entering data!

Related Terms

  • Synonym: format - to prepare a storage medium for use, especially by arranging it into sectors

Etymology

From 'initial' + '-ize', modeled after similar technical coinages in English.

Source: WordNet
WordNet ID: oewn-00564629-v
Synset POS: v
