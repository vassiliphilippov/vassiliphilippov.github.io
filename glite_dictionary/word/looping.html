looping

1. NOUN

/ ˈluːpɪŋ / (AmE), / ˈluːpɪŋ / (BrE)
(computing) the process of executing the same set of instructions repeatedly, either a specified number of times or until a particular condition is met.
repeating instructions in computing

Examples

In most programming languages, looping allows a block of code to run multiple times.
Proper use of looping can make your code more efficient and easier to maintain.
Mistakes in looping structures can lead to infinite loops that crash the program.

Synset Examples

the solution is obtained by iteration

Related Terms

  • Synonym: iteration - repeated execution of a process or set of instructions

Etymology

From 'loop' (in programming, a sequence of instructions that is continually repeated) + -ing suffix.
