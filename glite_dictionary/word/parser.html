parser

1. NOUN

/ ˈpɑrsər / (AmE), / ˈpɑːsə / (BrE)
A computer program that analyzes a string of symbols, such as code, and divides it into functional components for further processing.
computer program for analyzing code structure

Examples

A parser converts human-readable code into data structures for computer processing.
The programming language uses a sophisticated parser to interpret user commands.
Errors in the parser can cause the compiler to misinterpret the code's structure.

Synset Examples

compilers must parse source code in order to translate it into object code

Related Terms

  • Synonym: compiler - a computer program that translates source code into executable code

Etymology

From 'parse', itself from Latin 'pars (orationis)' meaning 'part (of speech)'.
