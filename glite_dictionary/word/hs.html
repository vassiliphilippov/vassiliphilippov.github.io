Hs

1. NOUN

/ eɪtʃ ɛs / (AmE), / eɪtʃ ɛs / (BrE)
the chemical symbol for hassium, a radioactive transuranic element with the atomic number 108
symbol for hassium, element 108

Examples

Scientists first identified Hs in a laboratory experiment in Germany.
Because Hs is highly unstable, it can only be produced in minute amounts.
The element Hs was named after the German state of Hesse.

Related Terms

  • Synonym: hassium - a radioactive transuranic element, atomic number 108

Etymology

From Latin 'Hassium', named after the German state of Hesse.
