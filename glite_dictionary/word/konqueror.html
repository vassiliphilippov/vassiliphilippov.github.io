Konqueror

1. PROPN

/ kɑnˈkɛrər / (AmE), / kɒnˈkɛrərə / (BrE)
a free web browser and file manager designed for the Linux operating system, created as part of the KDE project
Linux browser and file manager

Examples

I used Konqueror as my primary browser when I first tried Linux.
Konqueror is known for its versatility as both a file manager and a web browser.
Many KDE users appreciate the integration between Konqueror and the rest of the desktop environment.

Etymology

A playful alteration of 'conqueror', from the KDE (K Desktop Environment) project, following the tradition of software names starting with 'K'.
