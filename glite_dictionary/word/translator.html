translator

1. NOUN

/ trænzˈleɪtər / (AmE), / trænzˈleɪtə / (BrE)
a person who translates written texts from one language into another
person who translates written texts

Examples

She works as a translator for a publishing company, converting novels from French into English.
Finding a qualified translator is essential for ensuring accurate legal documents.
He hired a translator to help him read ancient manuscripts.

Related Terms

  • Synonym: interpreter - a person who translates spoken words between languages

Etymology

From Latin 'translator', from 'transferre' meaning 'to carry across'.

Source: WordNet
WordNet ID: oewn-10744916-n
Synset POS: n

2. NOUN

/ trænzˈleɪtər / (AmE), / trænzˈleɪtə / (BrE)
a computer program that converts code written in one programming language into another, such as a compiler or interpreter
program converting programming languages

Examples

The translator converts the source code into a form that the computer can execute.
Many modern IDEs include a built-in translator for multiple programming languages.
Using an automatic translator can help migrate legacy code to a new software platform.

Related Terms

  • Synonym: compiler - a program that translates code from a high-level programming language to machine code
  • Synonym: interpreter - a program that executes instructions written in a programming language directly

Etymology

From Latin translator (one who translates), from translatus, past participle of transferre (to transfer).

Source: WordNet
WordNet ID: oewn-06593444-n
Synset POS: n

3. NOUN

/ trænzˈleɪtər / (AmE), / trænsˈleɪtə / (BrE)
a person who translates spoken or written language from one language to another
person who translates language

Examples

She works as a translator for the United Nations, facilitating communications between delegates from different countries.
The book was so beautifully rendered that the translator received an award for her work.
Hiring a skilled translator is crucial when negotiating international business contracts.

Related Terms

  • Synonym: interpreter - a person who translates spoken language orally

Etymology

From Latin translatorem (one who carries over), from translatus, past participle of transferre (to carry across).

Source: WordNet
WordNet ID: oewn-10232171-n
Synset POS: n
