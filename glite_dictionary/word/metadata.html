metadata

1. NOUN

/ ˈmɛ.təˌdeɪ.tə / (AmE), / ˈmɛt.əˌdeɪ.tə / (BrE)
data that provides information about other data, such as details describing properties, origin, or structure of digital files or records
data about other data

Examples

The photo contains metadata that shows when and where it was taken.
Library catalogs use metadata to help users find books by subject, author, or title.
Webpages often include metadata to improve their search engine rankings.

Synset Examples

a library catalog is metadata because it describes publications

Etymology

From meta- (meaning 'beyond' or 'about') and data, first used in information science in the 1960s.
