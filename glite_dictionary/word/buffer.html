2. NOUN

/ ˈbʌfɚ / (AmE), / ˈbʌfə / (BrE)
(computing) a region of memory used for temporary storage of data while it is being transferred between two places, especially to compensate for differences in processing speed.
temporary data storage area in computing

Examples

Before saving the file, the program placed the data in a buffer to ensure nothing was lost.
The video paused frequently because the buffer could not keep up with the data stream.
When printing large documents, a buffer helps manage the flow between the computer and the printer.

Related Terms

  • Synonym: cache - temporary storage for frequently accessed data
