denumerable

1. ADJ

/ dɪˈnuːmərəbl̩ / (AmE), / dɪˈnjuːmərəbl̩ / (BrE)
(of a set or quantity) able to be counted; countable, especially in a mathematical context.
that can be counted

Examples

In mathematics, a set is denumerable if its elements can be matched with the natural numbers.
The list of rational numbers is denumerable, but the set of real numbers is not.
She explained to the students that only denumerable quantities can be listed one after another.

Synset Examples

countable sins
numerable assets

Related Terms

  • Synonym: countable - able to be counted
  • Antonym: uncountable - not able to be counted

Etymology

From Latin 'denumerare' (to count out) + -able.
