parameter

1. NOUN

/ pəˈræm.ə.tɚ / (AmE), / pəˈræm.ɪ.tə / (BrE)
(Computer science) a value or reference that is passed to a function, procedure, subroutine, command, or program to customize its operation.
value or reference passed to a function (computing)

Examples

In this programming language, you must define the type of each parameter in your function.
The function failed because it received an invalid parameter from the user input.
Optional parameters allow for more flexible and reusable code.

Related Terms

  • Synonym: argument - value passed to a function or procedure (computing)

Etymology

From Greek 'para-' (beside) + 'metron' (measure), via French and English.

WordNet ID: oewn-06367706-n
Synset POS: n

2. NOUN

/ pəˈræm.ɪ.tɚ / (AmE), / pəˈræm.ɪ.tə / (BrE)
a constant in the equation of a curve that can be varied to produce a family of similar curves
variable constant in curve equations

Examples

Changing the parameter in the equation results in different parabolas being graphed.
The shape of the ellipse depends on the values of its parameters.
By adjusting each parameter, mathematicians can explore entire families of similar curves.

Related Terms

  • Antonym: variable - a symbol representing a quantity that can change

Etymology

From Greek 'para-' (beside) and 'metron' (measure).

Source: WordNet
WordNet ID: oewn-05867655-n
Synset POS: n

3. NOUN

/ pəˈræmɪtɚ / (AmE), / pəˈræmɪtə / (BrE)
a factor that defines a system and influences or limits its performance or behavior
factor defining/limiting a system

Examples

The success of the experiment depends on carefully controlling each parameter within the system.
One key parameter in this study is the sample size.
Engineers must define all relevant parameters before beginning the design process.

Related Terms

  • Synonym: criterion - a standard or principle by which something is judged or decided

Etymology

From Greek 'parametros'—'para' (beside) + 'metron' (measure).

Source: WordNet
WordNet ID: oewn-07342905-n
Synset POS: n

4. NOUN

/ pəˈræmɪtɚ / (AmE), / pəˈræmɪtə / (BrE)
A quantity, such as the mean or variance, that characterizes a statistical population and can be estimated from sample data.
statistical quantity characterizing a population

Examples

The population mean is a parameter that can be estimated from sample data.
The variance is another parameter used to describe the distribution.
Researchers calculated the parameter using statistical methods.

Related Terms

  • Antonym: statistic - a numerical value calculated from sample data

Etymology

From Greek 'para-' (beside) + 'metron' (measure) via French 'paramètre'.

Source: WordNet
