ununquadium

1. NOUN

/ ˌʌn.ʌnˈkwɑː.di.əm / (AmE), / ˌʌn.ʌnˈkwæd.i.əm / (BrE)
the former temporary name for the radioactive chemical element with atomic number 114, now officially named flerovium
former name for element 114 (flerovium)

Examples

The element once called ununquadium is now officially named flerovium.
Scientists synthesized ununquadium in a laboratory before it received its permanent name.
Older textbooks may refer to element 114 as ununquadium rather than flerovium.

Related Terms

  • Synonym: flerovium - the official name for the element with atomic number 114

Etymology

From Latin roots: 'un' (one) repeated for atomic digits 1-1-4, and 'quadium' for four; systematic element naming before official discovery.
