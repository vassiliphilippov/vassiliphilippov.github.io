applet

1. NOUN

/ ˈæplət / (AmE), / ˈæplɪt / (BrE)
a small application, especially one written in Java, that runs within a web browser to provide interactive features
small web-based application program

Examples

The website uses a Java applet to display real-time stock prices.
When I opened the page, a small applet loaded to play a simple game.
Thanks to the weather applet, visitors can see current conditions instantly.

Related Terms

  • Degree Higher: application - a computer program designed to perform a group of coordinated functions

Etymology

Blend of 'application' and 'applet' with the diminutive suffix '-let.'
