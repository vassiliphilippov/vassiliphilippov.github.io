filename

1. NOUN

/ ˈfaɪlˌneɪm / (AmE), / ˈfaɪlˌneɪm / (BrE)
the name assigned to a computer file, often including an extension that indicates its type
name of a computer file

Examples

Please save the document using the filename your teacher provided.
Each filename must be unique within the same folder to avoid confusion.
He accidentally deleted the image because the filename was similar to another file.

Related Terms

  • Degree Higher: file extension - the suffix at the end of a filename, indicating its file type

Etymology

From 'file' + 'name'; adopted in the context of computing in the mid-20th century.

Source: WordNet
WordNet ID: oewn-06346543-n
Synset POS: n

2. NOUN

/ ˈfaɪlˌneɪm / (AmE), / ˈfaɪlneɪm / (BrE)
the name given to a computer file to identify it, often including an extension that indicates the file type
name of a computer file

Examples

Please save the document using the filename format specified by your instructor.
Make sure the filename does not contain any special characters before uploading.
He forgot the exact filename of the report and had to search for it.

Related Terms

  • Synonym: file - a resource for storing information on a computer

Etymology

Formed from 'file' and 'name', widely used since the early days of computing.

Source: WordNet
WordNet ID: oewn-06346543-n
Synset POS: n
