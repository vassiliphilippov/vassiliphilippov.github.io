washable

1. ADJ

/ ˈwɑːʃəbəl / (AmE), / ˈwɒʃəbəl / (BrE)
(of a fabric or item) able to be washed without being damaged.
able to be washed without damage

Examples

The children's paint set is completely washable, so cleaning up is easy.
Always check if a garment is washable before putting it in the laundry.
They chose a sofa with washable covers for convenience.

Synset Examples

washable woolens
acrylic blankets are both warm and washable

Related Terms

  • Antonym: non-washable - not able to be washed without damage
  • Degree Higher: machine-washable - able to be washed in a washing machine without damage

Etymology

From 'wash' + '-able', where '-able' denotes capability.
