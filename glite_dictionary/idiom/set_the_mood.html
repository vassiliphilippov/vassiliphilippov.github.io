to set the mood

1. VERB

/ sɛt ðə muːd / (AmE), / sɛt ðə muːd / (BrE)
to create a particular atmosphere or feeling in a place or situation
create a particular atmosphere

Examples

Candlelight and soft music helped set the mood for a romantic evening.
The coach's speech set the mood for the team's intense practice session.
The decorations and lighting were chosen carefully to set the mood for the ceremony.

Synset Examples

The party was filled with jazz music that set the mood.

Related Terms

  • Synonym: to set the tone - establish a general feeling or character for something
