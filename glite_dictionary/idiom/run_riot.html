to run riot

1. VERB

/ rʌn ˈraɪət / (AmE), / rʌn ˈraɪət / (BrE)
to behave in a wild, uncontrolled, or excessive way; to become unrestrained or out of control
behave wildly or uncontrollably

Examples

The students began to run riot when the teacher left the room.
During the festival, the crowd seemed to run riot in the streets.
When left unchecked, his imagination can run riot and create fantastic stories.

Synset Examples

The children began to run riot when the movie started.
His imagination can run riot when he's bored.

Related Terms

  • Synonym: go wild - to behave in an uncontrolled or excitable way
  • Synonym: to run amok - to behave in a frenzied, out-of-control way
