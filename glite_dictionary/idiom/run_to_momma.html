to run to momma

1. VERB

/ rʌn tə ˈmɑmə / (AmE), / rʌn tə ˈmɒmə / (BrE)
to seek comfort, protection, or help from one's mother, often in a childlike or dependent manner
seek comfort or help from one's mother

Examples

Whenever things get tough, he tends to run to momma instead of solving problems himself.
Some people accuse her of always wanting to run to momma for advice.
It's time to be independent and stop trying to run to momma every time you're upset.

Synset Examples

The kids always run to momma for comfort.

Related Terms

  • Synonym: run to daddy - to seek comfort or help from one's father in a similar manner
