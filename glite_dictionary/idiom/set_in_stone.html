set in stone

1. ADJ

/ sɛt ɪn stoʊn / (AmE), / sɛt ɪn stəʊn / (BrE)
fixed and unchangeable; not able to be altered or modified
unchangeable or fixed

Examples

The company's policy isn't set in stone and may change in the future.
Our vacation dates aren't set in stone yet, so we can be flexible.
Just because it's in the plan doesn't mean it's set in stone.

Synset Examples

the agreement is not yet set in stone

Related Terms

  • Synonym: final - not able to be changed or reversed
  • Antonym: changeable - able to be altered or modified
