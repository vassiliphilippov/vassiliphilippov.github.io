built like a brick shipyard

1. ADJ

/ bɪlt laɪk ə brɪk ˈʃɪpˌjɑrd / (AmE), / bɪlt laɪk ə brɪk ˈʃɪpˌjɑːd / (BrE)
Extremely strong, sturdy, or solidly constructed, typically used humorously to describe a person or object.
humorously very strong or sturdy
informal

Examples

Despite its age, the old safe is built like a brick shipyard and impossible to move.
The new suitcase is built like a brick shipyard, so I’m not worried about it breaking during travel.
Everyone at the gym agrees that Sarah is built like a brick shipyard thanks to her intense training.
