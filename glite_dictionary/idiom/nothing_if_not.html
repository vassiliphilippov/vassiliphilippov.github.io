nothing if not

1. ADV

/ ˈnʌθɪŋ ɪf nɑt / (AmE), / ˈnʌθɪŋ ɪf nɒt / (BrE)
used to emphasize that someone or something displays a particular quality to a very great degree; certainly or above all else
emphatically, above all else

Examples

She was nothing if not enthusiastic about the new project.
His explanation was nothing if not thorough, covering every possible detail.
The decor in the restaurant was nothing if not extravagant.

Synset Examples

She is nothing if not determined to succeed.
Their hospitality was nothing if not generous.
He is nothing if not punctual, always arriving on time.

Related Terms

  • Synonym: certainly - without doubt; definitely
  • Synonym: emphatically - in a forceful way; with emphasis
