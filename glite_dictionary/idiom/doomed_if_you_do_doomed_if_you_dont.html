doomed if you do, doomed if you don't

1. X

/ ˈduːmd ɪf juː duː, ˈduːmd ɪf juː doʊnt / (AmE), / ˈduːmd ɪf juː duː, ˈduːmd ɪf juː dəʊnt / (BrE)
used to describe a situation in which any possible course of action will lead to negative or unwanted consequences
no-win situation; negative outcome either way

Examples

No matter what decision I make, I'm doomed if you do, doomed if you don't.
Politicians often feel doomed if you do, doomed if you don't when faced with controversial issues.
Sometimes parenting can feel like you're doomed if you do, doomed if you don't.

Synset Examples

You're doomed if you do, doomed if you don't, so just choose.
In this situation, you're just doomed if you do, doomed if you don't.

Related Terms

  • Synonym: damned if you do, damned if you don't - no-win situation; condemned regardless of action taken
