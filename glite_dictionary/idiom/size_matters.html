size matters

1. X

/ saɪz ˈmætərz / (AmE), / saɪz ˈmætəz / (BrE)
used to emphasize that the size of something is important for its effectiveness, function, or suitability in a particular context
importance of size in a context

Examples

When buying a new television, size matters if you want the best viewing experience.
In sports like basketball, size matters when playing certain positions.
They learned that in packaging and marketing, size matters almost as much as quality.

Synset Examples

In this case, size matters for better performance.
When it comes to shoes, size matters a lot.
He believes size matters when choosing a vehicle.
