to make a run for it

1. VERB

/ meɪk ə rʌn fər ɪt / (AmE), / meɪk ə rʌn fə ɪt / (BrE)
to attempt to escape quickly, especially in a sudden or risky situation
escape quickly or suddenly

Examples

As soon as the guard turned his back, they decided to make a run for it.
The rain was coming down hard, so we had to make a run for it to the car.
If we don't make a run for it now, we'll be trapped inside.

Synset Examples

When the alarm sounded, they decided to make a run for it.
We should make a run for it before it starts raining.

Related Terms

  • Synonym: to flee - to run away from danger or trouble
  • Synonym: bolt - to move suddenly or quickly, especially to escape
