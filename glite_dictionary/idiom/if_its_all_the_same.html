if it's all the same

1. SCONJ

/ ɪf ɪts ɔl ðə seɪm / (AmE), / ɪf ɪts ɔːl ðə seɪm / (BrE)
used to indicate that one has no preference between alternatives, or that a proposed change or suggestion does not matter
expressing lack of preference

Examples

You can choose the restaurant—if it's all the same to you, I don't mind where we eat.
If it's all the same, let's meet an hour later so I can finish my work.
We can leave now or later, if it's all the same to everyone.

Synset Examples

I’d prefer tea; if it's all the same, make it hot.
I’ll take the bus; if it's all the same, it’s cheaper.

Related Terms

  • Synonym: it doesn't matter - indicating indifference or lack of importance
