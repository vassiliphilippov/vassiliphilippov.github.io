with any luck

1. ADV

/ wɪð ˈɛni lʌk / (AmE), / wɪð ˈɛni lʌk / (BrE)
if everything goes well; if things happen as hoped or planned
if things go well; hopefully

Examples

With any luck, the tickets will still be available when we arrive.
We'll catch the early train, with any luck.
With any luck, I'll finish my assignment before dinner.

Synset Examples

We’ll finish the project by Friday, with any luck.
We’ll find a solution, with any luck.
With any luck, the weather will be perfect for the picnic.

Related Terms

  • Synonym: hopefully - expressing hope that something will happen
