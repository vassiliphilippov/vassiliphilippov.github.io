be so kind as to

1. X

/ bi soʊ kaɪnd æz tu / (AmE), / bi səʊ kaɪnd az tuː / (BrE)
used as a very polite way of asking someone to do something
polite phrase to make a request

Examples

Would you be so kind as to close the window for me?
Could you be so kind as to show me the way to the station?
If you would be so kind as to wait here, the manager will see you soon.

Synset Examples

Would you be so kind as to help me?
Would you be so kind as to pass the salt?
Would you be so kind as to lend me your book?

Related Terms

  • Synonym: would you mind - polite phrase used to ask someone to do something
  • Synonym: please - word used to ask for something politely
  • Antonym: refuse - to decline or say no to a request
