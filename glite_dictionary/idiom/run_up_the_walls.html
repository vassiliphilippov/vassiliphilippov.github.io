to run up the walls

1. VERB

/ rʌn ʌp ðə wɔlz / (AmE), / rʌn ʌp ðə wɔːlz / (BrE)
to feel extremely anxious, agitated, or stressed to the point of restlessness
become extremely anxious or agitated
informal

Examples

The stress from work made me run up the walls last month.
When the kids started fighting again, I thought I would run up the walls from the noise.
He was running up the walls waiting for the test results.

Synset Examples

With deadlines looming, I was ready to run up the walls.

Related Terms

  • Synonym: to climb the walls - to feel very anxious or agitated
