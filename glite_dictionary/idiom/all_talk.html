all talk

1. NOUN

/ ɔl tɔk / (AmE), / ɔːl tɔːk / (BrE)
used to describe someone or something that makes promises, claims, or threats but does not take any real action
makes claims without action

Examples

He says he'll quit smoking, but he's all talk and never actually tries to stop.
The negotiations turned out to be all talk with no real progress made.
She's all talk when it comes to helping out but never actually volunteers.

Synset Examples

The project was all talk; there is no there there.

Related Terms

  • Synonym: empty talk - insubstantial or meaningless conversation
  • Antonym: action - the process of doing something to achieve an aim
