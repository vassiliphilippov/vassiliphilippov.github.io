to run with the hare and hunt with the hounds

1. VERB

/ rʌn wɪð ðə hɛr ənd hʌnt wɪð ðə haʊndz / (AmE), / rʌn wɪð ðə heə ənd hʌnt wɪð ðə haʊndz / (BrE)
to try to support or please two opposing sides at the same time
try to please both sides

Examples

Whenever a controversial issue arose, the politician would run with the hare and hunt with the hounds.
You can't run with the hare and hunt with the hounds if you want people to trust you.
She was accused of trying to run with the hare and hunt with the hounds during the negotiations.

Synset Examples

To please everyone, she decided to run with the hare and hunt with the hounds.
He tried to run with the hare and hunt with the hounds during the election.
You can't run with the hare and hunt with the hounds in this business.

Related Terms

  • Synonym: sit on the fence - to avoid making a decision or taking sides

Etymology

From hunting, originally referring to simultaneously helping both prey and hunters; recorded in English since the early 16th century.
