to run the table

1. VERB

/ rʌn ðə ˈteɪbəl / (AmE), / rʌn ðə ˈteɪbl̩ / (BrE)
to win every game or match in a series without a loss
win all matches in a series

Examples

If they run the table this season, they'll finish undefeated.
With two games left, the team must run the table to reach the finals.
Determined to run the table, she trained harder than ever for the tournament.

Synset Examples

Our team managed to run the table in the tournament.
If we run the table, we can secure the championship.

Related Terms

  • Synonym: sweep - to win all games in a series, especially in sports
