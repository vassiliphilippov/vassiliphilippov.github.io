like an elephant's memory

1. X

/ laɪk ən ˈɛləfənts ˈmɛm(ə)ri / (AmE), / laɪk ən ˈɛlɪfənts ˈmɛm(ə)ri / (BrE)
To have a very good memory; to be able to recall things easily and accurately.
have a very good memory

Examples

She says her grandmother has like an elephant's memory and never forgets a birthday.
With like an elephant's memory, he remembered every detail from his childhood.
You need like an elephant's memory to keep track of all these facts.

Synset Examples

She joked that her memory was like an elephant's.

Related Terms

  • Synonym: have a memory like an elephant - have an exceptionally good memory

Etymology

From the popular belief that elephants possess excellent long-term memory.
