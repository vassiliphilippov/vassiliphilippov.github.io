to set the wheels in motion

1. VERB

/ sɛt ðə wilz ɪn ˈmoʊʃən / (AmE), / sɛt ðə wiːlz ɪn ˈməʊʃən / (BrE)
to do something that will start a series of actions or events
begin a process or series of events

Examples

The board decided to set the wheels in motion for the merger.
Her resignation set the wheels in motion for significant changes at the company.
We need to set the wheels in motion if we want the event to happen this year.

Synset Examples

He set the wheels in motion to start the charity.
They set the wheels in motion for the festival last month.
Let's set the wheels in motion for the project.

Related Terms

  • Synonym: to get the ball rolling - start something; initiate an activity or process
