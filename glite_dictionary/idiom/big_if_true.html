big if true

1. X

/ bɪg ɪf tru / (AmE), / bɪg ɪf truː / (BrE)
used to emphasize that if a statement or claim proves to be correct, it would be very significant or have major implications
significant if correct (informal)
informal

Examples

That rumor about the company splitting is big if true.
He told me aliens landed in his garden, which is big if true.
The article says the new device will revolutionize medicine—big if true.

Synset Examples

The news is a big if true, but it’s still unconfirmed.
That’s a big if true regarding the merger rumors.
