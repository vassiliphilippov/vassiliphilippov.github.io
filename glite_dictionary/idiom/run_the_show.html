to run the show

1. VERB

/ ɹʌn ðə ʃoʊ / (AmE), / ɹʌn ðə ʃəʊ / (BrE)
to be in charge of or manage an activity, organization, or operation
be in charge or manage something

Examples

As project manager, she really knows how to run the show.
When Mr. Lopez left, his assistant was asked to run the show.
If you want things done efficiently, let Jacob run the show.

Synset Examples

He loves to run the show during our team meetings.
She knows how to run the show effectively.
They let him run the show while they handled logistics.

Related Terms

  • Synonym: be in charge - to have control or responsibility for something
  • Synonym: to call the shots - to be the person who makes the decisions
