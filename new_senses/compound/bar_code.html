bar code

1. NOUN

/ ˈbɑr ˌkoʊd / (AmE), / ˈbɑː ˌkəʊd / (BrE)
A code consisting of a series of vertical bars of varying widths, usually printed on products and readable by machines, used to identify items and retrieve information such as price and inventory.
machine-readable code of vertical bars

Examples

Every product in the store has a unique bar code for inventory tracking.
If the bar code is damaged, the scanner may not read it properly.
The cashier scanned the bar code to ring up the item quickly.

Related Terms

  • Degree Higher: QR code - two-dimensional matrix barcode
