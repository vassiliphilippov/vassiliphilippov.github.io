first in first out

1. NOUN

/ fɜrst ɪn fɜrst aʊt / (AmE), / fɜːst ɪn fɜːst aʊt / (BrE)
An accounting method in which the oldest inventory items are recorded as sold first.
inventory accounting method (oldest sold first)

Examples

Stores managing perishable goods often prefer a first in first out system to reduce waste.
The company uses the first in first out method to value its inventory.
Under first in first out accounting, the oldest stock is sold before newer arrivals.

Related Terms

  • Antonym: last in first out - inventory accounting (newest items sold first)

Etymology

From the phrase 'first in, first out', describing the order in which items are managed.

2. NOUN NEW

/ fɜrst ɪn fɜrst aʊt / (AmE), / fɜːst ɪn fɜːst aʊt / (BrE)
A queue discipline or processing order (especially in computing and data systems) in which the earliest-arrived item is processed or removed before later ones.
queue order: oldest processed first

Examples

The network router uses first in first out to preserve packet order during bursts of traffic.
When designing the shared buffer, they decided a first in first out policy would reduce latency spikes.
The print spooler follows first in first out, so the earliest submitted documents print before later ones.

Related Terms

  • Synonym: queue - an ordered list of items waiting to be processed
  • Antonym: last in first out - a processing order in which the most recently added item is processed first (opposite of FIFO)
