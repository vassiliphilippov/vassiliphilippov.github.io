style sheet

1. NOUN

/ ˈstaɪl ˌʃit / (AmE), / ˈstaɪl ˌʃiːt / (BrE)
a document summarizing the editorial conventions and rules to be followed when preparing text for publication
editorial guidelines document

Examples

Before submitting your article, consult the style sheet to ensure consistency.
Each department was given a separate style sheet for internal documents.
The publisher updated their style sheet to reflect new grammar rules.

Related Terms

  • Synonym: style guide - a set of standards for writing and designing documents

2. NOUN NEW

/ c8sta6al cc83it / (AmE), / c8sta6al cc83id0t / (BrE)
(computing) a file that contains rules describing the presentation and layout of a document, especially a web page (for example, a CSS file).
file of rules defining webpage appearance

Examples

Web developers maintain a style sheet to ensure consistent layouts across pages.
She edited the site's style sheet to change fonts and button colors quickly.
If the browser ignores the style sheet, the page will display with default fonts and spacing.
