(in computing, digital forensics, and virtualization) a complete copy of the contents of a computer's volatile memory (RAM), saved as a file for analysis, debugging, or restoration.
snapshot of RAM for analysis

Examples

The forensic team captured a memory image of the suspect's machine before powering it down.
Developers loaded the virtual machine's memory image into the debugger to inspect running processes.
After the crash, they restored the server from a recent memory image to reproduce the bug.
