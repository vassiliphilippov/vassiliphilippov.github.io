garbage collection

1. NOUN

/ ˈɡɑrbɪdʒ kəˌlɛkʃən / (AmE), / ˈɡɑːbɪdʒ kəˌlɛkʃən / (BrE)
the process of collecting and removing household or commercial waste and refuse.
removal of waste or garbage

Examples

Because of the holiday, garbage collection was delayed by one day.
Proper garbage collection helps keep the streets clean and prevents pollution.
The city schedules garbage collection twice a week in this neighborhood.

Related Terms

  • Synonym: waste collection - the collection of unwanted materials for disposal or recycling

2. NOUN NEW

(in computing) the automatic process by which a runtime system reclaims memory taken by objects that are no longer reachable so that that memory can be reused.
automatic memory reclamation (computing)

Examples

Java's runtime uses garbage collection to automatically reclaim unused memory during execution.
Performance issues often appear when garbage collection runs too frequently on a production server.
To reduce pauses, the engineer tuned the JVM's garbage collection settings for smoother throughput.

Etymology

Metaphorical extension of the literal phrase 'garbage collection' (collecting waste), applied in computing to the automatic removal of unused memory.
