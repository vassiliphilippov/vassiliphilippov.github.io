train set

1. NOUN

/ treɪn sɛt / (AmE), / treɪn sɛt / (BrE)
A toy consisting of miniature models of trains and tracks for them to run on.
toy model of trains with track

Examples

A train set can be a fun way to introduce children to engineering concepts.
He received a train set for his birthday and spent hours assembling the track.
My grandfather still has the train set he played with as a child.

Related Terms

  • Synonym: model railway - miniature railway system as a hobby
  • Synonym: toy train - small toy version of a train

2. NOUN NEW

/ treɪn sɛt / (AmE), / treɪn sɛt / (BrE)
A subset of a dataset used to train a machine-learning model; it contains examples the model learns from.
dataset used to train models

Examples

To prevent overfitting, the team separated the dataset into a train set and a test set.
The neural network improved dramatically after we augmented images in the train set with rotations.
Many competitions provide multiple train set versions so teams can test different preprocessing strategies.
