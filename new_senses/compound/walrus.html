(computing, informal) the assignment-expression operator ':=' that allows assignment within an expression (commonly used in Python).
nickname for ':=' assignment operator

Examples

The walrus operator lets you assign values inside an if statement in one line.
Using the walrus operator, she avoided recalculating the list length before the loop.
Many style guides discourage the walrus operator because it can make complex expressions harder to read.

Etymology

Nickname from the operator symbol ':='; introduced and popularized in the Python community (PEP 572, 2018) because the symbol was likened to a walrus' face.
