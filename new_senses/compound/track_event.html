track event

1. NOUN

/ træk ɪˈvɛnt / (AmE), / træk ɪˈvɛnt / (BrE)
a footrace held on a running track, either indoors or outdoors, especially in athletics competitions
a race on a running track

Examples

Athletes trained all season to compete in multiple track events at the championship.
She won a gold medal in her first ever track event last year.
The 100-meter dash is a popular track event at the Olympic Games.

Related Terms

  • Antonym: field event - an athletic competition taking place on the field, such as jumping or throwing

2. NOUN NEW

/ træk ɪˈvɛnt / (AmE), / træk ɪˈvɛnt / (BrE)
(in software and analytics) a recorded user action or system occurrence captured as a data point for measurement, logging, or telemetry.
analytics event

Examples

The analytics team defined a track event for each user action that changes the shopping cart.
You should log a track event when the payment succeeds so engineers can debug failures.
After the update, our dashboard showed every track event grouped by country and device type.
