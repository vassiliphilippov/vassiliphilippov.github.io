web log

1. NOUN

/ wɛb lɑɡ / (AmE), / wɛb lɒɡ / (BrE)
a shared online journal where people regularly post entries about their personal experiences, opinions, or hobbies
online journal or diary

Examples

He updates his web log every week with reviews of new books.
Many companies maintain a web log to update customers on new products and services.
She started a web log to share her travel stories and photos with friends.

Synset Examples

postings on a blog are usually in chronological order

Related Terms

  • Synonym: blog - a regularly updated website or web page, typically run by an individual or small group

Etymology

From 'web' (the World Wide Web) and 'log' (a record of events).

2. NOUN NEW

/ wɛb lɑg / (AmE), / wɛb lɒg / (BrE)
(computing) a file produced by a web server that records requests and activity, typically including timestamps, client IP addresses, requested URLs, and status codes.
web server access log file

Examples

After the security incident, the team examined the web log to trace suspicious IP addresses.
The analytics tool parsed the web log and produced a report of peak traffic hours.
Every night the backup job compresses the web log files to save disk space and retain history.

Synset Examples

After the security incident, the team examined the web log to trace suspicious IP addresses.
The analytics tool parsed the web log and produced a report of peak traffic hours.
Every night the backup job compresses the web log files to save disk space and retain history.

Related Terms

  • Synonym: web server log - a file created by a web server that records details of requests it receives
  • Synonym: access log - a log file listing client accesses to a server or service, often used for analytics

Etymology

From 'web' (the World Wide Web) and 'log' (a record of events).
