episodic memory

1. NOUN

/ ˌɛpɪˈsɑdɪk ˈmɛməri / (AmE), / ˌɛpɪˈsɒdɪk ˈmɛməri / (BrE)
the ability to remember specific events or experiences from your own life, often including their context and details
memory of personal life events
medical

Examples

As people age, their episodic memory for past events can sometimes decline.
Damage to certain parts of the brain may impair episodic memory but leave other types of memory intact.
He relied on his episodic memory to recall the details of his childhood vacation.

Related Terms

  • Antonym: semantic memory - memory for facts and general knowledge, not linked to personal experience

Etymology

From 'episode' (Greek 'epeisodion', an addition) + 'memory' (Latin 'memoria').

(in artificial intelligence and cognitive science) a computational memory system that records, stores, and retrieves representations of past events (episodes) to support learning, planning, or decision-making.
computational memory for past events

Examples

The robot's episodic memory allowed it to recall yesterday's tool location to complete the task.
In the simulator, the agent updated its episodic memory after each trial to improve planning.
Researchers gave the neural network an explicit episodic memory to store past experiences for decision-making.
