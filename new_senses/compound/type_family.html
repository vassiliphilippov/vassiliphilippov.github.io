type family

1. NOUN

/ taɪp ˈfæməli / (AmE), / taɪp ˈfæməli / (BrE)
a complete set of related typefaces that share common design features but may vary in weight, width, or style
group of related typefaces

Examples

She mixed italics and bold from the same type family in her brochure design.
The designer chose a type family with several weights for the new website.
Using a single type family throughout the publication created a cohesive look.

Related Terms

  • Synonym: typeface - a set of characters of the same design

2. NOUN NEW

/ taɪp ˈfæməli / (AmE), / taɪp ˈfæməli / (BrE)
(in programming, especially type theory) a named group of related types defined together so code can associate or compute types based on other types or inputs.
programming group of related types

Examples

In Haskell, the compiler uses a type family to relate types for different implementations.
The library provides a type family so functions can return types depending on their input.
Students learned how a type family can simplify generic code by grouping related type rules.
