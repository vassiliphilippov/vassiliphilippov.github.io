ECC

1. NOUN

/ i si si / (AmE), / iː siː siː / (BrE)
a system for detecting and correcting errors in data by using extra parity bits, commonly used in telecommunications and computing
system for detecting and correcting data errors

Examples

Most modern memory modules use ECC to prevent data corruption.
The server's ECC feature automatically corrects single-bit errors.
Without ECC, data transmissions can be more vulnerable to faults.

Related Terms

  • Synonym: error-correcting code - a coding scheme used to detect and correct errors in data transmission or storage

Etymology

Abbreviation of error-correcting code.

2. NOUN NEW

/ i si si / (AmE), / iːsiːsiː / (BrE)
(in cryptography) an abbreviation for elliptic curve cryptography, a form of public-key cryptography that uses elliptic curves to provide strong security with smaller key sizes than many older systems.
elliptic-curve public-key cryptography

Examples

Security engineers favor ECC keys for faster, smaller cryptography in modern TLS implementations.
Mobile devices often use ECC because it gives strong security with much smaller key sizes.
The library supports both RSA and ECC, allowing developers to choose performance over legacy compatibility.
