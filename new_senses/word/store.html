store

1. NOUN

/ stɔr / (AmE), / stɔː / (BrE)

electronic memory device for data storage

Examples

Increasing the store capacity can significantly improve system performance.
The computer's store holds all the data necessary for processing tasks.

Synset Examples

2. NOUN

/ stɔːr / (AmE), / stɔː / (BrE)
a building or place where goods are kept for future use or sale; a storage facility
place where goods are stored

Examples

Grain was kept in a store until it was needed for the winter.
Most of the equipment is safely locked in the back store.
The company rented a store to keep extra inventory near the distribution center.

Synset Examples

Related Terms

  • Synonym: warehouse - a large building where raw materials or manufactured goods may be stored
  • Antonym: shop - a building or room where goods and services are sold

Etymology

From Old French 'store' (provisions, supplies), from Late Latin 'instaurare' (to furnish, provide).

7. NOUN NEW

/ stɔr / (AmE), / stɔː / (BrE)
a software data repository or persistence layer used by an application to hold and retrieve structured data or application state.
software data repository / persistence layer

Examples

The web app retrieves session data from its store when the user logs in.
We moved the Redux store into a separate module to simplify state management across components.
During backup, the engineer exported the application's store to a cloud database for safekeeping.

Synset Examples

Related Terms

  • Synonym: repository - a central place in software where data or code is stored and managed
  • Synonym: database - an organized collection of structured data, often used as persistent storage for applications
  • Synonym: persistence layer - the part of an application that handles reading and writing data to long-term storage

Etymology

Metaphorical extension of 'store' (a place for keeping things), from Old French 'estore', from Latin 'instaurare' (to restore, renew).
