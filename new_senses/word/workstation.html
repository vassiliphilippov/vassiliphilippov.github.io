workstation

1. NOUN

/ ˈwɝːkˌsteɪʃən / (AmE), / ˈwɜːkˌsteɪʃən / (BrE)
a desktop computer designed to be more powerful than a standard personal computer, often used for technical or scientific applications
powerful desktop computer

Examples

Many engineers require a specialized workstation to run resource-intensive software.
The graphics designer chose a workstation because it could handle large image files efficiently.
Upgrading your workstation can significantly improve productivity in scientific research.

Related Terms

  • Degree Lower: personal computer - a general-purpose computer for personal use

Etymology

From 'work' + 'station', originally used for specialized computer setups in technical environments.
