columnar

1. ADJ

/ kəˈlʌmnɚ / (AmE), / kəˈlʌmnə / (BrE)
having the shape or form of a column; tall and cylindrical.
having the shape of a column

Examples

Some trees, such as the Lombardy poplar, have a distinctly columnar growth habit.
The columnar structure of the rock formations was striking against the landscape.
The building was supported by columnar pillars made of marble.

Synset Examples

trees with columniform trunks
columnar forms
a columnlike tree trunk

Related Terms

  • Synonym: cylindrical - having the shape of a cylinder

Etymology

From Latin columna (column) with the suffix -ar forming adjectives.

3. ADJ NEW

/ kəˈlʌmnə / (AmE), / kəˈlʌmnə / (BrE)
(in computing, especially databases and analytics) relating to or using a column-oriented organization of data, in which values from the same field are stored together to speed analytical queries and improve compression.
relating to column-oriented data storage

Examples

Modern analytics systems often use a columnar storage layout to speed up queries over large tables.
The data warehouse adopted a columnar format to improve compression and query performance.
Adopting columnar formats reduces read time for analytics but can slow transactional updates in some systems.

Synset Examples

Modern analytics systems often use a columnar storage layout to speed up queries over large tables.
The data warehouse adopted a columnar format to improve compression and query performance.
Adopting columnar formats reduces read time for analytics but can slow transactional updates in some systems.

Related Terms

  • Synonym: column-oriented - organized or stored by columns rather than by rows; often used to describe database storage formats.
  • Antonym: row-oriented - organized or stored by rows; a contrasting storage approach optimized for transactional workloads.

Etymology

From Latin columna (column); used metaphorically in computing to describe storage organized by columns.
