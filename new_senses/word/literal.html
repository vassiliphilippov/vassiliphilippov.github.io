literal

1. ADJ

/ ˈlɪt̬.ɚ.əl / (AmE), / ˈlɪt.ər.əl / (BrE)
used to emphasize that something is exactly true and not exaggerated or embellished; without exaggeration.
exactly true; not exaggerated

Examples

His account was the literal version of what happened, unembellished and direct.
The judge asked for the literal facts, without any exaggeration.
When she said she was starving, she didn't mean it was the literal truth.

Synset Examples

it's the literal truth

Related Terms

  • Antonym: figurative - not literal; using metaphorical or symbolic meaning

Etymology

From Latin 'litteralis' (of or relating to letters).

5. NOUN NEW

/ ˈlɪtərəl / (AmE), / ˈlɪtərəl / (BrE)
(in computing) a fixed value written directly in source code, such as a number, string, boolean, or character, that represents itself rather than a variable.
constant value in code

Examples

The programmer used the number literal 100 to set the initial counter value in the function.
Many beginners confuse the string literal 'hello' with a variable that can change later.
In the exam, she explained that a boolean literal like true represents a fixed value in code.

Synset Examples

The programmer used the number literal 100 to set the initial counter value in the function.
Many beginners confuse the string literal 'hello' with a variable that can change later.
In the exam, she explained that a boolean literal like true represents a fixed value in code.

Related Terms

  • Synonym: constant - a value that does not change during program execution
  • Antonym: variable - a named storage location whose value can change

Etymology

From Late Latin 'litteralis' (of or relating to letters); extended in modern computing to mean a value written literally in code.
