to containerise

1. VERB

/ kənˈteɪnəˌraɪz / (AmE), / kənˈteɪnəraɪz / (BrE)
to package or transport something in a container, especially for shipping or storage
package in a container
british

Examples

All the goods were containerised before being shipped overseas.
Many modern ports are designed to efficiently handle containerised cargo.
The company decided to containerise their products to reduce damage in transit.

Synset Examples

The cargo was containerized for safe and efficient shipping

Related Terms

  • Synonym: to containerize - package in a container (American spelling)

Etymology

From noun 'container' + suffix '-ise', modelled after 'containerize'.

WordNet ID: oewn-01485374-v

2. VERB NEW

/ kənˈteɪnəˌraɪz / (AmE), / kənˈteɪnəraɪz / (BrE)
(in computing) to package and deploy software by placing it in a lightweight, isolated runtime container so it runs reliably across different systems.
package software in a runtime container
british

Examples

Tech teams containerise their apps so they run consistently across development and production environments.
By containerising the service, the engineer isolated dependencies and simplified deployment to the cloud.
After the update, they had containerised the database and rolled back when tests failed.

Synset Examples

Tech teams containerise their apps so they run consistently across development and production environments.
By containerising the service, the engineer isolated dependencies and simplified deployment to the cloud.
After the update, they had containerised the database and rolled back when tests failed.

Related Terms

  • Synonym: to containerize - package and deploy software in containers (American spelling)
  • Synonym: dockerize - adapt or package software to run within Docker containers (often used informally)
  • Synonym: package - prepare software and its dependencies for distribution and deployment

Etymology

From noun 'container' (software container) + suffix '-ise', modelled on American 'containerize'.
