event

1. NOUN

/ ɪˈvɛnt / (AmE), / ɪˈvɛnt / (BrE)
A phenomenon occurring at a single point in space and time, considered as the basic observable entity in the theory of relativity.
phenomenon at a specific space-time point (physics)

Examples

In Einstein's theory of relativity, an event is defined by its unique position in both space and time.
Physicists use space-time diagrams to represent the position of an event in the universe.
Two events that appear simultaneous for one observer may not be for another.

/ ɪˈvɛnt / (AmE), / ɪˈvɛnt / (BrE)
a particular situation or condition, often imagined or specified, especially for considering what might happen; a set of circumstances

Examples

In the event of an emergency, please follow the exit signs.

5. NOUN NEW

/ ɪˈvɛnt / (AmE), / ɪˈvɛnt / (BrE)
(in computing) a software-generated occurrence or signal that indicates something has happened and may trigger a response (such as a handler or callback) in an event-driven system.
software signal that triggers a response

Examples

In event-driven systems, an event notifies the program that user input needs immediate handling.
The UI library emits an event when the button is clicked, invoking registered callbacks.
After detecting a network packet, the server created an event that started the processing pipeline.

Synset Examples

Related Terms

  • Synonym: signal - a transmitted indication or message that prompts a system to take action
  • Synonym: callback - a function provided to be called in response to an event or completed operation

Etymology

From Latin eventus (occurrence, result); metaphorically extended to computing to mean a software signal that something has happened.
