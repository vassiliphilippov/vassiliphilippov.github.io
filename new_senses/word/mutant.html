mutant

1. ADJ

/ ˈmjuːtənt / (AmE), / ˈmjuːtənt / (BrE)
tending to undergo or resulting from mutation, especially used in genetics or biology.
undergoing or resulting from mutation

Examples

Her research focuses on a mutant protein that affects cell division.
Only a mutant cell exhibited resistance to the disease.
The scientist studied several mutant strains of bacteria.

Synset Examples

a mutant gene

Etymology

From Latin 'mutans', present participle of 'mutare' (to change).

2. NOUN

/ ˈmjuːtənt / (AmE), / ˈmjuːtənt / (BrE)
An animal or organism that has undergone genetic mutation, resulting in a change from the normal type.
organism with genetic mutation

Examples

A mutant was discovered in the population that was resistant to the disease.
Some crops are engineered to contain a beneficial mutant gene.
The scientists studied a mutant mouse with white fur instead of brown.

Related Terms

  • Synonym: mutation - the process by which genetic material changes

Etymology

From Latin 'mutare' meaning 'to change', via mutation.

WordNet ID: oewn-01326946-n

3. NOUN

/ ˈmjutənt / (AmE), / ˈmjuːtənt / (BrE)
(biology) an organism that exhibits characteristics resulting from a genetic or chromosomal alteration.
organism with genetic alteration

Examples

A single mutant in the population displayed resistance to the pesticide.
Some farmers fear that mutant crops could spread uncontrollably in the environment.
The scientists studied the mutant to understand how the gene affected pigmentation.

Related Terms

  • Synonym: mutation - change in the structure of a gene or chromosome

Etymology

From Latin mutare (to change), via French mutant.

(computing, software testing) a version of a program produced by making small, deliberate changes to source code so that test suites can be evaluated by seeing which modified versions are detected.
modified program for testing

Examples

The mutation testing framework creates a mutant program to assess the effectiveness of the test suite.
Developers ran the test suite against each mutant to find missing checks in the code.
If many mutants survived, the team improved the unit tests before releasing the feature.
