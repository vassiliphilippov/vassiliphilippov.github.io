to recode

1. VERB

/ ˌriːˈkoʊd / (AmE), / ˌriːˈkəʊd / (BrE)
to convert information or data into a different code or format, often to change its structure or adapt it for a new purpose
convert information into a different code or format

Examples

She was able to recode the information in her mind to make it easier to remember.
The researchers had to recode the survey responses to fit the updated categories.
You may need to recode this data set before analysis to ensure consistency.

Synset Examples

People recode and restructure information in order to remember it

Related Terms

  • Synonym: encode - to convert information into a particular code
  • Antonym: decode - to convert coded information back into its original form

Etymology

Formed from the prefix 're-' (again) + 'code', meaning to code again.

2. VERB NEW

/ riːˈkoʊd / (AmE), / riːˈkəʊd / (BrE)
(in genetics, especially synthetic biology) to change an organism's genetic code by reassigning codons so they encode different or nonstandard amino acids.
alter an organism's genetic code

Examples

The team plans to recode the bacterium's genome to resist viral infection during the experiment.
After extensive work, researchers recoded the virus genome to prevent translation of harmful proteins.
Current synthetic biology papers describe methods for recoding genomes to expand the genetic code for novel proteins.

Synset Examples

The team plans to recode the bacterium's genome to resist viral infection during the experiment.
After extensive work, researchers recoded the virus genome to prevent translation of harmful proteins.
Current synthetic biology papers describe methods for recoding genomes to expand the genetic code for novel proteins.

Related Terms

  • Synonym: codon reassignment - the deliberate change of which amino acid a codon specifies in a genetic code
  • Synonym: genome editing - techniques used to make targeted changes to an organism's DNA
