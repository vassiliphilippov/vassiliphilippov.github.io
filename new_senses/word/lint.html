lint

1. NOUN

/ lɪnt / (AmE), / lɪnt / (BrE)
Very fine fibres, especially loose cotton or linen fibres, that separate from fabric, often accumulating as fluff or dust.
fine cotton or linen fibres; fluff

Examples

After doing the laundry, I always find lint in the dryer filter.
He brushed the lint off his black trousers before the meeting.
Old bandages used to be made of lint for dressing wounds.

Related Terms

  • Synonym: fluff - soft, light, and loose mass of fibres or down

Etymology

From Middle English *lynt*, from Old English *līn* (flax, linen).

Source: WordNet
WordNet ID: oewn-14962605-n
Synset POS: n

2. NOUN

/ lɪnt / (AmE), / lɪnt / (BrE)
Cotton or linen fabric with a raised nap on one side, used for dressing wounds.
wound dressing fabric
medical

Examples

After cleaning the cut, the nurse covered it with a sterile piece of lint.
During the war, wounded soldiers often had their injuries wrapped with lint.
The doctor asked for some lint to dress the burn before bandaging it.

Related Terms

  • Synonym: gauze - thin transparent fabric used for surgical dressings

Etymology

Probably from Middle English 'lin' (flax, linen), from Old English 'līn'.

Source: WordNet
WordNet ID: oewn-03681106-n
Synset POS: n

3. NOUN NEW

/ lɪnt / (AmE), / lɪnt / (BrE)
A program or tool that performs static analysis of source code to find possible errors, bugs, or style problems.
static code-analysis tool

Examples

The IDE now includes a built-in lint that flags unused variables and inconsistent formatting.
Before committing, she ran the lint to ensure the project met the team's style rules.
Many open-source projects require passing continuous integration including a lint step for code quality.

Synset Examples

Related Terms

  • Synonym: linter - a program that checks source code for stylistic or programming errors
  • Synonym: static analysis - automated checking of source code for potential errors without running the program

Etymology

Named after the Unix program 'lint' (1978), so-called because it finds small, undesirable "fluff" in source code; ultimately from Middle English *lynt* (flax, linen).

4. VERB NEW

/ lɪnt / (AmE), / lɪnt / (BrE)
To run a linting tool on source code; to perform static analysis that reports possible errors, bugs, or style violations.
run a linter / perform static analysis

Examples

Junior developers are taught to lint their code before asking for a code review.
The build script will automatically lint all changed files and report style violations.
Please remember to lint your functions so errors are easier to find during testing.
