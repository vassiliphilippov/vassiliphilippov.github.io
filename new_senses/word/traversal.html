traversal

1. NOUN

/ trəˈvɜrsəl / (AmE), / trəˈvɜːsəl / (BrE)
the act of traveling across or through something
the act of traveling across

Examples

Data structures are often discussed in terms of their traversal methods.
Her traversal of the dense forest was both challenging and rewarding.
The traversal of the mountain range took the explorers several days.

Related Terms

  • Synonym: crossing - the act of going across something

Etymology

From Latin 'traversare' (to cross), from 'trans-' (across) + 'versare' (to turn).

2. NOUN

/ trəˈvɜrsəl / (AmE), / trəˈvɜːsəl / (BrE)
The act of moving across a slope on skis in a zigzag path, usually to control speed or direction.
zigzag movement on skis across a slope

Examples

A smooth traversal can help you maintain balance while skiing downhill.
She perfected her traversal technique on the steepest parts of the mountain.
The instructor demonstrated a controlled traversal to show how to shift weight on skis.

Related Terms

  • Synonym: traverse - to move across a slope sideways on skis

Etymology

From traverse, from Old French traverser, from Latin transversare (to cross).

3. NOUN NEW

/ trəˈvɜrsəl / (AmE), / trəˈvɜːsəl / (BrE)
(in computing, especially cybersecurity) a vulnerability or attack in which an attacker manipulates file-path input to access files or directories outside the intended location (also called path or directory traversal).
path/directory traversal vulnerability

Examples

A directory traversal vulnerability allowed attackers to read configuration files on the web server.
The security team found a traversal flaw in the upload form that exposed sensitive user data.
Patching the server eliminated the traversal exploit that had been used to access password files.

Synset Examples

A directory traversal vulnerability allowed attackers to read configuration files on the web server.

Related Terms

  • Synonym: path traversal - a cyberattack that manipulates file paths to access files outside the intended directory
  • Synonym: directory traversal attack - an attack that uses traversal techniques to read or modify restricted server files

Etymology

From traverse, via Old French traverser and Latin transversare 'to cross'; used metaphorically in computing to mean moving across directory paths.
