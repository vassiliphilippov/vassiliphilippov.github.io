idempotent

1. ADJ

/ aɪˈdɛmpətənt / (AmE), / ˌɪdɛmˈpəʊtənt / (BrE)
(of a mathematical element, such as a matrix) remaining unchanged in value when multiplied by itself.
unchanged when multiplied by itself (math)

Examples

If a function is idempotent, applying it twice yields the same outcome as once.
In abstract algebra, an idempotent element has special properties.
The idempotent matrix showed the same result after repeated multiplication.

Synset Examples

this matrix is idempotent

Related Terms

  • Antonym: nilpotent - (of an element) becoming zero when raised to some power (math)

Etymology

From Latin idem (same) + potent (present participle of posse, to be able), meaning 'having the same power'.

(in computing) of an operation or request that, if performed multiple times, produces the same effect as performing it once; repeating it causes no additional change.
same effect when repeated (computing)

Examples

An idempotent operation in REST APIs can be retried safely after a network failure.
Because PUT is designed to be idempotent, repeating it should not create duplicate resources.
To simplify error handling, the team made key write endpoints idempotent wherever possible.

  • Antonym: non-idempotent - an operation where repeating it can produce different effects or additional changes

Etymology

From Latin idem (same) + potent (having power); borrowed from mathematics and extended to computing to describe repeated operations.
