trace

1. NOUN

/ treɪs / (AmE), / treɪs / (BrE)
a visible mark, such as a footprint, left by the passage of a person, animal, or vehicle
a visible mark left by passage

Examples

After the rain, all trace of the car had disappeared from the muddy road.
The detective found a trace of footprints near the window.
The hunters followed the trace left by the deer through the forest.

Related Terms

  • Synonym: track - a mark or line left by something that has passed; a path or trail

Etymology

From Old French *tracer* (to draw, mark), from Latin *tractus* (dragged, pulled).

15. NOUN NEW

/ treɪs / (AmE), / treɪs / (BrE)
(in computing) a recorded sequence of a program's execution events (for example function calls, errors, or timestamps) used to analyse behaviour or locate bugs.
program execution log for debugging

Examples

The debugger produced a detailed trace of function calls that led to the program crash.
Engineers reviewed the server trace to see which request caused the memory leak during testing.
She enabled system trace logging so they could reproduce the error with exact timestamps.
