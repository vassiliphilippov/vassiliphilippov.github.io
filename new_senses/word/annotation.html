annotation

1. NOUN

/ ˌænəˈteɪʃən / (AmE), / ˌænəˈteɪʃən / (BrE)
a note or comment added to a text or diagram to explain or criticize it
a note or comment added to a text

Examples

Her annotation pointed out a factual error in the diagram.
Please review the annotations in the margins before submitting your manuscript.
The editor's annotation clarified the historical context of the poem.

Synset Examples

his notes were appended at the end of the article
he added a short notation to the address on the envelope

Related Terms

  • Synonym: note - a brief comment or explanation added to a text
  • Synonym: commentary - an explanatory series of notes or interpretations

Etymology

From Latin annotātiō (a noting, marking or making known), from annotāre (to note down).

2. NOUN

/ ˌænəˈteɪʃən / (AmE), / ˌænəˈteɪʃən / (BrE)
the act of adding explanatory notes or comments to a text, diagram, or other data
the act of adding notes or comments

Examples

Each figure in the report includes a short annotation for reference.
Please submit your annotation of the article by the end of the week.
The teacher's annotation in the margins helped clarify the poem's meaning.

Related Terms

  • Synonym: commentary - a series of explanatory notes or comments on a text

Etymology

From Latin 'annotatio', from 'annotare' (to mark, to note).

3. NOUN NEW

/ ˌænəˈteɪʃən / (AmE), / ˌænəˈteɪʃən / (BrE)
(in computing) a metadata tag attached to source-code elements (such as classes, methods, or fields) that provides information to tools, frameworks, or compilers and may guide processing or configuration.
metadata tag in source code

Examples

The framework reads the annotation on the class to configure dependency injection.
You should add an annotation above the method to indicate it handles HTTP POST requests.
During compilation, the tool used the annotation to generate extra validation code automatically.

Synset Examples

The framework reads the annotation on the class to configure dependency injection.
You should add an annotation above the method to indicate it handles HTTP POST requests.
During compilation, the tool used the annotation to generate extra validation code automatically.

Related Terms

  • Synonym: metadata - data that provides information about other data; descriptive information attached to data objects.
  • Synonym: attribute - a language-specific term (for example, C#) for a construct that attaches metadata to program elements, similar to an annotation.

Etymology

Derived from the general English noun 'annotation' (from Latin annotatio 'a noting'); extended in computing to mean a metadata tag attached to program code.
