callable

1. ADJ

/ ˈkɔl.ə.bəl / (AmE), / ˈkɔːl.ə.bəl / (BrE)
(of a financial instrument, especially a bond) able to be redeemed by the issuer before its maturity date, usually at a specified price.
(finance) able to be redeemed early

Examples

Because the bond was callable, it was redeemed three years before its maturity date.
Investors must consider the risks associated with purchasing callable securities.
The company decided to issue callable bonds to retain financial flexibility.

Synset Examples

callable bonds

Related Terms

  • Synonym: redeemable - (finance) able to be bought back or paid off

Etymology

From 'call' + '-able'; used in finance to describe bonds that can be 'called' (redeemed) early by the issuer.

2. NOUN NEW

(computing) an object (for example, a function, method, or object implementing a call operator) that can be invoked or called to perform an action or return a value.
programmable object that can be invoked

Examples

In Python, a callable is any object that can be invoked with parentheses to run code.
The library's API accepts a callable parameter to customize how each item is processed asynchronously.
To test, she passed a simple callable returning a fixed value instead of a complex service.
