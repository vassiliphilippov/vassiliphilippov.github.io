instance

1. NOUN

/ ˈɪnstəns / (AmE), / ˈɪnstəns / (BrE)
an example or a specific case that is typical of a general category or group
a specific example or case

Examples

Can you provide an instance of successful collaboration in your experience?
In this instance, the law does not apply.
The report includes an instance where the system failed unexpectedly.

Synset Examples

this patient provides a typical example of the syndrome
there is an example on page 10

Related Terms

  • Synonym: example - a representative form or pattern; something used to illustrate a rule or concept

Etymology

From Latin 'instantia', meaning presence or example.

2. NOUN

/ ˈɪn.stəns / (AmE), / ˈɪn.stəns / (BrE)
an example or single occurrence of something; a particular case.
particular occurrence or example

Examples

Can you provide an instance where this strategy was successful?
In this instance, I believe the rules should be applied strictly.
There was one instance last year when the plan did not work as expected.

Synset Examples

it was a case of bad judgment
another instance occurred yesterday
but there is always the famous example of the Smiths

Related Terms

  • Synonym: example - something that is representative of a group or category
  • Synonym: case - a particular occurrence or situation

Etymology

From Latin 'instantia' (presence, example), from 'instare' (to stand upon).

3. NOUN NEW

/ ˈɪnstəns / (AmE), / ˈɪnstəns / (BrE)
(in cloud computing) a running virtual machine or compute resource created from a machine image, used to run software or services.
cloud virtual machine

Examples

We launched a new instance of the application on the cloud to handle traffic spikes.
Each instance runs its own operating system and can be resized without affecting other services.
The admin terminated several idle instances overnight to reduce cloud costs and free resources.

Synset Examples
