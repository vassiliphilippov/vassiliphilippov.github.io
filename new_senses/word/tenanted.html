tenanted

1. ADJ

/ ˈtɛnəntɪd / (AmE), / ˈtɛnəntɪd / (BrE)
(of a property) occupied or lived in by tenants.
occupied by tenants

Examples

Many of the city's older buildings remain tenanted, despite their age.
Only a small number of the apartments in the block are currently tenanted.
The cottages are tenanted year-round by families and students.

Synset Examples

not all the occupied (or tenanted) apartments were well kept up

Related Terms

  • Synonym: occupied - being lived in or used by people
  • Antonym: untenanted - (of property) not occupied by tenants

2. ADJ NEW

/ \u02c8t\u025bn\u0259nt\u026ad / (AmE), / \u02c8t\u025bn\u0259nt\u026ad / (BrE)
(in computing) used of software, platforms, or services that host multiple separate customers (tenants) on the same system while keeping each customer's data and settings logically separate.
serving multiple customers (multi-tenant)

Examples

The enterprise now uses a tenanted cloud platform to serve several subsidiaries.
Developers designed the application as a tenanted service so each customer has isolated data.
Security audits confirmed the database remained tenanted without any leaks between client partitions.

Synset Examples

The enterprise now uses a tenanted cloud platform to serve several subsidiaries.
Developers designed the application as a tenanted service so each customer has isolated data.
Security audits confirmed the database remained tenanted without any leaks between client partitions.

Related Terms

  • Synonym: multi-tenant - describing systems that support multiple separate customers (tenants) on shared infrastructure
  • Antonym: single-tenant - a system architecture that dedicates resources to a single customer rather than serving multiple tenants

Etymology

From noun 'tenant' + -ed; extended metaphorically from property occupancy to computing (multi-tenant) in the 21st century.
