STM

1. NOUN

/ ˌɛs tiː ˈɛm / (AmE), / ˌɛs tiː ˈɛm / (BrE)
abbreviation for 'short-term memory', the capacity for holding a small amount of information in an active, readily available state for a short period.
short-term memory (abbreviation)
medical

Examples

After hearing the phone number, I kept it in my STM until I found a pen.
Research shows that anxiety can disrupt STM performance in students.
She repeated the list aloud to strengthen her STM before writing it down.

Related Terms

  • Synonym: short-term memory - the capacity for retaining information briefly
  • Antonym: long-term memory - the system for storing information for a long time

Etymology

/ cc5bs tid0 c85bm / (AmE), / cc5bs tid0 c85bm / (BrE)
(in computer programming) an abbreviation for Software Transactional Memory, a concurrency-control system that lets groups of memory operations execute atomically and in isolation, simplifying multithreaded code without using explicit locks.
Software Transactional Memory (programming)

Examples

The team used STM to avoid complex locks while developing the multithreaded server.
Using STM, the programmer wrote simpler code that kept shared memory consistent across threads.
Researchers compared performance of lock-based methods with STM in several concurrent benchmarks.
