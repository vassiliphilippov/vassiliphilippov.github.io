to relativize

1. VERB

/ ˈrɛləˌtɪvaɪz / (AmE), / ˈrɛlətɪvaɪz / (BrE)
To consider or treat something as relative rather than absolute; to view in relation to something else.
to treat as relative, not absolute

Examples

In his essay, the philosopher attempts to relativize the concept of truth by examining it across cultures.
Sociologists often relativize moral values, arguing that they depend on social context.
Some critics argue that too much effort to relativize standards can undermine universal principles.

Related Terms

  • Antonym: absolutize - to make absolute; to treat as absolute

Etymology

Formed from 'relative' + '-ize', following the pattern of verbs meaning 'to make X' or 'to treat as X'.

(in computing) to convert an absolute file path or URL into a relative path with respect to a specified base location, so it can be used portably or within a different directory structure.

Examples

The build tool can relativize file paths to the project root for portability.

You should avoid relativizing paths that must remain absolute for system security.
