2. NOUN NEW

/ ˈeɪdʒæks / (AmE), / ˈeɪdʒæks / (BrE)
A web-development technique for making asynchronous requests from a web page to a server so parts of the page can update without a full reload; originally an acronym for Asynchronous JavaScript and XML.
asynchronous web-request technique

Examples

Modern web apps use Ajax calls to fetch data without refreshing the page.
The tutorial explained how Ajax can load new items into a list without reloading the whole site.
Developers replaced several full-page requests with Ajax to improve speed and user experience.

Synset Examples

Modern web apps use Ajax calls to fetch data without refreshing the page.
The tutorial explained how Ajax can load new items into a list without reloading the whole site.
Developers replaced several full-page requests with Ajax to improve speed and user experience.

Related Terms

  • Synonym: asynchronous request - a request that allows a web page to continue operating while waiting for a server response
  • Antonym: synchronous request - a request that blocks a page or operation until the server response is received

Etymology

From the acronym AJAX (Asynchronous JavaScript and XML), coined in the early 2000s to describe asynchronous browser–server requests.
