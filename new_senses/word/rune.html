rune

1. NOUN

/ ruːn / (AmE), / ruːn / (BrE)
a character from any of the ancient Germanic alphabets, especially those used in Scandinavia from the 3rd century to the Middle Ages
ancient Germanic alphabet character

Examples

Each rune was believed to possess magical properties in ancient times.
He spent years studying the meaning of every rune carved into the stone.
The archaeologists discovered an inscription containing a rare rune never seen before.

Synset Examples

each rune had its own magical significance

Etymology

From Old Norse 'rún' (mystery, secret), related to Old English 'rūn'.

2. NOUN NEW

/ ruːn / (AmE), / ruːn / (BrE)
(computing, especially in the Go programming language) a value representing a single Unicode code point (a single character) stored in a string or variable.
Unicode code point (programming)

Examples

In Go, a rune represents a single Unicode code point inside a string.
She used a slice of runes to iterate over multilingual text correctly.
When printing the file, convert each rune into its numeric code to debug encoding errors.
