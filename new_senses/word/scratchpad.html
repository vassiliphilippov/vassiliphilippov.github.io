scratchpad

1. NOUN

/ ˈskrætʃˌpæd / (AmE), / ˈskrætʃpad / (BrE)
(computing) a high-speed internal memory used for temporary storage of data or instructions.
temporary high-speed computer memory

Examples

Programmers optimize performance by utilizing the available scratchpad memory.
Some embedded systems employ a dedicated scratchpad instead of conventional RAM.
The processor uses a scratchpad to store frequently accessed variables for faster computation.

Related Terms

  • Synonym: cache - a hardware or software component that stores data so future requests are served faster

2. NOUN NEW

/ \u02c8skr\u00e6t\u0283\u02ccp\u00e6d / (AmE), / \u02c8skr\u00e6t\u0283pad / (BrE)
(in AI and software) a temporary, writable workspace used by a program or artificial agent to record intermediate reasoning steps, calculations, notes, or other transient information while performing a task.
temporary AI/workspace for intermediate reasoning

Examples

The chatbot uses a scratchpad to record intermediate calculations before finalizing the answer.
During training, researchers saved the model's chain-of-thought notes in a simple scratchpad file.
When solving puzzles, the virtual assistant wrote partial proofs into an internal scratchpad for later steps.
