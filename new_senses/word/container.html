container

1. NOUN

/ kənˈteɪnər / (AmE), / kənˈteɪnə / (BrE)
an object used for holding, storing, or transporting things, especially a large standardized metal box used for shipping goods
an object used to hold or transport things

Examples

He loaded the books into a large container for storage.
Please use a sealed container to keep the food fresh.
The shipping container arrived at the port yesterday.

Related Terms

  • Synonym: receptacle - a container used to hold something
  • Synonym: vessel - a hollow container, especially for liquids
  • Synonym: package - an object or group of objects wrapped in paper or packed in a box

Etymology

From Latin continere (to hold together, contain) via Late Latin container and French contener.

2. NOUN NEW

/ kənˈteɪnər / (AmE), / kənˈteɪnə / (BrE)
(in computing) a lightweight, standalone software package that bundles an application together with its libraries and dependencies so it can run reliably across different computing environments.
software package bundling an application

Examples

Developers put their application and libraries into a container so it works the same on any server.
The team used several containers to run microservices independently in the production environment.
After updating the code, she rebuilt the container and deployed it to the cloud.

Synset Examples

Etymology

Metaphorical extension of the physical noun 'container' (an object that holds things); popularized in the 2010s with containerization tools such as Docker.
