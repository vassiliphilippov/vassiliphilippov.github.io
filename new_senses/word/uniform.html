uniform

1. ADJ

/ ˈjuːnəˌfɔrm / (AmE), / ˈjuːnɪfɔːm / (BrE)
not varying or not different in form, character, or degree; remaining the same at all times or in all parts.
not varying; consistent; not differentiated

Examples

Students are expected to wear uniform clothing during the ceremony.
The paint was applied in a uniform layer across the wall.
The temperature in the room remained uniform throughout the night.

Related Terms

  • Synonym: consistent - always behaving or happening in a similar way
  • Antonym: variable - liable to change; not consistent or having a fixed pattern

Etymology

From Latin 'uniformis', from 'uni-' (one) + 'forma' (form).

7. ADJ NEW

/ ˈjuːnɪfɔrm / (AmE), / ˈjuːnɪfɔːm / (BrE)
(in computer graphics and shading languages) a qualifier or variable declared as holding a value that is constant across all shader invocations and is supplied by the host application.
shader qualifier; constant across shader invocations

Examples

The shader declares a uniform variable to hold the scene's ambient light intensity.
Before each frame, the engine updates the uniform values sent to the GPU for rendering.
They grouped related settings into a single uniform block to reduce shader state changes.

Synset Examples

Related Terms

  • Synonym: uniform variable - a shader variable declared with the 'uniform' qualifier; holds values provided by the host program
  • Antonym: varying - a shader variable that changes between vertices or fragments and is not constant across invocations

Etymology

From Latin 'uniformis' (of one form); used in graphics programming as a technical term for values that remain the same across shader executions.
