exit

1. NOUN

/ ˈɛɡzɪt, ˈɛksɪt / (AmE), / ˈɛksɪt, ˈɛɡzɪt / (BrE)
An opening or way out that allows escape or release from a place or situation.
an opening for escape or release

Examples

Steam poured from the exit valve at the back of the machine.
The only exit from the building was blocked by debris after the storm.
We searched for another exit but found ourselves trapped in the maze.

Synset Examples

he blocked the way out
the canyon had only one issue

Related Terms

  • Antonym: entrance - an opening or passage allowing entry

Etymology

From Latin exitus (a going out, exit), from exire (to go out).

terminate a program or process

Examples

The script will exit gracefully if the input is invalid or missing.
After the error, the background process exited with status code 1, prompting a restart.
At the shell prompt, type exit to end the remote session and close the connection.
