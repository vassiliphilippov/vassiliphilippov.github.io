typescript

1. NOUN

/ ˈtaɪpˌskrɪpt / (AmE), / ˈtaɪpˌskrɪpt / (BrE)
A typewritten copy of a document, especially a manuscript prepared for editing, review, or publication.
typewritten copy of a manuscript

Examples

After typing the novel, she kept the typescript safe in a folder.
He submitted his typescript to the publisher for consideration.
The original typescript had several handwritten edits in the margins.

Related Terms

  • Synonym: manuscript - a handwritten or typed document, especially a writer's draft

Etymology

From 'type' (from typewriter) + 'script' (from Latin scriptum, meaning writing).

2. NOUN NEW

/ ta6apccskr6apt / (AmE), / ta6apccskr6apt / (BrE)
(computing) A statically typed programming language developed by Microsoft that is a superset of JavaScript and compiles to JavaScript; commonly used to build large-scale web applications.
statically typed superset of JavaScript

Examples

We decided to rewrite the legacy app in TypeScript to improve code safety before release.
The tutorial shows how TypeScript compiles annotations into plain JavaScript for browser compatibility.
She learned TypeScript quickly by following online courses and building small projects.

Synset Examples

Related Terms

  • Synonym: JavaScript - a widely used scripting language for the web; TypeScript is a typed superset that compiles to it.
  • Synonym: static typing - a programming-language feature where types are checked at compile time; a core feature of TypeScript.

Etymology

Named from 'type' (static typing) + 'script' (from JavaScript); introduced by Microsoft in 2012.
