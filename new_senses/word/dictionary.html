dictionary

1. NOUN

/ ˈdɪkʃəˌnɛri / (AmE), / ˈdɪkʃənəri / (BrE)
A reference book, in print or electronic form, listing words of a language, usually in alphabetical order, and giving their meanings, definitions, pronunciations, and other information.
reference book listing words and meanings

Examples

Every student is advised to own a reliable English dictionary.
He reached for the dictionary to check the meaning of the word.
The online dictionary provides audio pronunciations and example sentences.

Related Terms

  • Synonym: lexicon - the vocabulary of a language or a reference book listing words
  • Degree Lower: thesaurus - a reference book listing synonyms and related words

Etymology

From Medieval Latin 'dictionarium', from Latin 'dictio' (saying, expression) + '-arium' (a place for).

2. NOUN NEW

/ \u02c8d\u026ak\u0283\u0259\u02ccn\u025bri / (AmE), / \u02c8d\u026ak\u0283\u0259n\u0259ri / (BrE)
(in computing) a data structure that stores associations between keys and values, allowing programs to look up, add, or remove a value by its key quickly.
key–value data structure

Examples

In Python, a dictionary maps keys to values for fast lookups in many programs.
She stored user preferences in a dictionary so the app could quickly retrieve settings.
When parsing the file, the developer built a dictionary from pairs of names and IDs.

  • Synonym: map - a data structure that associates keys with values; often used in high-level programming languages.
  • Synonym: hash table - an implementation of a dictionary that uses a hash function to map keys to storage locations, enabling fast access.

Etymology

Named by analogy with the reference book "dictionary" (which maps words to definitions); applied in computing to a structure that maps keys to values.
