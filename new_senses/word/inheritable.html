inheritable

1. ADJ

/ ɪnˈhɛrɪtəbəl / (AmE), / ɪnˈhɛrɪtəbəl / (BrE)
Able to be passed from one generation to another, especially through genetic transmission or legal succession.
able to be passed on or inherited

Examples

Eye color is an inheritable characteristic determined by several genes.
Genetic disorders can be inheritable, passing from parents to their children.
The estate included an inheritable title that was passed down for generations.

Synset Examples

inheritable traits such as eye color
an inheritable title

Related Terms

  • Synonym: heritable - able to be inherited; inheritable
  • Antonym: non-inheritable - not able to be inherited or passed on

Etymology

From inherit + -able, patterned after Medieval Latin *hereditabilis* (heritable).

2. ADJ NEW

/ \u026an\u02c8h\u025br\u026at\u0259b\u0259l / (AmE), / \u026an\u02c8h\u025br\u026at\u0259b\u0259l / (BrE)
(in computing) capable of being passed from a parent to a child entity, for example a class, process, or object, so that the child receives the same property, handle, or permission.
capable of being inherited (computing)

Examples

The base class defines an inheritable property that all subclasses automatically receive.
Make the file descriptor inheritable so child processes can access the same resource after fork.
Security settings should avoid marking credentials as inheritable to prevent accidental privilege escalation.

Synset Examples

Related Terms

  • Synonym: heritable - capable of being inherited; equivalent in meaning
  • Antonym: non-inheritable - not able to be inherited or passed on (opposite in computing contexts)

Etymology

From inherit + -able; extended metaphorically from biological and legal uses to computing.
