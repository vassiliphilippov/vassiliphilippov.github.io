untracked

1. ADJ

/ ʌnˈtrækt / (AmE), / ʌnˈtrakt / (BrE)
(of a place) not having any paths, roads, or tracks; untouched or unexplored.
without paths or tracks

Examples

Satellite images revealed miles of untracked snow in the mountains after the storm.
She longed to hike across untracked wildernesses where few humans had ever set foot.
The explorers ventured deep into the untracked forest, far from civilization.

Synset Examples

trackless wilderness
roadless areas

Related Terms

  • Synonym: trackless - having no paths or tracks
  • Synonym: roadless - lacking roads

Etymology

Formed by prefix 'un-' (not) + 'tracked' (provided with a track or path).

2. ADJ NEW

/ \u028cn\u02c8tr\u00e6kt / (AmE), / \u028cn\u02c8trakt / (BrE)
(in computing, especially version control) not recorded or managed by a version-control system (e.g., Git); a file or set of files that the system is not tracking and so are not included in commits until added.
not tracked by version control

Examples

The build failed because there were untracked files in the repository.
Before committing, she ran git status to find any untracked files to add.
He lost his changes when an important untracked file was not included in the commit.

Synset Examples

The build failed because there were untracked files in the repository.
Before committing, she ran git status to find any untracked files to add.
He lost his changes when an important untracked file was not included in the commit.

Related Terms

  • Synonym: unversioned - not placed under version control; not managed by a VCS
  • Antonym: tracked - (in version control) recorded and monitored by the VCS; included in commits

Etymology

Formed by prefix 'un-' (not) + 'tracked' (provided with a track or path).
