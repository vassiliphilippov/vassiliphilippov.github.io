DDT

1. NOUN

/ ˌdiː ˌdiː ˈtiː / (AmE), / ˌdiː ˌdiː ˈtiː / (BrE)
a synthetic insecticide that is also toxic to animals and humans; once widely used but now banned in many countries due to environmental and health concerns
a synthetic insecticide, now banned

Examples

After DDT was banned, bird populations began to recover.
Scientists discovered that DDT accumulates in the tissues of animals and humans.
The widespread use of DDT in agriculture caused serious environmental problems.

Etymology

From the chemical name dichlorodiphenyltrichloroethane.

2. NOUN NEW

/ \u02ccdi\u02d0 \u02ccdi\u02d0 \u02c8ti\u02d0 / (AmE), / \u02ccdi\u02d0 \u02ccdi\u02d0 \u02c8ti\u02d0 / (BrE)
(in software testing) a method in which the same test logic is executed repeatedly with different external data inputs so many cases can be checked efficiently.
data-driven testing (software)

Examples

The QA lead introduced DDT to reduce duplicated test code across multiple input cases.
Using DDT, testers ran the same test logic with hundreds of different input files quickly.
Many frameworks support DDT by reading test data from CSV or JSON configuration files.
