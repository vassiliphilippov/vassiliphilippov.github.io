marked-up

1. ADJ

/ ˈmɑrkt ˌʌp / (AmE), / ˈmɑːkt ˌʌp / (BrE)
(of a manuscript or document) having been written on or altered with corrections, annotations, or changes.
containing corrections or annotations

Examples

She spent hours reviewing the marked-up pages before making the final draft.
The document was so marked-up that it was difficult to read the original text.
The editor returned the marked-up manuscript with comments in the margins.

Synset Examples

foul (or dirty) copy

Related Terms

  • Synonym: corrected - having had errors or mistakes removed or amended

Containing markup language (such as HTML or XML) used to indicate structure, formatting, or semantic information for documents or data.
contains markup language (HTML/XML)

Examples

The marked-up document rendered correctly in the browser after we fixed the missing closing tag.
Developers inspected the marked-up file to find which HTML tags were breaking the mobile layout.
