zygote

1. NOUN

/ ˈzaɪɡoʊt / (AmE), / ˈzaɪɡəʊt / (BrE)
A cell formed by the fusion of a sperm and an egg, which is the first stage in the development of a new organism.
fertilized egg cell; early stage organism
medical

Examples

A fertilized egg is called a zygote, which will divide repeatedly to form an embryo.
Every multicellular organism begins as a single zygote after fertilization.
Once the sperm penetrates the ovum, a zygote is formed and development begins.

Related Terms

  • Degree Higher: embryo - an early stage of development after the zygote
  • Degree Lower: gamete - a reproductive cell (sperm or egg) that fuses to form a zygote

Etymology

From Greek zygōtos (joined, yoked), from zygoun (to join).

2. NOUN NEW

/ ˈzaɪɡoʊt / (AmE), / ˈzaɪɡət / (BrE)
(in computing) a startup process that preloads code and libraries and forks new application processes to speed startup and reduce memory use.
startup process that forks app processes

Examples

Android's zygote process preloads libraries so new apps start faster on low-end phones.
During boot the system spawned several zygote processes to reduce memory use across applications.
Server engineers sometimes prestart multiple zygotes to speed client startup during peak traffic.

Synset Examples

Android's zygote process preloads libraries so new apps start faster on low-end phones.

Related Terms

  • Synonym: fork - to create a new process by duplicating an existing one (common term in operating systems).
  • Synonym: process - an instance of a running program in an operating system.

Etymology

Metaphorical extension of the biological term from Greek zygōtos ‘joined’ (via biological zygote); applied in computing to a process that 'gives rise' to others.
