2. NOUN NEW

/ aɪ ɪl / (AmE), / aɪ ɪl / (BrE)
(in computing, especially Microsoft .NET) the Intermediate Language used by .NET language compilers — a low-level, platform-agnostic form of code (also called MSIL or CIL) that is later compiled or JIT-compiled to native instructions.
Microsoft .NET intermediate language

Examples

Developers inspect the generated IL to understand how C# code will be executed.
You can use a disassembler to view the IL produced by the compiler for debugging.
Performance tools sometimes show both native instructions and the original IL for comparison.

Synset Examples

Related Terms

  • Synonym: CIL - Common Intermediate Language — the standardized name for Microsoft's IL in the .NET specification.
  • Synonym: bytecode - low-level, platform-independent code executed by a virtual machine (e.g., Java bytecode); a similar concept in other platforms.

Etymology

Abbreviation of 'Intermediate Language'. Originally called MSIL (Microsoft Intermediate Language) and standardized as CIL (Common Intermediate Language) in the .NET specification.
