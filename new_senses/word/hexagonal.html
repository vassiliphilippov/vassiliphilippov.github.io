hexagonal

1. ADJ

/ hɛkˈsæɡənl̩ / (AmE), / hɛkˈsæɡənl̩ / (BrE)
having six sides, or shaped like a hexagon; relating to or divided into hexagons.
having six sides or a hexagon shape

Examples

He noticed hexagonal patterns forming in the ice crystals on the window.
The honeycomb is known for its hexagonal structure, which is highly efficient.
The tiles were arranged in a hexagonal grid across the floor.

Related Terms

  • Synonym: hexagon - a six-sided polygon
  • Degree Lower: pentagonal - having five sides or relating to a pentagon
  • Degree Higher: heptagonal - having seven sides or relating to a heptagon

Etymology

From Greek hexagonos (six-angled), from hex (six) + gonia (angle).

2. ADJ NEW

/ hɛkˈsæɡənəl / (AmE), / hɛkˈsæɡənəl / (BrE)
(in software engineering) relating to or using hexagonal architecture, a design pattern that isolates core application logic from external systems by using ports and adapters.
relating to hexagonal software architecture

Examples

The team adopted hexagonal architecture to keep core logic independent from external interfaces.
Using a hexagonal design allowed their microservices to be tested without depending on databases or UI.
Many engineers prefer hexagonal patterns because they separate application rules from delivery mechanisms cleanly.

Synset Examples

The team adopted hexagonal architecture to keep core logic independent from external interfaces.
Using a hexagonal design allowed their microservices to be tested without depending on databases or UI.
Many engineers prefer hexagonal patterns because they separate application rules from delivery mechanisms cleanly.

Related Terms

  • Synonym: hexagonal architecture - a software design pattern that decouples core logic from external interfaces using ports and adapters
  • Synonym: ports and adapters - an approach to organising software so the core logic communicates with the outside world through defined ports and adapter implementations

Etymology

Named after the hexagonal- shaped diagram used to illustrate the architecture; ultimately from Greek hexagonos 'six-angled'.
