/ bæʃ / (AmE), / bæʃ / (BrE)
A Unix/Linux command-line shell and scripting language (the Bourne Again SHell) used to run commands and write shell scripts.
Unix/Linux command shell

Examples

I wrote a quick backup script in bash to automate the nightly database dump and uploads.
She learned basic bash commands to navigate the server and manage files during the internship.
The tutorial explained how to loop over filenames using a simple bash for-loop on the command line.
