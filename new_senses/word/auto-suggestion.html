auto-suggestion

1. NOUN

/ ˌɔtoʊ səˈdʒɛstʃən / (AmE), / ˌɔːtəʊ səˈdʒɛstʃən / (BrE)
a psychological technique in which a person influences their own thoughts or behavior by repeating positive statements to themselves, as developed by Émile Coué
self-influence method via repeated suggestions

Examples

She practiced auto-suggestion every morning to boost her confidence.
The concept of auto-suggestion became popular in the early twentieth century.
Through repeated auto-suggestion, some believe that individuals can improve their mental well-being.

Related Terms

  • Synonym: affirmation - a positive statement repeated to encourage belief or change

Etymology

From French auto- (self) + suggestion, popularized by Émile Coué in early 20th century.

(in computing, especially user interfaces) a feature that automatically suggests or completes words, queries, or entries as the user types, helping them find results more quickly.
automatic suggestions in a UI

Examples

The search box offers auto-suggestion to speed up queries as you type.
When typing the address bar, her browser displayed helpful auto-suggestions based on browsing history.
Developers implemented auto-suggestion so users can find products with fewer keystrokes.

  • Synonym: autosuggest - a system that suggests likely completions for input fields in a user interface
  • Synonym: typeahead - a UI pattern that displays likely matches while the user types

From French auto- (self) + suggestion; later extended to computing to mean automatic suggestions in user interfaces.
