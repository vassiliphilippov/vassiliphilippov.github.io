ROM

1. NOUN

/ ɑr oʊ ɛm / (AmE), / ɑːr əʊ ɛm / (BrE)
(computing) Memory in a computer that can be read but not written to, which retains its contents even when the computer is turned off.
read-only computer memory
american

Examples

Older video game consoles used cartridges containing ROM chips to store games.
The BIOS is stored in ROM so it is not erased when the computer is powered off.
You can't modify data on ROM since it's not writable like RAM.

Related Terms

  • Antonym: RAM - computer memory that allows both reading and writing

Etymology

Acronym of 'Read-Only Memory'.

Source: WordNet
WordNet ID: oewn-04065351-n
Synset POS: n

2. NOUN NEW

/ ɑr oʊ ɛm / (AmE), / ɑːr əʊ ɛm / (BrE)
A digital file that contains a copy of the data from a read-only memory device (for example, a game cartridge), typically used with emulators.
digital image of cartridge/software

Examples

Retro gamers often download ROM images to play classic games on emulators.
She kept a backup of the original ROM after dumping the cartridge to her computer.
Distributing commercial ROM files is illegal in many countries despite emulator availability.
