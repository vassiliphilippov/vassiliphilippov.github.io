Wayland

1. PROPN

/ ˈweɪlənd / (AmE), / ˈweɪlənd / (BrE)
(in European mythology) a legendary supernatural smith and king of the elves, identified with the Norse figure Volund.
mythical smith from European legend

Examples

Archaeologists found artifacts linked to the myth of Wayland the Smith.
Many ancient ballads tell the story of Wayland, the master smith of legend.
The sword was said to be forged by Wayland himself, giving it magical powers.

Related Terms

  • Synonym: Volund - Norse mythology: legendary smith, counterpart of Wayland

Etymology

From Old English Weland, related to Old Norse Vǫlundr, legendary smith.

2. PROPN NEW

/ ˈweɪlənd / (AmE), / ˈweɪlənd / (BrE)
A display server protocol and reference compositor for Linux and other Unix-like systems, intended to replace the older X Window System (X11).
Linux display server protocol

Examples

Many Linux distributions now use Wayland as their default display server instead of X11.
The graphics driver caused flicker only under Wayland, but it worked fine with the X server.
Developers praised Wayland for its simpler security model and smoother frame timing on modern desktops.

Synset Examples
