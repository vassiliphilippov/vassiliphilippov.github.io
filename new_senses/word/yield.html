17. VERB NEW

/ jiːld / (AmE), / jiːld / (BrE)
(in programming) to pause a function or routine and return a value to the caller, allowing the function to resume later and produce values incrementally (generator/coroutine behaviour).
pause function and return a value (programming)

Examples

In Python, a function can yield values one at a time, creating a generator for large data.
The coroutine yielded control back to the event loop while waiting for the network response.
By yielding values lazily, the function avoided building the entire list in memory.

Synset Examples
