synchronized

1. ADJ

/ ˈsɪŋkrəˌnaɪzd / (AmE), / ˈsɪŋkrənaɪzd / (BrE)
happening, existing, or operating at the same time or in unison; made to occur together precisely
operating in unison; simultaneous

Examples

The synchronized flashing of the fireflies lit up the night.
The athletes performed a synchronized dive into the pool.
Their actions were so synchronized that it seemed rehearsed.

Synset Examples

the synchronized flapping of a bird's wings

Related Terms

  • Synonym: simultaneous - happening or done at the same time
  • Antonym: unsynchronized - not made to occur at the same time; not synchronized

Etymology

From Greek 'synchronos' (happening at the same time), via French 'synchroniser' and English 'synchronize'.

(in computing) protected so that only one thread or process can execute the code or access the resource at a time, preventing data races and ensuring thread-safe access.
protected for thread-safe execution

Examples

The Java method was declared synchronized to avoid concurrent writes to the list.
During debugging she noticed the synchronized block causing a performance bottleneck under heavy threading.
Make the cache access synchronized so multiple threads do not corrupt stored entries.
