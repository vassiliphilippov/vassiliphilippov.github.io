dumper

1. NOUN

/ ˈdʌmpər / (AmE), / ˈdʌmpə / (BrE)
a truck designed so that its contents can be emptied by raising the front of its platform, allowing the load to be discharged by gravity
truck for emptying loads by tipping

Examples

A mechanical problem caused the dumper to stop unloading its cargo halfway.
He operated a dumper to transport soil from the excavation area to the landfill.
The construction site was busy with several dumper trucks hauling gravel and sand.

Related Terms

  • Synonym: dump truck - a vehicle for carrying and dumping loose material such as sand or gravel

Etymology

From 'dump' (to tip out), with the suffix '-er' indicating an agent or thing that performs an action.

2. NOUN NEW

/ ˈdʌmpər / (AmE), / ˈdʌmpə / (BrE)
(computing) a program or tool that creates a copy (a "dump") of a computer's memory or other raw data for debugging, analysis, or forensic examination.
software tool that creates data/memory dumps

Examples

The crash analyst used a dumper to dump the program's memory for debugging.
They ran the dumper tool to capture the server's RAM after the unexpected reboot.
Several dumpers produced different-sized files for later forensic analysis of the breach.

Synset Examples

Related Terms

  • Synonym: memory dump - a file or image containing the contents of a computer's memory at a point in time, used for debugging or analysis
  • Synonym: core dump - a recorded state of a program's memory, typically produced when a program crashes

Etymology

From 'dump' (to empty or copy out) + agentive suffix '-er'.
