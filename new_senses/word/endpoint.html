endpoint

1. NOUN

/ ˈɛn(d)ˌpɔɪnt / (AmE), / ˈɛn(d)pɔɪnt / (BrE)
the place or moment at which something comes to an end or is completed
place where something ends

Examples

Each experiment had a clearly defined endpoint for measuring success.
The endpoint of the road marks the boundary of the city.
This trail's endpoint is at the top of the mountain, offering an amazing view.

Related Terms

  • Synonym: terminus - final point or end of something, especially in travel or transport
  • Antonym: beginning - the point in time or space at which something starts

(in computing) a specific network address or URL provided by a web service or API that clients call to access a resource or perform an action.
network address for a web API

Examples

Before release, the team tested the new endpoint to ensure correct authentication and responses.
You can call the endpoint with curl to retrieve JSON-formatted user profile data for debugging.
The mobile app communicates with a secure endpoint that returns product availability and pricing information.
