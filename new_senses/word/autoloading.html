autoloading

1. ADJ

/ ˌɔː.t̬əˈloʊ.dɪŋ / (AmE), / ˌɔː.təˈləʊ.dɪŋ / (BrE)
(Of a firearm) capable of automatically loading the next round after firing, allowing for continuous or rapid fire.
automatic loading (firearms)

Examples

Early 20th-century innovations introduced the first widely used autoloading rifles.
The soldier carried an autoloading pistol for quick follow-up shots.
This hunting shotgun is a reliable autoloading model used by many professionals.

Synset Examples

an autoloading rifle

Related Terms

  • Synonym: semiautomatic - (Of a firearm) partially automatic but requiring a trigger pull for each shot
  • Antonym: manual - (Of a firearm) requiring manual loading of each round

Etymology

From auto- (self) + loading, indicating self-loading capability.

(in computing) the automatic loading of code modules, classes, or components on first reference, so they are loaded only when needed.
automatic module loading (computing)

Examples

Frameworks increasingly rely on autoloading to load components as needed.
Thanks to autoloading, the application starts faster because modules load only when used.
In Ruby on Rails, autoloading reduces memory usage by deferring class loading until runtime.

Synset Examples

Frameworks increasingly rely on autoloading to load components as needed.

Related Terms

  • Synonym: lazy loading - a programming technique that delays initialization of resources until they are needed
  • Synonym: on-demand loading - loading resources only when a program requires them rather than at startup
  • Synonym: dynamic loading - the process of loading code or libraries into a program at runtime

Etymology

From auto- (self) + loading; extended metaphorically into computing to mean automatically loading code on first use.
