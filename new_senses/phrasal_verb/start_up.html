to start up

1. VERB

/ stɑrt ʌp / (AmE), / stɑːt ʌp / (BrE)
to begin operating or cause something (such as a machine, engine, or system) to begin operating
begin operating; set in motion

Examples

I always start up my laptop as soon as I get to work.
It took several tries to start up the old car in the morning.
The company plans to start up production again next month.

Synset Examples

We simply could not start the engine
start up the computer

Related Terms

  • Antonym: shut down - to cause a machine or system to stop operating

/ stɑrt ˈʌp / (AmE), / stɑːt ˈʌp / (BrE)

to begin or launch something

Examples

She decided to start up her own bakery after years of dreaming about it.
The computer takes a few minutes to start up when you first turn it on.
