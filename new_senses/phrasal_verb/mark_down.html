to mark down

1. VERB

/ mɑrk daʊn / (AmE), / mɑːk daʊn / (BrE)
to reduce the price of something
reduce in price

Examples

She waited for the shoes to be marked down before buying them.
The store decided to mark down all winter clothing by 30%.
Yesterday, they marked down the old model televisions to clear out inventory.

Related Terms

  • Synonym: discount - to reduce the price of something
  • Antonym: mark up - to increase the price of something

2. VERB NEW

/ mɑrk daʊn / (AmE), / mɑːk daʊn / (BrE)
to write or format text using the Markdown lightweight markup language (so plain text becomes structured, styled text).
format text using Markdown

Examples

Before publishing, please mark down the headings so they render correctly on the site.
She learned to mark down meeting notes so colleagues could copy and paste formatted text.
The documentation requires that you mark down code blocks using triple backticks for clarity.

Synset Examples

Before publishing, please mark down the headings so they render correctly on the site.
She learned to mark down meeting notes so colleagues could copy and paste formatted text.
The documentation requires that you mark down code blocks using triple backticks for clarity.

Related Terms

  • Synonym: Markdown - a lightweight plain-text markup language for formatting text, commonly used in documentation and web content
  • Synonym: format - to arrange or style text or data so it appears in a particular structure

Etymology

From Markdown (the lightweight markup language coined by John Gruber in 2004), a play on the term 'markup'.
