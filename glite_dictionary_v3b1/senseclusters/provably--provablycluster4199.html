provably

adverb
in a way that can be proven
6.7
Pronunciation: / ˈproʊvəbli / (AmE), / ˈpruːvəbli / (BrE)
In a manner that can be proven to be true; so as to be demonstrable by evidence or argument.

Examples

To be provably true
It is provably false that the earth is flat, given all the scientific evidence.
She made a provably accurate statement about the company's profits.
Their theory is not provably correct, since there is currently no supporting data.

Related Terms

  • synonym: demonstrably - in a way that can be demonstrated or shown to be true
