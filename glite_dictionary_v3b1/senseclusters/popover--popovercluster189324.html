popover

noun
small floating UI panel
7.0 0.03/m

(in user interfaces) a small on-screen floating panel that appears near a control or element to show information or actions without leaving the current view.

Examples

A popover window
Click the help icon and a small popover will appear next to the field with quick instructions.
On mobile, the settings popover hides nonessential options to keep the screen uncluttered and simple.
Developers used a CSS library to style the interactive popovers so they matched the brand colors.

Related Terms

  • antonym: modal - a dialog that requires the user to interact before returning to the main interface; unlike a popover, it blocks other actions.
  • synonym: tooltip - a small information box that appears on hover or focus; similar to a popover but often simpler and purely informational.
