keyboard buffer

noun
temporary memory storage for keystrokes
8.3 3.9
Pronunciation: /ˈkiː.bɔːrd ˈbʌf.ər/ (AmE), /ˈkiː.bɔːd ˈbʌf.ə/ (BrE)
a temporary area in a computer's memory that stores keystrokes until they can be processed

Examples

The keyboard buffer memory
Clearing the keyboard buffer can prevent unwanted input from being processed accidentally.
The software reads input from the keyboard buffer instead of waiting for the user to press a key.
When too many keys are pressed quickly, the keyboard buffer may overflow and some characters will be lost.
