minification

noun
code size reduction
Pronunciation: / ˌmɪnɪfɪˈkeɪʃən / (AmE), / ˌmɪnɪfɪˈkeɪʃən / (BrE)
the process of making computer code smaller by removing unnecessary characters or parts without changing its functionality

Examples

Code minification
Web developers often use minification to decrease the loading time of their websites.
After minification, the JavaScript file was much smaller and loaded more quickly.
Tools for minification automatically remove spaces and comments from code.

Related Terms

  • synonym: compression - the act of reducing the size of data or files
  • antonym: obfuscation - the process of making code difficult to understand
