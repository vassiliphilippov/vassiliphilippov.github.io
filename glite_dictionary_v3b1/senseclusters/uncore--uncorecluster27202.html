uncore

noun
non-core processor part
9.9
Pronunciation: / ˈʌnˌkɔr / (AmE), / ˈʌnˌkɔː / (BrE)
the part of a computer processor that is not included in the main processing cores, often handling memory and input-output functions

Examples

The uncore part
The uncore is responsible for managing communication between the CPU and memory.
Modern processors have a complex uncore to support high-speed data transfer.
Upgrading the uncore can improve overall system performance in some computer architectures.

Related Terms

  • antonym: core - the main processing unit of a computer processor
