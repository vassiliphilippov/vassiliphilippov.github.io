programmatically

adverb
using programming
Pronunciation: / ˌproʊɡræˈmætɪkli / (AmE), / ˌprəʊɡræˈmætɪkli / (BrE)

Examples

Programmatically update data

You can generate reports programmatically instead of creating them by hand.
The website content is changed programmatically whenever new data is available.
