requester

noun
initiator of API or network requests
Pronunciation: / rɪˈkwɛstər / (AmE), / rɪˈkwɛstə / (BrE)
(in computing) an entity (for example, a program, user, or process) that initiates a request to a service, API, or network resource.

Examples

API requester
Before sending data, the requester must include a valid API key in the header.
The server logged many failed attempts because anonymous requesters lacked proper authentication tokens.
In the load test, each simulated requester opened dozens of concurrent connections to measure throughput.

Related Terms

  • synonym: client - a program or device that requests services or resources from a server
  • antonym: server - a program or system that receives and responds to requests from clients or requesters
