face

noun
specific type style or size in typography
7.8 3.10/m
Pronunciation: / feÉªs / (AmE), / feÉªs / (BrE)
A particular size and style of type within a type family, used in printing and typography.

Examples

Change the face
Switching the face can dramatically change the appearance of a document.
The printer recommended using a classic face for the main text of the book.
This magazine prefers a modern face for its headlines to attract attention.

Synonyms

  • fount
    9.0 0.02/m
    fount is the traditional printing name for a complete set of characters in one size and style, while face is that typographic unit.
  • font
    2.8 6.32/m
    font is the modern printed or digital set of letters in a specific size and design, while face is the typographic name for that unit.
  • typeface
    4.1 0.71/m
    typeface is the overall design shared by many sizes and styles, while face refers to a single size and style within that family.
