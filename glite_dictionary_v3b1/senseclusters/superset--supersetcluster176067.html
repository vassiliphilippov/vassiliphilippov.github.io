superset

noun
expanded system
Pronunciation: / ˈsuːpərˌsɛt / (AmE), / ˈsuːpəsɛt / (BrE)
a set, system, or specification that contains all the elements or features of another and possibly more (used in mathematics, programming, etc.)

Examples

Superset of numbers
TypeScript is a superset of JavaScript that adds optional static typing and tooling.
The new protocol is a superset of the old one, so all previous features are still supported.
Python 3 can be seen as a superset of Python 2, with additional improvements and features.
In mathematics, the set of mammals is a superset of the set of dogs.
If set A includes every element of set B, then A is a superset of B.
The teacher explained that a superset can have extra elements not found in the smaller set.
