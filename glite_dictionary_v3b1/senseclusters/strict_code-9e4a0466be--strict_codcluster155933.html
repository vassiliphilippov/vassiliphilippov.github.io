strict code

noun
firmly enforced set of rules
Pronunciation: / strɪkt koʊd / (AmE), / strɪkt kəʊd / (BrE)
a set of rules or principles that are rigidly enforced and must be followed without exception

Examples

A strict code of honor
As a samurai, he adhered to a strict code of honor throughout his life.
Religious communities often have a strict code that all members are expected to follow.
The company operates under a strict code of ethics that guides every decision.

Related Terms

  • synonym: code of conduct - a set of rules outlining the norms or responsibilities
