fdisk

noun
disk partition tool
9.6
Pronunciation: / ˈɛfˌdɪsk / (AmE), / ˈɛfˌdɪsk / (BrE)
a computer program used to create, delete, or manage disk partitions on a hard drive

Examples

Run fdisk program
Before installing the new operating system, he ran fdisk to create the necessary partitions.
The technician used fdisk to delete an old partition and free up space on the drive.
If you are setting up a hard drive from scratch, fdisk can help you organize the storage.

Related Terms

  • antonym: format - to prepare a storage device for use by erasing all data and setting up a file system
  • synonym: partition manager - a program for creating and managing disk partitions
