inheritable

adjective
capable of being inherited (computing)
6.5
Pronunciation: /ɪn.ˈhɜːr.ɪ.tə.bəl/ (AmE), /ɪn.ˈhɜː.rɪ.tə.bəl/ (BrE)
(in computing) capable of being passed from a parent to a child entity, for example a class, process, or object, so that the child receives the same property, handle, or permission.

Examples

Inheritable property
The base class defines an inheritable property that all subclasses automatically receive.
Make the file descriptor inheritable so child processes can access the same resource after fork.
Security settings should avoid marking credentials as inheritable to prevent accidental privilege escalation.

Related Terms

  • synonym: heritable - capable of being inherited; equivalent in meaning
  • antonym: non-inheritable - not able to be inherited or passed on (opposite in computing contexts)
