borrow

verb
take a temporary reference (programming)

(in computing) to obtain a temporary reference to a value without transferring ownership, allowing code to read or (if permitted) modify the value while the original owner keeps ownership.

Examples

To borrow a value
In Rust, you can borrow a value immutably so multiple functions can read it safely.
She borrowed a mutable reference to the array to update its elements directly.
The library design avoids borrowing ownership; instead it hands out short-lived references for access.

Related Terms

  • antonym: ownership - the property or control over a value or resource; contrasts with temporary access
  • synonym: reference - a pointer or handle that gives access to data without transferring ownership
