autoincrement

verb
to increase automatically
7.3
Pronunciation: / ˌɔːtoʊˈɪnkrəmənt / (AmE), / ˌɔːtəʊˈɪŋkrɪmənt / (BrE)
to increase a value automatically, especially a number or counter in a computer system or database, without manual input

Examples

To autoincrement a number
In many databases, the primary key will autoincrement each time a new record is added.
The system is designed to autoincrement the invoice number for every new transaction.
If you enable this option, the counter will autoincrement without any user action.

Related Terms

  • antonym: decrement - to decrease a value by a set amount
  • synonym: increment - to increase a value by a set amount
