typable

adjective
can be typed
Pronunciation: / ˈtaɪpəbəl / (AmE), / ˈtaɪpəbəl / (BrE)
able to be entered or represented using a keyboard or in printed type

Examples

Typable password
Make sure your password is easily typable on any standard keyboard.
The document contains only typable characters, so you can enter it without special software.
Some symbols are not typable on a regular laptop keyboard.
Please use only typable characters when creating your password for the online form.
Some mathematical symbols are not typable on a standard keyboard.
The document must contain only typable text to ensure compatibility with all printers.

Related Terms

  • synonym: printable - able to be printed or displayed in print
  • antonym: untypable - not able to be entered using a keyboard
