true pine

noun
a coniferous tree of the genus Pinus
4.4
Pronunciation: / truː paɪn / (AmE), / truː paɪn / (BrE)
a coniferous tree of the genus Pinus, as distinguished from other trees called 'pine' that are not in this genus

Examples

A true pine tree
Several true pine cones were scattered across the forest floor.
Some botanists distinguish between true pine and other trees commonly called pine.
The forest was dominated by true pine species native to the region.
