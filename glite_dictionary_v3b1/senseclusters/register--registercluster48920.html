register

noun
memory device in computer processors
Pronunciation: / ˈrɛdʒɪstər / (AmE), / ˈrɛdʒɪstə / (BrE)
(computing) a small, fast storage location within a computer's processor or memory used for holding data or instructions temporarily.

Examples

A CPU register
Each register in the processor has a specific function and address.
The CPU uses a register to store intermediate calculation results.
The value was moved from memory into a register for faster processing.
