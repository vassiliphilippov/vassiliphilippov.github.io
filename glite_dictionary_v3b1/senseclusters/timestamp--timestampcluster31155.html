timestamp

verb
to record date and time
3.7
Pronunciation: / ˈtaɪmˌstæmp / (AmE), / ˈtaɪmˌstæmp / (BrE)
to add a record of the date and time when something happens or is created

Examples

To timestamp a file
The software will automatically timestamp every file you save for better tracking.
Please timestamp each document so we know when it was created.
All emails are timestamped to show exactly when they were sent.

Related Terms

  • antonym: erase - to remove or delete information
  • synonym: log - to record details of an event or activity
