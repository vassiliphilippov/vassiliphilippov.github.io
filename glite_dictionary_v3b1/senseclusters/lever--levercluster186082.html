lever

noun
metal tumbler in a lever lock
4.4 7.4
Pronunciation: / ˈliːvər, ˈlɛvər / (AmE), / ˈliːvə, ˈlɛvə / (BrE)
A flat metal tumbler inside a lever lock, which must be lifted to a certain height by the key to allow the lock to open.

Examples

Lever lock mechanism
A locksmith explained how a broken lever could prevent the mechanism from turning.
Each lever must be raised to the correct height for the lock to open.
The safe's security depended on the arrangement of each lever inside the lock.
