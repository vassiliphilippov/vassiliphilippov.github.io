clean-up event

noun
scheduled maintenance to remove temporary data

(in computing and IT) a scheduled or triggered maintenance operation that removes temporary files, caches, logs, or other obsolete data to free space or restore system health.

Examples

System clean-up event
The server runs a nightly clean-up event to delete temporary files and free disk space.
After the failed update, the ops team scheduled a clean-up event to remove corrupted cache entries.
Cloud platforms often trigger automatic clean-up events when storage usage exceeds predefined thresholds.
