grayout

noun
dimmed UI option
7.3
Pronunciation: / ˈɡreɪˌaʊt / (AmE), / ˈɡreɪˌaʊt / (BrE)
a state in a user interface where an option or control appears dimmed or inactive, usually shown in grey

Examples

Grayout of a button
When the feature is unavailable, a grayout appears over the button to indicate it cannot be used.
The software uses a grayout effect to show which menu items are currently disabled.
If you see a grayout on the option, it means you need to complete another step first.

Related Terms

  • antonym: activate - to make a function or option active
  • synonym: disable - to make a function or option inactive
