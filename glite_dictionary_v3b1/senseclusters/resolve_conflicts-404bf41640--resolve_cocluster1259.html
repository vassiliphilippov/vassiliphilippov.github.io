resolve conflicts

verb
reconcile merge conflicts in version control
3.5
Pronunciation: /riː.ˈzɑːlv ˈkɑːn.flɪkts/ /; / /rɪ.ˈzɑːlv ˈkɑːn.flɪkts/ (AmE), /rɪ.ˈzɒlv ˈkɒn.flɪkts/ /; / /riː.ˈzɒlv ˈkɒn.flɪkts/ (BrE)
(in software development) to reconcile conflicting changes when merging branches or versions in a version-control system by choosing, editing, or combining edits so the files can be merged.

Examples

To resolve conflicts in code
When merging branches, the team had to resolve conflicts before the automated tests could run.
I spent an hour trying to resolve conflicts in the code after pulling my colleague's changes.
Git marked three files as conflicted, so she needed to carefully resolve conflicts manually.

Related Terms

  • synonym: merge - to combine changes from different branches in version control; may produce conflicts that must be resolved
  • synonym: reconcile - to make two sets of changes agree; often used for resolving merge conflicts in code
