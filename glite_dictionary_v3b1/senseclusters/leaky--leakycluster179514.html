leaky

adjective
abstraction exposing implementation details (computing)
7.8 0.03/m
Pronunciation: / ÀàliÀêki / (AmE), / ÀàliÀêki / (BrE)
(in computing) of an abstraction, API, or library: failing to hide internal implementation details so that users must understand those details to use it correctly or avoid errors.

Examples

Leaky abstraction
The library's leaky abstraction forced developers to understand the network internals to fix performance issues.
They replaced the leaky API because its hidden caching behavior caused inconsistent results across platforms.
A leaky abstraction often increases debugging time, as engineers chase implementation-specific errors.
