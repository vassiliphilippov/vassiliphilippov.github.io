return on investment

noun
profit gained from an investment
Pronunciation: / rɪˈtɝːn ɑn ɪnˈvɛstmənt / (AmE), / rɪˈtɜːn ɒn ɪnˈvɛstmənt / (BrE)
the amount of profit or benefit gained from an investment, expressed as a percentage of the original amount invested

Examples

A return on investment rate
A good return on investment is crucial for attracting new shareholders.
Investors calculate return on investment before deciding where to allocate funds.
The company's latest project delivered a higher return on investment than expected.
