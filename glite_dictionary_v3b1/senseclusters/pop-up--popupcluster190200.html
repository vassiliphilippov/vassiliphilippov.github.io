pop-up

noun
sudden on-screen window
3.4
Pronunciation: / ˈpɑp ˌʌp / (AmE), / ˈpɒp ˌʌp / (BrE)
a small window or element that suddenly appears on a computer screen, often to show options, messages, or advertisements.

Examples

Pop-up window
The site displayed a pop-up window asking users to subscribe to the newsletter.
He blocked intrusive pop-ups in his browser to stop the constant advertising interruptions.
When the download finished, a small pop-up confirmed the file saved successfully to the folder.

Related Terms

  • synonym: dialog box - a small window that prompts the user to make a choice or enter information
  • synonym: modal window - a window that requires the user to interact with it before returning to the main interface
