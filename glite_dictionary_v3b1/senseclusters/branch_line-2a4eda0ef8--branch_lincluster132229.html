branch line

noun
secondary railway line off a main line
Pronunciation: / ˈbræntʃ laɪn / (AmE), / ˈbrɑːntʃ laɪn / (BrE)
a railway line that diverges from a main line to serve a smaller area or locality

Examples

A branch line railway
A new branch line connects the rural town to the main network.
Freight trains regularly use the branch line to deliver goods to local industries.
The old branch line was closed after passenger numbers fell.

Related Terms

  • antonym: main line - the principal railway route between two places
