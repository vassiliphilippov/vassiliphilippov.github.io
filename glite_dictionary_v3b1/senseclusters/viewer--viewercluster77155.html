viewer

noun
software application for viewing files
3.5
Pronunciation: / c8vjud059r / (AmE), / c8vjud059 / (BrE)
(in computing) a software application or component that displays digital media or documents (for example images, PDFs, or videos) for viewing rather than editing.

Examples

PDF viewer
I opened the PDF in the viewer to copy the text.
The photo viewer crashed when she tried to open the large RAW image file.
Developers added a dark mode to the document viewer for easier night reading.

Related Terms

  • antonym: editor - a program that lets you modify files rather than only view them.
  • synonym: reader - a program that opens and displays documents (e.g., PDF reader).
