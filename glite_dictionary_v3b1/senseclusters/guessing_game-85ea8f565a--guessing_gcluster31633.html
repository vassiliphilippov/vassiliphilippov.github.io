guessing game

noun
game of identifying with limited clues
1.0
Pronunciation: / ˈɡɛsɪŋ ɡeɪm / (AmE), / ˈɡɛsɪŋ ɡeɪm / (BrE)
a game in which participants try to identify something with limited or obscure clues

Examples

A guessing game with clues
The children played a guessing game at the party using animal noises.
Their relationship status has become a constant guessing game for gossip magazines.
We turned the mystery into a fun guessing game for everyone to join.
