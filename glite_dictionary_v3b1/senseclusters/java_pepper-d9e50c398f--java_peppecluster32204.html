Java pepper

noun
tropical vine with spicy fruits
Pronunciation: / ˈdʒɑːvə ˈpɛpər / (AmE), / ˈdʒɑːvə ˈpɛpə / (BrE)
A tropical Southeast Asian shrubby vine that produces spicy, berrylike fruits, used as a spice.

Examples

Java pepper spice
Botanists studied the growth patterns of Java pepper in tropical climates.
Historically, Java pepper was a highly prized spice in the spice trade.
The chef used Java pepper to give the curry a distinctive flavor.

Related Terms

  • synonym: black pepper - another spice from the pepper family, commonly used worldwide
  • synonym: long pepper - a similar spice-producing vine bearing elongated fruits
