noun
pull request (code merge proposal)

A pull request: a proposal to merge proposed code changes into a project's repository, submitted for review and discussion.

Examples

GitHub PR
The team merged the PR after a thorough code review and discussion.
She left comments on the PR requesting minor changes before approving the merge.
Continuous integration failed, so they updated the PR and pushed another commit.

Related Terms

  • synonym: pull request - a request to merge code changes into a shared repository, typically reviewed by teammates
