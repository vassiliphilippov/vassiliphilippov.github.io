macro

noun
computer command that expands into a set of instructions
6.7
Pronunciation: / ˈmækroʊ / (AmE), / ˈmakrəʊ / (BrE)
a single instruction in a computer program that automatically expands to carry out a set of instructions

Examples

Spreadsheet macro
He created a macro to automate repetitive spreadsheet tasks at work.
The software allows users to record and edit macros for more efficient workflows.
You can assign a macro to this button to speed up the formatting process.

Related Terms

  • synonym: script - a set of commands that automate tasks in a computer program
