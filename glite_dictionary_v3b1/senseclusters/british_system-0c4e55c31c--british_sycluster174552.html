British system

noun
British weights and measures system
1.0 4.8
Pronunciation: / ˈbrɪtɪʃ ˈsɪstəm / (AmE), / ˈbrɪtɪʃ ˈsɪstəm / (BrE)
the system of weights and measures used in Britain, based on units such as the foot, pound, second, and pint

Examples

The British system of weights
Measurements in the British system can be confusing for those used to the metric system.
The British system uses the pint as a common measure for liquids.
The engineer converted the data from the British system to metric for the report.
