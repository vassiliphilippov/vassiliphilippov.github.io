if I'm lying, I'm dying

x
emphatic assertion of truthfulness
4.3
Pronunciation: / ɪf aɪm ˈlaɪɪŋ aɪm ˈdaɪɪŋ / (AmE), / ɪf aɪm ˈlaɪɪŋ aɪm ˈdaɪɪŋ / (BrE)
(informal) used to strongly say that what you are telling is completely true

Examples

If I'm lying, I'm dying promise
If I'm lying, I'm dying—this is the best pizza I've ever tasted.
Honestly, if I'm lying, I'm dying, he finished the race in under four minutes.
She said, 'if I'm lying, I'm dying, I saw a shooting star last night.'
If I'm lying, I'm dying, that's exactly how it happened.
He swore the fish was ten feet long; if I'm lying, I'm dying.
I saw her win the whole game, if I'm lying, I'm dying.
