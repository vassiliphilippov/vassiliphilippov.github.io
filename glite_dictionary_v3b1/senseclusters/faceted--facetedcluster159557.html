faceted

adjective
relating to search interfaces with facet filters
Pronunciation: / ˈfæsɪtɪd / (AmE), / ˈfæsɪtɪd / (BrE)
(in computing and user-interface design) describing a search or navigation interface that lets users refine results by selecting among multiple facet filters (for example: category, price, brand).

Examples

Faceted search filters
Users can refine results with faceted search options on the catalog site.
The library's new system shows categories and counts for quick faceted browsing across collections.
When building the shop, they added a faceted menu so customers could filter by size and brand.

Related Terms

  • synonym: faceted search - a search interface that allows users to narrow results by selecting from several categories (facets).
  • synonym: filter - to narrow results by applying criteria or selecting options.
