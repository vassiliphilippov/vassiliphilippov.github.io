false morel

noun
a poisonous mushroom resembling a morel
9.9
Pronunciation: / fɔls məˈrɛl / (AmE), / fɔːls məˈrɛl / (BrE)
A species of poisonous mushroom that resembles true morels but belongs to the family Helvellaceae.

Examples

A false morel mushroom
In some regions, collecting false morels is strictly discouraged due to their toxic properties.
Many inexperienced foragers mistake the false morel for the edible true morel, which can be dangerous.
The false morel contains compounds that can be deadly if not properly prepared or consumed raw.
