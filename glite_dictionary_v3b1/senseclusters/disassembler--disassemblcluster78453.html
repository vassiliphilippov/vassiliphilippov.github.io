disassembler

noun
machine code converter
Pronunciation: / dɪsəˈsæmbələr / (AmE), / dɪsəˈsæmbələ / (BrE)
a computer program that converts machine code into human-readable assembly language instructions

Examples

Software disassembler
The security analyst used a disassembler to study the malware's behavior.
Learning to operate a disassembler is essential for reverse engineering software.
A disassembler can help programmers understand how compiled code works.

Related Terms

  • antonym: assembler - a program that converts assembly language into machine code
  • synonym: decompiler - a program that translates machine code into a higher-level programming language
