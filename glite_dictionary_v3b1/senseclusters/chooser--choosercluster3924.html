chooser

noun
software selection control
6.3
Pronunciation: /ˈtʃuː.zər/ (AmE), /ˈtʃuː.zə/ (BrE)
(in computing) a graphical user interface control or dialog that presents options and lets the user select a file, colour, or other item.

Examples

File chooser
The chooser window allowed her to pick multiple image files at once for upload.
He opened the color chooser to find a shade that matched the company logo.
In the settings, the file chooser defaulted to the Documents folder for convenience.

Related Terms

  • synonym: selector - a UI control that lets a user choose an item from a list or options
