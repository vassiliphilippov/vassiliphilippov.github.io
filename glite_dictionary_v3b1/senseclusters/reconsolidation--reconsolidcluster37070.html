reconsolidation

noun
memory restabilization
Pronunciation: / ˌriːkənˌsɑːlɪˈdeɪʃən / (AmE), / ˌriːkɒnˌsɒlɪˈdeɪʃən / (BrE)
the process by which a memory becomes stable again after being recalled or reactivated

Examples

Memory reconsolidation process
Scientists are studying how reconsolidation can be influenced to treat traumatic memories.
After recalling an old event, the process of reconsolidation helps the memory become stable again.
Therapies targeting reconsolidation may help people change unwanted emotional responses.

Related Terms

  • synonym: consolidation - the process of making a memory stable in the brain
