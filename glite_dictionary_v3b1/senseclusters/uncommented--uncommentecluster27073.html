uncommented

adjective
active code line
5.7
Pronunciation: / ʌnˈkɑːmɛntɪd / (AmE), / ʌnˈkɒmɛntɪd / (BrE)
not marked as a comment in computer code, so it is active and will be executed.

Examples

Uncommented code line
Be careful not to leave sensitive information in an uncommented line of code.
The script will only run if the command is uncommented in the configuration file.
After editing, she made sure all necessary settings were uncommented before saving the file.

Related Terms

  • antonym: commented - marked as a comment in code and not executed
