increasing monotonic

adjective
consistently increasing or never decreasing
7.9
Pronunciation: / ɪnˈkriːsɪŋ məˈnɑtənɪk / (AmE), / ɪnˈkriːsɪŋ məˈnɒtənɪk / (BrE)
(of a sequence, function, or process) consistently increasing or never decreasing.

Examples

A increasing monotonic sequence
A strictly increasing monotonic sequence must have each term greater than the one before.
The function is increasing monotonic, so it never decreases at any point on its domain.
This data set displays an increasing monotonic trend, as each value is equal to or greater than the previous one.
