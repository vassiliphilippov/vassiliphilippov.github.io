stub

verb
create or use placeholder code
8.4 0.00/m
Pronunciation: / st8cb / (AmE), / st8cb / (BrE)
to create or use a stub (a minimal placeholder implementation) in code, typically to simulate a component during testing or development.

Examples

To stub a function
The developer stubbed the payment function to avoid calling the live gateway.
To test the UI, they are stubbing network calls with canned responses for speed.
In rehearsal, she stubs the database access so the demo runs offline.

Synonyms
