buffer store

noun
temporary computer memory for data transfer
8.2
Pronunciation: / ˈbʌfɚ stɔr / (AmE), / ˈbʌfə stɔː / (BrE)
(computing) a section of a computer's memory used to temporarily hold data while it is being transferred between devices or components operating at different speeds.

Examples

A buffer store memory
The hard drive writes data to a buffer store before saving it permanently.
The printer used a buffer store to manage the incoming print jobs efficiently.
When streaming video, a buffer store helps prevent interruptions caused by network delays.

Related Terms

  • synonym: buffer - memory area for temporary data storage
