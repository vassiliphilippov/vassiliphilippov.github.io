prop

noun
object used on stage or film set
4.0 5.03/m
Pronunciation: / prÉ‘p / (AmE), / prÉ’p / (BrE)
an object used by actors on the stage or set of a play, film, or television show, usually as part of the action or setting

Examples

Stage prop
Each prop must be returned to its designated area after use.
She designed a realistic crown as a prop for the school play.
The actor forgot to pick up the prop sword before going on stage.

Synonyms

  • property
    6.8 7.20/m
    property is a formal production term for movable set objects used in scenes, while prop is the common short word actors use.

Related Terms

  • synonym: stage property - an object used in theatrical productions
