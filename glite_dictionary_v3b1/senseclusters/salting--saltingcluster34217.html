salting

noun
adding random data to passwords before hashing
7.7
Pronunciation: /ˈsɔːl.tɪŋ/ (AmE), /ˈsɔːl.tɪŋ/ (BrE)
(in computing, especially information security) the practice of adding random data (a "salt") to a password or other secret before hashing, so that precomputed attacks (for example, rainbow-table attacks) are ineffective.

Examples

Salting passwords
The company began salting user passwords to make their stored hashes much harder to crack.
During the security audit, she recommended salting and hashing every password with a unique random value.
Modern authentication libraries enable automatic salting so developers do not implement it incorrectly.

Related Terms

  • synonym: hashing - the process of transforming data into a fixed-size string (a hash) using a hash function, commonly used to store passwords securely
  • synonym: salt - a random value added to data (especially passwords) before hashing to make attacks harder
