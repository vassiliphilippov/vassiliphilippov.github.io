definer

noun
database authority account
8.5
Pronunciation: / dɪˈfaɪnər / (AmE), / dɪˈfaɪnə / (BrE)
the user account or owner specified as the authority for a database routine, view, or function in computing

Examples

Definer of a function
The definer of the stored procedure determines which privileges are used during execution.
If the definer account is deleted, the database function may stop working.
You should check the definer setting to ensure the correct permissions are applied.

Related Terms

  • antonym: invoker - the user who executes a database routine or function
  • synonym: owner - the user who has control over a database object
