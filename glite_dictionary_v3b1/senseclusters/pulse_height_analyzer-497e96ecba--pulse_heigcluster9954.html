pulse height analyzer

noun
electronic instrument for selecting voltage pulses by height
8.2
Pronunciation: / pʌls haɪt əˈnælaɪzɚ / (AmE), / pʌls haɪt əˈnælʌɪzə / (BrE)
a scientific instrument consisting of an electronic circuit that allows only voltage pulses of predetermined height to pass through

Examples

A pulse height analyzer device
Modern nuclear laboratories typically use a pulse height analyzer to process and classify signal voltages.
Technicians adjusted the pulse height analyzer to improve the accuracy of particle detection.
The pulse height analyzer was set to detect only signals above a certain threshold.
