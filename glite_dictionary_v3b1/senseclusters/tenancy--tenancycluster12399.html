tenancy

noun
multi-tenant software arrangement
Pronunciation: / ˈtɛnənsi / (AmE), / ˈtɛnənsi / (BrE)
(in computing, especially cloud and SaaS) the arrangement by which a single software instance serves multiple customers (tenants) while keeping each tenant's data and configuration logically isolated; often referred to as multi-tenancy.

Examples

Cloud tenancy model
The platform's tenancy model isolates each customer's data in a separate logical partition for security.
When designing the app, they chose a multi-tenant tenancy approach to reduce infrastructure costs.
Migrating to single-tenant hosting required changing the tenancy boundaries and exporting each customer's data separately.

Related Terms

  • synonym: multi-tenancy - an architecture in which a single software instance serves multiple customers with isolated data and configurations
  • antonym: single-tenant - an architecture in which each customer has a separate instance of the software (opposite of multi-tenancy)
