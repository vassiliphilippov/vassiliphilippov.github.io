run on

verb
continue uninterrupted
2.8 1.27/m
Pronunciation: / rÊŒn É‘n / (AmE), / rÊŒn É’n / (BrE)
to continue for longer than expected or without interruption

Examples

Meetings run on late
His lectures often run on longer than scheduled.
She let the engine run on for a few minutes before turning it off.
The meeting ran on for hours without any resolution.

Synonyms

  • keep going
    1.6 11/m
    keep going is to continue without stopping, often said when things remain working, while run on suggests continuing longer than expected.
  • run for
    2.8 5.11/m
    run for means to last for a set period, often with a stated time, while run on implies continuing unexpectedly long.
  • carry on
    2.7 11/m
    carry on is to continue after a break or despite problems, while run on often means continuing without stopping or too long.
  • carry on with
    2.6 0.98/m
    carry on with is to resume or continue a specific task after a pause, while run on suggests going on without stopping.
