cookbook

noun
collection of automation recipes (DevOps)
Pronunciation: / c8k8a k.b8a k / (AmE), / c8k8a k.b8a k / (BrE)
(in computing, especially DevOps) a collection of reusable automation recipes or configuration files used to provision, configure, or deploy software and infrastructure.

Examples

DevOps cookbook
The DevOps team published a cookbook for automating database backups across environments.
When onboarding new interns, we point them to the internal cookbook with deployment procedures.
You can import the Chef cookbooks to quickly configure servers with required packages.

Related Terms

  • synonym: playbook - In tools like Ansible, a file describing automation tasks; similar in purpose to a cookbook.
  • synonym: recipe (computing) - An individual automation script or set of steps within a cookbook that configures part of a system.
