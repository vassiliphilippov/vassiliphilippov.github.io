run roughshod

verb
treat harshly or inconsiderately
7.0 6.7
Pronunciation: /rʌn ˈrʌf.ʃɑːd/ (AmE), /rʌn ˈrʌf.ʃɒd/ (BrE)
to treat someone or something with little care or consideration; to act with disregard for others' feelings or rights

Examples

To run roughshod over rules
If we run roughshod over the rules, it will create chaos for everyone.
The new manager was determined not to run roughshod over the staff's concerns.
They can't just run roughshod over community traditions whenever they want a new development.
