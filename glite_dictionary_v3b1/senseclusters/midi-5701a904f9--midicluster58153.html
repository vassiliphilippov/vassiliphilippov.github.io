MIDI

noun
musical communication protocol
Pronunciation: / ˈmɪdi / (AmE), / ˈmɪdi / (BrE)
a standard protocol for communication and control between electronic musical instruments, computers, and other related devices

Examples

MIDI interface
He used a MIDI interface to control multiple synthesizers at the same time.
The new keyboard connects via MIDI to the computer for easy recording.
With MIDI, musicians can synchronize their instruments and create complex arrangements.

Related Terms

  • synonym: protocol - a set of rules governing data exchange between devices
