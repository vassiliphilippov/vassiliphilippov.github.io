fixed-cycle operation

noun
operation performed in a set number of cycles
Pronunciation: / fɪkst ˈsaɪ.kəl ˌɑː.pəˈreɪ.ʃən / (AmE), / fɪkst ˈsaɪ.kəl ˌɒp.əˈreɪ.ʃən / (BrE)
an operation performed in a set number of regularly timed execution cycles, especially in computing or engineering processes.

Examples
