file clerk

noun
employee who maintains files and records
Pronunciation: / ˈfaɪl ˌklɜrk / (AmE), / ˈfaɪl ˌklɑːk / (BrE)
an employee responsible for organizing and maintaining the files and records of an organization

Examples

A file clerk organizes files
A file clerk needs to be organized and detail-oriented to keep track of important documents.
After several years as a file clerk, she was promoted to office manager.
The file clerk carefully labeled each folder before placing it in the cabinet.

Related Terms

  • synonym: clerk - a person employed in an office for administrative duties
