Devanagari script

noun
Indian syllabic writing system
Pronunciation: / ˌdeɪvəˈnɑːɡəri skrɪpt / (AmE), / ˌdeɪvəˈnɑːɡəri skrɪpt / (BrE)
a syllabic writing system used for Sanskrit, Hindi, and several other Indian languages

Examples

Devanagari script writing system
Learning the Devanagari script is essential for reading and writing in Hindi.
Many ancient Hindu scriptures were originally written in Devanagari script.
The Sanskrit language uses the Devanagari script for most of its classical texts.
