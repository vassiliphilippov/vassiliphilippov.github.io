canonicalize

verb
to standardize format
7.8
Pronunciation: /kə.ˈnæn.ɪ.kə.laɪz/ (AmE), /kə.ˈnɒn.ɪ.kə.laɪz/ (BrE)
to convert something to a standard or recognized form, especially in computing or data processing

Examples

To canonicalize a file path
The program will canonicalize all URLs to ensure consistency in the database.
Before comparing the documents, the system must canonicalize their formatting.
Developers often canonicalize file paths to avoid errors caused by different directory structures.
