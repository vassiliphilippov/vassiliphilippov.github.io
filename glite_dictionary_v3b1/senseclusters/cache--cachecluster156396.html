cache

noun
(computing) memory for quick data access
Pronunciation: / kæʃ / (AmE), / kæʃ / (BrE)
(computing) a special area of memory used to store frequently accessed data temporarily, making data retrieval faster for future requests.

Examples

CPU cache
A larger CPU cache can significantly improve system performance.
Clearing your browser's cache can resolve many loading issues on websites.
The processor uses a cache to speed up access to frequently used instructions.

Related Terms

  • synonym: buffer - a temporary storage area used to hold data while it is being transferred
