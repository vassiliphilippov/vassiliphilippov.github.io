int

noun
software interrupt instruction
9.9 0.03/m
Pronunciation: / Éªnt / (AmE), / Éªnt / (BrE)
an assembly-language instruction that triggers a software interrupt to request services from the operating system

Examples

Int instruction call
The programmer used the int instruction to access system-level functions in the operating system.
In x86 assembly, int 21h is commonly used for DOS system calls.
Debuggers often monitor the int instruction to catch software interrupts during program execution.

Synonyms

  • supervisor call instruction
    9.8 0.00/m
    a supervisor call instruction is a machine command that transfers control to the system supervisor for OS services, while int is the assembly name that triggers that software interrupt.

Related Terms

  • antonym: call - an instruction to invoke a subroutine or function
  • synonym: interrupt - a signal that temporarily halts a program to allow a special process
