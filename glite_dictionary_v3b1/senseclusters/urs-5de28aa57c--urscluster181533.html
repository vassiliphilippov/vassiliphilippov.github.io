URS

proper noun
Soviet sports code
9.5
Pronunciation: / juː ɑːr ɛs / (AmE), / juː ɑːr ɛs / (BrE)
the three-letter code used in international sports to represent the Union of Soviet Socialist Republics (USSR) from 1922 to 1991

Examples

Olympic URS code
During the 1980 Olympics, all athletes from the Soviet Union competed under the URS code.
The scoreboard displayed URS whenever a Soviet athlete won a medal.
After 1991, the URS code was no longer used in international competitions.
