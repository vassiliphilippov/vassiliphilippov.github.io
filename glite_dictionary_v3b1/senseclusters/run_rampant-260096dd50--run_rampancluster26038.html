run rampant

verb
grow or spread wildly and uncontrollably
Pronunciation: / rʌn ˈræmpənt / (AmE), / rʌn ˈræmpənt / (BrE)
to grow, spread, or behave in an uncontrolled or wild manner, especially to a harmful degree

Examples

To run rampant in school
After the principal left, chaos seemed to run rampant in the school halls.
If pests are left unchecked, they will run rampant throughout the crops.
Without regulations, misinformation can run rampant on social media platforms.
