shim

noun
software compatibility layer
7.6
Pronunciation: /ʃɪm/ (AmE), /ʃɪm/ (BrE)
(computing) a small piece of code or library that provides compatibility by intercepting, emulating, or translating calls so older software can run on a newer system.

Examples

A software shim
The developers added a shim so the old plugin runs on the new browser.
We loaded several shims to emulate missing APIs during the migration test.
When the OS updated, a small shim intercepted calls and provided compatibility.

Related Terms

  • synonym: adapter - a component that converts or mediates between two different interfaces or formats
  • synonym: wrapper - a small piece of code that 'wraps' another component to change or adapt its interface
