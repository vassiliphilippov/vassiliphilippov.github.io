mashup

noun
combined web application
4.8
Pronunciation: /ˈmæ.ʃʌp/ (AmE), /ˈmæ.ʃʌp/ (BrE)
a web or software application that combines content, data, or functions from multiple sources into a single service or interface

Examples

Web mashup

Many music fans enjoy a mashup that blends songs from different artists into a single track.
Our team created a mashup that displays weather updates alongside local news headlines.
