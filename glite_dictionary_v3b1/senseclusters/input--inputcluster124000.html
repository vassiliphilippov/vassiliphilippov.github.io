input

noun
signal or data sent into a system
Pronunciation: / ˈɪnˌpʊt / (AmE), / ˈɪn.pʊt / (BrE)
a signal or piece of data that is sent into an electronic system for processing

Examples

Audio input
The audio input on the computer is not working.
The engineer checked the sensor input to diagnose the problem.
You need to connect the microphone to the correct input port.
