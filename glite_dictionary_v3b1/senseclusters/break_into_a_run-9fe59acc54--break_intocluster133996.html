break into a run

verb
suddenly start running
Pronunciation: / breɪk ˈɪntu ə rʌn / (AmE), / breɪk ˈɪntuː ə rʌn / (BrE)
to suddenly begin running, typically after walking or standing still

Examples

To break into a run suddenly

When the thunder rumbled, the children broke into a run toward their house.
