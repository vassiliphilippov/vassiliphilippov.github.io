memory trace

noun
theoretical neural change storing a memory
Pronunciation: / ˈmɛməri treɪs / (AmE), / ˈmɛməri treɪs / (BrE)
a theoretical change within neural tissue believed to store specific memories

Examples

Brain memory trace
Neuroscientists are still investigating how a memory trace is formed and maintained in the brain.
Some researchers argue that each experience leaves a unique memory trace in our nervous system.
The loss of a memory trace can lead to the forgetting of specific information.
