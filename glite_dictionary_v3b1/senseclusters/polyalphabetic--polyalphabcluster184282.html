polyalphabetic

adjective
multiple alphabets used
9.8
Pronunciation: / ˌpɑliˌælfəˈbɛtɪk / (AmE), / ˌpɒliˌælfəˈbɛtɪk / (BrE)
using or involving more than one alphabet, especially in codes or ciphers to increase security

Examples

A polyalphabetic cipher
A polyalphabetic cipher makes it much harder for attackers to decode secret messages.
The security of the system relies on a polyalphabetic substitution method.
Students learned how polyalphabetic encryption improved upon earlier, simpler ciphers.

Related Terms

  • antonym: monoalphabetic - using only one alphabet in a code or cipher
