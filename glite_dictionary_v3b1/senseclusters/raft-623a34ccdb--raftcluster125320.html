Raft

noun
distributed consensus algorithm
9.9
Pronunciation: / ræft / (AmE), / rɑːft / (BrE)
(computing) A consensus algorithm used in distributed systems to manage a replicated log or state machine and ensure that multiple servers agree on the same sequence of updates.

Examples

Raft consensus algorithm
Many distributed databases use the Raft protocol to keep replicas synchronized across data centers.
During leader failure, the cluster automatically elects a new Raft leader to continue processing client requests.
The team implemented Raft in their service to ensure consistent state across multiple server replicas.

Related Terms

  • synonym: Paxos - another well-known family of distributed consensus algorithms used to achieve agreement among unreliable processors
  • synonym: consensus algorithm - an algorithm that helps multiple computers agree on the same value or sequence of updates
