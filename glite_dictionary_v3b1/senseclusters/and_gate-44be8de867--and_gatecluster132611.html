AND gate

noun
digital logic circuit; needs all inputs
Pronunciation: / ænd ɡeɪt / (AmE), / ænd ɡeɪt / (BrE)
a digital logic circuit that produces an output signal only when all of its input signals are present

Examples

Digital AND gate
Combining an AND gate with other logic gates allows for complex decision-making circuits.
In this experiment, students built a simple circuit using an AND gate to control the LED.
The AND gate produces a high output only when both of its inputs are high.

Related Terms

  • degree higher: NAND gate - a logic circuit that outputs false only if all inputs are true
  • antonym: OR gate - a logic circuit that outputs true if any input is true
