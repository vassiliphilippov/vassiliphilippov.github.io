erasable programmable read-only memory

noun
computer memory chip that can be erased and reprogrammed
7.7
Pronunciation: / ɪˈreɪsəbəl proʊˌgræməbl riːd ˈoʊnli ˈmɛməri / (AmE), / ɪˈreɪsəbl prəˈɡræməbl riːd ˈəʊnli ˈmɛməri / (BrE)
(in computing) a type of memory chip whose contents can be erased using ultraviolet light and then reprogrammed with new data.

Examples

Erasable programmable read-only memory chip
Engineers replaced the faulty erasable programmable read-only memory to update the system's instructions.
Many early computer systems used erasable programmable read-only memory for storing their firmware.
The erasable programmable read-only memory chip can be reprogrammed after exposure to ultraviolet light.
