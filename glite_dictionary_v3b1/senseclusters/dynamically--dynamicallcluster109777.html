dynamically

adverb
updated in real time (computing)
5.1
Pronunciation: / daɪˈnæmɪkli / (AmE), / daɪˈnæmɪkli / (BrE)
(in computing) in a way that is generated, updated, or changed at runtime by software or a system rather than being fixed in advance.

Examples

Update dynamically
The dashboard updates dynamically as new sensor data streams in from devices.
Instead of static pages, the site delivers content dynamically based on user behavior and preferences.
Developers used an API to render the notifications dynamically without requiring a full page reload.

Related Terms

  • synonym: automatically - without manual intervention; performed by a system in response to events or conditions
  • antonym: static - fixed or not changing over time; often used of content that does not update at runtime
