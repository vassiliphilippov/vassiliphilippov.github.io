payload

noun
data portion of a packet
4.9 0.16/m
Pronunciation: / \u02c8pe\u026a\u02cclo\u028ad / (AmE), / \u02c8pe\u026a.l\u0259\u028ad / (BrE)
the part of a transmitted packet, frame, or message that contains the actual data being sent, excluding headers, metadata, or protocol overhead.

Examples

Data payload
The network packet's payload contained the encrypted user data securely.
During troubleshooting, the engineer inspected the payload to identify the corrupted file transfer.
IoT devices send small payload segments frequently to report sensor readings to the server.

Related Terms

  • synonym: data - information, often in digital form, that is stored or transmitted.
  • antonym: header - control or addressing information attached to a packet or message that precedes the payload.
