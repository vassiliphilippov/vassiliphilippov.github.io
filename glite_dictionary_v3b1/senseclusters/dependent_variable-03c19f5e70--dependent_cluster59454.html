dependent variable

noun
variable whose value depends on another
1.7 4.3
Pronunciation: /dɪ.ˈpen.dənt ˈve.rɪ.ə.bəl/ /; / /dɪ.ˈpen.dənt ˈvæ.rɪ.ə.bəl/ (AmE), /dɪ.ˈpen.dənt ˈveə.rɪ.ə.bəl/ /; / /dɪ.ˈpen.dənt ˈvæ.rɪ.ə.bəl/ (BrE)
(statistics) a variable in an experiment or mathematical expression whose value is determined by one or more other variables, usually called independent variables.

Examples

The dependent variable value
In a scientific study, the dependent variable is the outcome the researchers are trying to measure.
Researchers adjusted the independent variable to observe changes in the dependent variable.
When analyzing the function y = 2x + 5, y is the dependent variable.

Related Terms

  • antonym: independent variable - a variable whose variation does not depend on that of another
