Synonyms

  • logical argument
    4.3 0.2/m
    logical argument is a carefully ordered proof using formal logic, while argument is a broader set of reasons meant to persuade.
  • line of reasoning
    4.2 0.4/m
    line of reasoning is a connected series of steps that lead to a conclusion, while argument is a general set of reasons used to persuade.
  • case
    2.7 20/m
    case is a presentation of facts and reasons to support a view, while argument is the general set of reasons offered to persuade.
  • demonstration
    4.1 0.2/m
    demonstration is a logical proof that shows something is true, while argument is often a persuasive set of reasons, not full proof.
