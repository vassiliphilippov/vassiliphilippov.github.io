mock

noun
simulated test object
7.0
Pronunciation: / mɑk / (AmE), / mɒk / (BrE)
a simulated object or service used in software testing that stands in for a real component to provide predictable behaviour or data.

Examples

A mock object
To isolate the module, the developer replaced the real database with a simple mock.
The test suite failed because the mock did not mimic the API's error responses accurately.
They created a mock of the payment gateway to run offline demonstrations for clients.

Related Terms

  • synonym: stub - a simplified implementation used in place of a real component during testing
  • synonym: test double - a generic term for objects that replace real components in tests (mocks, stubs, fakes)
