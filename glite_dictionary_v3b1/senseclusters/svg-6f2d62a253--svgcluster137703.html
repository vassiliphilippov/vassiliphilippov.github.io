SVG

noun
vector image format
4.4
Pronunciation: / ˌɛs vi ˈdʒi / (AmE), / ˌɛs viː ˈdʒiː / (BrE)
a file format and markup language for two-dimensional vector graphics, commonly used for images on the web that can be resized without losing quality

Examples

An SVG file
The designer exported the logo as an SVG to keep it sharp at any size.
You can edit an SVG file with a text editor because it is based on XML.
Web developers prefer using SVG images for icons due to their scalability.
The company logo was saved as an SVG so it could be resized without losing quality.
You can edit an SVG file with a simple text editor or a graphics program.
Modern browsers support displaying SVG images directly on web pages.

Related Terms

  • antonym: JPEG - a commonly used compressed image file format
  • antonym: PNG - a raster image file format commonly used for web graphics
  • synonym: vector graphic - an image made from lines and shapes, not pixels
