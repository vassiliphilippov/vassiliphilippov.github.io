reconstruction

noun
memory recall by constructing from features
7.0 0.33/m
Pronunciation: / ÀåriÀê.k…ônÀàstr åk. É…ôn / (AmE), / ÀåriÀê.k…ônÀàstr åk. É…ôn / (BrE)
The process of recalling a memory by using stored abstract features to construct the original experience, rather than retrieving it exactly as it was.

Examples

Memory reconstruction
Eyewitness testimony is sometimes unreliable due to the reconstruction involved in memory recall.
In cognitive science, reconstruction refers to building a memory from fragments rather than retrieving an exact copy.
Psychologists believe that reconstruction explains why people often remember events differently over time.
