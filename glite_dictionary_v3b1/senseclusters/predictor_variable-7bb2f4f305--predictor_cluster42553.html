predictor variable

noun
variable used to predict another (statistics)
5.9 5.4
Pronunciation: /prɪ.ˈdɪk.tər ˈver.ɪ.ə.bəl/ (AmE), /prɪ.ˈdɪk.tə ˈveə.ri.ə.bəl/ (BrE)
a variable in statistical analysis that is used to forecast or estimate the value of another variable

Examples

Predictor variable in study
In the study, age was used as a predictor variable for income level.
Researchers tested several predictor variables to see which best explained the differences in test scores.
The strength of the predictor variable determines how well it can estimate the outcome.

Related Terms

  • antonym: dependent variable - a variable whose value depends on another variable in a statistical analysis
  • synonym: independent variable - a variable whose variation does not depend on another, often used similarly in statistics
