set up

verb
produce or cause to happen
1.0
Pronunciation: / sɛt ʌp / (AmE), / sɛt ʌp / (BrE)
to establish, start, or create something such as a business, organization, system, or to arrange for someone to be accommodated or placed somewhere

Examples

To set up a business
Her remarks set up a lively debate among the audience.
The researchers set up a controlled experiment to study the effects.
They set up a chain reaction by mixing the chemicals.
Her company set up the delegation in a luxury hotel during the conference.
They set up the new manager in an office overlooking the courtyard.
When he first arrived in the city, friends set up John at a guesthouse.
She helped set up the computer network for the office.
The city decided to set up a new task force to address housing issues.
They plan to set up several booths at the career fair.
