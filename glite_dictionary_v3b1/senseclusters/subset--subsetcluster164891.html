subset

noun
a set contained within another set
5.5 3.0
Pronunciation: /ˈsʌb.set/ /; / /ˈsʌb.sət/ (AmE), /ˈsʌb.set/ /; / /ˈsʌb.sət/ (BrE)
a set whose elements are all contained in another set

Examples

A subset of animals
All mammals are a subset of animals.
If B is a subset of A, then every element of B is also in A.
The set of even numbers is a subset of the set of integers.

Related Terms

  • antonym: superset - a set that contains all elements of a given set and possibly more
