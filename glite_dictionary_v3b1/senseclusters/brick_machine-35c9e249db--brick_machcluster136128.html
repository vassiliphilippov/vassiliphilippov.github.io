brick machine

noun
machine for making bricks
Pronunciation: / brɪk məˈʃin / (AmE), / brɪk məˈʃiːn / (BrE)
a machine used industrially for the manufacture of bricks from clay or other materials.

Examples

A brick machine factory
A modern brick machine can produce thousands of uniform bricks every hour.
Early versions of the brick machine required manual labor to operate.
The new factory installed an automatic brick machine to increase production efficiency.
