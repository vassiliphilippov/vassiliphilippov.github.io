div

noun
integer division operator
Pronunciation: / dɪv / (AmE), / dɪv / (BrE)
a programming or mathematical operator that returns the integer part of a division, discarding any remainder

Examples

Div operator
In many programming languages, the div operator is used to perform integer division.
When you use div with 7 and 2, the result is 3 because the remainder is discarded.
The function calculates the number of full groups using the div operator.

Related Terms

  • synonym: divide - to separate or be separated into parts; in mathematics, to perform division
  • antonym: mod - an operator that returns the remainder after division
