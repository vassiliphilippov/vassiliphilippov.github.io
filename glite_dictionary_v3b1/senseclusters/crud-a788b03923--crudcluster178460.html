CRUD

noun
create/read/update/delete operations (computing)
Pronunciation: / krʌd / (AmE), / krʌd / (BrE)
(computing) An acronym standing for Create, Read, Update, Delete; the basic operations used to create, retrieve, modify, and remove persistent data in databases or APIs.

Examples

CRUD operations
The API supports CRUD operations so developers can create, read, update, and delete records.
To build the app's backend, she documented all required CRUD endpoints for user profile management.
Junior engineers learned SQL quickly because each table needed clear CRUD functionality for everyday tasks.

Related Terms

  • synonym: data management operations - basic actions used to handle and maintain stored data, such as creating and updating records
  • synonym: database operations - standard operations performed on database records, including insertion, retrieval, modification, and deletion
