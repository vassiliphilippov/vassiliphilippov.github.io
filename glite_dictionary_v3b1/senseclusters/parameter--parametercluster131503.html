parameter

noun
value or reference passed to a function (computing)
Pronunciation: / pəˈræm.ə.tɚ / (AmE), / pəˈræm.ɪ.tə / (BrE)
(Computer science) a value or reference that is passed to a function, procedure, subroutine, command, or program to customize its operation.

Examples

Parameter in a function
In this programming language, you must define the type of each parameter in your function.
Optional parameters allow for more flexible and reusable code.
The function failed because it received an invalid parameter from the user input.
