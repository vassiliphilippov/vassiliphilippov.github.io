facet

noun
filterable attribute in search
Pronunciation: / ˈfæsət / (AmE), / ˈfæsɪt / (BrE)
(in information retrieval and user interfaces) a property or attribute of items (for example brand, price, or color) used to classify, group, and filter search results; one selectable category in faceted navigation.

Examples

Search facet
Users can refine search results by selecting a facet such as brand or price range.
The online store added a new facet for material, improving product filtering and discovery.
Some sites show multiple facets so customers can combine filters like size and color.

Related Terms

  • synonym: attribute - a characteristic or property used to describe an item
  • synonym: filter - a criterion or control used to narrow search results
