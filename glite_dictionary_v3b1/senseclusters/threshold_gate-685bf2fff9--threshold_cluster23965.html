threshold gate

noun
logic gate with output based on threshold sum
7.7
Pronunciation: /ˈθreʃ.hoʊld geɪt/ (AmE), /ˈθreʃ.həʊld geɪt/ (BrE)
a logic circuit element that produces an output only when the weighted sum of its inputs reaches or exceeds a certain threshold value

Examples

A threshold gate circuit
In digital circuits, a threshold gate can perform more complex decisions than simple AND or OR gates.
Researchers designed a new threshold gate that improves processing speed in neural networks.
The threshold gate outputs a signal only if enough inputs are active at the same time.
