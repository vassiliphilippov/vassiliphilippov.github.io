virtual storage

noun
simulated computer memory using disk storage
Pronunciation: / ˈvɝː.tʃu.əl ˈstɔːr.ɪdʒ / (AmE), / ˈvɜː.tʃu.əl ˈstɔː.rɪdʒ / (BrE)
(in computing) memory created by using disk space to simulate additional random-access memory, allowing programs to use more memory than is physically available.

Examples

Virtual storage for memory
Running too many programs at once can cause virtual storage to slow down your computer.
Thanks to virtual storage, multitasking is possible even on systems with limited memory.
The computer uses virtual storage to handle large applications that exceed its physical RAM.

Related Terms

  • synonym: virtual memory - (computing) memory management technique using hardware and software to simulate larger memory
