ongoing evaluation

noun
continuous evaluation during project implementation

An evaluation carried out continuously during the implementation phase of a project or program to monitor progress and inform decision-making.

Examples

The team instituted an ongoing evaluation process to ensure issues could be addressed as the project progressed.
