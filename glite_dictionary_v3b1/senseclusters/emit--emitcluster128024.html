emit

verb
trigger or send an event (computing)
6.0
Pronunciation: / ɪˈmɪt / (AmE), / ɪˈmɪt / (BrE)
(in computing, especially event-driven programming) to trigger or send out an event or signal so that other parts of a program can respond.

Examples

To emit a signal
The event engine can emit a notification when a background job completes.
When the server receives complete data, it emits a success event to subscribed clients.
The debugger emitted a trace event showing the variable values at the breakpoint.

Related Terms

  • synonym: dispatch - to send out an event or message to handlers or listeners
  • synonym: trigger - to cause an action or event to occur in a system
