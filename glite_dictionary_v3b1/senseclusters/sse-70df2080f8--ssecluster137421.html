SSE

noun
Server-Sent Events (web streaming)
8.1
Pronunciation: / cc5bs 5bs c8i / (AmE), / cc5bs 5bs c8id0 / (BrE)
Abbreviation of Server-Sent Events: a web technology that sends automatic, one-way real-time updates from a server to a client over an HTTP connection.

Examples

SSE connection
The app uses SSE to receive live updates from the server.
We implemented SSE so the news page pushes breaking headlines to users instantly.
Unlike WebSocket, SSE uses a simple HTTP connection for one-way event streaming to clients.

Related Terms

  • synonym: Server-Sent Events - the full name of the web API for one-way event streaming from server to client over HTTP
