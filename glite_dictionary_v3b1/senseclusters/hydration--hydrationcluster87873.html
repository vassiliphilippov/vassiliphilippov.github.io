hydration

noun
making server-rendered HTML interactive
Pronunciation: / haɪˈdreɪʃən / (AmE), / haɪˈdreɪʃən / (BrE)
(in web development) the process of turning server-rendered HTML into a fully interactive client-side page by attaching JavaScript state, event handlers, and behavior to the existing DOM.

Examples

Page hydration process
During the initial load, the framework performs hydration to activate interactivity on the page.
After server-side rendering, the app waits for scripts to run before hydration completes and events work.
Debugging was hard because the hydration step failed, leaving components rendered but not interactive for users.

Related Terms

  • antonym: client-side rendering - rendering and building the page entirely in the browser without first sending fully rendered HTML from the server
  • synonym: rehydration - an alternative name used for the same process of attaching client-side behavior to server-rendered markup
