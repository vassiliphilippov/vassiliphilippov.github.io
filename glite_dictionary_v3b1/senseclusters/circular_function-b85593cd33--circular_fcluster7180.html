circular function

noun
trigonometric function of an angle
7.2
Pronunciation: / ˈsɝːkjələr ˈfʌŋkʃən / (AmE), / ˈsɜːkjʊlə ˈfʌŋkʃən / (BrE)
a trigonometric function of an angle, defined as a ratio of the lengths of sides in a right-angled triangle containing that angle

Examples

A circular function graph
Students learn how to graph each circular function when studying circular motion.
The sine, cosine, and tangent are each a type of circular function used in trigonometry.
Understanding a circular function is essential for solving many problems involving right triangles.

Related Terms
