wave function

noun
quantum state function
Pronunciation: / ˈweɪv ˌfʌŋkʃən / (AmE), / ˈweɪv ˌfʌŋkʃən / (BrE)
a mathematical function that describes the quantum state and behavior of a particle or system in quantum mechanics

Examples

Quantum wave function
The wave function allows scientists to calculate the probability of finding a particle in a certain location.
In quantum mechanics, the wave function contains all the information about a system's possible states.
When measured, the wave function collapses to a specific value, revealing the particle's position.

Related Terms

  • synonym: probability amplitude - a complex number describing the likelihood of a quantum event
  • synonym: quantum state - the complete description of a system in quantum mechanics
