utility

noun
software tool for system tasks
Pronunciation: / juˈtɪləti / (AmE), / juːˈtɪlɪti / (BrE)
(computing) a computer program designed to perform a specific task, usually to manage system resources or provide general support for the operating system or applications.

Examples

Computer utility
After installing the new security utility, the computer ran much more efficiently.
System administrators often rely on utility programs to back up or restore data.
This utility scans your hard drive and removes temporary files automatically.

Related Terms

  • antonym: application - a program designed for a specific purpose or use by the user
