Teredo

noun
IPv6-over-UDP tunneling protocol
9.8
Pronunciation: /tə.ˈriː.doʊ/ /; / /ˈte.rɪ.doʊ/ (AmE), /tə.ˈriː.dəʊ/ /; / /ˈte.rɪ.dəʊ/ (BrE)
A networking protocol that provides IPv6 connectivity by encapsulating IPv6 packets inside UDP/IPv4 packets so they can traverse NATs and reach IPv6 endpoints.

Examples

Teredo protocol
Windows automatically configures Teredo to provide IPv6 connectivity when native IPv6 is unavailable.
When their router blocked IPv6, the system fell back to Teredo for basic internet access.
Many network tools can detect active Teredo tunnels and report them to the administrator.

Related Terms

  • synonym: IPv6 transition technology - a class of methods (including Teredo) used to provide IPv6 connectivity over IPv4 networks
  • synonym: Teredo tunneling - the same tunneling method that encapsulates IPv6 in UDP to traverse NATs
