reactivity

noun
automatic change propagation (computing)
6.8
Pronunciation: /riː.æk.ˈtɪ.və.tiː/ /; / /rɪ.æk.ˈtɪ.və.tiː/ (AmE), /riː.æk.ˈtɪ.və.tiː/ /; / /rɪ.æk.ˈtɪ.və.tiː/ (BrE)
(in computing) the property of a system, library, or framework by which changes to data automatically propagate to dependent values, components, or the user interface without manual intervention.

Examples

UI reactivity
The framework's reactivity automatically updates the UI when the data changes.
During testing, we measured the library's reactivity under heavy load to ensure stability.
Good reactivity in the dashboard meant charts updated instantly after the server pushed new values.

Related Terms

  • synonym: data binding - the technique of synchronizing data between a model and its representation in the UI
  • synonym: reactive programming - a programming paradigm centred on data flows and the propagation of change
