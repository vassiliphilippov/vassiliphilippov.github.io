focused

adjective
UI: having keyboard/input focus
4.3 0.00/m
Pronunciation: / \u02c8fo\u028ak\u0259st / (AmE), / \u02c8f\u0259\u028ak\u0259st / (BrE)
(computing) (of a user-interface element) having keyboard or input focus and therefore receiving input or interaction events.

Examples

Focused text box
When you click the textbox, the focused field displays a blinking cursor and accepts keyboard input.
Screen readers announce which element is focused so users know where their keystrokes will go.
You can style a focused button in CSS using the :focus selector to show it is active.

Related Terms

  • synonym: active - in computing, currently selected or accepting input (often used interchangeably in UI contexts)
