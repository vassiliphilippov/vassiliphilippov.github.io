work table

noun
table for a specific task or work
Pronunciation: / wɝk ˈteɪbəl / (AmE), / wɜːk ˈteɪbəl / (BrE)
a table specially designed for carrying out a particular task or type of work

Examples

A work table for sewing
She keeps her sewing machine and supplies on her work table in the corner of the room.
The carpentry classroom had a work table for each group of students.
The designer spread fabric across the work table to cut the new dress pattern.

Related Terms

  • synonym: workbench - a strong table for working on with tools or machinery, especially in a workshop
