compile

verb
convert source code into executable machine code
3.7 5.3
Pronunciation: / kəmˈpaɪl / (AmE), / kəmˈpaɪl / (BrE)
(computing) to use a computer program to translate source code written in a particular programming language into machine code that can be executed by a computer.

Examples

To compile code
After editing the source file, she tried to compile it again to check for errors.
The software failed to compile due to a missing semicolon in the code.
You need to compile the program before running it on your computer.
