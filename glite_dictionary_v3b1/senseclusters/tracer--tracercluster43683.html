tracer

noun
software tool for recording program execution
7.4
Pronunciation: / ˈtreɪsər / (AmE), / ˈtreɪsə / (BrE)
(computing) a software tool or component that records the execution path, events, or data flows of a program (often across services) to help debugging, profiling, or performance analysis.

Examples

Software tracer
The developers used a tracer to monitor API calls and memory allocations.
Distributed systems commonly use tracers to collect timing and causal information for requests.
A lightweight tracer helped them identify the slow database call causing the performance regression.

Related Terms

  • synonym: debugger - a program used to find and fix errors in other programs; often used alongside tracers for diagnosing faults
  • synonym: profiler - a tool that measures where a program spends time or resources; related to tracers for performance analysis
