vectorization

noun
matrix to column vector
8.4
Pronunciation: /vek.tə.raɪ.ˈzeɪ.ʃən/ (AmE), /vek.tə.raɪ.ˈzeɪ.ʃən/ (BrE)
the process of converting a matrix into a single column vector by stacking its columns, often using the vec operator in mathematics.

Examples

Matrix vectorization
The algorithm uses vectorization to simplify complex matrix calculations.
By applying vectorization, the data was transformed into a single column for further analysis.
Understanding vectorization is essential for working with advanced mathematical models.
