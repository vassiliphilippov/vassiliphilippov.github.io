personal information manager

noun
software for managing personal data
Pronunciation: / ˈpɝsənl ˌɪnfərˈmeɪʃən ˈmænɪdʒɚ / (AmE), / ˈpɜːsənl ˌɪnfəˈmeɪʃən ˈmænɪdʒə / (BrE)
A software application designed to help individuals record, organize, and manage personal data such as contacts, calendars, and tasks.

Examples

A personal information manager software
Many professionals rely on a personal information manager to coordinate their schedules and to-do lists.
She uses a personal information manager to keep track of her meetings and contacts.
The personal information manager integrates email, calendar, and task management into one convenient tool.
