500

noun
HTTP 500 server error
5.6
Pronunciation: /faɪv ˈhʌn.drəd/ (AmE), /faɪv ˈhʌn.drəd/ (BrE)
(computing) an HTTP response code meaning 'Internal Server Error', used to refer to a server-side failure or the error message returned by a web server.

Examples

500 error message
The website returned a 500 error after the request failed during the deployment.
Developers checked the server logs and found repeated 500 responses during the spike in traffic.
When users reported pages timing out, the dashboard showed several 500 errors from the backend.

Related Terms

  • synonym: HTTP status code - a three-digit code in an HTTP response that indicates the result of a client's request
  • synonym: Internal Server Error - an HTTP response indicating the server encountered an unexpected condition and could not fulfill the request
