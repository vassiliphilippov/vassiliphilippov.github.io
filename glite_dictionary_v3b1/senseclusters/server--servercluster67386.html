server

noun
utensil for serving food or drink
9.9
Pronunciation: / ˈsɝːvər / (AmE), / ˈsɜːvə / (BrE)
a flat or shaped utensil used for serving food or drink, such as a cake server or pie server

Examples

Cake server
Please pass me the server so I can dish out the lasagna.
She used a server to neatly distribute slices of cake to the guests.
The silver server matched the rest of the cutlery on the table.
