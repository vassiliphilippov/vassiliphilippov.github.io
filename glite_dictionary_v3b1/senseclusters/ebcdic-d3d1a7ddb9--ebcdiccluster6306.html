EBCDIC

adjective
using EBCDIC encoding
Pronunciation: / ˈɛb sɪ dɪk / (AmE), / ˈɛb sɪ dɪk / (BrE)
encoded or formatted using the EBCDIC character encoding system, typically used in older computer systems.

Examples

EBCDIC format
The mainframe stores all its data in EBCDIC format, requiring special conversion tools.
Some legacy printers only support EBCDIC encoded files for document processing.
When migrating to modern systems, EBCDIC data must be converted to a more common encoding.

Related Terms

  • antonym: ASCII - a widely used character encoding standard for electronic communication
