value

noun
duration length of a musical note
Pronunciation: / ˈvæljuː / (AmE), / ˈvæljuː / (BrE)
(music) The relative length or duration of a musical note, such as a whole note, half note, or quarter note.

Examples

Note value
Each type of note has a different value that determines how long it is held.
Musicians must pay attention to the value of the notes to play the rhythm correctly.
The composer indicated the value of each note with specific symbols.
