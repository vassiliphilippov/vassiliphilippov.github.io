uncoded

adjective
not assigned a code
Pronunciation: / ʌnˈkoʊdɪd / (AmE), / ʌnˈkəʊdɪd / (BrE)
not converted into a code or not assigned a code for classification or processing

Examples

Uncoded data files
The survey responses were still uncoded at the end of the research project.
Sensitive information should never be left uncoded in a public database.
Because the files were uncoded, the software could not recognize them.

Related Terms

  • antonym: coded - converted into a code or assigned a code
  • synonym: unclassified - not assigned to a category or class
