plugin

adjective
extending software
Pronunciation: / ˈplʌɡ ɪn / (AmE), / ˈplʌɡ ɪn / (BrE)
designed to be installed into and extend the features of a larger program or system

Examples

Plugin module
The new plugin module allows users to add extra features to the main application.
We prefer a plugin approach so that updates can be made without changing the core system.
Many modern programs support plugin components for greater flexibility and customization.

Related Terms

  • synonym: modular - made up of separate parts that can be connected or combined
  • antonym: standalone - able to function independently without additional components
