REPL

noun
interactive coding environment
8.4
Pronunciation: /ˈrep.əl/ /; / /ɑːr iː piː el/ (AmE), /ˈrep.əl/ /; / /ɑː iː piː el/ (BrE)

Many developers use the REPL to experiment with new programming ideas.
The REPL makes it easy to debug small pieces of code interactively.
