REPL

noun
interactive coding environment
Pronunciation: / ˈrɛpəl / (AmE), / ˈrɛpəl / (BrE)
a computer environment that allows users to enter commands, see results, and interact with code in real time

Examples

A REPL environment
Python provides a REPL where you can test code and see results immediately.
Many developers use the REPL to experiment with new programming ideas.
The REPL makes it easy to debug small pieces of code interactively.

Related Terms

  • synonym: command line - a text-based interface for entering commands
  • synonym: interpreter - a program that executes code line by line
