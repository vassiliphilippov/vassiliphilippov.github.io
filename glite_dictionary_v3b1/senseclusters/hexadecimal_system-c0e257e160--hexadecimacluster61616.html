hexadecimal system

noun
numerical notation using base sixteen
Pronunciation: / ˌhɛksəˈdɛsəməl ˈsɪstəm / (AmE), / ˌhɛksəˈdɛsɪməl ˈsɪstəm / (BrE)
a system of numerical notation that uses base sixteen, employing the digits 0–9 and the letters A–F to represent values

Examples

The hexadecimal system notation
Learning the hexadecimal system is essential for understanding how computers process data.
Many computer programmers use the hexadecimal system to represent memory addresses.
The color code #FF5733 is written in the hexadecimal system.
