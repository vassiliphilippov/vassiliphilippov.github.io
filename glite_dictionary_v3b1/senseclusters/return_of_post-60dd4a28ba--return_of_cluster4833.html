return of post

noun
immediate reply by mail
Pronunciation: / rɪˈtɝn əv poʊst / (AmE), / rɪˈtɜːn əv pəʊst / (BrE)
the next available delivery or collection of mail, used to mean an immediate reply by post

Examples

By return of post delivery
Please let us know your decision by return of post as time is short.
The letter requested a reply by return of post to confirm attendance.
We would appreciate it if you could respond by return of post.
