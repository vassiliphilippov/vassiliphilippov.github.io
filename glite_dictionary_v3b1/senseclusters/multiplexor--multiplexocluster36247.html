multiplexor

noun
signal-combining device
8.1
Pronunciation: / ˈmʌltɪˌplɛksər / (AmE), / ˈmʌltɪˌplɛksə / (BrE)
an electronic device that manages several input signals and sends them through a single output line

Examples

A multiplexor device
The multiplexor in the network allows several computers to share the same communication line.
Engineers installed a new multiplexor to improve the efficiency of data transmission.
Without a multiplexor, each signal would require its own dedicated channel.
The technician installed a multiplexor to manage the data flow between different sensors.
A multiplexor is essential in digital circuits for selecting which input signal to process.
By using a multiplexor, the system can efficiently handle multiple data sources.

Related Terms

  • antonym: demultiplexer - a device that separates a single input signal into multiple output signals
  • synonym: multiplexer - a device that selects one of several input signals and forwards it to a single output
