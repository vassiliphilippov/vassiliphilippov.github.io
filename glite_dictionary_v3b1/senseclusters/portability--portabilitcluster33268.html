portability

noun
ability to run on multiple platforms
4.3
Pronunciation: / ccp54rt59c8b6al59ti / (AmE), / ccp54d0t59c8b6al59ti / (BrE)
the ability of software, code, or digital content to be moved to and run on different computing environments or platforms with little or no modification.

Examples

Software portability
The project's portability across Windows, macOS, and Linux was a major selling point.
Engineers improved the library's portability by removing platform-specific system calls and dependencies.
High portability allows software to be deployed in containers, cloud services, or on-premises servers.

Related Terms

  • synonym: port - to adapt software so it can run on a different computing platform
