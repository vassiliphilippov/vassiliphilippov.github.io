character set

noun
defined collection of characters for writing/printing
Pronunciation: / ˈkærɪktɚ ˌsɛt / (AmE), / ˈkærɪktə ˌsɛt / (BrE)
A defined collection of characters used together for writing, printing, or processing text, often in a specified order.

Examples

A character set order
Selecting the correct character set is essential for displaying international text properly.
The ASCII character set includes both uppercase and lowercase letters, digits, and special symbols.
The printer was unable to recognize the new character set in the document.
