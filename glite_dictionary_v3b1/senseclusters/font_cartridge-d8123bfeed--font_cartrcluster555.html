font cartridge

noun
removable printer module with fonts
6.6 6.3
Pronunciation: /fɑːnt ˈkɑːr.trɪdʒ/ /; / /fɑːnt ˈkɑːr.trədʒ/ /; / /fɔːnt ˈkɑːr.trɪdʒ/ (AmE), /fɒnt ˈkɑː.trɪdʒ/ /; / /fɒnt ˈkɑː.trədʒ/ (BrE)
a removable hardware module containing one or more fonts, designed to be inserted into a computer printer to expand its available typefaces

Examples

A font cartridge for printer
Early printers often came with only one style, unless you bought a font cartridge to expand the options.
He installed a new font cartridge to get additional fonts for his reports.
The old laser printer requires a font cartridge to print special typefaces.

Related Terms

  • synonym: font - a set of type of one particular face and size
  • synonym: printer cartridge - removable module used in a printer, typically for ink or toner
