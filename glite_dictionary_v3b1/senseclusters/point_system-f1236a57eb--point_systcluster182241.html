point system

noun
system for standardizing type sizes in points
5.7
Pronunciation: / pɔɪnt ˈsɪstəm / (AmE), / pɔɪnt ˈsɪstəm / (BrE)
A system for classifying the sizes of type or print using multiples of a standardized unit called the point.

Examples

Point system for type size
Before the point system, type sizes often varied from one foundry to another.
Many modern publishers use the point system to measure both text and spacing.
The point system ensures that fonts are produced in consistent and precise sizes.
