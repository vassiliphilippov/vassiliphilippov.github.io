failover

verb
to switch to backup
Pronunciation: / ˈfeɪlˌoʊvər / (AmE), / ˈfeɪlˌəʊvə / (BrE)
to switch automatically from a primary system to a backup system when the primary fails

Examples

To failover to backup
If the main server crashes, the system will failover to the backup server within seconds.
Our IT department tested how quickly the network could failover during a simulated outage.
Automatic failover ensures that users experience minimal downtime during technical problems.

Related Terms

  • synonym: recover - to return to a normal state after a failure
  • synonym: switch over - to change from one system or process to another
