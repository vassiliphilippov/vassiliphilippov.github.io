unindexed

adjective
without an index
Pronunciation: / ʌnˈɪndɛkst / (AmE), / ʌnˈɪndɛkst / (BrE)
not included in or assigned to an index, so it cannot be found or accessed directly (especially in computing, databases, or documents)

Examples

Unindexed database
The database remained unindexed, making searches much slower than necessary.
If a list is unindexed, you must check each item one by one to find what you need.
Some files are unindexed and cannot be found quickly using the search function.
The unindexed documents could not be found using the search function.
Researchers struggled to access the unindexed articles in the old database.
Many files remained unindexed, making the archive difficult to organize.

Related Terms

  • antonym: indexed - assigned an index number or position for direct retrieval
  • synonym: unlisted - not included in a list or directory
