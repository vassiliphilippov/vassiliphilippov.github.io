trace element

noun
element needed in very small amounts
3.5 3.6
Pronunciation: /treɪs ˈel.ə.mənt/ (AmE), /treɪs ˈel.ɪ.mənt/ (BrE)
a chemical element present in very small amounts in a living organism but essential for normal physiological processes

Examples

An essential trace element
A deficiency of any essential trace element can lead to health problems.
Iron is a trace element necessary for the production of hemoglobin.
Selenium is a trace element found naturally in many foods.
