regression line

noun
line fitted to data in regression analysis
7.8 4.9
Pronunciation: /rɪ.ˈgreʃ.ən laɪn/ /; / /rə.ˈgreʃ.ən laɪn/ (AmE), /rɪ.ˈgreʃ.ən laɪn/ /; / /rə.ˈgreʃ.ən laɪn/ (BrE)
a straight or smoothly curved line fitted to a set of paired data in regression analysis, showing the relationship between variables

Examples

The regression line slope
A positive slope in the regression line suggests that increases in one variable are associated with increases in the other.
After running the analysis, he drew a regression line through the scatter of data points.
The researcher plotted the regression line to visualize the link between study time and test scores.
