program

noun
set of instructions for a computer
Pronunciation: / ˈproʊɡræm / (AmE), / ˈprəʊɡræm / (BrE)
A sequence of coded instructions that a computer can interpret and execute in order to perform a specific task.

Examples

A computer program
She downloaded a new program to help her edit photos on her laptop.
The program crashed unexpectedly and lost all my work.
Writing a simple program can help beginners understand the basics of coding.
He spent the afternoon writing a programme to sort large databases.
The company released an updated programme that fixes several security issues.
Without the correct programme, the computer cannot process the data.

Related Terms

  • synonym: program - set of coded instructions that a computer can execute (American spelling)
  • synonym: software - programs and other operating information used by a computer
