rehydrate

verb
to restore program state
Pronunciation: / ˌriːˈhaɪdreɪt / (AmE), / ˌriːˈhaɪdreɪt / (BrE)
to restore a computer program's or data structure's state by loading saved or serialized data into memory

Examples

To rehydrate data
The web application will rehydrate its state from the server when the user refreshes the page.
After a crash, the system can rehydrate user sessions using data stored in the database.
Developers often rehydrate components to ensure a seamless user experience after loading saved data.

Related Terms

  • synonym: deserialize - to convert saved data back into a usable program structure
  • antonym: initialize - to set up or prepare a program or data structure for use
