canvas

noun
HTML5 drawing surface
Pronunciation: / ˈkænvəs / (AmE), / ˈkæn.vəs / (BrE)
(computing, web) a blank drawing surface provided by the HTML5 <canvas> element, used for rendering graphics programmatically (usually with JavaScript).

Examples

HTML canvas
The webpage uses a canvas element to render interactive graphics with JavaScript.
He drew the game's background directly onto the canvas every time the level loaded.
Before saving, she exported the canvas content as a PNG for use in the article.

Related Terms

  • synonym: HTML5 canvas element - the HTML element (<canvas>) that provides a surface for drawing graphics in web pages
  • synonym: drawing surface - a surface on which images or graphics are drawn or rendered
