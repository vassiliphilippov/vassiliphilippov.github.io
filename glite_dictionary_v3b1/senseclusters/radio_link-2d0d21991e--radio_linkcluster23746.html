radio link

noun
two-way communication channel via radio waves
5.1 3.3
Pronunciation: / ˈreɪdioʊ lɪŋk / (AmE), / ˈreɪdiəʊ lɪŋk / (BrE)
a two-way communication channel established using radio waves, usually as part of a larger telecommunication system

Examples

A radio link connection
A radio link can provide connectivity where laying cables is impractical or too expensive.
Engineers established a secure radio link between the two buildings for critical data transfer.
The new rural broadband service is delivered using a radio link to remote villages.
