(in computing) a message object used to request an action or transfer data between components of a software system (for example, Android intents used to start activities or services).

Examples

Send an intent
Developers send an intent to start the camera app with initial data.
The activity received the intent and opened the correct view with the supplied extras.
You can create implicit intents to request actions from other apps without specifying a package.

Related Terms

  • synonym: message - an object or packet conveying data or a request between software components
