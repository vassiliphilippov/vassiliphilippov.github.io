syntax checker

noun
program that checks for syntax errors
5.0
Pronunciation: / ˈsɪn.tæks ˌtʃɛk.ər / (AmE), / ˈsɪn.tæks ˌtʃɛk.ə / (BrE)
a computer program that examines text for syntax errors according to the rules of a language

Examples

A syntax checker program
A good syntax checker can save writers from making grammatical mistakes in documents.
Some word processors include a built-in syntax checker for multiple languages.
The syntax checker flagged several errors in my code before I could compile it.
