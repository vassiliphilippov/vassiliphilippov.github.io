token

verb
to split into tokens
7.5
Pronunciation: / ˈtoʊkən / (AmE), / ˈtəʊkən / (BrE)
to divide text or data into discrete units called tokens, often for processing or analysis

Examples

To token a sentence
Many natural language processing tools token sentences before further analysis.
The script will token the data to make it easier to search for keywords.
Before building the index, the program must token all the input documents.

  • synonym: segment - to divide something into separate parts or sections
