master key

noun
cryptographic key that unlocks multiple resources
4.2
Pronunciation: /ˈmæ.stər kiː/ (AmE), /ˈmɑː.stə kiː/ (BrE)
(in computing and cryptography) a single cryptographic key or central credential that can be used to decrypt, sign, or otherwise access multiple protected resources, services, or data sets.

Examples

A master key for data
The cloud provider assigns a master key that can decrypt all customer data across services.
After the breach, the security team rotated the master key to protect user records from further access.
Developers store the encrypted secrets separately and limit access to the master key to a few trusted admins.

Related Terms

  • synonym: encryption key - a key used to encode or decode data so it cannot be read without authorization
  • synonym: root key - a privileged key in a hierarchical key system that can derive or control other keys
