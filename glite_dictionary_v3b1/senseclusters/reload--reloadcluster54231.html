reload

verb
refresh content (web/app)
3.2 0.71/m

to fetch or load updated content again in a software context, for example by reloading a web page or app view so it shows the latest information.

Examples

To reload a page
When the chat stopped updating, she clicked reload to see the latest messages.
He reloaded the browser after installing the plugin so the new settings would appear.
The webpage kept reloading every few minutes, making it hard to finish the article.

Synonyms

  • synonym: update - to make current by adding the most recent information or data
