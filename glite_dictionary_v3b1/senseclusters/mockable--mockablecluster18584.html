mockable

adjective
able to be mocked
Pronunciation: / ˈmɑːkəbəl / (AmE), / ˈmɒkəbəl / (BrE)
able to be replaced, simulated, or emulated by a mock object or test double during unit testing in software development.

Examples

Mockable API
The API was designed to be mockable, making it easier to write unit tests.
Developers prefer mockable services so they can isolate components during testing.
If a class is not mockable, it can be difficult to test its behavior independently.

Related Terms

  • synonym: testable - able to be tested or examined
  • antonym: unmockable - not able to be replaced or simulated by a mock object
