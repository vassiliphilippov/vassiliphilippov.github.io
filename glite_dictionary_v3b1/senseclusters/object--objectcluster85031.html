(Grammar) a noun, noun phrase, or pronoun that receives the action of a verb or shows the result of an action.

Examples

Object of the verb
In the sentence 'She reads books,' the object is 'books.'
Some verbs require an object, while others do not.
The grammar lesson focused on identifying the object in each example.
