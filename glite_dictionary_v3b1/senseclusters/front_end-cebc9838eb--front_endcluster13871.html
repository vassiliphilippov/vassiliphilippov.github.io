front end

noun
user interface part
Pronunciation: / ˈfrʌnt ɛnd / (AmE), / ˈfrʌnt ɛnd / (BrE)
the part of a website or application that users see and interact with directly

Examples

Front end of a website
The front end of the app was redesigned to improve user experience and accessibility.
Developers often specialize in either the front end or the back end of web development.
She created a visually appealing front end for the company's new online store.
