front matter

noun
metadata block at top of a file
7.0
Pronunciation: /frʌnt ˈmæt.ər/ (AmE), /frʌnt ˈmæt.ə/ (BrE)
a block of metadata placed at the start of a digital document or file (often in YAML or TOML) that defines properties such as title, date, layout, or tags for processing by static-site generators and publishing tools.

Examples

YAML front matter
In Jekyll, the front matter at the top of a post sets layout, title, and date.
Before converting, check that the document's front matter uses valid YAML keys for metadata extraction.
The static site generator failed because the page's front matter was missing the required permalink field.

Related Terms

  • synonym: YAML front matter - a common format of front matter written in YAML used by many static-site generators
  • synonym: metadata - data that provides information about other data, such as title, author, or date
