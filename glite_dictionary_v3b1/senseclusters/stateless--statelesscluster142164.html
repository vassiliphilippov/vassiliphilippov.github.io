stateless

adjective
not retaining state
6.0
Pronunciation: / ˈsteɪtləs / (AmE), / ˈsteɪtləs / (BrE)
not retaining information or memory about previous interactions, conditions, or states

Examples

Stateless protocol
A stateless protocol does not keep track of previous communication between computers.
The web server is designed to be stateless, making it easier to scale and manage.
Because the application is stateless, each request must include all necessary information.

Related Terms

  • antonym: stateful - retaining information or memory about previous interactions or states
