loop

noun
one run of a repeated set of instructions
Pronunciation: / luːp / (AmE), / luːp / (BrE)
a sequence of instructions in a computer program that is repeated until a condition is met

Examples

A loop in code
Each loop in the program checks for new data and updates the results.
He noticed a bug that only appeared during the second loop of the process.
The instructions inside the loop executed faster after optimization.
An infinite loop can cause the application to become unresponsive.
The program enters a loop until the user presses the exit key.
We used a for-loop to process each item in the array.
