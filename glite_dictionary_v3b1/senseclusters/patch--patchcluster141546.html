patch

noun
software update for fixing bugs
Pronunciation: / pætʃ / (AmE), / pætʃ / (BrE)
a piece of code or set of instructions added to a computer program to fix a problem or improve its functioning

Examples

Software patch
After installing the latest patch, the game ran much more smoothly.
Developers issued a new patch that corrected several minor bugs.
The company released a patch to fix the security vulnerability in their software.
