host

noun
a computer providing resources on a network
3.7 4.2
Pronunciation: /hoʊst/ (AmE), /həʊst/ (BrE)
(computing) a computer on a network that provides other computers (clients) with access to files, printers, or other shared resources.

Examples

A host computer
Each client must connect to a central host to access important documents.
The main host manages all shared files and printers for the office network.
Upgrading the host improved performance for everyone using the network.

Related Terms

  • synonym: server - a computer or program that provides services to other computers on a network
