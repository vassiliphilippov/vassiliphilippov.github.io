if it had teeth

x
would be dangerous if it could harm
Pronunciation: / ɪf ɪt hæd tiθ / (AmE), / ɪf ɪt hæd tiːθ / (BrE)
used to suggest that something would be dangerous, harmful, or painful if it were capable of causing physical harm

Examples

If it had teeth it hurt
Her criticism was so sharp, if it had teeth, it'd draw blood.
His wit is so biting that if it had teeth, no one would be safe.
The remark was harsh; if it had teeth, it would have bitten her.
