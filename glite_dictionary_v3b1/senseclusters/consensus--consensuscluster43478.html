consensus

noun
agreement among computers in a distributed system
6.8 0.38/m
Pronunciation: / k59nc8s5bns59s / (AmE), / k59nc8s5bns59s / (BrE)
(in computing, especially distributed systems) agreement among networked computers or processes about the state of shared data, reached by running a consensus protocol that tolerates failures and ensures consistency.

Examples

Consensus protocol
The distributed ledger must reach consensus among all nodes before finalizing a block.
After a node failed, the cluster used Raft to reestablish consensus and resume accepting writes.
Designing fault-tolerant systems requires careful testing of the protocol that achieves consensus reliably.
