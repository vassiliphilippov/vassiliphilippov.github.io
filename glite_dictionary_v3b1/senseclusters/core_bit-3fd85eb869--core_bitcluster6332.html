core bit

noun
hollow cutting tool for drilling cores
Pronunciation: / kɔr bɪt / (AmE), / kɔː bɪt / (BrE)
a hollow cylindrical cutting tool used in a core drill to extract a core sample from material such as rock or concrete

Examples

A core bit tool
For geological surveys, a diamond-tipped core bit is often used to obtain clean samples.

The engineer replaced the worn core bit before continuing to drill into the rock.
