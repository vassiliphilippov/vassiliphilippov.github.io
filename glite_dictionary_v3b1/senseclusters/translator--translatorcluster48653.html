translator

noun
program converting programming languages
Pronunciation: / trænzˈleɪtər / (AmE), / trænzˈleɪtə / (BrE)
a computer program that converts code written in one programming language into another, such as a compiler or interpreter

Examples

Code translator
Many modern IDEs include a built-in translator for multiple programming languages.
The translator converts the source code into a form that the computer can execute.
Using an automatic translator can help migrate legacy code to a new software platform.

Related Terms

  • synonym: compiler - a program that translates code from a high-level programming language to machine code
  • synonym: interpreter - a program that executes instructions written in a programming language directly
