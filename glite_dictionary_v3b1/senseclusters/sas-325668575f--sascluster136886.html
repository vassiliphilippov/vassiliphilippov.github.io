SAS

noun
high-speed storage interface
7.9
Pronunciation: /es eɪ es/ /; / /sæs/ (AmE), /es eɪ es/ /; / /sæs/ (BrE)
(computing) Serial Attached SCSI, a high-speed point-to-point data storage interface used to connect and transfer data between storage devices and servers.

Examples

SAS hard drives
The server array uses SAS drives for faster I/O during heavy database operations.
Enterprise storage systems prefer SAS connections because they support high throughput and reliability.
After upgrading to SAS, the backup server completed large file transfers far more quickly overnight.

Related Terms

  • synonym: SCSI - Small Computer System Interface; a family of standards for connecting and transferring data between computers and peripheral devices; SAS is the serial version of SCSI.
