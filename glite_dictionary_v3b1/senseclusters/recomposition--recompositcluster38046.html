recomposition

noun
layout recalculation
Pronunciation: / ˌriːˌkɑːmpəˈzɪʃən / (AmE), / ˌriːˌkɒmpəˈzɪʃən / (BrE)
the process of recalculating or re-rendering a document or user interface layout after content or state changes

Examples

UI recomposition
When the user resized the window, the application performed a recomposition to adjust the layout.
Frequent recomposition can slow down the performance of a web page if not managed properly.
Modern frameworks optimize recomposition to ensure smooth and responsive user interfaces.

Related Terms

  • synonym: refresh - to update the display or content of a screen or page
  • synonym: rendering - the process of generating a visual display from data
