smart set

noun
group of connected smart devices
Pronunciation: / smɑrt sɛt / (AmE), / smɑːt sɛt / (BrE)
A group or collection of interconnected 'smart' electronic devices (for example speakers, lights, sensors) intended to be controlled together as part of a single system.

Examples

A smart set of devices
The smart set of speakers, lights, and sensors automates the living room.
Before guests arrived, she linked the smart set to a single app for easy control.
The installer recommended a basic smart set including a hub, two bulbs, and a door sensor.
