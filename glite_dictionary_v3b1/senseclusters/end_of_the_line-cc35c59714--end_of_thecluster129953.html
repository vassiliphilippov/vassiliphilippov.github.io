(computing) the character or marker that indicates the end of a line of text (a newline or line-termination marker).

Examples

End of the line character
In many editors, the end of the line is represented by a newline character.
When converting files, remember that Windows uses CRLF as the end of the line marker.
A missing end of the line at the end of the file can cause parsing errors in scripts.
