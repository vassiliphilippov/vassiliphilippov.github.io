ribbon

noun
tabbed toolbar in a graphical interface
Pronunciation: / ˈrɪbən / (AmE), / ˈrɪbən / (BrE)
(in computing) a horizontal bar near the top of an application window that groups commands and tools under labeled tabs to help users find features.

Examples

The ribbon menu
To change the font size, click the 'Home' ribbon tab and use the font controls.
She customized the Word ribbon so her most-used tools appeared on a single tab.
Many tutorials explain how to find commands using the new ribbon interface in modern apps.

Related Terms

  • synonym: menu bar - a horizontal bar that contains menus of commands at the top of an application window
  • synonym: toolbar - a row or panel of buttons and controls that give quick access to common functions
