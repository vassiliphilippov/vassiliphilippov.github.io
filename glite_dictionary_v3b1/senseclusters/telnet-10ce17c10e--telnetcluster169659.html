Telnet

verb
to access via Telnet
7.6
Pronunciation: / ˈtɛlnɛt / (AmE), / ˈtɛlnɛt / (BrE)
to connect to a remote computer or device using the Telnet protocol or a Telnet client

Examples

To Telnet into a server
The technician had to Telnet into the server to troubleshoot the network issue.
Before SSH became popular, many administrators would Telnet to remote systems for maintenance.
You can Telnet to the device using its IP address and make configuration changes.

Related Terms

  • synonym: SSH - to connect securely to a remote computer using the SSH protocol
  • antonym: disconnect - to end a connection to a remote computer or device
