problem-oriented language

noun
computer programming language using user-specific terminology
5.4
Pronunciation: / ˈprɑːbləm ˈɔːriˌɛntɪd ˈlæŋɡwɪdʒ / (AmE), / ˈprɒbləm ˈɔːriəntɪd ˈlæŋɡwɪdʒ / (BrE)
a computer programming language in which commands and constructs closely resemble the terminology and concepts of a particular problem area or user domain.

Examples

A problem-oriented language design
A good problem-oriented language can simplify programming for users in specialized fields.
COBOL is an example of a problem-oriented language designed for business applications.
Using a problem-oriented language allows engineers to express solutions in familiar technical terms.

Related Terms

  • synonym: high-level language - a programming language with strong abstraction from the computer's details
  • antonym: machine language - the lowest-level programming language, consisting of binary code
