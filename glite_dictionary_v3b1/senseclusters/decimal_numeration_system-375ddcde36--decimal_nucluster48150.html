decimal numeration system

noun
system of numeration with base ten
Pronunciation: / ˈdɛsɪml nuːˈmɛrəˌʃən ˈsɪstəm / (AmE), / ˈdɛsɪməl ˌnjuːməˈreɪʃən ˈsɪstəm / (BrE)
a system of numeration in which numbers are expressed in base ten, using the digits 0 to 9

Examples
