decimal numeration system

noun
system of numeration with base ten
Pronunciation: / ˈdɛsɪml nuːˈmɛrəˌʃən ˈsɪstəm / (AmE), / ˈdɛsɪməl ˌnjuːməˈreɪʃən ˈsɪstəm / (BrE)
a system of numeration in which numbers are expressed in base ten, using the digits 0 to 9

Examples

The decimal numeration system
Early calculators were designed to operate using the decimal numeration system.
The decimal numeration system is the standard system for denoting integer and non-integer numbers.
When teaching children math, most educators begin with the decimal numeration system before introducing other bases.
