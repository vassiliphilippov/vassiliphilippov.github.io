vault

noun
secure room for valuables
4.6 3.9
Pronunciation: / vɔlt / (AmE), / vɔːlt / (BrE)
a strong room or secure place (physical or digital) for keeping valuable items safe

Examples

Bank vault
Access to the company’s vault is restricted to authorized personnel only.
The bank keeps gold bars in a high-security vault deep underground.
They discovered ancient coins locked away in the old family vault.
Developers store API keys in a vault to prevent accidental exposure during deployments.
The security team configured role-based access so only admins can retrieve secrets from the vault.
After the breach, they rotated all tokens stored in the central vault and revoked old keys.

Related Terms

  • synonym: key management system - a service for generating, storing, and managing cryptographic keys and related access policies.
  • synonym: password manager - an application for storing and autofilling user passwords for websites and services.
  • synonym: safe - a strong container for storing valuables securely
  • synonym: secret manager - software or service that stores and manages access to secrets such as passwords and API keys.
