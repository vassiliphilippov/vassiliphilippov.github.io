reflog

noun
Git reference log
Pronunciation: / ˈrɛfˌlɑɡ / (AmE), / ˈrɛfˌlɒɡ / (BrE)
a record in Git that tracks updates and changes made to references such as branches and HEAD

Examples

Git reflog record
If you lose a branch, you can use the reflog to find its previous state.
The reflog shows all recent changes to the HEAD in your Git repository.
Before restoring a deleted commit, check the reflog for its reference.

Related Terms

  • synonym: branch - a separate line of development in a Git repository
  • synonym: commit history - a list of all commits made in a Git repository
