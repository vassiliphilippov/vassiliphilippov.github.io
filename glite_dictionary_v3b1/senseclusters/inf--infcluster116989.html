inf

noun
Windows setup file
9.9
Pronunciation: / ɪn ɛf / (AmE), / ɪn ɛf / (BrE)
a plain-text configuration or installation file used by Microsoft Windows, typically with the .inf file extension

Examples

Inf file update
The printer would not work until I updated the inf file with the correct driver information.
You can edit the inf file using any text editor to change the installation settings.
After downloading the software, make sure the inf file is present before starting the setup.
