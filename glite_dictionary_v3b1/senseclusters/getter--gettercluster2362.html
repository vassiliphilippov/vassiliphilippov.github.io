getter

noun
function retrieving value
6.1
Pronunciation: /ˈge.tər/ (AmE), /ˈge.tə/ (BrE)
a function or method that retrieves the value of a property from an object or class in programming

Examples

Property getter method
The getter allows you to read the value of the private variable safely.
In many programming languages, a getter is paired with a setter for managing object properties.
You should define a getter if you want to control how a property is accessed.

Related Terms

  • synonym: accessor - a method that accesses or retrieves data from an object
  • antonym: setter - a function or method that sets the value of a property in an object or class
