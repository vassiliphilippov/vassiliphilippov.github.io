dumper

noun
software tool that creates data/memory dumps
8.2
Pronunciation: /ˈdʌm.pər/ (AmE), /ˈdʌm.pə/ (BrE)
(computing) a program or tool that creates a copy (a "dump") of a computer's memory or other raw data for debugging, analysis, or forensic examination.

Examples

Memory dumper
The crash analyst used a dumper to dump the program's memory for debugging.
They ran the dumper tool to capture the server's RAM after the unexpected reboot.
Several dumpers produced different-sized files for later forensic analysis of the breach.

Related Terms

  • synonym: core dump - a recorded state of a program's memory, typically produced when a program crashes
  • synonym: memory dump - a file or image containing the contents of a computer's memory at a point in time, used for debugging or analysis
