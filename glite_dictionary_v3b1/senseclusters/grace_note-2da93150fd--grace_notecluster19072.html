grace note

noun
decorative musical note, played quickly
3.5
Pronunciation: / ˈɡreɪs ˌnoʊt / (AmE), / ˈɡreɪs nəʊt / (BrE)
a short, decorative musical note added as an ornament to the main notes in a piece of music

Examples

A grace note ornament
He struggled with the fast grace note passages in the intricate baroque composition.
The pianist played the grace note before the final chord, adding a touch of elegance to the piece.
You can see the grace note written in small type above the main melody on the sheet music.
Adding a grace note gave the melody more character and expression.
Some traditional Irish tunes are known for featuring multiple grace notes in quick succession.
The pianist skillfully included a grace note at the end of each phrase.

Related Terms

  • synonym: acciaccatura - a type of grace note played quickly before the main note
  • synonym: appoggiatura - an ornamented note that delays the next note, of which it usually takes some value
