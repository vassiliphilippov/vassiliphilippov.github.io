input signal

noun
signal entering an electronic device
3.5
Pronunciation: / ˈɪnˌpʊt ˈsɪɡnəl / (AmE), / ˈɪn.pʊt ˈsɪɡ.nəl / (BrE)
a signal that enters an electronic system, serving as data or a stimulus for processing

Examples

Noise can interfere with the quality of an input signal in audio equipment.
The amplifier boosts the input signal to a higher voltage level.
