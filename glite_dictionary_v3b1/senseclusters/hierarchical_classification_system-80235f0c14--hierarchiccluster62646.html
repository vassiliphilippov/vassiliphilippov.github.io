hierarchical classification system

noun
system for ranked categorization
6.6 5.4
Pronunciation: / haɪ.əˈrɑːr.kɪ.kəl ˌklæs.ɪ.fɪˈkeɪ.ʃən ˈsɪs.təm / (AmE), / haɪəˈrɑː.kɪ.kəl ˌklɑːs.ɪ.fɪˈkeɪ.ʃən ˈsɪs.təm / (BrE)
a system of organizing items or information in which each entry is arranged according to a ranked or graded hierarchy

Examples

A hierarchical classification system structure
A hierarchical classification system helps users find information quickly by narrowing down from broad to specific categories.
In biology, the hierarchical classification system groups organisms from kingdom down to species.
The library uses a hierarchical classification system to organize its books by subject and subcategory.
