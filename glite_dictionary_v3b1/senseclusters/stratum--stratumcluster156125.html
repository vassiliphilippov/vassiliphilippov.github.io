stratum

noun
cryptocurrency mining protocol
8.7
Pronunciation: / ˈstrætəm / (AmE), / ˈstrɑːtəm / (BrE)
(in cryptocurrency mining) a network protocol used by mining pools to distribute work to miners and to receive their submitted results (shares).

Examples

Stratum protocol
Miners connect to pools using the stratum protocol to receive work.
The server dropped the connection, so his mining rig stopped using stratum for several minutes.
Software updates improved stratum compatibility with new ASIC miners and reduced rejected shares.
