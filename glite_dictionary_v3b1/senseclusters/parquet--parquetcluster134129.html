parquet

noun
columnar data file format
7.6
Pronunciation: /ˈpɑːr.keɪ/ /; / /pɑːr.ˈkeɪ/ (AmE), /ˈpɑː.keɪ/ (BrE)
(in computing) a columnar storage file format used to store large datasets efficiently, enabling fast analytical queries and good compression.

Examples

Parquet data file
The dataset was stored in an open-source parquet file to accelerate queries.
Analysts converted the raw logs into parquet to make analytical queries much faster.
Cloud data warehouses often organize large tables using parquet files for efficient compression and scan performance.

Related Terms

  • synonym: Apache Parquet - an open-source columnar storage file format widely used in big data ecosystems
  • synonym: columnar storage - a data storage layout that stores values column-by-column to improve analytic query performance
