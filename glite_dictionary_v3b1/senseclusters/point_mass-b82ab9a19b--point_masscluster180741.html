point mass

noun
object with mass but no size (physics)
9.9 6.3
Pronunciation: /pɔɪnt mæs/ (AmE), /pɔɪnt mɑːs/ /; / /pɔɪnt mæs/ (BrE)
An object or particle considered to have mass but no size or dimensions, used as an idealization in physics.

Examples

A point mass object
In many physics problems, the planet is treated as a point mass to simplify calculations.
The point mass model is useful for analyzing the motion of projectiles.
You can ignore the rotation if you approximate the satellite as a point mass.
