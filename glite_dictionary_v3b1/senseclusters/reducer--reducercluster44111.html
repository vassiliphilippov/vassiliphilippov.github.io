reducer

noun
function that combines values into one
7.2 0.13/m
Pronunciation: / r…™Ààdus…ôr / (AmE), / r…™ÀàdjuÀês…ô / (BrE)
In computer science, a function that combines multiple inputs (such as elements of a collection or an existing state and an action) into a single value or new state by repeatedly applying a specified operation.

Examples

MapReduce reducer function
The Hadoop job used a custom reducer to sum all sales values for each product id.
In Redux, every reducer receives the current state and an action, then returns new state.
You can implement a reducer in JavaScript with reduce to combine array items into a single total.

Related Terms

  • synonym: aggregate - to combine several pieces of data into a single summary value
  • synonym: fold - a functional programming operation that aggregates a sequence into a single result by applying a function repeatedly
