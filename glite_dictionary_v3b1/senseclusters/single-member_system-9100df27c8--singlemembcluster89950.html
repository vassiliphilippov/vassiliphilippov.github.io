single-member system

noun
electoral system with one member per district
5.1 5.9
Pronunciation: / ˈsɪŋɡəl ˈmɛmbɚ ˈsɪstəm / (AmE), / ˈsɪŋɡəl ˈmɛmbə ˈsɪstəm / (BrE)
An electoral system in which only one representative is elected from each district or constituency.

Examples

A single-member system election
Many countries use a single-member system to elect their legislatures.
Some reformers argue that the single-member system does not fairly represent minority interests.
The single-member system often leads to a two-party political structure.

Related Terms

  • antonym: proportional representation - an electoral system in which parties gain seats in proportion to the number of votes cast for them
