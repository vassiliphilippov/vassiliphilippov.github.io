egress

noun
outbound network traffic
7.4
Pronunciation: /ˈiː.gres/ /; / /ˈɪ.gres/ (AmE), /ˈiː.gres/ /; / /ˈɪ.gres/ (BrE)
(computing, networking) The flow of data leaving a network or the route/interface through which data exits an internal network to an external network; outbound traffic.

Examples

Network egress traffic
The network administrator monitored egress to detect any unusual transfers of sensitive files to external servers.
Cloud providers often charge for egress when customers download large datasets from remote storage.
During the audit, the team blocked suspicious egress routes to prevent potential data exfiltration.

Related Terms

  • antonym: ingress - incoming network traffic or the route by which data enters a network; opposite of egress
  • synonym: outbound traffic - data packets or network traffic that travel from an internal network to external destinations
