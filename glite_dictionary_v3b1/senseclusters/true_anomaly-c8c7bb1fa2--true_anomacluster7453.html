true anomaly

noun
angular position in orbit from periapsis
5.2 8.5
Pronunciation: /truː ə.ˈnɑː.mə.liː/ /; / /truː ə.ˈnɑː.mə.lɪ/ (AmE), /truː ə.ˈnɒ.mə.lɪ/ (BrE)
the angular distance, measured in degrees, of an orbiting object from the point of periapsis along its orbital path

Examples

true anomaly angle
At periapsis, the true anomaly is defined as zero degrees.
Engineers use true anomaly to calculate the position of a spacecraft relative to periapsis.
The satellite's true anomaly determines its exact location along the orbit at any time.
