fuzzy logic

noun
mathematical logic with degrees of truth
2.3 5.6
Pronunciation: /ˈfʌz.iː ˈlɑː.dʒɪk/ /; / /ˈfʌz.ɪ ˈlɑː.dʒɪk/ (AmE), /ˈfʌz.iː ˈlɒ.dʒɪk/ /; / /ˈfʌz.ɪ ˈlɒ.dʒɪk/ (BrE)
a system of mathematical logic in which truth values may range continuously between false and true, typically between 0 and 1

Examples

Fuzzy logic system
Many modern control systems use fuzzy logic to make decisions in uncertain environments.
Researchers apply fuzzy logic in artificial intelligence to handle imprecise or ambiguous information.
The use of fuzzy logic helps washing machines determine the appropriate cycle for different loads.
