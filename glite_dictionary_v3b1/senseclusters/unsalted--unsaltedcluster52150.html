unsalted

adjective
not using a cryptographic salt (hashing)
8.7

(in computing, especially information security) (of a password hash or stored credential) created without adding a cryptographic salt, making it weaker and more vulnerable to precomputed attacks.

Examples

Unsalted password
The IT audit flagged unsalted password hashes, leaving users vulnerable to rainbow-table attacks.
Developers updated the code after discovering unsalted hashes being stored in the legacy database.
Modern frameworks warn if you attempt to save unsalted credentials without using a secure hashing function.
