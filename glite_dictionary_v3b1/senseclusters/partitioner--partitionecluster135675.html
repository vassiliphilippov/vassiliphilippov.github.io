partitioner

noun
software that assigns partitions
7.8
Pronunciation: / pɑːrˈtɪʃənər / (AmE), / pɑːˈtɪʃənə / (BrE)
a software component or algorithm that assigns data, messages, or tasks to specific partitions in a distributed system

Examples

Data partitioner
The partitioner ensures that each message is sent to the correct partition in the system.
A well-designed partitioner can improve the performance of distributed databases.
Developers often customize the partitioner to balance the workload evenly across servers.

Related Terms

  • antonym: aggregator - a component that collects data from multiple sources
  • synonym: distributor - a component that distributes data or tasks among multiple destinations
