code monkey

noun
programmer doing repetitive tasks
Pronunciation: / koʊd ˈmʌŋki / (AmE), / kəʊd ˈmʌŋki / (BrE)
an informal and sometimes disparaging term for a computer programmer who is assigned repetitive coding tasks and has little involvement in project planning or decision-making

Examples

A code monkey programmer
After months as a code monkey, he finally got a chance to lead his own project.
Many startups rely on code monkeys to handle routine bug fixes and updates.
She didn't want to be just a code monkey and sought a more creative role in the company.

Related Terms

  • synonym: programmer - a person who writes computer programs
