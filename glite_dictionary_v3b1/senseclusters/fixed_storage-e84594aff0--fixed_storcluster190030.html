fixed storage

noun
computer memory accessible but not modifiable
Pronunciation: / fɪkst ˈstɔrɪdʒ / (AmE), / fɪkst ˈstɔːrɪdʒ / (BrE)
(computing) A type of computer memory whose contents can be accessed and read but cannot be modified by the user or computer.

Examples

Fixed storage memory
Data in fixed storage is protected from being overwritten or erased.
Older computer consoles often used fixed storage for storing their operating system.
The system BIOS is typically stored in fixed storage to prevent accidental modification.

Related Terms

  • synonym: read-only memory - memory that can be read but not modified
