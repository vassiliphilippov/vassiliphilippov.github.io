unary

noun
base-1 numeral system
6.9
Pronunciation: / ˈjuːˌnɛri / (AmE), / ˈjuː.nəri / (BrE)
A numeral system, also called base-1, in which a single symbol is repeated as many times as the value of the number being represented.

Examples

Unary numeral system
Counting in unary is inefficient for large numbers, but simple for small amounts.
Some primitive counting methods use a unary system, where each stick or mark represents one item.
The unary numeral system represents seven as seven tally marks.
