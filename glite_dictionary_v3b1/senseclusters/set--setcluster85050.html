set

verb
adapt (text) for musical or performance use
Pronunciation: / sɛt / (AmE), / sɛt / (BrE)
to adapt or arrange something for performance, such as putting words to music or arranging text for printing

Examples

Set a poem to music
Many poets have had their works set to classical compositions over the centuries.
The composer decided to set Shakespeare's sonnet to music for the choir.
The playwright set the ancient story as a modern musical.
She carefully set the title in bold type at the top of each page.
The designer is setting the text in italics for emphasis.
The printer will set the manuscript in a clear, readable font.
