ARP

sym
network mapping protocol
Pronunciation: / ɑːr piː / (AmE), / ɑː pɪ / (BrE)
a computer protocol used to map network addresses to physical hardware addresses on a local network

Examples

ARP network protocol
The technician used ARP to find the physical address of the device on the network.
If you suspect a conflict, check the ARP table for duplicate entries.
Understanding how ARP works is essential for troubleshooting local network issues.

Related Terms

  • synonym: DNS - a system that translates domain names to IP addresses
  • synonym: MAC address - a unique identifier assigned to network hardware
