currying

noun
function transformation
9.8
Pronunciation: / ˈkɜːriɪŋ / (AmE), / ˈkʌriɪŋ / (BrE)
a programming technique in which a function is transformed into a sequence of functions, each taking a single argument

Examples

Currying a function
Understanding currying can make your code more modular and easier to test.
Many functional programming languages support currying as a built-in feature.
By using currying, developers can create more reusable and flexible functions.

Related Terms

  • synonym: partial application - a technique where a function is applied to some arguments, producing another function
