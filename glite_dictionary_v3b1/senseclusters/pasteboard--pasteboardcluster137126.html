pasteboard

noun
digital clipboard
Pronunciation: / ˈpeɪstˌbɔrd / (AmE), / ˈpeɪstˌbɔːd / (BrE)
(computing) a temporary area in a computer's memory or operating system used to store data (text, images, files) that has been cut or copied so it can be pasted elsewhere.

Examples

Copy to pasteboard
She copied the image to the pasteboard and pasted it into the report.
Developers use the pasteboard API to share text and images between apps on macOS.
After copying the link, check the pasteboard before closing the browser to ensure it saved.

Related Terms

  • synonym: clipboard - a software feature or area that temporarily holds cut or copied data for pasting
