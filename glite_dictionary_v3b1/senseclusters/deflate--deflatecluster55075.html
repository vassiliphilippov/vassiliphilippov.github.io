deflate

verb
compress digital data (computing)
Pronunciation: / dɪˈfleɪt / (AmE), / dɪˈfleɪt / (BrE)
(computing) to compress digital data or files to reduce their size, especially using the DEFLATE compression method.

Examples

To deflate a file
The backup script will deflate old log files before moving them to cold storage.
After transfer, the system deflated the package using the DEFLATE algorithm to save bandwidth.
Many libraries offer functions for deflating data streams in web applications to improve load times.

Related Terms

  • synonym: compress - to reduce the size of data or files by encoding them more efficiently
  • antonym: inflate - (computing) to decompress data previously compressed (the opposite operation to deflating)
