nofollow

noun
HTML link attribute
7.9
Pronunciation: /ˈnoʊ.fɑː.loʊ/ (AmE), /ˈnəʊ.fɒl.əʊ/ (BrE)
an HTML attribute value used to tell search engines not to follow a specific web link for ranking purposes

Examples

Nofollow attribute
Webmasters often add the nofollow attribute to links in comments to prevent spam from affecting search rankings.
The article explained how using nofollow can help control which links search engines consider.
Many websites automatically mark external links as nofollow to avoid passing search engine authority.
