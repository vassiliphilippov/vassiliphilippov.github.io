retriever

noun
software tool that fetches information
Pronunciation: / rəˈtrivər / (AmE), / rɪˈtriːvə / (BrE)
(in computing, especially information retrieval and AI) a software component or tool that locates and returns relevant information or documents from a database or corpus, often using indexing or vector search.

Examples

AI retriever tool
The new AI retriever returned several relevant research papers within seconds of my query.
Our engineering team integrated a dense-vector retriever to fetch short passages that answer user questions.
During the demo, the retriever failed to locate archived reports because the index had not been updated.

Related Terms

  • synonym: retrieval system - a software system designed to store, index, and return relevant documents or data
  • synonym: search engine - a system that finds information on the web or in a database in response to user queries
