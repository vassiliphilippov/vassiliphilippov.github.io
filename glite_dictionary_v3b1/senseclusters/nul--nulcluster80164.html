nul

noun
ASCII zero byte
Pronunciation: / nʌl / (AmE), / nʌl / (BrE)
the ASCII control character with code 0, used in computing to represent a zero byte or end-of-string marker

Examples

Nul character
The file ended with a nul character to indicate the end of the data.
In C programming, a string is terminated by a nul byte.
If the system encounters a nul character, it stops reading the input.

Related Terms

  • synonym: null character - the ASCII character with code 0, often used to mark the end of a string
