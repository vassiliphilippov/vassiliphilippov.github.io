exit

verb
terminate a program or process
3.6
Pronunciation: /ˈeg.zɪt/ /; / /ˈek.sət/ (AmE), /ˈek.sɪt/ /; / /ˈeg.zɪt/ (BrE)
(in computing) to terminate the execution of a program or process, often returning a status (exit) code to the operating system.

Examples

To exit the program
The script will exit gracefully if the input is invalid or missing.
After the error, the background process exited with status code 1, prompting a restart.
At the shell prompt, type exit to end the remote session and close the connection.

Related Terms

  • synonym: quit - to stop running; to leave an application or session.
  • antonym: start - to begin running or launch a program or process (opposite of exit in this technical sense).
  • synonym: terminate - to bring something to an end; stop a program or process.
