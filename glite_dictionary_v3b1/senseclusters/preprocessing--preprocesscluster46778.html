preprocessing

noun
code transformation stage
6.5
Pronunciation: / ˌpriːˈprɑːsɛsɪŋ / (AmE), / ˌpriːˈprɒsɛsɪŋ / (BrE)
the stage in computer programming where source code is modified by a preprocessor before main compilation or interpretation

Examples

Code preprocessing
The preprocessing phase checks for macros and includes necessary files before compiling the program.
Errors in preprocessing can cause the final code to behave unexpectedly.
Automated tools often handle preprocessing to simplify the programmer's work.

Related Terms

  • synonym: compilation - the process of converting source code into executable code
  • antonym: postprocessing - processing that occurs after the main operation or compilation
