(computing) to save or store data or program state to non-volatile storage so that it remains available after a program ends, a device reboots, or power is lost.

Examples

To persist data
The application will persist user preferences to disk so settings remain between sessions.
Database transactions must persist changes safely before acknowledging the client's request.
By default, cookies do not persist across browser restarts unless explicitly configured.
