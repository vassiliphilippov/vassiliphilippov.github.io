Kronecker delta

noun
function equaling 1 if variables match, 0 otherwise
Pronunciation: / ˈkroʊnɛkər ˈdɛltə / (AmE), / ˈkrɒnɛkə ˈdɛltə / (BrE)
a function of two variables that equals 1 if the variables are equal and 0 otherwise; used especially in mathematics and physics

Examples

Kronecker delta function
In tensor notation, the Kronecker delta acts as the identity matrix.
The Kronecker delta is frequently used to simplify expressions in linear algebra.
When i and j are not equal, the Kronecker delta yields zero.
