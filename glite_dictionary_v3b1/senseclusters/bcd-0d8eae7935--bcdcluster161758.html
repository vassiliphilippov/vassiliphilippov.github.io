BCD

noun
binary-coded decimal
9.2
Pronunciation: / biː siː diː / (AmE), / biː siː diː / (BrE)
a method of representing decimal numbers in binary form, where each digit is encoded separately as a group of four bits

Examples

BCD number format
The old computer system used BCD to store and process numerical data.
When converting numbers, the engineer explained how BCD differs from pure binary representation.
Many calculators rely on BCD for accurate decimal calculations.

Related Terms

  • synonym: binary - a system of representing numbers using only two digits, 0 and 1
  • antonym: decimal - a number system based on ten digits, 0 through 9
