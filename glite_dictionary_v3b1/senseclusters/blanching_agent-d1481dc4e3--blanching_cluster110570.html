blanching agent

noun
substance that whitens or decolorizes
Pronunciation: / ˈblæn.tʃɪŋ ˈeɪ.dʒənt / (AmE), / ˈblɑːn.tʃɪŋ ˈeɪ.dʒənt / (BrE)
A substance used to make things white or colorless by removing color or impurities.

Examples

A blanching agent in food
Adding a blanching agent helps remove natural pigments from the food product.
Make sure to handle the blanching agent carefully, as it can be caustic in large quantities.
The textile factory uses a blanching agent to whiten cotton fibers before dyeing.
