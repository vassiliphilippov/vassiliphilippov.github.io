(in software development) intended for use only within the same module, library, or assembly; not part of the public interface available to external code or users.

Examples

Internal use only
The team marked several classes as internal to prevent external code from depending on them.
To fix the bug, modify the internal method that handles session tokens on the server.
The developer documented why the internal API should not be used by third-party extensions.

Related Terms

  • antonym: public - available to all code or users; the opposite of access restricted to the same module or assembly
