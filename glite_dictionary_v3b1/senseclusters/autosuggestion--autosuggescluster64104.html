autosuggestion

noun
automatic UI suggestion (autocomplete)
5.4
Pronunciation: /ɑː.toʊ.sə.ˈdʒes.tʃən/ /; / /ɔː.toʊ.sə.ˈdʒes.tʃən/ (AmE), /ɔː.təʊ.sə.ˈdʒes.tʃən/ (BrE)
an automatic, algorithmic suggestion of possible completions, queries, or terms shown by a user interface (for example a search box or form) as the user types.

Examples

Autosuggestion in search
The website's search bar shows autosuggestion to help users find results faster.
He disabled autosuggestions in his browser to avoid seeing irrelevant query completions.
Developers added autosuggestion so users finish long product names with fewer keystrokes.

Related Terms

  • synonym: autocomplete - a feature that predicts and completes user input in a text field
  • synonym: typeahead - a UI behavior that shows likely completions as the user types
