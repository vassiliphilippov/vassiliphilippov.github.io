loop gain

noun
gain in a feedback system
5.7 7.2
Pronunciation: / luːp ɡeɪn / (AmE), / luːp ɡeɪn / (BrE)
the amount of amplification of a feedback amplifier or system, expressed as the ratio of the feedback signal to the input; used to describe how much output is fed back to the input, affecting the system's stability and response.

Examples

Loop gain value
If the loop gain is set too high, unwanted feedback may occur.
Reducing the loop gain can help maintain amplifier stability.
The engineer adjusted the loop gain to ensure the system would not oscillate.
