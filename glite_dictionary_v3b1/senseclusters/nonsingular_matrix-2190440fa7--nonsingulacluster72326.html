nonsingular matrix

noun
square matrix with nonzero determinant
6.8
Pronunciation: / nɑnˈsɪŋ.jə.lɚ ˈmeɪ.trɪks / (AmE), / nɒnˈsɪŋ.gjʊ.lə ˈmeɪ.trɪks / (BrE)
A square matrix whose determinant is not zero, and which therefore has an inverse.

Examples

A nonsingular matrix inverse
Every nonsingular matrix has a unique inverse in linear algebra.
Solving the system is possible only if the coefficient matrix is nonsingular matrix.
To compute the inverse, first check that the matrix is a nonsingular matrix.
