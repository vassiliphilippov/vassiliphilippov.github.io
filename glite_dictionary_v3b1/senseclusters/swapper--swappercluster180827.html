swapper

noun
memory management program
8.2
Pronunciation: / ˈswɑːpər / (AmE), / ˈswɒpə / (BrE)
a computer program or process that moves data between main memory and disk to manage available RAM

Examples

The swapper process

If the swapper fails, the operating system may run out of usable RAM.
System performance improved after the swapper was optimized to handle large workloads.
