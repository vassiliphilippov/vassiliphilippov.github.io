guid

noun
unique identifier
Pronunciation: / ɡaɪd / (AmE), / ɡaɪd / (BrE)
a unique reference number used in computer systems to identify information or objects

Examples

A unique guid number
Each user account is assigned a unique guid for identification.
The database uses a guid to ensure every record is distinct.
When creating a new file, the system generates a guid to track it.

Related Terms

  • synonym: identifier - a value used to identify something in a system
  • synonym: uuid - a universally unique identifier used in computing
