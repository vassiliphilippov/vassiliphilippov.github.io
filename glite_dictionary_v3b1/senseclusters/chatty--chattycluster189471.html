chatty

adjective
many small requests (computing)
7.5
Pronunciation: /ˈtʃæ.tiː/ /; / /ˈtʃæ.tɪ/ (AmE), /ˈtʃæ.tiː/ /; / /ˈtʃæ.tɪ/ (BrE)
(in computing) describing an interface, API, or protocol that requires many small, frequent back-and-forth requests, often increasing latency or overhead.

Examples

Chatty API
The mobile app's backend became slow because the REST API was too chatty with excessive small calls.
To improve performance, the engineers replaced several chatty endpoints with a single bulk-fetch method.
A chatty message protocol can significantly increase latency on mobile networks.

Related Terms

  • antonym: coarse-grained - describing an interface that groups work into larger requests to reduce the number of calls.
  • synonym: verbose - sending more messages or information than necessary; in computing, producing excessive requests or data.
