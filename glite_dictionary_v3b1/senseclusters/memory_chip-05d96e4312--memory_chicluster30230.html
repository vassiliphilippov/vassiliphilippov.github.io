memory chip

noun
neural implant that stores memories
4.4
Pronunciation: /ˈmem.ər.iː tʃɪp/ (AmE), /ˈmem.ə.riː tʃɪp/ (BrE)
(in neuroscience/biotechnology) a device implanted in the brain or connected to neural tissue that records, stores, or replays human memories or memory-related data.

Examples

Brain memory chip
Researchers implanted a memory chip in volunteers to store and replay short autobiographical memories.
After the accident she relied on a memory chip to recover details she could not recall.
Ethicists debated whether companies should sell devices such as the memory chip to alter personal identity.

Related Terms

  • synonym: brain–computer interface - a system that enables direct communication between the brain and external devices; used to read or write neural signals
  • synonym: memory implant - a device implanted in the brain intended to store or modify memories
