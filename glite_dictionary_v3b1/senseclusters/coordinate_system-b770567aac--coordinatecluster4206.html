coordinate system

noun
system assigning positions via coordinates
3.9 0.79/m
Pronunciation: / ko äÀà…îrd…ôÀåne…™t Ààs…™st…ôm / (AmE), / k…ô äÀà…îÀêd…™n…ôt Ààs…™st…ôm / (BrE)
A system that uses numbers or other symbols (coordinates) to uniquely determine the position of points or objects in a space.

Examples

A coordinate system in space
Astronomers use a specialized coordinate system to map the positions of stars in the sky.
In the Cartesian coordinate system, each point is identified by an ordered pair of numbers.
The coordinate system allows us to plot points on a graph accurately.

Synonyms

  • reference system
    4.9 0.07/m
    a reference system maps positions for navigation or astronomy in practice, while a coordinate system is the basic mathematical tool to locate points.
  • reference frame
    7.2 0.58/m
    a reference frame gives position, orientation, and motion information in physics, while a coordinate system simply assigns coordinates to locate points.
  • frame of reference
    6.1 0.38/m
    a frame of reference is an observer's viewpoint for measuring events, whereas a coordinate system is the formal set of coordinates to locate points.

Related Terms
