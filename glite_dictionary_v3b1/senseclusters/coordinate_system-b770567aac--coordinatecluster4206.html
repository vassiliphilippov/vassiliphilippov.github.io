coordinate system

noun
system assigning positions via coordinates
Pronunciation: / koʊˈɔrdəˌneɪt ˈsɪstəm / (AmE), / kəʊˈɔːdɪnət ˈsɪstəm / (BrE)
A system that uses numbers or other symbols (coordinates) to uniquely determine the position of points or objects in a space.

Examples

A coordinate system in space
Astronomers use a specialized coordinate system to map the positions of stars in the sky.
In the Cartesian coordinate system, each point is identified by an ordered pair of numbers.
The coordinate system allows us to plot points on a graph accurately.

Related Terms

  • degree higher: Cartesian coordinate system - a coordinate system specifying each point uniquely by a set of numerical coordinates
  • synonym: reference frame - a framework for specifying position and orientation in space
