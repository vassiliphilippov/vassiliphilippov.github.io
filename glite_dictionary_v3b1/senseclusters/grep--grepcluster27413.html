grep

verb
to search text files
Pronunciation: / ɡrɛp / (AmE), / ɡrɛp / (BrE)
to search for specific text patterns in files, especially using a command-line tool on a computer

Examples

To grep a file
You can grep for error messages in the log file to troubleshoot problems.
Many programmers grep through code to find specific functions or variables.
She used a script to grep thousands of files for the keyword 'confidential'.

Related Terms

  • synonym: scan - to examine something quickly for specific information
  • synonym: search - to look for information or text
