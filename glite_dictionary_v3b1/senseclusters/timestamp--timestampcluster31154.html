timestamp

noun
recorded date and time
3.5
Pronunciation: / ˈtaɪmˌstæmp / (AmE), / ˈtaɪmˌstæmp / (BrE)
a record or mark showing the exact date and time when something happens, is created, or is changed

Examples

The photo had a timestamp in the corner showing when it was taken.
Each message in the chat displays a timestamp so you know when it was sent.
The document's timestamp proved it was created before the deadline.
Each file in the folder has a timestamp showing when it was last modified.
The server automatically adds a timestamp to every message it receives.
You can sort the entries by their timestamp to see the most recent updates.
The timestamp on the document shows it was last edited yesterday afternoon.
Every photo taken with her phone includes a timestamp in the file details.
The security system records a timestamp each time someone enters the building.

  • synonym: log - a written record of events or times
