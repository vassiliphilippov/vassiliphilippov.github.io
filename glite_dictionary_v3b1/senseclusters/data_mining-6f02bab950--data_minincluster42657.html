data mining

noun
analyzing data to discover patterns
Pronunciation: / ˈdeɪtə ˌmaɪnɪŋ / (AmE), / ˈdeɪtə ˌmaɪnɪŋ / (BrE)
The process of analyzing large sets of data using advanced search techniques and statistical algorithms to discover patterns and relationships.

Examples

Data mining techniques
Advances in data mining have enabled better prediction models in weather forecasting.
Many businesses use data mining to identify trends in customer behavior.
The researcher applied data mining techniques to detect previously unknown patterns in medical records.

Related Terms

  • synonym: data analysis - the process of inspecting and modeling data to discover useful information
  • synonym: machine learning - computational methods that enable computers to find patterns in data
