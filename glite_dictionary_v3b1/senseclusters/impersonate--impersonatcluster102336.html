impersonate

verb
assume a digital identity (computing)
7.9 0.1/m
Pronunciation: / …™mÀàp…úrs…ône…™t / (AmE), / …™mÀàp…úÀês…ône…™t / (BrE)
(in computing or cybersecurity) to assume the digital identity of another user, account, process, or service within a system or network, either legitimately (for testing or delegated access) or maliciously to gain unauthorized access.

Examples

To impersonate a user
During security testing, the tool can impersonate a user to verify login controls.
A misconfigured script impersonates the admin account, granting the attacker full control of the system.
By impersonating the service's certificate, the malware tricked the network into trusting it.

Related Terms

  • synonym: masquerade - in security contexts, to present oneself as a different user or system
  • synonym: spoof - to falsify data (for example an address or certificate) to appear as another entity
