staging

noun
pre-production testing environment (software)

a pre-production environment in software development where updates, changes, or releases are tested and previewed before deployment to the live (production) system.

Examples

Staging server
We moved the update to staging for final QA before releasing it to production.
The bug appeared only on staging, so developers fixed it before the public rollout.
Their deployment script automatically syncs the database with staging after each nightly build.

Synonyms

    6.2 3/m

  • preproduction
    5.3 0.01/m
    preproduction is a formal name for a test environment that mimics live systems for final release checks, while staging is the more common term in development.
