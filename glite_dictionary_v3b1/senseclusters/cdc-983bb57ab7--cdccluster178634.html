Change Data Capture (database technique)

Pronunciation: / si di si / (AmE), / si di si / (BrE)
(computing) Change Data Capture: a technique that records or streams changes made to a database so those changes can be replicated, synchronized, or audited.

Examples

CDC system update

Related Terms

  • synonym: change data capture - the process of detecting and capturing data changes in a database for downstream use
  • synonym: data replication - the copying of data between databases or systems to ensure consistency and availability
  • synonym: data streaming - continuous transfer of data updates in real time for processing or analytics
