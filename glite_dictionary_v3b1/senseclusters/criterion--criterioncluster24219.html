criterion

noun
loss / objective function (ML)
Pronunciation: / kraɪˈtɪəriən / (AmE), / kraɪˈtɪəriən / (BrE)
(in machine learning) a loss or objective function used to measure error and guide a model's learning during training.

Examples

Training criterion in AI
The team changed the training criterion from MSE to cross-entropy to improve accuracy.
Choosing the right criterion helped the model converge faster during the supervised learning experiments.
Researchers compared several loss functions and settled on a robust criterion for noisy labels.

Related Terms

  • synonym: loss function - a mathematical function that measures the difference between predicted and true values during model training
  • synonym: objective function - a function that an algorithm seeks to minimize or maximize during optimization
