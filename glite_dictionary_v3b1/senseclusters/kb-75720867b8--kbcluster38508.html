KB

noun
unit of digital information (1024 bytes)
3.0
Pronunciation: / keɪ bi /; / ˌkeɪˈbi / (AmE), / keɪ bi /; / ˌkeɪˈbiː / (BrE)
a unit of digital information (kilobyte), used to measure computer memory or data size; usually equals 1,000 or 1,024 bytes

Examples

KB of data
Each photo takes up about 300 KB of space on my phone.
My email attachment was only 250 KB, so it uploaded quickly.
The document size is less than one KB, making it easy to send via text.
The document must not exceed 500 KB in size to be uploaded.
The email attachment was too large at 1,200 KB and could not be sent.
This file is only 200 KB, so it should download quickly.
