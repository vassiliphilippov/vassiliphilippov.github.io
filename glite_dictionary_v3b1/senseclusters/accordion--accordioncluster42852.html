accordion

noun
expandable UI panel
5.5
Pronunciation: / əˈkɔr.di.ən / (AmE), / əˈkɔː.di.ən / (BrE)
(in computing and user-interface design) a vertical list of headers that the user can open or close to reveal or hide the content beneath each header.

Examples

Accordion menu
The website's FAQ uses an accordion so each question expands without leaving the page.
Designers recommend using an accordion to hide long lists on small-screen interfaces.
In the prototype, the accordion component collapsed unexpectedly when the user rapidly tapped headers.

Related Terms

  • synonym: collapsible panel - a UI element that can be shown or hidden to save space
  • synonym: expandable panel - a section of an interface that expands to show more content and collapses to hide it
