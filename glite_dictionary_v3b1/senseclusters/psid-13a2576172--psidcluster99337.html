PSID

noun
unique identifier
Pronunciation: / pi ɛs aɪ di / (AmE), / piː ɛs aɪ diː / (BrE)
an abbreviation for 'psychological identification' or 'process session identifier', used in technical or psychological contexts to refer to a unique identifier

Examples

Unique PSID code
Each user is assigned a unique PSID for secure access.
The system logs every action under the corresponding PSID to track user activity.
You must enter your PSID to access the confidential files.

Related Terms

  • synonym: ID - a general term for identification or identifier
  • synonym: session ID - a unique identifier for a user session in computing
