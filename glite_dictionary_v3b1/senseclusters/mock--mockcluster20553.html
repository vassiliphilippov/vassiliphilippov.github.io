mock

verb
simulate a component in tests
4.9
Pronunciation: / mɑk / (AmE), / mɒk / (BrE)
(in software development) to replace or simulate the behaviour of a real component (such as an API, database, or service) with a controlled stand-in during testing.

Examples

To mock an API
During tests, the team mocked the payment API so failures wouldn't hit production.
I mock the email service in my unit tests to return controlled data for each case.
They were mocking the remote server during integration tests to simulate slow responses.

Related Terms

  • synonym: simulate - to imitate the behaviour or operation of a system or process
  • synonym: stub - a small piece of code that stands in for a component during testing
