Unicode

noun
text encoding standard
4.5
Pronunciation: / ˈjuːnɪˌkoʊd / (AmE), / ˈjuːnɪˌkəʊd /; / ˈjuːnɪkəʊd / (BrE)
a universal standard for encoding and representing text from many of the world's writing systems

Examples

Unicode standard
The document was saved using Unicode to support multiple languages.
Many modern applications rely on Unicode to display international characters correctly.
Each emoji is represented by a different Unicode in the standard we use.
Modern websites use Unicode to display text in many different languages.
The document was saved in Unicode format to support special characters.
Thanks to Unicode, emojis and symbols can appear correctly on most devices.

Related Terms

  • antonym: ASCII - an earlier character encoding standard for electronic communication
  • synonym: UTF-8 - a common Unicode encoding format for computers
  • synonym: character set - a collection of characters that can be used by a computer system
