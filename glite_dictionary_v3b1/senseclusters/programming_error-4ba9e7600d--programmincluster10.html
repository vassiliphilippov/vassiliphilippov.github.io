programming error

noun
software error caused by faulty code
Pronunciation: / ˈproʊɡræmɪŋ ˈɛrər / (AmE), / ˈprəʊɡræmɪŋ ˈɛrər / (BrE)
an error that occurs in computer software as a result of incorrect or faulty code written by a programmer, leading to unintended outcomes or malfunctions

Examples

A programming error message
A simple programming error caused the system to crash every time it processed a large file.
Debugging tools can help identify and fix programming errors before software is released.
The unexpected behavior was traced to a programming error in the application's login module.

Related Terms

  • synonym: bug - an error or flaw in a computer program that causes it to operate incorrectly
