sandbox

noun
isolated computing environment
5.4
Pronunciation: /ˈsænd.bɑːks/ (AmE), /ˈsænd.bɒks/ (BrE)
(in computing) a controlled, isolated environment in which untrusted or experimental code or software can run safely without affecting the host system or production data.

Examples

Software sandbox
Developers run experiments in a sandbox to observe behavior without risking the main system.
The security team executed the suspicious program inside a sandbox to analyze its network activity safely.
Students used a cloud-based sandbox to try coding exercises without changing the shared classroom servers.

Related Terms

  • synonym: test environment - an environment used for testing software changes without affecting production systems
  • synonym: virtual machine - a software emulation of a computer system often used to isolate and run code safely
