unilingual

adjective
one language only
Pronunciation: / ˌjuːnɪˈlɪŋɡwəl / (AmE), / ˌjuːnɪˈlɪŋɡwəl / (BrE)
using or involving only one language

Examples

Unilingual website
The official website is unilingual and provides information only in Spanish.
In some regions, government documents remain strictly unilingual despite a diverse population.
Her school was unilingual, so all classes were taught in German.

Related Terms

  • antonym: bilingual - using or involving two languages
  • synonym: monolingual - using or knowing only one language
  • antonym: multilingual - using or involving several languages
