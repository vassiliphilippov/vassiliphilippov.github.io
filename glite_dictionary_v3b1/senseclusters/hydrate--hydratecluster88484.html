hydrate

verb
initialize or populate data/UI (computing)
7.9
Pronunciation: /ˈhaɪ.dreɪt/ (AmE), /ˈhaɪ.dreɪt/ (BrE)
(in computing) to fill or initialize a data structure, object, or user interface with data retrieved from a source, especially after deserialization or server-side rendering.

Examples

To hydrate app data
After the API call returns, the framework will hydrate the client-side state with user data.
They hydrated the React component tree on the server to enable faster initial rendering.
The script is still hydrating the object after deserialization, restoring references and nested arrays.

Related Terms

  • synonym: initialize - to set up a data structure or variable with an initial value
  • synonym: populate - to fill a data structure or database with data
