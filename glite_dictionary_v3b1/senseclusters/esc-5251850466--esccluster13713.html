Esc

noun
keyboard escape key
Pronunciation: / ɛsk / (AmE), / ɛsk / (BrE)
the Escape key on a computer keyboard, used to exit or cancel actions

Examples

Esc key on keyboard
If a program freezes, try pressing the Esc key to stop it.
She quickly hit Esc to exit the full-screen mode during her presentation.
Many games use Esc to open the pause menu or quit.

Related Terms

  • antonym: Enter - the key used to confirm or submit actions on a keyboard
  • synonym: Escape key - the key on a keyboard labeled 'Esc' used to exit or cancel
