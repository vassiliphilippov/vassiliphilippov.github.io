viewport

noun

Examples

Screen viewport
The website automatically adjusts its layout to fit the viewport of any device.
Designers must consider how images appear within the viewport on different screen sizes.
If the viewport is too small, some content may not be visible without scrolling.
The designer resized the viewport to see how the website would look on different devices.
In 3D modeling software, you can rotate the scene within the viewport to view it from any angle.
Adjusting the viewport settings can improve the performance of graphics applications.
The website layout changes depending on the size of the viewport.
When you zoom in, only part of the image appears in the viewport.
Developers often test how their site looks in different viewports to ensure compatibility.
