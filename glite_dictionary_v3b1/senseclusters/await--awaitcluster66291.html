(in computer programming) to pause the execution of an asynchronous function until a promise or future completes, then continue with the resolved value or error handling.

Examples

To await a promise
In modern JavaScript, you can await a Promise to resolve inside an async function.
The function will await the API response, then update the user interface with the data.
Make sure to handle errors when you await network requests to avoid unhandled rejections.
