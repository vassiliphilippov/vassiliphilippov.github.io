passive matrix display

noun
LCD type with wire matrix control
7.0 7.4
Pronunciation: / ˈpæs.ɪv ˈmeɪ.trɪks dɪˈspleɪ / (AmE), / ˈpæs.ɪv ˈmeɪ.trɪks dɪˈspleɪ / (BrE)
a type of liquid crystal display (LCD) in which rows and columns of parallel wires control the pixels, typically offering lower image quality compared to active matrix displays

Examples

A passive matrix display
Consumers generally prefer active matrix screens over a passive matrix display because of better image clarity.
Many early laptops used a passive matrix display, which resulted in slower response times and lower contrast.
The passive matrix display on this old device can't match modern screen resolutions.

Related Terms

  • antonym: active matrix display - an LCD using a transistor for each pixel, providing higher quality
