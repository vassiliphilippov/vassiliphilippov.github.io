updatable

adjective
able to be updated
Pronunciation: / ʌpˈdeɪtəbəl / (AmE), / ʌpˈdeɪtəbəl / (BrE)
able to be updated or changed to a newer version or state

Examples

An updatable app
The app is updatable so you can always get the latest features.
Only updatable documents can be edited by multiple users online.
Make sure your device is updatable before installing the new software.

Related Terms

  • antonym: fixed - not able to be changed or updated
  • synonym: modifiable - able to be changed or adjusted
