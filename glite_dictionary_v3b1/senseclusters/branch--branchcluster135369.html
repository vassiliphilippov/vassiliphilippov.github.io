branch

noun
separate line of development (version control)
5.7 2.55/m
Pronunciation: / br√¶nt É / (AmE), / br…ëÀênt É / (BrE)
(in software development, version control) a separate line of development in a repository that diverges from the main codebase, used to develop features, fix bugs, or experiment independently.

Examples

Git branch
The team created a new branch to implement the feature without disrupting the main codebase.
Before testing, she pushed her changes to a temporary branch on the remote repository.
When multiple developers work concurrently, regularly merging branches prevents long-lived conflicts and integration problems.

Related Terms

  • synonym: fork - a copy of a repository made to develop independently, often used in open-source projects; related to branching.
  • antonym: main - the primary or central branch of a repository from which other branches diverge (often called 'main' or 'master').
