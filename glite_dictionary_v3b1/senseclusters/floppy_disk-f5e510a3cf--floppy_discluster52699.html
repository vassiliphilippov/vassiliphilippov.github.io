floppy disk

noun
save-icon (software)
3.7
Pronunciation: /ˈflɑːp.ɪ dɪsk/ /; / /ˈflɑːp.iː dɪsk/ (AmE), /ˈflɒp.ɪ dɪsk/ (BrE)
A small graphical icon (derived from the physical floppy disk) used in software interfaces to represent the command to save a file or document.

Examples

Floppy disk icon
On the toolbar, she clicked the floppy disk icon to save her changes before closing.
Even though no one uses them, the small floppy disk symbol still means 'save' to most people.
He taught his grandmother to click the floppy disk button whenever she wanted to keep a document.

Related Terms

  • synonym: icon - a small pictorial symbol used in computer interfaces to represent a function, file, or object
  • synonym: save icon - a graphical button or symbol in a user interface that saves a file or document
