socket

noun
software network endpoint
5.6
Pronunciation: /ˈsɑː.kət/ /; / /ˈsɑː.kɪt/ (AmE), /ˈsɒ.kɪt/ /; / /ˈsɒ.kət/ (BrE)
(computing) a software endpoint provided by an operating system that allows an application to send and receive data over a network, typically using protocols such as TCP or UDP.

Examples

Network socket
The client opens a socket to connect to the server and start data transfer.
After the crash, the service could not bind the socket, causing multiple failed connections.
The professor explained how a TCP socket ensures ordered, reliable data delivery between applications.
