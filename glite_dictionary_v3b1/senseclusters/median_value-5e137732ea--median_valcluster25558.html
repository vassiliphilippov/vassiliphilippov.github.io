median value

noun
middle number in a set of numbers
4.9 3.0
Pronunciation: /ˈmiː.di.ən ˈvæl.juː/ /; / /ˈmiː.dɪ.ən ˈvæl.juː/ (AmE), /ˈmiː.di.ən ˈvæl.juː/ /; / /ˈmiː.dɪ.ən ˈvæl.juː/ (BrE)
the middle number in a set of numbers, or the average of the two middle numbers when the set has an even number of items

Examples

The median value number
If the list has an even number of entries, the median value is the average of the two central numbers.
The median value helps provide a more accurate picture of typical income than the mean in skewed distributions.
To find the median value of the data set, arrange the numbers in order and locate the middle one.
