uninjected

adjective
not one-to-one (math)
9.9
Pronunciation: /ʌn.ɪn.ˈdʒek.tɪd/ (AmE), /ʌn.ɪn.ˈdʒek.tɪd/ (BrE)
not injective; describing a function that maps two or more distinct inputs to the same output

Examples

Uninjected function
The function g(x) = x squared is uninjected because both 3 and -3 give the same result.
If a function is uninjected, it cannot have an inverse that is also a function.
Mathematicians often check whether a mapping is uninjected or injective when analyzing its properties.

Related Terms

  • antonym: injective - describing a function that maps each input to a unique output
  • synonym: many-to-one - a function where multiple inputs can have the same output
