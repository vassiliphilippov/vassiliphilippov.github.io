semantic error

noun
logic or computation error detected at run time
6.1
Pronunciation: / səˈmæntɪk ˈɛrər / (AmE), / sɪˈmæntɪk ˈɛrər / (BrE)
an error in logic or computation that occurs when a program runs, causing it to behave incorrectly despite being syntactically correct

Examples

A semantic error in code
A semantic error in the code caused the program to calculate the wrong result.
Debugging a semantic error can be challenging because the program does not crash but behaves unexpectedly.
While compiling, there were no issues, but a semantic error led to incorrect output at run time.

Related Terms

  • antonym: syntax error - an error in the code's structure or grammar, detected when compiling or interpreting
