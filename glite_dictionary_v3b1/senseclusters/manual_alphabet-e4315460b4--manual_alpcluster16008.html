manual alphabet

noun
set of hand signs for letters
4.8
Pronunciation: / ˈmænjuəl ˈælfəbɛt / (AmE), / ˈmanjʊəl ˈalfəbɛt / (BrE)
a set of hand signs, each representing a letter of the alphabet, used especially by deaf people for spelling words

Examples

The manual alphabet system
Children at the school learn to use the manual alphabet to spell names and unfamiliar words.
She communicated with her deaf friend by fingerspelling using the manual alphabet.
The teacher demonstrated each letter of the manual alphabet to the class.
