grep

noun
text search operation
8.6
Pronunciation: / ɡrɛp / (AmE), / ɡrɛp / (BrE)
a search operation that finds lines in text matching a specified pattern, especially using a command-line tool

Examples

A quick grep
The developer ran a grep to find all instances of the function in the codebase.
A quick grep for the error message helped her locate the problem.
You can use a grep to search through large log files efficiently.
