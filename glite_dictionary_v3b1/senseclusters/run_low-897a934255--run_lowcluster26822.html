run low

verb
to become nearly used up
Pronunciation: / rʌn loʊ / (AmE), / rʌn ləʊ / (BrE)

We need to buy more milk because we're running low at home.
Our patience is running low with these endless delays.

We're running low on milk, so I'll need to buy some more.
