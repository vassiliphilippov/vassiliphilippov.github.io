unmappable

adjective
not able to be represented
Pronunciation: / ʌnˈmæpəbəl / (AmE), / ʌnˈmæpəbəl / (BrE)
not able to be converted into or represented by a particular system, format, or encoding.

Examples

Unmappable character error
Some Unicode symbols are unmappable in the old email system, so they appear as question marks.
The file format was unmappable by the software, resulting in an error message.
If a character is unmappable, it may be replaced with a placeholder during conversion.

Related Terms

  • antonym: convertible - able to be changed into a different form or system
  • synonym: incompatible - not able to work or exist together
