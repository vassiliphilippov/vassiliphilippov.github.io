runtime error

noun
error during program execution
Pronunciation: / ˈrʌnˌtaɪm ˈɛrər / (AmE), / ˈrʌntaɪm ˈɛrə / (BrE)
An error occurring during the execution of a program, often due to invalid operations or conditions not detected before the program runs.

Examples

Runtime error message
Debugging tools can help you identify and fix runtime errors in your code.
If you divide by zero, the program will throw a runtime error.
The software crashed because of a runtime error caused by invalid input.

Related Terms

  • antonym: syntax error - an error in the code detected before execution, typically during compilation
