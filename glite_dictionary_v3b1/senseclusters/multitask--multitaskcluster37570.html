multitask

verb
to run multiple programs
3.8
Pronunciation: /ˈmʌl.tɪ.tæsk/ /; / /ˈmʌl.tiː.tæsk/ (AmE), /ˈmʌl.tɪ.tɑːsk/ /; / /ˈmʌl.tiː.tɑːsk/ (BrE)
to run or allow several computer processes or programs to operate at the same time

Examples

Computers multitask easily
Modern operating systems can multitask, allowing users to run several applications at once.
The server is designed to multitask efficiently, handling many requests simultaneously.
If your computer cannot multitask, you may notice it slows down when opening multiple programs.

Related Terms

  • antonym: single-task - to run only one process or program at a time
