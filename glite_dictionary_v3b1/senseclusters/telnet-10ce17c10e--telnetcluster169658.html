Telnet

noun
remote access protocol
7.0
Pronunciation: / ˈtɛlnɛt / (AmE), / ˈtɛlnɛt / (BrE)
a network protocol (and program) that allows users to connect to and control remote computers over the internet or a local network

Examples

Telnet connection
The technician used Telnet to configure the router from his office.
Many modern systems have replaced Telnet with more secure alternatives.
She learned how to connect to remote servers using Telnet in her networking class.
The administrator used Telnet to troubleshoot the server from his office.
Many older systems still support Telnet for remote management tasks.
Security experts recommend using SSH instead of Telnet because it encrypts data.

Related Terms

  • synonym: FTP - a protocol for transferring files between computers
  • synonym: SSH - a secure protocol for remote computer access
