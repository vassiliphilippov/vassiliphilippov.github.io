parula

noun
MATLAB default colormap (color gradient)
Pronunciation: / pəˈruːlə / (AmE), / pəˈruːlə / (BrE)
A colormap (a continuous color gradient) used in data visualization and plotting; named 'parula' and used as MATLAB's default colormap since 2014.

Examples

The parula colormap
The default MATLAB colormap is parula, which replaced the older 'jet' palette in 2014.
When exporting figures, she changed the line colors to parula to create a subtler gradient.
Many plotting libraries now offer a parula option to give charts a modern, perceptually uniform look.

Related Terms

  • synonym: color palette - a set or scheme of colors used together in a graphic or visualization
  • synonym: colormap - a mapping from numerical values to colors used in data visualization
