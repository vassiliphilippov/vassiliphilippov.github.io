domain

noun
set of input values of a function (math)
5.1
Pronunciation: / doʊˈmeɪn / (AmE), / dəʊˈmeɪn / (BrE)
(Mathematics) The set of all possible values of the independent variable for which a function is defined.

Examples

Function domain
For the function f(x) = 1/x, zero is not included in its domain.
In this equation, the domain is all real numbers greater than zero.
You must specify the domain before graphing the function.
