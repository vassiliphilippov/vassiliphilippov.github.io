lex

noun
lexical analyzer generator
9.9
Pronunciation: / lɛks / (AmE), / lɛks / (BrE)
a computer program that generates lexical analyzers (tokenizers) from pattern specifications, commonly used in compiler construction

Examples

Lex program
The students used lex to create a tokenizer for their compiler assignment.
Many programming courses teach how to use lex for building lexical analyzers.
Before writing the parser, the team generated tokens with lex.

  • synonym: yacc - a computer program that generates parsers from grammar specifications
