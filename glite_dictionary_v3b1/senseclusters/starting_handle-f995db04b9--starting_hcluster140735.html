starting handle

noun
crank used to start an engine manually
Pronunciation: / ˈstɑrtɪŋ ˌhændl̩ / (AmE), / ˈstɑːtɪŋ ˌhændl̩ / (BrE)
a metal crank that is inserted into an engine and turned by hand to start it, especially in older vehicles or machinery

Examples

A starting handle crank
Always keep the starting handle in the glove compartment for emergencies.
Early tractors required a starting handle before they could operate.
The mechanic used the starting handle to get the old car's engine running.
