node

noun
device or computer connected in a network
4.9 4.7
Pronunciation: /noʊd/ (AmE), /nəʊd/ (BrE)
(computing) a device or computer connected as part of a network, capable of sending, receiving, or forwarding information.

Examples

Network node
Every node on the network must have a unique address.
If one node fails, the others can still communicate.
The system administrator added a new node to improve performance.

Related Terms

  • synonym: client - a computer or program that requests services from a server on a network
  • synonym: server - a computer or program that provides services to other computers on a network
