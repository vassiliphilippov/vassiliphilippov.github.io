in memory of

preposition
to honor someone's memory
Pronunciation: / ɪn ˈmɛməri əv / (AmE), / ɪn ˈmɛməri ɒv / (BrE)
done to honor or remember someone who has died

Examples

In memory of someone
A scholarship was established in memory of the late professor.
The park bench was donated in memory of her grandfather.
They held a candlelight vigil in memory of the victims.
