private

adjective
restricted access in programming
5.6
Pronunciation: /ˈpraɪ.vət/ /; / /ˈpraɪ.vɪt/ (AmE), /ˈpraɪ.vət/ /; / /ˈpraɪ.vɪt/ (BrE)
(in computing) restricted so that a program element (for example a variable, method, or member) is accessible only within a specified class, object, or module and not from external code.

Examples

Private variable
To hide the data, declare the member as private so it is inaccessible outside the class.
The developer changed the method to private after discovering it should not be part of the API.
Many languages enforce private visibility so unit tests cannot call internal helper functions directly.

Related Terms

  • synonym: encapsulation - the programming principle of hiding internal details and restricting access to a component.
  • antonym: public - accessible from outside the class or module; the opposite of private in programming.
