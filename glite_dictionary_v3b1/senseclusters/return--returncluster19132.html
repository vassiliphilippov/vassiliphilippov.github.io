return

noun
key for new line or command on keyboard
1.0 2.8
Pronunciation: /rɪ.ˈtɜːrn/ /; / /riː.ˈtɜːrn/ (AmE), /rɪ.ˈtɜːn/ (BrE)
the key on a typewriter or computer keyboard that moves the cursor to a new line or executes a command

Examples

On older typewriters, the return key would physically move the carriage to the next line.
Press the return key to start a new paragraph.
