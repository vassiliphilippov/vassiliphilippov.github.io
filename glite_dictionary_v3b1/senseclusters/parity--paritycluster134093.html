parity

noun
error-checking bit in computing
Pronunciation: / ˈpɛr.ɪ.ti / (AmE), / ˈpær.ɪ.ti / (BrE)
(Computer science) a technique in which an extra bit is added to a group of binary digits to make the total number of 1's either odd or even, used for error detection.

Examples

Parity bit check
Odd parity adds a bit so the sum of ones is always odd.
The system uses parity to detect simple transmission errors in the data.
When the parity check fails, the hardware flags an error.

Related Terms

  • synonym: parity bit - a bit that is added to ensure parity
