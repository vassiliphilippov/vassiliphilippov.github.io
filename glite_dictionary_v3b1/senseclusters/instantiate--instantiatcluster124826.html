instantiate

verb
to create an instance
8.8
Pronunciation: / ɪnˈstænʃiˌeɪt / (AmE), / ɪnˈstænʃieɪt / (BrE)
to create a specific instance of something, especially in computing or programming

Examples

To instantiate an object
Before you can use the object, you must instantiate it in your code.
The software automatically instantiates a new user profile when you sign up.
By instantiating multiple classes, the program can handle different types of data.
In object-oriented programming, you must instantiate a class before you can use its methods.
The software will instantiate a new user profile each time someone registers.
To test the function, the developer decided to instantiate several different objects.
