eyes-only

adjective
restricted, confidential document designation
4.2 7.4
Pronunciation: /aɪz ˈoʊn.liː/ /; / /aɪz ˈəʊn.liː/ (AmE), /aɪz ˈəʊn.liː/ /; / /aɪz ˈəʊ.niː/ (BrE)
(of a document) restricted so that only the person to whom it is addressed may read it; confidential and not for wider circulation.

Examples

An eyes-only document
Access to the eyes-only dossier was strictly limited to the chief investigator.
The ambassador received an eyes-only memo containing sensitive information about the negotiations.
This report is classified as eyes-only for senior management.
