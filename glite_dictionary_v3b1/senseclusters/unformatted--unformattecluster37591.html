unformatted

adjective
not prepared for use
5.1
Pronunciation: /ʌn.ˈfɔːr.mæt.ɪd/ /; / /ʌn.ˈfɔːr.mə.tɪd/ (AmE), /ʌn.ˈfɔː.mæt.ɪd/ /; / /ʌn.ˈfɔː.mə.tɪd/ (BrE)
not prepared with a file system or structure, so unable to store or organize files until formatted

Examples

Unformatted hard drive
The external drive was unformatted and needed to be set up before use.
If a disk is unformatted, your computer will not recognize it for saving files.
