vectorize

verb
to use vector operations
Pronunciation: / ˈvɛktəˌraɪz / (AmE), / ˈvɛktəˌraɪz / (BrE)
to convert computer code or data so that it uses vector operations or SIMD instructions for improved performance

Examples

To vectorize code
The software can automatically vectorize certain mathematical operations to speed up calculations.
Engineers worked to vectorize the image processing code for better performance on modern hardware.
If you vectorize your data, the program will run much faster on compatible processors.

Related Terms

  • synonym: parallelize - to make code run in parallel on multiple processors or cores
  • antonym: serialize - to process data or code in a single sequence rather than in parallel
