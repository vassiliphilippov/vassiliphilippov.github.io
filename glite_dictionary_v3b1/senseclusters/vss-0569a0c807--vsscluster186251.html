VSS

proper noun
Windows backup feature
Pronunciation: / vi ɛs ɛs / (AmE), / viː ɛs ɛs / (BrE)
a Windows feature called Volume Shadow Copy Service that creates backup snapshots of files or volumes

Examples

Windows VSS backup
Before updating the system, Windows created a VSS snapshot to protect important files.
The IT administrator used VSS to recover lost documents after a software crash.
Many backup programs rely on VSS to ensure files are safely copied even while in use.

Related Terms

  • synonym: backup - a copy of data kept in case the original is lost or damaged
  • synonym: restore - to return data or a system to a previous state
