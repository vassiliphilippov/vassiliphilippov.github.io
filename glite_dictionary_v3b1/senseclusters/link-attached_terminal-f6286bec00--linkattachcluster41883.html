A computer terminal that is connected to a main computer or network by means of a data link rather than direct or local connection.

Examples

Link-attached terminal device
Our office uses link-attached terminals to access the central database remotely.
The new system replaces standalone PCs with link-attached terminals for greater security.
With a link-attached terminal, employees can log in to the server from different locations.

Synonyms

  • link-attached station
    7.4 0.00/m
    a link-attached station is a network endpoint on a data link, while a link-attached terminal is the user device connected to that link.
  • remote terminal
    5.8 0.02/m
    a remote terminal lets a user access a computer from far away over a data link, while a link-attached terminal highlights connection by that link.
  • remote station
    4.6 0.00/m
    a remote station sits away from the main computer and connects by a data link, while a link-attached terminal names the terminal on that link.
