setuid

adjective
with owner privileges
9.9
Pronunciation: /ˈset.juː.ɪd/ /; / /ˈset.juːd/ /; / /ˈset.juː.aɪ.diː/ (AmE), /ˈset.juː.ɪd/ /; / /ˈset.juːd/ /; / /ˈset.juː.aɪ.diː/ (BrE)
having a special permission that allows a program or file to run with the privileges of its owner rather than the user who runs it

Examples

Setuid program
Only trusted utilities should be installed as setuid executables on a multi-user system.
A setuid program can be a security risk if not properly managed.
The administrator checked which files were marked as setuid to ensure system safety.

Related Terms

  • synonym: executable - a file that can be run as a program
  • synonym: setgid - having a permission that allows execution with the group owner's privileges
