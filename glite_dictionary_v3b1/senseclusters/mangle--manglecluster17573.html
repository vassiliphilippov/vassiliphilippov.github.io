mangle

verb
transform or encode identifiers (computing)
8.9
Pronunciation: /ˈmæŋ.gəl/ (AmE), /ˈmæŋ.gəl/ (BrE)
(in computing) to change, encode, or shorten program identifiers or symbol names (often automatically) so they include extra information or become suitable for linking, obfuscation, or size reduction.

Examples

To mangle names
Modern compilers mangle C++ function names to include parameter types for linking.
The build tool mangled long identifiers to shorten symbol tables and reduce binary size.
Security researchers noticed obfuscated code where attackers had deliberately mangled variable names.

Related Terms

  • synonym: encode - to convert information into a particular form, often for storage or transmission
  • synonym: obfuscate - to make something (such as code) deliberately unclear or difficult to understand
