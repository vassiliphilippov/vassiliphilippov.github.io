northbound

adjective
interface/API toward higher-level systems
5.9
Pronunciation: /ˈnɔːrθ.baʊnd/ (AmE), /ˈnɔːθ.baʊnd/ (BrE)
(in computing and networking) describing an interface, API, or data flow that connects toward higher-level systems, controllers, or external applications.

Examples

Northbound API
The SDN controller exposes a northbound API that application developers use to manage network behavior.
Engineers documented the northbound interfaces to show how services request data from the control plane.
During integration testing, we verified the northbound protocol for compatibility with third-party apps.

Related Terms

  • synonym: API - application programming interface; a set of rules that allows software components to communicate (often described as northbound when it serves higher layers).
  • antonym: southbound - an interface or API that connects to lower-level systems or devices (opposite direction in network architectures).
