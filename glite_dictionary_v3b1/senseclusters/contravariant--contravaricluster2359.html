contravariant

adjective
reversing type relationship
9.9
Pronunciation: / ˌkɑːntrəˈvɛriənt / (AmE), / ˌkɒntrəˈveəriənt / (BrE)
in type theory and programming, describing a type or parameter that accepts supertypes, reversing the usual subtype relationship

Examples

Contravariant parameter
In some programming languages, function parameter types can be contravariant to allow greater flexibility.
The concept of contravariant types is important when designing safe and reusable code.
Understanding when a parameter is contravariant helps prevent type errors in complex systems.

Related Terms
