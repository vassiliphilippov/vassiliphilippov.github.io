listen

verb
monitor for network connections
6.7
Pronunciation: /ˈlɪs.ən/ (AmE), /ˈlɪs.ən/ (BrE)
(in computing) to monitor a network socket, port, or interface for incoming connections, data, or events, typically performed by a server or background process.

Examples

Server listens for connections
The server will listen for incoming connections on port 443 throughout the night.
My web process listens on localhost so developers can test API requests easily.
After restart, the daemon began listening on multiple ports to handle new client requests.

Related Terms

  • synonym: monitor - to observe or check activity or status (e.g., of a network or process)
