freezable

adjective
able to be suspended
4.3
Pronunciation: /ˈfriː.zə.bəl/ /; / /ˈfriːz.ə.bəl/ (AmE), /ˈfriː.zə.bəl/ /; / /ˈfriːz.ə.bəl/ (BrE)
able to be suspended or temporarily made inactive, especially in computing processes or program states.

Examples

Freezable app state
The application must be freezable so that updates can be applied without losing data.
Developers check if a process is freezable before attempting to pause it during maintenance.
Not all system threads are freezable, which can complicate software upgrades.
