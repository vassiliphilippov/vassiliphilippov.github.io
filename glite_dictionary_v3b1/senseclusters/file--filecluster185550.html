file

noun
line of people or things
2.8 4.29/m
Pronunciation: / faÉªl / (AmE), / faÉªl / (BrE)
A line of people or things arranged one behind another.

Examples

A file of people
A long file of cars waited at the border crossing.
Children walked in file down the corridor to their classrooms.
The soldiers marched in a single file through the narrow pass.

Synonyms

  • single file
    2.8 1.11/m
    single file is a line with each person directly behind the next, while file is the general name for such an arrangement.
  • Indian file
    7.6 0.00/m
    Indian file is a set phrase for people walking one behind another, while file is the simple, general word for that arrangement.
  • column
    2.5 2.88/m
    column is a formal line often of vehicles or troops, while file is a general term for any one-behind-another arrangement.
  • line
    1.0 50/m
    line can mean a row side-by-side or an organized formation, while file always means a one-behind-another narrow line.
