Wayland

proper noun
Linux display server protocol
9.5
Pronunciation: /ˈweɪ.lənd/ /; / /ˈweɪ.lænd/ (AmE), /ˈweɪ.lənd/ /; / /ˈweɪ.lænd/ (BrE)
A display server protocol and reference compositor for Linux and other Unix-like systems, intended to replace the older X Window System (X11).

Examples

Wayland display server
Many Linux distributions now use Wayland as their default display server instead of X11.
The graphics driver caused flicker only under Wayland, but it worked fine with the X server.
Developers praised Wayland for its simpler security model and smoother frame timing on modern desktops.

Related Terms

  • synonym: X11 - the X Window System, an older display server protocol commonly used on Unix-like systems.
  • synonym: display server - software that communicates between graphical applications and the display hardware, managing windows and input events.
