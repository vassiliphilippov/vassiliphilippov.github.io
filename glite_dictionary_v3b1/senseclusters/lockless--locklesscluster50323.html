lockless

adjective
without locks (computing)
8.2
Pronunciation: / ˈlɑːkləs / (AmE), / ˈlɒkləs / (BrE)
in computing, allowing safe concurrent access without using mutual-exclusion locks, often by using atomic operations or nonblocking synchronization

Examples

Lockless algorithm
The new database uses a lockless queue to improve performance under heavy load.
Developers often choose lockless algorithms to avoid bottlenecks in multithreaded applications.
A lockless data structure can help prevent deadlocks in concurrent systems.

Related Terms

  • synonym: lock-free - not requiring locks for safe concurrent access
  • antonym: synchronized - using locks or other mechanisms to control concurrent access
