inflater

noun
software decompressor (deflate)
7.2
Pronunciation: / ɪnˈfleɪtər / (AmE), / ɪnˈfleɪtər / (BrE)
(in computing) a software component or library module that decompresses data previously compressed (often by the 'deflate' algorithm), restoring the original uncompressed data.

Examples

ZIP file inflater
The ZIP library's inflater processed the compressed stream and returned the original file data.
Developers added multiple inflaters to handle different compression formats on the server.
When the archive was corrupted, the forensic tool's inflater still recovered several images.

Related Terms

  • synonym: decompressor - a program or component that restores compressed data to its original form
  • antonym: deflate - to compress data (often using the DEFLATE algorithm); the opposite operation to inflating/decompressing
