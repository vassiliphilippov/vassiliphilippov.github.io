public address system

noun
system for amplifying speech in public
1.0
Pronunciation: / ˌpʌblɪk əˈdrɛs ˌsɪstəm / (AmE), / ˌpʌblɪk əˈdrɛs ˌsɪstəm / (BrE)
An electronic system with microphones, amplifiers, and loudspeakers used to make spoken announcements audible in public areas.

Examples

The public address system
Emergency instructions were broadcast over the public address system during the fire drill.
Please ensure the public address system is working before the event begins.
The school principal used the public address system to make an important announcement.
