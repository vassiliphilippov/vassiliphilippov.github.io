cardinal

noun
number denoting quantity in a set (math)

Pronunciation: / Ààk…ërd…™n…ôl / (AmE), / Ààk…ëÀêd…™n…ôl / (BrE)
(Mathematics) A number representing the quantity of elements in a set, irrespective of their order.

Examples

Cardinal of a set
In set theory, the cardinal of a finite set is simply its number of elements.
Mathematicians use the concept of cardinal to compare the sizes of infinite sets.
The cardinal of the empty set is zero.

Synonyms

  • cardinal numeral
