linkable

adjective
able to be linked
Pronunciation: / ˈlɪŋkəbəl / (AmE), / ˈlɪŋkəbəl / (BrE)
able to be combined with other code or modules by a linker to form an executable or library

Examples

Linkable code modules
The software must be linkable with external libraries to function correctly.
Developers checked if the new module was linkable before integrating it into the main project.
If a file is not linkable, the build process will fail with an error.

Related Terms

  • synonym: compatible - able to work together without conflict
  • antonym: incompatible - not able to work together or be combined
