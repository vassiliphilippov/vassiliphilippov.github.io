spa

noun
single-page application (web architecture)
Pronunciation: / spɑ / (AmE), / spɑː / (BrE)
(in computing) a web application architecture in which a single HTML page is loaded and subsequent user interactions update content dynamically without full page reloads; short for single-page application.

Examples

Modern spa website
Many companies build a faster, app-like website using a spa that updates content without full reloads.
When the server is unreachable, the browser can still show cached data in a spa for offline use.
Testing frameworks make it easier to debug interactions inside a complex spa during development.

Related Terms

  • synonym: single-page application - a web application that runs on a single page and updates content dynamically, often using client-side routing
