decompile

verb
to reverse compiled code
8.1
Pronunciation: / ˌdiːkəmˈpaɪl / (AmE), / ˌdiːkəmˈpaɪl / (BrE)
to convert compiled computer code back into a human-readable programming language or form for analysis or modification

Examples

To decompile software code
The security analyst needed to decompile the application to check for vulnerabilities.
It is often illegal to decompile commercial software without permission.
Developers sometimes decompile code to understand how a program works internally.
The security expert needed to decompile the application to check for hidden vulnerabilities.
After hours of work, she managed to decompile the program and study its logic.
Some tools can decompile software, but the results may not match the original source code exactly.

Related Terms

  • antonym: compile - to convert source code into machine code
  • synonym: disassemble - to break down machine code into assembly language
