stillborn project

noun
project that fails before starting
4.0
Pronunciation: / ˈstɪlˌbɔrn ˈprɑːdʒɛkt / (AmE), / ˈstɪlˌbɔːn ˈprɒdʒɛkt / (BrE)
A project that fails before it even begins or has a chance to develop.

Examples

A stillborn project
Many stillborn projects can be traced back to unrealistic expectations and poor preparation.
The lack of stakeholder engagement rendered the new software initiative a stillborn project.
Without adequate support, the stillborn project never moved beyond the planning stage.

Related Terms

  • synonym: failed project - a project that does not achieve its objectives
