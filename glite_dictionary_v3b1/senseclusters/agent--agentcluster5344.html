agent

noun
autonomous software program
6.6
Pronunciation: /ˈeɪ.dʒənt/ (AmE), /ˈeɪ.dʒənt/ (BrE)
a computer program that acts autonomously to perform tasks or services on behalf of a user, system, or other program.

Examples

Software agent
The virtual assistant functions as an intelligent agent that schedules meetings and sends reminders.
Several monitoring agents run on the server to detect faults before they affect users.
Security researchers studied a malicious agent that automatically steals login credentials from browsers.

Related Terms

  • synonym: autonomous agent - a program or system that operates without continuous human control to achieve goals.
  • synonym: bot - an automated program that performs tasks, often on the internet (e.g., chatbots, web crawlers).
