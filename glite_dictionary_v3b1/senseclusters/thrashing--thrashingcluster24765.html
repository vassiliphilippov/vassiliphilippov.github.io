(computing) a system performance problem in which an operating system or process spends excessive time paging or swapping memory, causing severe slowdowns and reduced throughput.

Examples

System thrashing problem
After the memory leak, the database server suffered severe thrashing and became nearly unresponsive.
System administrators diagnosed constant thrashing as processes swapped in and out of RAM endlessly.
Enabling more physical memory eliminated the thrashing that had crippled the virtual machines during peak load.

Related Terms

  • synonym: paging - the process of moving pages of memory between physical RAM and disk (swap) used by operating systems
  • synonym: swap - to move data between RAM and disk (swap space), often causing delays when overused
