backfire

noun
deliberately set fire to stop a wildfire
7.5
Pronunciation: / ˈbækˌfaɪɚ / (AmE), / ˈbækˌfaɪə / (BrE)
A fire that is deliberately set to stop an approaching wildfire by burning off fuel in its path.

Examples

A wildfire backfire
A carefully controlled backfire helped contain the rapidly advancing grassfire.
The firefighters set a backfire to prevent the main blaze from spreading to nearby homes.
Using a backfire is a risky but sometimes necessary tactic in wildfire management.
