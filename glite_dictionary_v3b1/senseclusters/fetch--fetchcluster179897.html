fetch

verb
retrieve data from a server
4.9
Pronunciation: / fɛtʃ / (AmE), / fɛtʃ / (BrE)
(computing) to request and retrieve data or a resource from a remote server, network, or API.

Examples

To fetch data
The web app will fetch the user data from the server via the public API.
When the page loads, the script fetches a list of recent posts and displays them.
After the outage, engineers were fetching logs from multiple servers to diagnose the failure.

Related Terms

  • synonym: download - to copy data from a remote computer or server to a local device
  • synonym: retrieve - to get something back or obtain data from storage or a remote source
