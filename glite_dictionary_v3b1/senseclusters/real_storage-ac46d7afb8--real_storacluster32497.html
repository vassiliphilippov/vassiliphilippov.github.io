real storage

noun
main memory in a virtual memory system
Pronunciation: / ˈriəl ˈstɔrɪdʒ / (AmE), / ˈrɪəl ˈstɔːrɪdʒ / (BrE)
The physical main memory in a computer's virtual memory system, as opposed to virtual or secondary storage.

Examples

The real storage memory
The operating system allocates real storage to ensure that critical processes have enough memory to run efficiently.
Upgrading your laptop increases the amount of real storage available for applications and the operating system.
When real storage is exhausted, the system relies more on virtual memory, which can slow down performance.

Related Terms

  • antonym: virtual memory - memory management technique using addresses beyond physical memory
