multigraph

noun
graph with multiple edges
8.4
Pronunciation: /ˈmʌl.tɪ.ɡræf/ /; / /ˈmʌl.tə.ɡræf/ (AmE), /ˈmʌl.tɪ.ɡræf/ /; / /ˈmʌl.tə.ɡræf/ (BrE)
a graph in which two nodes can be connected by more than one edge

Examples

A multigraph with double edges
A multigraph can represent transportation networks where multiple routes connect the same cities.
In some problems, using a multigraph makes it easier to model repeated relationships.
Unlike a simple graph, a multigraph allows several edges between the same pair of vertices.

Related Terms

  • synonym: graph - a set of nodes connected by edges
  • antonym: simple graph - a graph with at most one edge between any two nodes
