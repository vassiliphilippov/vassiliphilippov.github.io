multicore

adjective
with multiple cores
4.8
Pronunciation: / ˈmʌltiˌkɔr / (AmE), / ˈmʌltiˌkɔː / (BrE)
having or involving more than one processing unit or core, especially in a computer

Examples

Multicore processor
Most modern smartphones use multicore processors to handle multiple tasks efficiently.
The new software is optimized for multicore systems, allowing faster data processing.
Upgrading to a multicore computer can improve performance for demanding applications.

Related Terms

  • degree lower: dual-core - having two processing units or cores
  • degree higher: quad-core - having four processing units or cores
  • antonym: single-core - having only one processing unit or core
