reassign

verb
assign a new value (computing)
4.2
Pronunciation: /riː.ə.ˈsaɪn/ (AmE), /riː.ə.ˈsaɪn/ (BrE)
(in computing) to assign a new value to an existing variable, name, or symbol, replacing its previous value.

Examples

To reassign a variable
In Python, you can reassign the variable to reflect the latest result.
After debugging, she reassigned the counter to zero to restart the loop from the beginning.
The documentation recommends reassigning configuration values only through the provided API to avoid errors.
