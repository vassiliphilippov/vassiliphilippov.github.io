endpoint

noun
network address for a web API
4.2
Pronunciation: /ˈend.pɔɪnt/ (AmE), /ˈend.pɔɪnt/ (BrE)
(in computing) a specific network address or URL provided by a web service or API that clients call to access a resource or perform an action.

Examples

API endpoint
Before release, the team tested the new endpoint to ensure correct authentication and responses.
You can call the endpoint with curl to retrieve JSON-formatted user profile data for debugging.
The mobile app communicates with a secure endpoint that returns product availability and pricing information.
