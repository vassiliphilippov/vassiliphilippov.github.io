callable

noun
programmable object that can be invoked
7.7
Pronunciation: /ˈkɔː.lə.bəl/ /; / /ˈkɑː.lə.bəl/ (AmE), /ˈkɔː.lə.bəl/ (BrE)
(computing) an object (for example, a function, method, or object implementing a call operator) that can be invoked or called to perform an action or return a value.

Examples

Python callable
In Python, a callable is any object that can be invoked with parentheses to run code.
The library's API accepts a callable parameter to customize how each item is processed asynchronously.
To test, she passed a simple callable returning a fixed value instead of a complex service.

Related Terms

  • synonym: function - a named block of code that performs a task and can be called or invoked
  • synonym: method - a function associated with an object or class, callable using object syntax
