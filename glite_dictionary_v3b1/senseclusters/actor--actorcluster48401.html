actor

noun
concurrent computational unit
8.5
Pronunciation: /ˈæk.tər/ (AmE), /ˈæk.tə/ (BrE)
(computing) a unit of computation that has its own state and behavior and communicates with other units by sending and receiving asynchronous messages.

Examples

Software actor
In modern distributed systems, an actor processes messages asynchronously across components.
Each actor maintains its own state and responds to incoming messages independently.
We implemented several actors to handle user sessions and scale request handling.

Related Terms

  • synonym: actor model - a model of concurrent computation in which actors are the fundamental units that communicate by message-passing
  • synonym: agent - an autonomous computational entity that performs actions and may communicate with others
