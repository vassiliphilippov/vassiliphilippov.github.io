Justinian code

noun
Roman civil law code under Justinian I
9.3
Pronunciation: /dʒʌs.ˈtɪn.iː.ən koʊd/ (AmE), /dʒʌs.ˈtɪn.iː.ən kəʊd/ (BrE)
