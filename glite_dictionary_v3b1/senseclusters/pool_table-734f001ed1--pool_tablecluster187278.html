pool table

noun
table used for playing pool
Pronunciation: / ˈpul ˌteɪbəl / (AmE), / ˈpuːl ˌteɪbl / (BrE)
A heavy, flat table with pockets, used for playing the game of pool.

Examples

A pool table game
He chalked his cue and circled the pool table before taking his shot.
The old pool table in the bar had worn green felt but was still popular.
We bought a pool table for our basement so we could play with friends.
