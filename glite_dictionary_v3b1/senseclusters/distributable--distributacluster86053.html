distributable

adjective
able to be spread
6.6
Pronunciation: / dɪˈstrɪbjətəbl / (AmE), / dɪˈstrɪbjʊtəbl / (BrE)
able to be deployed, spread, or operated across multiple computers, servers, or locations for performance or reliability.

Examples

Distributable software
The software is distributable so it can run on several servers at once.
A distributable application helps improve system reliability by spreading the workload.
They designed the database to be distributable for better performance and fault tolerance.

Related Terms

  • antonym: centralized - concentrated in a single location or system
  • synonym: distributed - spread or shared across multiple locations or systems
