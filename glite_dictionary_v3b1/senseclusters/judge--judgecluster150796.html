judge

noun
automated evaluation system
6.5
Pronunciation: / dʒʌdʒ / (AmE), / dʒʌdʒ / (BrE)
(in computing) a software program or automated system that evaluates, scores, or ranks submissions (for example, code or data) by running tests or applying rules.

Examples

Online judge
The platform's judge automatically scores each code submission against hidden test cases.
During the contest, several teams complained that the judge rejected correct solutions for edge cases.
Many universities use an automated judge to give instant feedback on students' programming assignments.

Related Terms

  • synonym: automated grader - a program that automatically assesses students' code or assignments
  • synonym: online judge - a web-based system that runs and scores programming contest submissions
