virtualization

noun
creating virtual resources
Pronunciation: / ˌvɝːtʃuələˈzeɪʃən / (AmE), / ˌvɜːtʃuəlaɪˈzeɪʃən /; / ˌvɜːtjʊəlaɪˈzeɪʃən / (BrE)
the process of creating virtual versions of computer resources, such as servers, storage, or networks, on a single physical machine

Examples

Server virtualization
Many companies use virtualization to run multiple operating systems on a single server.
By adopting virtualization, the IT department reduced hardware costs and improved flexibility.
Cloud computing often relies on virtualization to provide scalable and efficient services.
Many companies use virtualization to make better use of their servers.
With virtualization, you can run several operating systems on one computer at the same time.
The IT department improved efficiency by implementing virtualization across their network.

Related Terms

  • synonym: containerization - the use of containers to run applications in isolated environments
  • synonym: emulation - the imitation of one system by another
  • antonym: physicalization - the use of actual physical hardware instead of virtual resources
