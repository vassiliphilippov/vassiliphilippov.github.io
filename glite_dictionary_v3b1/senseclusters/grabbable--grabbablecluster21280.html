grabbable

adjective
able to be moved
4.3
Pronunciation: /ˈgræb.ə.bəl/ (AmE), /ˈgræb.ə.bəl/ (BrE)
able to be picked up, selected, or moved by a mouse pointer, touch, or other input device in a user interface.

Examples

Grabbable icon
The toolbar is grabbable, so you can move it anywhere on the screen.
If the icon is not grabbable, you will not be able to rearrange it.
Many modern apps use grabbable elements to make customization easier for users.
