batting machine

noun
machine for separating textile fibers
Pronunciation: / ˈbæɾɪŋ məˈʃin / (AmE), / ˈbætɪŋ məˈʃiːn / (BrE)
a machine used to separate fibers from stems and impurities in the processing of cotton, silk, or flax

Examples

A batting machine for cotton
Regular maintenance of the batting machine is essential for efficient textile production.
The factory recently installed a new batting machine to improve fiber quality.
This batting machine can process large quantities of cotton in a single shift.
