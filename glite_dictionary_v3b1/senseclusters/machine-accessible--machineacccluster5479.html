machine-accessible

adjective
able to be accessed by a computer system
Pronunciation: / məˈʃin əkˈsɛsəbəl / (AmE), / məˈʃiːn əkˈsɛsəbl̩ / (BrE)
(of information or a system) capable of being stored in, controlled by, or directly accessed by a computer or other machine.

Examples

Machine-accessible data
Only machine-accessible data can be used in these automated systems.
The documents were converted to a machine-accessible format for faster processing.
The new database ensures all records are machine-accessible at any time.
