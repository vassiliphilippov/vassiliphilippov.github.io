code

verb
convert into symbols or code
Pronunciation: / koʊd / (AmE), / kəʊd / (BrE)
to convert information or a message into a system of symbols, numbers, or signs, often for secrecy or computer processing

Examples

To code a message
All the important messages were coded before being sent to headquarters.
She learned how to code sensitive data to protect users' privacy.
The instructions were coded so that only authorized staff could understand them.

Related Terms

  • antonym: decode - convert coded information back into ordinary language
  • synonym: encode - convert information into code, especially for secrecy or transmission
