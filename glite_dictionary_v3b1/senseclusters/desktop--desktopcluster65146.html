desktop

noun
cloud-hosted virtual desktop
3.5
Pronunciation: /ˈdesk.tɑːp/ (AmE), /ˈdesk.tɒp/ (BrE)
A virtual desktop environment hosted on a remote server or in the cloud and accessed over a network, giving a user a familiar desktop interface, applications, and files as if on a local machine.

Examples

Cloud desktop access
The IT department gave employees a cloud-hosted desktop accessible from any device.
During the pandemic, many companies switched to a virtual desktop so staff could use standardized apps remotely.
She logged into the secure desktop and found all her files and business apps ready.

Related Terms

  • synonym: remote desktop - access to a desktop environment or computer over a network
  • synonym: virtual desktop - a desktop environment that runs remotely on a server and is presented to a user's device
