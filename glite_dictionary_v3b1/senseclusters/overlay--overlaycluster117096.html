overlay

noun
virtual network layer
6.1
Pronunciation: /ˈoʊ.vər.leɪ/ /; / /ˈəʊ.vər.leɪ/ (AmE), /ˈəʊ.və.leɪ/ (BrE)
(in computer networking) a virtual network built on top of another network; it provides routing or other services without changing the underlying physical network.

Examples

Overlay network
The service runs on an overlay network that isolates it from the underlying Internet.
Engineers built a secure overlay across several data centers to manage traffic.
Many peer-to-peer apps use an overlay to route messages without changing hardware.

Related Terms

  • synonym: overlay network - a virtual network created on top of another network to provide routing or services
  • antonym: underlay - the physical or underlying network that provides connectivity for an overlay
