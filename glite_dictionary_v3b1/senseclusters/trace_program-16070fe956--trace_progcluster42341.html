trace program

noun
utility that shows a program's execution steps
5.2
Pronunciation: / treɪs ˈproʊˌɡræm / (AmE), / treɪs ˈprəʊˌɡræm / (BrE)
a computer utility that displays the sequence and results of executing the instructions in another program, often for debugging purposes

Examples

A trace program utility
A trace program can help beginners understand how their algorithms execute step by step.
Developers often use a trace program to monitor the flow of complex applications.
The trace program revealed exactly where the error occurred in the code.
