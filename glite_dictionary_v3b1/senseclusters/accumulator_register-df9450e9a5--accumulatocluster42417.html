accumulator register

noun
CPU register for arithmetic operations
5.7
Pronunciation: / əˈkjuːmjəˌleɪtər ˈrɛdʒɪstər / (AmE), / əˈkjuːmjʊˌleɪtə ˈrɛdʒɪstə / (BrE)
(in computing) a register in a central processing unit (CPU) that stores the results of arithmetic or logical operations, typically by adding an input value to its existing contents.

Examples

The accumulator register stores results
Modern CPUs often use multiple accumulator registers to enhance parallel processing.
The accumulator register plays a vital role in performing calculations within the processor.
Understanding how an accumulator register works is fundamental to learning computer architecture.
