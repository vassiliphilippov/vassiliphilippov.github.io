stochastic variable

noun
variable representing a random value
Pronunciation: / stəˈkæstɪk ˈvɛriəbl̩ / (AmE), / stəˈkæstɪk ˈvɛːriəbl̩ / (BrE)
A variable whose possible values are outcomes of a random phenomenon; a random variable.

Examples

A stochastic variable outcome
Each stochastic variable has an associated probability distribution.
In probability theory, a stochastic variable is used to model uncertainty in experiments.
The value of a stochastic variable may change each time an experiment is performed.

Related Terms

  • synonym: random variable - a variable whose values depend on outcomes of a random phenomenon
