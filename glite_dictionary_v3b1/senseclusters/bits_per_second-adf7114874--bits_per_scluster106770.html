A unit measuring the speed at which data is transmitted, equal to the number of bits transferred every second.

Examples

Millions of bits per second
Data is transmitted between devices in millions of bits per second on modern networks.
Older modems operated at only 56,000 bits per second.
The internet connection has a speed of 100,000 bits per second.

Synonyms

  • bps
    3.8 0.08/m
    bps is the short form shown on technical charts and settings, while bits per second names the same unit in full.
  • bitrate
    4.2 0.08/m
    bitrate refers to the measured speed in digital systems, while bits per second is the exact unit used to express that speed.
  • data rate
    3.4 0.75/m
    data rate describes how fast information moves across a link in general, while bits per second names the specific unit of that speed.
