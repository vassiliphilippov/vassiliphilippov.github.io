monospace

noun
equal-width font
5.8
Pronunciation: /ˈmoʊ.nə.speɪs/ /; / /ˈmoʊ.noʊ.speɪs/ (AmE), /ˈmɒn.ə.speɪs/ /; / /ˈməʊ.nə.speɪs/ (BrE)
a typeface or font in which every character has the same horizontal width

Examples

A monospace font
Programmers often prefer a monospace font because it makes code easier to read.
The document was formatted in a monospace typeface to align all the columns neatly.
When writing scripts, using a monospace font helps keep the text organized.

Related Terms

  • antonym: proportional - a font in which characters have different widths
  • synonym: typewriter - a machine that prints letters of equal width, often using monospace fonts
