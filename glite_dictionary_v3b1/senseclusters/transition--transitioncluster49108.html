transition

noun
CSS animation of property changes
Pronunciation: / trænˈzɪʃ.ən / (AmE), / trænˈzɪʃ.ən / (BrE)
(in web design) a CSS feature that animates changes to CSS property values over a set duration, producing a smooth change between states.

Examples

CSS transition
The menu fades in with a transition effect when hovered.
You can adjust the transition duration in CSS to make hover effects feel smoother.
The developer prefers CSS transitions over JavaScript for simple property changes.

Related Terms

  • synonym: CSS animation - a CSS module using keyframes for more complex or controlled animations
  • synonym: animation - a general technique for creating motion or gradual change over time in digital content
