matcher

noun
software component that finds matches
6.4
Pronunciation: /ˈmætʃ.ər/ (AmE), /ˈmætʃ.ə/ (BrE)
(in computing) a software component or algorithm that identifies matches between inputs and patterns or candidate items, used in pattern matching, information extraction, or record linkage.

Examples

Pattern matcher software
The text processor uses a matcher to locate dates in the document.
Our email system's spam filter uses a rule-based matcher to flag suspicious messages each hour.
During testing, engineers compared two matchers to see which found duplicate records more accurately and faster.

  • synonym: pattern matcher - software that searches text or data for patterns or templates
