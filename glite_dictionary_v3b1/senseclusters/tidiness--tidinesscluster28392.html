tidiness

noun
organization of data into tidy format
4.5
Pronunciation: / ˈtaɪ.di.nəs / (AmE), / ˈtaɪ.di.nəs / (BrE)
(in data analysis) the state or quality of arranging data in a 'tidy' format, where each variable is a column and each observation is a row.

Examples

Data tidiness
Data scientists insist on tidiness so each variable occupies one column for reproducible analysis.
The instructor emphasized tidiness when teaching how to reshape messy spreadsheets into tidy tables.
After importing the CSV, she checked tidiness to ensure each observation was in a separate row.

Related Terms

  • synonym: data cleaning - the process of detecting and correcting or removing inaccurate, corrupted, or irrelevant data
  • antonym: messiness - the state in which data are disorganized or inconsistently structured, making analysis difficult
  • synonym: tidy data - a data format where each variable forms a column and each observation a row
