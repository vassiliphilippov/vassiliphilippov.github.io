list box

noun
GUI selection element
3.7
Pronunciation: / lɪst bɑks / (AmE), / lɪst bɒks / (BrE)
a graphical user interface element that displays a list of items from which a user can select one or more options

Examples

Click on the list box
To choose your country, click on the list box and scroll through the options.
The software update added a new list box for selecting multiple file types.
You can select several items at once by holding down Ctrl while clicking in the list box.
