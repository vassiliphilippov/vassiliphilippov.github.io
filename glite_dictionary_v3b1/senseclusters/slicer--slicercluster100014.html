slicer

noun
dashboard filter widget
7.8
Pronunciation: / c8sla6as59r / (AmE), / c8sla6as59 / (BrE)
(in data-analysis and dashboard software) a control or widget that filters a dataset by selecting categories or values and updates linked visuals or tables.

Examples

Dashboard slicer
She added a slicer to the dashboard so users could filter sales by product category quickly.
When he clicked the slicer, all related charts updated to show only the chosen region's data.
The analyst exported the report after configuring the slicer to display last quarter's results.

Related Terms

  • synonym: dashboard control - interactive elements on a dashboard used to change the data view
  • synonym: filter - a control that restricts data to items meeting selected criteria
