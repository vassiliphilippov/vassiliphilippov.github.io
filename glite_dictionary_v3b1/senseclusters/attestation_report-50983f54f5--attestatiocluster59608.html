attestation report

noun
digital proof of system state
7.5 0.00/m
Pronunciation: / \u0259\u02cct\u025bst\u0259\u02c8\u0283\u0259n r\u026a\u02c8p\u0254rt / (AmE), / \u0259\u02cct\u025bst\u0259\u02c8\u0283\u0259n r\u026a\u02c8p\u0254\u02d0t / (BrE)
In computing and cybersecurity, a digital document (often cryptographically signed) produced by a device, secure hardware module, or system component that reports or proves the system's measured hardware, firmware, or software state to a remote verifier.

Examples

Digital attestation report
The cloud provider issued an attestation report confirming the VM's measured boot and patch level.
Before onboarding, customers request attestation reports to verify that servers run approved firmware.
The security team automated attestation report generation each time a device completed a secure boot.

Related Terms

  • synonym: remote attestation - the process by which a device or platform proves its measured state to a remote verifier
  • related: trusted platform module (TPM) - a secure hardware component that can measure and report system state and often generates attestation reports
