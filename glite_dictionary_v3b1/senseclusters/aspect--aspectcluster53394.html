aspect

noun
module for cross-cutting concerns (AOP)
7.7
Pronunciation: / ˈæspɛkt / (AmE), / ˈæspekt / (BrE)
(in software engineering) a modular unit that encapsulates cross-cutting concerns (for example logging, security, or transactions) and can be applied across different parts of a program; used especially in aspect-oriented programming (AOP).

Examples

A logging aspect
In our AOP framework, each aspect intercepts method calls to handle logging and security.
She wrote an aspect that applies authentication checks across controllers without changing their code.
During review they identified an aspect responsible for cross-cutting error reporting in multiple modules.

Related Terms

  • synonym: aspect-oriented programming - a programming paradigm that uses aspects to separate cross-cutting concerns from business logic
  • synonym: cross-cutting concern - a concern in software that affects multiple modules (for example logging or security)
