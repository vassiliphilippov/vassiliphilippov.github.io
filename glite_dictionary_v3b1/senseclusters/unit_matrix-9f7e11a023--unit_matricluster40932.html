unit matrix

noun
matrix with 1s on diagonal and 0s elsewhere

A square matrix in which all the elements of the principal diagonal are 1 and all other elements are 0; also known as an identity matrix.

Examples

The unit matrix diagonal
Every square matrix has a unique unit matrix of the same size associated with it.
The unit matrix is often used as the multiplicative identity in linear algebra.
When you multiply any matrix by a unit matrix of the same order, the result is the original matrix.
