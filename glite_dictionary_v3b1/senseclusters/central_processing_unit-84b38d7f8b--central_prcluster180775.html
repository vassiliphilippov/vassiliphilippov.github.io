central processing unit

noun
main processing part of a computer
Pronunciation: / ˈsɛntrəl ˈproʊsɛsɪŋ ˌjunɪt / (AmE), / ˈsɛntrəl ˈprəʊsesɪŋ ˌjuːnɪt / (BrE)
the main part of a computer, typically a microprocessor chip, that performs most of the data processing and controls other components

Examples

The central processing unit chip
The central processing unit executes instructions and manages data within the computer.
The central processing unit works closely with memory to perform complex calculations.
Upgrading your central processing unit can significantly improve your computer's performance.

Related Terms

  • synonym: CPU - abbreviation for 'central processing unit', main data-processing chip in a computer
  • synonym: microprocessor - integrated circuit that contains the functions of a central processing unit
