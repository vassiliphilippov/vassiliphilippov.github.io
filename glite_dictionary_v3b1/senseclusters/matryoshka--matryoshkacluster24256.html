matryoshka

noun
nested structure
9.1
Pronunciation: /mə.triː.ˈɑː.ʃ.kə/ /; / /mæ.trɪ.ˈɑː.ʃ.kə/ (AmE), /mə.triː.ˈɒ.ʃ.kə/ /; / /mæ.trɪ.ˈɒ.ʃ.kə/ (BrE)
a set of objects, ideas, or structures nested inside one another, each similar in form to the next smaller one

Examples

Matryoshka structure
The software was designed like a matryoshka, with each module containing smaller, similar components.
Her explanation was a matryoshka of ideas, each concept leading to another, deeper one.
The museum displayed a giant matryoshka sculpture, symbolizing layers within layers.

Related Terms

  • synonym: layer - a level or part within a structure or system
  • synonym: nesting doll - a set of wooden dolls of decreasing size placed one inside another
