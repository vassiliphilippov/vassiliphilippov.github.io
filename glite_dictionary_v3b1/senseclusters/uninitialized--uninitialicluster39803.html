uninitialized

adjective
not given initial value
Pronunciation: / ˌʌnɪˈnɪʃəˌlaɪzd / (AmE), / ˌʌnɪˈnɪʃəlaɪzd / (BrE)
not having been given an initial value or state, especially in computing contexts

Examples

An uninitialized variable
The program crashed because it tried to use an uninitialized variable.
Leaving data uninitialized can lead to unpredictable results in your code.
Always check that no fields remain uninitialized before running the application.

Related Terms

  • antonym: initialized - having been given an initial value or state
  • synonym: undefined - not assigned a value or not having a defined meaning
