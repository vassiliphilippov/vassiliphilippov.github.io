dependency

noun
required software package or library
5.5
Pronunciation: /dɪ.ˈpen.dən.siː/ (AmE), /dɪ.ˈpen.dən.siː/ (BrE)
(in computing) a software package, library, or module that a project or program requires in order to build, run, or provide particular functionality.

Examples

Software dependency
The project build failed due to a missing dependency that could not be resolved.
Before running the app, install each dependency listed in the project's README to avoid errors.
Package managers automatically download required dependencies and keep them up to date for you.

Related Terms

  • synonym: library - a collection of reusable code that provides functionality other programs can use
  • synonym: module - a discrete component of code that provides a specific set of features or functions
  • synonym: package - a bundled distribution of code and metadata that can be installed into a project
