src

noun
abbreviation for source
7.5
Pronunciation: / sɑːrk / (AmE), / sɒːk / (BrE)
an abbreviation for 'source', commonly used in computing or web development to refer to the location of a file or resource

Examples

src attribute in HTML
In HTML, the src attribute tells the browser where to find the image file.
Developers often organize their code in a folder named src for source files.
You need to update the src path if you move the video to a different directory.
