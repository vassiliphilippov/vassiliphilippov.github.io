pickle

noun
serialized object (byte stream)
Pronunciation: / ˈpɪkəl / (AmE), / ˈpɪkəl / (BrE)
(computing) a byte-stream file or object produced by serializing a program object (especially using Python's pickle format).

Examples

A pickle file
I opened the pickle file to restore the application's user settings and cache.
The CI pipeline failed because a corrupted pickle caused the restore step to crash.
Always avoid loading an untrusted pickle because it can execute arbitrary code on your machine.

Related Terms

  • synonym: serialization - the process of converting an object into a format for storage or transmission
  • synonym: serialized object - an object converted into a storable or transmittable format
