descending node

noun
point where orbit crosses ecliptic southward
6.9
Pronunciation: / dɪˈsɛndɪŋ noʊd / (AmE), / dɪˈsɛndɪŋ nəʊd / (BrE)
the point at which the path of an orbiting object crosses the ecliptic plane from north to south

Examples

The descending node crosses
Each month, the Moon's descending node moves slowly westward along the ecliptic.
The eclipse can only occur when the Sun is close to a descending node of the Moon's orbit.
The satellite passed through its descending node at 3:15 AM UTC.

Related Terms

  • antonym: ascending node - the point where an orbit crosses the ecliptic going north
