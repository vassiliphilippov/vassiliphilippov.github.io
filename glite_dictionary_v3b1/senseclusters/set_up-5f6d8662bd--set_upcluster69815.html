set up

verb
to install or arrange something for use
1.0 2.3
Pronunciation: /set ʌp/ /; / /set əp/ (AmE), /set ʌp/ /; / /set əp/ (BrE)
to arrange, assemble, or prepare something so it is ready for use or in the correct order (including building or erecting objects or equipment)
