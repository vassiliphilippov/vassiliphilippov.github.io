authorizer

noun
software permission enforcer
6.6
Pronunciation: /ˈɑː.θə.raɪ.zər/ /; / /ˈɔː.θə.raɪ.zər/ (AmE), /ˈɔː.θə.raɪ.zə/ (BrE)
(in computing) a software component or service that enforces access control by validating credentials or tokens and granting or denying permissions to resources.

Examples

API authorizer checks access
The API gateway uses a Lambda authorizer to validate tokens before forwarding requests to services.
During the audit, engineers found the authorizer misconfigured, allowing excessive permissions to many users.
We replaced the legacy authorizer with a centralized service that enforces role-based access control across systems.

Related Terms

  • synonym: authorization service - a service that decides whether access should be allowed, often by evaluating policies or tokens
