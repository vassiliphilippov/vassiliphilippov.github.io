inference

noun
making predictions with a trained model
5.7
Pronunciation: /ˈɪn.fər.əns/ /; / /ˈɪn.fə.rəns/ (AmE), /ˈɪn.fə.rəns/ /; / /ˈɪn.fər.əns/ (BrE)
(in computing and machine learning) the process of using a trained model to generate predictions or outputs for new, unseen data; often called 'running inference'.

Examples

Machine learning inference
The server runs inference on incoming sensor streams to detect anomalies in real time.
On smartphones, performing inference locally reduces latency and avoids sending private data to the cloud.
After training the network, the team measured inference speed to estimate how many requests it could serve.
