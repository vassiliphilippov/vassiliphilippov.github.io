cube root

noun
number whose third power equals a value
1.8
Pronunciation: / kjuːb ruːt / (AmE), / kjuːb ruːt / (BrE)
a number that produces a specified number when multiplied by itself twice; that is, a number whose third power is the given number

Examples

The cube root of eight
She learned how to calculate the cube root of a number using a calculator.
The cube root of 27 is 3, because 3 × 3 × 3 equals 27.
To solve the equation, you need to find the cube root of 125.
