pickled

adjective
serialized (computing)
8.4
Pronunciation: /ˈpɪk.əld/ (AmE), /ˈpɪk.əld/ (BrE)
(computing) (of data or an object) serialized into a byte stream by a 'pickle' process or format so it can be stored or transmitted and later restored.

Examples

Pickled data file
The pickled object can be restored to its original class with pickle.load after transmission.
During backup, engineers stored several pickled configurations as binary files for quick deployment.
If you receive a pickled object from an untrusted source, avoid unpickling it for security.

Related Terms

  • antonym: deserialize - to convert serialized data back into its original object or data structure
  • synonym: marshal - to prepare and transform data for storage or transmission (often used interchangeably with serialize)
  • synonym: serialize - to convert a data structure or object into a format that can be stored or transmitted
