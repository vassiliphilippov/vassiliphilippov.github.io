exponential function

noun
mathematical function with variable as exponent
Pronunciation: / ˌɛkspoʊˈnɛnʃənl ˈfʌŋkʃən / (AmE), / ˌɛkspəˈnɛnʃənl ˈfʌŋkʃən / (BrE)
A mathematical function in which the variable appears in the exponent, commonly of the form f(x) = a^x where a is a constant.

Examples

An exponential function graph
Population growth can often be modeled with an exponential function.
Solving the equation requires understanding how an exponential function behaves.
The graph of an exponential function rises rapidly as x increases.

Related Terms
