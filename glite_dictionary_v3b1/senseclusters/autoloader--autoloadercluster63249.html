autoloader

noun
self-reloading firearm
8.9 0.29/m
Pronunciation: / Àà…îÀêt…ôÀålo äd…ôr / (AmE), / Àà…îÀêt…ô äÀål…ô äd…ô / (BrE)
a firearm or firearm mechanism that automatically reloads itself after each shot, without requiring manual action by the user

Examples

Autoloader rifle
He prefers using an autoloader for quicker target engagement during competitions.
Many modern tanks use an autoloader instead of a human loader for the main gun.
The new rifle features an autoloader that increases the rate of fire.

Synonyms

  • self-loader
    8.6 0.01/m
    a self-loader automatically reloads after each shot without user action, while an autoloader names any firearm mechanism that reloads itself.
  • semiautomatic firearm
    7.2 0.02/m
    a semiautomatic firearm automatically reloads but fires one round per trigger pull, while an autoloader just means it reloads without manual action.

Related Terms

  • synonym: automatic - a firearm that fires as long as the trigger is pressed and there is ammunition
