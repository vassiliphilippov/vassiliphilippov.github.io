to move an object or figure in space from one position to another without rotating it (technical/physics)

Examples

To translate an image
Computer graphics often translate images across the screen along a straight path.
If an object is translated five units to the right, every point moves the same distance.
In geometry, to translate a shape, you slide it to a new position without turning it.
In physics, a rigid body can translate without rotating along a straight path.
