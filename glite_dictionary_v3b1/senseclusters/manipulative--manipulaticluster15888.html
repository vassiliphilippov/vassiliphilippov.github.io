manipulative

noun
teaching aid object
Pronunciation: / məˈnɪpjələtɪv / (AmE), / məˈnɪpjʊlətɪv / (BrE)
an object that students can handle to help them learn concepts through hands-on activities

Examples

Math manipulative
The teacher gave each child a manipulative to help them understand addition.
Using a manipulative like blocks can make math lessons more interactive for young students.
Students used a different manipulative for each science experiment to explore new concepts.

Related Terms

  • synonym: teaching aid - an object or tool used to support learning
  • antonym: worksheet - a paper with exercises for students
