breakpoint

noun
web design threshold
Pronunciation: / ˈbreɪkˌpɔɪnt / (AmE), / ˈbreɪkˌpɔɪnt / (BrE)
a specific screen width or layout threshold in web design at which a page's appearance or structure changes

Examples

Screen breakpoint
The developer set a breakpoint at 1024 pixels to adjust the layout for tablets.
When the screen size hits a breakpoint, the website automatically rearranges its content.
Designers often test each breakpoint to ensure the site looks good on all devices.

Related Terms

  • synonym: milestone - an important stage or event in a process
  • synonym: threshold - a point at which something changes or begins
