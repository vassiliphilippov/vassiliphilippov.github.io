discretization

noun
making data discrete
Pronunciation: / dɪˌskriːtəˈzeɪʃən / (AmE), / dɪˌskriːtəˈzeɪʃən / (BrE)
the process of converting continuous data or functions into a set of discrete parts or values

Examples

Data discretization process
The discretization of the temperature data made it easier to analyze in a computer program.
Engineers use discretization to solve complex equations in numerical simulations.
Accurate discretization is essential for reliable results in digital signal processing.

Related Terms

  • antonym: interpolation - estimating values between known data points
  • synonym: quantization - the process of mapping input values from a large set to output values in a smaller set
