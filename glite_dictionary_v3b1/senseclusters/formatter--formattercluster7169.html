formatter

noun
device for preparing disks
Pronunciation: / ˈfɔrmætər / (AmE), / ˈfɔːmætə / (BrE)
a program or device that prepares a storage medium by erasing data and creating a file system or format for use

Examples

Disk formatter program
The formatter erased all the data on the hard drive before installing the new software.
Before using the memory card, she ran a formatter to set up the correct file system.
Technicians often rely on a formatter to prepare storage devices for customers.

Related Terms

  • synonym: format - to prepare a storage device for use by erasing data and setting up a file system
  • synonym: initialization - the process of preparing a device or system for use
