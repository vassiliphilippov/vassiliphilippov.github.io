UI cue indicating where to interact

(in user-interface and interaction design) a visible or otherwise perceivable cue that indicates how or where a user should interact with a control or element.

Examples

UI signifier
The new app uses clear signifiers like shadows and arrows to guide first-time users.
A faded button provides a poor signifier and many users miss the feature.
During testing, designers noted which signifier drew attention and which confused participants.

Related Terms

  • synonym: affordance - a property of an object that suggests how it can be used; often discussed alongside signifiers in design.
  • synonym: indicator - a thing that shows the state or existence of something; a visual cue.
