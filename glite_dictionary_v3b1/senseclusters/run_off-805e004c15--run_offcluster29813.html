run off

verb
make photocopies of
Pronunciation: / rʌn ɔf / (AmE), / rʌn ɒf / (BrE)
to reproduce or copy printed material using a photocopier or similar machine

Examples

To run off copies
Could you please run off twenty copies of this report for the meeting?
He runs off the class handouts before every lecture.
She quickly ran off a flyer on the office copier.
Could you run off ten more copies before the meeting starts?
He tried to run off after breaking the window, but the teacher saw him.
They decided to run off together and get married in secret.

  • synonym: photocopy - to make a photographic copy of a document or image
  • synonym: print - to produce text or images on paper
