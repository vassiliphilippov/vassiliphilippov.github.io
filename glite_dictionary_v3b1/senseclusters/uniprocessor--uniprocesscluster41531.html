uniprocessor

noun
single-CPU computer
7.7
Pronunciation: /juː.nɪ.ˈprɑː.ses.ər/ /; / /juː.ni.ˈprɑː.ses.ər/ (AmE), /juː.nɪ.ˈprɒ.ses.ə/ /; / /juː.ni.ˈprɒ.ses.ə/ (BrE)
a computer system that has only one central processing unit (CPU)

Examples

A uniprocessor computer
A uniprocessor is often less efficient at multitasking than a multiprocessor system.
Older desktop computers typically used a uniprocessor design.
Software optimized for a uniprocessor may not take advantage of multiple CPUs.

Related Terms

  • antonym: multiprocessor - a computer system with two or more central processing units
  • synonym: single-core - having only one processing core in a CPU
