production order

noun
instruction to start manufacturing
Pronunciation: / prəˈdʌk.ʃən ˈɔr.dər / (AmE), / prəˈdʌk.ʃən ˈɔː.də / (BrE)
an official instruction to begin manufacturing a specified product or quantity.

Examples

A production order form
Each production order must specify the raw materials and components required.
Once the production order is issued, the factory can start assembling the products.
The manager approved the production order for 500 units of the new model.

Related Terms

  • synonym: work order - an order specifying work to be done, especially in manufacturing or repair
