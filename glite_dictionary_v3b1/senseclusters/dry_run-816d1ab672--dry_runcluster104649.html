dry run

noun
practice session before an event
Pronunciation: / ˈdraɪ ˌrʌn / (AmE), / ˈdraɪ ˌrʌn / (BrE)
a practice session or rehearsal to prepare for an actual event or performance

Examples

A dry run rehearsal
Before launching the new system, the team did a dry run to test for any issues.
The actors gathered for a final dry run the night before the play's opening.
The crew held a dry run of the wedding ceremony to make sure everything would go smoothly.
Before the product launch, the team conducted a dry run to identify possible issues.
Let's do a dry run of the concert so we catch any technical problems.
The firefighters held a dry run to prepare for emergency situations.

  • synonym: trial run - a test or rehearsal of a procedure or process
