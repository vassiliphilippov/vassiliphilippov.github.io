trojan horse

noun
malicious computer program disguised as useful
6.1 0.01/m
Pronunciation: / Ààtro äd í…ôn h…îrs / (AmE), / Ààtr…ô äd í…ôn h…îÀês / (BrE)
a computer program that appears to be useful or harmless but actually contains malicious code intended to damage, disrupt, or gain unauthorized access to a system

Examples

A trojan horse program
