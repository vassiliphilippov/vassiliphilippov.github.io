trojan horse

noun
malicious computer program disguised as useful
4.3 4.9
Pronunciation: /ˈtroʊ.dʒən hɔːrs/ /; / /ˈtroʊ.dʒən hoʊrs/ (AmE), /ˈtrəʊ.dʒən hɔːs/ (BrE)
a computer program that appears to be useful or harmless but actually contains malicious code intended to damage, disrupt, or gain unauthorized access to a system

Examples

A trojan horse program
Always be cautious when downloading files, as a trojan horse can compromise your computer.
Many users were tricked into installing a trojan horse by downloading a fake software update.
The security team found a trojan horse hidden in an email attachment.

Related Terms

  • synonym: malware - software intended to damage or disable computers
  • synonym: virus - a type of malicious program that can replicate itself
  • synonym: worm - a standalone malware program that replicates itself
