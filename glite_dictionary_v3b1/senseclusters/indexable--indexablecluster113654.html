indexable

adjective
can be indexed
5.6
Pronunciation: / ˈɪndɛksəbəl / (AmE), / ˈɪndɛksəbəl / (BrE)
capable of being accessed or referenced by a numeric or positional index, especially in computing or data structures

Examples

Indexable array
The data structure is indexable, allowing quick access to any element by its position.
Only indexable collections can be used with this search algorithm in the software.
If a list is not indexable, you cannot retrieve items using their numeric position.

Related Terms

  • synonym: searchable - able to be searched or found using a search function
  • antonym: unindexable - not able to be accessed or referenced by an index
