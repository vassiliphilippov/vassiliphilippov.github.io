poker game

noun
a betting card game with ranked hands
Pronunciation: / ˈpoʊ.kɚ ɡeɪm / (AmE), / ˈpəʊ.kə ɡeɪm / (BrE)
any of various card games in which players bet on having the highest-ranking hand.

Examples

A poker game night
Every Friday night, the group gathered for a friendly poker game in the basement.
He lost a lot of money at the last poker game he attended.
The rules for each type of poker game can vary significantly.
