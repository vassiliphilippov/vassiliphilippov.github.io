(in computing and artificial intelligence) the loss of previously learned information or performance by a system or model when it is trained or updated on new data; often called catastrophic forgetting.

Examples

Memory loss in AI
Researchers observed severe memory loss in the neural network after training on several new tasks.
Continual-learning teams develop methods to prevent memory loss when models update with fresh data streams.
They measured the model's memory loss by testing accuracy on classes learned earlier in training.
