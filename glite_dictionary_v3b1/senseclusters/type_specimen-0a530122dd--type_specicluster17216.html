type specimen

noun
original specimen for species description
6.5
Pronunciation: / taɪp ˈspɛsəmɪn / (AmE), / taɪp ˈspesɪmɪn / (BrE)
the original specimen on which the description and name of a new species is based

Examples

A type specimen plant
Researchers traveled to collect a type specimen for the newly discovered plant.
The type specimen of the species is kept in the museum's collection.
The original type specimen was lost, requiring the selection of a neotype.

Related Terms

  • synonym: holotype - the single specimen designated or used as the type of a species
  • degree lower: paratype - a specimen used in a species description other than the holotype
