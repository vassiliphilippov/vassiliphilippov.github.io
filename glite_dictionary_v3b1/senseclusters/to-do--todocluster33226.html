to-do

noun
task note in code or tracker
3.6
Pronunciation: /ˈtuː.duː/ /; / /tə.ˈduː/ (AmE), /ˈtuː.duː/ /; / /tə.ˈduː/ (BrE)
An item or note indicating a task that should be done later, especially a marker in code or project trackers (often written TODO).

Examples

To-do list item
The project tracker shows a to-do for implementing the search feature by next sprint.
She left a to-do comment in the code reminding herself to fix the bug later.
During the meeting they assigned several to-dos to different team members for follow-up.

Related Terms

  • synonym: action item - a specific task assigned for someone to complete, typically after a meeting or discussion
  • synonym: task - a piece of work to be done or undertaken
