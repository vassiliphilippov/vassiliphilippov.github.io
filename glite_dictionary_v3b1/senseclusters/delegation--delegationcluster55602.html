delegation

noun
software design pattern
5.5
Pronunciation: / \u02ccd\u025bl.\u0259\u02c8\u0261e\u026a.\u0283\u0259n / (AmE), / \u02ccd\u025bl.\u026a\u02c8\u0261e\u026a.\u0283\u0259n / (BrE)
(in software engineering) a design pattern in which an object passes responsibility for a task or behavior to a separate helper object instead of implementing it itself.

Examples

Delegation in programming
The framework uses delegation to pass tasks to a helper object.
In their codebase, they implemented delegation so different objects handle events cleanly.
During the review, she recommended delegation of authentication logic to a dedicated service.

Related Terms

  • synonym: composition - a design approach in which objects are built from other objects to provide behavior
  • synonym: forwarding - the act of passing a request or message from one object or component to another
