feedforward

adjective
one-way process
8.2
Pronunciation: / ˈfiːdˌfɔrwərd / (AmE), / ˈfiːdˌfɔːwəd / (BrE)
describing a system or process in which information moves forward in one direction without feedback loops.

Examples

Feedforward control system
A feedforward control system anticipates changes and adjusts outputs without waiting for feedback.
In machine learning, a feedforward neural network processes data in a single direction through its layers.
Engineers designed the circuit to be feedforward so that signals would not loop back.

Related Terms

  • antonym: feedback - a process where output is returned to influence input
