logic diagram

noun
diagram using symbols to represent logic functions
1.0 4.5
Pronunciation: /ˈlɑː.dʒɪk ˈdaɪ.ə.ɡræm/ /; / /ˈlɑː.dʒɪk ˈdaɪ.ɡræm/ (AmE), /ˈlɒdʒ.ɪk ˈdaɪ.ə.ɡræm/ /; / /ˈlɒdʒ.ɪk ˈdaɪ.ɡræm/ (BrE)
