logic diagram

noun
diagram using symbols to represent logic functions
Pronunciation: / ˈlɑːdʒɪk ˈdaɪəˌɡræm / (AmE), / ˈlɒdʒɪk ˈdaɪəˌɡram / (BrE)
a graphical representation that uses symbols to show the relationships and functions of logic circuits or formal logical expressions, often used in computing and electrical engineering

Examples

A logic diagram circuit
The engineer created a logic diagram to illustrate the operation of the digital circuit.
The textbook includes several logic diagrams to explain how different gates work together.
Using a logic diagram helps programmers visualize the flow of information in complex software systems.
