slicer

noun
3D printing slicing software
6.6 0.02/m
Pronunciation: / c8sla6as59r / (AmE), / c8sla6as59 / (BrE)
(in 3D printing) software that converts a 3D model into a sequence of thin layers and produces machine instructions (G-code) for a 3D printer.

Examples

3D print slicer
Before printing, she loaded the STL file into the slicer and set layer height and supports.
The new slicer produced cleaner G-code, reducing print time and improving surface finish.
He compared two different slicer settings to find the best balance between strength and print speed.

Related Terms

  • synonym: G-code generator - a program that creates G-code instructions for controlling CNC or 3D printers
  • synonym: slicing software - software that converts 3D models into layered instructions for printers
