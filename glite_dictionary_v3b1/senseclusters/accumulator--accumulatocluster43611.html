accumulator

noun
computer register that accumulates values
Pronunciation: / əˈkjumjəˌleɪtər / (AmE), / əˈkjuːmjʊleɪtə / (BrE)
(computing) a register in a computer processor that stores intermediate arithmetic and logic results, especially by adding values to its contents.

Examples

CPU accumulator
If the accumulator overflows, an error flag may be set by the processor.
Instructions are executed sequentially, updating the accumulator with new values each time.
The CPU's accumulator stores the result of each addition during the calculation.

Related Terms

  • synonym: register - a small amount of fast storage in a computer processor
