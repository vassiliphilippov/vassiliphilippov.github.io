linkage editor

noun
program that resolves and links code modules
Pronunciation: / ˈlɪŋkɪdʒ ˈɛdətər / (AmE), / ˈlɪŋkɪdʒ ˈedɪtə / (BrE)
a computer program that combines several modules of compiled code into a single program by resolving cross-references among them

Examples

The linkage editor program
After compiling all the source files, they ran the linkage editor to create the executable.
Mainframe systems often relied on a linkage editor to produce final application binaries.
The linkage editor is responsible for resolving external references between program modules.

Related Terms

  • synonym: linker - a program that combines object modules into an executable program
