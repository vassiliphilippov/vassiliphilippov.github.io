PO box number

noun
number assigned to a post office box
2.7 0.01/m
Pronunciation: / pi o ä b…ëks Ààn åmb…ö / (AmE), / piÀê …ô ä b…íks Ààn åmb…ô / (BrE)
the unique number assigned to a post office box for collecting mail at a post office

Examples

The PO box number assigned
All correspondence should be sent to their PO box number in New York.
If you do not have a PO box number, mail cannot be delivered to you at the post office.
Please write your PO box number clearly on the application form.

Synonyms

  • post-office box number
    2.1 0.01/m
    post-office box number is the formal full name for a rented mailbox at the post office, while PO box number is the common short form.
  • box number
    3.0 0.02/m
    box number is a general number for any mailbox or parcel locker, while PO box number refers specifically to a rented post office box.
