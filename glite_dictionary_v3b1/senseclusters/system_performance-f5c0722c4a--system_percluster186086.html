system performance

noun
efficiency and effectiveness of a system
Pronunciation: / ˈsɪstəm pərˈfɔrməns / (AmE), / ˈsɪstəm pəˈfɔːməns / (BrE)
the efficiency and effectiveness with which a system operates, often measured in terms of speed, reliability, and resource usage

Examples

System performance improvement
After upgrading the hardware, the team's system performance improved significantly.
Regular maintenance can help sustain optimal system performance over time.
Software bugs can negatively affect system performance and user experience.
