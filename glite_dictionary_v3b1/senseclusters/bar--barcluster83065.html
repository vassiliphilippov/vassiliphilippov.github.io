noun
horizontal UI strip with controls or info
3.1 10/m
Pronunciation: / b…ër / (AmE), / b…ëÀê / (BrE)
(computing) A horizontal area or strip in a graphical user interface that contains buttons, menus, icons, or information (for example, a menu bar, status bar, or navigation bar).

Examples

Menu bar on screen
In the latest update, the top bar on the screen provides quick navigation between app sections.
She tapped the small bar at the bottom to view notifications and battery status.
Developers moved the search field into a floating bar so users could access it faster.
