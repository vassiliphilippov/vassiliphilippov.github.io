triplet code

noun
genetic code using nucleotide triplets
1.9 6.5
Pronunciation: / ˈtrɪplət koʊd / (AmE), / ˈtrɪplət kəʊd / (BrE)
the genetic coding system in which sequences of three nucleotides (triplets) specify particular amino acids in the synthesis of proteins

Examples

Triplet code in DNA
DNA information is stored in the triplet code, where each triplet represents an amino acid.
Scientists study mutations that alter the triplet code and disrupt protein synthesis.
The discovery of the triplet code was a major milestone in molecular biology.
