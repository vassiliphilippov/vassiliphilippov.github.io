preppy dress code

noun
classic, collegiate style of formal attire
4.6
Pronunciation: / ˈprɛpi drɛs koʊd / (AmE), / ˈprɛpi drɛs kəʊd / (BrE)
a style of formal attire characterized by neat, classic, and collegiate-inspired clothing, typically associated with preppy youth culture

Examples

A preppy dress code

The school's strict preppy dress code requires students to wear blazers and ties every day.
