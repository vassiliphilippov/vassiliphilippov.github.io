fmt

noun
Unix text formatter
9.9
Pronunciation: / ɛf ɛm ti / (AmE), / ɛf ɛm ti / (BrE)
a small Unix command-line program that reformats text by adjusting line breaks to a specified width

Examples

Unix fmt command
She used fmt to reformat the text file so each line was under eighty characters.
The fmt utility is helpful for cleaning up messy paragraphs in plain text documents.
After editing the README, he ran fmt to make the lines more readable.

Related Terms

  • synonym: cat - a Unix command that concatenates and displays files
  • synonym: fold - a Unix command that wraps each input line to fit a specified width
