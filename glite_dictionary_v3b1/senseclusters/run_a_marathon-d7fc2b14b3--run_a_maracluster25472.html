run a marathon

verb
participate in a long-distance race
1.6
Pronunciation: / rʌn ə ˈmærəˌθɑn / (AmE), / rʌn ə ˈmærəθən / (BrE)
to participate in a marathon race, typically a long-distance running event of 26.2 miles (42.195 kilometers)

Examples

Many people travel overseas just to run a marathon in famous cities like Boston or Berlin.
