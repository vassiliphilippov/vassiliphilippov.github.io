format

verb
prepare disk for data storage
Pronunciation: / ˈfɔːrˌmæt / (AmE), / ˈfɔːmæt / (BrE)
to prepare a disk or other storage medium for use by dividing it into marked sectors to store data

Examples

To format a disk
She is formatting the USB stick to ensure it is compatible with the computer.
The technician accidentally formatted the memory card, erasing all the photos.
You must format the new hard drive before installing the operating system.

Related Terms

  • synonym: initialize - to set something, especially a computer disk, to a starting value or state
