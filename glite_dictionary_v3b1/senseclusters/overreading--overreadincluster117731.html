overreading

noun
reading past boundary
9.6
Pronunciation: / ˌoʊvərˈriːdɪŋ / (AmE), / ˌəʊvəˈriːdɪŋ / (BrE)
the unintended reading of data beyond a memory or input boundary, often causing errors or security risks in computing

Examples

Data overreading
A software bug led to overreading of the input buffer, which caused the program to crash.
Security experts warned that overreading could expose sensitive information stored in memory.
The developer fixed the overreading issue by adding proper boundary checks to the code.

Related Terms

  • synonym: buffer overflow - an error where data exceeds a buffer's storage capacity
  • antonym: underreading - failing to read all required data from a source
