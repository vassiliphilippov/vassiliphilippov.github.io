cmd

noun
computer instruction
6.8
Pronunciation: / siː ɛm diː / (AmE), / siː ɛm diː / (BrE)
an abbreviation for 'command', referring to a single instruction given to a computer program or operating system

Examples

If you type the wrong cmd, the program may not work as expected.
