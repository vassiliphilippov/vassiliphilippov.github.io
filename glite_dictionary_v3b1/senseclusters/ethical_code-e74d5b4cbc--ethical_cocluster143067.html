ethical code

noun
system of moral principles
2.5 3.3
Pronunciation: /ˈeθ.ɪ.kəl koʊd/ (AmE), /ˈeθ.ɪ.kəl kəʊd/ (BrE)
A set of principles or rules that guide moral behavior and acceptable conduct within a group or profession.

Examples

An ethical code of conduct
Doctors are expected to follow a strict ethical code when treating their patients.
Every professional organization should have an ethical code to guide the behavior of its members.
The company's ethical code prohibits accepting gifts from clients.
