committer

noun
person making changes
7.9
Pronunciation: / kəˈmɪtər / (AmE), / kəˈmɪtə / (BrE)
a person or program that commits changes, especially in computing or law

Examples

Project committer
The committer pushed the latest code updates to the repository.
Each committer must review their changes before submitting them.
In the project, only a trusted committer can approve new features.

Related Terms

  • synonym: contributor - a person who adds or provides something, especially to a project
