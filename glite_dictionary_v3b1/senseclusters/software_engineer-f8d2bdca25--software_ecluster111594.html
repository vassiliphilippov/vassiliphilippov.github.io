software engineer

noun
professional who develops computer software
Pronunciation: / ˈsɔft.wɛr ˌɛn.dʒəˈnɪr / (AmE), / ˈsɒft.wɛə ˌɛn.dʒɪˈnɪə / (BrE)
a person whose job is to design, write, and test computer programs and applications

Examples

Software engineer designs programs
After graduating from university, she became a software engineer at a major tech company.
Many software engineers work remotely and collaborate across continents.
The software engineer fixed the bug that was causing the system to crash.
