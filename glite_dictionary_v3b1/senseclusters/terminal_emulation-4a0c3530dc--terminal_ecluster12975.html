terminal emulation

noun
computer acting like a terminal
6.1
Pronunciation: / ˈtɜrmɪnəl ˌɛmjəˈleɪʃən / (AmE), / ˈtɜːmɪnəl ˌɛmjuˈleɪʃən / (BrE)
the process of making a computer behave as if it were a terminal, typically to access mainframes or remote systems

Examples

Terminal emulation software
Many network administrators use terminal emulation to access servers remotely.
With terminal emulation, a desktop PC can communicate with legacy mainframe computers.
You can install software that offers terminal emulation on almost any modern operating system.
