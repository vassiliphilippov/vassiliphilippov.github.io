heartbeat

noun
periodic liveness signal (computing)
Pronunciation: / c8h51rtccbit / (AmE), / c8h51d0tccbid0t / (BrE)
(in computing) a periodic signal sent between systems or components to indicate they are operating and reachable; used to detect failures and trigger recovery.

Examples

Server heartbeat
The monitoring service sends a heartbeat every few seconds to confirm each server is still alive.
When a node stops replying to heartbeats, the cluster manager marks it as failed and reallocates tasks.
Engineers configured a watchdog to restart the service if no heartbeat arrives for thirty seconds.

Related Terms

  • synonym: liveness probe - a check or test that determines whether a service or component is still running and responsive
  • synonym: ping - a simple network request used to test whether a remote host is reachable
