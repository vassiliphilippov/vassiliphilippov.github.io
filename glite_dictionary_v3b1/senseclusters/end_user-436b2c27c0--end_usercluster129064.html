end user

noun
final user of a product or service
Pronunciation: / ˈɛnd ˌjuːzər / (AmE), / ˈɛnd ˌjuːzə / (BrE)
the person or group who actually uses a product or service after it has been developed and sold, rather than those involved in its creation or distribution

Examples

End users give feedback
Manufacturers gather feedback from end users to improve their products.
Software updates must be easy for the end user to install without technical help.
The new interface was designed with the end user in mind, focusing on simplicity.
