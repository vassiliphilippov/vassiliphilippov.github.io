statics

noun
static web assets
Pronunciation: / ˈstætɪks / (AmE), / ˈstætɪks / (BrE)
(in computing, especially web development) static files or assets served unchanged by a website or server, such as CSS, JavaScript, and images.

Examples

Website statics
The deployment serves all statics from a CDN to improve load times.
Please put the statics directory behind the cache-control header for longer caching.
When building the site, they copied small images and CSS into the statics folder.

Related Terms

  • antonym: dynamic content - web content generated or changed per request (not served as fixed files)
  • synonym: static asset - a file (CSS, JS, image) served unchanged by a web server
