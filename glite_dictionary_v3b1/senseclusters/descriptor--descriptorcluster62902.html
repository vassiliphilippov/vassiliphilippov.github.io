descriptor

noun
identifier in information systems
Pronunciation: / dɪˈskrɪptər / (AmE), / dɪˈskrɪptə / (BrE)
a piece of stored information used to identify or describe an item in an information storage and retrieval system, especially in computing.

Examples

File descriptor
A unique descriptor is assigned to every document in the archive.
The database uses a descriptor to keep track of each file.
The search function relies on metadata and descriptors to locate items quickly.

Related Terms

  • synonym: identifier - a symbol or name that uniquely identifies an item in a system
