context

noun
runtime environment or program state
6.8 4.06/m
Pronunciation: / \u02c8k\u0251n.t\u025bkst / (AmE), / \u02c8k\u0252n.t\u025bkst / (BrE)
(computing) The runtime environment or state in which a program, function, thread, or piece of code executes, including variables, memory, bindings, and the call stack.

Examples

Program context
The function executes within a dynamic context that includes variables and the call stack.
When debugging, she inspected the context to see which variable values caused the crash.
Different threads can run in separate contexts, each with its own memory bindings and permissions.

Related Terms

  • synonym: environment - the complete set of conditions and resources in which a program or process runs
  • synonym: runtime - the period when a program is executing and the associated runtime system or state
