org

noun
Org mode file format
9.3
Pronunciation: / ɔːɡ / (AmE), / ɔːɡ / (BrE)
a plain-text file extension and format used by Emacs Org mode for organizing notes, outlines, and tasks

Examples

Org file format
She saved her notes as an org file to keep them organized in Emacs.
Many programmers use the org format to manage their daily tasks and outlines.
The org file was easy to edit and share with her colleagues.
