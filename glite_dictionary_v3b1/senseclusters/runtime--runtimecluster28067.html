runtime

noun
program execution period
4.9
Pronunciation: /ˈrʌn.taɪm/ (AmE), /ˈrʌn.taɪm/ (BrE)
the period or environment in which a computer program, process, or movie is running or active

Examples

Movie runtime
The runtime of the application increased after the new features were added.
You can change some settings only during the runtime of the program.
The software encountered an unexpected error at runtime and had to close.
The runtime of the software determines how long it will stay active on your computer.
Before watching the movie, she checked its runtime to see if she had enough time.
The runtime for this process is usually less than five minutes.

Related Terms

  • antonym: compile time - the period when a program is being compiled, before it runs
  • synonym: duration - the length of time something lasts
  • synonym: execution - the process of carrying out a computer program
  • synonym: length - the amount of time something continues
