disprovable

adjective
able to be disproved
6.5
Pronunciation: / dɪsˈpruːvəbəl / (AmE), / dɪsˈpruːvəbəl / (BrE)
able to be shown to be false or incorrect through evidence or argument

Examples

Disprovable theory
A scientific theory should be disprovable if new evidence contradicts it.
The lawyer argued that the accusation was disprovable with the available facts.
Not every claim is disprovable using current technology or knowledge.

Related Terms

  • antonym: irrefutable - impossible to disprove or deny
  • synonym: refutable - able to be proven wrong or false
