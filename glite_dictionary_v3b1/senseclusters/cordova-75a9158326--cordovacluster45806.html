Cordova

proper noun
cross-platform mobile app framework
7.2 0.06/m
Pronunciation: / k…îrÀàdo äv…ô / (AmE), / k…îÀêÀàd…ô äv…ô / (BrE)
A cross-platform mobile development framework (Apache Cordova) that lets developers build apps using HTML, CSS, and JavaScript and package them for multiple mobile platforms.

Examples

Apache Cordova framework
Many developers choose Cordova when they need to build cross-platform mobile apps quickly.
She installed a plugin to enable camera access in her Cordova project before testing.
Their company published the update as a Cordova app targeting both Android and iOS stores.

Related Terms

  • synonym: PhoneGap - the original name / earlier distribution of the Cordova project, a tool for building hybrid mobile apps
  • synonym: hybrid app framework - a framework that uses web technologies to build mobile apps packaged to run on device platforms
