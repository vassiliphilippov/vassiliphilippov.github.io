inheritability

noun
ability to be inherited
8.2
Pronunciation: /ɪn.ə.rə.ˈbɪl.ɪ.tiː/ (AmE), /ɪn.ə.rə.ˈbɪl.ɪ.tiː/ (BrE)
the quality of a class member in object-oriented programming that allows it to be inherited by subclasses

Examples

Inheritability of a method
The inheritability of the method allows subclasses to reuse its functionality without rewriting code.
Changing the inheritability of a property can affect how future classes interact with it.
Good software design considers the inheritability of class members to promote code reuse.

Related Terms

  • antonym: encapsulation - the concept of restricting access to certain components of an object
  • synonym: inheritance - the mechanism by which one class acquires properties of another
