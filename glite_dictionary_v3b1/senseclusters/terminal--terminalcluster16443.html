terminal

noun
device for computer access
1.5 4.6
Pronunciation: /ˈtɜːr.mɪ.nəl/ /; / /ˈtɜːr.mə.nəl/ (AmE), /ˈtɜː.mɪ.nəl/ /; / /ˈtɜː.mə.nəl/ (BrE)
a device or program used to interact with a computer or computer network (such as a keyboard and screen, or a command-line window)

Examples

Computer terminal
Many banks still use a terminal for employees to process transactions quickly and securely.
The technician replaced the faulty terminal to restore access to the mainframe computer.
You can log in remotely through a terminal connected to the company's network.
He opened the terminal to run command-line tools and inspect system logs.
The developer prefers the terminal for file operations rather than using a graphical file manager.
After installing updates, she typed a command into the terminal to restart the service.

Related Terms

  • synonym: console - a control panel for operating a computer system
  • synonym: terminal emulator - a program that replicates the functions of a traditional hardware terminal in software
  • synonym: workstation - a desktop computer terminal typically used by one person
