word string

noun
sequence of words
Pronunciation: / wɝːd strɪŋ / (AmE), / wɜːd strɪŋ / (BrE)
a linear sequence of words, either spoken or written, treated as a single entity

Examples

A word string sequence
A password is often treated as a special word string in security systems.
She copied the entire word string from the document for reference.
The program analyzes each word string for grammatical errors.
