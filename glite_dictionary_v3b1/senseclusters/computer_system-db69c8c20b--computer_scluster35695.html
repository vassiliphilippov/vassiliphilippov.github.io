computer system

noun
a group of computers and software working together
Pronunciation: / kəmˈpjuːtər ˈsɪstəm / (AmE), / kəmˈpjuːtə ˈsɪstəm / (BrE)
A set of one or more computers and related software that operate together using shared storage and resources.

Examples

A computer system network
A backup of the entire computer system should be done regularly to prevent data loss.
Her job involves maintaining the school's computer system and ensuring all devices are connected.
The company upgraded its computer system to improve security and performance.
