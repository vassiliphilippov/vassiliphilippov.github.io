RAM disk

noun
virtual drive in RAM for fast data access
1.2 5.0
Pronunciation: /ræm dɪsk/ (AmE), /ræm dɪsk/ (BrE)
(computing) a virtual storage drive created by using system memory (RAM) to act like a disk drive, usually very fast but may lose data when power is lost unless backed by persistent memory

Examples

The RAM disk speed
All information on the RAM disk will be lost if you reboot the computer.
To speed up the program, the developer saved temporary files on the RAM disk.
Using a RAM disk can drastically improve data transfer performance for certain applications.
To ensure logs survive reboots, they mounted a RAM disk on NVDIMM persistent memory.
Performance testing showed the application's temp files accessed the RAM disk far faster than the SSD.
After the power failure, the team retrieved critical logs from the RAM disk that had persisted.

Related Terms

  • antonym: hard disk - data storage device using magnetic storage
  • synonym: virtual drive - software-based emulation of a physical disk drive
  • antonym: volatile RAM disk - a RAM disk implemented in ordinary RAM whose contents are lost when power is removed
