booter

noun
startup program or disk
Pronunciation: / ˈbuːtər / (AmE), / ˈbuːtə / (BrE)
a small program or disk that starts a computer and loads an operating system or software when powered on

Examples

Booter disk
The vintage computer came with a booter disk that started the game automatically.
You need a booter to load the operating system on that old machine.
Without a booter, the computer will not start up properly.

Related Terms

  • synonym: boot loader - a program that loads an operating system when a computer starts
  • synonym: startup disk - a disk used to start a computer and load its operating system
