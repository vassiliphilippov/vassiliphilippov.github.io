vectorization

noun
code optimization method
8.0
Pronunciation: / ˌvɛktərəˈzeɪʃən / (AmE), / ˌvɛktəraɪˈzeɪʃən / (BrE)
the process of converting computer code to perform operations on multiple data elements simultaneously using vector instructions to improve performance

Examples

Code vectorization
Modern compilers can perform vectorization to make programs run faster on new hardware.
The engineer explained how vectorization improved the speed of the image processing algorithm.
Without vectorization, the software would take much longer to analyze large datasets.

Related Terms

  • synonym: parallelization - the process of making code run on multiple processors or cores at the same time
  • antonym: serialization - the process of executing operations one after another rather than simultaneously
