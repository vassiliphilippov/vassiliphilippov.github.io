extend

verb
derive a subclass (computing)
6.2
Pronunciation: /ɪk.ˈstend/ /; / /ek.ˈstend/ (AmE), /ek.ˈstend/ /; / /ɪk.ˈstend/ (BrE)
(computing) to create a new class, component, or module by inheriting from an existing one, adding or overriding behaviour and properties.

Examples

To extend a class
In Java, you extend the Animal class to create specialized subclasses for dogs.
When she extended the base component, she overrode its render method to change the layout.
Frameworks often encourage extending existing modules rather than rewriting them to reuse tested functionality.

Related Terms

  • synonym: inherit - to receive properties or methods from a parent class in object-oriented programming
  • synonym: subclass - a class that derives from another (parent) class, gaining or overriding its behaviour
