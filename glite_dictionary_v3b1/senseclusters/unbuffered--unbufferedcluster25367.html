unbuffered

adjective
without temporary storage
6.9
Pronunciation: / ʌnˈbʌfərd / (AmE), / ʌnˈbʌfəd / (BrE)
not using temporary storage, so data is processed or transferred immediately without delay.

Examples

Unbuffered output mode
The program uses unbuffered output to display each character as soon as it is typed.
For real-time applications, unbuffered data transfer can reduce delays.
The technician recommended switching to unbuffered input for more immediate feedback.

Related Terms

  • antonym: buffered - using temporary storage to hold data before processing
