bug

noun
error or flaw in a computer program
3.0
Pronunciation: / bʌɡ / (AmE), / bʌɡ / (BrE)
an error or flaw in a computer program or system that causes it to work incorrectly

Examples

A bug in software
A critical bug in the code left users' data vulnerable to hackers.
She spent hours trying to track down a bug in the website's login system.
The software update fixed the bug that caused the program to crash unexpectedly.
After installing the update, users noticed a new bug that crashed the app.
Reporting a bug helps developers improve the quality of the program.
The team worked all night to fix a critical bug before the software launch.

Related Terms

  • antonym: feature - a distinctive attribute or aspect of something, sometimes used ironically in programming to mean an intentional behavior that seems like a bug
  • synonym: glitch - a minor, temporary malfunction or irregularity
