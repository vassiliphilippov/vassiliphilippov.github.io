remember

verb
store user data for future use

(in computing) to store a user's information, settings, or preferences so they are available automatically in future interactions.

Examples

Remember your password
The website will remember your language choice so pages load in the preferred language next time.
After the update, the app remembered my login details so I could access my account immediately.
Many sites ask if they should remember you on this device; on public computers it's safer to say no.

Synonyms

  • persist
    6.8 0.13/m
    persist writes data to permanent storage so it survives program end or reboot, while remember restores user settings automatically later.
  • save
    1.3 14/m
    save records user data or files when asked or manually, while remember silently stores preferences for automatic use in future interactions.

Related Terms

  • synonym: save - to record data so it can be used again later, often to persistent storage
  • synonym: store - to put or keep data in a place (such as a database or disk) for later retrieval
