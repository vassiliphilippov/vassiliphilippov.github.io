absolute value

noun
non-negative value of a number
2.8 3.2
Pronunciation: /ˈæb.sə.luːt ˈvæl.juː/ (AmE), /ˈæb.sə.luːt ˈvæl.juː/ (BrE)
the numerical value of a real number without regard to its sign; the non-negative value of a number

Examples

The absolute value of a number
Temperature drops below zero, but its absolute value tells us how far it is from zero.
The absolute value of -8 is 8.
When solving equations, we sometimes need to consider the absolute value of the result.
