Microsoft minute

noun
unpredictably long delay (humorous)
6.4 7.5
Pronunciation: /ˈmaɪ.krə.sɑːft ˈmɪn.ət/ /; / /ˈmaɪ.kroʊ.sɑːft ˈmɪn.ət/ /; / /ˈmaɪ.krə.sɔːft ˈmɪn.ət/ (AmE), /ˈmaɪ.krə.sɒft ˈmɪn.ɪt/ (BrE)
A humorous term referring to a brief but unpredictably long delay, especially when something is expected to happen quickly but takes longer than anticipated.

Examples

A Microsoft minute
I'll be back in a Microsoft minute, but you know how updates can be.
She promised she’d deliver the files in a Microsoft minute, which meant I shouldn’t hold my breath.
When the installer said it would finish in a Microsoft minute, I knew it might take all afternoon.
