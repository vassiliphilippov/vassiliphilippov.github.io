misconfiguration

noun
incorrect setup
6.8
Pronunciation: / ˌmɪskənˌfɪɡjəˈreɪʃən / (AmE), / ˌmɪskənˌfɪɡjʊˈreɪʃən / (BrE)
an incorrect or faulty setup of a device, system, or software that can cause problems or failures

Examples

A misconfiguration error
A simple misconfiguration in the server settings can make a website unavailable to users.
The security breach was traced back to a misconfiguration in the database permissions.
He spent hours troubleshooting the network, only to find a misconfiguration in the router.

Related Terms

  • antonym: configuration - the arrangement or setup of the parts of a system
  • synonym: error - a mistake or problem in a process or system
  • synonym: malfunction - a failure to function properly
