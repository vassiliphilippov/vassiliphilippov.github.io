stemming algorithm

noun
algorithm that reduces words to their stems
Pronunciation: / ˈstɛmiŋ ælˌɡɒrɪðəm / (AmE), / ˈstɛmɪŋ ˈalɡərɪðəm / (BrE)
a computer algorithm that removes inflectional and derivational endings from words to reduce them to their root form or stem

Examples

A stemming algorithm process
By applying a stemming algorithm, the program can improve search results by recognizing related terms.
Researchers compare the accuracy of each stemming algorithm before choosing one for their project.
The search engine uses a stemming algorithm to match different word forms to the same base word.
