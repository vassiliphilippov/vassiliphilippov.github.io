ASCII text file

noun
plain text file using ASCII characters only
5.0
Pronunciation: / ˈæski tɛkst faɪl / (AmE), / ˈaski tɛkst faɪl / (BrE)
a text file that contains only ASCII characters and no special formatting or encoding

Examples

An ASCII text file format
He converted the report into an ASCII text file so it could be read on any computer.
The software only accepts ASCII text files, so you can't use documents with special formatting.
When emailing code, it's safest to send it as an ASCII text file to avoid compatibility issues.
