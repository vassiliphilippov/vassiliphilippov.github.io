polling

noun
repeated status checking
Pronunciation: / ˈpoʊlɪŋ / (AmE), / ˈpəʊlɪŋ / (BrE)
a process in computing or electronics where a program or device repeatedly checks the status of an input, sensor, or peripheral to determine if action is needed

Examples

Device polling status
The software uses polling to check for new messages every minute.
Continuous polling of the sensor can slow down the system's performance.
Instead of polling, the device now responds to interrupts for greater efficiency.

Related Terms

  • antonym: interrupt - a signal that temporarily halts a program to give attention to an event
  • synonym: query - to request information from a system or database
