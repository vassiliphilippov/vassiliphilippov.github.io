drawing machine

noun
machine for stretching metal by drawing
Pronunciation: / ˈdrɔɪɪŋ məˌʃin / (AmE), / ˈdrɔːɪŋ məˌʃiːn / (BrE)
a machine tool that uses tensile forces to stretch metal into wire, rods, or other shapes by pulling it through a die.

Examples

A drawing machine tool
Each drawing machine is carefully calibrated to produce wires of a precise diameter.
The drawing machine requires regular maintenance to ensure smooth operation during the metal stretching process.
The new factory installed a state-of-the-art drawing machine to increase its wire production capacity.
