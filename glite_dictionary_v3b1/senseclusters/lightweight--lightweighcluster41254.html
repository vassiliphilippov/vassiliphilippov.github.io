lightweight

adjective
minimal-resource software
3.9
Pronunciation: /ˈlaɪ.tweɪt/ /; / /ˈlaɪt.weɪt/ (AmE), /ˈlaɪ.tweɪt/ /; / /ˈlaɪt.weɪt/ (BrE)
(computing) (of software, a library, or a framework) designed to require minimal system resources, dependencies, or configuration; efficient and easy to run.

Examples

Lightweight software
Developers chose a lightweight framework to keep the app fast on mobile devices.
The team preferred lightweight libraries with few dependencies to simplify deployment and reduce memory use.
After testing, the new service proved much more lightweight than the old one under peak load.

Related Terms

  • synonym: efficient - performing a task with minimal waste of resources such as time or memory
  • antonym: heavyweight - (in computing) software or libraries that use many resources or have large dependencies
