tuple

noun
grouped data elements
8.0
Pronunciation: / ˈtuːpəl / (AmE), / ˈtjuːpəl / (BrE)
an ordered group of elements, often of different types, treated as a single unit in programming or mathematics

Examples

A tuple in programming
In Python, a tuple can store both numbers and strings together.
The function returns a tuple containing the result and an error code.
Coordinates are often stored as a tuple in computer programs.
In programming, a tuple can store several values of different types together.
The database returns each record as a tuple containing three fields.
A coordinate point in space can be represented as a tuple of numbers.

  • synonym: list - an ordered sequence of elements in programming
  • antonym: set - a collection of distinct elements without order
