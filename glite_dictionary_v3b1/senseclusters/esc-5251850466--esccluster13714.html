Esc

noun
computer control character
8.9
Pronunciation: / ɛsk / (AmE), / ɛsk / (BrE)
a control character (ASCII code 27) used in computing to start control or escape sequences for terminals and devices

Examples

Esc character code
Pressing the Esc key will often cancel the current operation in many programs.
The terminal recognized the Esc character and changed the text color as instructed.
Some games use the Esc key to open the pause menu or exit to the main screen.

Related Terms

  • synonym: control character - a non-printing character used to control devices
  • synonym: escape key - a keyboard key used to generate the escape character
