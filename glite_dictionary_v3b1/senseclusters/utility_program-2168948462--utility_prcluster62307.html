utility program

noun
software for computer maintenance and support

(computing) a computer program designed to perform a specific task that helps manage, maintain, or support the computer system, such as file management or virus scanning.

Examples

A utility program tool
Many operating systems come with built-in utility programs for tasks like compressing files or managing printers.
She used a disk cleanup utility program to free up space on her computer.
The antivirus utility program runs a scan of your entire hard drive every week.

Synonyms

  • utility
    4.4 4.60/m
    utility is a small program that does a single maintenance or management task, while utility program is the full, formal name.
  • service program
    4.2 0.31/m
    service program runs mainly to support other software or users in the background, while utility program often means user tools for maintenance.
