(in information technology) the process, mechanism, or data (such as tokens or permissions) by which a system grants, restricts, or enforces access rights to resources or actions; distinct from authentication.

Examples

Authorization token access
The system issues an authorization token that allows the app to read the user's profile data.
Database administrators set role-based authorization so employees see only the records they need.
Unlike authentication, authorization determines what actions a verified user may perform in an application.
