extensible

adjective
able to be extended
Pronunciation: / ɪkˈstɛnsəbəl / (AmE), / ɪkˈstɛnsəbəl / (BrE)
able to be extended or made larger in scope, size, or function

Examples

Extensible software
The new platform is highly extensible, allowing developers to add new features easily.
We chose an extensible design so the system can grow with our needs.
An extensible table can be lengthened to seat more guests during dinner parties.

Related Terms

  • antonym: fixed - not able to be changed or extended
