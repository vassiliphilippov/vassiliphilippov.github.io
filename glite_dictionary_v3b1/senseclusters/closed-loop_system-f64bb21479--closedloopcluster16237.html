closed-loop system

noun
system with feedback control
5.7 4.9
Pronunciation: /ˈkloʊzd.luːp ˈsɪs.təm/ (AmE), /ˈkləʊzd.luːp ˈsɪs.təm/ (BrE)
A control system in which feedback is used to automatically adjust its operation for desired results.

Examples

Automatic closed-loop system
Automotive cruise control is a simple example of a closed-loop system in action.
In a closed-loop system, sensors constantly provide data to correct any deviations from the set target.
The new factory uses a closed-loop system to monitor and adjust the machinery automatically.
