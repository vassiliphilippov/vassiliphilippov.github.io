platform

noun
computer hardware and operating system combination
1.4
Pronunciation: / ˈplæt.fɔrm / (AmE), / ˈplæt.fɔːm / (BrE)
the combination of a particular type of computer hardware and a particular operating system, forming a base on which applications can run

Examples

Software platform
Developers often design applications to be compatible with more than one platform.
He switched to a new platform that offered better security features.
This software is available for every major platform, including Windows, macOS, and Linux.

Related Terms

  • synonym: environment - the hardware or software setting in which a program runs
