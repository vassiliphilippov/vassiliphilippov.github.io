Sysprep

verb
to prepare Windows system
9.9
Pronunciation: /ˈsɪs.prep/ /; / /ˈsɪz.prep/ (AmE), /ˈsɪs.prep/ /; / /ˈsɪz.prep/ (BrE)
to run the Microsoft Sysprep program on a Windows computer to prepare it for imaging, duplication, or deployment

Examples

To Sysprep a computer
Before cloning the computers, the IT team will Sysprep each machine to ensure unique settings.
You should always Sysprep a Windows system before deploying it to multiple users.
After making all necessary updates, she decided to Sysprep the device for imaging.
