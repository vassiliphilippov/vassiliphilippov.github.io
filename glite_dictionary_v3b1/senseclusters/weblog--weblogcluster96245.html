weblog

noun
web server activity record
Pronunciation: / ˈwɛbˌlɑɡ / (AmE), / ˈwɛbˌlɒɡ / (BrE)
a file or record that tracks activity or requests made to a web server

Examples

Server weblog
The administrator checked the weblog to identify unusual activity on the server.
After the cyberattack, experts analyzed the weblog for suspicious IP addresses.
Regularly reviewing the weblog helps maintain the security of the website.

Related Terms

  • synonym: access log - a log recording access requests to a server
  • synonym: server log - a file recording events or requests on a server
