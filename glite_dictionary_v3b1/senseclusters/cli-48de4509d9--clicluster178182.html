CLI

noun
text-based command interface
7.9 0.11/m
Pronunciation: / siÀê …õl a…™ / (AmE), / siÀê …õl a…™ / (BrE)
A computing interface in which users type textual commands, rather than using graphical menus or icons, to control software or the operating system.

Examples

The CLI interface
Learning to use the CLI can greatly improve your productivity as a developer.
Many servers are managed through a CLI because it offers more control for experienced users.
Unlike a GUI, a CLI requires you to memorize and type commands.

Synonyms

  • command line interface
    6.7 0.02/m
    command line interface is a text-based way to control programs using typed commands, while CLI is the short name used by technical users.
