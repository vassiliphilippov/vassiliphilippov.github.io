plumb rule

noun
tool for checking vertical alignment
7.9 0.00/m
Pronunciation: / pl åm ruÀêl / (AmE), / pl åm ruÀêl / (BrE)
A tool consisting of a straight narrow board with a plumb line attached, used by builders to check if a surface is perfectly vertical.

Examples

A plumb rule tool
Before spirit levels were common, builders relied on the trusty plumb rule for vertical alignment.
Even in modern construction, a plumb rule can be useful for checking accuracy.
The carpenter used a plumb rule to ensure the wall was perfectly upright.
