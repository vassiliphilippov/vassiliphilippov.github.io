Synonyms

  • run
    2.2 135/m
    run means a machine or system operates continuously, while go also means to begin or be in action generally.
  • work
    1.0 393/m
    work means a machine or plan functions effectively, while go can mean both starting and functioning in many situations.
  • operate
    2.8 38/m
    operate means to control or make a machine work properly, while go also means to begin or to function without control.
  • function
    2.6 35/m
    function describes how something works to do its role, while go means to start or be in action more generally.
  • start
    1.0 843/m
    start means to begin an action or cause something to begin, while go also means to work or function after beginning.
  • go on
    1.1 2.66/m
    go on means to begin running or functioning, often for machines, while go is broader and means to start or be in action.
  • get going
    2.3 8.54/m
    get going means to start being active, usually for people in an informal way, while go has broader, neutral uses.
  • go to work
    2.0 3.28/m
    go to work means a machine or system begins operating to do its job, while go also covers general starting or action.
