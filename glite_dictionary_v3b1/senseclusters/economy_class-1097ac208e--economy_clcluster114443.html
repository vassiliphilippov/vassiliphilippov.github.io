Examples

Economy class ticket

Synonyms

  • tourist class
    3.2 0.01/m
    tourist class offers basic, low-cost seating for many travelers, while economy class is the common modern airline term for that level.
  • second class
    3.2 0.22/m
    second class is a lower-priced tier on trains or ships and can be above the very cheapest options, while economy class is the lowest fare.
  • cabin class
    3.1 0.01/m
    cabin class refers to a non-first level of accommodation on ships or planes, while economy class specifically means the least expensive seats.
