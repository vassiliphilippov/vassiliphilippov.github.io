connected

adjective
joined to a power source or network
1.0
Pronunciation: / kəˈnɛktɪd / (AmE), / kəˈnɛktɪd / (BrE)
(of a device or system) joined or linked to a power source, network, or other devices so it can work or communicate

Examples

Connected to Wi-Fi
Always make sure the computer is connected before turning it on.
Several devices can be connected to the same Wi-Fi network at once.
The lamp won't work unless it is properly connected to the power outlet.
All the machines in the office are now connected to the central server.
The new printer is connected to the network, so everyone can use it.
With a connected home, you can control your lights remotely using an app.
Each smoke detector is connected to the main fire alarm panel for immediate alerts.
If the cameras aren't connected, the security system won't function properly.
Make sure all the window sensors are connected to the central alarm system.

Related Terms

  • antonym: disconnected - not joined to a power source or network
  • synonym: wired - connected using wire, especially for electrical or communication purposes
