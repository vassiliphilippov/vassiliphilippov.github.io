threshold function

noun
mathematical function; outputs 1 or 0 based on a threshold
Pronunciation: / ˈθrɛʃˌhoʊld ˈfʌŋkʃən / (AmE), / ˈθrɛʃhəʊld ˈfʌŋkʃən / (BrE)
a function that outputs 1 if a specified function of its arguments exceeds a given threshold, and 0 otherwise

Examples

A threshold function output
A simple threshold function returns 1 if its input exceeds the value of five.
Researchers analyzed how changing the threshold function affected classification performance.
The threshold function is widely used in neural networks to determine activation.
