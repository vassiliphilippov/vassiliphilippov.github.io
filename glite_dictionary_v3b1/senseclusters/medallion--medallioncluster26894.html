medallion

noun
three-layer data architecture (Bronze/Silver/Gold)
8.4
Pronunciation: /mə.ˈdæl.jən/ (AmE), /mə.ˈdæl.jən/ (BrE)
(in data engineering) a three-layer data architecture that organises data into Bronze (raw), Silver (cleaned), and Gold (curated) zones for processing and analytics.

Examples

Medallion architecture
The data team adopted a medallion architecture to organize Bronze, Silver, and Gold data.
Using a medallion model helped them separate raw logs, cleaned records, and analytics-ready tables.
Many modern platforms provide tools to implement a medallion workflow for reliable data delivery.

Related Terms

  • synonym: data architecture - the overall design and organization of data systems and how data flows between components
  • synonym: data pipeline - a sequence of data processing steps that move and transform data from source to destination
