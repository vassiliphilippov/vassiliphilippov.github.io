black out the memory

verb
deliberately suppress a memory
Pronunciation: / blæk aʊt ðə məˈmɛɹi / (AmE), / blæk aʊt ðə məˈmɛɹi / (BrE)

Some people black out the memory of traumatic events as a coping mechanism.
