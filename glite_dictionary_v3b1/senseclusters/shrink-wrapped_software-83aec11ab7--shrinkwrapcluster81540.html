shrink-wrapped software

noun
commercially packaged software product
Pronunciation: / ʃrɪŋk ræpt ˈsɔftwɛr / (AmE), / ʃrɪŋk ræpt ˈsɒftwɛː / (BrE)
Commercially packaged software sold as a physical product, typically boxed and sealed in shrink wrap, usually supporting standard platforms.

Examples

Boxed shrink-wrapped software
In the past, most users purchased shrink-wrapped software from electronics stores.
Support for standard hardware was crucial for shrink-wrapped software to run on various computers.
The rise of downloadable applications has reduced the popularity of shrink-wrapped software.

Related Terms

  • synonym: commercial software - software developed for sale to end users
  • antonym: open-source software - software with source code made freely available
