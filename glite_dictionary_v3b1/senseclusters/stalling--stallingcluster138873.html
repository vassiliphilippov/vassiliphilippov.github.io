stalling

noun
temporary CPU pipeline halt
7.2
Pronunciation: /ˈstɔː.lɪŋ/ /; / /ˈstɑː.lɪŋ/ (AmE), /ˈstɔː.lɪŋ/ (BrE)
(computing) a temporary suspension of a processor's instruction execution, usually in a CPU pipeline, caused by hazards, cache misses, or resource conflicts that delay subsequent instructions.

Examples

CPU stalling
During the benchmark, frequent stalling kept the CPU from reaching peak throughput.
Cache misses caused repeated stalling, so the program ran much slower than expected.
Engineers measured pipeline stalling during memory accesses to identify performance bottlenecks.

Related Terms

  • synonym: pipeline bubble - a short delay inserted into a CPU pipeline that leaves stages idle
  • synonym: pipeline stall - a pause in a CPU pipeline that prevents instructions from progressing
