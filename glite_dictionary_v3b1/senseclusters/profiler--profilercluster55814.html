profiler

noun
program performance analyzer
6.3
Pronunciation: /ˈproʊ.faɪ.lər/ (AmE), /ˈprəʊ.faɪ.lə/ (BrE)
a computer program that analyzes how another program runs to identify performance issues such as slow functions or high memory use

Examples

Software profiler
The developer used a profiler to check which parts of the code were running slowly.
After running the profiler, she discovered that one function was using too much memory.
A good profiler can help programmers make their applications faster and more efficient.

Related Terms

  • synonym: debugger - a tool used to test and find errors in computer programs
  • synonym: optimizer - a tool or program that improves the efficiency of code
