Examples

Synonyms

  • computer screen
    2.1 4.44/m
    computer screen is the everyday name for the flat glass you look at, while computer display sounds slightly more formal or technical.
  • computer monitor
    2.7 1.05/m
    computer monitor is the whole device with a screen and electronics used at a desk, while computer display means the screen showing images.
  • monitor
    3.3 10/m
    monitor is a general device that shows images from many sources, while computer display specifically names the screen showing computer output.
  • visual display unit
    4.1 0.00/m
    visual display unit is a technical British term for a monitor used in offices, while computer display is the everyday term for the screen.
