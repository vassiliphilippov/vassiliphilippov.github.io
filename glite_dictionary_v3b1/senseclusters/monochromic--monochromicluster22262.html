monochromic

adjective
having only one color
5.5
Pronunciation: / ˌmɑːnəˈkroʊmɪk / (AmE), / ˌmɒnəˈkrəʊmɪk / (BrE)
Having or appearing to have only one color; using shades of a single color.

Examples

Monochromic color scheme
Her living room has a simple, monochromic color scheme that looks very elegant.
Older computer monitors often displayed images in a monochromic format.
The artist created a monochromic painting using only shades of blue.
