functor

noun
function on functions
Pronunciation: / ˈfʌŋktər / (AmE), / ˈfʌŋktə / (BrE)
a function, operator, or structure in mathematics or computing that maps between categories or applies to other functions or objects

Examples

Mathematical functor
In category theory, a functor maps objects and morphisms between categories.
The concept of a functor is essential in functional programming languages.
A functor can be used to apply a function to every element in a data structure.
In category theory, a functor maps objects and morphisms from one category to another.
Many programming languages use the concept of a functor to handle data transformations.
Understanding how a functor works can help you write more flexible code.
