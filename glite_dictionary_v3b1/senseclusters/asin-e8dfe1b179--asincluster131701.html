ASIN

noun
Amazon product identifier
8.2
Pronunciation: /ˈeɪ.sɪn/ /; / /ˈeɪ.es.aɪ.en/ (AmE), /ˈeɪ.sɪn/ /; / /ˈeɪ.es.aɪ.en/ (BrE)
A unique alphanumeric identifier assigned by Amazon to a product or item in its online catalog, used to find, track, and reference that product.

Examples

Amazon ASIN code
She copied the ASIN from the Amazon listing to compare prices.
Enter the product's ASIN into the search box to find matching listings quickly.
The seller updated each item's database record with its correct ASIN last night.

Related Terms

  • synonym: SKU - a stock-keeping unit; a seller's code for tracking products (similar function to an ASIN)
  • synonym: identifier - a label or code used to uniquely identify something
