decomp

verb
to decompile code
Pronunciation: / diːˈkɑmp / (AmE), / diːˈkɒmp / (BrE)
to decompile a compiled computer program, converting executable code back into readable source code

Examples

To decomp a program
The developer had to decomp the app to understand how it worked.
Before making changes, they decided to decomp the binary file for analysis.
She used a special tool to decomp the program and recover the original code.

Related Terms

  • antonym: compile - to convert source code into executable code
  • synonym: decompile - to convert executable code into source code
