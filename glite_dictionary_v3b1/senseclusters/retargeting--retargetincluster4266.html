retargeting

noun
adapting for new platform
7.2
Pronunciation: / ˌriːˈtɑːrɡətɪŋ / (AmE), / ˌriːˈtɑːɡɪtɪŋ / (BrE)
the process of modifying software or systems so they can operate on a different hardware platform or environment

Examples

Retargeting software platforms
The development team spent months on retargeting the application for the new mobile operating system.
Successful retargeting allows software to reach a wider range of devices and users.
After retargeting, the program could run efficiently on both Windows and Linux platforms.

Related Terms

  • synonym: porting - the process of adapting software to run on a different system
