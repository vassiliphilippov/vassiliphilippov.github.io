DOS

noun
early disk-based operating system
7.5 3.59/m
Pronunciation: / di o ä …õs / (AmE), / diÀê …í …õs / (BrE)
an early computer operating system that runs from a disk, used widely in personal computers, especially IBM compatibles, in the 1980s and 1990s

Examples

DOS commands
Many early programmers learned to use DOS commands before graphical interfaces became popular.
The company's first product was a word processor that operated under DOS.
When I was a child, my family's computer ran DOS rather than Windows.

Synonyms

  • MS-DOS
    6.7 0.33/m
    MS-DOS is the Microsoft operating system used on many 1980s and 1990s IBM-compatible PCs, while DOS names that same disk-based system.
  • disk operating system
    4.4 0.00/m
    disk operating system is any operating system stored and run from a disk in early microcomputers, while DOS refers to the specific 1980s and 1990s PC system.
