order Araneae

noun
taxonomic order of spiders
Pronunciation: / ˈɔrdɚ ærəˈniː.iː / (AmE), / ˈɔːdə ˌærəˈniː.iː / (BrE)
the taxonomic order that comprises all spiders

Examples

The order Araneae
All true spiders belong to the order Araneae, which is characterized by the production of silk.
Scientists have discovered thousands of new species within the order Araneae in recent years.
The order Araneae includes both common house spiders and more exotic species.
