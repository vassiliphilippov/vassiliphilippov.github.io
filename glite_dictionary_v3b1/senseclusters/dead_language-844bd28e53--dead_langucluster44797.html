dead language

noun
a language no longer spoken as a native tongue
Pronunciation: / dɛd ˈlæŋɡwɪdʒ / (AmE), / dɛd ˈlaŋɡwɪdʒ / (BrE)
A language that is no longer learned or spoken as a native language by any community.

Examples

A dead language
Latin is considered a dead language because no one speaks it as a first language anymore.
Many ancient texts are written in dead languages, such as Sumerian or Ancient Egyptian.
Scholars often study a dead language to gain insight into historical cultures.

Related Terms

  • synonym: extinct language - a language that no longer has any speakers at all
  • antonym: living language - a language that is currently spoken as a native tongue
