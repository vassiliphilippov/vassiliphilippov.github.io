control operation

noun
operation regulating data handling
7.1
Pronunciation: / kənˈtroʊl ˌɑːpəˈreɪʃən / (AmE), / kənˈtrəʊl ˌɒpəˈreɪʃən / (BrE)
an operation that initiates, modulates, or stops the recording, processing, transmission, or interpretation of data, especially in computing or electronic systems

Examples

A control operation process
Each control operation must be verified before processing sensitive information.
The engineer initiated a control operation to start the backup system.
Without a proper control operation, the device could malfunction during data transmission.
