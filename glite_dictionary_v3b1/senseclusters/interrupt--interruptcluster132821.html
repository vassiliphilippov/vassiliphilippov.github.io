interrupt

noun
computing: temporary signal stopping a program
Pronunciation: / ˌɪntəˈrʌpt / (AmE), / ˌɪntəˈrʌpt / (BrE)
in computing, a signal that temporarily stops the execution of a program or process so that another procedure can be carried out

Examples

A hardware interrupt
Device drivers often respond to hardware interrupts that signal an event has occurred.
The CPU received an interrupt that caused it to pause the current task.
When a key is pressed, it generates an interrupt to alert the operating system.

Related Terms

  • synonym: signal - electronic communication alerting or triggering an action, especially in computing
