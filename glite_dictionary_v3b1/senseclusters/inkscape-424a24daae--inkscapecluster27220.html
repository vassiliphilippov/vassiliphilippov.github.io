Inkscape

noun
vector graphics file
Pronunciation: / ˈɪŋkˌskeɪp / (AmE), / ˈɪŋkˌskeɪp / (BrE)
a digital vector-graphics file created with or intended to be edited using the Inkscape software, usually in SVG format

An Inkscape file
She sent me the Inkscape so I could update the company logo before printing.
The designer prefers working with an Inkscape because it allows for easy editing of vector shapes.
Please upload the Inkscape to the shared drive so the team can access the latest version.
