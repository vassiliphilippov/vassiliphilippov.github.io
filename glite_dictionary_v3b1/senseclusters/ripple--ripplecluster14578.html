ripple

noun
UI touch/click animation
5.3
Pronunciation: /ˈrɪp.əl/ (AmE), /ˈrɪp.əl/ (BrE)
A circular visual effect in a graphical user interface that expands outward from the point of touch or click to indicate interaction or provide feedback.

Examples

Screen ripple effect
When she touched the screen, a small ripple spread from her finger to the button's edge.
The designer adjusted the animation so the ripple lasted just long enough to reassure users.
Native Android controls draw a circular ripple on touch as immediate visual feedback for interactions.

Related Terms

  • synonym: animation - a sequence of changing images or effects used to convey motion or state changes in an interface
  • synonym: visual feedback - a visual indication or response in a user interface that confirms an action or shows state
