goto

noun
programming jump statement
Pronunciation: / ˈɡoʊtuː / (AmE), / ˈɡəʊtuː / (BrE)
a programming statement that causes the execution of code to jump to another specified location in the program

Examples

Goto statement in code
Many programmers avoid using a goto because it can make code harder to follow.
The old software relied heavily on goto statements for controlling program flow.
Replacing a goto with structured loops can improve code readability and maintenance.

Related Terms

  • synonym: jump - a statement that transfers control to another part of a program
  • antonym: loop - a programming structure that repeats a set of instructions
