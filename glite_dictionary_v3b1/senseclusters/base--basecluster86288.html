base

noun
fundamental number in a numeration system
3.2
Pronunciation: / beɪs / (AmE), / beɪs / (BrE)
The positive integer that serves as the fundamental number in a numeration system, such that each digit position represents a power of this number.

Examples

Base 10 system
In the binary system, the base is 2.
The decimal system uses 10 as its base.
When converting numbers between different systems, knowing the base is essential.
