(in computing, especially IT/DevOps) a file or script that defines a set of automated tasks to configure, deploy, or manage computer systems.

Examples

IT playbook
The team updated the playbook to patch all servers across environments.
She ran the Ansible playbook to deploy the new version to the staging cluster.
Before the incident, they stored their disaster-recovery playbooks in a secure repository for automation.
