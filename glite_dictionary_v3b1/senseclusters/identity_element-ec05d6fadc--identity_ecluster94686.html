identity element

noun
element that leaves others unchanged under an operation
Pronunciation: / aɪˈdɛntəti ˈɛləmənt / (AmE), / aɪˈdɛntəti ˈɛlɪmənt / (BrE)
An element in a set which, when combined with any other element using a given operation, leaves that element unchanged.

Examples

The identity element in math
Every group in mathematics includes a unique identity element.
In matrix multiplication, the identity matrix serves as the identity element.
The number 0 is the identity element for addition in the set of integers.

Related Terms

  • synonym: neutral element - element that does not change other elements in an operation
