memory apparatus

noun
obsolete device for presenting memorization stimuli
8.1
Pronunciation: / ˈmɛməri ˌæpəˌrætəs / (AmE), / ˈmɛməri ˌapəˈreɪtəs / (BrE)
an outdated mechanical device used to present a series of items or stimuli for the purpose of memorization, especially in psychological experiments

Examples

A memory apparatus device
A memory apparatus can still be found in some museum collections as a relic of psychological research.
Researchers found that the memory apparatus improved recall by controlling the timing of each cue.
The psychologist demonstrated the memory apparatus used in early 20th-century experiments on learning.
