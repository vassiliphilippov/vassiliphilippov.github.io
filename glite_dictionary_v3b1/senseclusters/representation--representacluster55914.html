(in computing and machine learning) a numerical encoding (often a vector) that stands for an object, concept, or piece of data so that algorithms can process and compare it.

Examples

Word representation vector
In NLP, a word's representation is a list of numbers that captures its meaning.
Researchers train models to learn useful representations from large amounts of text.
A dense representation lets the classifier compare items by simple mathematical distance.

Synonyms

  • embedding
    7.6 0.00/m
    embedding is a numeric vector that maps items into a continuous space for machine learning, while representation is any numeric encoding used by algorithms.
  • internal representation
    8.0 0.04/m
    internal representation is a model's learned encoding of inputs used inside networks for prediction, while representation is a broader numeric encoding for algorithms to process data.

Related Terms

  • synonym: encoding - a representation of data in a particular form suitable for processing
  • synonym: feature vector - a list of numerical features that represents an object for use in algorithms
