stored-program

adjective
computer architecture with instructions stored in memory
3.9
Pronunciation: / ˈstɔrd ˈproʊɡræm / (AmE), / ˈstɔːd ˈprəʊɡræm / (BrE)
Relating to a computer architecture in which instructions to be executed by the computer are stored in its memory.

Examples

Stored-program computer
A stored-program computer can change its function by loading different software.
Modern CPUs operate on the stored-program principle.
The stored-program concept revolutionized early computer design.
