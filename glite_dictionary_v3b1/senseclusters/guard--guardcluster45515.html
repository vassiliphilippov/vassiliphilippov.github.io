guard

noun
programming early-exit check
7.4
Pronunciation: / ɡɑrd / (AmE), / ɡɑːd / (BrE)
(in programming) a statement or clause that checks a condition at the start of a block or function and causes an immediate exit or alternative path if the condition is not met.

Examples

Programming guard clause
In Swift, add a guard at the start of the function to return early on invalid input.
The code uses a guard to ensure the user object exists before proceeding with the operation.
Reviewers praised her use of a guard clause to simplify nested conditionals and improve readability.

Related Terms

  • synonym: guard clause - a conditional check placed at the start of a function that returns or breaks early when a requirement is not met
  • synonym: precondition - a condition that must be true before a piece of code runs; often verified using a guard
