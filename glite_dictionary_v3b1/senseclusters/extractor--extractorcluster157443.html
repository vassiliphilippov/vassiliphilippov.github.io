extractor

noun
software tool that extracts data
6.1
Pronunciation: /ɪk.ˈstræk.tər/ (AmE), /ɪk.ˈstræk.tə/ (BrE)
software that automatically identifies and extracts structured data from unstructured or semi-structured sources such as text, web pages, or databases.

Examples

Data extractor
The company uses an extractor to pull data from multiple websites.
Data scientists built a custom extractor to convert messy HTML tables into a clean CSV file.
During testing the team compared several extractors for speed and accuracy on real web pages.

Related Terms

  • synonym: parser - software that analyses text to identify and structure its components
  • synonym: web scraper - a program that automatically collects data from websites
