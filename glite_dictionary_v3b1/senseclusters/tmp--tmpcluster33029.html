tmp

noun
temporary computer file
Pronunciation: / ti ɛm pi / (AmE), / ti ɛm pi / (BrE)
a temporary file created by a computer program to store intermediate or unsaved data, often with the extension .tmp

Examples

Tmp file found
After the software crashed, I found a tmp file in the project folder.
You should regularly delete old tmp files to free up disk space.
The program automatically creates a tmp to store unsaved changes.

Related Terms

  • synonym: cache - a storage location for temporary data to speed up processes
  • synonym: temporary file - a file created to hold data temporarily while a program is running
