collector

noun
software component that gathers telemetry
Pronunciation: / kəˈlɛktər / (AmE), / kəˈlɛktə / (BrE)
(in computing) a software component or service that gathers telemetry, logs, metrics, or usage data from devices or applications and forwards them for storage or analysis.

Examples

Data collector
The cloud service uses a collector to gather performance metrics from all nodes.
During troubleshooting, the engineers deployed several collectors to capture logs from the failing microservices.
A lightweight collector running on each device sends telemetry to the central monitoring system every minute.

Related Terms

  • synonym: agent - a small program that runs on a host to collect and send monitoring data
  • synonym: aggregator - a system or component that collects and combines data from multiple sources
