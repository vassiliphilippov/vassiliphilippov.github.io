return the favor

verb
repay a kindness or favor
1.0
Pronunciation: / rɪˈtɝn ðə ˈfeɪvɚ / (AmE), / rɪˈtɜːn ðə ˈfeɪvə / (BrE)
to do something nice or helpful for someone who has previously done something for you

Examples

To return the favor kindly
After years of mentoring, she was glad to have the chance to return the favor.
He offered to fix my computer, so I'll return the favor by treating him to lunch.
I'll be happy to return the favor if you ever need a ride somewhere.
