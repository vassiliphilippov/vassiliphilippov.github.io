eye dropper

noun
color-sampling tool in graphics software
3.8
Pronunciation: /aɪ ˈdrɑːp.ər/ (AmE), /aɪ ˈdrɒp.ə/ (BrE)
(in image-editing or graphics software) a tool that samples or picks a color value from an image or screen so the same color can be used elsewhere.

Examples

Eye dropper tool
In Photoshop, use the eye dropper to pick the exact color from the image.
The designer used the eye dropper to match the button color across different files.
When recording the tutorial, she showed how the eye dropper reads hexadecimal color codes.

Related Terms

  • synonym: color picker - a software tool used to select or sample colors from an image, palette, or screen
  • synonym: pipette - a small tube used to transfer liquid; the physical instrument that gave rise to the software tool's name
