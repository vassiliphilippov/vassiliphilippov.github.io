Microsoft tax

noun
perceived extra cost for Microsoft software
5.7
Pronunciation: / ˈmaɪk.rəˌsɑft tæks / (AmE), / ˈmaɪ.krəˌsɒft tæks / (BrE)
an informal term for the perceived extra cost incurred by users for software upgrades or high-priced Microsoft products, especially when alternatives are unavailable or less compatible

Examples

The Microsoft tax cost
He argued that open source alternatives could reduce the Microsoft tax faced by consumers.
Many businesses factor in the Microsoft tax when budgeting for their IT needs.
Some users complain about the compulsory Microsoft tax on new computers sold with pre-installed Windows.
