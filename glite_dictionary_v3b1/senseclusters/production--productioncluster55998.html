production

noun
live software environment
Pronunciation: / prəˈdʌkʃən / (AmE), / prəˈdʌkʃən / (BrE)
(computing) the live, user-facing software environment where applications are deployed and used by end users.

Examples

Production server
We deployed the latest feature to production after comprehensive staging and testing.
The bug only appears in production, so the team scheduled an urgent rollback to fix it.
Monitor error rates in production continuously to detect issues before they affect many users.

Related Terms

  • synonym: deployment - the act of releasing or installing software into an environment, often used when moving changes to production
  • antonym: staging - a pre-production testing environment used to validate changes before deploying to production
