indexer

noun
software that builds search indexes
4.9
Pronunciation: /ˈɪn.dek.sər/ /; / /ˈɪn.dək.sər/ (AmE), /ˈɪn.dek.ə/ /; / /ˈɪn.dək.ə/ (BrE)
(computing) a software program or component that automatically builds, updates, or maintains an index of content to enable fast search and retrieval.

Examples

Search indexer
The operating system's indexer runs in the background to rebuild the search index.
Our database indexer updated its mappings overnight, so queries returned results much faster this morning.
During deployment, they disabled the indexer to prevent it from interfering with bulk data imports.

Related Terms

  • synonym: index - a data structure or list that enables fast lookup and search; the object created and used by an indexer.
  • synonym: search engine - software that finds and ranks information, often using indexes produced by an indexer.
