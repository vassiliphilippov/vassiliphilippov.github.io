function call

noun
instruction invoking a subroutine in programming
6.1
Pronunciation: / ˈfʌŋkʃən kɔl / (AmE), / ˈfʌŋkʃən kɔːl / (BrE)
an instruction in a computer program that passes control to a subroutine or function, after which control returns to the main program

Examples

A function call in code
Debugging the code revealed that an invalid function call caused the error.
Every time you make a function call, memory is allocated for arguments and local variables.
The function call triggered a series of calculations in the program.

Related Terms

  • antonym: return statement - an instruction that returns control from a function to the caller
