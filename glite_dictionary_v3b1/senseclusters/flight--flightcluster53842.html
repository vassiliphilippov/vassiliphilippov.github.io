flight

noun
set of stairs between floors
2.8 4.95/m
Pronunciation: / flaÉªt / (AmE), / flaÉªt / (BrE)
a set of stairs between one floor or landing and the next

Examples

A flight of stairs
Handrails ran along both sides of the narrow flight.
She paused to rest after climbing the steep flight between the first and second floor.
The building has a flight of stairs leading to the upper floor.

Synonyms

  • flight of stairs
    1.6 2.23/m
    flight of stairs is a continuous set of steps connecting two floors in a building, while flight is the single set between landings.
  • flight of steps
    3.0 0.38/m
    flight of steps is a short run of stairs often between two levels, while flight is the specific set between one floor and the next.
  • stairs
    1.0 0.51/m
    stairs are the whole series of steps people use to move between floors in a building, while flight is one short set.
  • stairway
    1.0 4.84/m
    stairway is a built route of steps inside or outside a building, while flight is a single run between two landings.
  • staircase
    1.0 8.89/m
    staircase is a full built structure often with a banister and support, while flight is merely the set of steps between floors.
  • steps
    1.0 0.00/m
    steps often refer to an outside set of shallow stairs leading to a door or garden, while flight is the indoor set between floors.
