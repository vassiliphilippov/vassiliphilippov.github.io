point size

noun
size of plotted markers
3.4
Pronunciation: /pɔɪnt saɪz/ (AmE), /pɔɪnt saɪz/ (BrE)
in graphs or plotting software, the diameter or visual size of a data marker; a parameter that controls how large plotted points appear.

Examples

Point size in graphs
In the plotting software, you can adjust the point size to emphasize data clusters.
She increased the point size on the scatter plot to make outliers more visible to reviewers.
When exporting the figure, check the point size so markers remain clear at publication scale.

Related Terms

  • synonym: marker size - the size or diameter of a marker used to represent data points on a plot
  • synonym: symbol size - the displayed size of symbols or points used in charts and graphs
