ARIA

noun
web accessibility attributes (ARIA)
Pronunciation: / eɪ.ɑr.aɪ.eɪ / (AmE), / eɪ.ɑː.aɪ.eɪ / (BrE)
An acronym for Accessible Rich Internet Applications: a set of HTML attributes, roles, and practices used to make dynamic web content and user interfaces accessible to people who use assistive technologies.

Examples

ARIA attributes
The developer added ARIA attributes to improve screen reader navigation on the website.
Before launch, they tested the product with assistive tech and fixed several ARIA issues.
Most modern frameworks include components that automatically apply appropriate ARIA roles and attributes.

Related Terms

  • synonym: WAI-ARIA - The W3C working-group specification that defines ARIA roles, states, and properties for web accessibility.
  • synonym: accessibility - Designing websites and applications so people with disabilities can perceive, understand, navigate, and interact with them.
