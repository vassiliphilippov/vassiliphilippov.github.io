initialize

verb
set an initial value (computing)
Pronunciation: / ɪˈnɪʃəˌlaɪz / (AmE), / ɪˈnɪʃəlaɪz / (BrE)
to set the initial value of a variable, object, or other element in computing; to prepare something for use by assigning a starting value

Examples

To initialize a variable
Before running the program, you need to initialize all variables.
She wrote a function to initialize the database with default values.
The system failed to initialize the hardware properly.
Before using the array, you must initialise all its elements to zero.
He spent the morning initialising the new database to prepare it for testing.
The program will not run correctly if you forget to initialise the variables.

Related Terms

  • synonym: initialize - assign an initial value to a computer program (US spelling)
  • synonym: reset - to set something again to its initial state
  • antonym: terminate - to bring to an end; in computing, to stop a process
