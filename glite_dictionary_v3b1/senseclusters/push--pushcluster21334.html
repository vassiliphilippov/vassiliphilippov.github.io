push

verb
server-initiated send of data
3.6 6.23/m
Pronunciation: / p\u028a\u0283 / (AmE), / p\u028a\u0283 / (BrE)
(in computing) to send updates, messages, or data from a server or service to a client or device without the client first requesting them.

Examples

To push updates
The server can push updates to devices in real time.
To reduce latency, engineers pushed new configurations to edge nodes overnight.
The messaging service is pushing urgent alerts to subscribers during the outage.

Synonyms

  • push out
    3.5 0.32/m
    push out releases updates to users or the market, while push is a server action that sends updates without the client asking.

Related Terms

  • antonym: pull - a model where the client requests data from the server (opposite of push in computing)
  • synonym: send - to cause data or a message to be transmitted to another device or person
