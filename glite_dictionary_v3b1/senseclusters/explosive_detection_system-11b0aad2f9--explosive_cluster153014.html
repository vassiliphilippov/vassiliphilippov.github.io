explosive detection system

noun
automated system for detecting explosives, often in airports
Pronunciation: / ɪkˈsploʊsɪv dɪˈtɛkʃən ˈsɪstəm / (AmE), / ɪkˈspləʊsɪv dɪˈtɛkʃən ˈsɪstəm / (BrE)
an automated system that uses X-ray technology and computers to rapidly detect plastic explosives in luggage, especially at airports

Examples

Airport explosive detection system
After the bomb threat, authorities upgraded the explosive detection system at all terminals.
Security officers rely on the explosive detection system to quickly screen checked luggage for banned materials.
The airport installed a new explosive detection system to enhance passenger safety.
