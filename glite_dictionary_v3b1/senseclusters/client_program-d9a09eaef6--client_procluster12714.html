client program

noun
software that accesses a remote server
4.0 3.9
Pronunciation: / ˈklaɪənt ˌproʊɡræm / (AmE), / ˈklaɪənt ˌprəʊɡræm / (BrE)
A software application that runs on a user's computer and accesses services or data from a remote server over a network.

Examples

A client program accesses server
A web browser is a common example of a client program that retrieves webpages from servers.
Companies use a client program to access their databases stored on remote servers.
You must install the client program before you can connect to the company's online services.

Related Terms

  • synonym: client - a computer or program that accesses a service provided by a server
  • antonym: server - a computer or program that provides data or services to client programs
