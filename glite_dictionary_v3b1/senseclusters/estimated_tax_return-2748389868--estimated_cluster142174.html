estimated tax return

noun
tax return for reporting estimated taxes
5.0 5.6
Pronunciation: / ˈɛstəˌmeɪtɪd tæks rɪˈtɝn / (AmE), / ˈɛstɪmeɪtɪd tæks rɪˈtɜːn / (BrE)
a tax return filed by a taxpayer to report and pay estimated taxes when tax withheld from income does not satisfy the year’s expected tax liability

Examples

An estimated tax return filed
Failure to submit an estimated tax return on time can result in fines from the tax authorities.
Freelancers often complete an estimated tax return each quarter to avoid penalties.
If you expect to owe more than $1,000 in taxes, you may need to file an estimated tax return.
