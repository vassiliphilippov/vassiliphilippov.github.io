test driver

noun
software program that runs automated tests

A software (or hardware) program or component that runs automated tests for another program or device by supplying inputs, controlling modules, and recording results.

Examples

Software test driver
The unit tests were automated by a test driver that exercises each module and reports failures.
Before integration, the team built a simple test driver to simulate inputs and capture outputs.
The embedded system requires a hardware test driver to run diagnostics during manufacturing.

Related Terms

  • synonym: test harness - a framework of code and tools used to run tests and provide inputs, stubs, or mocks
  • synonym: test runner - a tool that executes test cases and reports their results
