square matrix

noun
matrix with equal rows and columns
Pronunciation: / skwɛr ˈmeɪtrɪks / (AmE), / skwɛː ˈmatrɪks / (BrE)
a matrix in which the number of rows is equal to the number of columns

Examples

A square matrix
A square matrix is essential for calculating determinants.
Every square matrix has the same number of rows and columns.
The identity matrix is always a square matrix.

Related Terms

  • antonym: rectangular matrix - a matrix with a different number of rows and columns
