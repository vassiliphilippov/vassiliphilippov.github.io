interquartile

adjective
between quartiles
7.3
Pronunciation: / ˌɪntərˈkwɔrtaɪl / (AmE), / ˌɪntəˈkwɔːtaɪl / (BrE)
relating to the range between the first and third quartiles in a set of statistical data

Examples

The interquartile range
The interquartile range is often used to measure the spread of exam scores.
Researchers reported the interquartile values to show the variability in the data.
A box plot displays the interquartile range as the box itself.

Related Terms

  • synonym: quartile - one of four equal groups into which data can be divided
