cardinality

noun
number of elements in a set (math)
6.6
Pronunciation: / ˌkɑːrdɪˈnælɪti / (AmE), / ˌkɑːdɪˈnælɪti / (BrE)
(mathematics) The number of elements in a set or group, considered as a property of that set or group.

Examples

Set cardinality
A set with infinite cardinality cannot be counted by natural numbers.
In database design, understanding the cardinality of relationships is essential.
The cardinality of the set {2, 4, 6, 8} is four.

  • synonym: size - the total number of elements in a mathematical set
