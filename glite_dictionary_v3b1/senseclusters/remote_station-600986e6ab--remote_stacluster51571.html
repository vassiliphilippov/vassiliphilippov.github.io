remote station

noun
terminal linked remotely to a computer
1.0
Pronunciation: / rɪˈmoʊt ˈsteɪʃən / (AmE), / rɪˈməʊt ˈsteɪʃən / (BrE)
a terminal or device located at a distance from a central computer and connected to it by a data link

Examples

Remote station device
Each remote station collects data and transmits it back to the main server.
The company installed a new remote station to monitor the pipeline from a distant location.
The technician troubleshot communication issues between the remote station and the central system.
