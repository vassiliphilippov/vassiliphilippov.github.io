umpty

noun
long Morse signal
Pronunciation: / ˈʌmpti / (AmE), / ˈʌmpti / (BrE)
an informal term for a dash or long signal in Morse code or telegraphy

Examples

Umpty in Morse code
The operator tapped out an umpty to signal the start of the message.
In old telegraphy, a umpty was used to represent a dash.
He explained that each umpty in Morse code stands for a long sound.
