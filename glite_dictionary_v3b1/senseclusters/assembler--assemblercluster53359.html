assembler

noun
one who puts together
4.2
Pronunciation: / əˈsɛmblər /; / əˈsɛmbələr / (AmE), / əˈsɛmblə /; / əˈsɛmbələ / (BrE)
a person or program that puts parts together to make a complete product or system

Examples

Factory assembler
The assembler at the electronics plant carefully put together each circuit board.
A software assembler converts assembly language code into machine code for computers.
Each assembler on the team is responsible for a different part of the final product.
The assembler checked each part before putting them together on the production line.
As an experienced assembler, she could build a computer in under an hour.
Many assemblers work in teams to speed up the manufacturing process.

Related Terms

  • synonym: builder - a person who constructs or puts things together
  • synonym: compiler - a computer program that translates code from one language to another
  • antonym: disassembler - a computer program that converts machine code into assembly language
