drop line

noun
indented style headline (publishing)
Pronunciation: / drɑp laɪn / (AmE), / drɒp laɪn / (BrE)
A headline in which the first line is aligned with the left margin and each following line is progressively indented to the right.

Examples

A drop line headline
Designers experimented with a drop line to create a dramatic visual effect.
Many newspapers employed a drop line format in the 20th century.
The editor suggested using a drop line to make the headline stand out.
