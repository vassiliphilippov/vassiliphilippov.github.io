security review

noun
assessment of software security
4.0
Pronunciation: /sɪ.ˈkjʊr.ɪ.ti rɪ.ˈvjuː/ /; / /sə.ˈkjʊr.ɪ.ti rɪ.ˈvjuː/ /; / /sɪ.ˈkjʊr.ə.ti rɪ.ˈvjuː/ (AmE), /sɪ.ˈkjʊə.rɪ.ti rɪ.ˈvjuː/ /; / /sə.ˈkjʊə.rɪ.ti rɪ.ˈvjuː/ /; / /sɪ.ˈkjʊə.ti rɪ.ˈvjuː/ (BrE)
An assessment of the security of a software system, product, or service, including architecture, code, configurations, and threat models, carried out to identify vulnerabilities and recommend fixes.

Examples

Software security review
Before release, the team performed a security review of the web application to identify vulnerabilities.
The startup hired external experts to conduct a formal security review of their API and cloud configuration.
As part of compliance, each major change must undergo a security review before deployment to production.

Related Terms

  • synonym: security assessment - a systematic evaluation of an information system to determine the extent to which it meets specified security requirements
  • synonym: security audit - a formal review of an organization's security policies, controls, and practices to ensure compliance and effectiveness
