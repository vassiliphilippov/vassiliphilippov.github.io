SSI

noun
web scripting language
8.2
Pronunciation: /es es aɪ/ (AmE), /es es aɪ/ (BrE)
a simple server-side scripting language used to insert dynamic content into web pages, short for Server Side Includes

Examples

SSI code
The developer used SSI to automatically display the last updated date on every page.
Many older websites rely on SSI for simple dynamic content like headers and footers.
Learning SSI can help you understand how web servers process dynamic information.
