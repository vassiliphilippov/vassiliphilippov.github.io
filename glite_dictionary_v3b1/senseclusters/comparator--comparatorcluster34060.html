comparator

noun
function for ordering
6.3
Pronunciation: /kəm.ˈpær.ə.tɜːr/ (AmE), /kəm.ˈpær.ə.tə/ (BrE)
a function or routine that determines the order of elements by comparing them, especially in programming

Examples

Sorting comparator
The comparator determines whether one number should come before another in the sorted list.
You can write a custom comparator to sort the objects by their names.
If the comparator returns zero, the two elements are considered equal for sorting purposes.

Related Terms

  • synonym: comparator function - a function used to compare two values for sorting
  • synonym: sorter - a function or device that arranges items in order
