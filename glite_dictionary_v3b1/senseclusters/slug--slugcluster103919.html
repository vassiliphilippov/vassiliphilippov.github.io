slug

noun
URL-friendly page identifier
6.9
Pronunciation: / slʌg / (AmE), / slʌg / (BrE)
a short, URL-friendly string, usually derived from a page title, used to identify and locate a specific web page in an address.

Examples

URL slug
The article's slug was generated from its title to create a clean URL.
She edited the slug to include keywords that would help people find the page.
When migrating sites, remember to redirect old slugs so existing links continue to work.

Related Terms

  • synonym: URL - the address of a web page on the internet; slugs form part of a URL path
  • synonym: permalink - a permanent URL that links directly to a specific web page or post
