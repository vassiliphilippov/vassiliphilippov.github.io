datalink

noun
data connection
4.1
Pronunciation: /ˈdæt.ə.lɪŋk/ /; / /ˈdeɪ.tə.lɪŋk/ (AmE), /ˈdeɪ.tə.lɪŋk/ /; / /ˈdæt.ə.lɪŋk/ (BrE)
a system or channel that allows electronic devices or systems to exchange data with each other

Examples

A secure datalink
The engineers set up a secure datalink between the control center and the satellite.
A reliable datalink is essential for real-time communication in modern aircraft.
The new software improved the speed of the datalink connecting the two servers.
The pilot relied on the datalink to receive real-time instructions from the control tower.
Modern satellites use a secure datalink to send telemetry back to Earth.
Engineers tested the new datalink system to ensure reliable communication between vehicles.
The engineer checked the datalink to ensure reliable communication between the two computers.
A failure in the datalink can disrupt the entire local area network.
Modern routers often include advanced datalink features for better error correction.

Related Terms

  • synonym: interface - a point where two systems or devices connect and interact
  • synonym: network - a system of connected computers or devices
  • antonym: network layer - the layer responsible for routing data between networks (OSI layer 3)
  • antonym: physical layer - the layer that transmits raw bits over a physical medium (OSI layer 1)
  • synonym: telemetry - the process of recording and transmitting data from remote sources
