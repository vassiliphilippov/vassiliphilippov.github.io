size constancy

noun
perceiving objects as constant in size despite distance
9.9 5.6
Pronunciation: /saɪz ˈkɑːn.stən.siː/ (AmE), /saɪz ˈkɒn.stən.siː/ (BrE)
the tendency to perceive the actual size of a familiar object as constant, even when its distance from the observer changes and the image on the retina varies in size

Examples

Size constancy in perception
Children develop size constancy as they learn to recognize objects at different distances.
Even as the car drove away, size constancy allowed Sam to perceive it as being the same size.
The experiment was designed to test how lighting conditions affect size constancy in visual perception.
