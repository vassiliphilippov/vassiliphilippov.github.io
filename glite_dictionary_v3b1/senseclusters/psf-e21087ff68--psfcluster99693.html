PSF

noun
molecular structure file
Pronunciation: / pi ɛs ɛf / (AmE), / piː ɛs ɛf / (BrE)
a file format used in molecular modeling to describe the structure, topology, and connectivity of a molecular system

Examples

Molecular PSF file
The researcher created a PSF to define the molecular system for the simulation.
Before running the analysis, you must ensure the PSF matches the coordinate file.
Many molecular modeling programs can read and write the PSF format for complex systems.

Related Terms

  • synonym: PDB - a file format for three-dimensional structures of molecules
  • synonym: topology file - a file describing the connectivity of atoms in a molecule
