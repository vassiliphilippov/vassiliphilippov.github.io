orthographically

adverb
using orthographic projection
Pronunciation: / ˌɔrθəˈɡræfɪkli / (AmE), / ˌɔːθəˈɡræfɪkli / (BrE)
in a way that uses orthographic projection, showing objects without perspective so their dimensions remain accurate.

Examples

Orthographically drawn parts
The engineer drew the machine parts orthographically to ensure every measurement was precise.
In technical drawings, objects are often represented orthographically for clarity and accuracy.
The software allows users to view the 3D model orthographically to check the true dimensions.

Related Terms

  • synonym: isometrically - in a way that uses isometric projection, showing three-dimensional objects with equal scale along each axis
