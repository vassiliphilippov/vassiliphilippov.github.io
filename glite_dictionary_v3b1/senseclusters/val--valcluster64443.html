val

noun
immutable variable keyword
9.9
Pronunciation: / væl / (AmE), / væl / (BrE)
a reserved word in some programming languages used to declare an immutable (read-only) variable

Examples

Val keyword in code
In Kotlin, you use val to declare a read-only variable that cannot be reassigned.
The developer prefers using val for values that should not change after initialization.
If you need a constant value in Scala, declare it with val instead of var.

Related Terms

  • synonym: constant - a value that cannot be changed during program execution
  • antonym: var - a reserved word used to declare a mutable variable
