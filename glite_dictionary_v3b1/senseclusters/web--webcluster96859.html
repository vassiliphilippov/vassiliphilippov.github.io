web

noun
the World Wide Web; internet system of linked resources
2.7
Pronunciation: / wɛb / (AmE), / wɛb / (BrE)
the system of connected documents and resources on the internet (the World Wide Web), or a global network of linked data in computing

Examples

On the web
Children learn how to use the web safely at school.
Many businesses now depend on the web to reach their customers.
She found the information she needed by searching the web.
Researchers often describe the web of linked data as enabling automated discovery across scientific domains.
The museum published its collection metadata to the web so other institutions could query it programmatically.
Developers are building tools that make the web of data easier to search and integrate.

Related Terms

  • synonym: Semantic Web - the vision and standards for a web of linked, machine-readable data that supports automated processing
  • synonym: internet - the global system of interconnected computer networks
  • synonym: linked data - data published in a structured, interlinked form so it can be connected and queried across sources
  • degree lower: website - a set of related web pages under a single domain name
