argument

noun
variable in math or logic determining outcome
Pronunciation: / ˈɑːrɡjəmənt /; / ˈɑːrɡjumənt / (AmE), / ˈɑːɡjʊmənt / (BrE)
(mathematics, computing) a value or variable that is used as input for a function, procedure, or program

Examples

Function argument
In the function g(t) = t^2 + 1, the argument is the variable t.
The value of the argument determines the output of the sine function.
When programming, you often need to specify an argument for a function to run properly.
Each function in Python can take one or more arguments when it is called.
If you omit the required argument, the program may produce an error.
The command line allows users to specify multiple arguments to customize their output.

Related Terms

  • synonym: parameter - a quantity or value that defines a system or sets conditions within a function or operation (in computing)
