enclave

noun
secure isolated execution environment (computing)
7.4
Pronunciation: / \u02c8\u025bn.kle\u026av / (AmE), / \u02c8\u0252n.kle\u026av / (BrE)
(in computing) a secure, isolated execution environment provided by hardware or software that protects code and data from the rest of the system.

Examples

Secure enclave
The processor provides a trusted enclave that keeps encryption keys safe from the operating system.
Developers run sensitive algorithms inside a hardware enclave to prevent data leakage during computation.
During the audit they verified that the cloud provider isolated each tenant's keys within separate enclaves.

Related Terms

  • synonym: trusted execution environment - a protected area of a processor that runs code and stores data securely, isolated from the main operating system
