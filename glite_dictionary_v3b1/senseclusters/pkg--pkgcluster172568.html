pkg

proper noun
package manager program
Pronunciation: / piː keɪ dʒi / (AmE), / piː keɪ dʒi / (BrE)
the name of a command-line package management program used to install, remove, and update software on some Unix-like systems

Examples

Run pkg command
To update all installed software, simply run pkg upgrade in the terminal.
The FreeBSD system uses pkg to manage and install new applications.
If you want to remove a program, you can use the pkg delete command.

Related Terms

  • synonym: apt - a package management tool for Debian-based systems
  • antonym: remove - to take away or uninstall software
  • synonym: yum - a package manager for RPM-based Linux systems
