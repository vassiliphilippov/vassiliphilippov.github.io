init

verb
to initialize system
Pronunciation: / ˈɪnɪt / (AmE), / ˈɪnɪt / (BrE)
to start or set up a device, program, or system by performing initialization, especially in computing contexts

Examples

To init a system
Before running the application, you need to init the database to store user data.
The technician will init the system after installing the new hardware components.
If you forget to init the device, it may not function correctly.

Related Terms

  • synonym: initialize - to set up or prepare a system or program for use
  • antonym: shutdown - to turn off or stop a system or device
