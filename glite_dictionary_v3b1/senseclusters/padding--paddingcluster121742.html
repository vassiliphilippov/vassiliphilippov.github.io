(in computing) extra bytes or bits added to data so that it meets a required length or alignment, for example in block ciphers, file formats, or network frames.

Examples

Data padding
In cryptography, proper padding prevents attackers from exploiting incomplete encryption blocks.
The file format requires four bytes of padding after the header to align the data.
Network packets include padding so the payload meets the minimum frame size for transmission.
