health code

noun
official public health regulations
5.1
Pronunciation: / hɛlθ koʊd / (AmE), / hɛlθ kəʊd / (BrE)
a set of official rules or standards, established and enforced by government, relating to public health such as sanitation, food safety, or plumbing

Examples

Health code violations
Building inspectors review new constructions to ensure compliance with the local health code.
Changes to the city's health code have improved food safety standards in recent years.
The restaurant was shut down for violating several sections of the health code.

Related Terms

  • synonym: sanitary code - set of official rules about hygiene and sanitation
