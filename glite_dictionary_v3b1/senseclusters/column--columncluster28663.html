column

noun
a line of units one after another

A line of people, vehicles, or other units arranged one behind another.

Examples

A column of soldiers
A long column of cars followed the leader through the city.
The soldiers marched in a column down the main street.
They stood in a single column waiting to buy tickets at the booth.

Synonyms

  • file
    2.8 4.29/m
    a file is a line of people arranged one behind another, while a column often refers to organized groups like troops or vehicles.
  • single file
    2.8 1.11/m
    single file is a line where each person stands right behind another, while column refers to a long ordered group of people or vehicles.
  • Indian file
    7.6 0.00/m
    Indian file is a British phrase for people walking in a single line, while column is a general term for such ordered groups.
  • line
    1.0 50/m
    a line is a broad row or formation of people or things, while column means a single file with units one behind another.
