audit program

noun
list of procedures for an audit
Pronunciation: / ˈɔdɪt ˌproʊɡræm / (AmE), / ˈɔːdɪt ˌprəʊɡræm / (BrE)
a systematic list of procedures and steps to be followed when conducting an audit, detailing what is to be examined and how.

Examples

Detailed audit program
A comprehensive audit program helps auditors remain consistent and thorough during an investigation.
Before beginning fieldwork, the team reviewed the audit program for any necessary adjustments.
The accounting firm developed a detailed audit program to ensure all financial records were properly reviewed.
Changes to the audit programme were required when new regulations came into effect.
Following the audit programme, the team systematically checked each department's compliance with company policy.
The external auditors prepared an audit programme to ensure all key financial processes would be reviewed.

Related Terms

  • synonym: audit plan - an overall strategy and outline for the audit, including the scope, timing, and direction
