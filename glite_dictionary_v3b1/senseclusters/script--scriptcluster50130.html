script

noun
small computer program (often for browsers)
Pronunciation: / skrɪpt / (AmE), / skrɪpt / (BrE)
a short computer program, often written in a scripting language, that runs to automate tasks or add behavior (commonly run in web browsers or simple runtime environments).

Examples

Computer script
He installed a script that automatically fills forms on the page.
Web developers often write small scripts to add interactive features to a site.
The security team blocked a script that had been stealing users' login tokens.

Related Terms

  • synonym: macro - a small automated instruction or sequence used to automate repetitive tasks in software
  • synonym: program - a set of instructions that a computer executes to perform tasks
