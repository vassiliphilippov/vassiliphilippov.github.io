procedure

noun
set of instructions in programming
4.1 10/m
Pronunciation: / pr…ôÀàsid í…ö / (AmE), / pr…ôÀàsiÀêd í…ô / (BrE)
a set of instructions or operations in a computer program designed to perform a specific task

Examples

Computer procedure
Each procedure in the code is responsible for a different task.
The print procedure handles all output to the screen in this program.
You must call the initialization procedure before accessing the database.

Synonyms

  • subroutine
    6.5 0.96/m
    subroutine is a small reusable block of program instructions you call from different places, while procedure is a general set of program operations.
  • subprogram
    7.2 0.08/m
    subprogram is a named sequence of instructions inside a larger program used for one task, while procedure is the set of operations for that task.
  • routine
    3.8 1.70/m
    routine is a short program task that often runs regularly inside a system, while procedure is a defined set of instructions for one operation.
  • function
    4.9 51/m
    function maps inputs to outputs and often returns a value when used in code, while procedure is a set of program operations that may not return anything.

Related Terms
