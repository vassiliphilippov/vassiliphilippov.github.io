procedure

noun
set of instructions in programming
Pronunciation: / prəˈsidʒɚ / (AmE), / prəˈsiːdʒə / (BrE)
a set of instructions or operations in a computer program designed to perform a specific task

Examples

Computer procedure
Each procedure in the code is responsible for a different task.
The print procedure handles all output to the screen in this program.
You must call the initialization procedure before accessing the database.

Related Terms

  • synonym: function - a block of code that performs a specific task in programming
  • synonym: subroutine - a sequence of program instructions that perform a specific task, packaged as a unit
