focus on

verb
give keyboard/input focus to an element
3.3
Pronunciation: /ˈfoʊ.kəs ɑːn/ /; / /ˈfoʊ.kɪs ɑːn/ (AmE), /ˈfəʊ.kəs ɒn/ /; / /ˈfəʊ.kɪs ɒn/ (BrE)
(in computing, user interfaces) to give keyboard or input focus to a UI element so it can receive input or respond to keyboard events.

Examples

To focus on a field
The script will focus on the search field when the page loads.
After the modal opened, the browser automatically focused on the username field for faster login.
To improve accessibility, developers often program buttons to focus on form fields when clicked.

Related Terms

  • antonym: blur - to remove keyboard or input focus from a user-interface element
  • synonym: set focus - to programmatically give keyboard or input focus to a user-interface element
