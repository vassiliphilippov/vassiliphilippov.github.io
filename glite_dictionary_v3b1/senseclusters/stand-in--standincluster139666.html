stand-in

noun
software substitute used in testing
5.8
Pronunciation: /ˈstæn.dɪn/ /; / /ˈstæn.dən/ (AmE), /ˈstæn.dɪn/ /; / /ˈstæn.dən/ (BrE)
a non-human substitute used in software development or testing to take the place of a real component, service, or module (for example, a mock or stub).

Examples

A stand-in module
During testing, the team used a stand-in service to simulate the payment API.
The developer replaced the real database with several stand-ins to run unit tests quickly.
A lightweight stand-in returned canned responses when the external service was offline.

Related Terms

  • synonym: mock - a simulated object used in testing to mimic the behaviour of a real component
  • synonym: stub - a simple replacement for a component that provides fixed responses for testing
