quarter note

noun
musical note worth a quarter of a whole note
Pronunciation: / ˈkwɔrt̬ɚ ˌnoʊt / (AmE), / ˈkwɔːtə ˌnəʊt / (BrE)
a musical note with a duration equal to one fourth of a whole note

Examples

A quarter note sound
Each measure in four-four time contains exactly four quarter notes.
He tapped out the rhythm using only quarter notes on the piano.
When sight-reading, she mistook a quarter note for a half note and played it too long.

Related Terms

  • synonym: crotchet - British term for a quarter note
  • degree lower: eighth note - a musical note with half the value of a quarter note
