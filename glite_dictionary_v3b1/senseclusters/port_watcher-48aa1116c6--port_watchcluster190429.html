port watcher

noun
tool that monitors network port activity
Pronunciation: / pɔrt ˈwɑtʃər / (AmE), / pɔːt ˈwɒtʃə / (BrE)
A software program or hardware device that monitors network ports on a computer or network, reporting or logging open ports, connection attempts, and related activity for security or diagnostics.

Examples

Network port watcher
The administrator installed a port watcher to detect unauthorized port scans.
A simple port watcher alerted the team when the service unexpectedly opened a new port.
They used a cloud-based port watcher to continuously monitor thousands of virtual machines.

Related Terms

  • synonym: network monitor - a system that watches network traffic and devices for performance or security issues
  • synonym: port monitor - a tool that observes and reports on the status and activity of network ports
