monolith

noun
single-unit software system
7.1
Pronunciation: /ˈmɑː.nə.lɪθ/ /; / /ˈmɑːn.lɪθ/ (AmE), /ˈmɒn.ə.lɪθ/ /; / /ˈmɒn.lɪθ/ (BrE)
a software system built as a single, indivisible unit rather than as separate, independent services or modules.

Examples

Monolith software system
The legacy monolith architecture was costly to update and blocked rapid feature delivery.
After the merger, engineers decided to break the old monolith into smaller microservices for flexibility.
Startups often avoid building a monolith early to keep their codebase easy to change.

Related Terms

  • antonym: microservices - an architectural style that divides an application into small, independent services
  • synonym: monolithic architecture - a software design in which an entire application is built and deployed as a single unit
