loss

noun
error or cost function in machine learning
5.7 1/m
Pronunciation: / lÉ”s / (AmE), / lÉ’s / (BrE)
(in machine learning and statistics) a function or numeric value that measures the difference between predicted and true values; used to evaluate models and guide training (smaller loss indicates better fit).

Examples

Model loss value
During training, the model's loss decreased as it learned to predict housing prices more accurately.
We monitor the validation loss to detect overfitting before deciding to stop training the neural network.
Choosing cross-entropy as the loss helped the classifier separate cats from dogs more reliably.
