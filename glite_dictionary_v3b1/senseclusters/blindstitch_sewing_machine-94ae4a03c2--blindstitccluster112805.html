blindstitch sewing machine

noun
sewing machine for nearly invisible stitches
8.0 6.3
Pronunciation: /ˈblaɪnd.stɪtʃ ˈsoʊ.ɪŋ mə.ˈʃiːn/ (AmE), /ˈblaɪnd.stɪtʃ ˈsəʊ.ɪŋ mə.ˈʃiːn/ (BrE)
A sewing machine designed to join two pieces of fabric so that the stitch thread is invisible or nearly invisible from the outside.

Examples

A blindstitch sewing machine for fabric
In tailoring shops, a blindstitch sewing machine is essential for creating professional-looking seams.
She used a blindstitch sewing machine to hem the dress without any visible thread on the outside.
The instructions recommend a blindstitch sewing machine for this type of invisible hem.
