suggester

noun
software component that gives suggestions
Pronunciation: / səˈdʒɛstər / (AmE), / səˈdʒɛstə / (BrE)
(in computing) a software component or system that generates and presents suggestions to users, for example query completions, search suggestions, or product recommendations.

Examples

Search suggester
The search engine's suggester proposes related queries as you type.
Our e-commerce suggester learned to show accessories after customers bought phones.
Developers integrated a fast suggester module into the app to improve user search experience.

Related Terms

  • synonym: autocomplete - a feature that predicts and displays completions for partially typed input
  • synonym: recommender - a system that suggests items (products, media, etc.) to users based on data and algorithms
