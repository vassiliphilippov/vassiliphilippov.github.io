end run

noun
American football play running to the outside
Pronunciation: / ˈɛnd ˌrʌn / (AmE), / ˈɛnd ˌrʌn / (BrE)
(in American football) an attempt to advance the ball by running around the end of the line of scrimmage.

Examples

A football end run
He attempted an end run but was tackled just short of the goal line.
The running back broke free for a 20-yard end run in the second quarter.
Their coach called an end run to take advantage of the defense's weakness on the outside.
