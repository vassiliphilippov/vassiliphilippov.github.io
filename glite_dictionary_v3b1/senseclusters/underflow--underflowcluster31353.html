underflow

noun
data removal error
8.4
Pronunciation: /ˈʌn.dər.floʊ/ (AmE), /ˈʌn.də.fləʊ/ (BrE)
an error that occurs when a program tries to remove data from an empty data structure, causing failure or unexpected behavior

Examples

Underflow in empty stack
The program crashed due to an underflow when it tried to pop from an empty stack.
Careful error checking can help prevent underflow in your code.
If you attempt to remove an item from an empty queue, an underflow may occur.

Related Terms

  • synonym: error - a mistake or problem in a computer program
  • antonym: overflow - an error caused by exceeding the maximum capacity of a data structure
