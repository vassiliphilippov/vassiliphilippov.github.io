server

noun
computer managing network resources
3.0 28/m
Pronunciation: / Ààs…ùÀêv…ôr / (AmE), / Ààs…úÀêv…ô / (BrE)
A computer that manages access to resources, such as files or printers, for other computers on a network.

Examples

File server
If the server crashes, employees lose access to shared files until it's fixed.
Our company upgraded its server to improve network performance and handle more users.
You need the correct password to connect to the database server.

Synonyms

  • host
    5.2 7.88/m
    host can be any networked computer that offers resources, while server usually means a machine dedicated to providing services to clients.

Related Terms

  • antonym: client - a computer or program that accesses services provided by a server
