shredder

noun
tool for permanently erasing digital data
4.2
Pronunciation: /ˈʃred.ər/ (AmE), /ˈʃred.ə/ (BrE)
(in computing) a software program or hardware tool that permanently erases digital files or data by overwriting, wiping, or otherwise destroying information so it cannot be recovered.

Examples

File shredder
The IT department deployed a shredder tool to permanently wipe sensitive files from old laptops.
Before donating the server, he used a hardware shredder to overwrite every sector beyond recovery.
Many security suites include a file-shredder that replaces deleted data with random bytes.

Related Terms

  • synonym: data eraser - a tool or program that permanently deletes or overwrites digital files
  • antonym: data recovery - the process of retrieving lost, deleted, or inaccessible data (opposite goal)
  • synonym: wiper - software that wipes storage by overwriting data to prevent recovery
