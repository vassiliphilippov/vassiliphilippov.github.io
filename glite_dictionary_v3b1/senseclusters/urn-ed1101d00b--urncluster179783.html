URN

noun
persistent digital resource identifier (URN)
8.2
Pronunciation: /juː ɑːr en/ /; / /ɜːrn/ (AmE), /juː ɑː en/ /; / /ɜːn/ (BrE)
A Uniform Resource Name: a persistent, location-independent identifier for a digital resource (a type of URI).

Examples

A URN identifier
The database assigns a URN to each digital object to enable stable citation.
Researchers prefer citing the dataset's URN because it remains valid even if the URL changes.
The metadata included a service URN that uniquely identifies the collection in archival systems.

Related Terms

  • synonym: URI - Uniform Resource Identifier; a string that identifies a resource on the Internet (URN is a type of URI).
  • synonym: URL - Uniform Resource Locator; a web address pointing to a resource's location (related concept, not identical).
