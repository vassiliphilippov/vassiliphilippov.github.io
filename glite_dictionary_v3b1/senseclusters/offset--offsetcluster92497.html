offset

verb
to set apart or shift position
Pronunciation: / ˈɔfˌsɛt / (AmE), / ˈɒfˌsɛt / (BrE)
to place or arrange something so that it is set apart from, or not exactly aligned with, another element; to cause to be set off or shifted from a regular position

Examples

To offset a line
The builder will offset the new wall from the original foundation.
They decided to offset the windows for a more modern appearance.
To improve parking, the spaces were offset from the road by a few feet.
