undefined

adjective
programming: unassigned or unknown value
5.7
Pronunciation: /ʌn.dɪ.ˈfaɪnd/ /; / /ʌn.də.ˈfaɪnd/ (AmE), /ʌn.dɪ.ˈfaɪnd/ /; / /ʌn.də.ˈfaɪnd/ (BrE)
(in computing) not assigned a value or not known by a program; used to indicate an uninitialized, missing, or error state for a variable, field, or expression.

Examples

Undefined variable
The API returned an undefined value for the user's email until the profile finished loading.
During testing, several variables remained undefined, causing the script to throw errors unexpectedly.
The dashboard showed 'N/A' when the requested metric was undefined for that date range.

Related Terms

  • synonym: uninitialized - not given an initial value in a program; often causes the same behavior as 'undefined' in code.
