read-only memory

noun
computer memory that cannot be changed
6.3 3.6
Pronunciation: /riːd ˈoʊn.li ˈmem.ə.riː/ (AmE), /riːd ˈəʊn.li ˈmem.ə.rɪ/ (BrE)
(in computing) a type of memory in a computer or other device whose contents can be accessed and read but not changed.

Examples

Read-only memory chip
Data stored in read-only memory remains intact even when the device is powered off.
Firmware is often stored in read-only memory to prevent accidental modification.
Many early video game consoles used cartridges containing read-only memory chips.

Related Terms

  • synonym: ROM - abbreviation for read-only memory
  • antonym: random-access memory - computer memory that can be read and changed
