processor

noun
software component that processes data
3.5
Pronunciation: /ˈprɑː.ses.ər/ /; / /ˈprɑː.sə.sər/ (AmE), /ˈprɒ.ses.ə/ (BrE)
(in computing) a software component or module that performs operations on data within an application, pipeline, or system, typically transforming, filtering, or routing records.

Examples

Data processor module
The ETL pipeline uses a processor to clean and transform the incoming records.
Each message passes through several processors before reaching the analytics service for aggregation.
You can write a custom processor to filter events and enrich them with metadata.
