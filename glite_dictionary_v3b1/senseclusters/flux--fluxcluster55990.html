flux

noun
software architecture for unidirectional data flow
8.6
Pronunciation: /flʌks/ (AmE), /flʌks/ (BrE)
(in software engineering) a design pattern for front-end applications that enforces unidirectional data flow to manage state and updates predictably.

Examples

Flux architecture
Developers adopted the flux architecture to manage unidirectional data flow in their web apps.
After reading the docs, she implemented flux to keep component state predictable and easy to test.
Many tutorials compare flux with Redux to explain unidirectional state management patterns.

Related Terms

  • synonym: Redux - a popular JavaScript library inspired by Flux that implements unidirectional state management
  • synonym: state management - the practice and tools used to store, update, and coordinate application state in front-end software
