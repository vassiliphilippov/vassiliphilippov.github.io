joystick

noun
on-screen virtual control
Pronunciation: / \u02c8d\u0292\u0254\u026a\u02ccst\u026ak / (AmE), / \u02c8d\u0292\u0254\u026a.st\u026ak / (BrE)
a software on-screen control (usually on a touchscreen) that emulates a physical joystick and is used to control movement or direction in games or apps.

Examples

On-screen joystick
The mobile game uses an on-screen joystick to move the character during play.
She adjusted the sensitivity of the virtual joystick so the avatar would run smoothly.
Many players prefer physical controllers, but the on-screen joystick works well on smaller phones.

Related Terms

  • synonym: controller - a device or interface used to control a game or machine; similar function in software form
  • synonym: on-screen control - a graphical control element displayed on a screen for user interaction
