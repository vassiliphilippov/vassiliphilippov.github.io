Boolean logic

noun
system of symbolic logic used in computing
6.4 4.6
Pronunciation: / ˈbuːliən ˈlɑːdʒɪk / (AmE), / ˈbuːliən ˈlɒdʒɪk / (BrE)
A system of symbolic logic developed by George Boole, used especially in mathematics, computer science, and electronic circuits to manipulate true and false values.

Examples

Boolean logic system
Digital circuits are designed using the principles of Boolean logic.
Many programming languages rely on Boolean logic to control the flow of a program.
Understanding Boolean logic is essential for anyone studying computer science.

Related Terms

  • synonym: binary logic - logic based on binary variables (true/false, 1/0)
  • synonym: propositional logic - branch of logic dealing with propositions and their relationships
