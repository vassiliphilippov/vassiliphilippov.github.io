class list

noun
official list grouping students by degree honours
Pronunciation: / klæs lɪst / (AmE), / klɑːs lɪst / (BrE)
an official list issued by examiners, showing students grouped according to the class of honours they received in their degree examinations

Examples

University class list
Each summer, newspapers would publish the university class list, listing the top graduates.
She anxiously searched for her name on the class list to see if she had earned a first-class degree.
The class list was posted on the notice board after the results were confirmed.
