data rate

noun
speed of data transmission or processing
Pronunciation: / ˈdeɪtə ˌreɪt / (AmE), / ˈdeɪtə ˌreɪt / (BrE)
The speed at which data is transmitted or processed, usually measured in bits per second.

Examples

Maximum data rate
Different transmission technologies support different maximum data rates.
The data rate of this connection is limited to 100 megabits per second.
To stream high-definition video smoothly, you need a fast data rate.

Related Terms

  • synonym: bandwidth - the maximum rate of data transfer across a network
  • synonym: bit rate - the number of bits transmitted per unit of time
