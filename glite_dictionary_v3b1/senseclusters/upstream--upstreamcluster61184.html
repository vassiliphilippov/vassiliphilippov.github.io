upstream

noun
main source repository
6.8 0.79/m

(computing) the primary source repository or central project from which changes originate and to which contributors send patches or pull requests.

Examples

Submit to upstream
We submitted the patch to the upstream for review and potential merge.
Before releasing, they sync their fork with the upstream to avoid merge conflicts.
The company maintains the upstream repository that other projects clone and patch from regularly.

Related Terms

  • antonym: downstream - projects or repositories that receive changes from the upstream source (opposite role).
  • synonym: origin - the primary remote repository in workflows (for example, the default remote name in Git).
