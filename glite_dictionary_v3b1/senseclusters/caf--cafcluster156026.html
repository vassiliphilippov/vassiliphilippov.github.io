caf

noun
Apple audio file format
9.9
Pronunciation: / kæf / (AmE), / kæf / (BrE)
a digital audio file in Apple's Core Audio Format, usually with the .caf file extension

Examples

CAF audio file
The music producer exported the final track as a caf for use in the Apple software.
She converted the interview recording into a caf so it would open in the audio editor.
Some apps require audio files to be in the caf format for compatibility.
