rootkit

noun
hidden malicious software
9.8
Pronunciation: / ˈruːtkɪt / (AmE), / ˈruːtkɪt / (BrE)
a type of malicious software that hides itself and allows someone to access or control a computer system without permission

Examples

A rootkit attack
The security team discovered a rootkit on the company's server during a routine check.
Installing a rootkit can give hackers control over a computer without the user's knowledge.
Advanced rootkits are difficult to detect and remove from infected systems.

Related Terms

  • antonym: antivirus - software designed to detect and remove malicious software
  • synonym: malware - software designed to harm or gain unauthorized access to a computer
  • synonym: virus - a type of malicious software that can copy itself and infect computers
