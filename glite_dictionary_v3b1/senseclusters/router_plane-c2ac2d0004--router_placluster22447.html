router plane

noun
woodworking tool for smoothing grooves
5.0
Pronunciation: / ˈraʊtər pleɪn / (AmE), / ˈraʊtə pleɪn / (BrE)
a woodworking tool with a narrow cutting head, used to make grooves with smooth, level bottoms

Examples

A router plane tool
He used a router plane to level the bottom of the groove in the tabletop.
The router plane makes it easy to create smooth dados and mortises.
Vintage woodworkers often relied on a router plane for precision joinery.

Related Terms

  • synonym: chisel - a woodworking tool for carving or cutting hard material
  • synonym: plough plane - a woodworking tool for cutting grooves and channels, similar in function to a router plane
