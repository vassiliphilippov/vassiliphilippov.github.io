file-like

adjective
behaving like a file (computing)
5.8
Pronunciation: / ˈfaɪlˌlaɪk / (AmE), / ˈfaɪl.laɪk / (BrE)
(computing) behaving like a file by providing the standard file operations (for example, read, write, seek) so the object can be used wherever an open file is expected.

Examples

File-like object
In Python, a file-like object can be passed to libraries expecting an open file for reading.
To test streaming code, she created a file-like buffer that simulated network input and errors.
Many libraries accept any file-like object providing read and seek methods instead of a real file.

Related Terms

  • synonym: file object - a programming object representing an open file that supports read/write/seek operations
  • synonym: stream - a sequence of data elements made available over time; in some APIs streams are accepted like file-like objects
