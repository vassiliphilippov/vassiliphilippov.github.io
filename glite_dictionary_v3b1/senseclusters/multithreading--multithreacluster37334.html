multithreading

noun
simultaneous computing tasks
4.6
Pronunciation: /mʌl.tɪ.ˈθred.ɪŋ/ (AmE), /mʌl.tɪ.ˈθred.ɪŋ/ (BrE)
a method in computing where multiple tasks or threads are executed at the same time within a single program

Examples

Multithreading in software
Modern operating systems use multithreading to improve the performance of applications.
Learning about multithreading is important for developing efficient computer programs.
The software's multithreading capability allows it to handle many user requests at once.

Related Terms

  • synonym: concurrency - the ability of a system to run several tasks at the same time
  • synonym: parallelism - the simultaneous execution of multiple processes
  • antonym: single-threading - executing one task at a time in a program
