redirection

noun
changing data destination
4.1
Pronunciation: / ˌriːdəˈrɛkʃən / (AmE), / ˌriːdɪˈrɛkʃən / (BrE)
the process of sending data, input/output streams, or requests to a different destination, especially in computing.

Examples

Website redirection
The website uses redirection to send visitors to its new homepage automatically.
Command-line redirection allows users to save output from a program into a file.
After the server update, redirection ensured all requests reached the correct resource.

Related Terms

  • synonym: forwarding - the act of sending data or requests to another address or location
  • synonym: routing - the process of selecting a path for data in a network
