memory of a goldfish

noun
very poor or short memory
Pronunciation: / ˈmɛm.ə.ɹi əv ə ˈɡoʊld.fɪʃ / (AmE), / ˈmɛm.əri əv ə ˈɡəʊld.fɪʃ / (BrE)
a very poor memory; the tendency to forget things quickly, like a goldfish is (mistakenly) believed to do

Examples

Memory of a goldfish joke
I have the memory of a goldfish when it comes to remembering names.
If you have a memory of a goldfish, you might want to write things down.
She joked about having a memory of a goldfish after forgetting her keys again.

Related Terms

  • synonym: absent-minded - frequently forgetting things, inattentive
  • synonym: forgetful - tending to forget things easily
