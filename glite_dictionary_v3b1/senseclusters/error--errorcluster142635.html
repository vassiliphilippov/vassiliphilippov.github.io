error

noun
incorrect result by a computer
1.0 2.5
Pronunciation: /ˈe.rər/ (AmE), /ˈe.rə/ (BrE)
(in computing) a condition in which a computer produces an incorrect result, or a message indicating that something has gone wrong.

Examples

Error message appears
If you see this error message, please restart your computer.
The program crashed due to a critical error in the code.
The test results showed a calculation error by the database software.

Related Terms

  • synonym: bug - a flaw or fault in a computer program that causes it to operate incorrectly
  • antonym: correction - the act of removing or fixing an error
  • synonym: fault - a defect or imperfection in a computer system
