distinct

adjective
SQL keyword: return unique rows
Pronunciation: / dɪˈstɪŋkt / (AmE), / dɪˈstɪŋkt / (BrE)
(computing, SQL) a keyword used in SELECT statements to return only unique rows by removing duplicate results.
