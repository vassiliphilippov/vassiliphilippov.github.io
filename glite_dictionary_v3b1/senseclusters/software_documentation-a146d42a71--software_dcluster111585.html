software documentation

noun
manuals or guides for using software
1.0 3.0
Pronunciation: /ˈsɔːft.wer dɑː.kjə.ˈmen.teɪ.ʃən/ /; / /ˈsɑːft.wer dɑː.kjuː.ˈmen.teɪ.ʃən/ (AmE), /ˈsɒft.weə dɒk.jʊ.ˈmen.teɪ.ʃən/ /; / /ˈsɒft.weə dɒk.juː.ˈmen.teɪ.ʃən/ (BrE)
