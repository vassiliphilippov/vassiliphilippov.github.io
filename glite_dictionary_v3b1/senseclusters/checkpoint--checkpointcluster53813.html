checkpoint

verb
to save progress
Pronunciation: / ˈtʃɛkpɔɪnt / (AmE), / ˈtʃɛkpɔɪnt / (BrE)
to save the current state or progress of a process or program so it can be resumed later from that point

Examples

To checkpoint progress
Before installing the update, the system automatically checkpointed your files to prevent data loss.
The game checkpointed after each major level, allowing players to continue from where they left off.
Developers often checkpoint their work to avoid losing important changes during testing.

Related Terms

  • antonym: restore - to return to a previously saved state
  • synonym: save - to store data or progress for future use
