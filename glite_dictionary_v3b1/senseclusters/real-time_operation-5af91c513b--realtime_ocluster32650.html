real-time operation

noun
processing data as events occur
Pronunciation: / ˈriːəl taɪm ˌɑːpəˈreɪʃən / (AmE), / ˈrɪəl taɪm ˌɒpəˈreɪʃən / (BrE)
the processing of data quickly enough to keep up with external events or inputs as they occur

Examples

Real-time operation system
Many industrial control systems require real-time operation to function safely and efficiently.
Successful real-time operation allows the computer to respond to sensor inputs instantly.
The new software ensures real-time operation so that critical data is never delayed.
