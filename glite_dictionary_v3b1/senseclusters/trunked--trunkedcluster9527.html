trunked

adjective
using shared channel
7.5
Pronunciation: /trʌŋkt/ (AmE), /trʌŋkt/ (BrE)
using or connected by a trunk line, especially sharing a high-capacity communication channel for multiple calls or data streams

Examples

Trunked radio system
The police department upgraded to a trunked radio system for more efficient communication.
Many modern networks are trunked to allow multiple users to share the same infrastructure.
A trunked telephone system can handle many calls at once by dynamically allocating channels.
