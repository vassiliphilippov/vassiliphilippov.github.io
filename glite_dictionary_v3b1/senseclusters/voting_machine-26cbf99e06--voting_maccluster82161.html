voting machine

noun
machine for recording and counting votes
1.0
Pronunciation: / ˈvoʊ.tɪŋ məˈʃin / (AmE), / ˈvəʊ.tɪŋ məˈʃiːn / (BrE)
a machine used to record and count votes in an election automatically

Examples

A voting machine malfunction
An investigation revealed that a malfunctioning voting machine caused delays at the polling station.
Poll workers demonstrated how to use the new electronic voting machine to each voter.
The city upgraded every voting machine before the last election to improve accuracy.
