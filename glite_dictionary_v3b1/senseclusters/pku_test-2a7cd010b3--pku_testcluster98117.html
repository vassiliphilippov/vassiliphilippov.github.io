PKU test

noun
newborn screening for phenylketonuria
Pronunciation: / piː keɪ juː tɛst / (AmE), / piː keɪ juː tɛst / (BrE)
a blood test performed on newborns to screen for phenylketonuria, a genetic disorder that can cause intellectual disability if untreated

Examples

PKU test for babies
All newborn babies in the hospital receive a PKU test before they go home.
Early detection through the PKU test allows for prompt dietary intervention.
The doctor explained the purpose of the PKU test during the baby’s first check-up.
