Snellen test

noun
test for measuring visual acuity
5.7
Pronunciation: / ˈsnɛlən tɛst / (AmE), / ˈsnɛlən tɛst / (BrE)
a test of visual acuity that uses a chart of letters of decreasing size to determine the smallest print a person can read from a set distance

Examples

The Snellen test chart
Every year, students undergo a basic vision screening that includes the Snellen test.
Her poor performance on the Snellen test indicated a need for corrective lenses.
The optometrist performed a Snellen test to assess the patient's eyesight.
