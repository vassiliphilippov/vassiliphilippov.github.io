command guidance

noun
missile guidance by external commands
6.1
Pronunciation: / kəˈmænd ˈɡaɪdns / (AmE), / kəˈmɑːnd ˈɡaɪdns / (BrE)
a method of controlling the flight of a missile using commands sent from the ground or from another missile

Examples

Command guidance system
Engineers improved the command guidance system for greater accuracy in missile control.
The anti-aircraft system uses command guidance to intercept incoming targets.
Unlike homing missiles, those with command guidance depend entirely on instructions from operators.
