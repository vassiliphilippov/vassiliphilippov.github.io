Avogadro number

noun
the number of particles in a mole (≈6.022×10²³)
6.9
Pronunciation: / ˌævəˈɡɑdroʊ ˈnʌmbɚ / (AmE), / ˌævəˈɡædrəʊ ˈnʌmbə / (BrE)
the number of constituent particles, usually atoms or molecules, that are contained in one mole of a substance, equal to approximately 6.022 × 10²³.

Examples

The Avogadro number constant
A mole of carbon atoms contains the Avogadro number of atoms.
Knowing the Avogadro number allows scientists to convert between atomic scale and macroscopic quantities.
The Avogadro number is a fundamental constant in chemistry and physics.

Related Terms

  • synonym: Avogadro's constant - the same numerical value as Avogadro number, used to specify number of entities per mole
