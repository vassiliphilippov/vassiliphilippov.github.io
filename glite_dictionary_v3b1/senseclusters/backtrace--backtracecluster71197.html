backtrace

noun
function call list
Pronunciation: / ˈbækˌtreɪs / (AmE), / ˈbækˌtreɪs / (BrE)
a list showing the sequence of function calls leading to a specific point in a computer program, often used to find errors

Examples

A program backtrace
The developer examined the backtrace to understand what caused the program to crash.
A detailed backtrace can help programmers quickly locate bugs in complex software.
After the error occurred, the system generated a backtrace for troubleshooting.

Related Terms

  • synonym: debug - to identify and remove errors from computer code
  • synonym: stack trace - a report of the active stack frames at a certain point in a program
