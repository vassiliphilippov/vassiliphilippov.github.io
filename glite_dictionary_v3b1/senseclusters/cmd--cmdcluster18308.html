cmd

noun
Mac keyboard key
8.6
Pronunciation: / siː ɛm diː / (AmE), / siː ɛm diː / (BrE)
the Command key on Apple Macintosh keyboards, used with other keys to perform shortcuts.

Examples

Press cmd key
To paste the copied text, hold the cmd key and press V.
Many Mac shortcuts require you to use the cmd key together with other keys.
She accidentally pressed the cmd key and closed her document.

Related Terms

  • synonym: control key - a keyboard key used in combination with others for shortcuts
  • synonym: option key - a modifier key on Mac keyboards used for shortcuts
