object-oriented database

noun
a database storing data as objects
5.3 6.7
Pronunciation: / ˌɑbdʒɛkt ˈɔriˌɛntɪd ˈdeɪtəˌbeɪs / (AmE), / ˌɒbdʒɛkt ˈɔːriˌɛntɪd ˈdeɪtəˌbeɪs / (BrE)
A database system in which data is stored as objects, with both data and its operations bundled together, following object-oriented programming principles.

Examples

An object-oriented database system
The development team chose an object-oriented database for better integration with the software's data structures.
The new application uses an object-oriented database to manage complex multimedia data.
Unlike traditional relational models, an object-oriented database can efficiently store objects such as images and videos.

Related Terms

  • antonym: relational database - A database structured to recognize relations among stored items of information
