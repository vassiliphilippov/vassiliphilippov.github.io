decommit

verb
to cancel a commit
8.8
Pronunciation: /diː.kə.ˈmɪt/ /; / /dɪ.kə.ˈmɪt/ /; / /də.kə.ˈmɪt/ (AmE), /diː.kə.ˈmɪt/ /; / /dɪ.kə.ˈmɪt/ /; / /də.kə.ˈmɪt/ (BrE)
to reverse or cancel a previous commit, especially in computing, such as undoing changes or releasing allocated resources

Examples

To decommit memory

The system will automatically decommit memory that is no longer needed by the application.
