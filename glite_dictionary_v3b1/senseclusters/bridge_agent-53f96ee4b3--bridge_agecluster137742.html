bridge agent

noun
software/hardware connector between systems
5.5
Pronunciation: /brɪdʒ ˈeɪ.dʒənt/ (AmE), /brɪdʒ ˈeɪ.dʒənt/ (BrE)
(in computing) a software program or hardware device that connects two different systems or networks and translates data or protocols so they can communicate.

Examples

A bridge agent for networks
The integration hub uses a bridge agent to translate data between the legacy ERP system and the cloud platform.
During deployment, the team installed multiple bridge agents to connect sensors with the central monitoring service.
If the bridge agent fails, messages may queue up and synchronization between databases will stop.

Related Terms

  • synonym: connector - a component or software that links two systems to allow data exchange
  • synonym: gateway - a device or program that translates between different network protocols
  • synonym: middleware - software that sits between applications to enable communication and data management
