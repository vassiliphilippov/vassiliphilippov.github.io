OBJ

noun
compiled object file
Pronunciation: / oʊ biː dʒeɪ / (AmE), / əʊ biː dʒeɪ / (BrE)
a compiled object file produced by a compiler, containing machine code and data to be linked into a program

Examples

OBJ file error
The developer checked the OBJ files for errors after compiling the source code.
You need to link all the OBJ files together to create the final program.
Each module in the project generates its own OBJ file during compilation.

Related Terms

  • synonym: object file - a file containing compiled code to be linked into a program
  • antonym: source file - a file containing human-readable program code before compilation
