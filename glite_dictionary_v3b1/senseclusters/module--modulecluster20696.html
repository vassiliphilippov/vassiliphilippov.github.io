module

noun
self-contained component of a system
3.7 22/m
Pronunciation: / Ààm…ëÀê.d íuÀêl / (AmE), / Ààm…íd.juÀêl / (BrE)
a self-contained unit or component that is part of a larger system or structure and can often be used independently or combined with others

Examples

Course module
Each module of the course covers a different topic.
Space stations are often built by connecting pre-fabricated modules.
The software consists of several modules that can be installed independently.
Each memory module slots into the motherboard of the server.
Technicians tested the faulty module before installing a new one.
The graphics module can be replaced to upgrade your computer's performance.
