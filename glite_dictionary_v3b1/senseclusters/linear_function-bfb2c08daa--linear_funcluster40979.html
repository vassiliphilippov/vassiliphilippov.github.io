linear function

noun
a function whose graph is a straight line
4.4 3.8
Pronunciation: /ˈlɪn.ɪ.ər ˈfʌŋk.ʃən/ /; / /ˈlɪn.ɪr ˈfʌŋk.ʃən/ (AmE), /ˈlɪn.ɪ.ə ˈfʌŋk.ʃən/ /; / /ˈlɪn.jə ˈfʌŋk.ʃən/ (BrE)
a mathematical function whose graph is a straight line when plotted on Cartesian coordinates, and which can be represented by an equation of the form y = mx + b

Examples

A linear function graph
If you plot y = 2x + 1, you get a linear function on the graph.
The relationship between distance and time at a constant speed is a linear function.
The teacher explained that a linear function has a constant rate of change.
