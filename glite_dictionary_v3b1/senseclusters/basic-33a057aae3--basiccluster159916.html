BASIC

noun
easy-to-learn computer programming language
4.9
Pronunciation: /ˈbeɪ.sɪk/ (AmE), /ˈbeɪ.sɪk/ (BrE)
a simple, easy-to-learn computer programming language, especially popular in the 1970s and 1980s, whose name stands for 'Beginner's All-purpose Symbolic Instruction Code'

Examples

BASIC programming language
Although technology has advanced, some still use BASIC to teach introductory programming concepts.
He learned to code his first game in BASIC on a vintage home computer.
Many early personal computers came with BASIC built in as their main programming language.
