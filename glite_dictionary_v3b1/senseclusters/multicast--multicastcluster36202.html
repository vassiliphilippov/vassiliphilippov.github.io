multicast

adjective
network group transmission
6.4
Pronunciation: /ˈmʌl.tɪ.kæst/ /; / /ˈmʌl.ti.kæst/ (AmE), /ˈmʌl.tɪ.kɑːst/ /; / /ˈmʌl.ti.kɑːst/ (BrE)
relating to a network transmission method that sends data to multiple recipients simultaneously.

Examples

Multicast network traffic
The multicast protocol allows efficient video streaming to many users at once.
A multicast address is used to deliver the same data to several devices on the network.
Engineers set up a multicast system for distributing real-time updates to all branches.
