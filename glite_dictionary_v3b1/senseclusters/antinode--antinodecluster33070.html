antinode

noun
point of maximum displacement in a wave
8.6
Pronunciation: / ˈæn.tiˌnoʊd / (AmE), / ˈæn.ti.nəʊd / (BrE)
(Physics) A point in a standing wave where the displacement is at its maximum.

Examples

The antinode point
At the antinode, the string vibrates with the greatest amplitude.
Nodes and antinodes are key features when analyzing patterns in standing waves.
The distance between each antinode in a standing wave is half a wavelength.

Related Terms

  • antonym: node - a point in a standing wave with zero displacement
