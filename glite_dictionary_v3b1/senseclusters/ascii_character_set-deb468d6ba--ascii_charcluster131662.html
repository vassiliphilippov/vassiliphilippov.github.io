ASCII character set

noun
characters defined by the ASCII standard
9.9
Pronunciation: / ˈæski ˈkærɪktɚ sɛt / (AmE), / ˈæski ˈkærɪktə sɛt / (BrE)
the set of 128 characters defined by the ASCII standard, used in computers to represent letters, digits, symbols, and control codes

Examples

The ASCII character set codes
Most early computers could only display the ASCII character set on their screens.
The ASCII character set does not include characters with accents or other diacritics.
When saving a file as plain text, only the ASCII character set is guaranteed to be preserved.

Related Terms

  • synonym: ASCII - a standard code for representing characters as numbers on computers
