polymorphism

noun
malware changing form to avoid detection

(computing, security) the property or technique by which malware or code automatically changes its appearance or structure while keeping the same behaviour, in order to evade detection by antivirus or other security tools.

Examples

Malware polymorphism
The malware used polymorphism to mutate its code and evade antivirus detection during scans.
Security researchers studied the polymorphism of the worm to develop signatures that catch its variants.
Modern packing techniques combined with polymorphism make reversing malware more time-consuming for analysts.

Related Terms

  • synonym: code obfuscation - techniques that make code harder to analyse or understand, often used to resist reverse engineering
  • synonym: mutation (computing) - the automated alteration of code or data to create varied forms with equivalent behaviour
