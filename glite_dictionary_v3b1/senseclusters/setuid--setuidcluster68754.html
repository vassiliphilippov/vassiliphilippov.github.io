setuid

verb
to set user permissions
9.9
Pronunciation: /ˈset.juː.ɪd/ /; / /ˈset.juːd/ /; / /ˈset.juː.aɪ.diː/ (AmE), /ˈset.juː.ɪd/ /; / /ˈset.juːd/ /; / /ˈset.juː.aɪ.diː/ (BrE)
to set the setuid permission on a file or program so it runs with the file owner's privileges

Examples

To setuid a file
You need to setuid the script so it can perform administrative tasks.
The administrator forgot to setuid the program, causing it to fail.
Always be careful when you setuid files to avoid security risks.
