typeable

adjective
can be typed (programming)
7.0
Pronunciation: / ˈtaɪpəbəl / (AmE), / ˈtaɪpəbəl / (BrE)
able to be assigned a data type by a programming language's type system.

Examples

Typeable variable
In this programming language, every variable must be typeable for the code to compile.
The function is not typeable because its return value cannot be determined.
Advanced type systems can make more expressions typeable than simpler ones.

Related Terms

  • synonym: typed - having a specific data type assigned
  • antonym: typeless - not having or requiring a data type
