cardinal number

noun
number indicating quantity, not order
Pronunciation: / ˈkɑːrdɪnəl ˈnʌmbər /; / ˈkɑːrdɪnl ˈnʌmbɚ / (AmE), / ˈkɑːdɪnəl ˈnʌmbə /; / ˈkɑːdɪn(ə)l ˈnʌmbə / (BrE)
a number that shows how many things there are (for example, one, two, three), not the order

Examples

Cardinal numbers one two
Children often learn cardinal numbers before ordinal numbers in school.
In mathematics, a cardinal number represents the size of a collection.
The cardinal number of the set {2, 4, 6} is three.
A cardinal number is used to show how many items are present.
The teacher asked the students to list every cardinal number up to ten.
Words like 'four' and 'seventeen' are examples of cardinal numbers.

Related Terms

  • antonym: ordinal number - number indicating position or order (e.g., first, second, third)
