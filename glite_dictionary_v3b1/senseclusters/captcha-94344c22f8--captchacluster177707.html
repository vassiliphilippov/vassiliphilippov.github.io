CAPTCHA

noun
human verification test
Pronunciation: / ˈkæptʃə / (AmE), / ˈkæptʃə / (BrE)
a test on a website designed to determine whether the user is a human or an automated program

Examples

Solve a CAPTCHA
Before creating an account, you must solve a CAPTCHA to prove you are not a robot.
The website uses a CAPTCHA to prevent automated spam submissions.
If you cannot read the CAPTCHA, you can request a new one or listen to an audio version.

Related Terms

  • antonym: bot - an automated program that performs tasks on the internet
  • synonym: reCAPTCHA - a specific type of CAPTCHA service developed by Google
