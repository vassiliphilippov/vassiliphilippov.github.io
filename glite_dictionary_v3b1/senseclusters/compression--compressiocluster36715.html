compression

noun
reducing data size by encoding
Pronunciation: / kəmˈprɛʃən / (AmE), / kəmˈprɛʃən / (BrE)
the process of encoding information using fewer bits or less bandwidth, especially to reduce data size for storage or transmission

Examples

Image compression
Modern web browsers support image compression to help websites load faster.
The video file was too large to send by email, so she used compression to make it smaller.
Without proper compression, audio files can quickly fill up your device’s memory.

Related Terms

  • antonym: decompression - restoring data to its original form after compression
  • synonym: encoding - converting data into a particular form
