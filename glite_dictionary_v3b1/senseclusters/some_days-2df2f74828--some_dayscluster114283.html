some days

noun
several days in a period
1.0
Pronunciation: / sʌm deɪz / (AmE), / sʌm deɪz / (BrE)
several individual days, but not all days, within a particular period of time

Examples

Some days I run
Some days I wake up early and go for a run before work.
The weather is unpredictable; some days are sunny while others are rainy.
He only works from home some days each week, not every day.

Related Terms

  • antonym: every day - on all days without exception
  • synonym: occasionally - happening sometimes but not regularly
