tree creeper

noun
a bird that climbs tree trunks like a woodpecker
8.0
Pronunciation: / tri ˈkripər /; / tri ˈkripɚ / (AmE), / triː ˈkriːpə / (BrE)
a small bird that climbs up tree trunks using its stiff tail feathers, searching for insects
