scalar matrix

noun
square matrix with equal diagonal entries
7.1 6.6
Pronunciation: /ˈskeɪ.lər ˈmeɪ.trɪks/ /; / /ˈskeɪ.lər ˈmæ.trɪks/ (AmE), /ˈskeɪ.lə ˈmeɪ.trɪks/ /; / /ˈskeɪ.lə ˈmæ.trɪks/ (BrE)
a square matrix in which all the entries on the main diagonal are equal and all other entries are zero

Examples

A scalar matrix
A scalar matrix is always proportional to the identity matrix.
Multiplying any vector by a scalar matrix preserves its direction.
The simplest form of a diagonal matrix is a scalar matrix.
