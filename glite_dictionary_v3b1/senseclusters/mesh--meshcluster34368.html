mesh

noun
network topology: full direct connections
Pronunciation: / mɛʃ / (AmE), / mɛʃ / (BrE)
a network topology in which every node is directly connected to every other node

Examples

Mesh network
In a mesh topology, data can be transmitted directly from any computer to another without passing through a hub.
Network engineers often choose a mesh for critical systems that require high resilience to connection failures.
The company's wireless network uses a mesh to ensure reliable communication between all devices.

Related Terms

  • antonym: ring topology - a network topology where nodes are connected in a closed loop
  • antonym: star topology - a network topology with each node connected to a central node
