catchable

adjective
able to be handled
Pronunciation: / ˈkætʃəbəl / (AmE), / ˈkætʃəbəl / (BrE)
able to be intercepted or handled, especially by an exception handler in a computer program.

Examples

Catchable error
The error must be catchable for the program to recover gracefully.
If the exception is not catchable, the application may terminate unexpectedly.
Developers should ensure all critical errors are catchable by the system.

Related Terms

  • synonym: handleable - able to be managed or dealt with
  • antonym: uncatchable - not able to be intercepted or handled
