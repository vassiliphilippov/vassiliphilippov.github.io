file sharing

noun
exchanging digital files
Pronunciation: / ˈfaɪl ˌʃɛrɪŋ / (AmE), / ˈfaɪl ˌʃeərɪŋ / (BrE)
the act of exchanging or distributing digital files over a network or the internet, sometimes without permission

Examples

File sharing online
Many people use file sharing to exchange documents and photos online.
The company implemented a secure file sharing system for its employees.
Illegal file sharing of copyrighted music can result in legal consequences.
Many people use peer-to-peer networks for file sharing of music and movies.
The company implemented strict policies to prevent illegal file sharing among employees.
Legal streaming services have reduced the need for unauthorized file sharing online.
