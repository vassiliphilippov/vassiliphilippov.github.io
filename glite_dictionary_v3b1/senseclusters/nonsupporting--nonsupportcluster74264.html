nonsupporting

adjective
lacking compatibility
6.0
Pronunciation: / ˌnɑn.səˈpɔrtɪŋ / (AmE), / ˌnɒn.səˈpɔːtɪŋ / (BrE)
not able to provide compatibility or the necessary functionality for a particular feature, standard, or operation

Examples

Nonsupporting software version
Many older, nonsupporting browser versions cause layout errors for users.
The device is nonsupporting of the latest wireless charging standard.
If your software is nonsupporting, you may need to update it to use this feature.

Related Terms

  • synonym: incompatible - not able to work or be used together
  • antonym: supporting - able to provide compatibility or necessary functionality
