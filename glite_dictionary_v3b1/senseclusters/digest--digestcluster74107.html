digest

noun
cryptographic hash value
8.2 0.29/m
Pronunciation: / daÉªËˆdÊ’est / (AmE), / daÉªËˆdÊ’est / (BrE)
(computing) a fixed-length value produced by a cryptographic hash function that represents data and is used to verify integrity or identify the data.

Examples

File digest
Before uploading, she computed the file's digest to ensure integrity during transfer.
The security team compared the downloaded package's digest with the official checksum for verification.
Modern protocols store a short digest of each message to detect accidental or malicious corruption.

Synonyms

  • hash

Related Terms

  • synonym: hash - a value produced by a hash function that uniquely represents data
