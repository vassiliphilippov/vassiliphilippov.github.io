run down

verb
deplete; use up resources
3.2 0.59/m
Pronunciation: / rÊŒn daÊŠn / (AmE), / rÊŒn daÊŠn / (BrE)
to use up or lose energy, power, or resources until little or nothing remains

Examples

Run down the battery
If you keep leaving the lights on, you'll run down the batteries much faster.
The charity began to run down its funds during the crisis.
We tried not to run down our food supplies before the next delivery arrived.
After working long hours all week, I finally ran down on Friday evening.
If you leave the lights on, they'll run down the car's battery.
The battery in my phone will run down if I don't charge it tonight.
