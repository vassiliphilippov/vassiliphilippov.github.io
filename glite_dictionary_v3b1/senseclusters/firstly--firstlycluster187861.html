firstly

adverb
introducing the first point
1.0 3.50/m
Pronunciation: / Ààf…ùst.li / (AmE), / Ààf…úÀêst.li / (BrE)
Used to introduce the first point or reason in a list of items or arguments.

Examples

Firstly, let me
Firstly, let me thank everyone for attending this meeting.
Firstly, we need to address the most urgent problem before discussing the others.
Firstly, you should read the instructions carefully before beginning the test.

Synonyms

  • first of all
    1.0 31/m
    first of all is a common phrase used to start a point in speech, while firstly sounds more formal in writing.
  • first
    1.0 396/m
    first is a short, flexible word speakers use to mark order, while firstly is a more formal list marker in arguments.
  • first off
    2.8 2.99/m
    first off is an informal opener people use in speech to start actions, while firstly is more formal and fits written arguments.
  • first up
    3.7 1.49/m
    first up is a casual phrase presenters use for the next person or item, while firstly introduces the first argument or reason.

Related Terms

  • synonym: first - before anything else (as an adverb); in the first place
  • degree higher: secondly - used to introduce the second point in a list
