time-sharing

noun
shared computer use
3.6
Pronunciation: /ˈtaɪm.ʃeɪr.ɪŋ/ /; / /ˈtaɪm.ʃer.ɪŋ/ (AmE), /ˈtaɪm.ʃeə.rɪŋ/ (BrE)
a method in computing where multiple users use a computer system simultaneously by sharing its resources

Examples

Time-sharing computer system
In the 1960s, time-sharing allowed several people to use a single mainframe computer at once.
Modern cloud computing is based on principles similar to time-sharing from earlier decades.
The university introduced time-sharing so students could access the computer lab more efficiently.

Related Terms

  • synonym: multiprocessing - the use of two or more CPUs within a single computer system
