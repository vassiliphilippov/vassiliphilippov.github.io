inline

adjective
expanded at call site
6.9
Pronunciation: /ɪn.ˈlaɪn/ /; / /ˈɪn.laɪn/ (AmE), /ɪn.ˈlaɪn/ /; / /ˈɪn.laɪn/ (BrE)
in computing, describing a function or routine that is expanded at its call site by the compiler, avoiding a normal function call

Examples

Inline function
The inline function helped speed up the program by reducing call overhead.
Marking a small routine as inline can improve performance in critical code sections.
Some compilers automatically make simple functions inline to optimize execution.

Related Terms

  • antonym: function call - an instruction that invokes a function in programming
  • synonym: macro - a code fragment that is expanded by a compiler or preprocessor
