RLE

sym
Unicode RTL control
Pronunciation: / ɑr ɛl i / (AmE), / ɑː ɛl iː / (BrE)
a Unicode control character (U+202B) that starts a right-to-left embedding for displaying text in right-to-left order

Examples

RLE character in text
The software automatically inserts RLE when you type Arabic text in an English document.
To fix the display issue, the developer added an RLE before the Hebrew phrase.
Without the RLE character, the mixed-direction text appeared incorrectly on the screen.

Related Terms

  • antonym: LRE - a Unicode control character (U+202A) that starts a left-to-right embedding
  • synonym: RLM - a Unicode control character (U+200F) that marks right-to-left text direction
