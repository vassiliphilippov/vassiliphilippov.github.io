seeded

adjective
pre-populated with initial data (computing)
5.7
Pronunciation: /ˈsiː.dɪd/ /; / /ˈsiː.dəd/ (AmE), /ˈsiː.dɪd/ /; / /ˈsiː.dəd/ (BrE)
(in computing) supplied or pre-populated with initial 'seed' data so the system or dataset is ready for testing, demonstration, or development.

Examples

Seeded database
The development database is seeded with user accounts and sample records for testing.
Before demos, they always ensure the demo site has been seeded with realistic example data.
The migration script seeded the new environment with default settings, admin users, and test orders.
