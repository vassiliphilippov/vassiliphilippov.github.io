memorizer

noun
software that caches function results

(in computing) a software component or function that caches the results of expensive function calls so repeated calls with the same inputs return the stored results quickly (see memoization).

Examples

Software memorizer
The library module includes a memorizer that stores previous query results to reduce database calls.
During testing, the team found the memorizer caused stale data when inputs were not properly invalidated.
High-performance servers often use memorizers to avoid repeating expensive computations for identical requests.

Related Terms

  • synonym: cache - a storage layer that keeps data for faster repeated access
  • synonym: memoization - an optimization technique that stores the results of function calls and returns the cached result when the same inputs occur again
