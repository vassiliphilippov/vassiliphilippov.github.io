import

verb
transfer data into a database or document
Pronunciation: / ɪmˈpɔrt / (AmE), / ɪmˈpɔːt / (BrE)
to bring data, files, or software modules into a computer program or system

Examples

To import data
After we imported the files, all the records were available in the database.
The spreadsheet allows users to import information from other programs.
You can easily import the customer data into the new software.
The script will import the numpy module to perform advanced numerical operations on arrays.
She accidentally forgot to import the logging library, so error messages were not recorded.
After they imported the module, its functions were available throughout the application.

Related Terms

  • antonym: export - to transfer electronic data out of a database or document
  • synonym: include - to add the contents of one file or module to another so its definitions are available
  • synonym: load - to place a program, module, or data into memory so it can be executed or used
