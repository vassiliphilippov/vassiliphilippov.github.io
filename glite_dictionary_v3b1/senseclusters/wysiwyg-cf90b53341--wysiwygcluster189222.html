WYSIWYG

adjective
displaying text and graphics as printed
Pronunciation: / ˈwɪziˌwɪɡ / (AmE), / ˈwɪziˌwɪɡ / (BrE)
(of a computer display or editor) showing content exactly as it will appear when printed or published, with formatting and layout preserved.

Examples

WYSIWYG editor
He prefers WYSIWYG word processors over plain text editors.
Modern web builders use WYSIWYG interfaces for easier design.
The WYSIWYG editor allows users to see the final layout as they type.
