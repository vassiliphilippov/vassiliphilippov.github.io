home page

noun
main or start page of a website or browser
Pronunciation: / hoʊm peɪdʒ / (AmE), / həʊm peɪdʒ / (BrE)
the main or introductory page of a website, or the page automatically displayed when a web browser is launched

Examples

The home page of a website
Every time I open my browser, the home page appears automatically.
I set my favorite news website as my home page for quick access to headlines.
The company's home page features links to all of its main products and services.

Related Terms

  • synonym: landing page - the web page where a visitor arrives after clicking a link or advertisement
