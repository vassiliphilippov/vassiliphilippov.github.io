manifest

noun
software/package metadata file
6.9
Pronunciation: /ˈmæn.ɪ.fest/ /; / /ˈmæn.ə.fest/ (AmE), /ˈmæn.ɪ.fest/ /; / /ˈmæn.ə.fest/ (BrE)
(in computing) a file (often JSON, XML, or similar) that describes a software package or application by listing its contents, versions, dependencies, permissions, or other metadata needed for installation or execution.

Examples

App manifest
Before publishing, the build system updated the web app's manifest to include new permissions and icons.
The installer read the package manifest to determine required library versions and installation order.
Developers commit a concise manifest with dependencies so continuous integration can reproduce the environment.

Related Terms

  • synonym: metadata - data that provides information about other data, such as file attributes or configuration details
  • synonym: package descriptor - a file that describes a software package's contents and dependencies (for example, package.json)
