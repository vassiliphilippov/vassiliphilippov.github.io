codebook

noun
table of codewords
6.1
Pronunciation: / ˈkoʊdbʊk / (AmE), / ˈkəʊdbʊk / (BrE)
a set or table of codewords used for encoding, decoding, or compressing data, especially in signal processing or machine learning

Examples

Digital codebook
The algorithm searches the codebook to find the closest match for each data vector.
Engineers updated the codebook to improve the accuracy of the compression process.
In machine learning, a codebook helps convert complex data into simpler representations.

Related Terms

  • synonym: dictionary - a set of mappings between symbols or data representations
  • synonym: lookup table - a table used to substitute values or data quickly
