parameterizable

adjective
configurable by parameters
7.9
Pronunciation: / pəˌræmɪtəˈraɪzəbəl / (AmE), / pəˌræmɪtəˈraɪzəbəl / (BrE)
able to be configured or defined by setting parameters.

Examples

A parameterizable system
The new algorithm is fully parameterizable, allowing users to adjust its behavior easily.
A parameterizable template can be reused for different projects with minimal changes.
Developers prefer parameterizable systems because they offer greater flexibility and control.

Related Terms

  • antonym: fixed - not able to be changed or adjusted
