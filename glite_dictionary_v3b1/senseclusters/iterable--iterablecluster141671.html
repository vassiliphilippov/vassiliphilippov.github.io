iterable

adjective
able to be looped over
6.3
Pronunciation: / ˈɪt̬ərəbəl / (AmE), / ˈɪtərəbl / (BrE)
able to be repeated or looped over, especially referring to objects in computing that can be processed one item at a time

Examples

Iterable object type
In Python, a list is considered an iterable because you can loop through its elements.
The function only works if the input is an iterable object, such as a string or a tuple.
You can use a for loop to process each item in any iterable data structure.

Related Terms

  • antonym: non-iterable - not able to be looped over or repeated
  • synonym: repeatable - able to be repeated
