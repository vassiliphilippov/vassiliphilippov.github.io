CDF

noun
scientific data file format
Pronunciation: / si di ɛf / (AmE), / siː diː ɛf / (BrE)
a binary file format used for storing and sharing scientific multi-dimensional data such as arrays and time series

Examples

CDF data file
The research team stored their experimental results in CDF files for easy sharing.
Many scientific software tools can read and write CDF formats for data analysis.
To ensure compatibility, the satellite data was converted to CDF before distribution.

Related Terms

  • synonym: NetCDF - a file format for storing array-oriented scientific data
