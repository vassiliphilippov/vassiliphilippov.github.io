(in computing, especially user interfaces) a feature that automatically suggests or completes words, queries, or entries as the user types, helping them find results more quickly.

Examples

Auto-suggestion in search box
The search box offers auto-suggestion to speed up queries as you type.
When typing the address bar, her browser displayed helpful auto-suggestions based on browsing history.
Developers implemented auto-suggestion so users can find products with fewer keystrokes.

Synonyms

  • autosuggestion
    5.4 0.00/m
    autosuggestion shows automatic suggestion lists as a user types in a search box, while auto-suggestion is the same user interface feature but hyphenated.
  • autosuggest
    4.2 0.04/m
    autosuggest offers word and query suggestions as you type and often appears as a verb, while auto-suggestion names the user interface feature.
  • autocomplete
    3.9 0.01/m
    autocomplete fills in or completes the current word or phrase automatically, while auto-suggestion shows a list of possible searches as you type.

Related Terms

  • synonym: autosuggest - a system that suggests likely completions for input fields in a user interface
  • synonym: typeahead - a UI pattern that displays likely matches while the user types
