cards

noun
rectangular UI content elements
3.5 0.01/m
Pronunciation: / k…ërdz / (AmE), / k…ëÀêdz / (BrE)
Rectangular elements in graphical user interfaces that group related content and actions (for example, an image, short text, and buttons); often clickable or swipeable.

Examples

Dashboard cards
The dashboard presents key metrics in cards to compare figures at a glance.
The design system includes reusable cards for news items, events, and user profiles.
Mobile apps often stack cards vertically so users can swipe through different content quickly.

Related Terms

  • synonym: card (UI) - a single rectangular interface element used to present a unit of content and associated actions
  • synonym: widget - a reusable component or element in a user interface
