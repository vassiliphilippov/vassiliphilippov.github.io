accessor

noun
method to get value
Pronunciation: / ˈæksɛsər /; / əkˈsɛsər / (AmE), / ˈæksɛsə /; / əkˈsɛsə / (BrE)
a method or function in programming that retrieves data or the value of a property from an object without changing it

Examples

A accessor method
The accessor in this class returns the current temperature value.
You should use an accessor to read the object's private data safely.
Every property in the object has an accessor for retrieving its value.
The accessor provides a safe way to read the value of a private variable.
In this class, the accessor returns the current status of the object.
You should use an accessor instead of accessing the field directly.

Related Terms

  • synonym: getter - a method that returns the value of a property
  • antonym: mutator - a method that changes the value of a property
