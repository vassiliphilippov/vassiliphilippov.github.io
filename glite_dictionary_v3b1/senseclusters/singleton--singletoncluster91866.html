singleton

noun
a set with one member
6.6
Pronunciation: / ˈsɪŋɡəlˌtən / (AmE), / ˈsɪŋɡ(ə)ltən / (BrE)
a set containing exactly one element, especially in mathematics and computer science

Examples

A singleton set
In set theory, a singleton is a set that contains only one element.
Mathematicians often use a singleton to represent a unique value in a set.
The function returns a singleton containing the result.
