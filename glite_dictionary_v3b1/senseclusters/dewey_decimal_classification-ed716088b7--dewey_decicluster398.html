Dewey decimal classification

noun
library classification system using numbers
6.2
Pronunciation: / ˈduːi ˈdɛsɪməl ˌklæsɪfɪˈkeɪʃən / (AmE), / ˈdjuːi ˈdɛsɪməl ˌklɑːsɪfɪˈkeɪʃən / (BrE)
