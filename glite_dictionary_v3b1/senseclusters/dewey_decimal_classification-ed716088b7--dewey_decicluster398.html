Dewey decimal classification

noun
library classification system using numbers
6.2
Pronunciation: / ˈduːi ˈdɛsɪməl ˌklæsɪfɪˈkeɪʃən / (AmE), / ˈdjuːi ˈdɛsɪməl ˌklɑːsɪfɪˈkeɪʃən / (BrE)
a system for organizing books and other materials in libraries by dividing nonfiction subjects into numbered categories, using a three-digit number and decimals for more specific topics

Examples
