trial run

noun
test of something before regular use
2.8
Pronunciation: / ˈtraɪ.əl ˌrʌn / (AmE), / ˈtraɪ.əl ˌrʌn / (BrE)
an occasion when something is tested to see if it works well before being used or done for real

Examples

A trial run test
Her bakery had a trial run last weekend, serving free pastries to friends and neighbors.
The new software is going through a trial run before its official release.
We organized a trial run of the event to identify any possible issues.
