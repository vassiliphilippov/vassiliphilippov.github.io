written language

noun
language using written symbols
3.7
Pronunciation: / ˈrɪtn ˈlæŋɡwɪdʒ / (AmE), / ˈrɪtn ˈlaŋɡwɪdʒ / (BrE)
language expressed through written symbols rather than speech, typically using letters or characters to communicate ideas.

Examples

Written language symbols
Children often learn to read and write in their native written language before studying a foreign one.
Chinese is an example of a written language that uses characters rather than an alphabet.
The rules of written language can be very different from those of spoken language.
