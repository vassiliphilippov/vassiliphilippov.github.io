DDT

noun
data-driven testing (software)
6.7
Pronunciation: /diː.diː.ˈtiː/ (AmE), /diː.diː.ˈtiː/ (BrE)
(in software testing) a method in which the same test logic is executed repeatedly with different external data inputs so many cases can be checked efficiently.

Examples

DDT testing method
The QA lead introduced DDT to reduce duplicated test code across multiple input cases.
Using DDT, testers ran the same test logic with hundreds of different input files quickly.
Many frameworks support DDT by reading test data from CSV or JSON configuration files.

Related Terms

  • synonym: data-driven testing - testing approach where test logic is supplied with external data sets to run multiple cases.
  • synonym: parameterized testing - a testing technique in many frameworks where tests are run with different parameters or data sets.
