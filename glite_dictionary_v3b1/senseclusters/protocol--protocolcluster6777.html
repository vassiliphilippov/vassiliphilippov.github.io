protocol

noun
rules for data transmission in computing
Pronunciation: / ˈproʊtəˌkɑl / (AmE), / ˈprəʊtəkɒl / (BrE)
a set of rules governing the format and transmission of data between electronic devices, especially in computing and telecommunications

Examples

Network protocol
Engineers are developing a more secure network protocol for encrypted communication.
The HTTP protocol is used for transferring web pages over the Internet.
To connect the devices, both must support the same protocol.

Related Terms

  • synonym: procedure - an established way of doing something, especially according to fixed rules
  • synonym: standard - an accepted model or requirement used for comparison in measurements, quality, or specifications
