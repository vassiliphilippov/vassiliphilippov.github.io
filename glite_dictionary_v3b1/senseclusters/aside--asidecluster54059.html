aside

noun
HTML5 element for tangential content

(in web design, HTML5) a section of a webpage marked with the <aside> element that contains content tangentially related to the main content, such as sidebars, pull quotes, or author notes.

Examples

HTML aside element
The author included a short biography in an aside element to avoid interrupting the main text.
Designers often place related links in a aside so readers can explore further without leaving the article.
The site uses several asides to show definitions, advertisements, and author notes beside each chapter.
