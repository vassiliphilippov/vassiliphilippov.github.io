symbol

noun
programming unique identifier
6.5
Pronunciation: /ˈsɪm.bəl/ (AmE), /ˈsɪm.bəl/ (BrE)
(computing) a primitive data value that represents a unique, immutable identifier, often used as a key or token in programs.

Examples

Symbol key in code
In JavaScript, a symbol provides a unique key for private object properties and avoids name collisions.
Library functions often accept symbols to look up configuration values without risking accidental overrides.
Because each symbol is unique, comparing two created now will usually return false even if descriptions match.

  • synonym: identifier - a name or token used to identify a variable, property, or other program element
  • synonym: key - a value used to access or index entries in a map, object, or dictionary structure
