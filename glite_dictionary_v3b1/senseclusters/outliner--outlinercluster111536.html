outliner

noun
tool for outlining
4.8
Pronunciation: /ˈaʊt.laɪ.nər/ /; / /aʊt.ˈlaɪ.nər/ (AmE), /ˈaʊt.laɪ.nə/ /; / /aʊt.ˈlaɪ.nə/ (BrE)

Examples

The art teacher recommended using an outliner to define the main features of the sketch.
