bloat

noun
excessive software size or resource use
7.6
Pronunciation: /bloʊt/ (AmE), /bləʊt/ (BrE)
Excessive increase in the size, complexity, or resource use of a software program, typically caused by added features, redundant code, or inefficient implementation.

Examples

Software bloat
Users blamed the update for the bloat that made the app slow and use more battery.
Developers refactored the code to remove unnecessary features and reduce bloat in the system.
The startup's product was praised for being lean, with minimal bloat compared to competitors' offerings.

Related Terms

  • synonym: code bloat - excessive or inefficient code that increases program size or reduces performance
  • antonym: efficiency - the degree to which a program uses resources effectively; the opposite of bloat
  • synonym: feature creep - the tendency for product scope to expand by adding more features, often causing complexity and delays
