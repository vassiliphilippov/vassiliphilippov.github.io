nan

noun
floating-point 'Not a Number' value
9.4
Pronunciation: / næn / (AmE), / næn / (BrE)
(computing) A special floating-point value meaning 'Not a Number', used to represent undefined or unrepresentable numeric results in calculations and data.

Examples

NaN error
When dividing zero by zero, the calculator displayed nan instead of a numeric result.
The data frame contained several nan entries after the faulty import, causing analysis errors.
To handle nan values, the script replaced them with the column mean before modeling.

  • synonym: undefined - not defined or not having a valid numeric value in a computational context
