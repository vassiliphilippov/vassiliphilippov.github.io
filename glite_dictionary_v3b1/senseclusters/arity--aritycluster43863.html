arity

noun
number of arguments a function takes
Pronunciation: / ˈɛrəti / (AmE), / ˈærɪti / (BrE)
the number of arguments or operands that a function, operation, or relation can take

Examples

Function arity number
Changing the arity of an operator can dramatically affect how it is parsed or understood.
In programming, you must know the arity of a function to use it correctly.
The arity of this function is two, so it requires exactly two arguments.
