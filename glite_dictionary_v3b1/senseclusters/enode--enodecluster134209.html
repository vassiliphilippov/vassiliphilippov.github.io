enode

noun
network node (blockchain)
9.7
Pronunciation: /ˈiː.noʊd/ /; / /ˈɪ.noʊd/ (AmE), /ˈiː.nəʊd/ /; / /ˈɪ.nəʊd/ (BrE)

Examples

Blockchain enode
Each enode in the network must be properly configured to communicate.
The blockchain relies on every enode to validate and share transactions.
If an enode goes offline, the network can still function using the remaining nodes.
