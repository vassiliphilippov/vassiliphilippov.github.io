enode

noun
network node (blockchain)
9.7
Pronunciation: / ˈiːnoʊd / (AmE), / ˈiːnəʊd / (BrE)
a node in a computer network, especially in blockchain or distributed systems, that participates in communication and data exchange

Examples

Blockchain enode
Each enode in the network must be properly configured to communicate.
The blockchain relies on every enode to validate and share transactions.
If an enode goes offline, the network can still function using the remaining nodes.

Related Terms

  • synonym: node - a point of connection in a network
  • synonym: peer - an equal participant in a peer-to-peer network
