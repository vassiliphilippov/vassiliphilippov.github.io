populate

verb
fill a database or form with data
4.9 1.39/m
Pronunciation: / Ààp…ëÀê.pj…ô.le…™t / (AmE), / Ààp…íp.j ä.le…™t / (BrE)
(in computing) to fill a database, table, form, or user interface with data, especially by inserting or initializing records programmatically.

Examples

To populate a database
The migration script will populate the user table with initial test records before deployment.
When creating the demo, she quickly populated the web form with sample names and emails.
The API supports batch requests for populating databases with thousands of records in minutes.

Related Terms

  • synonym: fill - to make something full or to supply it with the necessary contents.
  • synonym: seed - to insert an initial set of data into a database or system (often used in software development).
