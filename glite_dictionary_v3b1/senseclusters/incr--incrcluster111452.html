incr

verb
to increment by one
8.8
Pronunciation: / ɪŋkɹ̩ / (AmE), / ɪŋkɹ̩ / (BrE)
to increase a value by one, especially as a command or function in computing and programming

Examples

To incr a counter
In many programming languages, you can use incr to add one to a variable.
The script will automatically incr the counter each time the function is called.
If you want to track the number of attempts, just incr the value after each try.

Related Terms

  • synonym: increment - to increase by a set amount, often by one
