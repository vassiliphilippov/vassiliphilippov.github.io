building code

noun
regulations for safe building construction
6.9
Pronunciation: / ˈbɪldɪŋ koʊd / (AmE), / ˈbɪldɪŋ kəʊd / (BrE)
a set of official standards and regulations established by local authorities, governing the structural design, construction, and safety of buildings

Examples

Building code rules
Contractors must follow the building code to ensure all new homes pass inspection.
Failure to adhere to the building code can result in hefty fines and demolition orders.
The city updated its building code to require stronger materials for earthquake safety.
