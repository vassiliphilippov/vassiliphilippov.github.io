(in computing, especially cybersecurity) a program or piece of code that attaches to or runs within another program, system, or file without authorization, often to persist, spread, or steal data.

Examples

A computer parasite
Security researchers detected a parasite inside the mobile app that secretly exfiltrated user data.
The forensic tool managed to remove several parasites that had attached themselves to the host process.
During analysis she found a parasite that injected code into every running instance of the service.

Related Terms

  • synonym: malware - software designed to harm, exploit, or otherwise compromise computers or networks
  • synonym: virus - a type of malicious program that can replicate and spread by attaching to other programs or files
