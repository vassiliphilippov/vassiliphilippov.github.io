editor program

noun
software for editing data or files
4.0
Pronunciation: / ˈɛdɪtər ˈproʊɡræm / (AmE), / ˈɛdɪtə ˈprəʊɡræm / (BrE)
A computer program designed to modify, rearrange, or delete data, such as text or code files.

Examples

A editor program for code
I use a specialized editor program to modify source code for my projects.
Many writers rely on an editor program to check and revise their manuscripts before publishing.
The editor program allows easy rearrangement and deletion of data in large files.

Related Terms

  • synonym: word processor - software for editing and formatting text documents
