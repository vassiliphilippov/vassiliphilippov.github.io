inertial guidance system

noun
navigation system using inertial forces
6.2 5.7
Pronunciation: /ɪ.ˈnɜːr.ʃəl ˈɡaɪ.dəns ˈsɪs.təm/ (AmE), /ɪ.ˈnɜː.ʃəl ˈɡaɪ.dəns ˈsɪs.təm/ (BrE)
a navigation system used in aircraft or spacecraft that determines position and controls movement by relying on inertial forces rather than external references

Examples

Inertial guidance system in aircraft
During the test flight, engineers carefully monitored the inertial guidance system for any errors.
Modern missiles often use an inertial guidance system to improve their accuracy.
The inertial guidance system allows the spacecraft to navigate without external signals.
