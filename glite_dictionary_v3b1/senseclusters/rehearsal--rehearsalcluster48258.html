Synonyms

  • run-through
    4.2 0.5/m
    a run-through is an uninterrupted practice of the whole show, while a rehearsal often includes shorter, stopped sessions.
  • playthrough
    4.4 0.00/m
    a playthrough is a complete run of a piece from start to finish, while a rehearsal often stops and repeats parts.
  • walk-through
    5.1 0.08/m
    a walk-through is an informal read-through with scripts and notes, while a rehearsal normally aims to perform without scripts.
  • dry run
    4.1 0.2/m
    a dry run is a practice used to test systems or timings in any event, while a rehearsal focuses on performing arts.
