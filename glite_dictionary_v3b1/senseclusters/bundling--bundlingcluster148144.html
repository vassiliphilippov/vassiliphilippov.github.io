(in computing) the process of combining multiple resources (for example, JavaScript files, stylesheets, and images) into a single file or package for distribution or to improve loading and deployment performance.

Examples

Asset bundling process
The build pipeline performs bundling of assets to reduce HTTP requests.
After updating dependencies, their automatic bundling failed, causing slow page loads in production.
You can disable bundling in the dev configuration to debug individual JavaScript modules.
