screen memory

noun
concealing memory that hides distressing truth
9.4 0.04/m
Pronunciation: / skrin Ààm…õm…ôri / (AmE), / skriÀên Ààm…õm…ôri / (BrE)
A recollection of a childhood experience that is imagined or distorted, serving to mask or conceal another memory of greater emotional significance, often distressing.

Examples

A screen memory in childhood
In psychoanalysis, a screen memory may protect the individual from recalling distressing experiences.
She realized her early childhood happiness might have been a screen memory concealing deeper emotional wounds.
The therapist suggested that his vivid recollection of the birthday party was actually a screen memory hiding more traumatic events.
