unimodal

adjective
single data modality (ML)
8.6
Pronunciation: / ccjud0n6ac8mo8ad59l / (AmE), / ccjud0n6ac8m598ad59l / (BrE)
(in computing, especially machine learning) relating to or designed to use a single data modality or input type (for example, only text, only images, or only audio).

Examples

Unimodal model
The research team developed an unimodal model that processes only images for object recognition.
Unlike multimodal systems, this pipeline remains unimodal and analyzes text without audio or visual inputs.
For the competition, students built a fast unimodal classifier trained solely on speech recordings.

Related Terms

  • synonym: monomodal - having a single data modality or channel; often used interchangeably with 'unimodal'.
  • antonym: multimodal - involving two or more data modalities (for example, text and images together).
