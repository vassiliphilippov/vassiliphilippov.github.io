persistence

noun
durable storage of data/state
5.5 0.37/m
Pronunciation: / p59rc8s6ast59ns / (AmE), / p59c8s6ast59ns / (BrE)
(in computing) the property of data or program state being stored durably so it remains available after the program ends, a reboot, or a power loss.

Examples

Data persistence in memory
The framework provides transparent persistence by automatically saving user sessions to the database.
Configuring disk-based persistence ensured the application's state survived server restarts and crashes.
Developers debated whether to use in-memory caching or durable persistence for critical transaction records.

Synonyms

  • permanence
    5.5 2.00/m
    permanence is general lasting without major change for a long time, while persistence stores data so it survives program end or reboot.

Related Terms

  • synonym: durability - the quality of data remaining intact and usable over time
  • synonym: non-volatile storage - storage that retains data when power is removed, such as SSDs or hard drives
  • antonym: transience - the state of being temporary or short-lived; the opposite of durable storage
