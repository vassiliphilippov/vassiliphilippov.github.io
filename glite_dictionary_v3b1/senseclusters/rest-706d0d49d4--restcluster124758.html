(computing) an architectural style for designing networked applications that uses stateless HTTP methods to access and manipulate resources identified by URIs; acronym of Representational State Transfer.

Examples

REST architecture
The company rebuilt its public endpoints using REST to simplify client-server interactions.
Many developers prefer REST APIs because they are stateless and easy to cache and scale.
During the design review, she argued that a REST approach would match our microservices architecture.
