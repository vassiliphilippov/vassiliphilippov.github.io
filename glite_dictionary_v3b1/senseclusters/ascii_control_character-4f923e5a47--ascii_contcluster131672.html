ASCII control character

noun
non-printable control character in ASCII
7.4
Pronunciation: /ˈæs.kiː kən.ˈtroʊl ˈkær.ək.tər/ (AmE), /ˈæs.kiː kən.ˈtrəʊl ˈkær.ək.tə/ (BrE)
An ASCII character that performs a control function, such as carriage return, tab, or backspace, typed by pressing a key along with the control key.

Examples

ASCII control character key
Certain programs ignore ASCII control characters that do not affect the display output.
Pressing the Control-M key combination sends an ASCII control character representing a carriage return.
Some ASCII control characters are used to format text, such as tabs or new lines.
