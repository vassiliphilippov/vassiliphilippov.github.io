internal representation

noun
learned features inside a model
8.0
Pronunciation: /ɪn.ˈtɜːr.nəl re.prə.zen.ˈteɪ.ʃən/ /; / /ɪn.ˈtɜːr.nəl riː.prə.zen.ˈteɪ.ʃən/ (AmE), /ɪn.ˈtɜː.nəl re.prə.zen.ˈteɪ.ʃən/ /; / /ɪn.ˈtɜː.nəl riː.prə.zen.ˈteɪ.ʃən/ (BrE)
(in artificial intelligence and machine learning) a model's learned internal encoding of input data—often a vector or pattern of activations—used for prediction or downstream tasks.

Examples

AI internal representation
Modern deep networks learn an internal representation that captures important image features for classification.
Researchers visualized the internal representation of the network's layer to understand what it detects.
Transfer learning reuses learned internal representations from large models to speed up training on new tasks.

Related Terms

  • synonym: feature vector - an ordered set of numerical features representing an example for use in machine learning
  • synonym: latent representation - an internal, often lower-dimensional representation learned by a model to summarize input data
