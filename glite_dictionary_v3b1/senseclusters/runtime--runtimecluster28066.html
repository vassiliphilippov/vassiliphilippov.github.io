runtime

adjective
during program execution
5.3
Pronunciation: /ˈrʌn.taɪm/ (AmE), /ˈrʌn.taɪm/ (BrE)
happening or existing during the execution of a computer program, especially describing errors, checks, or behaviors that occur while a program runs

Examples

Runtime error
A runtime error can cause a program to stop unexpectedly while it is running.
The software includes runtime checks to catch invalid user input as the program executes.
Developers must test for runtime issues that may not appear during compilation.

Related Terms

  • antonym: compile-time - happening during the compilation of a program, before it runs
