integration

noun
software practice of frequent automated merging (CI)
4.2 4.23/m
Pronunciation: / cc6an.t59c861re6a.8359n / (AmE), / cc6an.t6ac861re6a.8359n / (BrE)
(in software engineering) the practice of frequently merging code changes into a shared repository and automatically building and testing them, often called continuous integration (CI).

Examples

Continuous integration
The team adopted integration practices to automatically merge code changes and run tests every time.
Their CI server failed, so the developers couldn't complete the nightly integration before the release deadline.
Automated tests caught a merge conflict during integration, preventing buggy code from reaching production.

Synonyms

  • continuous integration
    4.5 0.01/m
    continuous integration is the full developer practice of frequent automated merging, building, and testing, while integration is a shorter, more general term.
