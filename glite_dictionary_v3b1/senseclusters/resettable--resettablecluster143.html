resettable

adjective
able to be set again
4.0
Pronunciation: / rɪˈsɛtəbəl / (AmE), / rɪˈsɛtəbəl / (BrE)
able to be set again to its original or default state after being changed or interrupted

Examples

Resettable fuse
This type of fuse is resettable, so you do not need to replace it after every overload.
The password is resettable if you forget it, making account recovery easier.
Many modern devices have resettable settings to help users fix problems quickly.

Related Terms

  • antonym: irreversible - not able to be changed back to a previous state
  • synonym: reusable - able to be used again after being used
