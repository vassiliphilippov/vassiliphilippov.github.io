run down the clock

verb
waste or use time strategically

to deliberately use up or waste time in order to gain an advantage or prevent the opposing side from having the opportunity to act

Examples

To run down the clock deliberately
She attempted to run down the clock by taking extra time to answer questions.
The basketball team tried to run down the clock in the final two minutes of the game.
Their strategy was to run down the clock and leave the opponents with no chance to score.
