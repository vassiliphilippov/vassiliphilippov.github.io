loopback

noun
virtual network interface
Pronunciation: / ˈluːpbæk / (AmE), / ˈluːpbæk / (BrE)
a virtual network interface on a computer that routes outgoing network traffic back to the same machine, often for testing or local communication

Examples

Loopback address use
The developer used the loopback address to test the web application locally.
Configuring the loopback interface allows programs to communicate within the same computer.
Network troubleshooting often involves sending data to the loopback device to check local connectivity.

Related Terms

  • synonym: localhost - the standard hostname referring to the local computer
  • synonym: network interface - a point of connection for sending and receiving data on a network
