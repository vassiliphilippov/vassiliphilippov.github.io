gzip

verb
to compress files
7.6
Pronunciation: / ˈdʒiːˌzɪp / (AmE), / ˈdʒiːˌzɪp / (BrE)
to compress a file or files using the gzip program or the gzip compression format

Examples

To gzip a file
Before sending the documents, please gzip them to reduce the file size.
He forgot to gzip the backup, so it took longer to upload.
You can easily gzip large log files to save storage space.

Related Terms

  • synonym: compress - to reduce the size of a file or data
  • antonym: decompress - to restore a compressed file to its original state
