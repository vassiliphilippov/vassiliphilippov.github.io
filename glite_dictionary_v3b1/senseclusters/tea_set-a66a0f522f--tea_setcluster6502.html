tea set

noun
set of items for serving tea
1.4 0.28/m
Pronunciation: / Ààti s…õt / (AmE), / ÀàtiÀê s…õt / (BrE)
A matching set of china or silverware items, such as teapot, cups, and saucers, used for serving tea.

Examples

A tea set of china
She inherited her grandmother's beautiful tea set made of fine porcelain.
The museum displayed a Victorian tea set in a glass case.
We used the silver tea set during the afternoon gathering.

Synonyms

  • tea service
    2.4 0.33/m
    tea service is a fuller matching set that often includes a sugar bowl and cream jug, while tea set refers to the basic teapot, cups, and saucers.
