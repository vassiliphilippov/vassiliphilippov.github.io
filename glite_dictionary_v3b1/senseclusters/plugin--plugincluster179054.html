plugin

noun
add-on device
3.8
Pronunciation: / ˈplʌɡˌɪn / (AmE), / ˈplʌɡˌɪn / (BrE)
a device or module that is inserted into an electronic system to add or modify its functions

Examples

Audio plugin
The technician installed a new plugin to improve the sound quality of the audio system.
You can upgrade your synthesizer by adding a compatible plugin module.
This plugin allows the device to support additional input types.

Related Terms

  • synonym: extension - something that adds extra features to a system
  • synonym: module - a separate unit that can be added to a system
