hibernate

verb
save session and power off (computers)
6.3
Pronunciation: / c8ha6ab5accne6at / (AmE), / c8h8c6ab59ne6at / (BrE)
(computing) to save the current contents of a computer's memory to non-volatile storage and power the device down so the session can be restored later.

Examples

Laptop hibernates overnight
Before the flight, she set her laptop to hibernate so it would save work and turn off.
The technician hibernated the server overnight to apply updates and preserve memory state.
Closing all apps and hibernating the tablet helped him conserve battery during travel.

Related Terms

  • antonym: shutdown - to turn a computer off without saving the current session; in practice the opposite of hibernating.
  • synonym: suspend - to put a computer into a low-power state that keeps the session in memory (similar low-power mode).
