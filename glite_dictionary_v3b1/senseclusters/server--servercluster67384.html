server

noun
player serving in a court game
1.9
Pronunciation: / ˈsɝːvɚ / (AmE), / ˈsɜːvə / (BrE)
the player who serves to start a point in court games such as tennis or volleyball

Examples

Tennis server
After losing a point, she switched roles and became the server for the next game.
In tennis, the server must stand behind the baseline before hitting the ball.
The server delivered a powerful serve that scored an ace.

Related Terms

  • antonym: receiver - the player who receives the serve in court games
