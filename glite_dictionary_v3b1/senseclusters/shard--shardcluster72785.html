shard

noun
database or blockchain partition
7.2 0.11/m
Pronunciation: / ÊƒÉ‘rd / (AmE), / ÊƒÉ‘Ëd / (BrE)
a partition of a database, ledger, or other data store that contains only a subset of the total data, used to distribute storage and processing across multiple machines.

Examples

Database shard
The web app routed requests to the correct shard to reduce database latency during peak traffic.
Developers split the user table across multiple shards so each server handles fewer records.
In some blockchains, a shard processes only a portion of transactions to increase throughput.
