iframe

noun
embedded web element
Pronunciation: / ˈaɪfreɪm / (AmE), / ˈaɪfreɪm / (BrE)
an HTML element that allows one webpage to be embedded within another webpage

Examples

HTML iframe
The video tutorial was placed inside an iframe on the homepage for easy viewing.
Developers often use an iframe to display external content like maps or forms within a site.
Some websites block their pages from being shown in an iframe for security reasons.

Related Terms

  • synonym: frame - an HTML element used to divide a browser window into multiple sections
