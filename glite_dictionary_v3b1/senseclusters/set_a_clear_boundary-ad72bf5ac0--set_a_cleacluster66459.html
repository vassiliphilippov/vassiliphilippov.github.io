set a clear boundary

verb
establish a definite limit or separation
1.3
Pronunciation: / sɛt ə klɪr ˈbaʊndəri / (AmE), / sɛt ə klɪə ˈbaʊndəri / (BrE)
to establish a definite or distinct limit or separation between two areas, situations, or concepts

Examples

To set a clear boundary
As a manager, it's important to set a clear boundary between professional and personal relationships.
Parents should set a clear boundary for their children's use of technology.
To avoid misunderstandings, we agreed to set a clear boundary regarding our responsibilities.
