datasheet

noun
information document
3.7
Pronunciation: /ˈdeɪ.tə.ʃiːt/ /; / /ˈdɑː.tə.ʃiːt/ (AmE), /ˈdeɪ.tə.ʃiːt/ /; / /ˈdɑː.tə.ʃiːt/ (BrE)
a document, often in table form, that provides detailed information or technical specifications about a person, product, or system

You can export the results as a datasheet for further analysis in a spreadsheet program.
Each datasheet in the database contains detailed information about the listed products.
