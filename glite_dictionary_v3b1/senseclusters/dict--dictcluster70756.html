dict

noun
computing data structure
Pronunciation: / dɪkt / (AmE), / dɪkt / (BrE)
a data structure in computing, short for 'dictionary', that stores key–value pairs for fast lookup by key

Examples

Python dict object
In Python, a dict allows you to store and retrieve values using unique keys.
The configuration settings were saved in a dict for easy access during the program's execution.
She converted the list of tuples into a dict to improve lookup speed.

Related Terms

  • synonym: dictionary - a data structure that stores key–value pairs for fast access
  • antonym: list - an ordered collection of items
  • synonym: map - a data structure that associates keys with values
