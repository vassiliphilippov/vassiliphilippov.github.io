single file

noun
line with each behind the next
Pronunciation: / ˌsɪŋ.ɡəl ˈfaɪl / (AmE), / ˌsɪŋ.ɡəl ˈfaɪl / (BrE)
A line of people or things arranged with each directly behind the other.

Examples

Single file line
The children walked in single file along the narrow path.
The hikers formed a single file to cross the bridge.
Visitors must move through the exhibit in single file for safety reasons.
