tun

noun
virtual network tunnel interface
9.8
Pronunciation: / tʌn / (AmE), / tʌn / (BrE)
a virtual network interface on an operating system that provides a point-to-point IP tunnel, commonly used by VPNs and network tools to send or receive IP packets in user space.

Examples

Tun network device
A tun device provides a virtual network tunnel for IP packets in many VPN implementations.
When we installed the VPN, the system created a new tun interface visible to ifconfig.
Developers often write to a tun device to send crafted IP packets for testing.

Related Terms

  • synonym: virtual network interface - a software-based network interface that routes packets between user-space programs and the kernel
