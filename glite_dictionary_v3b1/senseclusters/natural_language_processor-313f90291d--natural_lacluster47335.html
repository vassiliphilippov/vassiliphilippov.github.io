natural language processor

noun
program for processing human language text
Pronunciation: / ˈnætʃrəl ˈlæŋɡwɪdʒ prəˈsɛsər / (AmE), / ˈnætʃ(ə)rəl ˈlaŋɡwɪdʒ prəˈsɛsə / (BrE)
A computer program or system designed to analyze, understand, or generate human language text.

Examples

A natural language processor system
Many virtual assistants rely on a natural language processor to interpret spoken commands.
Researchers are developing a natural language processor to translate text between multiple languages.
The new natural language processor can understand user queries with remarkable accuracy.
