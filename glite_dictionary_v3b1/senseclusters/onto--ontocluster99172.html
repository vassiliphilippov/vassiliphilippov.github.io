onto

adjective
surjective function
9.0
Pronunciation: / ˈɑntu / (AmE), / ˈɒntu / (BrE)
in mathematics, describing a function in which every element of the target set is the image of at least one input; surjective

Examples

Onto function mapping
A function is called onto if every value in the codomain is mapped by some input.
The teacher asked whether the mapping was onto or not for the given sets.
In mathematics, proving a function is onto often requires checking all possible outputs.

Related Terms

  • antonym: injective - a function in which each input maps to a unique output
  • synonym: surjective - a function that covers every element of its target set
