noncombining

adjective
not combining with diacritics (Unicode)
8.6
Pronunciation: /nɑːn.kəm.ˈbaɪ.nɪŋ/ /; / /nən.kəm.ˈbaɪ.nɪŋ/ (AmE), /nɒn.kəm.ˈbaɪ.nɪŋ/ /; / /nən.kəm.ˈbaɪ.nɪŋ/ (BrE)
(in Unicode and typography) not participating in combining; describing a character or code point that does not accept or attach combining diacritical marks.

Examples

Noncombining character
In Unicode, a noncombining character cannot accept following combining diacritical marks.
The font treats the space as noncombining, so accents remain unattached in rendering.
When normalizing text, the parser skips noncombining code points to preserve original diacritics.
