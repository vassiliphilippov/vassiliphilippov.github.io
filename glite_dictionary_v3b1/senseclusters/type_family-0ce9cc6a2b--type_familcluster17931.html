type family

noun
group of related typefaces
Pronunciation: / taɪp ˈfæməli / (AmE), / taɪp ˈfæməli / (BrE)
a complete set of related typefaces that share common design features but may vary in weight, width, or style

Examples

A type family in design
She mixed italics and bold from the same type family in her brochure design.
The designer chose a type family with several weights for the new website.
Using a single type family throughout the publication created a cohesive look.

  • synonym: typeface - a set of characters of the same design
