masquerade

noun
NAT hiding of source IP addresses
7.5
Pronunciation: /mæ.skə.ˈreɪd/ (AmE), /mæ.skə.ˈreɪd/ (BrE)
(in computing, especially networking) the practice or configuration of hiding private IP addresses by rewriting the source address of outgoing packets, i.e., NAT masquerading.

Examples

Network masquerade
The router applied a masquerade so all internal devices shared the ISP's public address.
During the course, engineers demonstrated how NAT masquerade rewrites source addresses for outgoing traffic.
To troubleshoot connectivity, we temporarily disabled the firewall's masquerade rule and tested direct access.

Related Terms

  • synonym: network address translation (NAT) - a method for remapping one IP address space into another by modifying network address information in IP packet headers
