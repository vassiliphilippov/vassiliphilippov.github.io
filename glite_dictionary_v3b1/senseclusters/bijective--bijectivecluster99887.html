describing a function in mathematics that is both one-to-one and onto, pairing each element of one set with exactly one element of another set

Examples

Bijective function
A function is bijective if every element in the target set is matched with exactly one element from the source set.
The teacher explained that a bijective mapping has both injective and surjective properties.
To prove the sets are the same size, we constructed a bijective correspondence between them.
