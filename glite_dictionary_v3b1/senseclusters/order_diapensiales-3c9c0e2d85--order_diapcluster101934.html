order Diapensiales

noun
a botanical order of flowering plants
Pronunciation: / ˈoʊrdər daɪˌæpɛnˈsiːəˌliːz / (AmE), / ˈɔːdə daɪˌæpɛnˈsiːəliːz / (BrE)
A botanical order of flowering plants, coextensive with the family Diapensiaceae, used in some plant classification systems.

Examples

The order Diapensiales
Botanists have debated the classification of the order Diapensiales in modern taxonomy.
The order Diapensiales contains only one family, Diapensiaceae.
The order Diapensiales is not widely recognized in all classification systems.
