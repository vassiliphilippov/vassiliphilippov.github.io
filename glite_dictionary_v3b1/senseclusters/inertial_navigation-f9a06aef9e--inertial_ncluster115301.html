inertial navigation

noun
navigation using inertial forces
Pronunciation: / ɪnˈɝːʃəl ˌnævɪˈɡeɪʃən / (AmE), / ɪnˈɜːʃəl ˌnævɪˈɡeɪʃən / (BrE)
a method of determining the position and velocity of a moving object, such as a missile or aircraft, using devices that measure inertial forces without relying on external references

Examples

Inertial navigation system
Engineers are improving inertial navigation to reduce error growth over lengthy flights.
Modern missiles rely on inertial navigation systems to maintain their course without external signals.
The aircraft's inertial navigation allowed it to fly accurately even when GPS was unavailable.

Related Terms

  • synonym: inertial guidance - a system using inertial forces to guide vehicles
  • antonym: satellite navigation - navigation using satellite signals
