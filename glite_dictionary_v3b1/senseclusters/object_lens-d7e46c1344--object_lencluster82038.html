object lens

noun
the main lens closest to the viewed object in optical instruments
Pronunciation: / ˈɑbdʒɪkt ˌlɛnz / (AmE), / ˈɒbdʒɪkt ˌlɛnz / (BrE)
the lens or system of lenses in a microscope or telescope nearest to the object being viewed, responsible for forming the primary image

Examples

The object lens in microscope
Dust on the object lens can greatly affect image quality in microscopy.
She carefully adjusted the object lens to bring the cells into sharp focus.
The object lens magnifies the specimen so the viewer can see it in detail.
