syntax error

noun
programming error breaking language rules
4.1 3.5
Pronunciation: /ˈsɪn.tæks ˈe.rər/ (AmE), /ˈsɪn.tæks ˈe.rə/ (BrE)
an error in a program caused by code that does not follow the grammatical rules of the programming language

Examples

A syntax error in code
For beginners, missing a semicolon often causes a syntax error.
The compiler reported several syntax errors in the source code.
The program crashed because of a syntax error on line 15.
