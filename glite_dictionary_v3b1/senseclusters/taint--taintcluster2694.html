taint

noun
untrusted or unsafe data (computing)
8.9
Pronunciation: /teɪnt/ (AmE), /teɪnt/ (BrE)
(computing, especially security) the state of a piece of data or a value that has been identified as originating from an untrusted or potentially unsafe source and therefore requiring validation or sanitization before use.

Examples

Taint in data
The static analyzer flagged the user input as taint before it reached the database layer.
During testing they tracked taint through the application to find paths leading to injection bugs.
The security team documented every taint source and required sanitization before using any input values.

Related Terms

  • synonym: taint analysis - a technique for tracking the flow of tainted data through a program to find security vulnerabilities
  • synonym: untrusted input - data received from external sources that cannot be assumed safe without validation
