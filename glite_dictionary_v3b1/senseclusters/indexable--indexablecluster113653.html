indexable

adjective
can be indexed
Pronunciation: / ˈɪndɛksəbəl / (AmE), / ˈɪndɛksəbəl / (BrE)
able to be included in an index or made searchable by an indexing system

Examples

Indexable web pages
Only indexable pages will appear in the search results.
The document must be saved in an indexable format for the database to recognize it.
Webmasters work to make their sites more indexable by search engines.

Related Terms

  • synonym: searchable - able to be searched or found using a search function
  • antonym: unsearchable - not able to be searched or found using a search function
