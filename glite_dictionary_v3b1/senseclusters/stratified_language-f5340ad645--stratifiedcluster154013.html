stratified language

noun
language not usable as its own metalanguage
9.9
Pronunciation: / ˈstræt̬.əˌfaɪd ˈlæŋ.ɡwɪdʒ / (AmE), / ˈstræt.ɪˌfaɪd ˈlaŋ.ɡwɪdʒ / (BrE)
A language that cannot be used as its own metalanguage, often requiring a separate system to describe its own structure or rules.

Examples

A stratified language system
Each level in a stratified language hierarchy serves a unique purpose in formal analysis.
In some theories, a stratified language is contrasted with a universal language that can describe itself.
Mathematical logic often relies on a stratified language to avoid self-referential paradoxes.

Related Terms

  • degree higher: metalanguage - a language used to talk about another language
  • synonym: object language - the language being described by a metalanguage
  • antonym: self-describing language - a language that can be used as its own metalanguage
