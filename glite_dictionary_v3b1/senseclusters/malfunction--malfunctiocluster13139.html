Examples

Synonyms

  • failure
    3.3 31/m
    failure is a complete stop or loss of function in a system or organ, while malfunction is any problem that prevents normal operation.
  • equipment failure
    3.4 0.17/m
    equipment failure is when a machine stops working completely, while malfunction can be any fault that makes it work poorly.
  • fault
    3.2 4.53/m
    fault is a defect in a part or system that causes errors, while malfunction is the event of not working properly.
  • glitch
    5.8 0.16/m
    glitch is a small, often temporary problem in a system, while malfunction is a broader failure to work correctly.
