connector

noun
software component that links systems
3.7
Pronunciation: / kəˈnɛktə / (AmE), / kəˈnɛktə / (BrE)
A software component or service that links two or more computer systems or applications to enable data exchange, synchronization, or API integration.

Examples

Software connector
Businesses often install a connector to synchronize customer data between their ERP and CRM systems overnight.
The cloud service provides a ready-made connector that moves orders from the online store to the fulfillment system.
Developers wrote custom connectors to transform data formats between legacy databases and modern APIs.

Related Terms

  • synonym: adapter - a component that allows incompatible systems or formats to work together
  • synonym: integration - the process or result of combining separate systems so they function together
  • synonym: middleware - software that connects different applications or services and manages data exchange
