anomalousness

noun
numeric anomaly score
9.1
Pronunciation: /ə.ˈnɑː.mə.ləs.nəs/ (AmE), /ə.ˈnɒ.mə.ləs.nəs/ (BrE)
(in data science and computing) a numeric score or metric that indicates how much a data point or event deviates from a model of normal behaviour; used to rank or flag anomalies.

Examples

Anomalousness score
The system outputs an anomalousness score for each transaction to flag potential fraud.
Investigators set a threshold so any transaction with high anomalousness triggers manual review immediately.
During model evaluation we compared predicted anomalousness values with true labels to measure detection accuracy.
