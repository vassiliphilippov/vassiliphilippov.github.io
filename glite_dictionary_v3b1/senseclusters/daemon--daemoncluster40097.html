daemon

noun
background program or autonomous software agent
7.6 0.48/m
Pronunciation: / ÀàdiÀêm…ôn / (AmE), / ÀàdiÀêm…ôn / (BrE)
(computing) a background program or autonomous software agent that runs without direct user interaction to provide services, coordinate tasks, or respond to events; (also in fiction/speculative contexts) an autonomous digital actor that controls systems or networks.

Examples

System daemon process
The system runs several daemons to handle printing, networking, and scheduled tasks automatically.
In the novel, a hidden daemon quietly manipulates traffic signals to create chaos citywide.
Security researchers described the autonomous daemon that coordinates sensor data across thousands of devices.

Related Terms

  • synonym: agent - a software entity that performs actions on behalf of users or other programs
  • synonym: background process - a program that runs behind the scenes and performs tasks without direct user interaction
