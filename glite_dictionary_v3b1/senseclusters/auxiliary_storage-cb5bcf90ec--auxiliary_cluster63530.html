auxiliary storage

noun
secondary data storage device
Pronunciation: / ɔgˈzɪljəri ˈstɔrɪdʒ / (AmE), / ɔːgˈzɪl.jə.ri ˈstɔː.rɪdʒ / (BrE)
a data storage device used in a computer system that is not the main memory, typically used to store data long-term

Examples

External auxiliary storage
Floppy disks and external hard drives are examples of auxiliary storage.
Many large databases are kept on auxiliary storage devices to free up RAM.
The backup files were written to auxiliary storage for long-term preservation.

Related Terms

  • synonym: secondary storage - memory or devices used to store data permanently, not as main memory
