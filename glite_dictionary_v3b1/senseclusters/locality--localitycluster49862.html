locality

noun
data/cache locality (computing)
Pronunciation: / loʊˈkælɪti / (AmE), / ləʊˈkælɪti / (BrE)
(computing) The tendency for a processor to access the same or nearby memory locations within a short time; keeping data or computation physically or logically close to reduce memory/cache access time and improve performance.

Examples

Locality of data
Modern compilers optimize locality to reduce cache misses and make programs run faster.
Good locality in data structures means related items are stored near each other in memory.
Engineers improve locality by reorganizing loops to access arrays in a cache-friendly order.

Related Terms

  • synonym: cache locality - the property of a program's memory access pattern that makes good use of the CPU cache
  • synonym: locality of reference - the principle that programs tend to access the same set of memory locations repeatedly in a short period
