backend

adjective
server-side operations
Pronunciation: / ˈbækˌɛnd / (AmE), / ˈbækˌɛnd / (BrE)
relating to the part of a computer system or application that is not directly accessed by users, typically handling data processing or server-side operations

Examples

Backend server software
The company hired new engineers to improve the backend systems for faster data processing.
She specializes in backend development, focusing on databases and server logic.
A secure backend is essential for protecting user information in any web application.

Related Terms

  • antonym: frontend - relating to the part of a computer system users interact with directly
  • synonym: server-side - relating to operations performed on a server
