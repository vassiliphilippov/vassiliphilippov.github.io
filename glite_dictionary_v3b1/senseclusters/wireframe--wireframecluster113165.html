wireframe

noun
website layout sketch
Pronunciation: / ˈwaɪərˌfreɪm / (AmE), / ˈwaɪəˌfreɪm / (BrE)
a simple visual outline or sketch that shows the basic structure and layout of a website or application interface

Examples

Website wireframe
Before designing the final website, the team created a wireframe to plan the layout.
The client requested changes after reviewing the initial wireframe of the mobile app.
A clear wireframe helps developers understand where each element should be placed.

A wireframe helps developers and clients agree on the structure before adding details.
