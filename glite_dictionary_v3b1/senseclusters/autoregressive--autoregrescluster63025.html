autoregressive

adjective
predicting from past values
9.1
Pronunciation: / ˌɔtoʊrɪˈɡrɛsɪv / (AmE), / ˌɔːtəʊrɪˈɡrɛsɪv / (BrE)
describing a statistical model in which future values are predicted using previous values in the same series

Examples

Autoregressive statistical model
An autoregressive process is often used in time series analysis to model economic data.
The autoregressive approach assumes that each value depends on its own previous values.
Researchers applied an autoregressive model to forecast next month's temperature trends.

Related Terms

  • synonym: predictive - relating to the ability to predict future events
  • antonym: random - lacking any predictable order or plan
