OAS

noun
OpenAPI Specification (API description standard)
Pronunciation: / oʊ eɪ ɛs / (AmE), / əʊ eɪ ɛs / (BrE)
Abbreviation for the OpenAPI Specification, a machine-readable standard for describing RESTful APIs, including endpoints, parameters, and responses.

Examples

OAS file
The development team published the REST endpoints in a single OAS file for the frontend.
He used an online tool to validate the project's OAS before generating client code.
Companies often include a version number when they release a new OAS to partners.

Related Terms

  • synonym: API specification - A formal description of API endpoints, parameters, request and response formats.
  • synonym: OpenAPI - A specification and ecosystem for describing RESTful APIs in a standard, machine-readable format.
  • synonym: Swagger - An earlier, related format and toolset for describing and working with REST APIs, historically linked to OpenAPI.
