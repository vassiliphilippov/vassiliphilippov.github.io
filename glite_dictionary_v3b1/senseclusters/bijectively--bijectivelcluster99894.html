bijectively

adverb
one-to-one mapping
Pronunciation: / baɪˈdʒɛktɪvli / (AmE), / baɪˈdʒɛktɪvli / (BrE)
in a way that establishes a one-to-one correspondence between elements of two sets

Examples

To bijectively map sets
The function f maps set A bijectively onto set B, ensuring every element is paired uniquely.
If two sets are related bijectively, they have the same number of elements.
The mathematician proved that the two groups correspond bijectively under the given transformation.
