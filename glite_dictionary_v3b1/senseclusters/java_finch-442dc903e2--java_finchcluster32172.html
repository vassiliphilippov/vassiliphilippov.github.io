Java finch

noun
Indonesian finch-like weaverbird
Pronunciation: / ˈdʒɑːvə fɪntʃ / (AmE), / ˈdʒɑːvə fɪntʃ / (BrE)
a small finch-like weaverbird native to Indonesia, often found in rice fields and commonly kept as a cage bird

Examples

A Java finch bird
Rice farmers often see flocks of Java finches feeding in their fields.
The Java finch is a popular pet due to its attractive plumage and pleasant song.
The zoologist observed a pair of Java finches nesting near the edge of the wetland.
