alt

noun
keyboard modifier key
Pronunciation: / ælt / (AmE), / ɒlt / (BrE)
a key on a computer keyboard labeled 'Alt' that is used with other keys to perform special functions or shortcuts

Examples

Alt key shortcut
To select multiple files, hold down the alt key while clicking each one.
Some shortcuts require you to press the alt key together with another key.
The alt key is often used to access menu options in many programs.

Related Terms

  • synonym: control key - a keyboard key used with others to perform special operations
  • synonym: shift key - a keyboard key used to type capital letters or access alternate characters
