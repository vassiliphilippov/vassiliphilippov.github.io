jail

noun
isolated software container
Pronunciation: / dʒeɪl / (AmE), / dʒeɪl / (BrE)
(computing) an isolated environment or container that restricts an application's access to other parts of the system and its resources.

Examples

A software jail
The administrator launched a jail to run the test service in isolation.
Developers deployed multiple jails to separate each microservice on the same host.
Security guidelines recommend running untrusted code inside a jail to limit system access.

Related Terms

  • synonym: container - a lightweight, portable unit for packaging software and its dependencies, often isolated from the host system
  • synonym: sandbox - an isolated environment for running untrusted code with restricted privileges
