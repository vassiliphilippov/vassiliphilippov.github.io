federated

adjective
relating to computing federation

(computing) relating to a federation model in which multiple independent systems, services, or organizations cooperate using shared protocols or standards while keeping local control; used in phrases such as "federated identity," "federated learning," and "federated search."

Examples

Federated identity system
Many universities rely on federated identity systems to authenticate students across partner services.
The project uses federated learning so hospitals can train models without sharing patient records.
A federated search pulls results from several independent databases while keeping each database private.

Related Terms

  • antonym: centralized - organized so that control and decision-making are concentrated in a single center; opposite of federated
  • synonym: federation - a system in which multiple entities cooperate under common rules while remaining autonomous
