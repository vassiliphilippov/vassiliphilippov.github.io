authoring language

noun
software for creating interactive programs
Pronunciation: / ˈɔθɚɪŋ ˌlæŋɡwɪdʒ / (AmE), / ˈɔːθərɪŋ ˌlæŋɡwɪdʒ / (BrE)
A software tool that enables users to create interactive computer programs without needing to perform detailed computer programming.

Examples

An authoring language tool
Many educational websites are built using an authoring language that allows teachers to design interactive lessons.
The company released a new authoring language aimed at simplifying e-learning content creation.
Using an authoring language, developers can quickly prototype multimedia applications without advanced programming knowledge.
