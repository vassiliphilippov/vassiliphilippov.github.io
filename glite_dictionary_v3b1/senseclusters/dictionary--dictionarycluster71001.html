dictionary

noun
key–value data structure
4.3
Pronunciation: /ˈdɪk.ʃə.nər.iː/ /; / /ˈdɪk.ʃən.riː/ (AmE), /ˈdɪk.ʃən.riː/ /; / /ˈdɪk.ʃə.nə.riː/ (BrE)
(in computing) a data structure that stores associations between keys and values, allowing programs to look up, add, or remove a value by its key quickly.

Examples

Python dictionary
In Python, a dictionary maps keys to values for fast lookups in many programs.
She stored user preferences in a dictionary so the app could quickly retrieve settings.
When parsing the file, the developer built a dictionary from pairs of names and IDs.

Related Terms

  • synonym: associative array - a collection of (key, value) pairs where each key maps to a value; commonly used term in programming.
  • synonym: hash table - an implementation of a dictionary that uses a hash function to map keys to storage locations, enabling fast access.
  • synonym: map - a data structure that associates keys with values; often used in high-level programming languages.
