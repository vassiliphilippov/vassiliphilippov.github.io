multithreaded

adjective
capable of parallel tasks
Pronunciation: / ˌmʌltiˈθrɛdɪd / (AmE), / ˌmʌltiˈθrɛdɪd / (BrE)
able to perform multiple tasks or processes at the same time, especially in computing

Examples

Multithreaded program
The new software is multithreaded, allowing it to process data much faster than before.
A multithreaded server can handle many user requests at the same time.
Developers often design applications to be multithreaded for better performance on modern computers.

Related Terms

  • synonym: concurrent - happening or done at the same time
  • antonym: single-threaded - able to perform only one task or process at a time
