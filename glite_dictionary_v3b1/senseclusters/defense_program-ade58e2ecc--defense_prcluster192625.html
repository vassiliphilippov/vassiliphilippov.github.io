defense program

noun
computer security software
Pronunciation: / dɪˈfɛns ˈproʊɡræm / (AmE), / dɪˈfens ˈprəʊɡræm / (BrE)
A software application or suite designed to protect computers and networks from cyber threats such as malware, viruses, or intrusions.

Examples

Computer defense program
The organization deployed a new defence program to shield its servers from cyber intrusions.
After the breach, the IT team installed a lightweight defence program on every workstation.
Consumers choose a reputable defence program when seeking protection against malware and phishing scams.

Related Terms

  • synonym: antivirus - software specifically designed to detect and remove malicious software
  • synonym: security program - a software tool that protects devices and networks from cyber threats
