freestyle

noun
race allowing any style (esp. swimming)
Pronunciation: / ˈfriːˌstaɪl / (AmE), / ˈfriːˌstaɪl / (BrE)
a swimming race, or similar competition, in which competitors may use any style or technique they choose

Examples

Freestyle swimming race
At the next meet, they will compete in both the butterfly and the freestyle events.
He prefers the freestyle because it allows him to swim using his strongest stroke.
She set a new world record in the women's 100-meter freestyle.

Related Terms

  • antonym: backstroke - a swimming race using only the backstroke technique
  • antonym: breaststroke - a swimming race using only the breaststroke technique
  • antonym: butterfly - a swimming race using only the butterfly stroke
