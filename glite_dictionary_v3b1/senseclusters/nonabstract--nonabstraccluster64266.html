nonabstract

adjective
concrete in programming
7.5
Pronunciation: / ˌnɑnˈæbstrækt / (AmE), / ˌnɒnˈæbstrækt / (BrE)
in programming, not declared as abstract and able to be instantiated or provide a concrete implementation

Examples

Nonabstract class
The developer created a nonabstract class so it could be used directly in the application.
Unlike abstract classes, a nonabstract class can be instantiated to create objects.
They refactored the code to make the base class nonabstract for easier testing.

Related Terms

  • antonym: abstract - not providing a concrete implementation; cannot be instantiated
  • synonym: concrete - providing a specific, usable implementation
