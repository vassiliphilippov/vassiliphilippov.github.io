shape

noun
array/tensor dimensions (computing)
6.5
Pronunciation: /ʃeɪp/ (AmE), /ʃeɪp/ (BrE)
(computing) the dimensions of a multi‑dimensional array or tensor; a tuple or list that gives the size along each axis.

Examples

Array shape
The NumPy array's shape indicates its three dimensions: (3, 4, 2).
Before concatenating, check that both arrays have the same shape to avoid broadcasting errors.
The model expected inputs with shape (1, 28, 28), so I reshaped the image batch accordingly.

Related Terms

  • synonym: dimension - one axis or size measure of a data array or tensor
  • synonym: size - the number of elements along an axis or overall number of elements in an array
