highlighter

noun
software code colorizer
3.6
Pronunciation: /ˈhaɪ.laɪ.tər/ (AmE), /ˈhaɪ.laɪ.tə/ (BrE)
(computing) A software feature or component (often in an editor or IDE) that uses colors or styles to mark and differentiate parts of source code (keywords, strings, variables) to improve readability.

Examples

Code highlighter
The IDE's highlighter colorizes keywords and strings to help developers spot errors quickly.
She enabled the highlighter to show variables in different colors while debugging the application.
Many text editors include highlighters for dozens of languages from Python to JavaScript.

Related Terms

  • synonym: syntax highlighting - a feature that displays programming language elements in different colors or fonts according to their syntax
