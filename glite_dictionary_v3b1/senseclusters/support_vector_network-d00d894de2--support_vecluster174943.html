support vector network

noun
machine learning classifier using hyperplanes
8.1
Pronunciation: /sə.ˈpɔːrt ˈvek.tər ˈnet.wɜːrk/ (AmE), /sə.ˈpɔːt ˈvek.tə ˈnet.wɜːk/ (BrE)
A machine learning classifier that constructs one or more hyperplanes in a high- or infinite-dimensional space for classification, regression, or other tasks.

Examples

A support vector network model

The support vector network effectively separated the two classes of data points in the dataset.
Training a support vector network on large datasets can require significant computational resources.
