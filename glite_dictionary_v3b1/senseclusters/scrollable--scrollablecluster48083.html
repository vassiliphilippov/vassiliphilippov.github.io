scrollable

adjective
can be scrolled
Pronunciation: / ˈskroʊləbəl / (AmE), / ˈskrəʊləbəl / (BrE)
able to be moved up, down, or sideways on a screen to view additional content

Examples

Scrollable screen area
The chat window is scrollable, allowing you to read earlier messages easily.
Make sure the image gallery is scrollable on mobile devices for better user experience.
If the page is not scrollable, you might miss important information at the bottom.

Related Terms

  • synonym: draggable - able to be moved by dragging with a mouse or finger
  • antonym: fixed - not able to be moved or changed
