EBCDIC

sym
IBM character encoding
Pronunciation: / ˈɛb sɪ dɪk / (AmE), / ˈɛb sɪ dɪk / (BrE)
a character encoding system developed by IBM, mainly used on mainframe computers

Examples

EBCDIC code
The legacy banking system still processes data using EBCDIC instead of modern encodings.
To transfer files between platforms, the EBCDIC format must be converted to ASCII.
Many IBM mainframes rely on EBCDIC for representing text characters internally.

Related Terms

  • antonym: ASCII - a widely used character encoding standard for computers
