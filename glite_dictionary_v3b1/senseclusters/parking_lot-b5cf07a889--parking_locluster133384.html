parking lot

noun
list for deferred ideas or issues
4.4
Pronunciation: /ˈpɑːr.kɪŋ lɑːt/ /; / /ˈpɑːr.kɪŋ lɔːt/ (AmE), /ˈpɑː.kɪŋ lɒt/ (BrE)
an informal, metaphorical term for a list or place where ideas, topics, or issues are put aside temporarily to be discussed or dealt with later

Examples

Parking lot for ideas
We moved that suggestion to the parking lot so the team could focus on the urgent issues.
During the meeting, the manager added several technical questions to the parking lot for later review.
If you have ideas, write them on the shared board under the parking lot so they aren't forgotten.
