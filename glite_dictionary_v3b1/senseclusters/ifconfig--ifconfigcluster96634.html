ifconfig

verb
to configure network interface
Pronunciation: / ɪfˈkɒnˌfɪɡ / (AmE), / ɪfˈkɒnˌfɪɡ / (BrE)
to configure or check network interface settings by running the ifconfig command on a computer system

Examples

To ifconfig a device
Before troubleshooting the connection, try to ifconfig the device to check its IP address.
He asked me to ifconfig the server after installing the new network card.
You should always ifconfig after making changes to the network settings.

Related Terms

  • synonym: configure - to set up or arrange settings on a computer or device
  • synonym: ping - to test the reachability of a host on a network
