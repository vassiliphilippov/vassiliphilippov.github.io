stack

noun
LIFO data structure in computing
Pronunciation: / stæk / (AmE), / stak / (BrE)
a data structure in computing where the last item added is the first to be removed (LIFO stack)

Examples

Call stack
Every function call in this program uses the call stack.
Pushing and popping elements are basic operations on a stack.
The stack allows you to access the last item you placed in it first.
The program uses a stack to keep track of function calls.
Understanding how a stack works is essential for solving certain algorithmic problems.
When you push items onto a stack, the last one added will be the first one removed.

Related Terms

  • antonym: queue - a data structure in which the first item added is the first to be removed (FIFO)
