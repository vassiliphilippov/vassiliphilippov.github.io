unsee

verb
remove data from datasets or models
9.1 0.00/m
Pronunciation: /  ånÀàsi / (AmE), /  ånÀàsiÀê / (BrE)
(in computing and data science) to remove, delete, or exclude data or information from a dataset, memory, or model so it no longer affects future outputs or recalls.

Examples

To unsee data
The firm promised to unsee the biased data from its training set to reduce model skew.
After the audit, the team started unseeing personal images from the dataset and logging each removal.
Engineers wrote scripts to unsee identifiers so the model would no longer reproduce private details.
