typable

adjective
can be typed (programming)
Pronunciation: / ˈtaɪpəbəl / (AmE), / ˈtaɪpəbəl / (BrE)
able to be assigned a type according to the rules of a programming language's type system.

Examples

Typable function
The function is typable in most statically typed languages, making it easier to catch errors.
If a variable is not typable, the compiler will usually produce an error message.
Advanced type systems can determine whether complex expressions are typable or not.

Related Terms

  • antonym: untypable - not able to be assigned a type in a type system
  • synonym: well-typed - having a type that fits the rules of a type system
