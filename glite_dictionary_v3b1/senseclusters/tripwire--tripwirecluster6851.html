tripwire

noun
security monitoring tool
Pronunciation: / ˈtrɪpˌwaɪr / (AmE), / ˈtrɪpˌwaɪə / (BrE)
a security program or tool that monitors files or systems to detect unauthorized changes or activity

Examples

Install a tripwire
The company uses a tripwire to alert administrators if any system files are changed unexpectedly.
After a cyberattack, the IT team relied on their tripwire to identify which files had been altered.
Installing a tripwire can help organizations detect unauthorized access to sensitive data.

Related Terms

  • antonym: firewall - a security system that controls incoming and outgoing network traffic
  • synonym: intrusion detection system - a system that monitors networks or systems for malicious activity
