architectural style

noun
software design constraints and conventions
6.2
Pronunciation: /ɑːr.kə.ˈtek.tʃə.rəl staɪl/ /; / /ɑːr.kɪ.ˈtek.tʃə.rəl staɪl/ (AmE), /ɑː.kɪ.ˈtek.tʃə.rəl staɪl/ /; / /ɑː.kə.ˈtek.tʃə.rəl staɪl/ (BrE)
(in computing) a set of constraints, conventions, and organization principles that determine how components of a software system interact and are structured.

Examples

REST architectural style
The REST architectural style defines constraints for building scalable web services.
Our team decided to adopt an architectural style that emphasizes independent services and asynchronous communication.
When evaluating each architectural style, consider trade-offs for scalability, maintainability, and performance under load.

  • synonym: architectural pattern - a reusable solution or template for solving common problems in software architecture
  • synonym: software architecture - the overall structure and organization of a software system, including components and their relationships
