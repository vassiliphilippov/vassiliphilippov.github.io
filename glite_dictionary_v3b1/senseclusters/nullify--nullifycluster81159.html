nullify

verb
set to null (computing)
6.6
Pronunciation: /ˈnʌl.ə.faɪ/ /; / /ˈnʌl.ɪ.faɪ/ (AmE), /ˈnʌl.ɪ.faɪ/ /; / /ˈnʌl.ə.faɪ/ (BrE)
(computing) to set a variable, field, or other data storage to a null value, removing or erasing its previous content.

Examples

To nullify a field
After the migration, the script will nullify empty usernames to prevent legacy data errors.
The database trigger nullified the outdated field whenever a user requested data removal.
Developers warned that accidentally nullifying the session token could log a user out immediately.

Related Terms

  • synonym: erase - to remove recorded data or information.
  • synonym: null - to set a value to null (often used in computing to indicate an absence of data).
  • synonym: reset - to restore a system, value, or setting to an initial or default state.
