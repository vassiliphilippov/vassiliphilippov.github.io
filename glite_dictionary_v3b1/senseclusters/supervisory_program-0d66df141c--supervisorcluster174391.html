supervisory program

noun
program that manages other programs
2.7 4.6
Pronunciation: /suː.pər.ˈvaɪ.zər.iː ˈproʊ.græm/ /; / /suː.pər.ˈvaɪ.zər.iː ˈproʊ.grəm/ (AmE), /suː.pə.ˈvaɪ.zə.riː ˈprəʊ.græm/ /; / /suː.pə.ˈvaɪ.zə.riː ˈprəʊ.grəm/ (BrE)
A program, especially in computing, that manages and controls the execution of other programs.

Examples

A supervisory program manages tasks
A fault in the supervisory program can cause system-wide failures.
Many mainframe systems rely on a supervisory program to coordinate multiple tasks efficiently.
The supervisory program ensures that user applications do not interfere with each other's operation.
