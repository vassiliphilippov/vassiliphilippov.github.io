eval

noun
code evaluation function
9.4
Pronunciation: / ˈiːˌvæl / (AmE), / ˈiːˌvæl / (BrE)
a function or command in programming that evaluates or executes a string or expression as code at runtime

Examples

Eval function call
Many programming languages include an eval function to execute code stored as a string.
Using eval can be risky if the input is not properly checked for security.
The developer replaced eval with a safer alternative to prevent code injection attacks.
