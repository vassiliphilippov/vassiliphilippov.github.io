unpatched

adjective
not connected (network)
7.5
Pronunciation: / ʌnˈpætʃt / (AmE), / ʌnˈpætʃt / (BrE)
not connected or routed using patch cables or a patch panel, especially in a network or system setup

Examples

Unpatched network cable
The technician noticed that the cable was still unpatched and quickly connected it to the network.
Several ports on the switch remained unpatched after the initial installation.
If the device is unpatched, it will not be able to access the internet.

Related Terms

  • synonym: disconnected - not joined or linked to a system or network
  • antonym: patched - connected or routed using patch cables or a patch panel
