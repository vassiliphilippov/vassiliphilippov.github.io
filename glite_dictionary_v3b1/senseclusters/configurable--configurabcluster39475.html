configurable

adjective
able to be customized
Pronunciation: / kənˈfɪɡjərəbəl / (AmE), / kənˈfɪɡjʊrəbəl / (BrE)
able to be changed or set up according to preferences or needs

Examples

A configurable system
The software offers a highly configurable interface to meet different business requirements.
Each user can create their own configurable dashboard with preferred widgets and layouts.
The system is configurable so that it can adapt to various operating environments.

Related Terms

  • synonym: customizable - able to be modified to suit individual needs
  • antonym: fixed - not able to be changed or adjusted
