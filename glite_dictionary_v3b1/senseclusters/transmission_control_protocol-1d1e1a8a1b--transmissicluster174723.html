Transmission Control Protocol

noun
protocol for reliable data transfer on networks
3.9
Pronunciation: /trænz.ˈmɪʃ.ən kən.ˈtroʊl ˈproʊ.tə.kɑːl/ /; / /træns.ˈmɪʃ.ən kən.ˈtroʊl ˈproʊ.tə.kɑːl/ /; / /trænz.ˈmɪʃ.ən kən.ˈtroʊl ˈproʊ.tə.kɔːl/ (AmE), /trænz.ˈmɪʃ.ən kən.ˈtrəʊl ˈprəʊ.tə.kɒl/ /; / /træns.ˈmɪʃ.ən kən.ˈtrəʊl ˈprəʊ.tə.kɒl/ (BrE)
a standardized set of rules (protocol) used to enable reliable communication and data transfer between devices on a computer network, particularly the internet

Examples

Transmission Control Protocol rules
Data sent via the Transmission Control Protocol can be checked for errors and retransmitted if necessary.
Many websites use Transmission Control Protocol to ensure that information sent between servers and clients is accurate and reliable.
The Transmission Control Protocol establishes a connection before data begins to flow across the internet.
