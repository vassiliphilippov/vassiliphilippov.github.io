derived function

noun
function found by differentiation
Pronunciation: / dɪˈraɪvd ˈfʌŋkʃən / (AmE), / dɪˈraɪvd ˈfʌŋkʃən / (BrE)
a function obtained by differentiating another function; it gives the rate of change of the original function with respect to its variable

Examples

The derived function rate
If the original curve is quadratic, its derived function will be linear.
In calculus, finding the derived function is one of the main operations.
The derived function of f(x) tells us the slope at any given point.

Related Terms

  • synonym: derivative - the standard term for the function representing the instantaneous rate of change
