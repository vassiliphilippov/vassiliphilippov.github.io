journalize

verb
to log system changes
7.9
Pronunciation: / ˈdʒɝːnəˌlaɪz / (AmE), / ˈdʒɜːnəlaɪz / (BrE)
to record changes or events in a computer or system journal so they can be recovered after a failure

Examples

To journalize system events
Modern databases journalize every transaction to ensure data can be restored after a crash.
The system will automatically journalize all changes made during the update process.
Engineers rely on software that can journalize events for troubleshooting and recovery.

Related Terms

  • synonym: log - to record events or data systematically
  • synonym: record - to write down or store information for future use
