CDF

noun
interactive document format
Pronunciation: / si di ɛf / (AmE), / siː diː ɛf / (BrE)
a file format developed by Wolfram Research for interactive documents containing live computations and dynamic content

Examples

Wolfram CDF document
The professor shared the simulation as a CDF so students could explore the results interactively.
You need special software to open a CDF and use its dynamic features.
Researchers published their findings in a CDF to allow readers to manipulate the data.

Related Terms

  • antonym: PDF - a widely used file format for fixed-layout documents
  • synonym: notebook - an interactive document used in computational software
