TPM

noun
transactions per minute
7.5
Pronunciation: / ti pi ɛm / (AmE), / tiː piː em / (BrE)
a measure of the number of transactions completed in one minute, used to assess computer or database performance

Examples

TPM value rises
The database administrator monitored the TPM to ensure the system could handle increased traffic.
During the stress test, the server reached a peak TPM of 5,000.
Improving hardware can significantly boost the TPM of a transaction processing system.

Related Terms

  • antonym: latency - the delay before a transfer of data begins following an instruction
  • synonym: throughput - the amount of work performed by a system in a given period
