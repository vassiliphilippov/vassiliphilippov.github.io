semibold

adjective
medium-heavy typeface
5.2
Pronunciation: / ˈsɛmiˌboʊld / (AmE), / ˈsɛmiˌbəʊld / (BrE)
describing a typeface weight that is heavier than regular but lighter than bold

Examples

Semibold text style
The designer chose a semibold font for the subheadings to create a clear hierarchy.
Please use semibold text for emphasis instead of bold in this document.
The logo features the company name in a modern semibold typeface.
