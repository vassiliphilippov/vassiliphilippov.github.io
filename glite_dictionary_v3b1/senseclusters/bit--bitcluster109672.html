bit

noun
unit of information in computing
4.4 10/m
Pronunciation: / bÉªt / (AmE), / bÉªt / (BrE)
a unit of information in computing, equal to the amount of information that can be stored by a device or system with two possible states, usually 0 or 1

Examples

Computer bit
A single bit can represent either a 0 or a 1 in a computer.
Each character in this file takes up eight bits of memory.
The speed of the processor is measured by how quickly it can handle each bit of information.
