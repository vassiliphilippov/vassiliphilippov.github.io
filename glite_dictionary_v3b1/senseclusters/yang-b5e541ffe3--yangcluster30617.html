YANG

noun
network data‑modeling language
Pronunciation: / jæŋ / (AmE), / jæŋ / (BrE)
(in computing, especially network management) a data‑modeling language used to define configuration and state data for network devices; commonly used with protocols such as NETCONF and RESTCONF.

Examples

YANG data model
Network engineers wrote a YANG module to describe the router's interface and routing settings.
The team used a YANG schema to automatically push configuration to hundreds of switches.
After upgrading, she validated the device with a YANG model to prevent configuration errors.

Related Terms

  • synonym: data model - a formal description of the structure and meaning of data used by software or systems
  • synonym: schema - a structured definition of how data is organized, similar to a YANG module
