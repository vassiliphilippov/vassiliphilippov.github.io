DMZ

noun
network subzone hosting public services
7.3 0.03/m
Pronunciation: / di …õm zi / (AmE), / diÀê …õm z…õd / (BrE)
(in computing, especially networking and cybersecurity) a subnetwork that separates an internal trusted network from an untrusted external network, while hosting public-facing services (for example, web or mail servers) to reduce risk to internal systems.

Examples

Network DMZ
The company's public web server is placed in the DMZ to reduce risk.
Network administrators often monitor the DMZ more closely than internal segments during attacks.
Many organizations operate multiple DMZs to separate services with different security policies.

Related Terms

  • antonym: internal network - the trusted, private network inside an organization that the DMZ is designed to protect
  • synonym: perimeter network - a network segment that provides a boundary between an internal network and external networks, often hosting public services
