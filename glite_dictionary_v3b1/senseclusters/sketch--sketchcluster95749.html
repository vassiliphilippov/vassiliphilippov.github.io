sketch

noun
compact probabilistic data summary
8.6
Pronunciation: /skɛtʃ/ (AmE), /skɛtʃ/ (BrE)
(in computer science) a compact, probabilistic data structure that summarizes a large data stream so one can estimate frequencies, counts, or distributions approximately using very little memory.

Examples

A data sketch
The system uses a sketch to estimate item frequencies in a data stream with limited memory.
Researchers compared different sketch implementations to find the fastest with acceptable error rates.
They used a Count-Min sketch to approximate heavy hitters in the network traffic efficiently.

Related Terms

  • synonym: probabilistic data structure - a data structure that provides approximate answers with bounded error using less memory than exact methods
  • synonym: summary - a brief statement giving the main points of something
