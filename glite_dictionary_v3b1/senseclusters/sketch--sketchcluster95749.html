(in computer science) a compact, probabilistic data structure that summarizes a large data stream so one can estimate frequencies, counts, or distributions approximately using very little memory.

Examples

A data sketch

Researchers compared different sketch implementations to find the fastest with acceptable error rates.
They used a Count-Min sketch to approximate heavy hitters in the network traffic efficiently.
