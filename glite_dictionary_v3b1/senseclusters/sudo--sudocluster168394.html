sudo

verb
to run as administrator
Pronunciation: / ˈsuːdoʊ / (AmE), / ˈsuːdəʊ / (BrE)
to run a command or program with superuser (administrator) privileges by using the sudo command in a Unix-like system

Examples

To sudo a command
You must sudo the command to install new software on your computer.
If you forget to sudo, the system will deny you permission to change those files.
She taught her colleague how to sudo when updating the server configuration.

Related Terms

  • antonym: restrict - to limit access or permissions
  • synonym: run as administrator - to execute a program with administrative privileges
