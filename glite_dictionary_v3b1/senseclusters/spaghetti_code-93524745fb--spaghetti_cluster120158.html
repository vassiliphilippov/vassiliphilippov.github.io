spaghetti code

noun
poorly structured, tangled computer code
9.9 5.8
Pronunciation: /spə.ˈget.iː koʊd/ (AmE), /spə.ˈget.iː kəʊd/ (BrE)
computer code that is poorly structured, tangled, and difficult to understand or maintain

Examples

Poorly structured spaghetti code
After months of quick fixes, the software evolved into spaghetti code that no one wanted to touch.
The new developer struggled to make sense of the spaghetti code left by the previous team.
To prevent spaghetti code, the team agreed to follow strict coding standards and regular code reviews.
