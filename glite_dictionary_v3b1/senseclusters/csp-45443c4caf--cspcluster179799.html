CSP

noun
website security rules
Pronunciation: / si ɛs pi / (AmE), / siː ɛs piː / (BrE)
a set of rules that control which content can be loaded on a website to improve security

Examples

Website CSP
The company implemented a strict CSP to protect its website from harmful scripts.
Developers must update the CSP whenever new resources are added to the site.
A well-configured CSP can prevent many common web attacks.

Related Terms

  • synonym: firewall - a system that blocks unauthorized access to a network
  • antonym: malware - software designed to harm or exploit computer systems
