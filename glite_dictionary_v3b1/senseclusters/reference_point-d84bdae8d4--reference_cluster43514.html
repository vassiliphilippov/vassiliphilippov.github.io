reference point

noun
standard or indicator for comparison
4.2
Pronunciation: / ˈrɛf.ɚəns pɔɪnt / (AmE), / ˈrɛf.ərəns pɔɪnt / (BrE)
a standard or indicator used for comparison or orientation, especially to judge or measure something else

Examples

A reference point for comparison
In scientific experiments, data must be compared to a reliable reference point.
The guidepost in the park serves as a useful reference point for new visitors.
The old church has long been a reference point for local directions.

Related Terms

  • synonym: baseline - a minimum or starting point used for comparisons
  • synonym: benchmark - a standard or point of reference in measuring or judging something
