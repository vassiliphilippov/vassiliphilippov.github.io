random access memory

noun
computer memory for temporary data storage
3.6 3.1
Pronunciation: /ˈræn.dəm ˈæk.ses ˈmem.ər.iː/ (AmE), /ˈræn.dəm ˈæk.ses ˈmem.ə.riː/ (BrE)
a type of computer memory that can be accessed randomly; used by programs to temporarily store and retrieve data while a computer is running
