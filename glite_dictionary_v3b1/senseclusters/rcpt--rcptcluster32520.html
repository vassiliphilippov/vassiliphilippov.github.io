rcpt

noun
SMTP recipient command
9.9
Pronunciation: /ɑːr siː piː tiː/ /; / /rɪ.ˈsɪp.ɪ.ənt/ (AmE), /ɑː siː piː tiː/ /; / /rɪ.ˈsɪp.ɪ.ənt/ (BrE)
a command in the SMTP email protocol used to specify the recipient address, commonly seen as 'RCPT TO' in technical contexts

Examples

rcpt to address
The server responded with an error after the rcpt command was sent to an invalid address.
In the email log, you can see each rcpt entry corresponding to a recipient.
Technicians often check the rcpt command to troubleshoot email delivery issues.
