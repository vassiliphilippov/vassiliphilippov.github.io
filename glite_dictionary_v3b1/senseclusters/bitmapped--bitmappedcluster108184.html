bitmapped

adjective
using a bitmap
Pronunciation: / ˈbɪtˌmæpt / (AmE), / ˈbɪtˌmæpt / (BrE)
represented, stored, or displayed as a bitmap (as a grid of pixels or an array of bits)

Examples

Bitmapped graphics
The graphics in this old video game are entirely bitmapped rather than vector-based.
A bitmapped font can appear pixelated when enlarged on high-resolution screens.
The software supports both bitmapped and scalable images for different uses.
The old computer game used bitmapped graphics that looked blocky on modern screens.
When you zoom in on a bitmapped image, you can see the individual pixels clearly.
Logos should not be bitmapped if you want them to scale smoothly to any size.

Related Terms

  • synonym: bitmap - an array of bits representing data such as an image
  • synonym: raster - an image made up of a grid of pixels
  • antonym: vector - an image or data structure based on geometric shapes, not bitmaps
