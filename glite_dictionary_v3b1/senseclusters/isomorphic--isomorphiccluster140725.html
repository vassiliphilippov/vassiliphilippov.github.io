isomorphic

adjective
runs the same code on client and server
Pronunciation: / cca6a.s59c8m54r.f6ak / (AmE), / cca6a.s59c8m54d0.f6ak / (BrE)
(in web development and computing) describing software or code that can run both on the server and in the client (browser) with little or no modification, enabling universal or server‑and‑client rendering.

Examples

Isomorphic JavaScript code
The web team rewrote the frontend so it uses isomorphic JavaScript for faster first-page loads.
By building an isomorphic app, they share validation logic between server and browser, reducing bugs.
Many frameworks now offer templates for isomorphic development to simplify universal rendering strategies.

  • synonym: universal - in web development, able to run or be rendered on both server and client
