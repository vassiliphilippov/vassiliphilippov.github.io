decoder

noun
neural-network output component
Pronunciation: / dɪˈkoʊdər / (AmE), / dɪˈkəʊdə / (BrE)
(in machine learning and AI) a component of a neural network or generative model that converts an encoded representation (for example, a latent vector) into an output sequence or reconstructed data.

Examples

Neural network decoder
The translation model's decoder turned the source-language vector into fluent target sentences.
During training, the decoder learned to reconstruct images from noisy latent vectors produced by the encoder.
Researchers compared several decoders to see which architecture generated the most natural-sounding audio.

Related Terms

  • antonym: encoder - a model component that converts input data into a compact encoded representation (often paired with a decoder).
  • synonym: generator - a model that produces data (for example, text or images) from latent inputs; plays a similar role in generative architectures.
