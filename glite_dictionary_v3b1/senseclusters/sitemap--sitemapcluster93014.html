sitemap

noun
website page list
3.6
Pronunciation: / ˈsaɪtˌmæp / (AmE), / ˈsaɪtˌmæp / (BrE)
a file, page, list, or diagram that shows all the pages and structure of a website, used to help users or search engines find information

Examples

Website sitemap file
The developer created a sitemap to help search engines find every page on the website.
Before launching the new site, they updated the sitemap to include all recent changes.
A clear sitemap makes it easier for visitors to navigate large websites.
The sitemap helped me quickly find the section I was looking for on the website.
Web developers often create a sitemap to organize all the pages before building a site.
Search engines use the sitemap to index every page of the website efficiently.

Related Terms

  • synonym: index - a list of items or pages for reference
  • synonym: navigation - the process of finding your way through a website
  • synonym: navigation menu - a list of links to main sections of a website
