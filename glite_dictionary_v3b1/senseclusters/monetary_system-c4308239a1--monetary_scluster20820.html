monetary system

noun
system for supplying and managing money
4.6
Pronunciation: / ˈmɑːnəˌtɛri ˈsɪstəm / (AmE), / ˈmɒnɪtəri ˈsɪstəm / (BrE)
the system by which money is supplied and managed in a country or region, including the standards by which its value is measured
