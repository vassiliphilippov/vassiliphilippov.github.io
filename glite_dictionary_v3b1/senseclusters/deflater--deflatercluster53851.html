deflater

noun
data compression tool
Pronunciation: / dɪˈfleɪtər / (AmE), / dɪˈfleɪtə / (BrE)
a software component or program that compresses data using the DEFLATE algorithm

Examples

File deflater
The deflater reduced the file size by more than half before sending it over the network.
Many backup programs include a deflater to save storage space.
If the deflater fails, the files will not be compressed properly.

Related Terms

  • synonym: compressor - a program or device that reduces the size of data
  • antonym: inflater - a program that decompresses data compressed with the DEFLATE algorithm
