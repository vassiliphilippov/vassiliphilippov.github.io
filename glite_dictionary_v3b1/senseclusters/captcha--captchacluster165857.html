captcha

noun
website human test
5.6
Pronunciation: / ˈkæptʃə / (AmE), / ˈkæptʃə / (BrE)
a test on a website used to determine whether the user is a human or an automated program

Examples

A captcha test
Before submitting the form, you must complete the captcha to prove you are not a robot.
Many websites use captchas to prevent automated spam registrations.
If you cannot read the captcha, you can request a new one.

Related Terms

  • antonym: bot - an automated computer program that performs tasks
  • synonym: verification - the process of confirming identity or authenticity
