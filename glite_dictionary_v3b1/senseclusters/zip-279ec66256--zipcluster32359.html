ZIP

noun
US postal code system
4.9
Pronunciation: / zɪp / (AmE), / zɪp / (BrE)
A system of letters and digits added to a postal address in the United States to help with the sorting and delivery of mail; also called ZIP code.

Examples

A ZIP code
He forgot to include his ZIP when filling out the form online.
Please write your ZIP on the envelope so the letter arrives quickly.
The package was delivered to the wrong ZIP due to a typing error.

Related Terms

  • synonym: postcode - a code of letters and digits used in postal addresses, especially in the UK
