storable

adjective
able to be saved
3.4
Pronunciation: / ˈstɔːrəbəl / (AmE), / ˈstɔːrəbəl / (BrE)
able to be saved, recorded, or kept for future use, especially in electronic memory or on a storage device

Examples

Storable files

Only storable files can be backed up to the cloud automatically.
Make sure the data is in a storable format before transferring it to another device.
