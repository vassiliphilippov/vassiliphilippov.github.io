specializer

noun
program that generates specialized code
9.0
Pronunciation: /ˈspeʃ.ə.laɪ.zər/ /; / /ˈspeʃ.ɪ.laɪ.zər/ (AmE), /ˈspeʃ.ə.laɪ.zə/ /; / /ˈspeʃ.ɪ.laɪ.zə/ (BrE)
(in computing) a program that automatically produces a specialized or optimized version of another program, typically by using information about a particular architecture or input.

Examples

Software specializer
The compiler team developed a specialiser to produce optimized binaries for embedded processors.
Using a specialiser, researchers generated input-specific versions of the interpreter to speed up tests.
Several machine-learning libraries include specialisers that tailor models to particular hardware during deployment.

Related Terms

  • synonym: partial evaluator - a program transformation tool that precomputes parts of a program given known inputs, producing a specialized program.
  • synonym: specializer - US spelling of 'specialiser'; a program that produces optimized, input- or architecture-specific code.
