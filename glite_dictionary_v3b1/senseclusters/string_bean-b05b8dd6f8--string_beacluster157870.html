string bean

noun
a green bean with fibrous strings on the pod
2.3 2.3
Pronunciation: /strɪŋ biːn/ (AmE), /strɪŋ biːn/ (BrE)
a variety of green bean with tough fibrous strings running along its pods, which must be removed before eating

Examples

Fresh string bean salad
Many modern varieties of green beans are bred without the tough strings found on older string beans.
She snapped the ends off each string bean before putting them in the pot.
The recipe calls for a pound of fresh string beans, trimmed and washed.
