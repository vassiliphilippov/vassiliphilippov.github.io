data multiplexer

noun
device sharing data channels
5.2
Pronunciation: / ˈdeɪtə ˌmʌltɪˌplɛksər / (AmE), / ˈdeɪtə ˌmʌltɪˌplɛksə / (BrE)
A device that allows two or more data sources to share a single transmission channel or medium.

Examples

A data multiplexer device
Each data multiplexer efficiently manages the flow of information from several sensors into a central controller.
The network administrator installed a data multiplexer to connect multiple computers to the same communication line.
Upgrading to a new data multiplexer allowed the office to maximize the use of their existing bandwidth.

Related Terms

  • synonym: multiplexer - a device that combines several input signals for transmission over a single line
