multipath

noun
redundant data paths
Pronunciation: / ˈmʌltipaθ / (AmE), / ˈmʌltipaːθ / (BrE)
a technique in storage or networking that uses multiple physical data paths between devices for redundancy and better performance

Examples

Multipath storage setup
The IT team implemented multipath to ensure continuous access to the storage system.
Using multipath can help prevent data loss if one connection fails.
Many enterprise servers rely on multipath for both speed and reliability.

Related Terms

  • synonym: redundancy - the inclusion of extra components to increase reliability
  • antonym: single-path - using only one data path between devices
