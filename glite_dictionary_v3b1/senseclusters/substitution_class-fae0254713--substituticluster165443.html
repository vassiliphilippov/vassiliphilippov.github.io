substitution class

noun
linguistic group of interchangeable words
4.2
Pronunciation: / ˌsʌbstɪˈtuːʃən klæs / (AmE), / ˌsʌbstɪˈtjuːʃən klɑːs / (BrE)
(Linguistics) The group of words or elements that can occupy the same position in a grammatical structure and be substituted for each other without disrupting syntactic acceptability.

Examples

The substitution class group
In the sentence, the substitution class for the subject includes words like 'dog', 'child', and 'teacher'.
Linguists use the concept of substitution class to analyze sentence structures across different languages.
The pronouns 'he', 'she', and 'it' all belong to the same substitution class in this grammatical slot.

Related Terms

  • synonym: paradigm - a set of forms which a word or element can take based on grammatical function
