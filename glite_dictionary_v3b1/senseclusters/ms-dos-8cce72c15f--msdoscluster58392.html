MS-DOS

noun
early personal computer operating system
Pronunciation: / ɛm ɛs dɑs / (AmE), / ɛm ɛs dɒs / (BrE)
an early computer operating system for IBM-compatible personal computers, originally developed by Microsoft.

Examples

MS-DOS operating system
He learned to write batch scripts for MS-DOS in the 1980s.
Many early computers relied on MS-DOS to run their basic programs and manage files.
Some modern computers can still emulate MS-DOS for running classic software.
