RTI

noun
live data feed
Pronunciation: / ɑr ti aɪ / (AmE), / ɑː tiː aɪ / (BrE)
current, live data about a situation, such as transit or system status, provided in real time

Examples

RTI updates
The transit app displays RTI so passengers can see train arrivals and delays instantly.
With RTI, drivers receive live traffic updates to help avoid congestion.
Airports now use RTI to inform travelers about gate changes and flight statuses.

Related Terms

  • antonym: delayed information - data that is not provided instantly but after some time
  • synonym: live update - information that is updated instantly as events occur
  • synonym: real-time information - data provided instantly as events happen
