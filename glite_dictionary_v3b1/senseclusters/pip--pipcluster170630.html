pip

noun
Python package manager
8.0
Pronunciation: / pɪp / (AmE), / pɪp / (BrE)
a tool and package manager for the Python programming language, used to install, upgrade, and manage Python libraries and packages (often from the Python Package Index, PyPI).

Examples

Install with pip
Developers commonly install Python packages with pip after creating a virtual environment.
To reproduce the project, run the requirements file using pip with your virtualenv activated.
She upgraded the library by running pip install --upgrade in the project's root directory.

Related Terms

  • synonym: package installer - a program used to install software libraries or packages into a project environment
  • synonym: package manager - software that automates installing, upgrading, configuring, and removing software packages
