batteries included

adjective
provided with all necessary components
Pronunciation: / ˈbæt̬.ɚiz ɪnˈkluː.dɪd / (AmE), / ˈbæt.ər.iz ɪnˈkluː.dɪd / (BrE)
supplied or provided with all necessary components or features, especially so that it is ready to use immediately

Examples

Software batteries included
Many modern software tools come 'with batteries included,' meaning they have built-in functionality.
The camera arrived batteries included, saving us an extra trip to the store.
This toy is batteries included, so you can play with it right away.
