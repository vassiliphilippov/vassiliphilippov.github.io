text box

noun
rectangular text area
Pronunciation: / tɛkst bɑks /; / ˈtɛkst bɑks / (AmE), / tɛkst bɒks /; / ˈtɛkst bɒks / (BrE)
an area on a screen or in a document where you can enter or display text

Examples

A text box on screen
She added a text box to the poster to show the event details.
You can move the text box anywhere on the slide to improve the layout.
The designer resized the text box to fit all the important information.
Please enter your email address in the text box before continuing.
The website has a text box for users to leave comments on each article.
If the text box is empty, the form will not submit successfully.

Related Terms

  • antonym: button - an area on a screen that you click to perform an action
  • synonym: caption - a short piece of text describing an image or graphic
  • synonym: input field - a space on a form or screen for entering data
  • synonym: textbox - an area in a document or interface for displaying or entering text
