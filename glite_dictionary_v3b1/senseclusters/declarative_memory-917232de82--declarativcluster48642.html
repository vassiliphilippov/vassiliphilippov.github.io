declarative memory

noun
memory for facts and events
4.9
Pronunciation: / dɪˈklɛr.ə.tɪv ˈmɛm.ə.ri / (AmE), / dɪˈklær.ə.tɪv ˈmɛm.əri / (BrE)
A type of long-term memory involving information that can be consciously recalled, such as facts and events.

Examples

Declarative memory recall
Damage to the hippocampus can severely impair declarative memory while leaving other types of memory intact.
Her decline in declarative memory made it difficult for her to remember recent conversations.
Students rely on declarative memory to recall facts needed on exams.

Related Terms

  • antonym: procedural memory - a type of memory for skills and procedures, not consciously recalled
