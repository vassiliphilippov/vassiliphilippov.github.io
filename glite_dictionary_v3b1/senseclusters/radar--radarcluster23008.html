radar

noun
system for detecting objects using radio waves
4.1 20/m
Pronunciation: / Ààre…™Àåd…ër / (AmE), / Ààre…™Àåd…ëÀê / (BrE)
a system for detecting the presence, direction, distance, and speed of objects by sending out pulses of radio waves that are reflected off the object back to the source

Examples

A radar system
Heavy rain can sometimes disrupt the accuracy of radar systems.
Military ships rely on radar for early detection of enemy vessels.
The airport uses radar to monitor all incoming and outgoing flights.

Synonyms

  • radio detection and ranging
    4.2 0.01/m
    radio detection and ranging is the full formal name used in technical writing, while radar is the common short name in speech.
  • radiolocation
    8.4 0.00/m
    radiolocation uses radio waves to find object locations broadly, while radar is a specific pulse system for detecting distance, direction, and speed.

Related Terms

  • synonym: sonar - a system that uses sound waves to detect and locate objects underwater
