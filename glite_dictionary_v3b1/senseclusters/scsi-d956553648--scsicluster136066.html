SCSI

noun
computer interface for connecting peripherals
7.9 0.51/m
Pronunciation: / ËˆskÊŒzi / (AmE), / ËˆskÊŒzi / (BrE)
a standard interface for connecting computers and peripheral devices, such as hard drives and scanners, especially used in older computers

Examples

SCSI interface
Many scanners in the 1990s required a SCSI port to connect to a computer.
The old server uses a SCSI connection for its external hard drives.
Upgrading to a new system meant replacing all the SCSI cables and drives.

Synonyms

  • small computer system interface
    4.2 0.01/m
    small computer system interface is the full formal name used in manuals and documents, while SCSI is the short acronym common in technical writing.
