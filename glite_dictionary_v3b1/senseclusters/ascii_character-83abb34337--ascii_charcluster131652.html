ASCII character

noun
character encoded in ASCII standard
Pronunciation: / ˈæski ˈkærəktɚ / (AmE), / ˈaski ˈkarɪktə / (BrE)
any of the characters represented by the American Standard Code for Information Interchange (ASCII), which assigns each character a unique binary number.

Examples

ASCII character code
Each ASCII character is stored as a single byte in older computer systems.
The text file only contains basic ASCII characters, so it's compatible with most applications.
To represent a newline, the ASCII character with the value 10 is used.

Related Terms

  • antonym: Unicode character - a character represented in the Unicode character encoding standard
