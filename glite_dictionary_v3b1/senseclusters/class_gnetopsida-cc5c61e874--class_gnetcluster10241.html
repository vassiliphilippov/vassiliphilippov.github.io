class Gnetopsida

proper noun
a class of gymnospermous plants
Pronunciation: / klæs dʒniːˈtɒpsɪdə / (AmE), / klɑːs dʒniːˈtɒpsɪdə / (BrE)
A class of gymnospermous plants thought to be a link between conifers and angiosperms, sometimes classified as a class (Gnetopsida) or subdivision (Gnetophytina or Gnetophyta).

Examples

Class Gnetopsida plants
Scientists debate whether class Gnetopsida is more closely related to conifers or angiosperms.
Species within the class Gnetopsida display a number of unusual reproductive features.
Welwitschia is a remarkable member of class Gnetopsida found in the deserts of Africa.
