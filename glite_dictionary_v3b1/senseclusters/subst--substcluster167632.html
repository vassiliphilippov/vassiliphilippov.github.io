subst

proper noun
MS-DOS drive command
9.9
Pronunciation: / sʌbst / (AmE), / sʌbst / (BrE)
the name of a computer command in MS-DOS and Windows that assigns a local folder to a drive letter

Examples

Subst command Windows
He used the subst command to make the reports folder appear as drive R: on his computer.
To simplify file access, she ran subst so her project directory was mapped to drive P:.
After restarting, the subst drive was no longer available and had to be recreated.

Related Terms

  • synonym: map - to assign a drive letter to a folder or network location
  • antonym: unmap - to remove an assigned drive letter from a folder or network location
