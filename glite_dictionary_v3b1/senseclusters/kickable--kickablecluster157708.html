kickable

adjective
able to be expelled
Pronunciation: / ˈkɪkəbəl / (AmE), / ˈkɪkəbəl / (BrE)
able to be removed or expelled from a group, server, or service by an administrator or moderator

Examples

Kickable user account
If a user breaks the rules repeatedly, their account may be marked as kickable by moderators.
The chat system lists all kickable members so admins can manage the group easily.
Only kickable players can be removed from the online game during a match.

Related Terms

  • synonym: banishable - able to be officially sent away or excluded
  • antonym: permanent - lasting or intended to last indefinitely
  • synonym: removable - able to be taken away or dismissed
