slurp

verb
ingest large amounts of data (computing)
Pronunciation: / sl5crp / (AmE), / sl5cd0p / (BrE)
(in computing) to read, transfer, or ingest a large amount of data quickly and often automatically, typically from a file, database, or remote source, sometimes in a single operation.

Examples

To slurp data
A script can slurp thousands of records from the database in minutes for analysis.
They slurped the compressed log files into memory and began searching for errors.
The backup tool is slurping remote API responses each night to build a local cache.

Related Terms

  • synonym: import - to bring data into a program or database from another source
  • synonym: ingest - to take data into a system or process for storage or processing
