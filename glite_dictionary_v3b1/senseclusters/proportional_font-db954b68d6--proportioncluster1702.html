proportional font

noun
typeface with variable character widths
4.1 3.8
Pronunciation: /prə.ˈpɔːr.ʃə.nəl fɑːnt/ /; / /prə.ˈpɔːr.ʃə.nəl fɔːnt/ (AmE), /prə.ˈpɔː.ʃə.nəl fɒnt/ (BrE)
a typeface in which different characters have different widths, as opposed to a monospaced font

Examples

Proportional font style
Many books and magazines use proportional fonts for easier readability.
The document looked more polished after I switched to a proportional font.
Unlike a monospaced font, a proportional font allocates different width to each character.
