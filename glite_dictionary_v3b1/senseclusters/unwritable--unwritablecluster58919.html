unwritable

adjective
cannot be written
Pronunciation: / ʌnˈraɪtəbəl / (AmE), / ʌnˈraɪtəbəl / (BrE)
not able to be written, recorded, or saved, especially due to restrictions or limitations

Examples

Unwritable file
The document became unwritable after the administrator changed the permissions.
If the disk is unwritable, you will not be able to save any changes.
Some sections of the report are intentionally set as unwritable to prevent accidental edits.

Related Terms

  • synonym: read-only - able to be read but not written or modified
  • antonym: writable - able to be written or modified
