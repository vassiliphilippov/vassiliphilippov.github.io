Scrabble

noun
word board game using letter tiles
3.7
Pronunciation: / ˈskræb.əl / (AmE), / ˈskræb.əl / (BrE)
A board game in which players use lettered tiles to create words in a crossword-style grid for points determined by letter values.

Examples

Play Scrabble game
He's very good at Scrabble and always manages to use the highest scoring words.
She brought out her favorite Scrabble set for the family game night.
We spent the evening playing Scrabble by the fireplace.
