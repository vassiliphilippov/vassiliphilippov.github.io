mountable

adjective
can be connected (computing)
4.9
Pronunciation: /ˈmaʊn.tə.bəl/ (AmE), /ˈmaʊn.tə.bəl/ (BrE)
able to be connected to a computer so that its file system or contents can be accessed.

Examples

Mountable USB drive
The USB drive was not mountable on the computer after the system update.
If the disk is mountable, you can access your files through the file manager.
Some network shares are only mountable with the correct permissions.

Related Terms

  • synonym: accessible - able to be reached or entered
  • antonym: unmountable - not able to be connected or accessed by a computer
