default option

noun
option chosen automatically unless changed
Pronunciation: / dɪˈfɔlt ˈɑpʃən / (AmE), / dɪˈfɔːlt ˈɒpʃən / (BrE)
an option that is automatically selected if no alternative is specified

Examples

The default option
If you don't choose a payment method, the default option will be used.
She accepted the default option because she was unsure which alternative to select.
The software's default option is to save files locally unless you specify otherwise.

Related Terms

  • synonym: default setting - pre-set value or configuration used if no alternative is given
