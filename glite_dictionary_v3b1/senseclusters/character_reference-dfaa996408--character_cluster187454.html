character reference

noun

Pronunciation: /ˈkær.ək.tər ˈref.ər.əns/ /; / /ˈkær.ɪk.tər ˈref.ər.əns/ /; / /ˈkær.ək.tər ˈref.rəns/ (AmE), /ˈkær.ək.tə ˈref.ə.rəns/ /; / /ˈkær.ɪk.tə ˈref.ə.rəns/ /; / /ˈkær.ək.tə ˈref.rəns/ (BrE)
In computing (especially HTML and other markup languages), a sequence of characters used to represent a particular character or code point in text, for example named entities (&nbsp;, &eacute;) or numeric references (&#233;).

  • synonym: escape sequence - a sequence of characters that represents another character in text or code when direct use is not possible
