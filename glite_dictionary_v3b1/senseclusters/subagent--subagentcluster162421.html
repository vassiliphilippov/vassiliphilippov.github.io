subagent

noun
secondary software agent
Pronunciation: / ˈsʌbˌeɪdʒənt / (AmE), / ˈsʌbˌeɪdʒənt / (BrE)
a software program or component that operates under a main agent to perform specific tasks or monitor events

Software subagent

Each subagent is responsible for monitoring a specific part of the network.
The main agent delegates routine checks to a subagent to improve efficiency.
