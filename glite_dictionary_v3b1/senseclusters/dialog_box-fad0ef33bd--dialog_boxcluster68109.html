(computing) a small window on a computer screen that appears to prompt the user for input or provide information, usually requiring a response such as 'OK' or 'Cancel'.

Examples

A dialog box window

The software update process starts with a dialog box prompting for administrator approval.
Whenever you save your work, a dialog box appears with several options.
