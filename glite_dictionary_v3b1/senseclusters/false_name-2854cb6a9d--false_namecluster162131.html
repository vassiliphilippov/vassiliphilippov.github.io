false name

noun
name used instead of the real one
3.1
Pronunciation: / fɔls neɪm / (AmE), / fɔːls neɪm / (BrE)
a name that someone uses which is not their real name, especially to conceal their identity or avoid detection

Examples

Use a false name
Criminals often create a false name to conceal their identity online.
She checked into the hotel under a false name to avoid being recognized.
The witness agreed to testify but requested to use a false name for safety reasons.

Related Terms

  • antonym: real name - a person's actual legal or birth name
