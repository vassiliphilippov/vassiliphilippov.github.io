multithread

verb
to use multiple threads
Pronunciation: / ˌmʌltiˈθrɛd / (AmE), / ˌmʌltiˈθrɛd / (BrE)
to design or run software so that it uses multiple threads of execution at the same time

Examples

To multithread code
The team decided to multithread the application to handle more users at once.
Learning how to multithread code can greatly improve a program's performance.
If you multithread the process, it will finish much faster on modern computers.

Related Terms

  • synonym: parallelize - to arrange for tasks to be performed at the same time
  • antonym: serialize - to arrange for tasks to be performed one after another
