serialize

verb
convert data for storage/transmission
6.3
Pronunciation: /ˈsɪ.rɪ.ə.laɪz/ /; / /ˈsɪr.i.ə.laɪz/ (AmE), /ˈsɪə.rɪ.laɪz/ (BrE)
to convert a data structure or object into a standardized, portable serialized form (for example a byte stream or text format) so it can be stored or sent and later reconstructed.

Examples

To serialize data
The API can serialize the user profile into JSON for network transmission.
She serialized the session object to a binary file before shutting down the server.
When debugging, they avoided serializing live sockets because the operation caused errors.
The program can serialise the user profile into a JSON string for storage.
After saving, the service serialised the session data so it could be sent to another server.
They're serialising complex objects into a binary stream for faster network transmission today.

Related Terms

  • antonym: deserialize - to reconstruct a data structure or object from its serialized form (the reverse of serialization).
  • synonym: encode - to convert information into a particular form or code, often for storage or transmission.
  • synonym: serialize - American spelling of the same computing process: convert data into a linear format.
