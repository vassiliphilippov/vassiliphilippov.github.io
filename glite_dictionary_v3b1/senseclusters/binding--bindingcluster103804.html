binding

noun
linking data and UI for automatic updates
Pronunciation: / ˈbaɪndɪŋ / (AmE), / ˈbaɪndɪŋ / (BrE)
(in computing) a connection or mapping that links a data source or model to a user-interface element so that changes in one are automatically reflected in the other.

Examples

Data binding
The framework supports binding of UI fields to data models for real-time updates.
Using two-way binding, changes in the form immediately update the underlying database model.
He added a simple binding so the slider value always reflects the current temperature reading.

Related Terms

  • synonym: data binding - the technique of connecting UI elements to data sources so that changes propagate automatically
