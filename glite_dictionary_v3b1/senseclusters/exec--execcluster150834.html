exec

noun
executable file
Pronunciation: / ɪɡˈzɛk / (AmE), / ɪɡˈzɛk / (BrE)
a file or program that can be run directly by a computer, also called an executable

Examples

Run the exec file
After downloading the software, run the exec to start the installation process.
The folder contains several files, but only the exec will launch the program.
Make sure the exec has the correct permissions before trying to open it on your computer.

Related Terms

  • synonym: executable - a file that can be run by a computer
  • degree lower: script - a file containing instructions for a computer to execute
