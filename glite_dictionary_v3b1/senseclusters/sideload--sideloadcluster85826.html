sideload

noun
unofficial app install
8.4
Pronunciation: /ˈsaɪd.loʊd/ (AmE), /ˈsaɪd.ləʊd/ (BrE)
an application or file installed on a device from an unofficial source rather than through an official app store

Examples

An app sideload
He installed a sideload on his tablet to access features not available in the app store.
Some users prefer a sideload to get early access to new software updates.
Be careful when using a sideload, as unofficial apps may pose security risks.

  • synonym: install - to put software or an application onto a device
  • antonym: official app - an application obtained from an official app store
