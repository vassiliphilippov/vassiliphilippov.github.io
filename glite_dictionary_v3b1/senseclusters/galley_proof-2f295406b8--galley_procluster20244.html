galley proof

noun
trial print of text before final formatting
Pronunciation: / ˈɡæli pruːf / (AmE), / ˈɡæli pruːf / (BrE)
a trial printed copy of text taken before the type is set in its final page format, used for checking errors before final printing

Examples

A galley proof copy
After reviewing the galley proof, the author requested several changes to the manuscript.
Each galley proof must be checked carefully for typographical errors before page layout begins.
The editor marked corrections on the galley proof before the book went to print.
