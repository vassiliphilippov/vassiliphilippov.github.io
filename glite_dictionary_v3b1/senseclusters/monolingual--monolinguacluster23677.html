monolingual

adjective
using or knowing only one language
3.7
Pronunciation: / ˌmoʊnəˈlɪŋɡwəl / (AmE), / ˌmɒnəˈlɪŋɡwəl / (BrE)
using or able to use only one language; relating to or expressed in only one language.

Examples

Monolingual dictionary
Although he lives in a multicultural city, he remains strictly monolingual.
Many monolingual children start learning a second language at school.
This is a monolingual edition, presented only in English.

Related Terms

  • antonym: multilingual - knowing or using several languages
