stage

noun
pre-production testing environment (software)
6.2
Pronunciation: / steɪdʒ / (AmE), / steɪdʒ / (BrE)
(in computing) a pre‑production testing environment that closely mirrors the live (production) system, used to run final checks before deployment.

Examples

Stage server testing
The patch was tested on the stage before going live.
Our team deployed the update to the stage server to run final acceptance tests today.
Users reported no errors after we verified the release on stage and then promoted it to production.

Related Terms

  • antonym: production - the live environment used by end users; the system where the software is deployed for real use
  • synonym: staging environment - a server or environment that mimics production for testing releases before deployment
