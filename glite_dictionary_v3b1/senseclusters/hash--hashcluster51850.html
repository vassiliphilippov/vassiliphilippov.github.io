hash

noun
unique data code
4.9
Pronunciation: / hæʃ / (AmE), / hæʃ / (BrE)
a unique code or value generated from data, often used for security, identification, or verification in computing

Examples

File hash
The software generates a hash for each file to detect any changes.
You should compare the hash values to confirm the download is authentic.
A strong hash function is essential for secure password storage.
