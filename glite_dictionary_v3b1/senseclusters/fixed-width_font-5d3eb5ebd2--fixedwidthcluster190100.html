fixed-width font

noun
typeface with equal-width characters
6.6
Pronunciation: / ˌfɪkstˈwɪdθ fɑnt / (AmE), / ˌfɪkstˈwɪdθ fɒnt / (BrE)
A style of typeface in which each character occupies the same amount of horizontal space.

Examples

A fixed-width font
For certain design tasks, a fixed-width font can offer clarity and uniformity.
The typewriter produced neat lines thanks to its fixed-width font.
When programming, many developers prefer using a fixed-width font to help align their code.

Related Terms

  • synonym: monospaced font - typeface in which all characters have the same width
