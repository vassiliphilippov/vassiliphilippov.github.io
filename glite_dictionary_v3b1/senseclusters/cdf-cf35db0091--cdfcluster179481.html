CDF

noun
statistical probability function
8.4
Pronunciation: /siː diː ef/ (AmE), /siː diː ef/ (BrE)
an abbreviation for cumulative distribution function, a function in statistics that shows the probability a variable is less than or equal to a value

Examples

Statistics CDF curve
The CDF is useful for calculating the probability of a random variable falling within a certain range.
In statistics class, we learned how to plot a CDF for different types of data.
You can use the CDF to determine the likelihood that a measurement does not exceed a specific value.

Related Terms

  • synonym: cumulative distribution function - a function giving the probability that a variable is less than or equal to a value
  • antonym: probability density function - a function showing the likelihood of a variable taking a specific value
