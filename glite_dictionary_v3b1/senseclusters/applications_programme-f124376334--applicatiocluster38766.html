applications programme

noun
software for user tasks on a computer
4.3
Pronunciation: / ˌæplɪˈkeɪʃənz ˈproʊɡræm / (AmE), / ˌæplɪˈkeɪʃənz ˈprəʊɡræm / (BrE)
a computer program designed to help the user perform specific tasks such as word processing or spreadsheets

Examples

An applications programme for work
Most offices rely on at least one applications programme to manage their documents.
She installed a new applications programme to help with graphic design projects.
The company develops custom applications programmes for clients in various industries.

Related Terms

  • antonym: system software - computer programs that control the functioning of the hardware and basic system operations
