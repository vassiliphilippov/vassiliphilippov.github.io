lunar excursion module

noun
spacecraft for lunar landings and return
Pronunciation: / ˈluːnɚ ɪkˈskɝːʒən ˈmɑːdʒuːl / (AmE), / ˈluːnə ɪkˈskɜːʃn ˈmɒdjuːl / (BrE)
a spacecraft designed to transport astronauts from a command module to the surface of the moon and return them during a lunar mission

Examples

The lunar excursion module spacecraft
Each Apollo mission relied on the lunar excursion module to land astronauts on the moon safely.
Restoring the lunar excursion module was crucial for the astronauts to return to the command module.
The lunar excursion module successfully separated from the command module and descended to the moon's surface.

  • synonym: lunar module - the vehicle used for landing on the moon
