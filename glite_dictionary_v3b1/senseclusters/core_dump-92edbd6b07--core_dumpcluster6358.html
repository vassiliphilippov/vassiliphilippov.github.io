core dump

noun
computer memory record for debugging
7.7
Pronunciation: /kɔːr dʌmp/ (AmE), /kɔː dʌmp/ (BrE)
(computing) a file containing a record of the memory contents of a computer's central processing unit at a specific time, usually created after a program crash for debugging purposes.

Examples

A core dump file
Developers examined the core dump to diagnose the cause of the crash.
If the server crashes, a core dump can help identify the faulty process.
The program generated a core dump after encountering a fatal error.

Related Terms

  • synonym: crash dump - record of computer memory created after a system crash
