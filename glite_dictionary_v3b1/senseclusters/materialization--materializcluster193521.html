materialization

noun
storing computed data for faster access (computing)

(in computing and data management) the process of creating and storing a concrete, stored copy of a computed or derived result (for example a materialized view or cached query result) so it can be retrieved more quickly.

Examples

Database materialization
The data warehouse uses materialization to store query results as a materialized view for faster reporting.
During maintenance they canceled the nightly materialization jobs after the schema change caused failures.
Developers debated whether materialization of the computed table was necessary given the small dataset size.
The database team improved performance by materialisation of the query results into a cached table.
To speed reporting, they scheduled nightly materialisation of the view so analysts had instant access.
Cloud data pipelines often include materialisation steps that store computed results for reuse.

Related Terms

  • synonym: cache - a stored copy of data kept for faster access; similar in purpose to materialisation
  • synonym: caching - the practice of storing computed or frequently accessed data to speed up future requests
  • synonym: materialized view - a stored, precomputed result of a database query that can be queried directly for faster access
