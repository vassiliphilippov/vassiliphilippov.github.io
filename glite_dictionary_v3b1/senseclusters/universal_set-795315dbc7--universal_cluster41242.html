universal set

noun
set containing all elements considered
1.0 5.6
Pronunciation: / juˌnɪvɚsəl sɛt / (AmE), / juˌnɪvəsəl sɛt / (BrE)
(Mathematics) The set that contains all elements or objects under consideration in a particular problem, of which all other sets are subsets.

Examples

The universal set
If A is a subset of the universal set, then every element of A is also in it.
In this discussion, the universal set includes all natural numbers.
When drawing Venn diagrams, the rectangle often represents the universal set.
