application

noun
computer program for specific tasks
Pronunciation: / ˌæplɪˈkeɪʃən / (AmE), / ˌæplɪˈkeɪʃən / (BrE)
a computer program designed to perform a specific task for the user

Examples

Mobile application
I downloaded a new application to help me organize my daily tasks.
Many smartphones come with several pre-installed applications for email and web browsing.
This graphics application allows users to edit photos with advanced features.

Related Terms

  • synonym: app - a shortened form of 'application', commonly used for mobile software
  • antonym: operating system - the software that supports a computer's basic functions
  • synonym: software - programs and other operating information used by a computer
