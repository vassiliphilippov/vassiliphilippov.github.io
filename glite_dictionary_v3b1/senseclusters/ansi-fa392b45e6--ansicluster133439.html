ANSI

noun
Windows text encoding
7.4
Pronunciation: / ˈænsi / (AmE), / ˈænsi / (BrE)
a character encoding standard, often referring to the Windows-1252 code page, used to represent text on Microsoft Windows systems

Examples

ANSI text format
The document was saved in ANSI format, which caused some special characters to display incorrectly.
Many older Windows applications use ANSI encoding by default for text files.
To support more languages, it is better to convert files from ANSI to UTF-8.

Related Terms

  • synonym: ASCII - a basic character encoding standard for English text
  • antonym: UTF-8 - a widely used Unicode character encoding
