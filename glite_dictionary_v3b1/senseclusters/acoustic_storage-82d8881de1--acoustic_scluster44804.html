acoustic storage

noun
computer memory using sound waves
3.1
Pronunciation: / əˈkuːstɪk ˈstɔrɪdʒ / (AmE), / əˈkuːstɪk ˈstɔːrɪdʒ / (BrE)
A type of computer memory that uses acoustic delay lines to store data as sound waves, mainly used in early computers.

Examples

Acoustic storage memory
Early computers sometimes used acoustic storage to hold information before the invention of magnetic memory.
Modern technology has replaced acoustic storage with faster and more reliable memory systems.
The principle behind acoustic storage relies on converting electrical signals into sound waves for temporary data retention.
