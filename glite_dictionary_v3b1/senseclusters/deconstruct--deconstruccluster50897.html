deconstruct

verb
unpack a data structure (computing)
7.1
Pronunciation: /diː.kən.ˈstrʌkt/ /; / /də.kən.ˈstrʌkt/ (AmE), /diː.kən.ˈstrʌkt/ /; / /diː.kən.ˈstrʊkt/ (BrE)
(in computing) to unpack a composite data structure into its constituent parts, typically using a language feature or method that assigns components to variables.

Examples

To deconstruct a structure
In C# 7 you can deconstruct a Point into its x and y coordinate variables.
The library function deconstructs the JSON object into named fields for easier processing.
When debugging, she deconstructed the tuple to inspect each value separately.

Related Terms

  • synonym: destructure - to break a data structure into variables using language syntax (often called destructuring in some languages).
  • synonym: unpack - to separate a composite data structure into individual parts (computing).
