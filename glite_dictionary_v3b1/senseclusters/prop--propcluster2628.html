prop

noun
software property / component attribute
7.7
Pronunciation: / prɑp / (AmE), / prɒp / (BrE)
(in computing, especially in user-interface programming) a short form of 'property': a named value or attribute supplied to a component or object that controls its behavior or appearance.

Examples

React prop
In our React app we pass a prop called 'color' to change the button's appearance.
Each widget reads its prop during rendering to decide which data to display.
You can set default values for a component's prop to avoid errors when none are provided.

Related Terms

  • synonym: attribute - a characteristic or parameter of a software element that affects its behavior or display
  • synonym: parameter - a value passed into a function or component that influences its operation
  • synonym: property - a named value or attribute belonging to an object or component in programming
