unicameral script

noun
script with only one letter case
Pronunciation: / ˌjuːnɪˈkæmərəl skrɪpt / (AmE), / ˌjuːnɪˈkæmərəl skrɪpt / (BrE)
a writing system in which there is only one form of each letter, with no distinction between uppercase and lowercase

Examples

A unicameral script system
Many ancient writing systems, including Phoenician, used a unicameral script.
The Greek alphabet originally had a unicameral script before the development of uppercase and lowercase letters.
Unlike English, which uses both cases, Cherokee is written using a unicameral script.

Related Terms

  • antonym: bicameral script - a script with distinct uppercase and lowercase forms
