truncate

verb
remove all rows from a database table
7.0 0.08/m

(in computing, especially databases) to remove all rows from a table or similar data structure, typically using a TRUNCATE command so the table's contents are quickly reset.

Examples

To truncate a table
The nightly maintenance script will truncate the staging table to reset data before loading fresh records.
Before backups, the DBA truncated the temporary tables to free space and improve restore speed.
Use caution when you truncate a production table; many systems cannot roll back the operation easily.
