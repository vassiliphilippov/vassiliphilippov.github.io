provision

verb
set up and configure computing resources
6.5
Pronunciation: / prəˈvɪʒən / (AmE), / prəˈvɪʒ.ən / (BrE)
(in information technology) to set up, configure, and allocate computing resources (such as servers, virtual machines, storage, or network services) so they are ready for use.

Examples

To provision a server
The IT team will provision a new virtual machine for the project.
After approval, the cloud admin provisioned storage and networking for the data pipeline.
Automated provisioning of containers reduced deployment time from hours to minutes for developers.

Related Terms

  • synonym: allocate - to assign computing resources (such as CPU, memory, or storage) for a specific purpose
  • synonym: configure - to arrange or set up software or hardware settings so they work as required
  • synonym: deploy - to release or make software or services available for use, often after setup
