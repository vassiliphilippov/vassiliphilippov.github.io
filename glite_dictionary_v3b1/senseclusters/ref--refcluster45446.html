ref

noun
programming keyword for pass-by-reference
Pronunciation: / rɛf / (AmE), / rɛf / (BrE)
(in programming) a keyword or annotation (e.g., in C#) indicating that a function parameter is passed by reference so the function can modify the caller's variable.

Examples

Ref keyword
In C#, declare a parameter with ref when you want the method to modify it directly.
She changed the tutorial to explain how the ref keyword differs from 'out' parameters in C#.
During code review, he warned that passing large structs by ref can improve performance but be error-prone.

Related Terms

  • synonym: pass-by-reference - a parameter-passing mechanism where a function receives a reference to the caller's variable, allowing modification.
  • synonym: reference - the concept of referring to a variable's storage rather than copying its value; basis for 'ref'.
