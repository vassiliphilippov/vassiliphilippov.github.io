instrumentation

noun
adding monitoring/diagnostic code to software
5.8
Pronunciation: / cc6an.str59.m59nc8te6a.8359n / (AmE), / cc6an.str59.menc8te6a.8359n / (BrE)
the practice of inserting code, libraries, or tools into software systems to collect metrics, traces, logs, and other diagnostic data for performance, reliability, or debugging purposes.

Examples

Software instrumentation
We added instrumentation to the service to record request latency and error counts in production.
The team's monitoring depends on good instrumentation, including traces, metrics, and structured logs.
During testing they inserted detailed instrumentation into the code to diagnose intermittent failures.

Related Terms

  • synonym: monitoring - the automatic collection and analysis of system metrics and events to observe system health and performance
  • synonym: telemetry - the automated transmission and collection of measurement data from systems for analysis
