correlation matrix

noun
table of pairwise correlation coefficients
Pronunciation: / ˌkɔrəˈleɪʃən ˈmeɪtrɪks / (AmE), / ˌkɒrəˈleɪʃən ˈmeɪtrɪks / (BrE)
A table showing the correlation coefficients between all possible pairs of variables in a dataset.

Examples

The correlation matrix table
Before building the model, she checked the correlation matrix for highly correlated variables.
Researchers use a correlation matrix to examine the relationships between multiple variables at once.
The correlation matrix revealed that most factors were only weakly related to each other.
