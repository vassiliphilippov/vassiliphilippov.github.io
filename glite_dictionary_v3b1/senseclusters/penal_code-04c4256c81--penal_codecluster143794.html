penal code

noun
set of laws on crimes and punishments
4.2
Pronunciation: / ˈpiː.nəl koʊd / (AmE), / ˈpiː.nəl kəʊd / (BrE)
the body of laws that defines crimes and sets out their punishments in a particular jurisdiction

Examples

The penal code
Lawyers must be well-versed in the penal code to effectively defend their clients.
The penal code outlines the punishments for theft, assault, and other offenses.
The country recently revised its penal code to address new types of cybercrime.
