triphase

noun
three-phase power system
Pronunciation: / ˈtraɪˌfeɪz / (AmE), / ˈtraɪˌfeɪz / (BrE)
an electrical power system that uses three alternating currents of the same frequency, each offset by one third of a cycle

Examples

A triphase supply
The factory upgraded its machinery to run on triphase for greater efficiency.
Electricians installed a triphase supply to handle the building's heavy power demands.
A triphase system is commonly used in industrial settings for large motors.

Related Terms

  • antonym: single-phase - an electrical system using only one alternating current
  • synonym: three-phase - an electrical system using three alternating currents offset in phase
