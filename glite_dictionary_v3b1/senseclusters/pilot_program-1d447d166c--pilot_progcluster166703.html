pilot program

noun
trial project or test initiative
3.1 3.0
Pronunciation: /ˈpaɪ.lət ˈproʊ.græm/ /; / /ˈpaɪ.lət ˈproʊ.grəm/ (AmE), /ˈpaɪ.lət ˈprəʊ.græm/ /; / /ˈpaɪ.lət ˈprəʊ.ɡrəm/ (BrE)
a small-scale project or program used to test if a larger project will work before it is fully started

Examples

The school launched a pilot program to test the new math curriculum before expanding it to all grades.
