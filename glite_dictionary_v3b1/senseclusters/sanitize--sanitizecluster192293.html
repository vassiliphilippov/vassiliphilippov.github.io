sanitize

verb
remove or mask sensitive digital data

(in computing and data management) to remove, mask, or modify sensitive or potentially harmful information in digital data, logs, or user input to protect privacy or security.

Examples

To sanitize data
Before releasing the log files, the engineer sanitized them to remove IP addresses and usernames.
They asked the data team to sanitize the dataset by replacing names with random identifiers.
To prevent SQL injection, the developer always sanitizes user input before inserting it into queries.
The framework will automatically sanitise user-submitted comments before storing them.
After the breach, the team sanitised log files to remove malicious payloads and secrets.
They are sanitising all uploaded files to block scripts that could enable XSS attacks.

Related Terms

  • synonym: anonymize - to remove or alter personal identifiers in data so individuals cannot be identified
  • synonym: escape - to add or replace characters in input so it cannot be interpreted as code (a common technique when sanitising).
  • synonym: redact - to edit or remove sensitive information from a document or record before release
  • synonym: sanitize - to clean or remove unsafe content from data or input (US spelling).
