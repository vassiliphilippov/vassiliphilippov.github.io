registerable

adjective

capable of being stored in or assigned to a register, especially in computing or electronics

Examples

Registerable data type
Only registerable data types can be processed efficiently by the CPU.
The compiler checks if a variable is registerable before assigning it to a hardware register.
Some complex structures are not registerable and must be stored in main memory.
