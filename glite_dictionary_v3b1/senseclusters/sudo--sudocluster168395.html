sudo

x
Unix privilege command
8.4
Pronunciation: / ˈsuːduː / (AmE), / ˈsuːduː / (BrE)
a command in Unix-like operating systems that allows a user to run programs with the security privileges of another user, typically the superuser

Examples

sudo command line
To install new software, you often need to type sudo before the command.
He used sudo to access files that required administrator privileges.
Without sudo, the system will not let you make those changes.

Related Terms

  • synonym: root - the superuser account in Unix-like systems
  • synonym: su - a Unix command to switch user accounts
