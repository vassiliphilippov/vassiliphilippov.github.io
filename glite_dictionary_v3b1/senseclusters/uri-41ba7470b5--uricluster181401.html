URI

noun
internet resource identifier
5.9
Pronunciation: /ˈjuː.ɑːr.aɪ/ /; / /ˈjuː.riː/ /; / /ˈuː.riː/ (AmE), /ˈjuː.ɑː.aɪ/ /; / /ˈjuː.riː/ /; / /ˈuː.riː/ (BrE)
a string of characters that identifies a resource on the internet, such as a webpage or file

Examples

URI of a website
To access the image, you must enter the correct URI in your browser.
Each online document is assigned a unique URI for identification.
The developer checked the URI to make sure the link was working properly.

Related Terms

  • synonym: URL - the address of a specific webpage or file on the internet
  • synonym: URN - a unique name that identifies a resource but does not specify its location
