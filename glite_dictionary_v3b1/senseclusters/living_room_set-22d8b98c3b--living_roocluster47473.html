living room set

noun
set of furniture for a living room
1.0 1.9
Pronunciation: /ˈlɪv.ɪŋ ruːm sɛt/ (AmE), /ˈlɪv.ɪŋ ruːm sɛt/ /; / /ˈlɪv.ɪŋ rʊm sɛt/ (BrE)
