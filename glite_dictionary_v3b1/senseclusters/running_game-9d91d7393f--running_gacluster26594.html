running game

noun
offensive running strategy in American football
1.0
Pronunciation: / ˈrʌnɪŋ ɡeɪm / (AmE), / ˈrʌnɪŋ ɡeɪm / (BrE)
(in American football) a style of offensive play in which a team primarily advances the ball by running rather than passing.

Examples

The running game strategy
Defenders focused on stopping their opponent's running game early on.
He believes a solid running game is essential for success in the playoffs.
The team's running game was especially strong in the second half of the match.
