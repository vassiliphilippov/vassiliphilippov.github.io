cmd

noun
Windows batch file
Pronunciation: / siː ɛm diː / (AmE), / siː ɛm diː / (BrE)
a computer file with the .cmd extension that contains a series of batch commands for the Windows command interpreter

Examples

Cmd file script
You can automate repetitive tasks by writing a cmd file for Windows.
The IT department sent a cmd script to help users update their settings.
Before running the cmd file, make sure you understand what each command does.

Related Terms

  • synonym: batch file - a text file containing a series of commands to be executed by the command interpreter
  • synonym: script - a file containing commands that are executed by a program or interpreter
