check out

verb
retrieve files from version control
Pronunciation: / tʃɛk aʊt / (AmE), / tʃɛk aʊt / (BrE)
(in computing) to obtain a working copy of files or a project from a version-control repository, or to switch to a particular branch or revision so you can work on or run the code.

Examples

To check out code
Developers often check out the latest code from the repository before starting to fix bugs.
She checked out the main branch to run the program and reproduce the error.
Before making changes, he's checking out a copy of the project to test locally.

Related Terms

  • antonym: check in - to submit changes back to a version-control repository (opposite operation in some workflows)
  • synonym: clone - to create a local copy of a repository (often used with distributed version control)
  • synonym: download - to transfer a copy of data from a remote system to your local machine
