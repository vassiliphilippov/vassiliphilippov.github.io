harmonizer

noun
software that standardizes and reconciles data
Pronunciation: / c8h51d0rm59ccna6az59r / (AmE), / c8h51d0m59ccna6az59 / (BrE)
(in data management) a software tool or program that standardizes, reconciles, and maps data fields from different sources so they can be integrated and analysed together.

Examples

A data harmonizer
The hospital used a harmonizer to merge patient records from two incompatible datasets.
Our data team bought a cloud-based harmonizer to standardize supplier information across systems.
Before analysis, engineers ran the CSV files through a harmonizer to reconcile inconsistent field names.

Related Terms

  • synonym: data harmonization - the process of bringing data from different sources into a consistent format
  • synonym: data integration - combining data from different sources to provide a unified view
  • synonym: normalize - to transform data into a common format or scale
