preload

noun
data loaded in advance
5.3
Pronunciation: /ˈpriː.loʊd/ /; / /ˈprɪ.loʊd/ (AmE), /ˈpriː.ləʊd/ /; / /ˈprɪ.ləʊd/ (BrE)
data, files, or resources that are loaded in advance so they are ready for immediate use when needed

Examples

Image preload
The software uses a preload to ensure images appear instantly when you open the app.
Developers often create a preload of assets to reduce waiting times for users.
A large preload can help games run more smoothly by minimizing loading screens.

Related Terms

  • synonym: buffer - a temporary storage area for data before processing
  • synonym: cache - a storage location for quick access to frequently used data
