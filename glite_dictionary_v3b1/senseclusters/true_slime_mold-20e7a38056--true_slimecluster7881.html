true slime mold

noun
a slime mold of class Myxomycetes
Pronunciation: / truː slaɪm moʊld / (AmE), / truː slaɪm məʊld / (BrE)
a type of slime mold belonging to the class Myxomycetes, characterized by being multicellular protists that move and feed by amoeboid motion

Examples

true slime mold on logs
True slime mold can be observed on decaying logs in humid forests.
Researchers study true slime mold for insights into cell movement and differentiation.
Unlike cellular slime molds, true slime mold forms a large, acellular mass called a plasmodium.
