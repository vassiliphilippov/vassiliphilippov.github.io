surjective function

noun
mathematics: function that is onto
9.9 0.00/m
Pronunciation: / s…öÀàd í…™kt…™v Ààf å≈ãk É…ôn / (AmE), / s…úÀêÀàd í…™kt…™v Ààf å≈ãk É…ôn / (BrE)
A function in mathematics in which every element of the target set is mapped to by at least one element of the domain; an onto mapping.

Examples

A surjective function
A surjective function ensures that every output value is associated with at least one input value.
In this exercise, you must determine whether the given mapping is a surjective function.
The concept of a surjective function is important in determining whether an equation has solutions in every case.

Synonyms

  • onto function

  • surjection
    9.9 0.00/m
    a surjection is the noun mathematicians use in definitions and proofs, while a surjective function is the phrase naming the same mapping.

Related Terms
