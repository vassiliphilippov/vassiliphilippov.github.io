loadable

adjective
able to be loaded
4.0
Pronunciation: / ˈloʊdəbl̩ / (AmE), / ˈləʊdəbl̩ / (BrE)
able to be loaded or capable of being loaded, especially into a system or device

Examples

A loadable file
The new application is only loadable on computers with updated software.
Make sure the file is loadable before sending it to your colleague.
Some games are not loadable on older consoles due to compatibility issues.

Related Terms

  • synonym: compatible - able to work with a particular system or device
  • antonym: unloadable - not able to be loaded
