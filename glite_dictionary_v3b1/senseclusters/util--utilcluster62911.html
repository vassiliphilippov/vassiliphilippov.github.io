util

noun
small software tool
Pronunciation: / ˈjuːtɪl / (AmE), / ˈjuːtɪl / (BrE)
an informal term for a small utility program or software tool that performs a simple, specific task

Examples

Handy util program
I found a useful util that quickly converts image files to different formats.
This util helps you organize your desktop icons with just one click.
He wrote a simple util to automate the backup process on his computer.

Related Terms

  • synonym: utility - a software program designed to perform a specific function
