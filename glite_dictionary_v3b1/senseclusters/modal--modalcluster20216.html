modal

adjective
blocking UI dialog
7.5 0.06/m
Pronunciation: / c8mo8a.dl29 / (AmE), / c8m598a.d59l / (BrE)
(computing) (of a window, dialog, or interface element) requiring the user to deal with it before returning to the rest of the application; blocking other interaction until dismissed.

Examples

Modal dialog box
The application opened a modal dialog requiring you to confirm before continuing.
Designers advise against too many modal interruptions because they break the user's flow.
When testing, she noticed the modal window prevented interaction with the background form.

Related Terms

  • synonym: modal window - a window or dialog that prevents interaction with other parts of the interface until dismissed
  • antonym: modeless - not modal; an interface element that allows the user to continue interacting with other parts of the application
