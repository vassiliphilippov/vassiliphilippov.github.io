image

noun
the set of a function's possible outputs
Pronunciation: / ˈɪmɪdʒ / (AmE), / ˈɪmɪdʒ / (BrE)
(mathematics) The set of all values that a function may take as its output, given its domain.

Examples

Function image
If you restrict the domain, the image of the function may change accordingly.
In linear algebra, the image of a matrix transformation is a key concept.
The image of the function g(x) = 2x is the set of all real numbers.

Related Terms

  • synonym: range - the set of all possible output values of a function
