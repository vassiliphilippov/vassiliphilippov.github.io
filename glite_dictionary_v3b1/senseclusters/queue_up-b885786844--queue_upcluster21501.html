queue up

verb
add items to a queue
3.5 0.06/m
Pronunciation: / kjuÀê  åp / (AmE), / kjuÀê  åp / (BrE)
to add or arrange one or more items (such as tasks, media, or jobs) into a queue so they will be executed, played, or processed later.

Examples

Queue up songs
The media player will queue up the next video to play automatically.
She queued up several songs for the party before the guests arrived.
Can you queue up the print jobs so they run without manual interruption?

Synonyms

  • enqueue
    7.6 0.02/m
    enqueue adds an item to the end of a processing queue in technical settings, while queue up means arranging items to run later.

Related Terms

  • synonym: enqueue - to add an item to the end of a queue (especially in computing).
