monospace

adjective
equal-width characters
5.4
Pronunciation: / ˈmɑnoʊˌspeɪs / (AmE), / ˈmɒnəʊˌspeɪs / (BrE)
having characters or letters that each occupy the same amount of horizontal space

Examples

Monospace font style
Many programmers use monospace fonts to make their code easier to read.
The document looked strange because it mixed monospace and proportional fonts.
Typewriters typically produce text in a monospace style.

Related Terms

  • synonym: fixed-width - having a set width for each character
  • antonym: proportional - having characters of varying widths
