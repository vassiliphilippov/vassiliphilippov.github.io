IFC

noun
BIM data-exchange standard (Industry Foundation Classes)
7.3
Pronunciation: /aɪ ef siː/ (AmE), /aɪ ef siː/ (BrE)
A standard open data model (Industry Foundation Classes) used in Building Information Modeling (BIM) to describe building components, their properties, and relationships so different software can exchange information.

Examples

IFC file format
The architecture firm exchanged project data in IFC to ensure software interoperability.
The contractor opened the IFC file to check quantities and component classifications before ordering materials.
Many modeling tools can export a coordinated IFC so different disciplines share one consistent building model.
