hard mode

noun
difficult setting in a game
Pronunciation: / hɑrd moʊd / (AmE), / hɑːd məʊd / (BrE)
a more difficult level or setting in a video game, designed to present greater challenge to players

Examples

Hard mode setting
She managed to defeat the final boss on hard mode after several attempts.
Some players only play on hard mode to challenge themselves.
Unlocking the secret ending requires completing the game in hard mode.

Related Terms

  • antonym: easy mode - a less difficult setting in a game
