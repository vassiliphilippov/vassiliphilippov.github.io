low-level

adjective
close to hardware; minimal abstraction
6.7
Pronunciation: /loʊ ˈlɛv.əl/ /; / /ˈloʊ.lɛv.əl/ (AmE), /ləʊ ˈlɛv.əl/ /; / /ˈləʊ.lɛv.əl/ (BrE)
relating to computer programming or systems that operate close to the hardware, using little abstraction or high-level language features.

Examples

Low-level programming
Developers often write low-level code in assembly to optimize performance.
The operating system includes several low-level routines for direct hardware access.
Students learned low-level programming to understand how processors execute instructions.

Related Terms

  • antonym: high-level - programming languages or code that use strong abstraction and are further from direct hardware control.
  • synonym: machine-level - code or operations expressed in terms of machine instructions executed directly by a processor.
