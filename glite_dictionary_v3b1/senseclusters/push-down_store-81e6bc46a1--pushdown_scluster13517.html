push-down store

noun
storage structure with last-in, first-out retrieval
8.6
Pronunciation: / ˈpʊʃ daʊn stɔɹ / (AmE), / ˈpʊʃ daʊn stɔː / (BrE)
a data storage device or system in which the most recently stored item is the first to be retrieved (last in, first out)

Examples

A push-down store system
In computer science, a push-down store is also known as a stack.
The error occurred because the push-down store was full.
The program uses a push-down store to manage function calls efficiently.

Related Terms

  • synonym: stack - a data structure based on the last-in, first-out (LIFO) principle
