archiver

noun
file-compressing tool
5.6
Pronunciation: / ˈɑːrkaɪvər / (AmE), / ˈɑːkaɪvə / (BrE)
a program or tool that collects multiple files into a single archive file, often compressing them for storage or transfer

Examples

File archiver software
The archiver quickly combined all the documents into a single compressed file.
You need an archiver to package these photos before sending them by email.
After downloading the software, use an archiver to extract its contents.

Related Terms

  • synonym: compressor - a tool that reduces the size of files
  • antonym: extractor - a tool that unpacks files from an archive
