defer

verb
schedule code to run at scope exit
7.9
Pronunciation: /dɪ.ˈfɜːr/ (AmE), /dɪ.ˈfɜː/ (BrE)
(in programming) to arrange that a function or block of code is executed automatically when the current function or scope finishes, typically to perform cleanup or release resources.

Examples

To defer code execution
Go programmers often use defer to close files automatically when a function returns.
The library automatically deferred the database close operation until the transaction finished.
The cleanup routine defers closing sockets until after the function's last statement executes.

Related Terms

  • synonym: postpone - to delay until a later time; in programming, to delay execution until scope exit.
  • synonym: schedule - to arrange for something to happen at a specific time or event (here, to run code later, at scope exit).
