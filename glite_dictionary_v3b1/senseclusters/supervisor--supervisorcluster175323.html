supervisor

noun
program controlling other programs (computing)
1.0 3.9
Pronunciation: /ˈsuː.pər.vaɪ.zər/ (AmE), /ˈsuː.pə.vaɪ.zə/ (BrE)
a program in a computer system that controls the execution of other programs, typically managing resources and enforcing rules

Examples

System supervisor
A fault in the supervisor can lead to failures in multiple applications running on the computer.
The supervisor ensures that each user's process does not interfere with others on the system.
When the system boots, the supervisor is loaded first to coordinate program execution.

Related Terms

  • synonym: operating system - main software controlling a computer and its resources
