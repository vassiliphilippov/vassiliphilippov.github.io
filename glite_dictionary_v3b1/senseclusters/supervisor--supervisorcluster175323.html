supervisor

noun
program controlling other programs (computing)
4.8 0.70/m
Pronunciation: / ÀàsuÀê.p…öÀåva…™.z…ö / (AmE), / ÀàsuÀê.p…ôÀåva…™.z…ô / (BrE)
a program in a computer system that controls the execution of other programs, typically managing resources and enforcing rules

Examples

System supervisor
A fault in the supervisor can lead to failures in multiple applications running on the computer.
The supervisor ensures that each user's process does not interfere with others on the system.
When the system boots, the supervisor is loaded first to coordinate program execution.

Synonyms

  • supervisory program
    5.6 0.12/m
    a supervisory program manages and controls other programs inside a computer system, while a supervisor enforces rules and manages resources for those programs.
  • executive program
    6.5 0.00/m
    an executive program runs inside an operating system to control other programs, while a supervisor focuses on enforcing resource rules.
  • supervisory software
    6.9 0.00/m
    supervisory software stays in main memory to manage user programs and processes, while a supervisor is the program that enforces rules.
