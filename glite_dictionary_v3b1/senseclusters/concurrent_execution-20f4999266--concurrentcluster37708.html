concurrent execution

noun
simultaneous running of programs on one computer
6.8
Pronunciation: / kənˈkɝːrənt ˌɛksɪˈkjuːʃən / (AmE), / kənˈkʌrənt ˌɛksɪˈkjuːʃən / (BrE)
the operation of two or more computer programs or tasks at the same time by a single computer

Examples

Concurrent execution of programs
Concurrent execution allows multiple users to run programs without noticeable delays.
Modern operating systems support concurrent execution to improve efficiency and resource use.
The software was optimized for concurrent execution on multi-core processors.

Related Terms

  • synonym: parallel processing - simultaneous processing of multiple tasks by different processors
