bootloader

noun
computer startup program
Pronunciation: / ˈbuːtˌloʊdər / (AmE), / ˈbuːtˌləʊdə / (BrE)
a program that starts a computer and loads the operating system into memory

Examples

The bootloader program
If the bootloader is damaged, your computer may not start at all.
Developers sometimes unlock the bootloader to install custom software on their devices.
The bootloader runs first when you power on your smartphone, before the operating system loads.

Related Terms

  • synonym: firmware - software programmed into hardware devices
  • degree higher: operating system - the main software that manages a computer's hardware and software resources
