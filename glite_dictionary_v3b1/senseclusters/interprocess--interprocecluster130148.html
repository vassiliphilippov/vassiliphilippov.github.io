interprocess

adjective
between processes
6.8
Pronunciation: / ˌɪntərˈprɑːsɛs / (AmE), / ˌɪntəˈprəʊsɛs / (BrE)
relating to or occurring between two or more separate processes, especially in computing

Examples

Interprocess communication
The operating system provides interprocess communication tools for developers.
Security is important when designing interprocess data exchanges.
Efficient interprocess coordination can improve overall system performance.

Related Terms

  • synonym: interprocess communication - methods for exchanging data between separate processes in computing
  • antonym: intraprocess - occurring within a single process
