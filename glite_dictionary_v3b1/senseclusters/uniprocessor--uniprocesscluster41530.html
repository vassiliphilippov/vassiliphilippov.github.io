uniprocessor

adjective
single-processor system
Pronunciation: / ˌjuːniˈprɑːsɛsər / (AmE), / ˌjuːnɪˈprɒsɛsə / (BrE)
designed for or operating on a computer system with only one processor

Examples

Uniprocessor system
This software is compatible only with uniprocessor systems and may not run on modern hardware.
The uniprocessor design limited the computer's ability to handle multiple tasks at once.
Older operating systems were often built for uniprocessor environments before multicore technology became common.

Related Terms

  • antonym: multiprocessor - designed for or using more than one processor
