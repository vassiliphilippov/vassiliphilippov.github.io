emerge

verb
install/build packages with Portage
7.9
Pronunciation: /ɪ.ˈmɜːrdʒ/ /; / /ˈiː.mərdʒ/ (AmE), /ɪ.ˈmɜːdʒ/ /; / /iː.ˈmɜːdʒ/ (BrE)
(in computing, especially on Gentoo Linux) to install or build a software package using the Portage package manager (the 'emerge' command).

Examples

To emerge a package
System administrators often emerge new packages on Gentoo to keep servers secure and up to date.
Before deployment, she emerged the updated library and tested it in a clean chroot.
When the build failed, the team re-ran emerge with verbose output to diagnose the error.

Related Terms

  • synonym: build - to compile or assemble software from source code into a usable program
  • synonym: install - to put software onto a computer so it is ready to use
