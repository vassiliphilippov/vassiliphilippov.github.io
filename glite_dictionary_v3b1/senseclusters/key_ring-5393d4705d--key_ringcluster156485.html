key ring

noun
digital repository for cryptographic keys
4.1
Pronunciation: / c8ki ccr6ab / (AmE), / c8kid0 ccr6ab / (BrE)
(in computing) a digital repository or service that securely stores cryptographic keys, passwords, and other credentials for use by applications and the operating system.

Examples

Digital key ring
The password manager integrates with the system key ring to auto-fill login details.
Developers used the key ring to store API credentials for service-to-service authentication.
After the breach, administrators rotated keys in the key ring and revoked old certificates.

Related Terms

  • synonym: keychain - an operating-system service (e.g., macOS Keychain) that stores passwords and keys for applications
  • synonym: keystore - a software storage mechanism for cryptographic keys and certificates
