main diagonal

noun
diagonal in a square matrix from top left to bottom right
Pronunciation: / meɪn daɪˈæɡənəl / (AmE), / meɪn daɪˈæɡənəl / (BrE)
The diagonal of a square matrix running from the upper left entry to the lower right entry.

Examples

The main diagonal of matrix
Only the main diagonal of an identity matrix contains ones; the rest are zeros.
Swapping the main diagonal entries can change the properties of a matrix.
The trace of a matrix is the sum of the elements on its main diagonal.

Related Terms

  • antonym: antidiagonal - the diagonal of a square matrix from upper right to lower left
