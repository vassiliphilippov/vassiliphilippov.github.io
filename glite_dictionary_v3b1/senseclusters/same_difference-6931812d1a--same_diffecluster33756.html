same difference

noun
no significant difference; essentially the same
5.1
Pronunciation: / seɪm ˈdɪfɚəns / (AmE), / seɪm ˈdɪfərəns / (BrE)
used to say that two things are essentially the same or have no significant difference, despite minor distinctions.

Examples

The same difference
Call it soda or pop; to me, it's the same difference.
He argued that using a taxi or a ride-share app was the same difference.
It doesn't matter if you write in pen or pencil—it's the same difference.
