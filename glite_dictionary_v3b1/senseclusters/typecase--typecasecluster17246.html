typecase

noun
printer's type box
9.8
Pronunciation: / ˈtaɪpˌkeɪs / (AmE), / ˈtaɪpˌkeɪs / (BrE)
a compartmentalized box used to store and organize movable type for printing presses

Examples

A typecase box
The printer carefully arranged the letters in the typecase before starting the press.
Each compartment in the typecase held a different letter or symbol for easy access.
Old typecases are now often used as decorative shelves in homes and studios.

Related Terms

  • synonym: compositor - a person who arranges type for printing
  • synonym: type tray - a tray for holding movable type in printing
