adjective

(in computing) following the REST (Representational State Transfer) architectural style; describing web services or APIs that are stateless, resource-based, and use standard HTTP methods to access resources.

Examples

Restful API design
The platform offers a restful API for developers to integrate with.
They redesigned their backend into a restful system to simplify mobile app requests.
A restful service uses standard HTTP verbs and clear resource URLs for consistency.

Related Terms

  • synonym: REST-compliant - following the principles of REST (Representational State Transfer) for web services
  • synonym: stateless - not retaining client session state between requests; a key property of RESTful services
