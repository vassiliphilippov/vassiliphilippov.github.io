second class

noun
less expensive travel accommodation

the category of accommodations on a ship, train, or plane that are less expensive and offer fewer privileges than those in first class

Examples

Second class ticket
He bought a second class ticket for the overnight train to Paris.
Most of the students traveled in second class to save money.
She preferred second class because it was affordable, even if it wasn't as comfortable.

Synonyms

  • cabin class
    3.1 0.01/m
    cabin class gives modest private space on ships or trains, while second class names a lower-priced option with fewer privileges.
  • economy class
    2.0 0.11/m
    economy class is the modern lowest-cost seating on planes and trains, while second class is a general lower-priced category.
  • tourist class
    3.2 0.01/m
    tourist class sells the cheapest tickets with very basic comfort for travelers, while second class signals lower price and fewer privileges.

Related Terms
