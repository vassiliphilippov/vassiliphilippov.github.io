fabric

noun
data-management architecture
6.0
Pronunciation: /ˈfæ.brɪk/ /; / /ˈfæ.brək/ (AmE), /ˈfæ.brɪk/ /; / /ˈfæ.brək/ (BrE)
(in computing) a data-management architecture that provides a unified, interconnected layer for accessing, integrating, and managing data across distributed systems and cloud environments.

Examples

Data fabric platform
Enterprises are adopting a data fabric to unify sources across on-premise and cloud systems.
The new analytics platform queries the distributed fabric in real time to produce combined reports.
Engineers designed the microservices to sit on a secure fabric that handles identity and data flow.

Related Terms

  • synonym: data architecture - the design and structure of systems and processes used to manage and store data
  • synonym: data integration - the process of combining data from different sources to provide a unified view
  • synonym: framework - a basic structure or set of rules that supports or organizes a system
