plaintext

noun
unencrypted text
Pronunciation: / ˈpleɪnˌtɛkst / (AmE), / ˈpleɪnˌtɛkst / (BrE)
text or data that is not encrypted; the original, readable message before encryption

Examples

A plaintext message
Never send your password in plaintext over the internet, as it can be easily intercepted.
The file contained sensitive information stored as plaintext instead of being encrypted.
Hackers were able to access the plaintext messages because the system lacked proper security.

Related Terms

  • antonym: ciphertext - text that has been encrypted and is not readable without decryption
  • synonym: cleartext - text that is not encrypted and can be read by anyone
