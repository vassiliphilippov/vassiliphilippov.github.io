truthiness

noun
Boolean true property
9.1
Pronunciation: / ˈtruːθinəs / (AmE), / ˈtruːθinəs / (BrE)
the property of a value or expression that determines whether it is considered true in a Boolean context, especially in computing

Examples

Truthiness in Python
In Python, the truthiness of an empty list is considered false.
You should always check the truthiness of a variable before using it in a conditional statement.
The function returns a value whose truthiness determines whether the next step will execute.
