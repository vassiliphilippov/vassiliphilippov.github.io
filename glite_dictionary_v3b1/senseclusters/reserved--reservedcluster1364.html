reserved

adjective
set aside for specific use
1.0 2.2
Pronunciation: /rɪ.ˈzɜːrvd/ /; / /riː.ˈzɜːrvd/ (AmE), /rɪ.ˈzɜːvd/ (BrE)
(of a place, seat, or thing) set aside for the use of a particular person or group.

Examples

Reserved table
She saw a table with a sign that said it was reserved.
The front row seats were reserved for special guests at the ceremony.
This parking space is reserved for employees only.

Related Terms

  • synonym: allocated - set aside for a particular purpose or person
  • antonym: available - free for use by anyone, not set aside
