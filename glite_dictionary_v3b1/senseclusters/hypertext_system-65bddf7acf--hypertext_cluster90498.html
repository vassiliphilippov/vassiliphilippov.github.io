hypertext system

noun
system for linking text nodes non-linearly
3.3 6.1
Pronunciation: /ˈhaɪ.pər.tekst ˈsɪs.təm/ (AmE), /ˈhaɪ.pə.tekst ˈsɪs.təm/ (BrE)
A database management system that enables text fragments ('objects') to be organized as a network of nodes linked together in a non-linear way, allowing users to navigate between related pieces of information.

Examples

A hypertext system network
A hypertext system lets users jump between related pieces of information by clicking on links.
Early hypertext systems revolutionized how electronic documents were navigated.
The World Wide Web is the largest existing hypertext system in the world.
