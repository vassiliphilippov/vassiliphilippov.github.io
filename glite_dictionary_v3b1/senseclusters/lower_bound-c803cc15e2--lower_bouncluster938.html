lower bound

noun
minimum possible value in a set
Pronunciation: / ˈloʊər baʊnd / (AmE), / ˈləʊə baʊnd / (BrE)
(mathematics) a value that is less than or equal to every element in a given set.

Examples

The lower bound value
If five is the lower bound of a set, no member of the set is less than five.
The sequence is said to be bounded below if it has a lower bound.
Zero is a lower bound for all non-negative numbers.

Related Terms

  • antonym: upper bound - a value greater than or equal to every element in a set
