goto

x
programming jump command
Pronunciation: / ˈɡoʊtuː / (AmE), / ˈɡəʊtuː / (BrE)
a command in some programming languages that causes the program to jump to another line or section of code

Examples

Goto command in programming
Many programmers avoid using the goto command because it can make code difficult to follow.
The old programming manual explains how to use goto to jump between different parts of the code.
Some modern languages do not include the goto statement to encourage better coding practices.

Related Terms

  • synonym: jump - a command that transfers control to another part of a program
  • antonym: return - a command that transfers control back to a previous point in a program
