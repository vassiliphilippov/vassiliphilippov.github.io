monad

noun
programming structure
9.9
Pronunciation: /ˈmoʊ.næd/ (AmE), /ˈmɒ.næd/ (BrE)
a structure in programming that organizes data and actions in a consistent, predictable way, often to manage side effects

Examples

A programming monad
In functional programming, a monad helps manage side effects like input and output.
The concept of a monad can be challenging for beginners to understand at first.
Many libraries use the monad pattern to simplify complex data flows.

Related Terms

  • synonym: container - a data structure that holds and organizes values
  • synonym: functor - a programming structure that applies a function to values inside a container
