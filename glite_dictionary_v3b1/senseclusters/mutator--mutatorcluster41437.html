mutator

noun
memory-modifying process
9.5
Pronunciation: / ˈmjuːteɪtər / (AmE), / ˈmjuːteɪtə / (BrE)
a program, thread, or process that modifies memory or objects during execution, especially in the context of garbage collection

Examples

Mutator process
The mutator continues to allocate and change objects while the program is running.
During garbage collection, the mutator may be paused to ensure memory safety.
Each thread in the application acts as a mutator when it updates shared data.

Related Terms

  • antonym: garbage collector - a system that automatically frees unused memory in a program
  • synonym: process - a running instance of a program
