native

adjective
built to run on a particular platform (software)
Pronunciation: / c8ne6a.t2c6av / (AmE), / c8ne6a.t6av / (BrE)
(in computing) designed to run directly on a particular platform or operating system, using that platform's built-in interfaces or APIs rather than through an intermediary.

Examples

Native app development
They decided to develop a native Android app to use device sensors more efficiently.
Because of performance needs, the team rewrote the web tool as a native desktop application.
On older phones, native code often runs faster than interpreted or cross-platform frameworks.

Related Terms

  • antonym: cross-platform - software designed to run on multiple platforms or operating systems.
  • synonym: platform-specific - software designed for and limited to a particular computing platform or operating system.
