relational database

noun
database organized as related tables
4.0
Pronunciation: / rɪˈleɪʃənəl ˈdeɪtəˌbeɪs / (AmE), / rɪˈleɪʃənəl ˈdeɪtəˌbeɪs / (BrE)
A database in which data is organized into tables with defined relationships between them, allowing for easy access and management of associated information.

Examples

A relational database table

Many companies use a relational database to efficiently store and retrieve customer information.
