rubber tree

noun
tree producing latex for natural rubber
2.8 0.16/m
Pronunciation: / Ààr åb…ôr tri / (AmE), / Ààr åb…ô triÀê / (BrE)
a South American tree that produces latex, the main source of natural rubber

Examples

A rubber tree plant
A single mature rubber tree can produce latex for more than twenty years.
Farmers in Southeast Asia have established vast rubber tree plantations.
The rubber tree is tapped for its valuable latex, used to make natural rubber products.

Synonyms

  • Hevea brasiliensis
    9.9 0.00/m

  • Para rubber tree
    7.2 0.00/m
    Para rubber tree is a regional common name tied to Amazon rubber trade and plantations, while rubber tree is the general everyday label.
