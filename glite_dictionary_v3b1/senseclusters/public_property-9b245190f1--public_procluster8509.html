public property

noun
publicly accessible class member (programming)
4.3
Pronunciation: /ˈpʌb.lɪk ˈprɑː.pər.tiː/ (AmE), /ˈpʌb.lɪk ˈprɒp.ə.tiː/ (BrE)
(in computer programming) a class or object member declared with public access, allowing external code to read or modify its value (often via get/set accessors).

Examples

Public property in code
Make the public property 'Name' so other classes can access the value directly.
In C#, you declare a public property with get and set accessors to expose data safely.
During refactoring she turned one private field into a public property for easier testing.

Related Terms

  • synonym: attribute - an object property or value associated with an object or class (often similar to a property in some languages)
  • antonym: private property - a class member accessible only within the defining class or module (opposite access level)
  • synonym: public member - a class or object member (method or data) accessible from other code
