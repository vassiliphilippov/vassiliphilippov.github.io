source code

noun
program text before compilation or execution
4.9 3.3
Pronunciation: /sɔːrs koʊd/ (AmE), /sɔːs kəʊd/ (BrE)
A set of program instructions written in a programming language as plain text, which must be converted into executable code by a compiler, interpreter, or assembler.

Examples

Most software developers store their source code in version control systems.
