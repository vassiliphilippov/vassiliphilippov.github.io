multitasking

noun
running multiple programs
3.5
Pronunciation: /ˈmʌl.tiː.tæ.skɪŋ/ /; / /ˈmʌl.tɪ.tæ.skɪŋ/ (AmE), /ˈmʌl.tɪ.tɑː.skɪŋ/ /; / /ˈmʌl.tiː.tɑː.skɪŋ/ (BrE)
the ability of a computer, operating system, or device to run more than one program or process at the same time

Examples

Multitasking computer
Modern smartphones are designed for efficient multitasking, letting users switch between apps easily.
The new operating system improved multitasking by allowing more programs to run at once.
With advanced multitasking, the computer can handle video editing and web browsing simultaneously.

Related Terms

  • synonym: parallel processing - simultaneous data processing using multiple processors
  • antonym: single-tasking - running only one program or process at a time
