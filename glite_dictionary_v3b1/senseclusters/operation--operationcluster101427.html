operation

noun
a mental process or cognitive activity
6.1 1.54/m
Pronunciation: / Àå…ëÀê.p…ôÀàre…™. É…ôn / (AmE), / Àå…íp.…ôÀàre…™. É…ôn / (BrE)
a mental process involving the performance of a composite cognitive activity, particularly in psychology

Examples

Memory operation
Memory is a complex operation involving several different parts of the brain.
Problem-solving requires the operation of analytical and creative thinking skills.
The psychologist described learning as a cognitive operation that organizes information in the mind.

Synonyms

  • cognitive process
    4.4 1.85/m
    cognitive process is any mental activity such as learning, memory, or reasoning, while operation names a complex activity used in psychology.
  • cognitive operation
    6.4 0.04/m
    cognitive operation changes or manipulates information in the mind, while operation refers more broadly to a composite cognitive activity in psychology.
