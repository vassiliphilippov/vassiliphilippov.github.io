Synonyms

  • attachment
    3.4 1/m
    attachment is a device fixed to something to add a function, while accessory is any extra item that improves use or appearance.
  • appurtenance
    9.9 0.03/m
    appurtenance is a formal word for equipment that helps an activity, while accessory is a common term for extra useful or decorative parts.
  • adjunct
    6.9 0.3/m
    adjunct is something added to a main thing that is not essential, while accessory is often a useful or decorative extra.
  • add-on
    3.9 1/m
    add-on is a part or software added to give extra features, while accessory can be any extra item that improves function or look.
  • supplement
    3.9 5/m
    supplement is something added to improve or replace a lack, while accessory is usually a physical extra that adds use or style.
