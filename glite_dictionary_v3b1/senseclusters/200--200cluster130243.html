(computing, web development) An HTTP status code meaning that a request succeeded and the server has returned the requested resource or confirmation.

Examples

HTTP 200 response
When the server responded with 200, the client displayed the fetched data without error.
After testing the endpoint, she saw a 200 and confirmed the API call succeeded.
Logs showed repeated 200 entries for the health check, indicating the service was available.

Related Terms

  • antonym: 404 - an HTTP status code meaning 'Not Found'; often contrasted with 200
  • synonym: HTTP 200 OK - the standard HTTP response phrase for the 200 status code indicating success
  • synonym: HTTP status code - a three-digit code sent by a web server to indicate the result of a client's request
