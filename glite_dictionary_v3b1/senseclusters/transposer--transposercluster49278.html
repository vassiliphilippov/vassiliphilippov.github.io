transposer

noun
order-changing tool
Pronunciation: / trænˈspoʊzər / (AmE), / trænˈspəʊzə / (BrE)
a person, device, or program that changes the order or arrangement of elements, such as swapping rows and columns in a table

Examples

Table transposer
The spreadsheet software includes a transposer to quickly switch rows and columns.
She used a transposer to rearrange the musical notes for a different instrument.
A skilled transposer can adapt data formats for various applications.
