unmarshal

verb
to decode data
Pronunciation: / ʌnˈmɑrʃəl / (AmE), / ʌnˈmɑːʃəl / (BrE)
to convert data from a stored or transmitted format back into a usable or native form

Examples

To unmarshal data
The application will unmarshal the JSON response into a usable object.
Before analysis, the system must unmarshal the data received from the server.
After transmission, the file is unmarshalled so the program can read its contents.

Related Terms

  • synonym: deserialize - to convert data from a serialized format back into its original structure
  • antonym: marshal - to convert data into a format suitable for storage or transmission
