word

noun
unit of data in computer memory
7.2
Pronunciation: / wɝd / (AmE), / wɜːd / (BrE)
a fixed-length sequence of bits as a unit of data in computer memory, typically used in processing or storage operations

Examples

A 64-bit word
A modern processor may handle instructions involving a 64-bit word.
Early computers used a 16-bit word size for their processors.
The memory address corresponds to each word stored in the system.
