subroutine

noun
sequence of program steps for a specific task
6.5 0.96/m
Pronunciation: / Ààs åb.ruÀåtin / (AmE), / Ààs åb.ruÀê.tiÀên / (BrE)
a sequence of instructions within a computer program designed to perform a specific task, which can be used repeatedly as part of the larger program

Examples

A subroutine in program
Carefully commenting your subroutine can make your code easier to maintain and understand.
Each subroutine is responsible for handling a specific type of calculation.
The main program calls the subroutine whenever it needs to sort the data.

Synonyms

  • subprogram
    7.2 0.08/m
    a subprogram is a sequence that performs a task inside a larger program, while a subroutine emphasizes repeated use as a callable part.
  • routine
    3.8 1.70/m
    a routine is a sequence of program instructions used inside software, while a subroutine specifically means a reusable callable sequence.
  • procedure
    4.1 10/m
    a procedure is a set of instructions in a program that does a task, while a subroutine is a reusable callable block.
  • function
    4.9 51/m
    a function links inputs to outputs and often returns a value, while a subroutine is a reusable sequence that performs a task.

Related Terms

  • synonym: function - a named section of a program that returns a value
  • synonym: procedure - a section of code in a program, often not returning a value
