backend

noun
hidden system part
4.1
Pronunciation: / ˈbækˌɛnd / (AmE), / ˈbækˌɛnd / (BrE)
the part of a computer system or application that operates behind the scenes and is not directly accessed by users

Examples

Website backend
The backend of the website manages user data and processes requests.
Developers often specialize in either backend or frontend technologies.
Security updates are regularly applied to the backend to protect sensitive information.

Related Terms

  • antonym: frontend - the part of a computer system or application that users interact with directly
  • synonym: server - a computer or program that provides services to other computers or users
