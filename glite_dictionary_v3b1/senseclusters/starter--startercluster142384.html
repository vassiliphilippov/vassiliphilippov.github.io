starter

noun
motor that starts an engine
3.5 2.14/m
Pronunciation: / Ààst…ërt…ö /; / Ààst…ërt…ôr / (AmE), / Ààst…ëÀêt…ô / (BrE)
a device (electric or manual) used to start the engine of a vehicle or machine

Examples

Car starter
After replacing the starter, the engine turned over smoothly every time.
He learned how to install a new starter in his pickup truck.
The car wouldn't start because the starter was faulty.
He reached for the starter to turn over the old tractor's engine.
The mechanic used a starter to crank the machine by hand.
This vintage car still requires a manual starter before it will run.

Synonyms

  • starter motor
    4.0 0.08/m
    starter motor is an electric motor that turns an engine to start it, while starter can be manual or electric device.
  • starting motor
    3.7 0.01/m
    starting motor is an electric motor that cranks an engine until it runs, while starter can also be a manual device.
  • self-starter
    6.1 0.00/m
    self-starter automatically turns an engine without hand cranking, while starter can be either a manual tool or an electric motor.
  • crank
    4.5 3.46/m
    crank is a hand-turned handle used to spin an engine for starting, while starter also covers electric motors.
