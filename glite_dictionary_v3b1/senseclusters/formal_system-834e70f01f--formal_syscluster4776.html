formal system

noun
abstract structure of symbols and rules in logic/mathematics
2.2 7.7
Pronunciation: /ˈfɔːr.məl ˈsɪs.təm/ (AmE), /ˈfɔː.məl ˈsɪs.təm/ (BrE)
an abstract structure consisting of a set of symbols, rules for forming formulas, and rules for inference, used to analyze deduction, especially in logic and mathematics

Examples

A formal system in logic
Completeness and consistency are important properties for any formal system in mathematical logic.
Gödel's incompleteness theorems have profound implications for the limits of any formal system.
The rules that govern a formal system ensure that each step in a proof is logically justified.

Related Terms

  • synonym: axiomatic system - a system in which statements are derived from axioms by rules of inference
