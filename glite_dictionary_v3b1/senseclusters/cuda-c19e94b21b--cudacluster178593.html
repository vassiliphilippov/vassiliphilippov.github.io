CUDA

noun
NVIDIA computing platform
Pronunciation: / ˈkuːdə / (AmE), / ˈkuːdə / (BrE)
a parallel computing platform and programming model developed by NVIDIA for using GPUs to accelerate computing tasks

Examples

CUDA programming model
Many machine learning libraries are optimized to run on CUDA devices.
Developers use CUDA to speed up complex calculations on NVIDIA graphics cards.
The new software update improved CUDA support for scientific computing applications.

Related Terms

  • synonym: GPU - graphics processing unit, a specialized processor for graphics and parallel tasks
  • synonym: OpenCL - an open standard for parallel programming of diverse processors
