firmware

noun
software programmed into hardware memory
5.9 1.61/m
Pronunciation: / Ààf…úrmÀåw…õr / (AmE), / Ààf…úÀêmÀåw…õ…ô / (BrE)
(computing) permanent software programmed into a read-only memory chip, providing low-level control for a device's specific hardware.

Examples

Firmware update
If the firmware becomes corrupted, the device may not start properly.
The manufacturer released an update to the device's firmware to improve performance.
You can check the version of the firmware in the system settings menu.
