memorization

noun
the act of committing to memory
Pronunciation: / ˌmɛmərəˈzeɪʃən /; / ˌmɛm.ə.raɪˈzeɪ.ʃən / (AmE), / ˌmɛmərʌɪˈzeɪʃ(ə)n /; / ˌmɛm.ər.aɪˈzeɪ.ʃən / (BrE)
the action or process of learning something so that it can be recalled exactly from memory
