control key

noun
keyboard key for special functions
1.0
Pronunciation: / kənˈtroʊl kiː / (AmE), / kənˈtrəʊl kiː / (BrE)
a key on a computer keyboard used together with other keys to perform special functions or enter control characters

Examples

The control key on keyboard
Many keyboard shortcuts require holding down the control key and pressing another key.
Press and hold the control key while selecting multiple files to copy them all at once.
The control key is often labeled as 'Ctrl' on most keyboards.

Related Terms

  • synonym: Ctrl - abbreviation for 'control key' on keyboards
  • synonym: alt key - keyboard key used to change function of other keys
  • synonym: shift key - keyboard key used to modify other keys
