stateful

adjective
retaining previous data
7.1
Pronunciation: / ˈsteɪtfəl / (AmE), / ˈsteɪtfʊl / (BrE)

Examples

Stateful server design
A stateful server remembers user sessions between requests.
The application uses a stateful protocol to track ongoing conversations.
Unlike stateless systems, stateful designs can store information about previous actions.
