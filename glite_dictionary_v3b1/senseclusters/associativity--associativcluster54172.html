associativity

noun
cache storage property
9.8
Pronunciation: / əˌsoʊsiəˈtɪvəti / (AmE), / əˌsəʊsiəˈtɪvəti / (BrE)
the property of a computer cache or memory system that determines how many locations a given address can be stored in

Examples

Cache associativity level
Increasing the associativity of a cache can help reduce the number of memory conflicts.
The engineer explained how associativity affects the performance of the computer's memory system.
A cache with higher associativity allows more flexibility in storing data addresses.

Related Terms

  • degree lower: direct-mapped - a cache organization where each address maps to exactly one location
  • degree higher: fully associative - a cache organization where any address can be stored in any location
