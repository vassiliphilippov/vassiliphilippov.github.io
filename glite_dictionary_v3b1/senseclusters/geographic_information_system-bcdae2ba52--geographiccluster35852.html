geographic information system

noun
system for managing geographic data
1.0
Pronunciation: / dʒiˈɑːɡrəfɪk ˌɪnfərˈmeɪʃən ˈsɪstəm / (AmE), / dʒɪˈɒɡrəfɪk ˌɪnfəˈmeɪʃən ˈsɪstəm / (BrE)
A computer-based system used for capturing, storing, analyzing, and displaying spatial or geographic data.

Examples

A geographic information system software
Researchers rely on a geographic information system to analyze changes in land use over time.
The city uses a geographic information system to manage and update its digital maps.
