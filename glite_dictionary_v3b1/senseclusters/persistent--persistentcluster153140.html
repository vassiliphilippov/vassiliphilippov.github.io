(in computing) remaining stored or maintained across program runs, sessions, or system reboots; not temporary or ephemeral.

Examples

Persistent storage
The app uses persistent storage to keep user preferences after restart.
Developers store session tokens in persistent databases so users stay logged in between visits.
By writing the settings file to persistent memory, the device remembers configuration after power loss.

Synonyms

  • durable
    5.5 0.40/m
    durable data survive hardware or software failures and remain retrievable, while persistent data simply remain stored across runs, sessions, or reboots.

Related Terms

  • synonym: non-volatile - storage that retains data when power is removed; not erased on reboot.
  • antonym: volatile - storage or memory that loses its contents when power is removed or the process ends.
