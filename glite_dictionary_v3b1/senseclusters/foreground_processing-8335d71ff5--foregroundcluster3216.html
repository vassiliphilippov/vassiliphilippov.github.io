foreground processing

noun
computer process with user priority
Pronunciation: / ˈfɔrˌɡraʊnd ˈproʊˌsɛsɪŋ / (AmE), / ˈfɔːˌɡraʊnd ˈprəʊ.ses.ɪŋ / (BrE)
the operation of computer tasks that receive priority and directly interact with the user, often interrupting other activities on the system

Examples

Foreground processing tasks
Foreground processing allows a user to interact directly with the currently running program.
On most modern computers, only one application can use foreground processing at a time.
Switching between foreground processing and background tasks is common in multitasking operating systems.

Related Terms

  • antonym: background processing - computer processing that occurs without direct user interaction and at lower priority
