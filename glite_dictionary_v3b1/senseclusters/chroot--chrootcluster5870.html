chroot

verb
to restrict process directory
9.9
Pronunciation: /tʃruːt/ (AmE), /tʃruːt/ (BrE)
to change the apparent root directory for a running process, restricting its access to a specific part of the filesystem

Examples

To chroot a process
System administrators often chroot services for added security.
You should chroot the process before running untrusted code on the server.
The script failed because it could not chroot to the specified directory.

Related Terms

  • synonym: jail - to confine a process to a limited part of the system
  • synonym: sandbox - to run a program in a restricted environment
