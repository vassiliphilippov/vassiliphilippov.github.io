open-source

adjective
software with freely available source code
3.7 5.37/m
Pronunciation: / Àào äp…ôn s…îÀêrs / (AmE), / Àà…ô äp…ôn s…îÀês / (BrE)
(of computer software) for which the source code is freely available to the public to use, modify, and distribute.

Examples

Open-source software
Linux is an open-source operating system widely used for servers and desktops.
Many developers prefer working with open-source software because they can modify it to suit their needs.
The company decided to release its new program as open-source to encourage community contributions.
