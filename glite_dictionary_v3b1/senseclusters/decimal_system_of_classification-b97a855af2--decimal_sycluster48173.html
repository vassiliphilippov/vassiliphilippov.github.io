decimal system of classification

noun
library classification system using decimal numbers
2.1
Pronunciation: / ˈdɛsɪməl ˈsɪstəm əv ˌklæsɪfɪˈkeɪʃən / (AmE), / ˈdɛsɪməl ˈsɪstəm əv ˌklɑːsɪfɪˈkeɪʃən / (BrE)
a system for organizing books and other materials in a library by assigning them numbers that represent subjects, typically using a three-digit main number and additional numerals after a decimal point for further detail.

Examples

The decimal system of classification
In the decimal system of classification, similar subjects are grouped together, making it easier to browse related materials.
Students quickly adapted to finding resources once they understood the decimal system of classification.
The library uses the decimal system of classification to organize all of its nonfiction books by subject.
