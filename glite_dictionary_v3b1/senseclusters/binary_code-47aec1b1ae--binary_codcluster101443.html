binary code

noun
system representing data as 0s and 1s
3.5 2.9
Pronunciation: /ˈbaɪ.nə.riː koʊd/ (AmE), /ˈbaɪ.nə.riː kəʊd/ (BrE)
a system of representing text or instructions in computers, using patterns of 0s and 1s, typically with groups of eight digits to encode individual characters

Examples

Binary code in computers
Every computer program is ultimately stored and executed as binary code.
Learning how to convert text to binary code can help you understand how computers process information.
The letter 'A' is represented by a unique binary code in ASCII.

Related Terms

  • synonym: ASCII - American Standard Code for Information Interchange; a character encoding based on binary code
  • degree higher: hexadecimal - a base-16 numeral system often used to express binary-coded values in a more compact form
