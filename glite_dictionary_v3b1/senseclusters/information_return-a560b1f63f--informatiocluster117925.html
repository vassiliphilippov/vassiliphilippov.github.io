information return

noun
tax document reporting financial information
9.7 5.5
Pronunciation: /ɪn.fər.ˈmeɪ.ʃən rɪ.ˈtɜːrn/ /; / /ɪn.fər.ˈmeɪ.ʃən riː.ˈtɜːrn/ (AmE), /ɪn.fə.ˈmeɪ.ʃən rɪ.ˈtɜːn/ /; / /ɪn.fə.ˈmeɪ.ʃən riː.ˈtɜːn/ (BrE)
a document submitted to a tax authority that reports information about financial transactions but does not calculate the tax owed

Examples

File an information return
Employers must file an information return to report employees' wages to the government.
Failure to submit an information return on time can result in penalties from the tax authority.
The accountant prepared the necessary information return for the client's investment income.
