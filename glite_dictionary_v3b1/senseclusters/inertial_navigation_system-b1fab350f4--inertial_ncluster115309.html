inertial navigation system

noun
system for self-contained navigation using inertial sensors
5.0 5.7
Pronunciation: /ɪ.ˈnɜːr.ʃəl næv.ɪ.ˈɡeɪ.ʃən ˈsɪs.təm/ (AmE), /ɪ.ˈnɜː.ʃəl næv.ɪ.ˈɡeɪ.ʃən ˈsɪs.təm/ (BrE)
a device or system that determines the position and velocity of a moving object, such as an aircraft or spacecraft, by using internal sensors to measure acceleration and rotation without relying on external references

Examples

Inertial navigation system device
A malfunction in the inertial navigation system caused the spaceship to veer off course.
Engineers are testing a new inertial navigation system for the Mars rover mission.
The aircraft relies on its inertial navigation system during flights over the ocean, where GPS signals are unavailable.
