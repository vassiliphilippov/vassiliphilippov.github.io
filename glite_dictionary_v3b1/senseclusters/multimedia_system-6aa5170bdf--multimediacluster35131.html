multimedia system

noun
system combining multiple media types
Pronunciation: / ˌmʌltiˈmiːdiə ˈsɪstəm / (AmE), / ˌmʌltiˈmiːdiə ˈsɪstəm / (BrE)
A system that transmits or delivers information using more than one type of communication medium, such as text, graphics, sound, and video.

Examples

A multimedia system delivers
Many museums now use a multimedia system to present historical information in engaging ways.
The new multimedia system in the classroom incorporates interactive whiteboards and high-definition video.
We installed a state-of-the-art multimedia system to enhance our home entertainment experience.
