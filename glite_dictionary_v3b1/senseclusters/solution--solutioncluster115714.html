solution

noun
set of values for an equation
Pronunciation: / səˈluːʃən / (AmE), / səˈluːʃən / (BrE)
a set of values that satisfy an equation, making it true when substituted into it

Examples

Solution to the equation
By substituting x = 3, we can check if it is a solution to the equation.
The teacher asked the students to find the solution for each problem in the set.
