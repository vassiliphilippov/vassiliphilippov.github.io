operating procedure

noun
set of instructions for operation
1.0
Pronunciation: / ˈɑpəˌreɪtɪŋ prəˈsiːdʒər / (AmE), / ˈɒpərətɪŋ prəˈsiːdʒə / (BrE)
A set of instructions or steps for operating a machine, system, or for dealing with a particular situation.

Examples

A operating procedure manual
A detailed operating procedure helps to minimize the risk of mistakes in the laboratory.
All staff must follow the operating procedure when handling hazardous materials.
Before using the new software, please read the operating procedure carefully.
