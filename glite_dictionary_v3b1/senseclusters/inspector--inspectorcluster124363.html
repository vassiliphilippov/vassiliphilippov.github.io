inspector

noun
web development debugging tool
4.8
Pronunciation: /ɪn.ˈspek.tər/ (AmE), /ɪn.ˈspek.tə/ (BrE)
a software tool (usually in a browser's developer tools) that lets developers examine, edit, and debug a web page's structure (DOM), styles (CSS), and resources in real time.

Examples

Web inspector
When the layout broke, she opened the inspector to find the CSS rule causing the shift.
The tutorial showed how to use the inspector to edit element attributes live and debug quickly.
During the interview, the candidate explained how an inspector helps locate broken scripts and layout bugs.

Related Terms

  • synonym: DOM inspector - a tool for examining and modifying the document object model (HTML structure) of a web page
  • synonym: developer tools - browser tools for debugging, inspecting, and profiling web pages and applications
