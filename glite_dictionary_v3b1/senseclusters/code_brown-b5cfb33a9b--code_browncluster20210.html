code brown

noun
emergency situation term (often hazardous spill)
Pronunciation: / koʊd braʊn / (AmE), / kəʊd braʊn / (BrE)
a term used, especially in hospitals and public facilities, to indicate a particular type of emergency, often involving a hazardous spill or an incident that requires immediate response

Examples

Code brown emergency
During the drill, we practiced our response to a code brown scenario.
The hospital declared a code brown after a chemical spill occurred in the laboratory.
When the animal escaped, zoo staff called a code brown to alert everyone.

Related Terms

  • synonym: code blue - term for a medical emergency involving cardiac arrest (hospital code)
  • synonym: code red - term for a fire or major emergency (hospital or institutional code)
