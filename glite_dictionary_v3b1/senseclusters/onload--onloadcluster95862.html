onload

noun
web loading event
7.2
Pronunciation: / ˈɑnˌloʊd / (AmE), / ˈɒnˌləʊd / (BrE)
an event in web programming that occurs when a page or element has finished loading

Examples

The onload event
The onload event is used to run code after a webpage finishes loading.
You can display a welcome message by handling the onload event in your script.
Developers often use the onload event to initialize interactive features on a website.

Related Terms

  • synonym: onclick - an event triggered when an element is clicked in web programming
  • antonym: onunload - an event triggered when a page or element is unloaded in web programming
