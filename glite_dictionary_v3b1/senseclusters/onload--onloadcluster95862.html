onload

noun
web loading event
7.2
Pronunciation: / ˈɑnˌloʊd / (AmE), / ˈɒnˌləʊd / (BrE)
an event in web programming that occurs when a page or element has finished loading

Examples

The onload event

You can display a welcome message by handling the onload event in your script.
