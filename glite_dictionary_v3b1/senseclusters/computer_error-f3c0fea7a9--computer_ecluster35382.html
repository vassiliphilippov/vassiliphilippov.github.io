computer error

noun
incorrect result produced by a computer
1.0
Pronunciation: / kəmˈpjuːtər ˈɛrər / (AmE), / kəmˈpjuːtə ˈɛrə / (BrE)
an occasion when a computer produces an incorrect or unexpected result, usually due to a fault in hardware, software, or data

Examples

A computer error message
A minor computer error led to the incorrect calculation of the monthly payments.
If you encounter a computer error, try restarting the program to resolve it.
The company's data loss was caused by a computer error during the backup process.
