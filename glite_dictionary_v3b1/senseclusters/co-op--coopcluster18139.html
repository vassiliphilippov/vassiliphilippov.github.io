co-op

noun
cooperative video‑game mode
3.7
Pronunciation: / ˈkoʊ.ɑp / (AmE), / ˈkəʊ.ɒp / (BrE)
a game mode in which two or more players cooperate with each other (often against the game or AI) to complete objectives or missions.

Examples

Co-op game mode
The game's new co-op lets friends complete missions together instead of fighting each other online.
They preferred playing the campaign in co-op to share resources and watch each other's strategies.
After the update, the studio added a challenging co-op mode with special objectives for four players.

Related Terms

  • antonym: competitive mode - a game mode where players compete against each other
  • synonym: cooperative mode - a game mode where players work together rather than compete
