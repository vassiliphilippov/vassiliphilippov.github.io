mendelevium

noun
a synthetic radioactive chemical element (Md)
9.9
Pronunciation: /men.də.ˈliː.vi.əm/ /; / /men.də.ˈleɪ.vi.əm/ (AmE), /men.də.ˈliː.vi.əm/ /; / /men.də.ˈleɪ.vi.əm/ (BrE)
a synthetic radioactive transuranic element with the symbol Md, produced by bombarding einsteinium with alpha particles

Examples

The element mendelevium
Because mendelevium is highly radioactive and scarce, it is mainly used in scientific research.
Scientists first produced mendelevium in 1955 by bombarding einsteinium with alpha particles.
The chemical symbol for mendelevium was originally Mv, but it is now Md.
