injective

adjective
unique mapping (math)
9.9
Pronunciation: /ɪn.ˈdʒek.tɪv/ (AmE), /ɪn.ˈdʒek.tɪv/ (BrE)
describing a function in which each input is mapped to a unique output, so no two inputs share the same output

Examples

Injective function
A function is called injective if it never assigns the same value to two different inputs.
The mathematician proved that the mapping was injective but not surjective.
Understanding whether a function is injective is important in algebra and set theory.

Related Terms

  • degree higher: bijective - a function that is both injective and surjective
  • synonym: one-to-one - a function where each input has a unique output
  • antonym: surjective - a function where every possible output is covered by some input
