mutator

noun
method that changes state
Pronunciation: / ˈmjuːteɪtər / (AmE), / ˈmjuːteɪtə / (BrE)
a method or function in programming that changes the internal state of an object or data structure

Examples

Mutator method
The mutator allows you to update the user's email address in the database.
Good programming practice separates mutator methods from those that only read data.
By using a mutator, the application ensures that changes to the object are controlled and validated.

Related Terms

  • antonym: accessor - a method that retrieves the value of a property without changing it
  • synonym: setter - a method that sets or updates the value of a property
