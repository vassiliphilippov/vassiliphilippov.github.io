been there, done that, bought the T-shirt

x
jaded by something experienced before
6.3
Pronunciation: / bin ðer dʌn ðæt bɔt ðə ˈtiʃɚt / (AmE), / biːn ðeə dʌn ðæt bɔːt ðə ˈtiːʃɜːt / (BrE)
used to say that you have experienced something and found it unimpressive, unoriginal, or no longer interesting

Examples

Been there, done that, bought the T-shirt phrase
She rolled her eyes and said, 'Politics? Been there, done that, bought the T-shirt.'
They asked me to join the club, but honestly, I've been there, done that, bought the T-shirt.
When it comes to bungee jumping, I've been there, done that, bought the T-shirt, so it doesn't thrill me anymore.
