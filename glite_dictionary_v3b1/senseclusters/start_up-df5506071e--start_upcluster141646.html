start up

verb
begin operating; set in motion
3.0 1.68/m
Pronunciation: / st…ërt  åp / (AmE), / st…ëÀêt  åp / (BrE)
to begin operating or cause something (such as a machine, engine, or system) to begin operating

Examples

To start up a computer
I always start up my laptop as soon as I get to work.
It took several tries to start up the old car in the morning.
The company plans to start up production again next month.

Synonyms

    1.0 843/m
    start is a general word for beginning actions, while start up focuses on causing machines or systems to begin operating.
  • come up
    3.5 9.62/m
    come up describes equipment or systems beginning to work, while start up means causing a machine or system to begin operating.
  • come on
    2.5 14/m
    come on is used for lights or services beginning to work, while start up refers to making a machine or system start operating.
  • go on
    1.1 2.66/m
    go on can mean a machine starts running, while start up specifically means causing or arranging a machine or system to begin operating.
  • turn on
    1.0 23/m
    turn on means causing a device to begin by using a switch or button, while start up often means bringing a system into operation.
  • switch on
    1.6 3.13/m
    switch on means flip a switch or press a button to power a device, while start up covers the whole process of making machines run.
  • begin
    1.0 680/m
    begin means to start an action or event in general, while start up focuses on causing machines or systems to begin operating.
  • get started
    1.8 15/m
    get started means to begin doing an activity or task, while start up mainly means to make machines or systems begin operating.
