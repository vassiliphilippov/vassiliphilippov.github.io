hard disk

noun
permanent magnetic data storage device
3.0 2.69/m
Pronunciation: / hÉ‘rd dÉªsk / (AmE), / hÉ‘Ëd dÉªsk / (BrE)
A rigid, permanently installed magnetic disk used to store data in a computer or similar device.

Examples

The hard disk drive
He accidentally deleted the files from the hard disk and couldn't recover them.
The company's new laptop comes with a larger hard disk for extra storage.
Upgrading your computer's hard disk can improve its performance and capacity.
Many older computers still use a traditional hard disc rather than a solid-state drive.
She backed up her files onto an external hard disc before updating the software.
The technician replaced the faulty hard disc with a new one to restore the computer's performance.

Synonyms

  • fixed disk
    4.4 0.01/m
    a fixed disk is a technical name for a rigid magnetic disk permanently installed in a computer, while a hard disk is the everyday term.
  • hard drive
    2.8 7.32/m
    a hard drive is the device that stores and retrieves data using one or more rigid disks, while a hard disk is the physical disk.

Related Terms

  • antonym: floppy disk - a soft, removable magnetic disk used for computer storage
  • synonym: hard drive - the complete device that houses a hard disk
  • antonym: solid-state drive - a data storage device using flash memory with no moving parts
