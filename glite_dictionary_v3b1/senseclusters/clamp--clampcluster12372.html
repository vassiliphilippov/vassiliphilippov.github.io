clamp

verb
limit a number to a range
6.6
Pronunciation: /klæmp/ (AmE), /klæmp/ (BrE)
(in computing) to restrict a numeric value so that it does not go below a minimum or above a maximum; to force a number to lie within a specified range.

Examples

To clamp a value
To avoid overflow, the function will clamp the input value between zero and 100 before processing.
She clamped the color channel values during editing so no pixel exceeded the allowed brightness.
Physics engines often clamp object positions to the world bounds to prevent visual glitches.

Related Terms

  • synonym: clip - (computing/graphics) to cut off values outside a specified range or region
  • synonym: limit - to set a maximum or minimum boundary for a value or quantity
