true fungus

noun
fungus of the division Eumycota
Pronunciation: / tru ˈfʌŋɡəs / (AmE), / truː ˈfʌŋɡəs / (BrE)
any fungus belonging to the division Eumycota, as distinct from similar fungus-like organisms that are not true fungi.

Examples

Common true fungi
Several common mushrooms are classified as true fungi.
The scientist explained that a true fungus has a cell wall made of chitin.
Unlike slime molds, a true fungus reproduces by spores.
