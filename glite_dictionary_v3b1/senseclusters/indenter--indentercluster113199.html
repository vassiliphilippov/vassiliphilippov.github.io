indenter

noun
text/code formatter
Pronunciation: / ɪnˈdɛntər / (AmE), / ɪnˈdɛntə / (BrE)
a person or computer program that formats text or code by adding spaces or tabs at the beginning of lines to show structure

Examples

Code indenter
The indenter made the source code much easier to read by organizing the lines clearly.
After running the indenter, the program's structure was much more apparent to new developers.
She used an automatic indenter to fix the inconsistent spacing in her script.

Related Terms

  • synonym: formatter - a tool or person that arranges text or data in a specific format
  • antonym: minifier - a tool that removes unnecessary characters from code
