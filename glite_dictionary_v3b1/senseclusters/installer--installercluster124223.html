installer

noun
setup person or program
Pronunciation: / ɪnˈstɔlɚ / (AmE), / ɪnˈstɔːlə / (BrE)
a person or computer program that sets up equipment or software for use

Examples

Software installer
The installer completed the software setup in just a few minutes.
A professional installer came to fit the new air conditioning unit in our office.
Before using the program, you must run the installer to set it up on your computer.

Related Terms

  • synonym: setup - the process of installing or configuring something
  • antonym: uninstaller - a program that removes software from a computer
