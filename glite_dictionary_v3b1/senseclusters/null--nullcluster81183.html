null

noun
no data value
5.9
Pronunciation: / nʌl / (AmE), / nʌl / (BrE)
a value in computing that represents the absence of data or no value assigned

Examples

Null value
The programmer used a null to indicate that no value was present in the field.
If the response is null, the application will display an error message.
Databases often use null to represent missing or unknown information.

Related Terms

  • synonym: empty - containing nothing; not filled
  • antonym: value - a quantity or data assigned to something
  • synonym: zero - the number 0; nothing
