erratum

noun
printer's or publisher's error in print
5.8 7.4
Pronunciation: / ɛˈrɑːtəm / (AmE), / ɪˈrɑːtəm / (BrE)
an error in a printed or published work, often with a correction notice

Examples

An erratum notice
An erratum was included in the following issue to address the typographical mistake.
Please refer to the erratum on page two for the correct figure.
The publisher issued an erratum to correct the misstated data in the report.
The journal published an erratum to correct the data in the original article.
If you find a mistake in the book, check the erratum at the end for corrections.
The erratum listed several spelling errors that appeared in the first edition.

  • antonym: error - a mistake or inaccuracy
