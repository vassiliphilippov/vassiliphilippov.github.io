boot

verb
start a computer's operating system
6.7
Pronunciation: / buːt / (AmE), / buːt / (BrE)
(of a computer) to load the operating system and start the initial processes.

Examples

To boot a computer
Before you can use the system, you need to boot the computer.
He forgot to boot his laptop before the meeting started.
The technician instructed me to boot from the recovery disk.

Related Terms

  • synonym: reboot - to restart a computer's operating system
