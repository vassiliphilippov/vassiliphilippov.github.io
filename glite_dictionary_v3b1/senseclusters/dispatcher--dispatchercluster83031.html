dispatcher

noun
software component that routes tasks/messages
5.2
Pronunciation: / dɪˈspætʃər / (AmE), / dɪˈspætʃə / (BrE)
(in computing) a software component that routes or assigns tasks, messages, or events to appropriate handlers, processes, or workers within a system or application.

Examples

Software dispatcher
The server uses a dispatcher to route incoming jobs to available worker processes for parallel execution.
In the GUI, the event dispatcher delivers user actions to the correct handler on the main thread.
Microservices rely on a central dispatcher to forward messages to services based on topic and load.

Related Terms

  • synonym: router - a component that forwards messages or data to the correct destination
  • synonym: scheduler - a system component that decides the order or timing of tasks to run on processors
