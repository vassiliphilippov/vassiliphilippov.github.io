dynamical system

noun
system describing time evolution in mathematics/physics
7.5
Pronunciation: / daɪˈnæmɪkəl ˈsɪstəm / (AmE), / daɪˈnæmɪkəl ˈsɪstəm / (BrE)
A system in which a function describes the time dependence of a point in a geometrical space, typically studied in mathematics and physics.

Examples

A dynamical system model
A simple pendulum can be considered an example of a dynamical system.
Researchers used computer models to analyze the evolution of the dynamical system over time.
The dynamical system exhibited chaotic behavior under certain conditions.
