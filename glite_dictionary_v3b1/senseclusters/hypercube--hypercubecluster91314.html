hypercube

noun
n-dimensional network
9.5
Pronunciation: /ˈhaɪ.pər.kjuːb/ (AmE), /ˈhaɪ.pə.kjuːb/ (BrE)
a network topology in parallel computing where processors or nodes are connected like the vertices of an n-dimensional cube

Examples

Hypercube network
The new supercomputer uses a hypercube topology to connect thousands of processors efficiently.
Researchers analyzed data transfer speeds within a twelve-dimensional hypercube network.
A hypercube allows for fast communication between nodes in parallel computing systems.
