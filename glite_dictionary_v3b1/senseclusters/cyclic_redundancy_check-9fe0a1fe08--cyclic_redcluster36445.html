cyclic redundancy check

noun
error-detecting code for data integrity
Pronunciation: / ˈsaɪklɪk rɪˈdʌndənsi tʃɛk / (AmE), / ˈsɪklɪk rɪˈdʌndənsi tʃɛk / (BrE)
a method of detecting errors in data, especially on storage devices, by using a mathematical algorithm to generate a checksum that verifies the integrity of the information

Examples

Cyclic redundancy check method
Engineers recommend enabling the cyclic redundancy check feature for maximum data reliability.
Many file transfer protocols use a cyclic redundancy check to ensure files are not altered during transmission.
The hard drive reported a cyclic redundancy check error, indicating possible data corruption.
