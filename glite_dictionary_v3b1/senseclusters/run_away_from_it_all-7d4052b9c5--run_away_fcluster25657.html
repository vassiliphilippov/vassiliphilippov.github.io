run away from it all

verb
escape from problems or responsibilities
1.0
Pronunciation: / rʌn əˈweɪ frəm ɪt ɔl / (AmE), / rʌn əˈweɪ frəm ɪt ɔːl / (BrE)
to escape from one's problems, responsibilities, or the stresses of everyday life, often by going somewhere else temporarily

Examples
