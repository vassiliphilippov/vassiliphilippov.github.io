feedback

noun
output looped back to input (system control)
Pronunciation: / ˈfidˌbæk / (AmE), / ˈfiːdbæk / (BrE)
the process in which part of the output of a system is returned to its input to regulate or modify further output

Examples

Feedback control system
Engineers designed the controller to use feedback from the sensors to maintain stability.
In audio systems, a feedback loop can cause unwanted screeching noises if not properly managed.
The climate system is complex and features several feedback mechanisms that regulate global temperature.

Related Terms

  • antonym: input - data or power supplied to a system
  • antonym: output - the result produced by a system
