feedback loop

noun
process where system output is fed back into input
4.2 1.46/m

a process in which part of the output of a system is returned to its input, often to control or modify its function

Examples

A feedback loop process
A malfunction in the feedback loop can cause the system to behave unpredictably.
Many biological systems depend on a feedback loop to maintain stability.

Synonyms

  • feedback
    3.7 9.27/m
    feedback describes the general process of returning output to input to regulate a system, while feedback loop names the repeating path itself.
  • closed loop
    5.1 0.33/m
    closed loop is a control system that sends output back to its own input to keep a chosen state, while feedback loop describes that returning process.
