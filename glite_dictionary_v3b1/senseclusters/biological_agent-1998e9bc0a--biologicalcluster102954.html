biological agent

noun
organism or toxin used in biological warfare
Pronunciation: / ˌbaɪ.əˈlɑ.dʒɪ.kəl ˈeɪ.dʒənt / (AmE), / ˌbaɪ.əˈlɒ.dʒɪ.kəl ˈeɪ.dʒənt / (BrE)
An organism, such as a bacterium, virus, or toxin, that can be used deliberately to cause disease, especially in biological warfare.

Examples

A biological agent weapon
Anthrax is a famous example of a biological agent that has been used as a weapon.
Laboratories handling any biological agent must follow strict safety protocols.
The government implemented safeguards to prevent the release of a biological agent into the water supply.

Related Terms

  • synonym: bioweapon - biological weapon; a harmful agent used to cause disease or death
  • antonym: chemical agent - substance used for chemical warfare
