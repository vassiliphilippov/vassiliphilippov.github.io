monoalphabetic

adjective
single alphabet system
9.0
Pronunciation: / ˌmɑːnoʊˌælfəˈbɛtɪk / (AmE), / ˌmɒnəʊˌælfəˈbɛtɪk / (BrE)
using only one alphabet or set of symbols for encoding, writing, or substitution

Examples

A monoalphabetic cipher
A monoalphabetic cipher replaces each letter with a fixed substitute throughout the message.
The security of a monoalphabetic code is limited because patterns can be detected easily.
Unlike polyalphabetic systems, monoalphabetic ciphers use only one substitution rule.

Related Terms

  • antonym: polyalphabetic - using more than one alphabet or set of symbols for encoding
