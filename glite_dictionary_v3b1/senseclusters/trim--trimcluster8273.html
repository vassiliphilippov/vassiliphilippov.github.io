trim

verb
send TRIM command to SSD to mark unused blocks
8.1
Pronunciation: /trɪm/ (AmE), /trɪm/ (BrE)
(computing) to send the TRIM (or discard) command to a solid-state storage device to inform it which data blocks are no longer in use so they can be erased or reclaimed.

Examples

To trim SSD blocks
The operating system will trim unused blocks on the SSD to improve long-term performance.
After the update, the storage controller automatically trimmed the drive to prepare blocks for erasure.
You should run fstrim regularly so the system can trim free space on NVMe drives.

Related Terms

  • synonym: discard - to mark blocks on a storage device as unused so they can be reclaimed (Linux term for TRIM).
  • synonym: erase - to remove or clear stored data so the storage space can be reused.
