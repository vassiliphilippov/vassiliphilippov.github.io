pipelining

noun
sequential task processing
Pronunciation: / ˈpaɪplaɪnɪŋ / (AmE), / ˈpaɪplaɪnɪŋ / (BrE)
a method of processing tasks in a sequence of steps so that multiple tasks are handled at different stages simultaneously to improve efficiency

Examples

Processor pipelining
Modern processors use pipelining to execute instructions faster and more efficiently.
The concept of pipelining allows different stages of a task to be processed at the same time.
By implementing pipelining, the company improved the speed of its data processing systems.

Related Terms

  • synonym: parallel processing - simultaneous processing of multiple tasks
  • antonym: sequential processing - processing tasks one after another in order
