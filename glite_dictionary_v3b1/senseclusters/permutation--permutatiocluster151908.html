permutation

noun
arrangement or ordering of a set
5.1 5.8
Pronunciation: /pɜːr.mjuː.ˈteɪ.ʃən/ /; / /pɜːr.mjʊ.ˈteɪ.ʃən/ (AmE), /pə.mjuː.ˈteɪ.ʃən/ /; / /pə.mjʊ.ˈteɪ.ʃən/ (BrE)
each of the possible ways in which a set of things can be arranged or ordered; the act or result of arranging elements in a group

Examples

A permutation of numbers
The code uses a complex permutation of letters and numbers to ensure security.

Trying every permutation of the puzzle pieces took him all afternoon.
Each new permutation of the list created a different result in the experiment.
Mathematicians study the permutation of objects to understand probability.
The number of possible permutations of these letters is 120.
