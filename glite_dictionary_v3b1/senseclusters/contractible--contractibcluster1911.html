contractible

adjective
can be shrunk to point
9.9
Pronunciation: / kənˈtræktəbəl / (AmE), / kənˈtræktɪbəl / (BrE)
able to be continuously reduced to a single point, especially in topology.

Examples

Contractible space
In algebraic topology, a sphere is not contractible but a disk is.
The mathematician explained why the space was contractible using a simple deformation.
If a space is contractible, it has the same homotopy type as a point.

Related Terms

  • synonym: collapsible - able to be folded or reduced in size
  • antonym: noncontractible - not able to be continuously reduced to a point
