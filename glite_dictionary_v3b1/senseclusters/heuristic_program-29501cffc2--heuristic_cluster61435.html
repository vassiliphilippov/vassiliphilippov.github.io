heuristic program

noun
software using heuristic rules to solve problems
9.9 8.4
Pronunciation: / hjuˈrɪstɪk ˈproʊɡræm / (AmE), / hjʊəˈrɪstɪk ˈprəʊɡræm / (BrE)
a computer program that uses heuristic methods or rules of thumb to find solutions or make decisions, often when an exact solution is impractical

Examples

A heuristic program can help diagnose medical conditions with limited data.
Many chess computers rely on a heuristic program rather than trying every possible move.
The university developed a new heuristic program to optimize delivery routes.
