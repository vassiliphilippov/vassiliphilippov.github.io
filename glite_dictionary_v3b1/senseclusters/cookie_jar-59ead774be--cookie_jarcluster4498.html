cookie jar

noun
storage for HTTP cookies

(in computing) A software object, storage area, or browser feature that holds HTTP cookies used for sessions, preferences, tracking, or authentication.

Examples

Browser cookie jar
Developers inspected the cookie jar in Chrome DevTools to debug a leaking session cookie.
The backend periodically cleared the cookie jar to remove expired cookies and protect user privacy.
She used the HTTP client's cookie jar to persist authentication cookies between requests.

Related Terms

  • synonym: HTTP cookie - a small piece of data stored by a web browser and sent to servers to manage sessions and preferences
  • synonym: cookie store - a storage mechanism in browsers or libraries for keeping HTTP cookies
