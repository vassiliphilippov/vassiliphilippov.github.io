subclass

verb
to create derived class
6.0
Pronunciation: / ˈsʌbˌklæs / (AmE), / ˈsʌbˌklɑːs / (BrE)
to create a new class in object-oriented programming that is based on an existing class and inherits its properties

Examples

To subclass a class
In Python, you can subclass a built-in class to customize its behavior.
Developers often subclass existing components to add new features in their applications.
To implement this functionality, you should subclass the base class and override its methods.

Related Terms

  • synonym: extend - to add new features to an existing class
  • synonym: inherit - to receive properties or methods from a parent class
  • synonym: override - to replace a method in a parent class with a new version in a subclass
