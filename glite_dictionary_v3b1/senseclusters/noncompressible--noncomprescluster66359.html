noncompressible

adjective
not able to be compressed
Pronunciation: / ˌnɑnkəmˈprɛsəbəl / (AmE), / ˌnɒnkəmˈprɛsəbəl / (BrE)
not able to be reduced in size by compression algorithms because it contains little or no redundant data

Examples

Noncompressible file
Because the image was noncompressible, it required more storage space than expected.
Some types of encrypted files are almost completely noncompressible by standard software.
The engineer explained that the data set was noncompressible due to its random structure.

Related Terms

  • antonym: compressible - able to be reduced in size by compression
