HSI

sym
color model abbreviation
Pronunciation: / eɪtʃ ɛs aɪ / (AmE), / eɪtʃ ɛs aɪ / (BrE)
an abbreviation for 'hue, saturation, intensity', a color model used in image processing and computer graphics

Examples

HSI color model
The HSI model is often used to separate color information from intensity in digital images.
Engineers prefer the HSI color space for certain image analysis tasks.
By converting an image to HSI, it becomes easier to adjust brightness without affecting color.

Related Terms

  • synonym: HSV - a color model using hue, saturation, and value
  • synonym: RGB - a color model using red, green, and blue components
