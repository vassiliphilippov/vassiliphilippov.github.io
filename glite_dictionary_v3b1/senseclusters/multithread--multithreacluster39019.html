multithread

adjective
using multiple threads
6.4
Pronunciation: / ˌmʌltiˈθrɛd / (AmE), / ˌmʌltiˈθrɛd / (BrE)
using or relating to multiple threads of execution in a computer program, allowing tasks to run simultaneously

Examples

Multithread software
A multithread application can handle several user requests at the same time.
Developers often design multithread systems to improve performance on modern computers.
The software was updated to include multithread processing for faster data analysis.

Related Terms

  • synonym: multithreaded - using or supporting multiple threads of execution
  • antonym: single-threaded - using only one thread of execution
