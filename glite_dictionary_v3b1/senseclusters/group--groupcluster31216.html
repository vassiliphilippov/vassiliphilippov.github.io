group

noun
mathematical set with certain properties
7.2
Pronunciation: / ɡruːp / (AmE), / ɡruːp / (BrE)
(Mathematics) A set of elements with an operation that is closed, associative, has an identity element, and in which every element has an inverse.

Examples

A math group
Every element in a group must have an inverse within the group.
If a group has only two elements, it is called a cyclic group of order two.
The set of integers under addition forms a group in mathematics.
