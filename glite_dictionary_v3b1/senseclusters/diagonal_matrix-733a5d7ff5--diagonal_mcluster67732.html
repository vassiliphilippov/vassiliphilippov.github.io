diagonal matrix

noun
square matrix with zeros off the main diagonal
Pronunciation: / daɪˈæɡənəl ˈmætrɪks / (AmE), / daɪˈaɡənəl ˈmatrɪks / (BrE)
a square matrix in which all elements not on the main diagonal are zero

Examples

A diagonal matrix
A diagonal matrix is easy to invert if none of its diagonal entries are zero.
The eigenvalues of a diagonal matrix are simply its diagonal elements.
When expressed as a diagonal matrix, the computation becomes much simpler.

Related Terms

  • degree lower: identity matrix - a diagonal matrix with all diagonal elements equal to one
  • degree lower: scalar matrix - a diagonal matrix with all diagonal elements equal to the same scalar
