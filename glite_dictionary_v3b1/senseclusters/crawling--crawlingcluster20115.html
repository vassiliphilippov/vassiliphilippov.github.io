crawling

noun
automated web discovery
4.9
Pronunciation: /ˈkrɔː.lɪŋ/ /; / /ˈkrɑː.lɪŋ/ (AmE), /ˈkrɔː.lɪŋ/ (BrE)
(in computing, especially web search and SEO) the automated process by which programs called web crawlers browse the internet to discover, fetch, and report pages for indexing.

Examples

Web crawling
Search engines use crawling to discover new pages and keep their indexes up to date.

Good site structure speeds up crawling, helping pages appear faster in search results.
