bookmarklet

noun
JavaScript browser tool
7.0
Pronunciation: / ˈbʊkmɑːrkˌlɛt / (AmE), / ˈbʊkmɑːkˌlɛt /; / ˈbʊkmɑːkˌlɪt / (BrE)
a small program (usually JavaScript code) saved as a browser bookmark that performs a task on the current web page when clicked
