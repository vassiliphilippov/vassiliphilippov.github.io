SSH

sym
secure remote protocol
5.4
Pronunciation: / ɛs ɛs eɪtʃ / (AmE), / ɛs ɛs eɪtʃ / (BrE)
a secure network protocol used to remotely access and manage computers over a network

Examples

SSH remote login
System administrators use SSH to securely log into remote servers from anywhere in the world.
You should always use SSH instead of Telnet to protect your data during remote access.
After installing the software, connect to your server using SSH for secure management.

Related Terms

  • synonym: FTP - a protocol for transferring files between computers over a network
  • antonym: Telnet - an older, less secure protocol for remote computer access
