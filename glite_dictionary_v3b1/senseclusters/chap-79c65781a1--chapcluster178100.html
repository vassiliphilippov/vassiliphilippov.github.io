CHAP

noun
network authentication protocol
8.7
Pronunciation: /tʃæp/ (AmE), /tʃæp/ (BrE)
(computing) An authentication protocol (Challenge-Handshake Authentication Protocol) used to verify the identity of a remote device or user over network links such as PPP or VPN by using a challenge–response handshake.

Examples

CHAP authentication protocol
The remote access server supports CHAP authentication as part of its PPP setup.
During the VPN setup, enable CHAP only if the network requires stronger verification.
Investigators found repeated CHAP failures in the logs when the client credentials were wrong.

Related Terms

  • synonym: PAP - an alternative PPP authentication protocol (Password Authentication Protocol)
  • synonym: authentication - the process of verifying the identity of a user or device
