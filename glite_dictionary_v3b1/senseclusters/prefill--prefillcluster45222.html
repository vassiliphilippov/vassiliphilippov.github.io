prefill

noun
default entry
4.4
Pronunciation: /ˈpriː.fɪl/ /; / /ˈprɪ.fɪl/ (AmE), /ˈpriː.fɪl/ /; / /ˈprɪ.fɪl/ (BrE)
a value or entry that is automatically placed in a field or form before the user provides input, often as a default or suggestion

Examples

Form prefill value
The registration form included a prefill for the country based on the user's location.
If the prefill is incorrect, you can always change the information before submitting.
Many online surveys use a prefill to speed up the process for returning users.

Related Terms

  • synonym: default - a preselected option or value used when no alternative is specified
  • synonym: placeholder - a value or text shown in a field to indicate expected input
