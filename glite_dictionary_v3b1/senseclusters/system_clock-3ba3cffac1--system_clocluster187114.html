system clock

noun
clock that tracks time in computers
1.5 2.9
Pronunciation: / ˈsɪstəm klɑk / (AmE), / ˈsɪstəm klɒk / (BrE)
A time-of-day clock in a computer system that keeps track of the current time and date.

Examples

Computer system clock
If your computer's system clock is incorrect, you may have trouble connecting to online services.
The system clock needs to be accurate to ensure security certificates are valid.
When you reset the system clock, it can affect how programs schedule tasks.
