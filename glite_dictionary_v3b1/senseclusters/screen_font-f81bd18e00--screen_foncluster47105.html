screen font

noun
font displayed on a computer screen
1.0
Pronunciation: / skrin fɑnt / (AmE), / skriːn fɒnt / (BrE)
A font or typeface as it appears on a computer or digital display screen.

Examples

A screen font style
Many websites use sans-serif as their default screen font for better legibility.
The document may appear different if the screen font doesn't match the printer font.
You can change the screen font settings to make the text easier to read.

Related Terms

  • antonym: printer font - font used by a printer or in printed output
