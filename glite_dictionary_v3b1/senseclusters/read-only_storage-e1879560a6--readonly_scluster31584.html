read-only storage

noun
memory that can be read but not changed
3.8 3.5
Pronunciation: /riː ˈoʊn.li ˈstɔːr.ɪdʒ/ /; / /rɛd ˈoʊn.li ˈstɔːr.ɪdʒ/ (AmE), /riː ˈəʊn.li ˈstɔː.rɪdʒ/ /; / /rɛd ˈəʊn.li ˈstɔː.rɪdʒ/ (BrE)
(computing) Memory that can be accessed and read but cannot be modified or written to.

Examples

Read-only storage memory
Many embedded devices use read-only storage to protect their operating system from being altered.
Some critical system files are kept in read-only storage to prevent accidental modification.
The program was loaded directly from read-only storage, ensuring its integrity.

Related Terms

  • synonym: ROM - type of non-volatile memory that is read-only
  • antonym: writable storage - memory that can be both read and written to
