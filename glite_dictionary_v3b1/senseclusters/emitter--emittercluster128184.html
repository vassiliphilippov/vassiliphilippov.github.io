emitter

noun
software event source
Pronunciation: / ɪˈmɪtər / (AmE), / ɪˈmɪtə / (BrE)
(in computing) a software component that generates and broadcasts events or messages that other parts of a program can listen to or handle.

Examples

Event emitter
The emitter fired a 'data' event each time the stream received a new chunk.
In Node.js, developers create a new emitter to broadcast custom events across modules.
The logging system's emitter published error notifications that monitoring tools could record and display.

Related Terms

  • synonym: event source - a component that creates or originates events in a program
  • synonym: publisher - a component that publishes messages or events to subscribers
