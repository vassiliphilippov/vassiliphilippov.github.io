CGM

noun
graphics file format
8.3
Pronunciation: / si dʒi ˈɛm / (AmE), / siː dʒiː ˈɛm / (BrE)
a standard computer graphics file format for storing two- and three-dimensional vector images and graphical data (Computer Graphics Metafile)

Examples

CGM image file
The designer saved the diagram as a CGM to ensure it could be scaled without losing quality.
Many engineering applications support importing and exporting CGM files for technical drawings.
She converted the image from CGM to SVG for use on the website.

Related Terms

  • synonym: SVG - a widely used vector graphics file format (Scalable Vector Graphics)
  • antonym: bitmap - a graphics file format using pixels rather than vectors
