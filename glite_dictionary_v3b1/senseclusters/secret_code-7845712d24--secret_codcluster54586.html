secret code

noun
access code or PIN
3.5 0.03/m
Pronunciation: / sikr59t ko8ad / (AmE), / sikr6at k598ad / (BrE)
a numeric or alphanumeric sequence used to prove identity and gain access to a device, account, or service (for example, a PIN or passcode).

Examples

Bank secret code
To access her banking app, she typed her secret code and waited for the confirmation.
At the ATM he realized he had forgotten his secret code and had to reset it online.
Parents often tell children not to share their secret codes for safety and privacy reasons.

Synonyms

  • passcode
    3.1 0.47/m
    a passcode is a short numeric or alphanumeric sequence used to unlock devices, while a secret code proves identity to gain access.
  • access code
    3.1 0.74/m
    an access code is a string of characters or digits entered to open a system, while a secret code proves your identity for access.
  • passkey
    4.6 0.04/m
    a passkey is a special code or digital credential used for secure login, while a secret code is usually a numeric or alphanumeric password.

Related Terms

  • synonym: PIN - a personal identification number used for authentication, especially at ATMs
  • synonym: passcode - a short code used to unlock a device or app
  • synonym: password - a secret word or string of characters used to access an account or system
