ARC

noun
automatic memory-management technique

(computing) Abbreviation of Automatic Reference Counting, a memory-management technique that automatically tracks and frees objects when they are no longer needed.

Examples

ARC memory management
In modern iOS development, ARC automatically frees unused objects, simplifying memory management for programmers.
After enabling ARC, he removed many manual retain and release calls from the Objective-C code.
Swift uses ARC to manage object lifetimes, which reduces memory leaks in most apps.

Related Terms

  • synonym: automatic memory management - a system that reclaims unused memory or objects without requiring manual deallocation
