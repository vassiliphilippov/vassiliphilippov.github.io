Capsicum

noun
capability-based sandboxing framework
9.9
Pronunciation: /ˈkæp.sɪ.kəm/ (AmE), /ˈkæp.sɪ.kəm/ (BrE)
(computing) A capability-based security framework and API (originally in FreeBSD) that provides sandboxing and fine-grained access control for untrusted code.

Examples

Capsicum security framework
The application runs in a Capsicum sandbox to limit filesystem access.
Developers used Capsicum capabilities to prevent the plugin from reading sensitive configuration files.
During the audit, they discovered the server's Capsicum profile blocked unauthorized network sockets.

Related Terms

  • synonym: capability-based security - a security model using unforgeable tokens (capabilities) to grant code restricted access to resources
  • synonym: sandbox - an isolated environment that restricts program access to the rest of the system
