strict rule

noun
regulation enforced rigidly
3.4 0.62/m
Pronunciation: / str…™kt ruÀêl / (AmE), / str…™kt ruÀêl / (BrE)
a regulation or requirement that is enforced rigidly, with little or no flexibility

Examples

A strict rule
Because of health concerns, the school has a strict rule about washing hands before lunch.
The company enforces a strict rule against using mobile phones during meetings.
There is a strict rule about wearing seatbelts in this state.

Synonyms

  • strict measure
    4.8 0.03/m
    a strict measure is an action enforced to keep order or safety, while a strict rule is a fixed regulation with little flexibility.
  • bright-line rule
    7.8 0.35/m
    a bright-line rule is a clear, simple test used in courts or policies, while a strict rule stresses rigid enforcement over clarity.
