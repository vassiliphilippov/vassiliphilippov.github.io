listening

noun
server/socket awaiting connections
6.5
Pronunciation: / ˈlɪsənɪŋ / (AmE), / ˈlɪsənɪŋ / (BrE)
(in computing) the state of a server process or network socket that is bound to a port and waiting to accept incoming network connections.

Examples

Server listening port
The server remained listening on port 443 for incoming client connections.
After the crash, the administrator checked whether the listening socket had been re-established automatically.
You can use netstat to see all processes that are currently listening on network ports.

Related Terms

  • synonym: awaiting connections - the condition of waiting for incoming network connections
