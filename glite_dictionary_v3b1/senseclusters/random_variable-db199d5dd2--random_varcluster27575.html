random variable

noun
mathematical variable with random outcomes
Pronunciation: / ˈrændəm ˈvɛriəbl̩ / (AmE), / ˈrændəm ˈvɛːriəbl / (BrE)
A quantity whose possible values are outcomes of a random phenomenon and which takes on those values according to a probability distribution.

Examples

A random variable outcome
Each toss of a fair coin can be modeled by a discrete random variable.
In statistics, a random variable represents the outcome of a random process.
The values that a random variable can take are determined by its probability distribution.

Related Terms

  • antonym: deterministic variable - a variable that is not random and has a fixed value
  • synonym: stochastic variable - another term for 'random variable' in mathematics
