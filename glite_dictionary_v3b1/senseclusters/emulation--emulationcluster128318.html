emulation

noun
computer system imitation technique
5.6 5.4
Pronunciation: / ˌɛm.jəˈleɪ.ʃən / (AmE), / ˌɛm.jʊˈleɪ.ʃən / (BrE)
(Computing) The technique by which one computer system imitates or replicates the functions of another, enabling the user to run software or use features originally designed for a different system.

Examples

Computer emulation
Developers rely on emulation to test apps on different operating systems before release.
The gaming console supports emulation of classic arcade systems, letting users play older titles.
With software emulation, a Windows program can run on a Mac computer without modification.

Related Terms

  • synonym: simulation - the imitation of a situation or process, often for training or testing
