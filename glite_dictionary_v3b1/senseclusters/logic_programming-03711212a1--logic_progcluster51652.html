logic programming

noun
computer programming using formal logic rules
4.0 5.4
Pronunciation: /ˈlɑː.dʒɪk ˈproʊ.græm.ɪŋ/ (AmE), /ˈlɒ.dʒɪk ˈprəʊ.grəm.ɪŋ/ (BrE)
