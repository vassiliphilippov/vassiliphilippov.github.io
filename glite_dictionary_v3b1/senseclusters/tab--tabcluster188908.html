tab

noun
browser page (UI element)
2.9
Pronunciation: / tæb / (AmE), / tæb / (BrE)
a single page or document opened in a web browser or application, represented by a labeled area (a tab) in the window that you click to switch between pages.

Examples

A browser tab
She opened a new tab to compare flight prices on two booking sites.
Before the meeting, he closed several tabs to reduce memory use on his laptop.
You can pin a tab in your browser so the website opens every time you start it.

Related Terms

  • synonym: panel - a distinct area of a user interface that displays content; similar in function to a tab in many applications.
