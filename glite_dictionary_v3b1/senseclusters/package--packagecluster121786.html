package

noun
set of software programs and documentation
Pronunciation: / ˈpækɪdʒ / (AmE), / ˈpækɪdʒ / (BrE)
(computing) a set of related programs, procedures, and documentation offered as a unit to perform specific tasks on a computer system.

Examples

Software package
After installing the latest package, several new features became available in the application.
Many developers use this package to process large data sets efficiently.
The IT department purchased a package that includes antivirus protection and backup tools.
