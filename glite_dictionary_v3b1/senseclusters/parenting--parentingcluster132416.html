parenting

noun
object linking in graphics
Pronunciation: / ˈpɛrəntɪŋ / (AmE), / ˈpɛːrəntɪŋ / (BrE)
the process in computer graphics or animation of linking one object to another so the child object moves, rotates, or scales with the parent object

Examples

Parenting objects together
By parenting the lamp to the table, any movement of the table also moves the lamp in the scene.
The animator used parenting to attach the character's hat so it followed every head movement.
Proper parenting of objects can save time when creating complex animations in 3D software.

Related Terms

  • synonym: grouping - combining multiple objects so they act as a single unit
  • antonym: unparenting - removing the parent-child relationship between objects
