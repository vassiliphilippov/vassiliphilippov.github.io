traversal

noun
path/directory traversal vulnerability
9.0
Pronunciation: / trəˈvɜrsəl / (AmE), / trəˈvɜːsəl / (BrE)
(in computing, especially cybersecurity) a vulnerability or attack in which an attacker manipulates file-path input to access files or directories outside the intended location (also called path or directory traversal).

Examples

Directory traversal attack
A directory traversal vulnerability allowed attackers to read configuration files on the web server.
The security team found a traversal flaw in the upload form that exposed sensitive user data.
Patching the server eliminated the traversal exploit that had been used to access password files.

Related Terms

  • synonym: directory traversal attack - an attack that uses traversal techniques to read or modify restricted server files
  • synonym: path traversal - a cyberattack that manipulates file paths to access files outside the intended directory
