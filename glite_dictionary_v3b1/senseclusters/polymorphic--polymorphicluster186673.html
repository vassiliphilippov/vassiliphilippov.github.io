polymorphic

adjective
changing form to avoid detection (computing)
8.8
Pronunciation: /pɑː.lɪ.ˈmɔːr.fɪk/ (AmE), /pɒl.ɪ.ˈmɔː.fɪk/ (BrE)
(in computing/security) describing software, code, or malware that repeatedly changes its internal form, code, or signature (usually by encryption, obfuscation, or mutation) in order to avoid detection by antivirus and other defensive tools.

Examples

Polymorphic malware
Security researchers analyzed a polymorphic malware strain that changed its signature after every infection.
The antivirus failed to detect the polymorphic sample because its code mutated on each run.
Analysts use emulation and heuristics to identify polymorphic binaries that try to hide malicious payloads.

Related Terms

  • synonym: metamorphic - in malware, code that rewrites itself so its form changes substantially between copies; closely related to polymorphic techniques
  • synonym: self-modifying - code that alters its own instructions while running; often used to evade analysis
