(computing) a single operation or command written as part of a computer program, typically executed by a computer's processor.

Examples

A computer instruction
An optimized compiler generates efficient machine instructions for faster execution.
Each instruction in the source code is executed sequentially by the processor.
The program crashed due to an invalid instruction in line 42.

Synonyms

  • statement
    4.6 1.52/m
    a statement is a line of code that tells the computer to do a specific action, while instruction is a single processor operation executed.
  • program line
    4.1 0.01/m
    a program line is one written line of source code, while instruction is the actual operation the processor runs.
  • command
    3.7 19/m
    a command is an order given to software or a shell, while instruction is a low-level operation executed by the processor.
