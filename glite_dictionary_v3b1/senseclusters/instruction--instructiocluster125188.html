instruction

noun
a line of code in a computer program
Pronunciation: / ɪnˈstrʌkʃən / (AmE), / ɪnˈstrʌkʃən / (BrE)
(computing) a single operation or command written as part of a computer program, typically executed by a computer's processor.

Examples

A computer instruction
An optimized compiler generates efficient machine instructions for faster execution.
Each instruction in the source code is executed sequentially by the processor.
The program crashed due to an invalid instruction in line 42.

Related Terms

  • synonym: command - a directive to a computer to perform a specific task
