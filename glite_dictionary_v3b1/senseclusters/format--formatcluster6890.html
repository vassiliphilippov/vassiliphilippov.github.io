format

verb
arrange data for storage/display (computing)
Pronunciation: / ˈfɔːrˌmæt /; / ˈfɔrˌmæt / (AmE), / ˈfɔːmæt / (BrE)
to arrange or organize text, images, or data into a particular layout or structure, especially for display, printing, or storage

Examples

To format a document
Make sure you format the disk before copying any files to it.
Please format the document according to the company's style guide.
The program can format your data for easier analysis.
Please format the document before sending it to the printer.
She formatted the report to make it easier to read.
The computer can automatically format your text based on the selected style.

Related Terms

  • antonym: disorganize - to make something lacking in order or organization
  • synonym: structure - to organize or arrange parts in a particular way
