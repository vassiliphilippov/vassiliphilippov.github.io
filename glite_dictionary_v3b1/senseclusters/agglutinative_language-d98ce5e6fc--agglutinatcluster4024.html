agglutinative language

noun
language type using joined morphemes
9.9 8.2
Pronunciation: / əˌɡluː.tɪˈneɪ.tɪv ˈlæŋ.ɡwɪdʒ / (AmE), / əˌɡluː.tɪˈneɪ.tɪv ˈlaŋ.ɡwɪdʒ / (BrE)
a language in which words are formed by stringing together distinct morphemes, each with a separate meaning or grammatical function

Examples

Agglutinative language structure
In an agglutinative language, each grammatical function is typically marked by its own distinct suffix.
Linguists often compare Finnish and Japanese as both are agglutinative languages.
Turkish is a well-known example of an agglutinative language with complex word forms built from many affixes.

Related Terms

  • antonym: analytic language - a language that primarily uses single morphemes per word, often with grammatical relationships shown by word order or helper words
  • antonym: fusional language - a language in which morphemes are fused together so that a single morpheme may express multiple grammatical categories
