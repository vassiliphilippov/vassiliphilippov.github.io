log line

noun
single line in a log file
3.7
Pronunciation: / lɑɡ laɪn / (AmE), / lɒɡ laɪn / (BrE)
A single line in a computer or system log file that records an event, message, or timestamp.

Examples

Error log line
The developer checked the log line to find when the error first occurred during the outage.
All log lines included a timestamp, the process ID, and a short message about the event.
A single log line showed that the service restarted unexpectedly at midnight.

Related Terms

  • synonym: log entry - a recorded item in a log file, often equivalent to a log line
  • synonym: log message - the textual message portion of a log line describing an event or status
