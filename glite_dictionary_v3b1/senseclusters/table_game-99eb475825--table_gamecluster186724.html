table game

noun
game played on a table
1.0
Pronunciation: / ˈteɪ.bəl ɡeɪm / (AmE), / ˈteɪ.bəl ɡeɪm / (BrE)
a game that is played on a table, such as a board or card game, rather than on the floor or outdoors

Examples

A table game of chess
Chess is one of the most famous table games in the world.
Many casinos offer various table games such as poker and blackjack.
We spent the evening enjoying a new table game with friends.

Related Terms

  • antonym: video game - a game played electronically on a video screen
