run

noun
regular trip by vehicle on a route
Pronunciation: / rʌn / (AmE), / rʌn / (BrE)
a regular journey or route taken by a vehicle, or a short trip for a purpose

Examples

A bus run
Our ferry makes three runs between the islands each afternoon.
She takes the last run home from work if it's too late to drive.
The morning run to the city departs at 7:15 every day.
He went on a run to the hardware store to get some tools.
Let's take a run downtown before dinner to pick up the tickets.
She made a quick run to the grocery store for some milk.

  • synonym: trip - a journey or excursion, especially for pleasure
