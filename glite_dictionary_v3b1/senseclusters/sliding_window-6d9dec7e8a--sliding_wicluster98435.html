sliding window

noun
window that opens by sliding horizontally
Pronunciation: / ˈslaɪdɪŋ ˈwɪndoʊ / (AmE), / ˈslaɪdɪŋ ˈwɪndəʊ / (BrE)
a window that opens by moving horizontally along a frame, rather than swinging open

Examples

An old sliding window
He struggled to open the old sliding window because it was stuck.
The new apartment features a large sliding window that overlooks the garden.
We decided to install a sliding window in the kitchen for better ventilation.
