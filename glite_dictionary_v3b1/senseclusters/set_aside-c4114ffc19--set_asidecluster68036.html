set aside

verb
reserve or allocate for a purpose
3.1
Pronunciation: / sɛt əˈsaɪd / (AmE), / sɛt əˈsaɪd / (BrE)
to reserve or allocate something, such as time or money, for a specific purpose or person

Examples

To set aside money
Each year, I set aside a portion of my salary for savings.
She always sets aside some time on Sundays to call her parents.
The government decided to set aside funds for disaster relief.

Related Terms

  • synonym: allocate - to distribute resources for a specific purpose
  • synonym: earmark - to designate something for a particular purpose
