DITA

noun
XML standard for technical documentation
9.9 0.2/m
Pronunciation: / ÀàdiÀêt…ô / (AmE), / ÀàdiÀêt…ô / (BrE)
(computing) an XML-based architecture for topic-based, structured authoring and publishing of technical documentation, used to create reusable content modules.

Examples

DITA technical documentation
Many software companies adopt DITA to standardize their help documentation and technical manuals.
She learned to write modular topics in DITA so content could be reused across multiple guides.
The documentation team converted the legacy files into DITA format for easier translation and publishing.
