class Bryopsida

noun
class of true mosses
9.9
Pronunciation: / klæs braɪˈɑpsɪdə / (AmE), / klɑːs braɪˈɒpsɪdə / (BrE)
a class of bryophyte plants known as true mosses, characterized by leafy gametophytes and including several orders such as Bryales and Sphagnales

Examples

Class Bryopsida moss plants
Members of class Bryopsida are distinguished by their leafy gametophytes rather than thalloid structures.
Scientists discovered a new order of moss within the class Bryopsida in the temperate rainforest.
The class Bryopsida includes most of the moss species found worldwide.
