documentation

noun
written material describing operation or use
Pronunciation: / ˌdɑkjəˌmɛnˈteɪʃən / (AmE), / ˌdɒkjʊmɛnˈteɪʃən / (BrE)
written material that provides official information or instructions about how to operate or use something, especially software or machinery

Examples

Software documentation
She spent the afternoon writing documentation for the new features in the program.
The software comes with extensive documentation to help users install and configure it.
Without proper documentation, troubleshooting technical issues can be very difficult.

Related Terms

  • synonym: manual - a book giving instructions or information
