scraper

noun
software that extracts website data
6.6
Pronunciation: / ˈskreɪpər / (AmE), / ˈskreɪpə / (BrE)
(computing) a software program that automatically extracts structured data from websites by downloading and parsing pages or APIs.

Examples

Web scraper
The marketing team used a scraper to collect product prices from several competitor websites.
After the change, the site blocked the scraper by refusing requests without a valid API key.
Developers prefer writing a custom scraper when public APIs don't provide the needed detailed information.

Related Terms

  • synonym: data extractor - software that extracts specific information from larger datasets or files
  • synonym: web scraper - a program specifically designed to collect data from web pages
