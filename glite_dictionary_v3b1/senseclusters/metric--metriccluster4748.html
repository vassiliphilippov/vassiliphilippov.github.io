metric

noun
function defining distance in mathematics
3.0
Pronunciation: / ˈmɛtrɪk / (AmE), / ˈmɛtrɪk / (BrE)
(Mathematics) A function that defines a distance between each pair of points in a topological space.

Examples

Mathematical metric
A space equipped with a metric is called a metric space.
Different choices of metric can lead to different notions of distance in geometry.
In topology, the Euclidean metric assigns the usual distance between two points in space.

Related Terms

  • synonym: distance function - a function that describes a distance between elements in a space
