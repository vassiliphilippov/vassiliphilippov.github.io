hacky code

noun
inelegant or poorly written computer code
4.4 6.6
Pronunciation: /ˈhæk.iː koʊd/ (AmE), /ˈhæk.iː kəʊd/ (BrE)
(informal) Computer code that is poorly written, inelegant, or created as a quick and temporary solution rather than following good programming practices.
