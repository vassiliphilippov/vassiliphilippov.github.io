non-volatile storage

noun
memory/storage retaining data without power
6.8 4.1
Pronunciation: /nɑːn.ˈvɑːl.ə.təl ˈstɔːr.ɪdʒ/ /; / /nə.ˈvɑːl.ə.təl ˈstɔːr.ɪdʒ/ (AmE), /nɒn.ˈvɒl.ə.taɪl ˈstɔː.rɪdʒ/ /; / /nɒn.ˈvɒl.ə.təl ˈstɔː.rɪdʒ/ /; / /nə.ˈvɒl.ə.taɪl ˈstɔː.rɪdʒ/ (BrE)
Computer memory or storage that retains data even when the power is turned off.

Examples

Non-volatile storage device
Critical data should be saved to non-volatile storage to prevent loss during a power outage.
Flash memory is often used as non-volatile storage in smartphones and tablets.
Solid-state drives are a popular form of non-volatile storage in modern computers.

Related Terms

  • antonym: volatile storage - computer memory that loses data when power is lost
