toolbox

noun
collection of software tools
3.7
Pronunciation: /ˈtuːl.bɔːks/ /; / /ˈtuːl.bɑːks/ (AmE), /ˈtuːl.bɒks/ (BrE)
(computing) a packaged collection of software tools, functions, or utilities provided within a program or platform to perform related tasks.

Examples

Software toolbox

The web platform offers a developer toolbox with APIs, debugging tools, and deployment scripts.
