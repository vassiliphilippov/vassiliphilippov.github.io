toolbox

noun
collection of software tools
3.7 0.55/m
Pronunciation: / tuclcbaks / (AmE), / tuclcbks / (BrE)
(computing) a packaged collection of software tools, functions, or utilities provided within a program or platform to perform related tasks.

Examples

Software toolbox
The new data visualization toolbox provides ready-made charts and cleaning routines for beginners.
Engineers installed several MATLAB toolboxes to add signal-processing and control-system functions.
The web platform offers a developer toolbox with APIs, debugging tools, and deployment scripts.
