grep

noun
search utility
8.1
Pronunciation: / ɡrɛp / (AmE), / ɡrɛp / (BrE)
a command-line program used to search files or text for lines matching a specified pattern, often using regular expressions

Examples

The grep command
Many programmers use grep to quickly find specific lines in large codebases.
The manual explains how to use grep with different search patterns.
She ran grep to filter out all lines containing the word 'error' from the log file.
