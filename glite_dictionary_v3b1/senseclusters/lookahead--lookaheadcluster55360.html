lookahead

noun
parser foresight
Pronunciation: / ˈlʊkəˌhɛd / (AmE), / ˈlʊkəˌhɛd / (BrE)
in computing, the ability of a parser or lexer to examine input symbols beyond the current position to help resolve ambiguities

Examples

Parser lookahead
The parser required a two-token lookahead to decide which grammar rule applied.
Increasing the lookahead can make the parsing process more accurate but also more complex.
Some programming languages need only a single-symbol lookahead for efficient parsing.

Related Terms

  • antonym: backtracking - reverting to a previous state in parsing to try a different path
