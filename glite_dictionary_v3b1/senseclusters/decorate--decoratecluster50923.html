decorate

verb
apply a decorator to code
Pronunciation: / ˈdɛkəˌreɪt / (AmE), / ˈdɛkəreɪt / (BrE)
(in programming) to modify or extend the behavior of a function, method, or class by applying a decorator — a callable or construct that wraps or alters it.

Examples

To decorate a function
In Python, you can decorate a function with @log to automatically record every call and its arguments.
She decorated the class with a validation decorator so instances validate input when created.
They avoided decorating simple functions to keep the code clear and easy to test.

Related Terms

  • synonym: modify - to change the behavior or properties of code or an object
  • synonym: wrap - to surround a function or object with another function that adds behavior
