von Neumann machine

noun
digital computer based on von Neumann architecture
Pronunciation: / vɑn ˈnɔɪmən məˈʃin / (AmE), / vɒn ˈnɔɪmən məˈʃiːn / (BrE)
a digital computer based on the architecture proposed by John von Neumann, featuring stored programs and serial processing of instructions

Examples

von Neumann machine architecture
Most modern computers are based on the von Neumann machine concept introduced in the 1940s.
Students learn about the fundamental parts of a von Neumann machine in their computer architecture courses.
The von Neumann machine revolutionized computer design by allowing instructions and data to be stored in the same memory.

Related Terms

  • synonym: von Neumann architecture - the basic design model for most modern computers, introduced by John von Neumann
