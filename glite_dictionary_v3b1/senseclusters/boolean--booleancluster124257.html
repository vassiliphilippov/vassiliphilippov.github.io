boolean

noun
true-or-false data type
4.1
Pronunciation: / ˈbuːliən / (AmE), / ˈbuːliən / (BrE)
a data type in computing that can have only two possible values: true or false

Examples

A boolean variable
In most programming languages, a boolean variable can only be true or false.
You should use a boolean to check if the user is logged in.
The function returns a boolean indicating whether the file exists.

Related Terms

  • synonym: binary - a system using only two values, typically 0 and 1
