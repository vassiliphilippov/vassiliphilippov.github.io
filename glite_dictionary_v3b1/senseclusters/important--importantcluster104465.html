important

adjective
CSS keyword that forces style priority
5.7 0.89/m
Pronunciation: / …™mÀàp…îrt…ônt / (AmE), / …™mÀàp…îÀêt…ônt / (BrE)
(in web design, CSS) a keyword used after a property value (written as !important) to give that declaration priority so it overrides other conflicting style rules.

Examples

CSS !important
When two rules conflict, add !important to the CSS property to force the browser to use it.
Because of maintainability issues, developers advise avoiding !important except for quick debugging fixes.
In his styleguide he explained why adding !important makes a rule override even specific selectors.
