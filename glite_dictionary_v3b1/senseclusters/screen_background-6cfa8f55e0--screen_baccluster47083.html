screen background

noun
area behind icons/windows on screen
1.0 2.7
Pronunciation: /skriːn ˈbæk.graʊnd/ (AmE), /skriːn ˈbæk.graʊnd/ (BrE)
in computing, the part of a display screen in a graphical user interface against which icons, folders, and windows are shown
