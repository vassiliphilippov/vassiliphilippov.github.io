cryptonym

noun
secret code name
Pronunciation: / ˈkrɪptəˌnɪm / (AmE), / ˈkrɪptəˌnɪm / (BrE)
a secret or code name used to hide a person's identity or the purpose of something

Examples

A cryptonym name
The agent was known only by the cryptonym 'Falcon' during the mission.
Each project was assigned a unique cryptonym to keep its details confidential.
The journalist used a cryptonym to protect their identity while reporting sensitive news.

Related Terms

  • synonym: pseudonym - a fictitious name used by someone, especially an author
  • antonym: real name - a person's actual name
