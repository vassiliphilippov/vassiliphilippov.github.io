spin up

verb
start or initialize a system or service
Pronunciation: / spɪn ʌp /; / spɪn / (AmE), / spɪn ʌp /; / spɪn / (BrE)
to start or initialize a computer system, service, or resource so it is ready for use (especially in computing)

Examples

To spin up a server
Before testing, you need to spin up the development environment.
It only takes a few minutes to spin up a virtual machine in the cloud.
The IT team will spin up a new server to handle the increased traffic.
Cloud engineers can spin up a new virtual instance in minutes for testing.
She spun up a temporary database container to reproduce the customer's error during debugging.
The deployment script automatically spins up test environments whenever a developer opens a pull request.
