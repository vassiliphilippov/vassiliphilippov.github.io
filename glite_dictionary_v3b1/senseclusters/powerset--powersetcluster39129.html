powerset

noun
all subsets of a set
9.8
Pronunciation: / ˈpaʊərˌsɛt / (AmE), / ˈpaʊəˌsɛt / (BrE)
the set containing all possible subsets of a given set, including the empty set and the set itself

Examples

The powerset of numbers
The powerset of a set with three elements contains eight different subsets.
In mathematics, the powerset is used to study the properties of sets and their relationships.
To solve the problem, she listed every subset in the powerset of the original set.

  • synonym: set - a collection of distinct objects considered as a whole
  • synonym: subset - a set whose elements are all contained in another set
