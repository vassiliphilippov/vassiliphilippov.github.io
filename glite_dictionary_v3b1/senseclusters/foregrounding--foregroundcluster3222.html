foregrounding

noun
execution of a priority computer process
9.9
Pronunciation: / ˈfɔrˌɡraʊn.dɪŋ / (AmE), / ˈfɔːˌɡraʊn.dɪŋ / (BrE)
The execution of a program or process in a computing system that takes precedence over others and fully occupies the system’s resources.

Examples

Foregrounding a program
After foregrounding the music player, you can interact with it directly.
Some applications support quick foregrounding for priority tasks.
The operating system allows foregrounding of a program so it can utilize most of the resources.

Related Terms

  • antonym: backgrounding - execution of a program in the background of a computing system
