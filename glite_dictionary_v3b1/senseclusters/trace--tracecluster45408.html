trace

noun
program execution log for debugging
7.2
Pronunciation: /treɪs/ (AmE), /treɪs/ (BrE)
(in computing) a recorded sequence of a program's execution events (for example function calls, errors, or timestamps) used to analyse behaviour or locate bugs.

Examples

A program trace
The debugger produced a detailed trace of function calls that led to the program crash.
Engineers reviewed the server trace to see which request caused the memory leak during testing.
She enabled system trace logging so they could reproduce the error with exact timestamps.

Related Terms

  • synonym: log - a record of events, messages, or actions produced by a program or system
  • synonym: stack trace - a list of active function calls showing where an error occurred in a program
