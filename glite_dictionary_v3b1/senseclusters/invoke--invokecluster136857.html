invoke

verb
call or execute a function (computing)
5.5
Pronunciation: /ɪn.ˈvoʊk/ /; / /ɪn.ˈvəʊk/ (AmE), /ɪn.ˈvəʊk/ (BrE)
(in computing) to call or run a function, method, or procedure so that its code executes.

Examples

To invoke a function
In many programming languages, you invoke a function by writing its name followed by parentheses and arguments.
The API documentation showed how to invoke the authentication method asynchronously to avoid blocking the main thread.
When the event fires, the framework automatically invokes any registered callback functions in order of priority.

Related Terms

  • synonym: call - to request that a function or routine run; commonly used interchangeably with 'invoke' in programming
  • synonym: execute - to run a program or routine so that its instructions are carried out
