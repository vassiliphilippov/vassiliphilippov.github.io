double quotes

noun
quotation marks for string literals (programming)
3.2
Pronunciation: /ˈdʌb.əl kwoʊts/ (AmE), /ˈdʌb.əl kwəʊts/ (BrE)
(in programming and data formats) a pair of quotation marks used to mark the start and end of a string literal or quoted token in source code or data.

Examples

Double quotes in code
In Java, double quotes mark the start and end of every string literal in source code.
JSON requires that object keys are enclosed in double quotes to be valid across parsers.
The tutorial explained how to escape a quotation mark inside double quotes so the string remains valid.

Related Terms

  • synonym: quotation marks - marks used to indicate speech, quotations, or quoted text (includes double and single marks).
  • antonym: single quotes - a pair of single quotation marks (often used as an alternative delimiter in programming or for quotations within quotations).
