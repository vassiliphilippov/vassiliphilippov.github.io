Synonyms

  • remember
    1.0 396/m
    remember is the everyday word for keeping something in mind, while recall is a general word used in formal or neutral contexts.
  • recollect
    3.7 1.91/m
    recollect is a slightly formal word for bringing an event to mind, while recall is more common in reports or tests.
  • retrieve
    4.1 3.98/m
    retrieve is used for bringing back a stored memory or fact, while recall covers remembering in everyday speech.
  • think back
    1.9 4.52/m
    think back means actively remembering past moments, while recall can be used more generally for any act of remembering.
  • call up
    3.2 1.09/m
    call up brings a memory quickly to mind, while recall is a neutral word for remembering from the past.
  • call back
    2.3 0.18/m
    call back can mean bringing a memory to mind in a sudden way, while recall is the general word for remembering.
  • cast one's mind back
    6.3 0.05/m
    cast one's mind back asks you to deliberately remember a past scene, while recall is more neutral about remembering.
  • think
    1.0 137/m
    think can mean using memory or reasoning broadly, while recall specifically means bringing a past fact or memory to mind.
