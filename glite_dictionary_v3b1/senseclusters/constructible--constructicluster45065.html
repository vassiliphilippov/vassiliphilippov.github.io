constructible

adjective
able to be created
Pronunciation: / kənˈstrʌktəbl̩ / (AmE), / kənˈstrʌktəbl̩ / (BrE)
able to be created or instantiated, especially by a constructor or initialization process in programming.

Examples

Constructible class
The class must be constructible with no arguments to work with this library.
Only types that are constructible from a string can be used in this function.
If a type is not constructible by default, you need to provide a custom constructor.

Related Terms

  • synonym: instantiable - able to be instantiated or created as an instance
  • antonym: unconstructible - not able to be created or instantiated
