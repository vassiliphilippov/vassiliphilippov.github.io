prototypal

adjective
relating to prototype-based programming
8.5
Pronunciation: /proʊ.tə.ˈtaɪ.pəl/ (AmE), /prəʊ.tə.ˈtaɪ.pəl/ (BrE)
(in computing) relating to or using prototype-based programming or inheritance, in which objects inherit properties and behavior directly from other objects (prototypes) rather than from classes.

Examples

Prototypal inheritance
JavaScript uses prototypal inheritance, so objects can inherit properties directly from other objects.
Many developers learn prototypal patterns to better understand frameworks built on prototype-based objects.
The course introduced prototypal inheritance with hands-on exercises manipulating object prototypes and properties.

Related Terms

  • antonym: class-based - relating to a programming model where objects are instances of classes (opposite of prototype-based)
  • synonym: prototype-based - relating to a programming model where objects inherit directly from other objects
