decimal system

noun
system of counting based on ten
3.6 2.6
Pronunciation: /ˈdes.ɪ.məl ˈsɪs.təm/ /; / /ˈdes.ə.məl ˈsɪs.təm/ (AmE), /ˈdes.ɪ.məl ˈsɪs.təm/ /; / /ˈdes.ə.məl ˈsɪs.təm/ (BrE)
A system of counting or expressing numbers in which each place value is ten times the value to its right, using the digits 0 to 9.

Examples

The decimal system of numbers
Children first learn to count using the decimal system before they are introduced to other numeration systems.
Computers use binary code, not the decimal system, to store and process data.
The decimal system is the most common way of representing numbers in everyday life.
