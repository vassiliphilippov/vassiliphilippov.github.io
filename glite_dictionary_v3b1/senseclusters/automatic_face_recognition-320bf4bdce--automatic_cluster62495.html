automatic face recognition

noun
computer identification of faces for security or recognition
Pronunciation: / ˌɔtəˈmætɪk feɪs ˌrɛkəɡˈnɪʃən / (AmE), / ˌɔːtəˈmætɪk feɪs ˌrɛkəɡˈnɪʃən / (BrE)
the process of identifying or verifying a person's identity by using computer systems to scan and analyze facial features, usually by comparing them against a database of known faces

Examples

Automatic face recognition system
Many airports now use automatic face recognition to speed up the security process.
Privacy advocates are concerned about the widespread use of automatic face recognition in public spaces.
The company's new phone model features automatic face recognition for secure unlocking.

  • synonym: facial recognition - identification of people using their facial features
