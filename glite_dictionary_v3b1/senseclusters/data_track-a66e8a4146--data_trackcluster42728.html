data track

noun
circular path for storing data on magnetic disks
Pronunciation: / ˈdeɪtə træk / (AmE), / ˈdeɪtə træk / (BrE)
(computing) one of the concentric circular paths on a magnetic disk where data is recorded and read.

Examples

Data track on disk
Each data track on the disk is precisely aligned to maximize storage capacity.
The technician examined the data track for errors during the recovery process.
When a disk is formatted, every data track is initialized for future use.
