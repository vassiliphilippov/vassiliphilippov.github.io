inlining

noun
code substitution process
7.5
Pronunciation: / ˈɪnˌlaɪnɪŋ / (AmE), / ˈɪnˌlaɪnɪŋ / (BrE)
the process of replacing a function call with the actual code of the function to improve performance

Examples

Inlining a function
The compiler uses inlining to make programs run faster by avoiding unnecessary function calls.
Manual inlining of code can sometimes make debugging more difficult for developers.
Some programming languages support automatic inlining to optimize performance during compilation.

Related Terms

  • antonym: function call - an instruction to execute a function in programming
  • synonym: optimization - the process of making code run more efficiently
