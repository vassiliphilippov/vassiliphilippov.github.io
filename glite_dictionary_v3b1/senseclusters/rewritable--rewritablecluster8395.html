rewritable

adjective
able to be edited
Pronunciation: / riˈraɪtəbəl / (AmE), / riˈraɪtəbəl / (BrE)
able to be written again, edited, or changed after the original version.

Examples

Rewritable document
The document is rewritable, so you can update it whenever necessary.
Many CDs are rewritable, allowing users to erase and record new data multiple times.
Before final approval, the proposal remains rewritable for further changes.

  • antonym: permanent - not able to be changed or edited
