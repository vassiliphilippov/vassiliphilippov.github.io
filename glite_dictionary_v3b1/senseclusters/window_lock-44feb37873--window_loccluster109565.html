window lock

noun
device for locking a window shut
Pronunciation: / ˈwɪndoʊ lɑk / (AmE), / ˈwɪndəʊ lɒk / (BrE)
a device attached to a window to secure it in the closed position

Examples

A window lock device
Always check that each window lock is engaged before leaving the house.
The old window lock was broken, making it easy for the window to be opened from outside.
We installed a new window lock on every upstairs window for added security.
