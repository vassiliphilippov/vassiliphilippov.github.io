chroot

noun
Unix directory isolation
Pronunciation: / tʃeɪnʤ ruːt / (AmE), / tʃeɪnʤ ruːt / (BrE)
a Unix command or system call that changes the apparent root directory for a process, isolating it from the rest of the system

Examples

The chroot command
The administrator used chroot to create a secure environment for testing new software.
Running a server inside a chroot can help limit potential security risks.
Before installing the package, she entered a chroot to avoid affecting the main system.

Related Terms

  • synonym: jail - a security mechanism that limits a program's access to the system
  • synonym: sandbox - a restricted environment for running programs safely
