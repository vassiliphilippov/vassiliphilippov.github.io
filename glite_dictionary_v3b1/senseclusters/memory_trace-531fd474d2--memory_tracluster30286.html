memory trace

noun
log of memory events
Pronunciation: / c8m5bm59ri tre6as / (AmE), / c8m5bm59ri tre6as / (BrE)
a recorded sequence or log of memory-related events (such as allocations, accesses, or frees) produced by a program or tool and used for debugging or performance analysis.

Examples

Program memory trace
The profiler generates a memory trace to analyze heap usage and find memory leaks.
Developers examined several memory traces to identify which allocation caused the crash.
You can record a memory trace during tests to measure peak usage and performance.

Related Terms

  • synonym: memory log - a recorded list of memory operations or events, used to inspect program behavior
