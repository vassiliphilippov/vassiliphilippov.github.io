code of behavior

noun
set of group rules for conduct
Pronunciation: / koʊd əv bɪˈheɪvjər / (AmE), / kəʊd əv bɪˈheɪvjə / (BrE)
a set of conventional principles and expectations regarded as obligatory for members of a particular group

Examples

The code of behavior
Every club member is expected to follow the code of behavior at all times.
The company's code of behavior requires employees to treat each other with respect.
Violating the school's code of behavior can result in disciplinary action.

  • synonym: code of conduct - a set of rules outlining the norms for members of a group
