silent

adjective
without user interaction (computing)
6.1
Pronunciation: /ˈsaɪ.lənt/ (AmE), /ˈsaɪ.lənt/ (BrE)
(computing) performed without user interaction, prompts, or visible messages; running unattended or without requiring input.

Examples

Silent install
The software offers a silent install that requires no user input during the whole setup process.
IT scheduled a silent update for overnight so employees would not see prompts or confirm dialogs.
Developers use a silent API call to check credentials without interrupting the user's workflow.

Related Terms

  • synonym: headless - running without a graphical user interface or direct user interaction
  • synonym: unattended - performed without a user present or without requiring user input
