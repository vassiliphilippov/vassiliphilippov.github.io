file in

verb
to enter in a line
2.9
Pronunciation: / faɪl ɪn / (AmE), / faɪl ɪn / (BrE)
to enter a place by moving in a line, especially one behind the other

Examples

To file in quietly
Every morning, employees file in to the building just before nine o'clock.
Guests file in through the main door as the ceremony is about to start.
The students filed in quietly and took their seats before the assembly began.

Related Terms

  • antonym: file out - to leave a place by moving in a line
  • synonym: march in - to enter by marching
