offscreen

adjective
not visible on screen
Pronunciation: / ˌɔfˈskrin / (AmE), / ˌɒfˈskriːn / (BrE)
stored, rendered, or occurring in memory rather than being currently visible on the screen, especially in computing or graphics.

Examples

Offscreen buffer
The animation was first created in an offscreen buffer to improve performance.
Developers often use offscreen rendering to prepare images before displaying them to users.
The video editor allows you to preview changes in an offscreen window before saving.

Related Terms

  • antonym: onscreen - visible or displayed on the screen
