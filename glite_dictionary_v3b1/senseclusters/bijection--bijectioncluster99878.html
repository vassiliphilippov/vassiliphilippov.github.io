bijection

noun
one-to-one correspondence
9.8
Pronunciation: / baɪˈdʒɛkʃən / (AmE), / baɪˈdʒɛkʃən / (BrE)
a function that pairs each element of one set with exactly one unique element of another set, and vice versa

Examples

A bijection function
A bijection exists between the set of natural numbers and the set of even numbers.
To prove the two sets are the same size, we constructed a bijection between them.
Understanding the concept of bijection is essential in higher mathematics.

Related Terms

  • degree lower: injection - a function that maps each element of one set to a unique element of another set
  • degree lower: surjection - a function that covers every element of the target set at least once
