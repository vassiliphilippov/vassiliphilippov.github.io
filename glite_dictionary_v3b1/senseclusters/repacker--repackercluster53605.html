repacker

noun
software packaging tool
Pronunciation: / ˌriːˈpækər / (AmE), / ˌriːˈpækə / (BrE)
a computer program or tool that repackages software or digital files, often by compressing or modifying installers

Examples

Software repacker
The developer used a repacker to compress the game files into a smaller installer for download.
Many software distributions rely on a repacker to create efficient installation packages.
After modifying the application, he ran a repacker to bundle all the necessary files together.

Related Terms

  • synonym: compressor - a tool that reduces the size of digital files
  • antonym: extractor - a tool that unpacks or extracts files from an archive
  • synonym: installer - a program that installs software on a computer
