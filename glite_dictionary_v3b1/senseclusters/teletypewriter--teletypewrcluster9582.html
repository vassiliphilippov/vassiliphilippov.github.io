Synonyms

  • teleprinter
    8.8 0.01/m
    teleprinter is an electromechanical typewriter used in many communication systems, while teletypewriter is the general name for the same printed messaging device.
  • teletype machine
    8.8 0.02/m
    teletype machine prints and receives typed messages over telegraph lines, while teletypewriter is the broader term for similar machines.
  • TTY
    9.8 0.01/m
    TTY is an abbreviation for electromechanical teletypewriters used before computers, while teletypewriter is the full device name.
  • telex machine
    9.6 0.01/m
    telex machine is a typewriter-like device used on the telex network for written messages, while teletypewriter is a general printed-message machine.
  • telex
    9.3 0.24/m
    telex is the name for the telex network's teleprinter system used for messaging, while teletypewriter refers to the physical printing machine.
