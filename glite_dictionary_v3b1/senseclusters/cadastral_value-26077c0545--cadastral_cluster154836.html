cadastral value

noun
official valuation of property for tax purposes
8.7 9.3
Pronunciation: / kəˈdæstrəl ˈvælju / (AmE), / kəˈdæstrəl ˈvaljuː / (BrE)
the official value assigned to a property, comprising the land and any buildings, used as a basis for calculating property taxes

Examples

The cadastral value of property
Disputes can arise if the cadastral value of a house is set much higher than its market value.
Every few years, the government updates the cadastral value to reflect changes in property assessments.
The local council used the cadastral value to determine the annual property tax bill.
