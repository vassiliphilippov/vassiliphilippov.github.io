ppm

noun
image file format
Pronunciation: / pi pi ɛm / (AmE), / piː piː em / (BrE)
a simple uncompressed image file format (Portable Pixmap) commonly used for storing pixel data, usually with the .ppm extension

Examples

PPM image file
The graphics program can export images as ppm files for easy editing.
She converted the scanned document from ppm to JPEG to reduce the file size.
Many open-source tools support reading and writing ppm images for testing purposes.

Related Terms

  • synonym: jpeg - a widely used compressed image file format
  • synonym: png - a popular compressed image file format
