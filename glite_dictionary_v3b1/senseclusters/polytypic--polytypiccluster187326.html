polytypic

adjective
type-generic
8.6
Pronunciation: / ˌpɑliˈtɪpɪk / (AmE), / ˌpɒlɪˈtɪpɪk / (BrE)
able to operate on or apply to many different types, especially in computing or programming contexts.

Examples

Polytypic function design
A polytypic function can process both lists and trees without rewriting the code.
The software's polytypic design allows it to handle various data structures efficiently.
Researchers developed a polytypic algorithm that adapts to different input types automatically.

Related Terms

  • synonym: generic - not specific to a single type; general
  • antonym: monotypic - restricted to a single type
