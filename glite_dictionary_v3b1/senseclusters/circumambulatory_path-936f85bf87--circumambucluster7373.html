circumambulatory path

noun
path that circles around something
Pronunciation: / ˌsɜrkəmˈæmbjələˌtɔri pæθ / (AmE), / ˌsɜːkəmˈæmbjʊlətəri pɑːθ / (BrE)
A path that goes around an area, object, or structure, typically for walking or ritual purposes.

Examples

A circumambulatory path around
A stone circumambulatory path encircles the temple, allowing visitors to take a meditative walk.
Pilgrims followed the circumambulatory path as part of the sacred ritual.
The garden was designed with a circumambulatory path lined with cherry trees.

Related Terms

  • synonym: perimeter path - a path that runs along the boundary or edge of an area
