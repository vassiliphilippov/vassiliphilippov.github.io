inscriptible

adjective
can be written on
Pronunciation: / ɪnˈskrɪptəbəl / (AmE), / ɪnˈskrɪptəbəl / (BrE)
able to be inscribed or written upon

Examples

Inscriptible label
The label is inscriptible, so you can write your name on it with a marker.
Not every surface is inscriptible with chalk, especially if it is too smooth.
They searched for an inscriptible stone to carve the date into.

Related Terms

  • synonym: inscribable - able to have words or symbols carved or written on
  • antonym: unwritable - not able to be written on
  • synonym: writable - able to be written on
