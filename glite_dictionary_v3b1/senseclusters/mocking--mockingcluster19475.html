mocking

noun
use of mock objects in testing
7.5 0.64/m
Pronunciation: / c8m51k6a4b / (AmE), / c8m52k6a4b / (BrE)
(in software development) the practice of using mock objects or simulated components to imitate real modules during automated testing so tests run independently of external systems.

Examples

Unit test mocking
The team adopted mocking to isolate the unit under test during development.
Our CI pipeline fails if developers rely on heavy mocking without integration tests.
Because of extensive mocking, they could simulate network errors and speed up debugging.

Related Terms

  • synonym: mock object - a simulated object that mimics the behavior of real components in tests
  • synonym: stub - a simple replacement for a software component used to provide predefined responses in tests
