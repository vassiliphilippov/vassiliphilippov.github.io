run

noun
row of unravelled stitches in fabric
8.2 0.47/m
Pronunciation: / rÊŒn / (AmE), / rÊŒn / (BrE)
a row in a knitted fabric, such as a stocking, where the stitches have become unravelled, especially forming a noticeable line or gap

Examples

A run in stockings
A small run in your sweater can quickly become a large hole if you aren't careful.
She noticed a run in her stockings and had to change them before leaving the house.
The dress was perfect except for a run in the hem that needed mending.
