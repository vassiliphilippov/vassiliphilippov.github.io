run to the store

verb
make a quick trip to a store
1.0
Pronunciation: / rʌn tə ðə stɔr / (AmE), / rʌn tə ðə stɔː / (BrE)
to go quickly to a store, often for a brief errand or to buy something needed

Examples

To run to the store quickly
I forgot the eggs, so I need to run to the store before dinner.
She asked her son to run to the store for some milk.
When they ran out of soda, Mark offered to run to the store and pick some up.

Related Terms

  • synonym: pop to the shop - go quickly to a shop, mainly British English
