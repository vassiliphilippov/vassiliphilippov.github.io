dividing line

noun
draggable UI separator
4.2
Pronunciation: /dɪ.ˈvaɪ.dɪŋ laɪn/ (AmE), /dɪ.ˈvaɪ.dɪŋ laɪn/ (BrE)
(in computing and user interfaces) a visible vertical or horizontal separator between panes or panels that can be dragged (or locked) to resize adjacent content areas.

Examples

Move the dividing line
Drag the dividing line to make the left panel wider when editing the document layout.
The app shows a thin dividing line between the editor and preview panes you can drag.
She locked the dividing line so users cannot accidentally resize essential interface panels during demonstrations.

Related Terms

  • synonym: divider - a line or bar in a user interface that separates and can often resize adjacent areas
  • synonym: panel - a distinct area of a user interface used to display content or tools
