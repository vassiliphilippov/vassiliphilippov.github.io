pluggable

adjective
can be added as module
5.3
Pronunciation: / ˈplʌɡəbəl / (AmE), / ˈplʌɡəbəl / (BrE)
able to be connected, added, or removed as a separate module or component, especially in software or hardware systems

Examples

Pluggable module
The application allows for pluggable authentication methods to increase security.
Developers can create pluggable extensions to add new features without changing the core system.
The device's pluggable components make it easy to upgrade or repair.

Related Terms

  • antonym: fixed - not able to be changed or added to
  • synonym: modular - made up of separate parts that can be connected or combined
