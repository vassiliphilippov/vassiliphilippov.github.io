nonlocal

x
Python reserved word
8.2
Pronunciation: / nɑnˈloʊkəl / (AmE), / nɒnˈləʊkəl / (BrE)
a reserved word in Python used in a nested function to refer to a variable in the nearest enclosing non-global scope

Examples

Python nonlocal keyword
The nonlocal keyword allows you to modify a variable from an enclosing function in Python.
If you want to change a variable defined outside the inner function, use nonlocal.
Without nonlocal, the assignment would create a new local variable instead of modifying the outer one.

Related Terms

  • antonym: global - a reserved word in Python used to refer to variables in the global scope
