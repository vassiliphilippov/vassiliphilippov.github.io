utility routine

noun
general-purpose routine used as needed
Pronunciation: / juˈtɪləti ruˈtin / (AmE), / juˈtɪlɪti ruːˈtiːn / (BrE)
a general-purpose program or function that performs a common task and can be used by other programs as needed

Examples

A utility routine program
By using a utility routine, you can simplify repetitive coding tasks.
Many libraries include a utility routine for error checking.
The programmer wrote a utility routine to handle file conversions automatically.
