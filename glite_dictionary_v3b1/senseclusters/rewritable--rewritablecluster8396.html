rewritable

adjective
can be written again
4.1
Pronunciation: / ˌriːˈraɪtəbəl / (AmE), / ˌriːˈraɪtəbəl / (BrE)
able to be written on, erased, and written on again, especially referring to digital storage media

Examples

Rewritable CD
This rewritable CD lets you update your files as often as you need.
Make sure you use a rewritable disk if you want to erase and reuse it later.
The camera stores photos on a rewritable memory card for easy editing.

Related Terms

  • synonym: erasable - able to be erased or removed
  • antonym: read-only - able to be read but not written or changed
