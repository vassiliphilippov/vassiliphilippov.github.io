nsec

noun
DNSSEC record type
9.9
Pronunciation: /ˈen.sek/ /; / /ˈen.es.iː.siː/ (AmE), /ˈen.sek/ /; / /ˈen.es.iː.siː/ (BrE)
a type of DNSSEC resource record used in domain name systems to prove that a specific domain name does not exist

Examples

nsec record in DNS
The administrator added an nsec record to the DNS zone for security purposes.
Without a valid nsec record, the server could not confirm the domain's nonexistence.
Learning about nsec records is important for anyone managing DNSSEC-enabled domains.
