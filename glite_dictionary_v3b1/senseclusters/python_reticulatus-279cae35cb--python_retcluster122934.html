Python reticulatus

noun
largest species of python; reticulated python
8.1
Pronunciation: / ˈpaɪθɑn rɛˌtɪkjəˈleɪtəs / (AmE), / ˈpaɪθən rɪˌtɪkjʊˈleɪtəs / (BrE)
A species of python native to Southeast Asia and the East Indies, recognized as the largest snake in the world.

The Python reticulatus snake
