portably

adverb
in a transferable way
5.2
Pronunciation: /ˈpɔːr.tə.bliː/ (AmE), /ˈpɔː.tə.bliː/ (BrE)
in a way that allows something, especially software or data, to be used or transferred easily between different systems or locations

Examples

Portably run software
The application can be used portably on different computers without installation.
By saving your files portably, you can access them from any device.
Many tools are designed to run portably from a USB drive for convenience.

Related Terms

  • synonym: flexibly - in a way that can be easily adapted or changed
  • antonym: locally - in a way that is limited to a specific place or system
