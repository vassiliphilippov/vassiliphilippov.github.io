env

noun
Unix environment utility
Pronunciation: / ɛnv / (AmE), / ɛnv / (BrE)
a Unix command-line utility used to run a program in a modified environment or display environment variables

Examples

The env command
You can use the env command to list all current environment variables in your terminal.
Before running the script, she set a variable with env to change its behavior.
The env utility is often used in shell scripts to control the execution environment.

Related Terms

  • synonym: environment variable - a variable that stores information about the operating system environment
  • synonym: set - a command to assign values to environment variables
