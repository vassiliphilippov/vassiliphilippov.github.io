preinstruction

noun
initializing command
8.3
Pronunciation: / ˌpriːɪnˈstrʌkʃən / (AmE), / ˌpriːɪnˈstrʌkʃən / (BrE)
a command or short sequence executed before the main program or operation, often to initialize hardware or software systems

Examples

Preinstruction code runs
The firmware runs a small preinstruction to initialize sensors before starting the main control loop.
A preinstruction is necessary to set up the environment before the main application loads.
Engineers added a preinstruction to check system status before launching the software update.
