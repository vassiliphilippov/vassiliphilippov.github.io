NAC

noun
Network Access Control (security system)
8.0
Pronunciation: /en eɪ siː/ /; / /næk/ (AmE), /en eɪ siː/ /; / /næk/ (BrE)
(computing) abbreviation for Network Access Control: a security system or framework that enforces policy-based access and device-compliance checks before allowing devices to connect to a network.

Examples

NAC system
The campus rolled out a NAC system to enforce device compliance before granting network access.
After a breach, the IT team configured the NAC to block noncompliant laptops immediately.
Many businesses use NAC combined with firewalls to ensure only approved devices join the network.

Related Terms

  • synonym: Network Access Control - a technology or process that controls which devices can connect to a network based on policy and compliance checks
  • synonym: access control - methods and policies for controlling who or what can access resources on a system or network
