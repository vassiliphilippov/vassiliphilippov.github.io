exp

noun
exponential function
Pronunciation: / ɛksˈp / (AmE), / ɛksˈp / (BrE)
the mathematical function that gives the value of e raised to a given power, written as exp(x) = e^x

Examples

Exp function math
To solve the equation, you need to use the exp function on your calculator.
The graph of the exp function rises rapidly as x increases.
In many scientific formulas, exp is used to represent exponential growth.
