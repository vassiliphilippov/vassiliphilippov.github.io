YARN

noun
Hadoop resource manager (acronym)
8.9
Pronunciation: / jɑrn / (AmE), / jɑːn / (BrE)
(computing, Hadoop) the resource‑management and job‑scheduling layer of Apache Hadoop that allocates cluster resources and manages containers for running distributed applications.

Examples

YARN resource manager
The data pipeline runs on YARN to allocate memory and CPU across the cluster.
During the outage, the team restarted the application manager to recover jobs from YARN.
Developers often tune YARN's container settings for better performance on large clusters.

Related Terms

  • synonym: Apache Hadoop - an open-source framework for distributed storage and processing of large data sets; YARN is its resource-management layer
  • synonym: resource manager - a system component that allocates and manages compute resources for running applications
