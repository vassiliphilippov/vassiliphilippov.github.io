rel

noun
relocation file
Pronunciation: / rɛl / (AmE), / rɛl / (BrE)
a computer file, often with the extension .rel, that contains relocation or linking information for assemblers and linkers

Examples

Rel file format
The developer checked the rel file for errors before proceeding to the linking stage.
After assembling the code, a rel was generated for the linker to process.
You need to upload the rel so the build system can create the final program.

Related Terms

  • antonym: executable - a file that can be run directly by a computer
  • synonym: object file - a file containing machine code output from a compiler or assembler
