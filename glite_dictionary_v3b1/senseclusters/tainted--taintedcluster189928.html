tainted

adjective
untrusted or potentially dangerous data (computing)
Pronunciation: / ˈteɪn.tɪd / (AmE), / ˈteɪn.tɪd / (BrE)
(in computing, especially security and program analysis) marked as coming from an untrusted or potentially dangerous source and therefore requiring validation or sanitization.

Examples

Tainted data
The security team marked the suspicious parameter as tainted before running the sanitization routine.
During dynamic analysis, the tool tracked which variables became tainted after receiving external network data.
Developers must treat any tainted user input as dangerous until it has been properly validated.

Related Terms

  • antonym: sanitized - processed to remove or neutralize harmful or unsafe content
  • synonym: untrusted - not trusted; data or sources that may be unsafe or malicious
