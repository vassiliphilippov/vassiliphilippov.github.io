compressor

noun
software tool that reduces file size
4.0
Pronunciation: /kəm.ˈpres.ər/ (AmE), /kəm.ˈpres.ə/ (BrE)
a software program or algorithm that reduces the size of digital data by encoding it more efficiently.

Examples

File compressor
The online compressor reduced the 50MB image to a small, web-friendly file in seconds.
Developers chose a fast compressor to speed downloads for users on slow mobile networks.
Using a lossless compressor, she recovered the original files without any quality loss after archiving.

Related Terms

  • synonym: archiver - a program that creates compressed archive files (often combining compression with packaging)
  • antonym: decompressor - a program or routine that restores compressed data to its original form
