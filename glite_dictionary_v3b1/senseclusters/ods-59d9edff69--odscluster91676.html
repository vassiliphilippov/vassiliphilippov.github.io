ODS

noun
spreadsheet file format
Pronunciation: / oʊ diː ɛs / (AmE), / əʊ diː ɛs / (BrE)
a spreadsheet file format (with the .ods extension) used by OpenDocument-compatible programs such as LibreOffice and OpenOffice

Examples

ODS file format
She exported the data as an ODS file to share with her team.
Most open-source office suites can open and edit ODS spreadsheets easily.
He prefers using the ODS format because it is compatible with many programs.
