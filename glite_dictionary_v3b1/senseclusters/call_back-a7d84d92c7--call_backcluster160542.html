call back

verb
invoke a registered function on an event
5.0
Pronunciation: /kɔːl bæk/ /; / /kɑːl bæk/ (AmE), /kɔːl bæk/ (BrE)
(in computing) to invoke or execute a function previously provided (a callback) in response to an event or the completion of an asynchronous operation.

Examples

To call back a function

The library is still calling back functions as events occur in the background.
