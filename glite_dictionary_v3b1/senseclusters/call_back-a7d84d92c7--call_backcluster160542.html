call back

verb
invoke a registered function on an event
Pronunciation: / kɔl bæk / (AmE), / kɔːl bæk / (BrE)
(in computing) to invoke or execute a function previously provided (a callback) in response to an event or the completion of an asynchronous operation.

Examples

To call back a function
The API will call back the registered function when the data arrives.
After processing the upload, the server called back the client's handler to report success.
The library is still calling back functions as events occur in the background.

Related Terms

  • synonym: invoke - to execute or call a function or routine in a program
  • synonym: trigger - to cause a function or process to start running, often in response to an event
