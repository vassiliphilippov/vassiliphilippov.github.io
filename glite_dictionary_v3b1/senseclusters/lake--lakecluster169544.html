lake

noun
central repository for raw data
6.1
Pronunciation: /leɪk/ (AmE), /leɪk/ (BrE)
(in computing, especially data engineering) a centralized repository that stores large volumes of raw, unprocessed data in its native format for later analysis or processing.

Examples

A data lake
Many organizations now run a lake of raw data in the cloud to support analytics.
The analytics team loaded terabytes into the lake and planned schema-on-read queries.
Instead of a warehouse, they chose a lake to store raw logs for machine learning.

Related Terms

  • synonym: data repository - a place or system for storing and managing data
  • antonym: data warehouse - a structured system for storing cleaned, processed data for reporting and analysis; often contrasted with a data lake
