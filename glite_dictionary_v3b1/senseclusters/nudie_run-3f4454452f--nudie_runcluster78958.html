nudie run

noun
running naked as a prank or dare
Pronunciation: / ˈnuːdi ˌrʌn / (AmE), / ˈnjuːdi ˌrʌn / (BrE)
an act of running naked in public, usually as a prank, dare, or for amusement

Examples

A nudie run prank
After losing a bet, he had to do a nudie run across the football field.
The group organized a nudie run to raise money for charity.
They all laughed when someone suggested a midnight nudie run at the lake.
