bicameral script

noun
script with two letter cases
5.8
Pronunciation: / baɪˈkæm.ɚ.əl skrɪpt / (AmE), / baɪˈkæm.ər.əl skrɪpt / (BrE)
a writing system or script that has two distinct cases, typically uppercase and lowercase forms of letters

Examples

Bicameral script system
Most modern European languages are written in bicameral scripts such as Latin or Cyrillic.
The English alphabet is a bicameral script, featuring both uppercase and lowercase letters.
Typographers must consider how each letter looks in both cases when designing a bicameral script.

Related Terms

  • antonym: unicameral script - writing system with only one case
