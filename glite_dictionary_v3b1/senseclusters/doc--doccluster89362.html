doc

noun
informal document
Pronunciation: / dɑk / (AmE), / dɒk / (BrE)

Examples

Can you send me the doc with the latest project updates by email?
I saved the report as a doc so everyone can edit it easily.
Please upload the doc to the shared folder before the meeting.
