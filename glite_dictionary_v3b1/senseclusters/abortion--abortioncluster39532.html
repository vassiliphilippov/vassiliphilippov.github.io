abortion

noun
premature termination (computing)
6.7 0.11/m
Pronunciation: / …ôÀàb…îr. É…ôn / (AmE), / …ôÀàb…îÀêr. É…ôn / (BrE)
(in computing) the premature termination of a process, transaction, or computation by a user or the system before normal completion.

Examples

System abortion
After detecting a conflict, the database performed an automatic abortion of the transaction to prevent corruption.
The application logged the unexpected abortion of the background task and restarted it immediately.
Users receive a warning when they request an abortion of the file upload during a slow connection.
