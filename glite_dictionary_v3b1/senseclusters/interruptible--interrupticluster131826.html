interruptible

adjective
can be paused
5.3
Pronunciation: / ɪn.təˈrʌpt.ə.bəl / (AmE), / ɪn.təˈrʌpt.ɪ.bəl / (BrE)
able to be stopped or paused temporarily and then continued later

Examples

An interruptible process
The software update is interruptible, so you can pause it and resume later if needed.
Some power supplies are interruptible, meaning they can be turned off without causing damage.
The meeting was designed to be interruptible in case urgent calls came in.
The process is interruptible, so it can respond quickly to urgent signals.
In some systems, only interruptible tasks can be safely paused without losing data.
The software was designed to be interruptible for better multitasking performance.
