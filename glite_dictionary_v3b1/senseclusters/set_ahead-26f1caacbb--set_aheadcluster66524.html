set ahead

verb
move something forward, especially a clock or watch
Pronunciation: / sɛt əˈhɛd / (AmE), / sɛt əˈhɛd / (BrE)
to move (something, especially a clock or watch) forward to a later time

Examples

To set ahead clocks

The staff will set ahead the conference room clock so everyone arrives on time.
We had to set ahead all the watches in the house after traveling east.
