ANSI

noun
terminal control codes
7.1
Pronunciation: / ˈænsi / (AmE), / ˈænsi / (BrE)
a set of control codes used to manage text color, cursor position, and formatting on computer terminals

Examples

ANSI color codes
The script uses ANSI codes to display colored output in the terminal.
Many modern terminals support ANSI escape sequences for advanced text formatting.
He learned how to use ANSI codes to move the cursor and clear the screen.

Related Terms

  • synonym: VT100 - a type of terminal that uses similar control codes
  • synonym: escape sequence - a series of characters used to control devices or formatting
