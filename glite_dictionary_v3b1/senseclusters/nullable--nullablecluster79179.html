nullable

adjective
can be null
Pronunciation: / ˈnʌləbəl / (AmE), / ˈnʌləbəl / (BrE)
able to be assigned a null value or left without a value, especially in computing or databases

Examples

Nullable database field
In this database, the email field is nullable, so you do not have to enter an address.
A nullable column can store either a value or no value at all.
The developer marked the variable as nullable to allow for missing information.

Related Terms

  • synonym: optional - not required; can be left empty or unassigned
  • antonym: required - must have a value; cannot be left empty
