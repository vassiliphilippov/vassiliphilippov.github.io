secondary storage

noun
computer data storage device (not main memory)
6.1 3.2
Pronunciation: /ˈsek.ən.dri ˈstɔː.rɪdʒ/ (AmE), /ˈsek.ən.dri ˈstɔː.rɪdʒ/ (BrE)
a data storage device, such as a hard disk or flash drive, that is used to store data permanently and is not the main memory of a computer

Examples

Secondary storage device
Because primary memory is limited, large files are often saved in secondary storage.
The computer uses secondary storage like hard drives to keep data even when the power is off.
Upgrading your secondary storage can significantly increase your computer’s capacity to hold documents and media.

Related Terms

  • antonym: primary storage - main memory of a computer (RAM)
