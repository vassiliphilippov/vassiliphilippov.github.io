external storage

noun
data storage device outside main memory
Pronunciation: / ɪkˈstɝːnəl ˈstɔːrɪdʒ / (AmE), / ɪkˈstɜːnəl ˈstɔːrɪdʒ / (BrE)
a data storage device that is not part of the main memory of a computer, often removable or separate from the main device

Examples

USB external storage
Many professionals use external storage to transfer data between computers.
She backed up her files onto an external storage device for safekeeping.
The laptop has limited internal memory, so external storage is recommended for large files.

Related Terms

  • antonym: internal storage - data storage within the main computer system
