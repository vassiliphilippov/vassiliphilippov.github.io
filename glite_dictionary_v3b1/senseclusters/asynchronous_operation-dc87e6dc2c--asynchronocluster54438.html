asynchronous operation

noun
operation that occurs independently of other processes
Pronunciation: / əˈsɪŋkrənəs ˌɑːpəˈreɪʃən / (AmE), / əˈsɪŋkrənəs ˌɒpəˈreɪʃən / (BrE)
an operation that occurs independently of other processes, without needing to happen at regular or predictable intervals relative to other events

Examples

An asynchronous operation process
Data is read from the disk using an asynchronous operation so the user interface remains responsive.
Switching from a synchronous to an asynchronous operation improved the system's efficiency significantly.
The server handles requests through an asynchronous operation to avoid blocking the main thread.

Related Terms

  • antonym: synchronous operation - operation that occurs in regular, predictable relation to other processes
