sourceless

adjective
without source code
8.2
Pronunciation: / ˈsɔrs.ləs / (AmE), / ˈsɔːs.ləs / (BrE)
distributed or provided without the original source code, only as compiled or executable files

Examples

Sourceless software release
The software was released as a sourceless application, so developers could not inspect or change its code.
Many users prefer open-source programs over sourceless ones for greater transparency and control.
Because it was sourceless, the program could not be customized to fit the company's needs.

Related Terms

  • antonym: open-source - with source code freely available for use and modification
  • synonym: proprietary - restricted and not freely available for modification or distribution
