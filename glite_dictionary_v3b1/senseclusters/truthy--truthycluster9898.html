truthy

adjective
evaluated as true
9.1
Pronunciation: / ˈtruːθi / (AmE), / ˈtruːθi / (BrE)
describes a value in programming that is treated as true when evaluated in a logical context

Examples

Truthy value in code
In JavaScript, the number 1 is considered truthy in conditional statements.
Empty strings are not truthy in most programming languages.
You should check if the variable is truthy before using it in your code.

Related Terms
