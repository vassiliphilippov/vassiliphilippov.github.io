prod

noun
production environment (software)
8.1
Pronunciation: / prɑd / (AmE), / prɒd / (BrE)
(computing, informal) the production environment — the live system where software runs for real users and where live data is processed.

Examples

Fixes in prod
We deployed the hotfix to prod to verify it in real user conditions.
Don't test experimental features in prod; use the staging environment instead.
The incident caused major outages, bringing down several services in prod for hours.

Related Terms

  • synonym: production environment - the live system used by end users to run software and services
  • antonym: staging - a pre-production environment used for testing before deploying to production
