USR

noun
Unix system directory
Pronunciation: / ˈjuː ɛs ɑːr / (AmE), / ˈjuː ɛs ɑː / (BrE)
a directory in Unix and Unix-like computer systems that stores user-related files and programs

Examples

USR directory
The program files are stored in the USR directory on most Linux systems.
You should not delete anything from the USR folder unless you know what you are doing.
System administrators often update software located in USR to keep the system secure.

Related Terms

  • synonym: home directory - a user's personal directory in a computer system
  • antonym: root directory - the top-level directory in a file system
