ingest

verb
import data into a system
6.3
Pronunciation: /ɪn.ˈdʒest/ (AmE), /ɪn.ˈdʒest/ (BrE)
(in computing) to import or take data, files, or streams into a computer system, database, or processing pipeline for storage or further processing.

Examples

To ingest data
The platform is designed to ingest data from multiple sources every hour for real-time analysis.
Their ETL tool ingested terabytes of sensor readings last night without any data loss.
By ingesting cleaned datasets first, the team avoided inconsistent results during model training.

Related Terms

  • antonym: export - to send data out of a system to another system or format
  • synonym: import - to bring data or files into a system from an external source
  • synonym: load - to place data into memory or storage for use or processing
