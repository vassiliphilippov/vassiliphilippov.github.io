(computing) to imitate the function of another system, usually by modifying hardware or software, so that one system behaves like another.

Examples

To emulate a computer
Engineers developed an application to emulate the original hardware functionality in software.
Many mobile phone apps emulate classic arcade games using modern devices.
This software allows you to emulate old video game consoles on your computer.

Synonyms

  • imitate
    2.7 12/m

  • mimic
    4.5 0.05/m
    mimic often copies surface behavior or sound in a simple way, while emulate reproduces full system function through hardware or software changes.
