freeware

noun
software provided at no charge
Pronunciation: / ˈfriˌwɛr / (AmE), / ˈfriːweə / (BrE)
Computer software that is available for use at no cost, though it may have usage restrictions or limited rights to modify or redistribute.

Examples

Freeware software
He prefers freeware because it allows him to try new tools without cost.
Many people use freeware to edit photos without having to pay for expensive programs.
Several popular utilities are available as freeware and can be downloaded from the developer's website.

Related Terms

  • antonym: commercial software - software that must be purchased to use legally
  • degree higher: open-source software - software whose source code is freely available for modification and distribution
  • synonym: shareware - software distributed for free but requiring payment for full functionality
