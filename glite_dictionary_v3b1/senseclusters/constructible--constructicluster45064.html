constructible

adjective
able to be built
Pronunciation: / kənˈstrʌktəbəl / (AmE), / kənˈstrʌktɪbəl / (BrE)
able to be built, created, or constructed, especially according to specific rules or methods

Examples

Constructible shape
The geometric figure is constructible with only a compass and straightedge.
Not every design is constructible within the given budget and time frame.
In mathematics, a number is constructible if it can be obtained using certain operations.

Related Terms

  • synonym: buildable - able to be built or constructed
  • antonym: inconstructible - not able to be built or constructed
