start

verb
to begin; to have a beginning
1.0
Pronunciation: / stɑrt /; / stɑːrt / (AmE), / stɑːt / (BrE)
to begin or cause something to begin; to set something in motion or start doing something

Examples

To start a project
Her new job started last Monday and she is already enjoying it.
Our meeting usually starts right after lunch.
The race will start at exactly 9 a.m. tomorrow.
She will start her presentation after lunch.
The race will start in just a few minutes.
We usually start work at nine o'clock each morning.
Could you please start the washing machine for me?
He forgot to start the recording before the concert began.
It took several tries to start the car on the cold morning.
