logical diagram

noun
diagram showing logical relationships
Pronunciation: / ˈlɑːdʒɪkəl daɪˌæɡræm / (AmE), / ˈlɒdʒɪkəl daɪˌæɡræm / (BrE)
a diagram that represents the structure or functioning of a system or program using symbols and formal logic

Examples

A logical diagram of system
Before coding, she created a logical diagram to represent the program's logical flow.
Students used a logical diagram to visualize how different program components interact.
The software engineer drew a logical diagram to illustrate the decision-making process of the system.
