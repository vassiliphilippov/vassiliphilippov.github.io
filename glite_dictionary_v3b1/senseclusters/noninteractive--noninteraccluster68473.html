noninteractive

adjective
without user interaction
6.1
Pronunciation: / ˌnɑnˌɪntəˈræktɪv / (AmE), / ˌnɒnˌɪntəˈræktɪv / (BrE)
not involving or allowing direct user interaction, especially in computer programs or systems

Examples

A noninteractive program
The installation process is noninteractive, so it does not ask the user any questions.
In noninteractive mode, the program completes all tasks automatically.
System updates can be performed in a noninteractive way to save time.

Related Terms

  • synonym: automated - operated automatically without human intervention
  • antonym: interactive - allowing direct communication or user input
