acceleration

noun
computing speed-up using specialized hardware
4.2
Pronunciation: / ækˌsɛləˈreɪʃən / (AmE), / əkˌsɛləˈreɪʃən / (BrE)
(in computing) the use of specialized hardware, firmware, or services to speed up particular computational tasks or processing.

Examples

Graphics acceleration chip
The software relies on acceleration to render 3D graphics in real time.
By enabling GPU acceleration, the video editor exports high-resolution clips much faster.
Cloud providers offer specialized acceleration instances for machine learning workloads.

Related Terms

  • synonym: hardware acceleration - use of dedicated hardware to perform functions more quickly than general-purpose CPUs
  • synonym: optimization - the process of making a system or program run more efficiently or quickly
