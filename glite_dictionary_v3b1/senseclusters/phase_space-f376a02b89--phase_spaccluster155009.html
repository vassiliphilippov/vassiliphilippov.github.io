phase space

noun
ideal space for system variables in physics
9.9 8.2
Pronunciation: /feɪz speɪs/ (AmE), /feɪz speɪs/ (BrE)
(in physics) an ideal multidimensional space where each axis represents one of the variables needed to specify the state of a system.

Examples

The phase space diagram
In quantum mechanics, the concept of phase space becomes more abstract and complex.
The behavior of the system can be visualized as a trajectory through its phase space.
To analyze the motion, plot each possible state in phase space for clarity.

Related Terms

  • synonym: state space - a mathematical space in which each point represents a possible state of a system
