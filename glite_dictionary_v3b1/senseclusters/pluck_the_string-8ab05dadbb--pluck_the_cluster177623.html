pluck the string

verb
gently pull a string on an instrument
4.2 3.2
Pronunciation: /plʌk ðə strɪŋ/ /; / /plʌk ðiː strɪŋ/ (AmE), /plʌk ðə strɪŋ/ /; / /plʌk ðiː strɪŋ/ (BrE)
to gently pull and release a string on a musical instrument, such as a guitar or harp, to produce a sound

Examples

To pluck the string gently
Remember to gently pluck the string rather than strumming it for a lighter sound.
She watched her teacher pluck the string to demonstrate the difference in tone.
The guitarist plucked the string to create a soft melody.

Related Terms
