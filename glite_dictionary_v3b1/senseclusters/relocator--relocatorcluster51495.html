relocator

noun
program that moves code
Pronunciation: / ˌriːloʊˈkeɪtər / (AmE), / ˌriːləʊˈkeɪtə / (BrE)
a computer program or routine that adjusts addresses or moves code and data so software runs at a new location

Examples

Software relocator
The relocator ensures that the program can run at any memory address.
After compiling the code, a relocator adjusts the addresses for the final executable.
Developers use a relocator to make software more flexible across different systems.

Related Terms

  • synonym: linker - a program that combines object files into a single executable
  • synonym: loader - a program that loads software into memory for execution
