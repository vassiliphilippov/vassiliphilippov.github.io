injector

noun
software dependency-injection component
6.5
Pronunciation: /ɪn.ˈdʒek.tər/ (AmE), /ɪn.ˈdʒek.tə/ (BrE)
(in computing) a component or framework that provides or supplies the required objects or services to other parts of a program by performing dependency injection, often at startup or runtime.

Examples

Software injector
The application's startup uses an injector to assemble services and wire dependencies.
Developers configured the injector so each controller received the correct service instances automatically.
Several frameworks include built-in injectors to simplify building large, modular applications.

Related Terms

  • synonym: DI container - a software module or framework that implements dependency injection to manage and supply dependencies
  • synonym: dependency injection - a programming pattern in which an object receives the components it depends on from an external source
