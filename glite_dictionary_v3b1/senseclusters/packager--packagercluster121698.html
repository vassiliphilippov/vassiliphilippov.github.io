packager

noun
software packaging tool
Pronunciation: / ˈpækɪdʒər / (AmE), / ˈpækɪdʒə / (BrE)
a person or software tool that prepares software or digital content into installable or distributable packages

Examples

Software packager
The packager automatically created an archive of the application for distribution.
As a packager, she ensures all software dependencies are included before release.
Many open-source projects rely on a packager to prepare updates for users.

Related Terms

  • synonym: developer - a person who creates software applications
  • synonym: installer - a program that installs software on a computer
