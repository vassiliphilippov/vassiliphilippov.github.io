exporter

noun
software component exposing metrics
6.0
Pronunciation: /ɪk.ˈspɔːr.tər/ (AmE), /ɪk.ˈspɔː.tə/ (BrE)
(in computing) a software component that exposes internal data or system metrics so external tools can collect, process, or store them.

Examples

Prometheus exporter software
The monitoring stack uses an exporter to publish host metrics to Prometheus.
To troubleshoot performance, install a network exporter that sends bandwidth data to your dashboard.
During deployment they configured several exporters to expose application metrics for collection.

Related Terms

  • synonym: agent - a small program that collects or forwards system data for monitoring or management
  • synonym: collector - software that gathers metrics or logs from one or more sources for processing
