whiteout

verb
mark lower-layer file as deleted
9.5 0.00/m
Pronunciation: / c8wa6at.a8at / (AmE), / c8wa6at.a8at / (BrE)
(computing) to create a placeholder or marker in a layered filesystem that marks a lower-layer file as deleted so it is hidden in the merged view.

Examples

To whiteout a file
The deployment script will whiteout the legacy file in the overlay before building the image.
When the updater ran, it whiteouted several lower-layer files to prevent conflicts.
Engineers often whiteout outdated files during image creation to keep containers clean.
