filo

noun
first in, last out
9.8
Pronunciation: / ˈfaɪloʊ / (AmE), / ˈfaɪləʊ / (BrE)
a method of organizing or processing items so that the first item added is the last one removed, often used in computing and inventory systems

Examples

A filo stack
The stack data structure operates on a filo principle, making it useful for undo features.
In some warehouses, inventory is managed using a filo system to keep newer items accessible.
Understanding filo is important for programmers working with certain types of memory storage.

Related Terms

  • antonym: fifo - a method where the first item added is the first one removed
  • antonym: lifo - a method where the last item added is the first one removed
