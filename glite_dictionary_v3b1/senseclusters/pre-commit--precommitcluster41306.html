pre-commit

noun
automated code check
7.1
Pronunciation: /priː.kə.ˈmɪt/ /; / /prɪ.kə.ˈmɪt/ (AmE), /priː.kə.ˈmɪt/ /; / /prɪ.kə.ˈmɪt/ (BrE)
an automated script or check that runs before changes are committed to a version-control system

Examples

A pre-commit check
The team added a pre-commit to ensure all code is properly formatted before being saved.
If the pre-commit fails, your changes will not be committed to the repository.
Developers often use a pre-commit to automatically check for errors in their code.

Related Terms

  • antonym: post-commit - a script or check that runs after changes are committed
  • synonym: pre-commit hook - a script that runs before a commit in version control
