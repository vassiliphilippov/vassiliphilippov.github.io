machine-readable

adjective
able to be read by a computer
2.3 3.3
Pronunciation: /mə.ˈʃiːn ˈriː.də.bəl/ (AmE), /mə.ˈʃiːn ˈriː.də.bəl/ (BrE)
(of information or data) in a format that can be directly processed by a computer without manual intervention.

Examples

Machine-readable digital file
Before processing, the forms must be converted to a machine-readable digital file.
Government agencies often publish open data sets in machine-readable formats to encourage transparency.
The new database stores records in a machine-readable format for quick analysis.
