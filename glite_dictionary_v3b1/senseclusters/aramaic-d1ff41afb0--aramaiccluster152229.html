Aramaic

noun
ancient script for Aramaic language
Pronunciation: / ˌærəˈmeɪɪk / (AmE), / ˌarəˈmeɪɪk / (BrE)
an alphabetic or syllabic script used since the 9th century BC to write the Aramaic language, from which many other scripts were later derived

Examples

Aramaic script
Scholars have studied ancient Aramaic inscriptions found in the ruins.
Some biblical texts were originally written in Aramaic characters.
The Hebrew script evolved from Aramaic during the Second Temple period.
