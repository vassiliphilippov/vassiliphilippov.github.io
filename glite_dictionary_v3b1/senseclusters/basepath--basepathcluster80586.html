basepath

noun
root directory or URL
Pronunciation: / ˈbeɪsˌpæθ / (AmE), / ˈbeɪsˌpɑːθ / (BrE)
the root directory or initial URL segment used to resolve relative file or resource paths in software or web applications

Examples

Set the basepath
You need to set the basepath in the configuration file before deploying the website.
If the basepath is incorrect, images and scripts may not load properly.
Developers often change the basepath when moving an app to a different server environment.

Related Terms

  • antonym: endpoint - a specific URL or location in a web application
  • synonym: root directory - the top-level directory in a file system
