weaving

noun
integrating aspects into program code
8.4 0.10/m

(in computing, especially aspect-oriented programming) the process of combining or integrating aspect code with a program's main code, so that cross-cutting concerns (for example logging or security checks) are inserted at compile time or run time.

Examples

Code weaving
In AspectJ, weaving inserts advice into Java classes during compilation.
The build tool failed because weaving couldn't find the pointcuts in the compiled classes.
Developers test the application both before and after weaving to ensure aspects behave correctly.

Related Terms

  • synonym: aspect weaving - weaving as used specifically in aspect-oriented programming to combine advice with target code
  • synonym: code weaving - the process of inserting additional code (aspects) into a program's base code
