HDR image file format
Pronunciation: / iː ɛks ɑːr / (AmE), / iː ɛks ɑː / (BrE)
a file format for high dynamic range images, widely used in visual effects and animation industries

Examples

EXR image file
The visual effects team saved each frame as an EXR to preserve image quality.
You should use the EXR format if you need a high dynamic range for your animation project.
Many professional rendering engines support exporting images directly to EXR files.
