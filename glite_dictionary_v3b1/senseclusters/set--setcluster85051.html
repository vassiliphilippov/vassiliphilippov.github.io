set

verb
cause to begin or apply
3.8
Pronunciation: / sɛt / (AmE), / sɛt / (BrE)
to cause something to begin or to apply something, especially something with a significant effect

Examples

Set a fire
Careless campers accidentally set the forest ablaze by leaving a smoldering campfire.
He was arrested after attempting to set a chain reaction that would cause the alarms to go off.
The protestors threatened to set fire to the abandoned building if their demands were not met.
