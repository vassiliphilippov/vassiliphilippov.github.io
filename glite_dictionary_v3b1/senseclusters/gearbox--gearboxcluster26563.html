gearbox

noun
case housing a set of gears
4.0 3.8
Pronunciation: /ˈɡɪr.bɑːks/ (AmE), /ˈgɪə.bɒks/ (BrE)
the metal casing that contains and protects a set of gears in a machine or vehicle, usually as part of the transmission system
