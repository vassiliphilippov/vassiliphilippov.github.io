sigmoid

noun
S-shaped activation function (logistic)
8.7 0.01/m
Pronunciation: / c8s6a61.m546ad / (AmE), / c8s6a61.m546ad / (BrE)
A mathematical S-shaped function (especially the logistic function) used as an activation or transfer function in neural networks and statistics, mapping real numbers to a limited range (commonly 0 to 1).

Examples

Sigmoid function
The neural network used a sigmoid to produce output values between zero and one for classification.
During training, she replaced the ReLU with a sigmoid to obtain smoother gradient updates.
The textbook describes the logistic sigmoid as an S-shaped curve mapping real numbers to probabilities.

Related Terms

  • synonym: activation function - a function used in neural networks to transform input signals into outputs; the sigmoid is one example.
  • synonym: logistic function - a common S-shaped function mapping real numbers to values between 0 and 1; often called the sigmoid.
