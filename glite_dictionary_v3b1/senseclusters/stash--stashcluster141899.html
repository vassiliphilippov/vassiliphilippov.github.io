stash

verb
temporarily save uncommitted changes (VCS)
6.1
Pronunciation: / stæʃ / (AmE), / stæʃ / (BrE)
(in computing, especially version control) to save uncommitted changes temporarily so you can switch contexts and restore them later.

Examples

To stash changes
Before switching branches, she stashed her local changes to avoid merge conflicts later.
He ran git stash to save unfinished edits so he could switch branches safely.
The CI tutorial explained how to stash changes and apply them later when the build succeeds.

Related Terms

  • antonym: commit - to record changes permanently in a version-control repository (contrast with temporarily stashing them)
  • synonym: save - to record data so it can be used again later; in VCS contexts, to preserve work
