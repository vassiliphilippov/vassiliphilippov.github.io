active matrix screen

noun
type of high-quality LCD screen
Pronunciation: / ˈæktɪv ˈmeɪtrɪks skrin / (AmE), / ˈæktɪv ˈmeɪtrɪks skriːn / (BrE)
a type of LCD display that uses a separate circuit for each pixel, resulting in better image quality and response time compared to passive matrix screens.

Examples

An active matrix screen
Compared to older models, the active matrix screen provides much better color reproduction.
Many tablets now use active matrix screens to display high-definition video.
The new laptop features an active matrix screen for sharper images and faster response times.

Related Terms

  • antonym: passive matrix screen - LCD screen with shared circuits, lower quality than active matrix
