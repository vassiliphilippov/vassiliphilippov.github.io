typescript

noun
statically typed superset of JavaScript
6.8
Pronunciation: /ˈtaɪp.skrɪpt/ (AmE), /ˈtaɪp.skrɪpt/ (BrE)
(computing) A statically typed programming language developed by Microsoft that is a superset of JavaScript and compiles to JavaScript; commonly used to build large-scale web applications.

TypeScript code

Related Terms

  • synonym: JavaScript - a widely used scripting language for the web; TypeScript is a typed superset that compiles to it.
  • synonym: static typing - a programming-language feature where types are checked at compile time; a core feature of TypeScript.
