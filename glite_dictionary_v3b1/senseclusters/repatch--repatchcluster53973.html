repatch

noun
second patch
Pronunciation: / ˌriːˈpætʃ / (AmE), / ˌriːˈpætʃ / (BrE)
an additional or replacement patch applied after an earlier patch, especially to fix or update software or repairs

Examples

A software repatch
After the initial fix caused new problems, the team released a repatch to address the issues.
The mechanic had to apply a repatch when the first repair did not hold.
Users were advised to install the latest repatch for improved security.

Related Terms

  • degree lower: patch - a piece of code or material used to fix or improve something
  • synonym: update - a new version or fix applied to software or a system
