trapdoor

noun
hidden software access
8.0
Pronunciation: / ˈtræpˌdɔr / (AmE), / ˈtræpˌdɔː / (BrE)
a hidden vulnerability or secret method in software or hardware that allows unauthorized or concealed access or control

Examples

Software trapdoor
The security team discovered a trapdoor that allowed hackers to access confidential files.
A programmer inserted a trapdoor into the code to bypass normal authentication checks.
After the breach, experts searched the system for any hidden trapdoors left by attackers.

Related Terms

  • synonym: backdoor - a secret way of getting access to a computer system
  • antonym: firewall - a security system that controls incoming and outgoing network traffic
  • synonym: vulnerability - a weakness in a system that can be exploited
