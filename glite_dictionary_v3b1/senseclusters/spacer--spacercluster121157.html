spacer

noun
element for separation
Pronunciation: / ˈspeɪsər / (AmE), / ˈspeɪsə / (BrE)
a blank character or thin element used to create space between text or graphical items in a layout or design

Examples

Text spacer block
The web developer added a spacer to improve the layout of the navigation bar.
In the printed brochure, a thin spacer separates each section for clarity.
Designers often use a spacer to create visual balance between images and text.

Related Terms

  • synonym: divider - a thing that separates or marks a division
  • synonym: separator - an element that divides or keeps things apart
