equal temperament

noun
musical tuning system with equal semitones
8.4 0.06/m
Pronunciation: / ÀàiÀêkw…ôl Ààt…õmp…ôr…ôm…ônt / (AmE), / ÀàiÀêkw…ôl Ààt…õmp…ôr…ôm…ônt / (BrE)
a system of tuning in which the octave is divided into twelve equal semitones, so that each key is equally in tune

Examples

Equal temperament tuning system
Bach wrote The Well-Tempered Clavier to demonstrate the advantages of equal temperament.
In equal temperament, each semitone has the same frequency ratio, making modulation easier.
Most modern pianos are tuned using equal temperament to allow for playability in all keys.
