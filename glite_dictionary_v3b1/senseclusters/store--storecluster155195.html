store

noun
electronic memory device for data storage
Pronunciation: / stɔr / (AmE), / stɔː / (BrE)
a device or software used to keep and retrieve data in a computer system

Examples

Computer store
Early computers often used magnetic drums as their main store.
Increasing the store capacity can significantly improve system performance.
The computer's store holds all the data necessary for processing tasks.
The web app retrieves session data from its store when the user logs in.
We moved the Redux store into a separate module to simplify state management across components.
During backup, the engineer exported the application's store to a cloud database for safekeeping.

Related Terms

  • synonym: database - an organized collection of structured data, often used as persistent storage for applications
  • synonym: memory - device or part of a computer that stores information
  • synonym: persistence layer - the part of an application that handles reading and writing data to long-term storage
  • degree lower: register - a small amount of storage available directly in the CPU
  • synonym: repository - a central place in software where data or code is stored and managed
