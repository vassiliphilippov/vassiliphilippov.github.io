picture element

noun
HTML element for responsive images
4.5
Pronunciation: /ˈpɪk.tʃər ˈel.ə.mənt/ /; / /ˈpɪk.tʃər ˈɛl.ə.mənt/ (AmE), /ˈpɪk.tʃə ˈɛl.ɪ.mənt/ /; / /ˈpɪk.tʃə ˈel.ɪ.mənt/ (BrE)
(in web development) an HTML element (<picture>) that lets authors provide multiple image sources so browsers can choose the best image for different screen sizes or resolutions.

Examples

HTML picture element
To support mobile screens, the designer used a picture element with several image sources.
Browsers choose the best image from a picture element based on viewport size and pixel density.
During testing, the team noticed the picture element fell back to the default image on older browsers.
