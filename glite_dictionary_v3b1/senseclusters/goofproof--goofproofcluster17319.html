goofproof

adjective
resistant to user error
5.1
Pronunciation: /ˈguːf.pruːf/ (AmE), /ˈguːf.pruːf/ (BrE)
Deliberately designed so that it is difficult or impossible for users to make mistakes when using it; foolproof.

Examples

A goofproof device
The new interface is goofproof, so even first-time users navigate easily.
They bought a goofproof coffee maker that shuts off automatically when the pot is empty.
Designers aim for goofproof systems to reduce training time and user errors in hospitals.
