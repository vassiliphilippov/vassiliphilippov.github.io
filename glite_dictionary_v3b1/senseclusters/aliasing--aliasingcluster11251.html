aliasing

noun
shared memory reference
Pronunciation: / ˈeɪliˌæsɪŋ / (AmE), / ˈeɪliːəsɪŋ / (BrE)
the situation in computer programming when two or more references point to the same memory location, potentially causing unexpected changes

Examples

Variable aliasing bug
The programmer fixed a bug caused by aliasing between two variables in the code.
Understanding aliasing is important to avoid unexpected side effects in complex software projects.
The team used special techniques to prevent aliasing when optimizing memory usage.

Related Terms

  • synonym: pointer - a variable that stores a memory address
  • synonym: reference - an object or variable that refers to another value or object in memory
