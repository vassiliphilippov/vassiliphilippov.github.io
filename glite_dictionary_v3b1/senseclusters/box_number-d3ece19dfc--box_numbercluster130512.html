box number

noun
number identifying a post office box
1.5
Pronunciation: / bɑks ˈnʌmbɚ / (AmE), / bɒks ˈnʌmbə / (BrE)
a number assigned to a mailbox (usually at a post office) and used as a mailing address to receive mail

Examples

A box number address
Her mail is delivered to a box number at the nearby post office.
Please use my box number when sending the parcel to the post office.
The company asked customers to reply by writing to a box number.
Please address your letter to box number 47 if you are interested in this opportunity.
Responses should be sent to box number 312 at the local newspaper office.
The advertisement instructed readers to reply via box number rather than a home address.
