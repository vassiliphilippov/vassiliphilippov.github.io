ALGOL

proper noun
early computer programming language
8.5
Pronunciation: / ˈælɡɔl / (AmE), / ˈælɡɒl / (BrE)
a high-level programming language developed in the late 1950s, used for expressing algorithms and influencing later programming languages

Examples

ALGOL programming language
ALGOL played a foundational role in the development of most modern programming languages.
Many university computer science courses in the 1960s taught students using ALGOL.
The syntax of ALGOL heavily influenced the design of C and Pascal.
