(in computing) a programming language in which expressions are represented as strings of characters, rather than in a multi-dimensional format.

Examples

One-dimensional language in computing
Expressions in a one-dimensional language are easy to parse using simple software tools.
LISP is often cited as an early example of a one-dimensional language in computing.
Some mathematicians prefer using a two-dimensional notation instead of a one-dimensional language for complex formulas.
