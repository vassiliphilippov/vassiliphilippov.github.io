supervisory software

noun
software that controls and manages other programs
2.4
Pronunciation: / ˌsuːpərˈvaɪzəri ˈsɔːf(t)wɛr / (AmE), / ˌsuːpəˈvaɪzəri ˈsɒf(t)weə / (BrE)
specialized computer programs that permanently reside in the main memory and control the execution of user programs and system processes.

Examples

Supervisory software controls programs
By updating its supervisory software, the company improved the overall reliability of their computer systems.
The supervisory software ensures that all user applications run smoothly and efficiently on the server.
Without proper supervisory software, the computer would struggle to manage multiple programs at once.

Related Terms

  • synonym: operating system - the software that supports a computer's basic functions and manages hardware and software resources
