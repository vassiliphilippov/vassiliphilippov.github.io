control character

noun
non-printing computer character (e.g., tab, carriage return)
3.9 5.1
Pronunciation: / kənˈtroʊl ˈkæɹəktɚ / (AmE), / kənˈtrəʊl ˈkærəktə / (BrE)
A non-printing character in a computer character set, such as ASCII, that is used to control the interpretation or display of text, e.g., by representing a carriage return, tab, or backspace.

Examples

Control character in ASCII
If you press Ctrl+C, the computer receives a control character that interrupts the process.
Older printers may not recognize every control character sent from your computer.
The tab control character is used for aligning columns in a text file.

Related Terms

  • antonym: printable character - visible or graphic character in a text display
