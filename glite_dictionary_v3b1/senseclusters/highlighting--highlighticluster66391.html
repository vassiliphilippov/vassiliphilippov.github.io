highlighting

noun
UI marking of text or code
Pronunciation: / \u02c8ha\u026a\u02ccla\u026at\u026a\u014b / (AmE), / \u02c8ha\u026a\u02ccla\u026at\u026a\u014b / (BrE)
(in computing and user interfaces) the act or feature of marking text or code with color, background, or emphasis to show selection, syntax categories, or important parts.

Examples

Text highlighting
The code editor's highlighting shows keywords and strings in different colors for easier reading.
Her careful highlighting of the contract helped the team find key clauses quickly.
When I copy text, the browser's highlighting sometimes obscures nearby buttons on mobile screens.

Related Terms

  • synonym: syntax highlighting - coloring of source code in editors to show different syntax categories (keywords, strings, comments).
