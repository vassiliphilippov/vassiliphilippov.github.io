Synonyms

  • infectious agent
    4.2 0.2/m
    infectious agent is a microorganism or other cause that can spread infection and disease, while pathogen is any agent that causes disease in a host.
  • infective agent
    5.1 0.01/m
    infective agent often stresses the ability to start an infection, while pathogen is a general term for any organism that causes disease in a host.
  • germ
    1.0 6/m
    germ is a simple informal word for a tiny organism that may cause illness, while pathogen is a precise scientific term for a disease-causing agent.
