system call

noun
computer instruction that requests service from OS
8.7
Pronunciation: / ˈsɪstəm kɔl / (AmE), / ˈsɪstəm kɔːl / (BrE)
an instruction in a computer program that interrupts current execution to request a service from the operating system's supervisor or kernel

Examples

A system call request
File operations like reading or writing a file are typically performed using a system call in most operating systems.
If a system call fails, the program may receive an error code and respond accordingly.
The system call allows user programs to interact with hardware through the operating system.
