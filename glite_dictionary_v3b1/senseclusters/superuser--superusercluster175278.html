superuser

noun
expert system user
Pronunciation: / ˈsuːpərˌjuːzər / (AmE), / ˈsuːpəˌjuːzə / (BrE)
a user with the highest level of access or privileges on a computer system, able to perform all administrative tasks

Examples

System superuser
Only the superuser can install new software on the company's main server.
If you have a problem with your account, ask the superuser for help.
Becoming a superuser requires years of experience with the system.
Only the superuser can change important system settings on this server.
If you forget your password, you may need a superuser to reset it for you.
The superuser account should be protected with a strong password to prevent unauthorized access.

Related Terms

  • synonym: administrator - a person with control over a computer system or network
  • antonym: novice - a person new to and inexperienced in a system or activity
  • synonym: root - the main user account with full privileges on a Unix system
  • antonym: user - a person who uses a computer or service, usually with limited access
