ethernet cable

noun
cable for Ethernet network connections
7.2
Pronunciation: / ˈiːθɚˌnɛt ˈkeɪbəl / (AmE), / ˈiːθəˌnɛt ˈkeɪbəl / (BrE)
A cable used to connect devices within a local area network (LAN) using Ethernet protocol, typically consisting of twisted pair or coaxial wiring.

Examples

A ethernet cable connection
She purchased a longer ethernet cable to reach from the router to her desk.
The computer was not connected to the internet because the ethernet cable had been unplugged.
To ensure a stable connection, they recommended using an ethernet cable instead of relying on Wi-Fi.

Related Terms

  • synonym: network cable - a generic cable used for network connections
  • antonym: wireless - without wires or cables, using radio frequencies
