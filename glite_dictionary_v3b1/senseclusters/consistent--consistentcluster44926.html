consistent

adjective
coherent across replicas (computing)
7.3
Pronunciation: /kən.ˈsɪs.tənt/ (AmE), /kən.ˈsɪs.tənt/ (BrE)
(in distributed computing) maintaining a coherent, synchronized state across multiple replicas so that operations (especially reads) observe a single, agreed view of the data according to a specified consistency model.

Examples

Consistent data state
The distributed datastore guarantees consistent reads across all replicas despite concurrent updates.
To achieve consistent state, the cluster applies each transaction to every replica synchronously.
Under the strict protocol, nodes must coordinate to remain consistent after leader failover.

Related Terms

  • antonym: eventual consistency - a model where replicas converge to the same state only after some time; contrasts with immediate or strong consistency
  • synonym: synchronized - kept the same or coordinated across systems or processes
