BIOS

noun
computer startup software
6.9
Pronunciation: /ˈbaɪ.oʊs/ (AmE), /ˈbaɪ.əʊs/ /; / /ˈbaɪ.ɒs/ (BrE)
the basic software in a computer that starts the system and manages hardware before the operating system loads

Examples

BIOS settings
If your computer will not start, you may need to check the BIOS settings.
The technician updated the BIOS to fix compatibility issues with new hardware.
You can access the BIOS by pressing a specific key during the computer's startup.

Related Terms

  • synonym: firmware - software programmed into hardware devices
  • antonym: operating system - the main software that manages a computer's resources and user interface
