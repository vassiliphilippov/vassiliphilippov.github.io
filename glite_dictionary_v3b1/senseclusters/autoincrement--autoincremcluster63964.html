autoincrement

noun
automatic value increase
6.4
Pronunciation: / ˌɔːtoʊˈɪnkrəmənt / (AmE), / ˌɔːtəʊˈɪŋkrɪmənt / (BrE)
a feature in databases that automatically increases a numeric value, typically to assign unique identifiers

Examples

Database autoincrement feature
The autoincrement feature saves time by automatically assigning IDs to new records.
If you use autoincrement, you do not need to specify the primary key value yourself.
Many database systems support autoincrement columns for unique identification.

Related Terms

  • synonym: auto-increment - a feature that automatically increases a value in a database
  • antonym: manual entry - entering values by hand instead of automatically
  • synonym: sequence - a database object that generates a sequence of unique numbers
