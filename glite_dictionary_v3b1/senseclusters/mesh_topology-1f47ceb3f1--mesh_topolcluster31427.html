A network layout in which each component is directly connected to every other component.

Examples

A mesh topology network
A mesh topology ensures that every device in the network can communicate directly with every other device.
In a mesh topology, the failure of a single connection does not disrupt the entire network.
Many modern wireless networks use a mesh topology to increase reliability and coverage.
