attrib

noun
file attribute command
9.9
Pronunciation: / ˈætrɪb / (AmE), / ˈætrɪb / (BrE)
a command in MS-DOS and Windows used to display or change file attributes such as read-only or hidden status

Examples

Attrib command prompt
To make a file read-only, use the attrib command in the command prompt.
The attrib utility allows users to hide or unhide files in Windows.
He typed attrib -h filename to remove the hidden attribute from the document.

Related Terms

  • synonym: chmod - a command in Unix and Linux to change file permissions
  • antonym: del - a command in MS-DOS and Windows to delete files
