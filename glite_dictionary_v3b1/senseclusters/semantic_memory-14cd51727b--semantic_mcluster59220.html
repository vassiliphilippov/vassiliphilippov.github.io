semantic memory

noun
general knowledge memory
5.3 4.7
Pronunciation: /sɪ.ˈmæn.tɪk ˈmem.ə.riː/ (AmE), /sɪ.ˈmæn.tɪk ˈmem.ə.riː/ (BrE)
the part of long-term memory that stores general knowledge, concepts, and facts not tied to personal experiences

Examples

Semantic memory knowledge
Children develop semantic memory as they learn words and concepts in school.
Damage to the temporal lobe can affect a person's semantic memory, making it hard to recall factual information.
Unlike episodic memory, semantic memory holds knowledge that is not tied to a specific event in your life.

Related Terms

  • antonym: episodic memory - memory of specific events and experiences
