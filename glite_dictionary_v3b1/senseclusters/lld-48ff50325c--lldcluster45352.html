LLD

noun
software low-level design (abbrev.)
8.5
Pronunciation: / ccel ccel c8di / (AmE), / ccel ccel c8did0 / (BrE)
(software engineering) Abbreviation for low-level design: a detailed design document or phase that specifies components, modules, interfaces, data structures, and implementation details.

Examples

Project LLD document
The team drafted the LLD to guide coding after requirements approval.
During code review, she compared the implementation against the LLD to ensure conformity.
Before starting unit tests, they updated the LLD with the new interface and data models.

Related Terms

  • antonym: HLD - high-level design: the architectural overview and system structure that precedes low-level design
  • synonym: detailed design - a thorough specification of software components, interfaces, and data needed for implementation
