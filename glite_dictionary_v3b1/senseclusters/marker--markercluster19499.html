marker

noun
conspicuous object used to mark something
1.0 7.41/m
Pronunciation: / Ààm…ërk…ö / (AmE), / Ààm…ëÀêk…ô / (BrE)
an object that is conspicuous and used to indicate a position, boundary, or route

Examples

Roadside marker
Buoys serve as markers to guide ships through dangerous waters.
The red flag acted as a marker for the finish line of the race.
We placed a marker at the trailhead so hikers would know where to start.

Synonyms

  • guidepost

  • landmark

  • signpost
