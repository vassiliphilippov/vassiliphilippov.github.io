lut

noun
data mapping table
Pronunciation: / lʌt / (AmE), / lʌt / (BrE)
a table used in computing or digital imaging to map input values to output values, often for color correction or data conversion

Examples

Apply a lut filter
The video editor applied a lut to achieve a cinematic color effect in the final footage.
Using a lut can speed up image processing by avoiding complex calculations.
She downloaded a new lut to enhance the colors in her landscape photographs.

Related Terms

  • synonym: lookup table - a table used to substitute input values with output values in computing
