work unit

noun
task or data chunk in distributed computing
Pronunciation: / w5dd0k c8jud0.n6at / (AmE), / w5cd0k c8jud0.n6at / (BrE)
In computing, a discrete task or piece of data assigned to a worker (process, thread, or machine) in distributed or parallel computing for processing.

Examples

A work unit for processing
The grid server dispatched a work unit to each volunteer computer for processing.
In our MapReduce job, each mapper receives a single work unit to transform and forward.
To balance load, the scheduler split the dataset into identical work units for parallel workers.

Related Terms

  • synonym: job - a unit of work submitted for processing by a computer system
  • synonym: task - a piece of work to be done or undertaken
