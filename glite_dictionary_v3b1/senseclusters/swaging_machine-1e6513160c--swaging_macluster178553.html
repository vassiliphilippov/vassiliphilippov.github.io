swaging machine

noun
machine for shaping sheet metal
8.6
Pronunciation: / ˈsweɪdʒɪŋ məˈʃin / (AmE), / ˈsweɪdʒɪŋ məˈʃiːn / (BrE)
A machine used to shape sheet metal by means of rapidly closing and separating split dies.

Examples

A swaging machine tool
A modern swaging machine can process hundreds of parts per hour with great precision.
Before being assembled, each pipe end is shaped using a swaging machine.
The technician operated the swaging machine to form the metal tubes for the frame.
