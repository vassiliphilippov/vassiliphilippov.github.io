plug-in

noun
software module that extends an application
5.4
Pronunciation: / ˈplʌɡ ɪn / (AmE), / ˈplʌɡ ɪn / (BrE)
a software module that adds new features or extends the functionality of an existing computer application

Examples

Audio plug-in
After adding the plug-in, the music software could recognize more file formats.
The browser supports various plug-ins to extend its capabilities.
You can install a video editing plug-in to add new effects to the program.

Related Terms

  • synonym: add-on - additional software component that supplements a main program
  • synonym: extension - a small software program that adds features to an application
