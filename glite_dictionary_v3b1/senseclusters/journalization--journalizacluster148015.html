journalization

noun
recording changes in logs
Pronunciation: / ˌdʒɝːnələˈzeɪʃən / (AmE), / ˌdʒɜːnəlaɪˈzeɪʃən / (BrE)
the process of recording changes to files or data in a journal so a system can recover after a failure

Examples

System journalization of data
Modern file systems use journalization to help prevent data loss after unexpected shutdowns.
The database administrator explained how journalization ensures that recent transactions can be recovered.
Without journalization, restoring the system after a crash would be much more difficult.

Related Terms

  • synonym: backup - a copy of data kept for recovery after loss or failure
  • synonym: logging - the act of keeping a record of events or changes
