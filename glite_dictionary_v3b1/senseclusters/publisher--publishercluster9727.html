publisher

noun
software component that sends messages (pub/sub)
4.4
Pronunciation: / c8p8cbl6a8359r / (AmE), / c8p8cbl6a8359 / (BrE)
(in computing) a software component or service that sends or makes available messages to a topic or channel so that subscribers can receive them in a publish/subscribe messaging system.

Examples

Software publisher sends data
The data system uses a publisher to broadcast updates to subscribers.
In the architecture diagram, each microservice acts as a publisher for event notifications.
When the sensor detected motion, the publisher immediately pushed the message to the topic.

Related Terms

  • antonym: subscriber - a component that receives messages from a topic or channel in a publish/subscribe system
