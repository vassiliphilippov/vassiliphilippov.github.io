dead set against

adjective
firmly opposed to something
Pronunciation: / dɛd sɛt əˈɡɛnst / (AmE), / dɛd sɛt əˈɡɛnst / (BrE)
firmly and completely opposed to something

Examples

Dead set against changes
He remained dead set against the proposed changes despite everyone's arguments.
Her parents are dead set against her traveling alone.
I'm dead set against allowing smartphones in class unless needed for learning.

Related Terms

  • synonym: opposed to - holding an opposite or resisting position
