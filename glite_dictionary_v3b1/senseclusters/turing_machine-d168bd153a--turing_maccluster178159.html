Turing machine

noun
hypothetical computing device model
Pronunciation: / ˈtʊrɪŋ məˌʃin / (AmE), / ˈtjʊərɪŋ məˌʃiːn / (BrE)
a hypothetical computing device with an infinitely long memory tape, used to formalize the concept of computation

Examples

A Turing machine model
A Turing machine is often used to illustrate the limits of what can be computed.
Researchers study how a Turing machine simulates algorithms step by step.
The concept of the Turing machine is fundamental to theoretical computer science.
