localhost

noun
network loopback address
Pronunciation: / ˈloʊkəlˌhəʊst /; / ˈloʊkəlˌhoʊst / (AmE), / ˈləʊkəlˌhəʊst / (BrE)
the standard network address or hostname that refers to the computer or device you are currently using, often used for testing or local services

Examples

localhost address
Developers often use localhost to test web applications before making them public.
The database is configured to accept connections only from localhost for security reasons.

When you type 'localhost' in your browser, it connects to your own localhost server.

Related Terms

  • synonym: loopback - a network interface that routes signals back to the source device
  • antonym: remote host - a computer or server accessed over a network from another device
