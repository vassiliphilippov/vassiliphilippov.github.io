decimal point

noun
dot in a decimal number
1.0
Pronunciation: / ˈdɛsɪməl pɔɪnt / (AmE), / ˈdɛsɪməl pɔɪnt / (BrE)
the dot used to separate the whole number part from the fractional part in a decimal number

Examples

The decimal point symbol
In some European countries, a comma is used instead of a decimal point.
