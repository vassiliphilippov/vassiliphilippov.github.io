bool

noun
binary data type
7.2
Pronunciation: / buːl / (AmE), / buːl / (BrE)
a data type in computing that can have only two possible values: true or false

Examples

A bool variable
In many programming languages, a bool variable can only be true or false.
You should use a bool to store the result of a yes-or-no question.
The function returns a bool indicating whether the file exists.

Related Terms
