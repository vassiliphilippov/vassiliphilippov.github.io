monadic

adjective
single unit
9.8
Pronunciation: /mə.ˈnæd.ɪk/ (AmE), /mə.ˈnæd.ɪk/ /; / /mɒ.ˈnæd.ɪk/ (BrE)
consisting of or relating to a single unit, element, or something that involves only one input or argument

Examples

Monadic function
In mathematics, a monadic function takes only one argument.
The philosopher discussed the monadic nature of consciousness in his lecture.
Each process in the system is treated as a monadic entity, independent from the others.
In logic, a monadic predicate applies to only one subject at a time.
The programming language supports both monadic and polyadic functions for different operations.
A monadic operator processes a single input value to produce a result.
