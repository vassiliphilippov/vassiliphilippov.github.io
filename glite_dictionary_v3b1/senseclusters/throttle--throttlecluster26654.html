throttle

noun
software/network rate limiter
6.9
Pronunciation: /ˈθrɑː.təl/ /; / /ˈθrɔː.təl/ (AmE), /ˈθrɒ.təl/ (BrE)
A software or network mechanism that limits the rate of actions (such as API requests or data transfer) to prevent overload or ensure fair use.

Examples

API throttle
The cloud provider applied a throttle to prevent any single user from overloading the service.
You may hit the API throttle during peak hours, so space out your requests.
Engineers added a per-minute throttle that limits data transfer during backup windows.

Related Terms

  • synonym: rate limit - a restriction on the number of requests or amount of data allowed in a given time period
  • synonym: throttling - the act or process of limiting the rate of data or requests
