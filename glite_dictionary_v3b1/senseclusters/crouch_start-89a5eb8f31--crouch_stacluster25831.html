crouch start

noun
low starting position in sprinting
5.6
Pronunciation: / kraʊtʃ stɑrt / (AmE), / kraʊtʃ stɑːt / (BrE)
A low starting position used by sprinters in short-distance races, in which the runner crouches close to the ground before the race begins.

Examples

The crouch start position
Coaches teach young runners how to master the crouch start technique.
Most sprinters prefer the crouch start for better acceleration at the beginning of a race.
The athlete assumed a crouch start before the starter's pistol was fired.

Related Terms

  • antonym: standing start - an upright starting position used in some races
