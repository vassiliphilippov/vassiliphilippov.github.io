function

noun
mathematical relation with single-valued outputs
1.0
Pronunciation: / ˈfʌŋkʃən / (AmE), / ˈfʌŋkʃən / (BrE)
(mathematics, science, or computing) a relationship, process, or sequence that links inputs to outputs, or a named part of a computer program that performs a specific task

Examples

A math function
Every value in the domain must correspond to one output in a function.
In calculus, the derivative measures how a function changes as its input changes.
The square root function is defined only for non-negative numbers.
Interest paid on a loan is a function of the principal and the interest rate.
The amount of light received is a function of the time of day.
The efficiency of the machine varies as a function of temperature.
Each function in the code is responsible for a different calculation.
The print function allows you to display text on the screen.
You should write a separate function to handle user input.
