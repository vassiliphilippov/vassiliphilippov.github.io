nonvolatile storage

noun
memory retaining data without power
1.0
Pronunciation: / nɑnˈvɑlət̬aɪl ˈstɔɹɪdʒ / (AmE), / nɒnˈvɒlətaɪl ˈstɔːrɪdʒ / (BrE)
computer memory or storage that retains data even when power is turned off

Examples

Nonvolatile storage memory
Flash memory is a popular form of nonvolatile storage in modern electronics.
Hard drives and solid-state drives are types of nonvolatile storage commonly used in computers.
Unlike RAM, nonvolatile storage preserves your files when the device is powered down.

Related Terms

  • synonym: flash memory - a type of nonvolatile storage used in computers and devices
  • antonym: volatile storage - storage that loses data when power is turned off
