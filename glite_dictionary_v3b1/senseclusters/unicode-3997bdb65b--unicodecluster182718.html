Unicode

adjective
character encoding standard
5.2
Pronunciation: / ˈjuːnɪˌkoʊd / (AmE), / ˈjuːnɪkəʊd / (BrE)
using or designed for the Unicode standard, able to represent characters from many writing systems.

Examples

Unicode format
The document must be saved in Unicode format to support international characters.
Many modern programming languages use Unicode strings by default.
A Unicode font can display text in dozens of different scripts.

Related Terms

  • antonym: ASCII - a character encoding standard for English characters only
  • synonym: multilingual - able to use or support several languages
