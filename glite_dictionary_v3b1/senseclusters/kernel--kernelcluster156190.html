kernel

noun
function computing similarity in ML
8.3
Pronunciation: /ˈkɜːr.nəl/ (AmE), /ˈkɜː.nəl/ (BrE)
(in mathematics and machine learning) a function that computes a measure of similarity between inputs, often corresponding to an inner product in a transformed feature space.

Examples

Gaussian kernel function
The Gaussian kernel enables the SVM to operate in a higher-dimensional feature space.
Researchers compare different kernels to see which similarity measure improves classification accuracy.
When the feature map is unknown, they use a kernel to compute inner products implicitly.
