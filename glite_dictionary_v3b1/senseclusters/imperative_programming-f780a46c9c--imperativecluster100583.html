imperative programming

noun
programming style using commands to change state
5.2 5.9
Pronunciation: /ɪm.ˈper.ə.tɪv ˈproʊ.græm.ɪŋ/ (AmE), /ɪm.ˈper.ə.tɪv ˈprəʊ.græm.ɪŋ/ (BrE)
a programming paradigm that describes computation in terms of statements that change the program's state

Examples

Imperative programming style
Learning imperative programming helps new programmers understand how computers execute instructions step by step.
Many early computer languages, such as Fortran and C, support imperative programming as their core paradigm.
Unlike functional programming, imperative programming relies on changing variables and program state.

Related Terms

  • antonym: declarative programming - a programming paradigm expressing logic without describing control flow
  • degree lower: procedural programming - a type of imperative programming using procedures or routines
