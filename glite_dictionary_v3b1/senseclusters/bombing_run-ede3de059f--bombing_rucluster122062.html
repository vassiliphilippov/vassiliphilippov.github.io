bombing run

noun
part of a bombing mission
Pronunciation: / ˈbɑmɪŋ ˌrʌn / (AmE), / ˈbɒmɪŋ ˌrʌn / (BrE)
the section of a bombing mission that begins with the approach to the target and ends with the release of the bombs

Examples

A bombing run mission
Each bombing run required precise coordination between the pilot and the bombardier.
Enemy fire intensified as the aircraft made its bombing run over the target.
The pilot focused intensely during the bombing run to ensure maximum accuracy.
