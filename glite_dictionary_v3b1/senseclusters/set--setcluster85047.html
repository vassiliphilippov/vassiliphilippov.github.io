set

noun
scenery and props for a play or film
6.0 17/m

Examples

Synonyms

  • stage set
    3.6 0.33/m
    stage set is the built scenery and props used to create a scene, while set is the general production word for that location.
  • stage setting
    3.6 0.12/m
    stage setting is the way scenery and props are arranged to show a location, while set names the actual built scene used.
  • setting
    3.0 1.90/m
    setting is the arranged scenery and props that represent a place in a production, while set is the physical scenery built for that place.
