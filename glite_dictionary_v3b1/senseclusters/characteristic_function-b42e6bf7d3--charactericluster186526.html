Pronunciation: / kɛrəktəˌrɪstɪk ˈfʌŋkʃən / (AmE), / kærɪktəˌrɪstɪk ˈfʌŋkʃən / (BrE)
(Electronics) A graph that shows how a particular property or characteristic of a device changes with respect to other parameters.

Examples

The characteristic function graph
By plotting the characteristic function, we can understand how the output changes with varying input voltage.
Each diode type has a unique characteristic function that helps determine its behavior in a circuit.
The engineer analyzed the characteristic function of the transistor to predict its performance.
