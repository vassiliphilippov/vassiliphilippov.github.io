round file

noun
cylindrical file for holes or curves
Pronunciation: / raʊnd faɪl / (AmE), / raʊnd faɪl / (BrE)
a file with a cylindrical shape, typically used for smoothing or enlarging the inside of holes or curved surfaces

Examples

A round file tool
A round file is ideal for shaping the inside of curved surfaces.
He used a round file to smooth the edges of the drilled hole.
The mechanic reached for a round file to enlarge the opening slightly.

Related Terms

  • synonym: rat-tail file - a long, thin, round file used for similar purposes
