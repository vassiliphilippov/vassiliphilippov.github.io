ECB

noun
encryption mode
9.6
Pronunciation: / iː siː biː / (AmE), / iː siː biː / (BrE)
a mode of operation for block ciphers in cryptography where each block of data is encrypted independently

Examples

ECB encryption mode
The security expert warned against using ECB for sensitive data because of its vulnerabilities.
In cryptography classes, students learn why ECB is not recommended for most applications.
The software defaulted to ECB mode, which made the encrypted files less secure.

Related Terms

  • antonym: CBC - a block cipher mode where each block is combined with the previous ciphertext block before encryption
  • synonym: block cipher - a method of encrypting data in fixed-size blocks
