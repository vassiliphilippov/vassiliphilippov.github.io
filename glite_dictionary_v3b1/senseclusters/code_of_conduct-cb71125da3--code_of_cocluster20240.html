code of conduct

noun
rules for group or organizational behavior
Pronunciation: / koʊd əv ˈkɑːndʌkt / (AmE), / kəʊd əv ˈkɒndʌkt / (BrE)
a set of rules or principles that guide the behavior of members of a particular group or organization

Examples

Company code of conduct
All employees must follow the company's code of conduct at all times.
Members are expected to adhere strictly to the code of conduct.
The school introduced a new code of conduct to improve student behavior.

Related Terms

  • synonym: code - a set of rules or principles agreed upon by a group
