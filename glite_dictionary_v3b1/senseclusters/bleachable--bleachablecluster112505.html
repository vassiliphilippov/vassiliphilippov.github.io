bleachable

adjective
can be bleached
3.7
Pronunciation: /ˈbliː.tʃə.bəl/ (AmE), /ˈbliː.tʃə.bəl/ (BrE)
able to be safely cleaned, disinfected, or made whiter using bleach without being damaged

Examples

Bleachable fabric
Always check if the fabric is bleachable before using any strong cleaning products.
Some stains are easier to remove from bleachable materials than from delicate ones.
The manufacturer recommends only using bleachable surfaces in hospital settings for safety.
Only bleachable fabrics should be washed with strong chemicals.
The label says this shirt is bleachable, so you can use whitening agents safely.
Not all stains will come out, even on bleachable materials.
