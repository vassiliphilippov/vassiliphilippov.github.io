generic

noun
parameterized type or function (programming)
Pronunciation: / dʒəˈnərɪk / (AmE), / dʒəˈnərɪk / (BrE)
(in programming) a type, class, or function parameterized by one or more type variables so the same code can operate on different data types while preserving type safety.

Examples

Generic type
Many programming languages now use generic types to implement reusable, type-safe collections.
In Java, you declare a generic class with angle brackets to accept different element types.
When designing the API, she preferred using a generic parameter to avoid repeating code for each type.

Related Terms

  • synonym: parametric polymorphism - a programming concept that allows functions and types to be written generically to operate on any type
  • synonym: template - the C++ mechanism for generic programming; similar idea of parameterizing code by type
  • synonym: type parameter - a variable representing a type used to parameterize generic types or functions
