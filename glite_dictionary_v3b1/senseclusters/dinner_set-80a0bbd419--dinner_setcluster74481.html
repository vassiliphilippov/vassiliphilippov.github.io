dinner set

noun
matching tableware for dinner
1.0
Pronunciation: / ˈdɪnər sɛt / (AmE), / ˈdɪnə sɛt / (BrE)
A matching set of dishes, plates, and utensils used for serving and eating dinner.

Examples

A dinner set of dishes
She bought a beautiful new dinner set for special occasions.
The dinner set includes plates, bowls, and cups for eight people.
They received a porcelain dinner set as a wedding gift.
