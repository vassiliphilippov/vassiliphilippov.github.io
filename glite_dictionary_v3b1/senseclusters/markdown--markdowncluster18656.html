markdown

noun
plain-text markup language
3.8
Pronunciation: / ˈmɑrkˌdaʊn / (AmE), / ˈmɑːkˌdaʊn / (BrE)
a lightweight plain-text markup language used to add simple formatting such as headings, lists, and links to text, especially for web content

Examples

Markdown file format
Many developers write documentation in markdown because it is easy to read and convert to HTML.
The website allows users to format their posts using markdown for headings and lists.
She learned how to use markdown to make her notes more organized and readable.

Related Terms

  • synonym: HTML - a standard markup language for creating web pages
  • degree lower: plaintext - text without any formatting or markup
