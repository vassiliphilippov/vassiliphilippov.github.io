reconsolidate

verb
to stabilize recalled memory
8.8
Pronunciation: / ˌriːkənˈsɑːlɪdeɪt / (AmE), / ˌriːkɒnˈsɒlɪdeɪt / (BrE)
to re-stabilize a memory after it has been recalled, allowing it to be stored again in long-term memory.

Examples

To reconsolidate a memory
Therapists use special techniques to help patients reconsolidate painful memories in a safer way.
After recalling the event, the brain must reconsolidate the memory to keep it stored long-term.
Scientists study how sleep can affect the brain's ability to reconsolidate information learned during the day.
