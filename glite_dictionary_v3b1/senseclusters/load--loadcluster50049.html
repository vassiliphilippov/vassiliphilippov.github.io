load

verb
transfer data from storage to computer memory
2.7
Pronunciation: / loʊd / (AmE), / ləʊd / (BrE)
(of a computer or program) to transfer data or a program from a storage device into the computer's memory for use or execution.

Examples

To load a file
He accidentally forgot to load the document from the hard drive.
Please load the software before running the program.
The file takes a few seconds to load into memory.

Related Terms

  • antonym: download - to transfer data from a remote computer to a local system or device
  • antonym: upload - to transfer data from a local computer to a remote system
