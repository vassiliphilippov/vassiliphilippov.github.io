virtual memory

noun
simulated computer memory using disk storage
1.0
Pronunciation: / ˈvɝtʃuəl ˈmɛməri / (AmE), / ˈvəːtʃʊəl ˈmɛməri / (BrE)
A computing technique in which part of a computer's hard disk is used to simulate additional main memory, allowing programs to access more memory than is physically installed.

Examples

Enable virtual memory
Enabling extra virtual memory can sometimes improve system performance.
If your computer runs out of RAM, it relies on virtual memory to store excess data.
The operating system uses virtual memory to run large applications efficiently.

Related Terms

  • synonym: RAM - random-access memory; the main memory in computers
