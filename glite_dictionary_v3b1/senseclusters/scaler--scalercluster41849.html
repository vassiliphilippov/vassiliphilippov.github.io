scaler

noun
machine-learning feature-scaling tool
7.4
Pronunciation: /ˈskeɪ.lər/ (AmE), /ˈskeɪ.lə/ (BrE)
(in computing, especially machine learning) a software tool or component that rescales or normalizes numerical features during data preprocessing (for example, by standardization or min–max normalization).

Examples

Data scaler normalizes features
We applied a scaler to normalize the features before training the model.
Modern ML libraries include several built-in scalers for standardization and normalization tasks.
When tuning the pipeline, she compared a min–max scaler with a Gaussian transformer for model performance.

Related Terms

  • synonym: normalizer - a tool or function that rescales numeric data to a common range or distribution
  • synonym: standardizer - a scaler that transforms features to have zero mean and unit variance (standardization)
