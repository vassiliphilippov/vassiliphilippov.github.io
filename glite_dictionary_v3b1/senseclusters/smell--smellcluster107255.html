smell

noun
software design indicator
7.9 0.11/m
Pronunciation: / smÉ›l / (AmE), / smÉ›l / (BrE)
(in software engineering) a symptom in source code that suggests a deeper problem in design, structure, or maintainability and often indicates the need for refactoring.

Examples

Code smell
During the review, she pointed out a smell in the authentication module that could cause future bugs.
Many teams try to refactor when they detect a smell that makes the code hard to change.
Experienced developers can often spot a smell quickly and suggest improvements before releases.

Related Terms

  • synonym: code smell - a pattern in source code that suggests a deeper design or maintainability problem
  • synonym: design flaw - a weakness in software design that can lead to bugs or difficulty maintaining the code
