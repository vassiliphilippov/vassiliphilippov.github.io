overhead

noun
administrative resource use in computing
5.3
Pronunciation: / ˈoʊvərˌhɛd / (AmE), / ˈəʊvəˌhɛd /; / ˈəʊvəhɛd / (BrE)
the extra processing time, disk space, or resources required by a computer system for administrative or non-user purposes

Examples

System overhead
Increasing block size can minimize storage overhead in certain databases.
The backup process failed because the overhead exceeded the maximum allowed disk usage.
The file system's overhead reduces the amount of space available for user data on the disk.
Reducing system overhead can lead to more efficient processing and quicker command responses.
The new encryption software introduces significant overhead, making file transfers slightly slower.
When optimizing the code, developers worked to minimize the overhead required during data retrieval.

Related Terms

  • synonym: latency - the delay before a transfer of data begins
  • synonym: metadata - data that provides information about other data
  • antonym: throughput - the amount of work performed by a system in a given period
