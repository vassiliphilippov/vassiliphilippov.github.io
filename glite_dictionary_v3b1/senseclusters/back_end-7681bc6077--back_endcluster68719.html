back end

noun
server-side of a software system
3.7
Pronunciation: /bæk end/ /; / /bæk ənd/ (AmE), /bæk end/ /; / /bæk ənd/ (BrE)
the server-side part of a software application or system that processes and stores data, runs business logic, and provides services (APIs), as opposed to the front end (user interface).

Examples

Back end server
The company migrated critical services to the back end to improve scalability and security.
On the new app, the back end handles database queries, authentication, and business logic for requests.
Developers updated the back end during the release, deploying a new API and optimizing response times.

Related Terms

  • antonym: front end - the part of a software system that users interact with (user interface); often contrasted with the back end
  • synonym: server-side - components or code that run on servers to process data and handle requests
