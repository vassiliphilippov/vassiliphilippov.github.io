diff

noun
file showing changes
Pronunciation: / dɪf / (AmE), / dɪf / (BrE)
a file or output that displays the differences between two versions of text or code, typically produced by a comparison tool

A code diff
The programmer reviewed the diff before approving the changes to the project.
Please attach the diff so I can see exactly what was modified in the document.
After running the tool, the diff highlighted every line that had been updated.
