ideographic script

noun
writing system based on idea-symbols
5.9
Pronunciation: / ˌaɪdiəˈɡræfɪk skrɪpt / (AmE), / ˌaɪdɪəˈɡrafɪk skrɪpt / (BrE)
a writing system in which symbols (ideograms) represent ideas or concepts, rather than specific words or sounds

Examples

An ideographic script system
Ancient Egyptian hieroglyphs were once thought to be a purely ideographic script rather than representing sounds.
Learning an ideographic script can be challenging because each symbol may represent a complex concept.
Scholars debated whether Chinese writing should be classified as an ideographic script or a logographic one.

Related Terms

  • antonym: alphabet - writing system using letters to represent sounds
  • synonym: logographic script - writing system where each symbol represents a word or morpheme
  • antonym: syllabary - writing system where symbols represent syllables
