vertical file

noun
filing cabinet with upright folders
7.1 4.7
Pronunciation: /ˈvɜːr.tɪ.kəl faɪl/ /; / /ˈvɜːr.tə.kəl faɪl/ (AmE), /ˈvɜː.tɪ.kəl faɪl/ /; / /ˈvɜː.tə.kəl faɪl/ (BrE)
