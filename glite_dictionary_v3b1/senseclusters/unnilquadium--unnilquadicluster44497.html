unnilquadium

noun
systematic name for element 104 (rutherfordium)
Pronunciation: / ˌʌn.nɪlˈkwɑːdiəm / (AmE), / ˌʌn.nɪlˈkwæd.i.əm / (BrE)
A synthetic radioactive transuranic element with atomic number 104, previously used as a temporary systematic name for rutherfordium.

Examples

Unnilquadium element 104
Prior to being officially named rutherfordium, unnilquadium appeared in scientific papers to reference element 104.
Scientists debated the discovery and naming rights for unnilquadium for several years.
The element unnilquadium was first synthesized in the 1960s by bombarding plutonium with neon ions.
