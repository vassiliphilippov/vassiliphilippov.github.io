deprecate

verb
mark a software feature as obsolete
7.9 0.3/m
Pronunciation: / \u02c8d\u025bpr\u0259\u02ccke\u026at / (AmE), / \u02c8d\u025bpr\u026ake\u026at / (BrE)
(in software) to mark a feature, API, or function as discouraged for use and intended to be removed or unsupported in a future release; to signal that developers should migrate to alternatives.

Examples

To deprecate a feature
The library maintainers decided to deprecate the old API and document alternatives for developers.
After the security flaw, the vendor deprecated several functions slated for removal in the next release.
Many teams avoid deprecating features without offering a migration path for existing users.

Related Terms

  • synonym: obsolete - no longer supported or recommended for use; typically replaced by a newer alternative
  • antonym: support - to actively maintain, recommend, or continue providing updates for a feature or API
