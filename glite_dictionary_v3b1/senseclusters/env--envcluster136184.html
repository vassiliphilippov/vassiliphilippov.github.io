env

noun
computing abbreviation
Pronunciation: / ɛnv / (AmE), / ɛnv / (BrE)
an abbreviation for 'environment', especially used in computing to refer to environment variables or configuration files

Examples

Env file settings
Before running the program, check the env file for the correct settings.
Developers often use an env variable to store sensitive information like passwords.
