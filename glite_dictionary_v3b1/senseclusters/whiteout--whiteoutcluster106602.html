whiteout

noun
overlay filesystem deletion marker
9.3
Pronunciation: /ˈwaɪ.taʊt/ /; / /ˈwaɪt.aʊt/ (AmE), /ˈwaɪ.taʊt/ /; / /ˈwaɪt.aʊt/ (BrE)
(computing) a special marker or placeholder in a layered (overlay/union) filesystem that indicates a file from a lower layer has been deleted and should be hidden in the merged view.

Examples

Filesystem whiteout marker
In overlayfs, a whiteout hides a lower-layer file so the merged view omits it.
The container image used a whiteout to remove legacy configuration files during the update.
When inspecting the union mount, she found several whiteout entries indicating deleted files.

Related Terms

  • synonym: delete - to remove a file or information so it no longer appears or is accessible
  • synonym: hide - to make a file or item not visible in the merged filesystem view
