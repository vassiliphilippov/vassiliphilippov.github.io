local call

noun
function call within same process
3.7
Pronunciation: /ˈloʊ.kəl kɔːl/ /; / /ˈloʊ.kəl kɑːl/ (AmE), /ˈləʊ.kəl kɔːl/ (BrE)
(in computing) a function or procedure call that executes within the same process or address space, not across a network or between processes.

Examples

A local call in code
The compiler optimizes a local call by inlining it to eliminate function-call overhead.
When debugging, she stepped through each local call to verify the correct variable values were passed.
Microservices turned many local calls into remote requests across containers during migration.

Related Terms

  • synonym: function call - an invocation of a subroutine, function, or procedure in program code
  • antonym: remote call - a call that invokes code in a different process or over a network, often with higher latency
