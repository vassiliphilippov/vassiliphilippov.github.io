value judgment

noun
an opinion based on personal values
2.2 5.0
Pronunciation: / ˈvælju ˈdʒʌdʒmənt /; / ˈvælju ˌdʒʌdʒmənt / (AmE), / ˈvæljuː ˈdʒʌdʒmənt /; / ˈvalju ˌdʒʌdʒmənt / (BrE)
an opinion or assessment based on personal values or standards rather than objective facts

Examples
