literalization

noun
making data literal
9.4
Pronunciation: /lɪt.ə.rə.lə.ˈzeɪ.ʃən/ (AmE), /lɪt.ə.rə.lə.ˈzeɪ.ʃən/ (BrE)
the process of converting data, code, or text into a literal form, especially in programming by turning values into string or character literals

Examples

Literalization of data
The literalization of variables made the code easier to read and debug.

During the export process, the tool performs literalization to store values as plain text.
