data structure

noun
organization of data in computing
Pronunciation: / ˈdeɪ.tə ˌstrʌk.tʃɚ / (AmE), / ˈdeɪ.tə ˌstrʌk.tʃə / (BrE)
An organized way of storing, managing, and arranging data in a computer so it can be accessed and modified efficiently.

Examples

A data structure in programming
A stack is a common data structure used in programming for storing temporary data.
The choice of data structure can greatly affect how quickly an algorithm operates.
Understanding the data structure of a program is crucial for optimizing its performance.
