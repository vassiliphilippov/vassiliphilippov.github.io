gzip

noun
file compression tool
6.3
Pronunciation: / ˈdʒiːˌzɪp / (AmE), / ˈdʒiːˌzɪp / (BrE)
a software program and file format used to compress and decompress computer files

Examples

Gzip file format
Before uploading large files, it is common to use gzip to reduce their size.
The server automatically compresses log files with gzip every night.
You can extract the contents by running the gzip command in the terminal.

Related Terms

  • antonym: decompress - to restore compressed files to their original state
  • synonym: zip - a popular file compression format and program
