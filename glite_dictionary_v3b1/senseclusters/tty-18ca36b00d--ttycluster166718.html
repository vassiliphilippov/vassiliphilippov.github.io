TTY

noun
computer terminal interface
Pronunciation: / ˈtiː ˈtiː ˈwaɪ / (AmE), / ˈtiː ˈtiː ˈwaɪ / (BrE)
a computer terminal or interface used for user interaction, especially in Unix and Linux systems

Examples

Unix TTY terminal
The system administrator logged in through the TTY to check the server status.
If the graphical interface fails, you can use a TTY to access the command line.
Each user session on the Unix server is assigned a separate TTY.

Related Terms

  • synonym: console - a text-based interface for interacting with a computer system
  • synonym: terminal - a device or program for entering data into and displaying data from a computer
