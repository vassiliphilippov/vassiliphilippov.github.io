(in computer graphics) a flat 2D image or textured quad placed in a 3D scene that always faces the camera, used to represent distant or numerous objects efficiently.

Examples

3D billboard image
The game engine uses a billboard for distant trees that always face the camera.
To improve performance, the artist replaced many background models with billboards that use a single texture.
In the tutorial, we learned how to align a billboard so it correctly faces the player's viewpoint.
