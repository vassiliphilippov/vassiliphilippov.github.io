plugin

noun
software add-on
Pronunciation: / ˈplʌɡˌɪn / (AmE), / ˈplʌɡˌɪn / (BrE)
a small software component that adds specific features or functions to a larger computer program

Examples

Browser plugin
You can install a plugin to add new functions to your browser.
The music software supports many plugins for different sound effects.
After adding the plugin, the program could read more file formats.

Related Terms

  • synonym: add-on - an extra piece of software that provides additional features
  • synonym: extension - a software module that adds capabilities to a program
