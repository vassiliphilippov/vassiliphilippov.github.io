holdable

adjective
able to be conducted
Pronunciation: / ˈhoʊldəbəl / (AmE), / ˈhəʊldəbəl / (BrE)
able to be organized, conducted, or performed, especially referring to an event, meeting, or activity.

Examples

Holdable meeting
The conference is only holdable if enough participants register in advance.
Outdoor concerts are not holdable during severe weather conditions.
The meeting became holdable once the technical issues were resolved.

Related Terms

  • antonym: impossible - not able to be done or achieved
  • synonym: possible - able to be done or achieved
