demo

noun
trial version of software or game
3.2 1.71/m
Pronunciation: / \u02c8d\u025bmo\u028a / (AmE), / \u02c8d\u025bm\u0259\u028a / (BrE)
a limited or sample version of a program, app, or game released so users can try its features before buying, often with some features disabled.

Examples

Game demo
You can download the demo of the game to try before you buy.
The studio released a free demo featuring the game's first level and a short tutorial.
He tried several demos before choosing which software best met his needs.

Synonyms

  • trial
    3.3 0.39/m
    a trial lets users test software or services fully for a short period, while a demo is a limited sample with disabled features.
  • test drive
    3.2 0.25/m
    a test drive lets people try a product or service in real use, while a demo is a short, limited software sample.

Related Terms

  • antonym: full version - the complete edition of software with all features enabled
  • synonym: trial version - a limited copy of software provided to let users test features before purchase
