relocatable program

noun
computer program that runs from different memory locations
Pronunciation: / ˌriːˈloʊkəteɪtəbl ˈproʊɡræm / (AmE), / ˌriːˈləʊkəteɪtəbl ˈprəʊɡram / (BrE)
a computer program that can be loaded into and run from different memory locations without modification

Examples

A relocatable program file
Many compilers generate object code as a relocatable program for flexibility in system design.
The operating system makes use of a relocatable program to efficiently manage memory allocation.
Using a relocatable program, developers can avoid conflicts when loading software modules into memory.

Related Terms

  • synonym: load module - a program ready to be loaded into memory
