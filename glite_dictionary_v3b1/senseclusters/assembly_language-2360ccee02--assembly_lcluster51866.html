assembly language

noun
low-level programming language close to machine code
3.9 6.4
Pronunciation: /ə.ˈsem.blɪ ˈlæŋ.ɡwɪdʒ/ (AmE), /ə.ˈsem.blɪ ˈlæŋ.ɡwɪdʒ/ (BrE)
a low-level programming language that is closely related to machine code and requires instructions to be written in a form specific to a computer's architecture

Examples

Assembly language programming
Learning assembly language can help programmers understand how computers execute instructions at the hardware level.
Modern developers rarely use assembly language except when optimizing code for critical tasks.
The software was originally written in assembly language to ensure maximum performance on early computers.

Related Terms

  • degree higher: high-level language - a programming language that is more abstracted from machine code than assembly language
  • degree lower: machine language - the most basic computer programming language, consisting of binary code
