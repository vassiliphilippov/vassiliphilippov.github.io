(in software development, especially version control) to retrieve previously shelved or stashed code changes and apply them to the current working copy so they can be tested or merged.

Examples

To unshelve code changes
Before merging, she decided to unshelve the feature branch to run the tests locally.
The team lead asked him to unshelve the changes and resolve the merge conflicts tonight.
After reviewing feedback, he unshelved the patch so the CI could rebuild and pass.

Related Terms

  • synonym: apply - to put changes or a patch into effect in the current working code
  • synonym: restore - to bring back something to a previous state; used for retrieving saved data or changes
  • antonym: shelve - to set aside or temporarily store code changes (often as a shelve or stash); the opposite of unshelve
