Cartesian coordinate system

noun
system for locating points using perpendicular axes
8.2 4.2
Pronunciation: /kɑːr.ˈtiː.ʒən koʊ.ˈɔːr.də.nət ˈsɪs.təm/ /; / /kɑːr.ˈtiː.ʒən koʊ.ˈɔːr.də.neɪt ˈsɪs.təm/ (AmE), /kɑː.ˈtiː.ʒən kəʊ.ˈɔː.dɪ.nət ˈsɪs.təm/ /; / /kɑː.ˈtiː.ʒən kəʊ.ˈɔː.dɪ.neɪt ˈsɪs.təm/ (BrE)
A system for locating points in a plane or space by using their distances from two or more perpendicular axes that intersect at an origin.

Examples

The Cartesian coordinate system
Engineers often rely on the Cartesian coordinate system when designing blueprints.
The Cartesian coordinate system makes it easy to plot points on a graph using x and y values.
The position of the object was identified in the Cartesian coordinate system with coordinates (3, -2).

Related Terms

  • antonym: polar coordinate system - a coordinate system in which each point is determined by a distance from a fixed point and an angle from a fixed direction
