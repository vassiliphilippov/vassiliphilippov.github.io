expected value

noun
mean value of a random variable
Pronunciation: / ɪkˈspɛktɪd ˈvælju / (AmE), / ɪkˈspɛktɪd ˈvæljuː / (BrE)
the mean of a random variable, calculated as the sum of all possible values each multiplied by its probability of occurrence

Examples

The expected value formula
In probability theory, the expected value provides a measure of the center of a random distribution.
Investors often use expected value to assess potential financial outcomes.
The expected value of rolling a fair die is 3.5.

Related Terms

  • synonym: mean - average value of a set of numbers
