(computing) A hardware device or software component that acts as an entry point between networks or services, routing, translating, or forwarding requests and data between different systems or protocols.

Examples

Network gateway device
The cloud platform uses an API gateway to route requests to microservices.
To secure the network, engineers placed a new gateway between the internal and external servers.
When the protocol changed, the team updated the gateway to translate calls into the new format.

Related Terms

  • synonym: proxy - a server or service that acts as an intermediary for requests from clients seeking resources from other servers
  • synonym: router - a device that forwards data packets between computer networks, often based on IP addresses
