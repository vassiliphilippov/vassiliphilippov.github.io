table

noun
set of data arranged in rows and columns
Pronunciation: / ˈteɪbəl / (AmE), / ˈteɪbl / (BrE)
a set of data or information systematically arranged in rows and columns, often for easy reference or comparison

Examples

Data table
All the survey responses were entered into a table for analysis.
Please refer to table 2 for the test results.
The financial report includes a table summarizing last year's expenses and revenues.
