groupware

noun
collaborative software for group work
Pronunciation: / ˈɡruːpˌwɛr / (AmE), / ˈɡruːpˌwɛə / (BrE)
software designed to help people working on a common task achieve their goals, especially by enabling communication and collaboration among users who may be in different locations

Examples

Groupware software tools
Many project management tools now include groupware features for file sharing and collaborative editing.
The company implemented groupware to facilitate better communication among remote teams.
Using groupware, employees can work together on documents even if they are on different continents.

Related Terms

  • synonym: collaborative software - software enabling multiple users to work together
