multicore

noun
processor with multiple cores
Pronunciation: / ˈmʌltiˌkɔr / (AmE), / ˈmʌltiˌkɔː / (BrE)
a processor or chip that contains two or more independent processing cores within a single unit

Examples

A multicore chip
Modern laptops often feature a multicore processor to handle demanding applications efficiently.
The software is optimized to take advantage of multicore systems for faster performance.
Upgrading to a multicore CPU can significantly improve your computer's multitasking abilities.

Related Terms

  • degree higher: quad-core - a processor with four cores
  • antonym: single-core - a processor with only one core
