voice command

noun
a command given by speaking aloud
Pronunciation: / vɔɪs kəˈmænd / (AmE), / vɔɪs kəˈmɑːnd / (BrE)
a command given to a device or system by speaking aloud, often used in the context of computers or electronic devices

Examples

Voice command system
Cars now often support hands-free operation through voice commands.
The new phone allows users to control apps with a simple voice command.
You can set an alarm by issuing a voice command to your smart speaker.

Related Terms

  • synonym: verbal command - an instruction given by speaking, especially in a clear or explicit way
