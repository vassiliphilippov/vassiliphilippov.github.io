disk cache

noun
RAM storage for frequently used disk data
Pronunciation: / dɪsk kæʃ / (AmE), / dɪsk kæʃ / (BrE)
a section of volatile memory that stores frequently accessed disk data to speed up future read operations by avoiding slower direct disk access

Examples

The disk cache size
If the disk cache is too small, the computer may access the hard drive more often.
Installing additional memory can help increase the size of your disk cache for better performance.
Many modern computers use a disk cache to improve data retrieval speed.

Related Terms

  • synonym: cache - a hardware or software component that stores data for faster future access
