docker

noun
containerization platform
6.1
Pronunciation: /ˈdɑː.kər/ (AmE), /ˈdɒ.kə/ (BrE)
(computing) a software platform that packages applications and their dependencies into lightweight, portable containers for consistent deployment and execution.

Examples

Docker container software
Developers now package applications in portable containers using Docker to ensure consistent deployments.
The CI pipeline builds and pushes a new Docker image whenever a developer merges to the main branch.
Security scans flagged several vulnerable libraries inside the Docker images before production release.
