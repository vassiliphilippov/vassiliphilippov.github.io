word square

noun
a puzzle with words reading the same across and down
6.9 5.2
Pronunciation: /wɜːrd skwer/ /; / /wɜːrd skweər/ (AmE), /wɜːd skweə/ (BrE)
a puzzle consisting of a square grid in which words run the same horizontally and vertically, so that each row and column contains the same set of words

Examples

A word square puzzle
Each row in a word square must spell a valid word.
