autoincrement

adjective
automatically increasing
7.2
Pronunciation: / ˌɔːtoʊˈɪnkrəmənt / (AmE), / ˌɔːtəʊˈɪŋkrɪmənt / (BrE)
set to increase automatically, especially referring to a database column or field that generates a new value for each entry

Examples

Autoincrement field setting
The primary key field is set to autoincrement so each new record gets a unique number.
If you enable the autoincrement option, the database will handle numbering automatically.
Developers often use autoincrement columns to simplify record management in large tables.

Related Terms

  • synonym: automatic - working by itself with little or no direct human control
  • antonym: manual - done by hand or requiring human intervention
