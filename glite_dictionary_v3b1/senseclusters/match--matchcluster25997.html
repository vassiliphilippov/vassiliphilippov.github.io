match

noun
an exact duplicate
Pronunciation: / mætʃ / (AmE), / matʃ /; / mætʃ / (BrE)
an object or result that is an exact duplicate or positive identification (such as in DNA or programming)

Examples

A DNA match
The police were excited when the DNA sample produced a positive match.
They searched the database until they found a match for the fingerprint.
This vase is an exact match to the one we found last year.
In Rust, the match expression selects a branch based on value patterns.
During the code review she suggested using match to simplify multiple if-else statements.
Python 3.10 added a new match statement that supports structural pattern matching.

Related Terms

  • synonym: duplicate - an exact copy of something
  • antonym: original - the thing from which a copy is made
  • synonym: pattern matching - a technique for checking a value against a pattern and extracting information from it
  • synonym: switch statement - a control structure that selects a branch to execute based on the value of an expression
