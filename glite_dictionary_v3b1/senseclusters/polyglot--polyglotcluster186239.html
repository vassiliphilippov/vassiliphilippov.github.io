polyglot

noun
file valid in multiple programming languages
7.5
Pronunciation: /ˈpɑː.lɪ.ɡlɑːt/ (AmE), /ˈpɒ.lɪ.ɡlɒt/ (BrE)
a program, script, or file that is valid in, or can be executed by, more than one programming language or runtime.
