World Wide Web

noun
global information network on the Internet
1.0 2.0
Pronunciation: /wɜːrld waɪd web/ (AmE), /wɜːld waɪd web/ (BrE)
A system of interlinked hypertext documents and other resources that are accessed via the Internet using a web browser.

Examples

The World Wide Web system
Many people use the World Wide Web every day to find news, entertainment, and communicate with others.
Some schools teach students how to use the World Wide Web safely and effectively.
The invention of the World Wide Web revolutionized the way information is shared globally.
