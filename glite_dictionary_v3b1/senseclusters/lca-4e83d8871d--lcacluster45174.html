LCA

noun
lowest common ancestor
9.1
Pronunciation: /ˈel.siː.eɪ/ (AmE), /ˈel.siː.eɪ/ (BrE)
the lowest node in a tree data structure that is an ancestor of two given nodes, used in computer science and algorithms

Examples

LCA in a tree
The algorithm quickly found the LCA of the two nodes in the binary tree.
In computer science, finding the LCA is important for many tree-based problems.
The LCA helps determine the relationship between different nodes in a hierarchical structure.

Related Terms

  • synonym: ancestor - a node from which another node is descended in a tree structure
  • antonym: descendant - a node that comes from another node in a tree structure
