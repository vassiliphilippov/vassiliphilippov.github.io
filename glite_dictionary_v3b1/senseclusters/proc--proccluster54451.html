proc

noun
programming subroutine
Pronunciation: / prɑk / (AmE), / prɒk / (BrE)
a procedure or subroutine in programming, which is a callable block of code that performs a specific task

Examples

A proc in code
The developer wrote a proc to handle user authentication in the system.
You can reuse the same proc in different parts of your program to avoid repetition.
Before saving the data, the application calls a proc to validate the input fields.

Related Terms

  • synonym: function - a named section of code that performs a specific task
  • synonym: method - a function associated with an object or class in programming
