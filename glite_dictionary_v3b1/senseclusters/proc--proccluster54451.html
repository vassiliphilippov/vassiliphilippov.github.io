proc

noun
programming subroutine
8.8 0.10/m
Pronunciation: / prÉ‘k / (AmE), / prÉ’k / (BrE)
a procedure or subroutine in programming, which is a callable block of code that performs a specific task

Examples

A proc in code
The developer wrote a proc to handle user authentication in the system.
You can reuse the same proc in different parts of your program to avoid repetition.
Before saving the data, the application calls a proc to validate the input fields.

Synonyms

  • procedure
    4.1 10/m
    procedure is the standard name for a set of program instructions used in documentation, while proc is a shorter programmer label for that callable block.
  • subprogram
    7.2 0.08/m
    subprogram is a formal term for a small program used inside a larger system, while proc refers to such a callable unit in code.
  • routine
    3.8 1.70/m
    routine is a short sequence of code often seen in everyday programs, while proc specifically names a callable procedure or subroutine.
