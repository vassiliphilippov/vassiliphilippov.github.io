plot

noun
graph or chart of data
3.6 5.08/m
Pronunciation: / plÉ‘t / (AmE), / plÉ’t / (BrE)
a chart or graph that shows the relationship between different variables or the progress of data over time

Examples

A line plot
A line plot makes it easier to compare rainfall levels across different months.
The scientist created a plot to visualize the temperature changes during the experiment.
You can see from this plot how the data trends upward over time.

Synonyms

  • graph
    2.8 21/m
    a graph uses axes and points or lines to show how numbers relate, while a plot often refers to the actual data points drawn.
  • chart
    1.3 28/m
    a chart arranges data in tables or diagrams for easy reference, while a plot focuses on showing variable relationships over time.

Related Terms

  • synonym: chart - a sheet of information in the form of a table, graph, or diagram
  • synonym: graph - a diagram showing the relation between variable quantities
