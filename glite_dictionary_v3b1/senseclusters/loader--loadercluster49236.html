loader

noun
software component that loads resources
4.2
Pronunciation: /ˈloʊ.dər/ (AmE), /ˈləʊ.də/ (BrE)
(computing) a software component or program that loads data, modules, or other resources into memory or makes them available to a program, typically at startup or on demand.

Examples

Software loader
The web app's loader fetched images and stylesheets before the content appeared.
A slow loader caused the game to hitch and delay the initial menu from showing.
He wrote a custom loader to initialize plugins when the editor started up.

Related Terms

  • synonym: bootloader - a small program that loads an operating system when a computer starts
  • synonym: load - to move data or a program into a computer's memory so it can be used
