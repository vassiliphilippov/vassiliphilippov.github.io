abscissa

noun
horizontal coordinate value (math)
Pronunciation: / æbˈsɪs.ə / (AmE), / æbˈsɪs.ə / (BrE)
the coordinate value measured along the horizontal (x) axis in a Cartesian coordinate system

Examples

The abscissa value
Each point is described by its abscissa and ordinate in the coordinate plane.
On the graph, the abscissa represents the distance along the x-axis.
The abscissa of the intersection point is 4.

Related Terms

  • antonym: ordinate - the value of a coordinate on the vertical axis
