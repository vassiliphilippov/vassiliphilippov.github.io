computing machine

noun
machine that performs calculations automatically
Pronunciation: / kəmˈpjutɪŋ məˈʃin / (AmE), / kəmˈpjuːtɪŋ məˈʃiːn / (BrE)
A machine designed to perform calculations automatically, especially as an early term for a computer.

Examples

An early computing machine
Before modern computers, a computing machine was a valuable tool for astronomers.
Early scientists used the computing machine to solve complex mathematical problems.
The invention of the computing machine revolutionized the field of engineering.
