placeable

adjective
able to be positioned
Pronunciation: / ˈpleɪsəbəl / (AmE), / ˈpleɪsəbəl / (BrE)
able to be put or set in a particular position or location

Examples

Placeable shelves
The lightweight shelves are easily placeable on any wall in the house.
We bought modular furniture that is highly placeable in small apartments.
The decorations are placeable wherever you want to add a personal touch.

Related Terms

  • antonym: fixed - firmly attached and not able to be moved
  • synonym: movable - able to be moved from one place to another
