fluent

adjective
programming: enabling readable method chaining
7.0
Pronunciation: /ˈfluː.ənt/ /; / /ˈflʊ.ənt/ (AmE), /ˈfluː.ənt/ /; / /ˈflʊ.ənt/ (BrE)
(in programming) of an API or DSL: designed to allow method chaining and produce code that reads smoothly and clearly, often resembling natural language.

Examples

Fluent API design
The library provides a fluent API that lets developers chain calls instead of nested callbacks.
During the code review she praised the team's fluent interface for its readable method chaining.
The DSL uses a fluent syntax to build queries that read like plain English.

Related Terms

  • synonym: fluent interface - a style of API design that emphasizes readable, chainable method calls
  • synonym: method chaining - a programming technique where multiple method calls are linked together in a single expression
