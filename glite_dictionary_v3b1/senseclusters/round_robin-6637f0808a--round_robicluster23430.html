round robin

noun
rotating distribution method (computing)
3.9 0.06/m

A method of distributing tasks, connections, or requests by assigning them in turn to each server or process in a fixed, cyclical order.

Examples

Round robin scheduling
The load balancer used a round robin algorithm to send each request to a different server.
DNS round robin returned multiple IP addresses, letting browsers connect to different servers for redundancy.
To prevent overload, the team configured the proxy to distribute sessions using strict round robin order among nodes.
