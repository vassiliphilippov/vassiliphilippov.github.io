multithread

noun
multi-threaded program
Pronunciation: / ˈmʌltiˌθrɛd / (AmE), / ˈmʌltiˌθrɛd / (BrE)
a program, process, or system that uses multiple threads of execution simultaneously

Examples

A multithread server
The new server is a high-performance multithread capable of handling thousands of simultaneous connections.
Developers prefer a multithread for tasks that require parallel processing.
Upgrading to a multithread can significantly improve application performance.

Related Terms

  • synonym: multithreading - the ability of a CPU or program to manage multiple threads at once
  • antonym: single-thread - a program or process that uses only one thread of execution
