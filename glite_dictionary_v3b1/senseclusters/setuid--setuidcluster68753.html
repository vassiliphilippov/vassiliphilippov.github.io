setuid

noun
Unix file permission
Pronunciation: / ˈsɛtˌjuːaɪdiː / (AmE), / ˈsɛtˌjuːaɪdiː / (BrE)
a Unix file permission that allows a user to run an executable with the file owner's privileges

Examples

Setuid bit
The setuid bit must be set for this program to run with elevated permissions.
System administrators should be cautious when enabling setuid on executables to avoid security risks.
He checked the file's properties to see if the setuid permission was enabled.

Related Terms

  • synonym: permission - authorization to perform certain actions on a computer system
  • synonym: setgid - a Unix file permission that allows execution with the file group's privileges
