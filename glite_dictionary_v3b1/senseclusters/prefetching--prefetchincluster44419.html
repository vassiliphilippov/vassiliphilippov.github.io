prefetching

noun
advance data loading
Pronunciation: / priːˈfɛtʃɪŋ / (AmE), / priːˈfɛtʃɪŋ / (BrE)
a computing process or technique where data or resources are loaded in advance to improve performance

Examples

Memory prefetching
Modern processors use prefetching to reduce the time needed to access memory.
Effective prefetching can significantly speed up the performance of certain applications.
The software developer improved the app's speed by implementing prefetching techniques.
Modern web browsers use prefetching to make websites load more quickly for users.
By implementing prefetching, the app reduces waiting times when opening new pages.
The video player relies on prefetching to ensure smooth playback even with slow internet.

Related Terms

  • synonym: caching - storing data in a cache for faster access
