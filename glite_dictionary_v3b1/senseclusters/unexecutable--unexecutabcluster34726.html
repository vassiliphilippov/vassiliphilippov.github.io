unexecutable

adjective
not able to run
Pronunciation: / ˌʌnˌɛksɪˈkjuːtəbl̩ / (AmE), / ˌʌnˌɛksɪˈkjuːtəbl̩ / (BrE)
not able to be carried out, performed, or run, especially by a computer or as a task

Examples

An unexecutable task
The software update was unexecutable due to missing system files.
He realized the plan was unexecutable given the limited resources available.
If the permissions are set incorrectly, the file will remain unexecutable.

Related Terms

  • antonym: executable - able to be carried out or run, especially by a computer
  • synonym: impossible - not able to be done or achieved
