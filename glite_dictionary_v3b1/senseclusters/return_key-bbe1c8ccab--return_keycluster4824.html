return key

noun
key on keyboard or typewriter to start new line
6.7
Pronunciation: / rɪˈtɜrn ki / (AmE), / rɪˈtɜːn kiː / (BrE)
the key on a computer keyboard or typewriter that moves the cursor to a new line or starts a new command

Examples

The return key function
If the return key stops working, typing becomes very difficult.
On many laptops, the return key is labeled as 'Enter'.
Press the return key to start a new paragraph.
