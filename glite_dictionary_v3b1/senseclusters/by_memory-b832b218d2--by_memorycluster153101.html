by memory

adverb
from recall, not from written copy
4.4
Pronunciation: / baɪ ˈmɛməri / (AmE), / baɪ ˈmɛməri / (BrE)
using one's memory rather than reading or referring to something; from recall.

Examples

Recite by memory

She was able to play the piano piece by memory after months of practice.
Tour guides often memorize historical facts and share them by memory with visitors.
