unaligned

adjective
not properly positioned
6.8
Pronunciation: /ʌn.ə.ˈlaɪnd/ /; / /ə.nə.ˈlaɪnd/ (AmE), /ʌn.ə.ˈlaɪnd/ /; / /ə.nə.ˈlaɪnd/ (BrE)
not placed at the expected position or boundary, especially in computing or electronics, which may cause slower access or errors

Examples

Unaligned memory
The processor encountered an unaligned address, causing the operation to fail.
Accessing unaligned data can slow down performance on some computer systems.
Engineers fixed the bug by ensuring no unaligned memory accesses occurred in the code.
