dir

noun
computer directory command
Pronunciation: / dɪr / (AmE), / dɪə / (BrE)
a computer command that displays a list of files and folders in a directory

Examples

Type dir command
To view the contents of your folder, type dir in the command prompt.
The dir command quickly showed all the files stored in the directory.
After running dir, she found the document she was looking for.

Related Terms

  • antonym: cd - a command to change the current directory
  • synonym: ls - a command in Unix-like systems to list files and directories
