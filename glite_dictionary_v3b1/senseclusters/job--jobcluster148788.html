job

noun
unit of computer work
5.0 5.13/m
Pronunciation: / dÊ’É‘b / (AmE), / dÊ’É’b / (BrE)
(computing) a unit of work for a computer to perform, often consisting of several steps but managed as a single operation.

Examples

A print job
Each job submitted to the cluster is processed in the order it arrives.
Some jobs require both compiling and executing code as separate steps in the workflow.
The system administrator scheduled the backup as a nightly job to run automatically.

Synonyms

  • work unit
