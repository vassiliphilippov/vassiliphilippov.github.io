observable

noun
reactive programming data stream
8.4
Pronunciation: /əb.ˈzɜːr.və.bəl/ (AmE), /əb.ˈzɜː.və.bəl/ (BrE)
(in computer science, especially reactive programming) a data source object that emits a sequence of values or events over time which other code can subscribe to and receive.

Examples

JavaScript observable
The framework exposes an observable that emits events whenever the user interacts with the page.
To handle server updates efficiently, she converted the API results into an observable and subscribed to changes.
Multiple components shared the same observable, ensuring they received identical data when the state changed.

Related Terms

  • synonym: publisher - a component that produces and sends data to subscribers (in reactive systems)
  • synonym: stream - a continuous flow of data or events that can be processed over time
