nonvolatile

adjective
computer memory that retains data without power

(computing) Not losing stored information when power is removed; describing memory or storage that retains data without electrical power.

Examples

Nonvolatile memory
The server uses nonvolatile memory to prevent data loss during unexpected power cuts.
Modern smartphones include nonvolatile storage so your photos remain after a battery dies.
Engineers chose nonvolatile EEPROM chips for calibration settings that must survive power cycling.

Related Terms

  • synonym: persistent - continuing to exist or remain; used for storage that preserves data across power cycles
  • antonym: volatile - in computing, memory that loses its stored data when power is removed
