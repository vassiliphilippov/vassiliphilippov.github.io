unsquashed

adjective
not combined commits
Pronunciation: / ʌnˈskwɑʃt / (AmE), / ʌnˈskwɒʃt / (BrE)
not combined into a single commit or change set, especially in version control systems.

Examples

Unsquashed commits
The developer submitted a pull request with several unsquashed commits for easier code review.
Please keep your changes unsquashed until the final review is complete.
Some teams prefer unsquashed histories to track the development process in detail.

Related Terms

  • antonym: squashed - combined into a single commit or change set
