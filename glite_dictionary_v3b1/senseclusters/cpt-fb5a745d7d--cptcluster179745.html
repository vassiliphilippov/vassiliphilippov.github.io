CPT

proper noun
medical procedure codes
Pronunciation: / si pi ti / (AmE), / siː piː tiː / (BrE)
a standardized set of codes used to describe and report medical, surgical, and diagnostic procedures in the United States

Examples

CPT code
Hospitals use CPT codes to bill insurance companies for medical procedures.
The doctor entered the correct CPT code for the patient's surgery.
Medical billing specialists must be familiar with the latest CPT updates each year.

Related Terms

  • synonym: HCPCS - a set of health care procedure codes based on CPT
  • synonym: ICD - a coding system for diseases and health conditions
