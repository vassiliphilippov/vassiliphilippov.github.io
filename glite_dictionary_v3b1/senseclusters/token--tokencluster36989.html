token

adjective
symbolic; minimal or for appearance only
Pronunciation: / ˈtoʊkən / (AmE), / ˈtəʊkən / (BrE)
(of an action, gesture, or amount) done for the sake of appearance or as a symbolic gesture, rather than as something meaningful or substantial.

Examples

Token gesture
Her apology was merely a token gesture, not a sincere attempt to make amends.
Only a token number of changes were introduced, so the system stayed much the same.
The company made a token donation to charity, but it was clear their priorities lay elsewhere.

Related Terms

  • synonym: nominal - very small in amount; existing in name only
