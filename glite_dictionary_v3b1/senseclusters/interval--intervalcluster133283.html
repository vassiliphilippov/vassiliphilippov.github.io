interval

noun
set of numbers between two endpoints
5.4
Pronunciation: / ˈɪn.tɚ.vəl /; / ˈɪn.tər.vəl / (AmE), / ˈɪn.tə.vəl / (BrE)
a range or set that includes all points or values between two endpoints (such as in mathematics or genomics)

Examples

A number interval
Find the length of the interval [3, 8] on the real number line.
In calculus, an interval represents all the real numbers between two specified values.
The function is only defined on the interval from 0 to 1.
The study identified disease associations within a genomic interval on chromosome 7.
Bioinformaticians compare intervals from different samples to find overlapping regions of variation.
Use a BED file to list each interval with start and end coordinates for analysis.
