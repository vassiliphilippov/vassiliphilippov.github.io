command

noun
instruction to a computer program
Pronunciation: / kəˈmænd / (AmE), / kəˈmɑːnd / (BrE)
an order or instruction given by someone in authority, or a line of code given to a computer

Examples

Give a command
If the command is incorrect, the program will return an error message.
The user typed a command into the terminal to list all files in the directory.
This software allows you to automate tasks by creating custom commands.
At his command, the crew began preparations for departure.
Students are expected to follow the teacher's command without question.
The general gave a command to advance at dawn.

  • synonym: order - a directive or instruction given by someone in authority
