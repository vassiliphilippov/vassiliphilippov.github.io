erf

noun
mathematical error function
Pronunciation: / ɜrf / (AmE), / ɜːf / (BrE)
a mathematical function used in probability, statistics, and diffusion problems, known as the error function

Examples

The erf function
The scientist calculated the erf to determine the probability of a random variable falling within a certain range.
In heat transfer problems, the erf is often used to describe how temperature changes over time.
Statisticians frequently use the erf when working with normal distributions and error analysis.
