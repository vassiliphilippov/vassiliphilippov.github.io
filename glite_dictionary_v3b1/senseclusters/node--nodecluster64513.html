node

noun
module in a visual programming/shader graph
6.8 0.43/m
Pronunciation: / no8ad / (AmE), / n598ad / (BrE)
(computing, especially visual programming and graphics) a modular unit in a node-based graph or shader system that performs a single operation or data-processing step and exposes inputs and outputs for connection to other units.

Examples

Shader node
In Blender's shader graph, connect the node to the material output to see the change.
The sound designer chained three nodes to create a dynamic reverb effect for the scene.
Each node in the visual script receives inputs, performs an operation, and passes results onward.

Related Terms

  • synonym: graph node - a node representing an operation or value in a dataflow or shader graph
  • synonym: module - a self-contained component that performs a specific function and can be connected to other components
