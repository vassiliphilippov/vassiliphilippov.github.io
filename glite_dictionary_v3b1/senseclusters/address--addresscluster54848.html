address

noun
computing location code for data storage
Pronunciation: / əˈdrɛs / (AmE), / əˈdrɛs / (BrE)
a unique code or string that identifies a location in a computer system, network, or blockchain

Examples

IP address
Each device connected to the network is given a specific IP address.
Entering the wrong memory address can cause a program to crash.
Every file on the hard drive has a unique address that the operating system uses to find it.
She copied her blockchain address to receive funds from the exchange.
Before sending coins, double-check the recipient's address to avoid permanent loss.
He created a new wallet and saved the address along with its backup phrase.

Related Terms

  • synonym: memory address - the specific memory location code in a computer system
  • synonym: wallet address - the string used to receive cryptocurrency; used interchangeably with blockchain address
