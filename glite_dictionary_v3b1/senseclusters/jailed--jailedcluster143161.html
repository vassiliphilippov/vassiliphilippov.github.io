jailed

adjective
confined to a computing sandbox
7.5
Pronunciation: /dʒeɪld/ (AmE), /dʒeɪld/ (BrE)
(in computing) confined or restricted to a controlled sandbox or isolated environment so it cannot access certain system resources.

Examples

Jailed app
The database process was jailed inside a container to block file system access during testing.
For security, the web server runs jailed so it cannot read user home directories on the host.
When the team deployed the update, they accidentally left the backup process jailed without network access.

Related Terms

  • synonym: isolated - kept separate from other processes or the host to limit interactions and access
  • synonym: sandboxed - run in a restricted environment that limits access to system resources
