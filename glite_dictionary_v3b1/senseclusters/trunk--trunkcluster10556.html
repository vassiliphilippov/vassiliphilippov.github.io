trunk

noun
primary branch in version control
7.5
Pronunciation: / trʌŋk / (AmE), / trʌŋk / (BrE)
(in software development) the main line of development in a version control system, where ongoing work is integrated and from which release branches are often created.

Examples

Software trunk
The team pushed the latest changes to the trunk to prepare for the scheduled release.
Before merging, she ran the tests locally to make sure the feature didn't break the trunk.
Our continuous-integration server builds every commit on the trunk and reports failures immediately.

Related Terms

  • synonym: branch - a separate line of development in version control, often diverging from the trunk
  • synonym: main branch - the primary branch in a version control system used for integration and releases
