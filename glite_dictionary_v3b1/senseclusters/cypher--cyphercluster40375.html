cypher

noun
graph-database query language
8.5
Pronunciation: /ˈsaɪ.fər/ (AmE), /ˈsaɪ.fə/ (BrE)
(in computing) a declarative query language used to retrieve and manipulate data stored in property graph databases, commonly associated with Neo4j.

Examples

Cypher query language
Developers write cypher queries to traverse graphs and extract relationships and properties for analysis.
She optimized the cypher that returned friend recommendations to improve performance on large datasets.
In class they practiced writing cypher statements to match patterns of connected nodes and edges.
