canister

noun
Internet Computer software module
Pronunciation: / \u02c8k\u00e6n\u026ast\u0259r / (AmE), / \u02c8k\u00e6n\u026ast\u0259 / (BrE)
(computing, Internet Computer) a self-contained software unit deployed on the Internet Computer that contains executable code and persistent state, similar to a smart contract or service module.

Examples

A software canister
The startup deployed their service as a canister on the Internet Computer.
Developers updated the canister to fix a bug affecting user authentication flows.
Users interacted with the payment canister through a simple web interface without downloading anything.

Related Terms

  • synonym: module - a self-contained component of software that performs a specific function; a canister is a deployed module with state
  • synonym: smart contract - a program running on a blockchain that enforces rules and holds state; similar in purpose to a canister
