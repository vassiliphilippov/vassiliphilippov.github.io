distributable

adjective
able to be shared
Pronunciation: / dɪˈstrɪbjətəbl / (AmE), / dɪˈstrɪbjʊtəbl / (BrE)
able to be distributed, shared, or handed out to others

Examples

Distributable materials
The new version of the document is distributable to all team members.
Only distributable files can be uploaded to the public server.
The company released a distributable version of the software for educational use.

Related Terms

  • antonym: nontransferable - not able to be given or shared with others
  • synonym: shareable - able to be shared with others
