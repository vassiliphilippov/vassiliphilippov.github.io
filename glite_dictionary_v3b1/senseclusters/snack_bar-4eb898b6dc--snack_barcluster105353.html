snack bar

noun
temporary on-screen notification (UI)
4.1
Pronunciation: /snæk bɑːr/ (AmE), /snæk bɑː/ (BrE)
(in computing and user-interface design) a short, temporary on-screen message, usually appearing at the edge of the screen to inform the user or offer a simple action (for example, 'undo').

Examples

A snack bar message
When you save a file, a snack bar briefly appears at the bottom confirming the action.
Developers prefer a snack bar for short messages because it doesn't block the main screen.
The UX guidelines recommend using a snack bar for undo actions with a clear call to action.

Related Terms

  • synonym: notification - a message that informs the user about events or changes in an app or system
  • synonym: toast - a brief popup message in a user interface; similar short-lived notification
