disk

noun
magnetic storage device for computers
1.0
Pronunciation: / dɪsk / (AmE), / dɪsk / (BrE)
A flat, circular device coated with magnetic material used to store digital information in computers and related systems.

Examples

Computer disk
Over time, dust can cause errors when reading data from a disk.
The computer cannot start because there is no disk in the drive.
We need to back up all the important files on the disk before updating the software.
Magnetic discs were commonly used before hard drives became popular.
She inserted the backup disc into the computer to restore the lost files.
The data was corrupted on the old disc, so it could not be read.

Related Terms

  • synonym: disk - Alternative spelling, used especially in American English for computer storage devices
  • synonym: floppy disk - A flexible removable magnetic disk for data storage
  • synonym: hard disk - A rigid magnetic disk used for computer storage
  • antonym: optical disc - A disk storing data read by lasers, such as a CD or DVD
