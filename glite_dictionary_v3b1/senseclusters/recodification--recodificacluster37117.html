recodification

noun
rewriting code
8.3
Pronunciation: / ˌriːˌkoʊdɪfɪˈkeɪʃən / (AmE), / ˌriːˌkəʊdɪfɪˈkeɪʃən / (BrE)
the process of rewriting or reorganizing computer program source code to improve its structure, performance, or maintainability

Examples

Software recodification
The software company began a major recodification project to modernize its outdated systems.
After several years, the team decided that recodification was necessary to improve performance and security.
Successful recodification can make a program easier to maintain and extend in the future.

Related Terms

  • synonym: optimization - the process of making code run more efficiently
  • synonym: refactoring - the process of restructuring existing computer code without changing its external behavior
