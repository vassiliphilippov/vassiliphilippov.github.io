noun

(in computing) the state in which two or more processes, threads, or devices attempt to use the same limited resource at the same time, causing delays, reduced performance, or interference.

Examples

Disk contention
During peak load, disk contention caused the database queries to take twice as long.
Developers reduced thread contention by partitioning the work and using finer-grained locks.
Network contention between virtual machines led to packet loss during the high-traffic test.
