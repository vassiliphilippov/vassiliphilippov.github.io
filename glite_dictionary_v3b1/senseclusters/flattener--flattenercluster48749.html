flattener

noun
layer-merging tool
6.8
Pronunciation: /ˈflæt.ən.ər/ (AmE), /ˈflæt.ən.ə/ (BrE)
a software tool or process that merges all layers in an image or document into a single layer to preserve appearance for printing or sharing

Examples

PDF flattener tool
Before sending the file to the printer, use the flattener to combine all the layers.
The designer ran a flattener to ensure the document looked the same on every device.
Some image editing programs include a built-in flattener for preparing files for export.

Related Terms

  • synonym: merger - a tool or process that combines multiple elements into one
  • antonym: splitter - a tool or process that separates combined elements into parts
