nofollow

adjective
not followed by search engines
7.3
Pronunciation: /ˈnoʊ.fɑː.loʊ/ (AmE), /ˈnəʊ.fɒl.əʊ/ (BrE)
describing a web link that tells search engines not to follow it or pass on ranking value

Examples

Nofollow link tag
The blog uses nofollow tags on all user-submitted links to reduce spam.
Webmasters often mark paid links as nofollow to comply with search engine guidelines.
Adding a nofollow attribute can help control which pages are indexed by search engines.

Related Terms

  • antonym: dofollow - describing a web link that search engines are allowed to follow
