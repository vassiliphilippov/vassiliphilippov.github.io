multiplex operation

noun
interleaved multi-activity operation
Pronunciation: / ˈmʌl.t̬ɪˌplɛks ˌɑː.pəˈreɪ.ʃən / (AmE), / ˈmʌl.tɪ.pleks ˌɒp.ərˈeɪ.ʃən / (BrE)
an operation in which two or more activities are carried out by interleaving them, often to maximize resource use or efficiency

Examples

Multiplex operation handles inputs
By using multiplex operation, the computer can handle several input sources at the same time.
Modern communication systems rely on multiplex operation to transmit multiple data streams simultaneously.
The efficiency of the network improved greatly after the introduction of multiplex operation.
