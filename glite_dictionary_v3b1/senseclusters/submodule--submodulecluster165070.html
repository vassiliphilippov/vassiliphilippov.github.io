submodule

noun
part of a module
6.3 0.02/m
Pronunciation: / Ààs åbÀåm…ëÀêd íuÀêl / (AmE), / Ààs åbÀåm…ídjuÀêl / (BrE)
a smaller, self-contained part of a larger module or system, often with a specific function or working independently (especially in software or engineering)

Examples

Software submodule
The software project was divided into several submodules to make development easier.
Each submodule can be updated without affecting the rest of the system.
The team assigned different submodules to individual developers for faster progress.
Each submodule in the software project handles a specific task.
The engineering team updated the faulty submodule to improve the system's performance.
You must configure every submodule before the main application will run correctly.
