claim

noun
asserted user attribute in identity systems
6.7
Pronunciation: /kleɪm/ (AmE), /kleɪm/ (BrE)
(in digital identity and security) a discrete piece of information asserted about a user or subject (for example, name, email, or role) used by authentication and authorization systems.

Examples

A user claim
The JWT contains a claim with the user's role for access control.
During login the identity server issued several claims about the user's email and permissions.
Developers must validate each claim in the token before granting application privileges.

Related Terms

  • synonym: assertion - a statement presented as true about a subject, often used in authentication or identity contexts
  • synonym: attribute - a piece of data that describes a user or object (for example, an email address or role)
