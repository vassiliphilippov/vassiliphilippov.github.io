A variable in an experiment or statistical analysis whose value is not affected by other variables and which is manipulated to observe its effect on a dependent variable.

Examples

The independent variable in experiment
In the experiment, the temperature was set as the independent variable and the reaction rate was measured.
Researchers changed the independent variable to see how it influenced plant growth.
You must identify the independent variable before designing the scientific study.
