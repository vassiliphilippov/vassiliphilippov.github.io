table d'hôte

noun
fixed-price menu with limited options
8.3
Pronunciation: / ˌtɑːbəl ˈdoʊt / (AmE), / ˌtæbl ˈdəʊt / (BrE)
a menu in a restaurant offering a complete meal with limited choices at a fixed price

Examples

A table d'hôte menu
Some diners prefer ordering à la carte, while others go for the value of the table d'hôte.
The restaurant offers a table d'hôte that changes every week, featuring local seasonal dishes.
We chose the table d'hôte for simplicity and enjoyed a delicious three-course meal.
