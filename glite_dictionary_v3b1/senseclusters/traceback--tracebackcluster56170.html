traceback

noun
error call report
7.7
Pronunciation: / ˈtreɪsbæk / (AmE), / ˈtreɪsbæk / (BrE)
a report that shows the sequence of function calls leading to an error in a computer program

Examples

Python traceback message
When the script crashed, a traceback appeared on the screen to show where the error happened.
The developer used the traceback to identify which function caused the problem.
If you send the traceback to support, they can help you fix the bug.

Related Terms

  • synonym: debug log - a record of events to help find and fix errors in software
  • synonym: stack trace - a report showing the call stack at a specific point in a program
