cassowary

noun
UI layout constraint solver
9.9
Pronunciation: / \u02c8k\u00e6s\u0259\u02ccw\u025bri / (AmE), / \u02c8k\u00e6s\u0259w\u0259ri / (BrE)
(computing) a constraint-solving algorithm used in user-interface layout to compute positions and sizes by solving systems of linear constraints, often incrementally and with priorities.

Examples

Cassowary algorithm
The app uses the cassowary solver to keep buttons aligned during window resizing.
When the window changed size, the framework called cassowary to recompute positions instantly.
Many GUI libraries adopt cassowary for flexible auto-layout because it solves linear constraints incrementally.

Related Terms

  • synonym: constraint solver - software or algorithm that finds values satisfying a set of constraints
  • synonym: layout engine - component that computes positions and sizes of elements in a user interface
