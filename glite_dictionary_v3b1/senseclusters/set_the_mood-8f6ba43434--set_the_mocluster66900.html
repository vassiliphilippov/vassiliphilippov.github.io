set the mood

verb
create a particular atmosphere
Pronunciation: / sɛt ðə muːd / (AmE), / sɛt ðə muːd / (BrE)

Examples

The coach's speech set the mood for the team's intense practice session.
The decorations and lighting were chosen carefully to set the mood for the ceremony.
