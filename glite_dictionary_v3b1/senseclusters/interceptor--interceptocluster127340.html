interceptor

noun
software component that intercepts requests
8.0
Pronunciation: /ɪn.tər.ˈsep.tər/ /; / /ɪn.ər.ˈsep.tər/ (AmE), /ɪn.tə.ˈsep.tə/ /; / /ɪn.ə.ˈsep.tə/ (BrE)
(in computing) a software component or middleware module that intercepts, inspects, and optionally modifies or handles data, requests, or responses passed between systems or components (for example, HTTP interceptors).

Examples

HTTP interceptor module
The Angular interceptor logs outgoing HTTP requests and automatically adds the user's auth token.
During testing, our interceptors blocked malformed messages before they reached the backend service.
You can implement a logging interceptor to measure response times for all API calls in production.

Related Terms

  • synonym: filter - a component that inspects, modifies, or blocks data as it passes through a system
  • synonym: middleware - software that connects or mediates between different applications or services
