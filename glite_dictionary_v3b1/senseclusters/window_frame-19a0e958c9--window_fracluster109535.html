window frame

noun
framework supporting a window
4.3
Pronunciation: / ˈwɪndoʊ freɪm / (AmE), / ˈwɪndəʊ freɪm / (BrE)
the fixed framework that surrounds and supports a window within a wall or structure

Examples

A window frame gap
Rain leaked in through a gap in the window frame during last night's storm.
She painted the window frame white to match the new shutters.
The old wooden window frame needs to be replaced because it has started to rot.
