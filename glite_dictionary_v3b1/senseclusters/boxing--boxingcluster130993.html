boxing

noun
wrapping a primitive into an object (programming)
6.4
Pronunciation: /ˈbɑːk.sɪŋ/ (AmE), /ˈbɒk.sɪŋ/ (BrE)
(in programming) the process of converting a primitive value into a corresponding object wrapper so it can be used where an object is required.

Examples

Boxing a value
In C#, assigning an int to an object triggers boxing and can affect performance.
The lecture explained how boxing creates a heap object from a primitive value in .NET.
Developers avoid unnecessary boxing to reduce memory allocations and improve application throughput.

Related Terms

  • antonym: unboxing - the opposite process of extracting a primitive value from its object wrapper
  • synonym: wrapping - the general act of enclosing or packaging a value or object; in programming, putting a value into a container object
