color code

noun
system of classification using colors
2.9
Pronunciation: / ˈkʌlər koʊd / (AmE), / ˈkʌlə kəʊd / (BrE)
a system of organizing or classifying items by assigning different colors to represent categories or information

Examples

A color code system
Each resistor has a color code printed on it to indicate its resistance value.
The folders are arranged by a color code to help staff find files easily.
The hospital uses a color code to quickly identify different types of emergencies.

Related Terms

  • synonym: code - a system of symbols or signals used for communication or classification
  • synonym: color coding - the act or process of assigning colors to items for classification
