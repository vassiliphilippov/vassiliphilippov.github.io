pure binary numeration system

noun
numerical system with base two using binary digits
7.8
Pronunciation: / pjʊr ˈbaɪˌnɛri nuˈmɛrˌeɪʃən ˈsɪstəm / (AmE), / pjʊə ˈbɪnəri njuːˈmɛrɪˌeɪʃən ˈsɪstəm / (BrE)
A positional numerical system that uses only binary digits (0 and 1) and has a base (radix) of two.

Examples

A pure binary numeration system
Computers represent data internally using a pure binary numeration system that relies on only two digits: 0 and 1.
Every number can be translated into the pure binary numeration system for processing by digital devices.
The efficiency of digital circuits stems from the simplicity of the pure binary numeration system.
