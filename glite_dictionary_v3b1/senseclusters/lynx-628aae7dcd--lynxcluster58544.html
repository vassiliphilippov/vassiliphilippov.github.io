Lynx

noun
popular text-based web browser
8.9
Pronunciation: / lɪŋks / (AmE), / lɪŋks / (BrE)
A text-based web browser that displays web pages as plain text, typically used on command-line interfaces or in minimal computing environments.

Examples

Lynx web browser
Because Lynx is so lightweight, it can load web pages much faster on older computers.
Many system administrators use Lynx to quickly check websites from the command line.
Some universities still provide access to Lynx for visually impaired users who prefer text browsing.
