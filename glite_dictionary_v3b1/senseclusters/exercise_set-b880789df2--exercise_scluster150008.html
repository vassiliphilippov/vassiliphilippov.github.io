exercise set

noun
group of consecutive repetitions in exercise
1.9
Pronunciation: / ˈɛksɚˌsaɪz sɛt / (AmE), / ˈɛksəˌsaɪz sɛt / (BrE)
a group of repetitions of a particular exercise performed consecutively without rest

Examples

Three exercise sets
I usually perform three exercise sets of push-ups every morning.
Make sure to rest for one minute between each exercise set to avoid fatigue.
She increased the weight after completing each exercise set at the gym.
