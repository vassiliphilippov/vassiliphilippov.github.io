bundler

noun
software dependency manager
6.8
Pronunciation: / ˈbʌndlər / (AmE), / ˈbʌndlə / (BrE)
a software tool that manages, groups, and installs packages or dependencies for a programming project

Examples

Software bundler
The team used a bundler to ensure all necessary libraries were included in the project.
Before running the application, install the required packages with a bundler.
A good bundler can save developers time by managing dependencies automatically.
