learning curve

noun
ML model performance graph
4.0
Pronunciation: /ˈlɜːr.nɪŋ kɜːrv/ (AmE), /ˈlɜː.nɪŋ kɜːv/ (BrE)
(in machine learning) a graph showing how a model's performance changes as it is trained with more data or iterations, used to diagnose learning behavior and data sufficiency.

Examples

Model learning curve
The researchers plotted the learning curve to assess data sufficiency for the model.
A flat learning curve suggests the model no longer improves with more training examples.
They compared learning curve plots for two algorithms to decide which generalizes better.

Related Terms

  • synonym: training curve - a plot of model performance versus training examples or epochs; often used interchangeably with 'learning curve' in ML contexts
