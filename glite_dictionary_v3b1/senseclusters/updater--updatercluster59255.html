updater

noun
software or person updating
3.8
Pronunciation: / ˈʌpˌdeɪtər / (AmE), / ˈʌpˌdeɪtə / (BrE)
a person or program (especially a software tool) that brings something up to date, such as by installing the latest changes or information

Examples

Software updater tool
The updater automatically installs the latest software patches on your computer.
Before using the new features, make sure the updater has finished its process.
Many apps include a built-in updater to keep them current with security fixes.
The updater checks for new versions of the app every time you start your computer.
Make sure the updater is enabled so your software stays secure.
After running the updater, all the latest features were available in the program.

Related Terms

  • synonym: installer - a program or person that sets up software or equipment
  • synonym: patch - a small update to fix or improve software
  • antonym: remover - a program or person that deletes or uninstalls something
