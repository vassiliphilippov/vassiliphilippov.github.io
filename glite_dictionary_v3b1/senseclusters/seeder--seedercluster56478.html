seeder

noun
program that populates a database
6.3
Pronunciation: / ˈsiːdər / (AmE), / ˈsiːdə / (BrE)
a program or script that populates a database with initial, sample, or test data, often used to set up development or testing environments.

Examples

Database seeder
The Laravel seeder automatically fills the development database with sample users and test accounts.
Before running the integration tests they wrote a seeder to create realistic product and user records.
The deployment script uses a lightweight seeder that inserts default settings when installing the app.

Related Terms

  • synonym: fixture - a set of predefined data used to prepare a program or database for testing or development.
  • synonym: seed - to insert initial or sample data into a database or dataset (often used in software development).
