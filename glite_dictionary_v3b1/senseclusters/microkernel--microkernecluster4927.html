microkernel

noun
minimal OS kernel
8.3
Pronunciation: / ˈmaɪkroʊˌkɝːnəl / (AmE), / ˈmaɪkrəʊˌkɜːnəl / (BrE)
a minimal computer operating system kernel that provides only the most basic services, with other functions handled separately

Examples

A microkernel system
The new operating system is based on a microkernel architecture.
A microkernel allows developers to add features without changing the core system.
Many modern systems use a microkernel to improve security and stability.

Related Terms

  • synonym: kernel - the central part of an operating system managing core functions
  • antonym: monolithic kernel - an operating system kernel that includes all core services in one large program
