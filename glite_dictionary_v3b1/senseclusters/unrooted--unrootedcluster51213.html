unrooted

adjective
without root access
7.7
Pronunciation: / ʌnˈruːtɪd / (AmE), / ʌnˈruːtɪd / (BrE)
not modified to allow root or administrative access, especially on a device or operating system

Examples

Unrooted phone software
The warranty is still valid because the phone is unrooted and has not been altered.
Many users prefer to keep their devices unrooted for security reasons.
Some apps will not run on a unrooted system due to security restrictions.
