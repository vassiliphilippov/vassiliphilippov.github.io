preimage

noun
set mapped to target
9.9
Pronunciation: / ˈpriːˌɪmɪdʒ / (AmE), / ˈpriːˌɪmɪdʒ / (BrE)
the set of all elements that are mapped to a given set by a function, especially in mathematics

Examples

The preimage of a set
In mathematics, the preimage of a set under a function includes all elements that map to that set.
To solve the problem, first find the preimage of the interval using the given function.
The preimage can help determine which values in the domain correspond to a specific output.
