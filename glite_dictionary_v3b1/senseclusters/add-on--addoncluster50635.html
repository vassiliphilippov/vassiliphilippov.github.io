Synonyms

  • extension
    4.0 7.41/m
    extension is an extra part or code that adds functions to a system, while add-on is any added feature or component.
  • plug-in
    4.2 1.60/m
    plug-in is a software module that adds new features to an application, while add-on covers hardware or software extras.
  • bolt-on improvement
    6.3 0.00/m
    bolt-on improvement is an enhancement added to a system to boost performance without changing its core, while add-on is more general.
  • accessory
    3.5 7.78/m
    accessory is a supplementary item often for comfort or appearance, while add-on is any added part or feature for function.
  • appurtenance
    9.9 0.17/m
    appurtenance is a formal word for extra equipment used for a particular activity, while add-on is a common term for added features.
  • supplement
    3.9 12/m

  • addition
