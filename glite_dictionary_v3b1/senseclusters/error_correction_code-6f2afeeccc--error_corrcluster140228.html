error correction code

noun
code for detecting and correcting transmission errors
1.0 3.5
Pronunciation: /ˈer.ər kə.ˈrek.ʃən koʊd/ /; / /ˈer.ər kər.ˈek.ʃən koʊd/ (AmE), /ˈeə.rə kə.ˈrek.ʃən kəʊd/ /; / /ˈer.ə kə.ˈrek.ʃən kəʊd/ (BrE)
