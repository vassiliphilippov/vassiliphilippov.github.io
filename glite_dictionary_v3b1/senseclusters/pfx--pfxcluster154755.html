pfx

noun
certificate file format
9.9
Pronunciation: / pi ɛf ɛks / (AmE), / piː ɛf ɛks / (BrE)
a computer file in PKCS#12 format, usually with the .pfx extension, that stores a digital certificate and its private key

Examples

pfx file
The IT department requested the pfx file to install the SSL certificate on the server.
You need to export your digital certificate as a pfx to use it on another device.
After generating the pfx, she uploaded it to the secure portal for authentication.

Related Terms

  • synonym: certificate - a digital document used to prove identity or encrypt data
  • synonym: pem - a file format for storing cryptographic keys and certificates
