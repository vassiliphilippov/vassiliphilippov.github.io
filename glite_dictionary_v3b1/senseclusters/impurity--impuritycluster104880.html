impurity

noun
measure of label heterogeneity in ML
Pronunciation: / ɪmˈpjʊr.ə.ti / (AmE), / ɪmˈpjʊə.rɪ.ti / (BrE)
(in machine learning and statistics) a numeric measure of how mixed or heterogeneous the labels are in a dataset or node, used to evaluate and choose splits (for example, Gini impurity or entropy).

Examples

Gini impurity
The decision tree algorithm chose splits that reduced impurity in each child node.
Data scientists compare Gini and entropy to measure impurity when building classification models.
High impurity in a dataset node indicates mixed labels and suggests further splitting.

Related Terms

  • synonym: Gini impurity - a common impurity measure used in decision trees that quantifies label heterogeneity
  • synonym: entropy - an information-theory measure often used as an impurity metric for splitting nodes
  • antonym: purity - the inverse concept indicating a node contains mostly the same label; high purity means low impurity
