delegate

noun
type-safe object referring to a method (computing)
8.8 0.3/m
Pronunciation: / \u02c8d\u025bl\u026a\u02cc\u0261e\u026at / (AmE), / \u02c8d\u025bl\u026a\u0261e\u026at / (BrE)
(in computing) a type-safe object that holds a reference to a method or function so the method can be called later, often used to implement callbacks or event handling.

Examples

C# delegate
In C#, a delegate encapsulates a method reference and enables flexible callbacks.
The team used delegate objects to pass behavior between modules without exposing implementation details.
Unlike a raw function pointer, a delegate is type-safe and can reference instance methods.

Video Examples

Related Terms

  • synonym: callback - a function passed to another function to be invoked later; delegates are often used to implement callbacks
  • synonym: function pointer - a programming feature that holds the address of a function; a lower-level concept similar to delegates in other languages
