nonconsuming

adjective
not using up input
6.5
Pronunciation: / ˌnɑnkənˈsuːmɪŋ / (AmE), / ˌnɒnkənˈsjuːmɪŋ / (BrE)
not using up, destroying, or changing something during a process or activity

Examples

The function performs a nonconsuming check, so the original data remains unchanged.
