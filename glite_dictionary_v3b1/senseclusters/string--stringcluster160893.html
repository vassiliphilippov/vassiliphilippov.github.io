Synonyms

  • cord
    1.0 8/m
    cord is a thicker, stronger length of twisted fibers used for heavier tying, while string is thinner and used for small fastening.
  • twine
    3.2 1/m
    twine is a strong thread made of several strands twisted together for binding, while string is a thinner single fiber used for light tying.
  • tie
    1.0 5/m
    tie is a ribbon or cord used to fasten things in general, while string is mainly a thin piece of cord for tying.
