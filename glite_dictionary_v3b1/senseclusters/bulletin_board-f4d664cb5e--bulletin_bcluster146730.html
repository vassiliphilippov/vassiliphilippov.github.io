bulletin board

noun
online message board system

a computer system running software that allows users to post messages and access shared information, often forming an online community

Examples

Online bulletin board
Before the internet was widely available, people joined bulletin boards to discuss shared interests.
Many enthusiasts still use bulletin boards to exchange technical information about vintage computers.
The local computer club hosts a bulletin board where members can post announcements and files.

Synonyms

  • electronic bulletin board
    3.7 0.17/m
    an electronic bulletin board is a digital space for community posts and groups, while a bulletin board is a computer system for shared messages.
  • bulletin board system
    7.2 0.08/m
    a bulletin board system is the full, slightly older technical name used before the internet, while a bulletin board is the general modern term.
  • BBS
    8.2 0.51/m
    a BBS is the old abbreviation for systems that used phone lines and file sharing, while a bulletin board is a broader online message system.
