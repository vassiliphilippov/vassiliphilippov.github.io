writable

adjective
can be written on
3.5
Pronunciation: / ˈraɪtəbəl / (AmE), / ˈraɪtəbəl / (BrE)
able to be written on or to accept writing or data

Examples

Writable file
Before you can update the document, make sure it is writable.
The CD is not writable once it has been finalized.
Some folders on the computer are only writable by administrators.

Related Terms

  • synonym: editable - able to be changed or modified
  • antonym: read-only - able to be read but not written to
