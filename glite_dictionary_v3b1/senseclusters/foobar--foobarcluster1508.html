foobar

verb
to break software
9.9
Pronunciation: / ˈfuːbɑːr / (AmE), / ˈfuːbɑː / (BrE)
to cause a computer system, program, or data to stop working properly or become corrupted (informal)

Examples

To foobar a program
If you delete that file, you might foobar the entire application.
He accidentally foobared the database while trying to update the records.
Be careful not to foobar the system with untested code changes.
