file off the serial numbers

verb
remove identifying marks
Pronunciation: / faɪl ɑf ðə ˈsɪriəl ˈnʌmbərz / (AmE), / faɪl ɒf ðə ˈsɪəriəl ˈnʌmbəz / (BrE)
to deliberately remove identifying marks or information, especially to conceal the origin of something

Examples

To file off the serial numbers
Companies sometimes metaphorically file off the serial numbers when releasing a rebranded product.
The criminal tried to file off the serial numbers on the stolen merchandise before reselling it.
To disguise the weapon, he decided to file off the serial numbers.
