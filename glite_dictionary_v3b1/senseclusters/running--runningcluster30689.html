running

adjective
done with a running motion
Pronunciation: / ˈrʌnɪŋ / (AmE), / ˈrʌnɪŋ / (BrE)
(of a jump, start, or move) executed or begun while running or with a running motion.

Examples

Running jump

The athlete made a running start to gain momentum before the race.
