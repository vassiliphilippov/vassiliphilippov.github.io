shiftable

adjective
able to be rescheduled
Pronunciation: / ˈʃɪftəbəl / (AmE), / ˈʃɪftəbəl / (BrE)
able to be postponed, moved, or rescheduled to a different time or date.

Examples

Shiftable meeting
The meeting is shiftable if something urgent comes up.
Some deadlines are strict, but others are shiftable depending on the situation.
Because the event is shiftable, we can easily find a new date if needed.

Related Terms

  • antonym: fixed - not able to be changed or moved
  • synonym: movable - able to be moved from one place or time to another
  • synonym: postponable - able to be delayed to a later time
