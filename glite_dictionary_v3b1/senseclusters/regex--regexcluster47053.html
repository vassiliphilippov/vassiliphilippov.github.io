regex

noun
regular expression engine
Pronunciation: / ˈriːˌdʒɛks / (AmE), / ˈriːɡɛks / (BrE)
a software tool or engine that interprets and applies regular expressions to search, match, or modify text

Examples

Regex engine
The developer used a regex to quickly find all email addresses in the document.
Learning how to write a regex can make text processing much more efficient.
Some programming languages include a built-in regex library for pattern matching.

Related Terms

  • synonym: regular expression - a sequence of characters defining a search pattern
