decompiler

noun
program for reverse coding
Pronunciation: / diːkəmˈpaɪlər / (AmE), / diːkəmˈpaɪlə / (BrE)
a computer program or tool that converts machine code or compiled programs back into readable source code

Examples

A decompiler tool

A decompiler can help recover lost source code from compiled programs.
Many security analysts rely on a decompiler to inspect suspicious files.
The team used a decompiler to analyze the suspicious software for hidden functions.
A skilled decompiler can help recover lost source code from an old application.
Many cybersecurity experts rely on a decompiler to understand how malware operates.
