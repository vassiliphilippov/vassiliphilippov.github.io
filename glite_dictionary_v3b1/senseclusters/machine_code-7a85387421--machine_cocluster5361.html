machine code

noun
binary instructions executed directly by a computer
4.9 0.14/m
Pronunciation: / m…ôÀà Éin ko äd / (AmE), / m…ôÀà ÉiÀên k…ô äd / (BrE)
the set of instructions in binary or numerical format that a computer's central processing unit can execute directly, without further translation

Examples

Machine code instructions
Only the CPU can directly interpret and execute machine code.

Writing programs in machine code is difficult and rarely done by most programmers today.

Synonyms

  • machine language
    5.8 0.01/m
    machine language is the set of binary instructions a CPU understands and runs directly, while machine code is their exact numeric form executed by the processor.
  • object code
    7.4 0.10/m
    object code is the machine code produced by a compiler for a specific computer, while machine code refers to any binary instructions a CPU can run.
