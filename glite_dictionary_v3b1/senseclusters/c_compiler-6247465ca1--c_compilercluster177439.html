C compiler

noun
a compiler for the C programming language
4.7
Pronunciation: / si kəmˈpaɪlər / (AmE), / si kəmˈpaɪlə / (BrE)
A computer program that translates source code written in the C programming language into machine code or another lower-level language.

Examples

C compiler program
After installing a new C compiler, she was able to build her project without issues.
Most Unix-based systems come with a default C compiler pre-installed.
The C compiler reported several syntax errors in the code.
