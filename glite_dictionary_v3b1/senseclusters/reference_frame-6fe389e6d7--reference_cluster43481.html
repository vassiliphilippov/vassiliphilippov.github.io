reference frame

noun
coordinate system for specifying position
5.7 7.2
Pronunciation: / ˈrɛf.ɚəns freɪm / (AmE), / ˈrɛf.ərəns freɪm / (BrE)
A system of coordinates that is used to specify the position, orientation, and other properties of objects within it, especially in physics or engineering.

Examples

A reference frame system
In classical mechanics, the laws of motion may look different depending on the reference frame you choose.
The motion of the planet can be described more simply in a heliocentric reference frame.
To solve the problem, we must first define a suitable reference frame for the moving train.

Related Terms

  • synonym: coordinate system - a system that uses numbers to determine the position of a point or object
