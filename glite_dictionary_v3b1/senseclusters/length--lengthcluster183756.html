length

noun
number of elements in a sequence (computing)
3.6
Pronunciation: /leŋkθ/ /; / /leŋθ/ (AmE), /leŋθ/ /; / /leŋkθ/ (BrE)
(in computing) the number of elements or characters in a sequence such as an array, list, or string; often exposed as a 'length' property or function.

Examples

Length of the string
The program checks the length of the input string before processing.
You can get the length of the array with a built-in property in most languages.
When debugging, she printed the length value to confirm how many elements were stored.

Related Terms

  • synonym: count - the total number of items; often used interchangeably with 'length' when counting elements in a collection
  • synonym: size - the amount of space or number of elements in a collection; in many APIs 'size' and 'length' are related or equivalent
