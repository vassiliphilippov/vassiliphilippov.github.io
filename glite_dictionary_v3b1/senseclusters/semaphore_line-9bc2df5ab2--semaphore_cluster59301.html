semaphore line

noun
system using towers to send visual signals
Pronunciation: / ˈsɛməˌfɔr laɪn / (AmE), / ˈsɛməˌfɔː laɪn / (BrE)
a system of sending messages at a distance by means of visual signals, typically using towers with moving shutters to encode information

Examples

The semaphore line system
Before the invention of the electric telegraph, messages were often transmitted across Europe using the semaphore line.
Engineers maintained each tower in the semaphore line to ensure signals could be relayed without interruption.
The semaphore line required clear weather for the signals to be visible over long distances.

Related Terms

  • synonym: optical telegraph - an earlier term for systems like the semaphore line, using visual signals for distance communication
  • degree higher: telegraph - an electric or electronic system for sending coded messages over wires
