microwave linear accelerator

noun
particle accelerator using microwaves
Pronunciation: / ˈmaɪ.kroʊˌweɪv ˈlɪn.i.ɚ əkˈsɛl.ɚˌeɪ.tɚ / (AmE), / ˈmaɪ.krəʊˌweɪv ˈlɪn.i.ə əksɛl.əˌreɪ.tə / (BrE)
a type of linear accelerator in which charged particles are accelerated using microwave electromagnetic fields

Examples

A microwave linear accelerator device
A microwave linear accelerator generates high-energy particles by applying powerful microwave fields.
Scientists use a microwave linear accelerator to conduct advanced experiments in particle physics.
The hospital installed a new microwave linear accelerator to improve the precision of its cancer treatments.

Related Terms

  • synonym: linear accelerator - a device that accelerates charged particles along a straight path
