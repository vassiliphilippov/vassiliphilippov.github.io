web spinner

noun
rotating loading indicator on a website
3.9
Pronunciation: / ˈwɛb ˌspɪnər / (AmE), / ˈwɛb ˌspɪnə / (BrE)
(computing) A small animated graphic on a website or app that rotates to show that content is loading or a process is in progress.

Examples

Loading web spinner
The page shows a web spinner while the images load in the background.
Clicking the button replaced the form with a web spinner until submission finished.
Users complained that the slow web spinner made the site feel unresponsive on mobile.

  • synonym: spinner - a small animated graphic used in interfaces to indicate loading or ongoing processing
