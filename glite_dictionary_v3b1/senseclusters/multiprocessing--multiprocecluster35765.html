multiprocessing

noun
using multiple processors at once
1.0
Pronunciation: / ˌmʌltiˈproʊsɛsɪŋ / (AmE), / ˌmʌltɪˈprəʊsɛsɪŋ / (BrE)
the simultaneous processing of tasks by two or more processing units within a computer system

Examples

Multiprocessing system
By implementing multiprocessing, they significantly improved the application's performance.
Modern servers rely on multiprocessing to handle thousands of user requests simultaneously.
The computer's multiprocessing capability allows for more efficient resource use.
