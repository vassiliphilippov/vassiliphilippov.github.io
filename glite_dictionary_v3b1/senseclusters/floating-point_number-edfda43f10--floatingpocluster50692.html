floating-point number

noun
number in floating-point notation
6.7
Pronunciation: / ˈfloʊtɪŋ pɔɪnt ˈnʌmbɚ / (AmE), / ˈfləʊtɪŋ pɔɪnt ˈnʌmbə / (BrE)
A number expressed using floating-point notation, allowing representation of very large or very small real values with a given precision.

Examples

A floating-point number value
A floating-point number can represent values much larger or smaller than those allowed by integer types.
Computers can store a floating-point number to approximate the value of pi.
In scientific computing, accuracy can be affected by the limitations of floating-point numbers.

Related Terms

  • antonym: integer - a whole number; a number without fractions
