client

noun
networked computer or program accessing a server
4.5
Pronunciation: / ˈklaɪənt / (AmE), / ˈklaɪənt / (BrE)
A computer or program on a network that accesses services provided by a server.

Examples

Network client
Each client on the network requires a unique IP address.
The company upgraded all client machines to improve performance.
The software allows the client to request data from the central database.
