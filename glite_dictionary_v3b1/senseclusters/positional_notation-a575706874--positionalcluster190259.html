positional notation

noun
system of digit placement in numbers
Pronunciation: / pəˈzɪʃ.ən.əl noʊˈteɪ.ʃən / (AmE), / pəˈzɪʃ.ən.əl nəʊˈteɪ.ʃən / (BrE)
A system of representing numbers in which the value of each digit is determined by its position in the sequence.

Examples

A positional notation system
Babylonian mathematicians were among the first to develop a form of positional notation.
In positional notation, the placement of a digit changes its value exponentially.
The decimal system is a classic example of positional notation used worldwide.

Related Terms

  • synonym: place value - the value given to the position of a digit in a number
