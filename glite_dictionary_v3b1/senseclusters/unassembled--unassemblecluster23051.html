unassembled

adjective
not converted to machine code
Pronunciation: / ˌʌnəˈsɛmbəld / (AmE), / ˌʌnəˈsɛmbəld / (BrE)
not converted from assembly language into machine code by an assembler; still in assembly-language form.

Examples

Unassembled code files
The source code was still unassembled and could not be executed by the computer.
Before running the program, make sure it is not unassembled or it will not work.
The developer reviewed the unassembled code to check for errors before compiling.

Related Terms

  • antonym: assembled - converted from assembly language into machine code
