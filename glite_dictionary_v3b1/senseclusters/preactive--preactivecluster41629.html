preactive

adjective
ready but inactive
7.4
Pronunciation: / priˈæktɪv / (AmE), / priˈæktɪv / (BrE)
existing in a state just before activation; not yet active but ready to be activated.

Examples

Preactive mode
The security system remained preactive until the building was occupied.
Certain chemicals are stored in a preactive state to ensure safety before use.
The software enters a preactive mode while waiting for user input.

Related Terms

  • degree higher: active - currently working or operating
  • antonym: inactive - not active or not currently operating
