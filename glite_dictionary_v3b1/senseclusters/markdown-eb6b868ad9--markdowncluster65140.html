Markdown

noun
text formatting language
4.3
Pronunciation: /ˈmɑːrk.daʊn/ (AmE), /ˈmɑːk.daʊn/ (BrE)
a simple markup language used to format plain text documents (for example, to add headings, lists, or links)

Examples

Markdown language
Many developers prefer writing documentation in Markdown because it is easy to read and write.
You can convert your Markdown file into a PDF or HTML document with the right tools.
The website supports Markdown formatting for comments and posts.
You can write your notes in Markdown to make them easy to read and share online.
The website automatically converts Markdown files into formatted web pages.
Learning Markdown helps you create documents with headings, lists, and links quickly.
