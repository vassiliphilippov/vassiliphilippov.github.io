notebook

noun
interactive digital document for code
3.7
Pronunciation: /ˈnoʊt.bʊk/ (AmE), /ˈnəʊt.bʊk/ (BrE)
A digital, interactive document used to write and execute code, combine code with text and results, and share reproducible analyses, commonly used in data science and education.

Examples

A digital notebook
She opened a notebook to run Python cells and document her results.
The research team shared the notebook containing code, data, and plots for reproducibility.
Students turned in several notebooks with executed cells, explanations, and clean output.

Related Terms

  • synonym: Jupyter Notebook - a popular web-based interactive notebook for running code (especially Python) and documenting data analyses
  • synonym: interactive document - a document that allows execution of code, dynamic output, and embedded narrative
