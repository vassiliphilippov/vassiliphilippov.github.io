fan-out

noun
software coupling metric
7.3
Pronunciation: / ˈfæn aʊt / (AmE), / ˈfæn aʊt / (BrE)
the number of other software modules, functions, or components that a single module directly calls or depends on

Examples

Software fan-out
The architect analyzed the fan-out of each component to improve the system's maintainability.
High fan-out can make software harder to test and more prone to errors.
Reducing the fan-out of a module often leads to simpler and more reliable code.

Related Terms

  • synonym: coupling - the degree of interdependence between software modules
  • antonym: fan-in - the number of modules that directly call or depend on a single module
