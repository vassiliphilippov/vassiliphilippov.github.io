polyvalent

adjective
having multiple truth values
9.6
Pronunciation: /ˈpɑː.lɪ.veɪ.lənt/ /; / /ˈpɑː.lɪ.və.lənt/ (AmE), /ˈpɒl.ɪ.veɪ.lənt/ /; / /ˈpɒl.ɪ.və.lənt/ (BrE)
(in logic and computer science) describing a logical system or algebra that allows more than two distinct truth values (i.e., many-valued logic).

Examples

Polyvalent logic
Some AI frameworks use polyvalent logic to handle uncertain truth values in sensor fusion.
Researchers implemented a polyvalent truth system to represent degrees of belief in the model.
Unlike Boolean algebra, polyvalent systems allow more than two distinct truth values for propositions.

Related Terms

  • antonym: bivalent - having exactly two truth values (true and false); two-valued logic
  • synonym: many-valued logic - a system of logic that permits more than two truth values for propositions
