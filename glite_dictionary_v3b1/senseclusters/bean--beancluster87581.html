bean

noun
Java/Spring software component
Pronunciation: / bin / (AmE), / biːn / (BrE)
(computing) A reusable software component, especially in Java frameworks (for example JavaBeans or Spring), that is created and managed by a container and often provides business logic or configuration.

Examples

Java bean object
The configuration creates a bean to manage user sessions in the Spring container.
Developers often test each bean independently to ensure the business logic works correctly.
In the microservice, a singleton bean holds configuration properties that other components can access.

Related Terms

  • synonym: JavaBean - a Java class that follows conventions to allow properties to be accessed and managed by frameworks
  • synonym: component - a modular piece of software that performs a specific function and can be reused
