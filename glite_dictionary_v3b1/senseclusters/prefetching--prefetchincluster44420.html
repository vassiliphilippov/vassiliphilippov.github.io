prefetching

adjective
loading in advance
6.0
Pronunciation: / priːˈfɛtʃɪŋ / (AmE), / priːˈfɛtʃɪŋ / (BrE)
relating to software or a process that loads data or resources in advance to improve performance

Examples

Prefetching software

Many modern browsers use prefetching techniques to make web pages load faster.
A prefetching system can anticipate user actions and prepare data in advance.
