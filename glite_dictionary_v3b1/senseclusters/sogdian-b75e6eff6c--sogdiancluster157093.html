Sogdian

noun
ancient writing system
Pronunciation: / ˈsɑːɡdiən / (AmE), / ˈsɒɡdiən / (BrE)
the historical script used to write the Sogdian language, spoken in Central Asia in ancient times

Examples

Sogdian script
The museum displayed a rare manuscript written in Sogdian from the Silk Road era.
Researchers studied Sogdian inscriptions to learn about ancient trade in Central Asia.
Few people today can read the Sogdian script, which was used over a thousand years ago.

Related Terms

  • synonym: Aramaic script - an ancient writing system from which Sogdian script is derived
  • synonym: Sogdian language - the Eastern Iranian language written using the Sogdian script
