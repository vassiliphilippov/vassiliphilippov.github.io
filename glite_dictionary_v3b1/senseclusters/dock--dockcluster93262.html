dock

noun
desktop bar for quick app access
3.8 1.10/m
Pronunciation: / dÉ‘k / (AmE), / dÉ’k / (BrE)
A bar or area on a computer desktop that holds icons for frequently used applications, files, or system controls for quick access and launching.

Examples

Computer dock
She pinned her most-used apps to the dock so they were always easy to open.
After installing the update, the dock now shows previews when I hover over an icon.
He cleared the bottom of his screen so the dock would not hide any app icons.

Synonyms

  • launcher
    4.0 0.67/m
    a launcher is an app interface that shows icons and shortcuts to start and organise applications, while a dock holds icons for quick access and launching.
  • taskbar
    3.3 0.75/m
    a taskbar is a screen bar that shows open programs and system buttons for running and quick access, while a dock holds icons mainly for launching.

Related Terms

  • synonym: launcher - a program or UI element used to start or open applications quickly
  • synonym: taskbar - a bar on the desktop (commonly Windows) that shows running applications and provides quick access to programs
