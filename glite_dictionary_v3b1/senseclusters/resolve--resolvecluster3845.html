(in computing) to determine the network address (such as an IP address) associated with a domain name or hostname, typically by querying the Domain Name System (DNS).

Examples

To resolve a domain
When you type a web address, your browser will resolve the domain to an IP address automatically.
The network engineer resolved the server's hostname to confirm which IP address was responding.
Use the dig tool to resolve example.com and inspect the DNS records returned by the server.
