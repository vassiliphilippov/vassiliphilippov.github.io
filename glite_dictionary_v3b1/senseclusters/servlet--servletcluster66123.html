servlet

noun
web server program
7.3
Pronunciation: / ˈsɜːrvlət / (AmE), / ˈsɜːvlɪt / (BrE)
a small program that runs on a web server to process requests and generate responses, often for web applications

Examples

A servlet program
The developer wrote a servlet to handle user login requests on the website.
Each servlet is responsible for processing specific types of web requests.
After updating the servlet, the website responded to users much faster.

Related Terms

  • synonym: script - a small program that automates tasks on a server or website
  • synonym: web application - a software application that runs on a web server and is accessed via a browser
