teredo

noun
IPv6-over-IPv4 tunneling protocol
9.9
Pronunciation: /tə.ˈriː.doʊ/ (AmE), /tə.ˈriː.dəʊ/ (BrE)
(in computer networking) an IPv6-over-IPv4 tunneling protocol that allows IPv6 packets to be carried through IPv4 networks, typically using UDP to traverse NATs.

Examples

Teredo tunneling protocol
Many home routers still support teredo tunneling to give IPv6 connectivity over IPv4 networks.
Windows used teredo as a fallback method when native IPv6 connectivity was unavailable.
Security analysts warned that teredo can accidentally expose hosts behind NAT to external scans.

Related Terms

  • synonym: NAT traversal - techniques that allow network traffic to pass through network address translators so hosts behind NATs can communicate.
  • synonym: tunneling - sending one network protocol inside another (for example, carrying IPv6 packets inside IPv4 packets).
