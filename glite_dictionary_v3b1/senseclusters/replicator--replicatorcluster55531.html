replicator

noun
data copying program
7.2
Pronunciation: / ˈrɛplɪˌkeɪtər / (AmE), / ˈrɛplɪkeɪtə / (BrE)
a software program or process that copies or synchronizes data between different systems or locations

Examples

A file replicator
The replicator automatically updated all files across the network every night.
After installing the replicator, the company no longer worried about data loss between offices.
A reliable replicator is essential for keeping databases consistent in real time.

Related Terms

  • synonym: backup - a copy of data kept for safety
  • synonym: synchronizer - a tool that keeps data consistent between systems
