sketch map

noun
rough hand-drawn map showing main features
3.1 3.4
Pronunciation: /sketʃ mæp/ (AmE), /sketʃ mæp/ (BrE)
A map drawn quickly from observation, showing only the main features of an area rather than precise measurements.

Examples

A sketch map quickly
He made a quick sketch map to show the location of the nearest gas station.
Students were asked to create a sketch map of their neighborhood from memory.
The guide drew a sketch map of the trail so we wouldn't get lost.
