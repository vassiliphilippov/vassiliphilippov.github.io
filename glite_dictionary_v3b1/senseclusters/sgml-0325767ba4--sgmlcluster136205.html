SGML

noun
standard markup language for documents (computing)
Pronunciation: / ˌɛs ˌdʒiː ɛm ˈɛl / (AmE), / ˌɛs ˌdʒiː ɛm ˈɛl / (BrE)
(computing) Standard Generalized Markup Language, a standardized system for describing the structure and content of electronic documents, specifying rules for markup vocabularies.

Examples

SGML standard system
Many early digital publishing projects were structured using SGML standards.
The development of XML was influenced by the earlier SGML framework.
You need a parser that can read SGML documents for archival purposes.

Related Terms

  • degree lower: HTML - hypertext markup language derived from SGML
  • degree higher: XML - extensible markup language derived from SGML
