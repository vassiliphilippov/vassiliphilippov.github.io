incognito

noun
browser privacy mode
5.7
Pronunciation: /ɪn.kɔːg.ˈniː.toʊ/ /; / /ɪn.kɔːg.ˈnɪ.toʊ/ (AmE), /ɪn.kɒg.ˈniː.təʊ/ /; / /ɪn.kɒg.ˈnɪ.təʊ/ (BrE)
(informal, computing) a browser or application privacy mode that does not save browsing history, cookies, or other session data for the current session.

Examples

Incognito mode
Open a new incognito window if you don't want the browser to save your history.
She checked the flight on incognito to avoid personalized price increases from trackers.
Journalists often recommend using incognito when researching sensitive topics on public computers.

Related Terms

  • synonym: private browsing - a browser mode that does not save browsing history, cookies, or site data for a session
  • synonym: private window - a browser window opened in private or incognito mode
