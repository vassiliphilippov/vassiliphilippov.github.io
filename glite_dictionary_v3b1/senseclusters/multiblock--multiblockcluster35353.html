multiblock

noun
grouped data blocks
Pronunciation: / ˈmʌltiˌblɑk / (AmE), / ˈmʌltiˌblɒk / (BrE)
a group of data or storage blocks managed as a single unit by a computer system or file system

Examples

A multiblock file
The file system allocated a multiblock to store the large database index efficiently.
When reading large files, the system processes each multiblock to improve performance.
A multiblock can help reduce fragmentation in modern storage solutions.

Related Terms

  • synonym: block - a unit of data storage in computing
  • synonym: cluster - a group of sectors or blocks treated as a single unit in storage
