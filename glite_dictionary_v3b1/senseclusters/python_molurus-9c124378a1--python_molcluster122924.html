Python molurus

proper noun
a large python species of South and Southeast Asia
Pronunciation: / ˈpaɪθɑn moʊˈlʊərəs / (AmE), / ˈpaɪθən məʊˈlɔːrəs / (BrE)
a very large nonvenomous python species found in southern and Southeast Asia, also known as the Indian python
