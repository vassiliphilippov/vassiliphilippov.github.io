Java sparrow

noun
small finch-like Indonesian weaverbird
6.6
Pronunciation: / ˈdʒɑːvə ˈspæroʊ / (AmE), / ˈdʒɑːvə ˈspærəʊ / (BrE)
a small finch-like weaverbird (native to Indonesia) with a stout bill, often found in rice fields
