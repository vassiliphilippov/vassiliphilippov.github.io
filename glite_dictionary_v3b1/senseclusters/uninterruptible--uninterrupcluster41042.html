uninterruptible

adjective
cannot be interrupted
6.6
Pronunciation: / ˌʌnɪnˈtɜːrəptəbl /; / ˌʌnɪnˈtɜːrəptəbl̩ / (AmE), / ˌʌnɪnˈtʌrɪptəbl /; / ˌʌnɪnˈtʌrɪptəbl̩ / (BrE)
not able to be stopped, paused, or interrupted once started

Examples

An uninterruptible power supply
The server is connected to an uninterruptible power supply to prevent data loss during outages.
Critical medical equipment must have an uninterruptible source of electricity at all times.
The backup system ensures uninterruptible operation even if the main power fails.
The backup process is uninterruptible to prevent data corruption during critical operations.
If a task becomes uninterruptible, the system may appear to freeze until it completes.
Some system updates run in an uninterruptible mode to ensure they finish safely.

Related Terms

  • synonym: continuous - happening without stopping
  • antonym: interruptible - able to be stopped or paused
