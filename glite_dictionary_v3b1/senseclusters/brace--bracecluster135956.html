brace

noun
punctuation mark: { or }
Pronunciation: / breɪs / (AmE), / breɪs / (BrE)
either of the two punctuation marks { or } used to enclose words, symbols, or lines of text, especially in mathematics, programming, or lists

Examples

A brace in code
Each function block must start and end with a brace in this programming language.
Enclose the set of options within curly braces in your code.
The mathematical expression is grouped using a brace to indicate set membership.

Related Terms

  • synonym: bracket - one of the punctuation marks [ or ]
  • synonym: parenthesis - round brackets ( ) used to enclose information
