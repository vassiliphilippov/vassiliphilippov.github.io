reversible

adjective
able to be reversed or used with either side out
1.6 3.0
Pronunciation: /rɪ.ˈvɜːr.sə.bəl/ /; / /riː.ˈvɜːr.sə.bəl/ /; / /rə.ˈvɜːr.sə.bəl/ (AmE), /rɪ.ˈvɜː.sə.bəl/ /; / /riː.ˈvɜː.sə.bəl/ /; / /rə.ˈvɜː.sə.bəl/ (BrE)
(of an object or garment) able to be used or worn with either side out; able to be reversed in direction, position, or function.

Examples

Reversible jacket
She bought a reversible coat that can be worn inside out for a different look.
The new sofa cushions are reversible, with stripes on one side and plain color on the other.
You should check whether the door is reversible before you install it to open the opposite way.
