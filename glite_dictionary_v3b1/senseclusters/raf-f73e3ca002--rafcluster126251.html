noun
Fujifilm RAW image file format
7.9
Pronunciation: /ræf/ /; / /ˈɑːr.eɪ.ef/ (AmE), /rɑːf/ /; / /ˈɑː.eɪ.ef/ (BrE)
A Fujifilm RAW image file format (file extension .raf) used by certain digital cameras to store unprocessed sensor data for later processing.
