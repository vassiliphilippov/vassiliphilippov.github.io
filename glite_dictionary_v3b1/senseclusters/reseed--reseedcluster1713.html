reseed

verb
give a new seed to an RNG
7.2
Pronunciation: /riː.ˈsiːd/ (AmE), /riː.ˈsiːd/ (BrE)
(in computing) to set or replace the seed value for a random number generator or algorithm

Examples

To reseed a generator
After detecting low entropy, the daemon automatically reseeded the generator to improve randomness.
You should reseed the pseudo-random number generator periodically in long-running simulations to avoid bias.
Reseeding the entropy pool requires collecting new hardware randomness before creating cryptographic keys.
The programmer had to reseed the random number generator to ensure different results each time.
If the simulation behaves unpredictably, try reseeding the algorithm with a new value.
After a system reset, the software automatically reseeds its random sequence for security.
