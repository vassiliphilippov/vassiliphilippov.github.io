infector

noun
malicious computer program
8.3
Pronunciation: /ɪn.ˈfek.tər/ (AmE), /ɪn.ˈfek.tə/ (BrE)
a program or piece of malicious code that infects computers or files by attaching itself to them

Examples

Computer infector
The security scan found an infector hidden in the downloads folder.
An infector can spread quickly if the system is not protected by antivirus software.
After opening the suspicious email, a dangerous infector compromised several files on her computer.

Related Terms

  • antonym: antivirus - software designed to detect and remove malicious programs
  • synonym: virus - a type of malicious software that can copy itself and infect computers
  • synonym: worm - a self-replicating computer program that spreads to other computers
