computer monitor

noun
screen for displaying computer output

Pronunciation: / k…ômÀàpjuÀêt…ö Ààm…ëÀên…™t…ö / (AmE), / k…ômÀàpjuÀêt…ô Ààm…ín…™t…ô / (BrE)
A device that displays the visual output from a computer, typically comprising a screen and associated electronics.

Examples

A computer monitor screen
I bought a new computer monitor with a high-resolution display for my office.
Make sure the computer monitor is plugged in before turning on your PC.
The computer monitor suddenly went black during the storm.

Synonyms

  • computer display
    2.5 0.23/m
    computer display is a screen that shows visual output to the user, while computer monitor is the full device with screen and electronics.
  • monitor
    3.3 10/m
    monitor is a general word for a display used with many devices, while computer monitor refers specifically to the computer device.
  • visual display unit
    4.1 0.00/m
    visual display unit is a formal technical name used especially in British English, while computer monitor is the common everyday term for that device.
  • computer screen
    2.1 4.44/m
    computer screen is the flat glass part you look at, while computer monitor is the complete device including electronics and a stand.
