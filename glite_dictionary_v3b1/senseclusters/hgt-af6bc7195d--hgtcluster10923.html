HGT

noun
elevation data file
Pronunciation: / eɪtʃ dʒiː tiː / (AmE), / eɪtʃ dʒiː tiː / (BrE)
a computer file format (file extension .hgt) used to store digital elevation or height data in geographic information systems

Examples

HGT file format
The mapping software requires an HGT file to display accurate terrain elevations.
Researchers downloaded several HGT files to analyze the topography of the mountain region.
You can convert an HGT file into other formats for use in different GIS applications.

Related Terms

  • synonym: DEM - a digital elevation model file used to represent terrain surfaces
  • synonym: TIFF - a file format often used for storing raster graphics and images, including geospatial data
