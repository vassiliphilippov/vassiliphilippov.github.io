interpolator

noun
estimation device or program
8.5
Pronunciation: / ɪnˈtɜːrpəˌleɪtər / (AmE), / ɪnˈtɜːpəleɪtə / (BrE)
a device or program that estimates values between known data points by interpolation

Examples

Digital interpolator
The engineer used an interpolator to estimate the missing temperature readings.
Modern graphics software often includes an interpolator to create smooth transitions between images.
An interpolator can help predict values in scientific experiments when some data points are unavailable.

Related Terms

  • synonym: calculator - a device or program that performs mathematical calculations
  • antonym: extrapolator - a device or program that estimates values beyond known data points
