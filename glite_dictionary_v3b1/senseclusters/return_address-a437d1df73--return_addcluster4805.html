return address

noun
sender's address for mail return
2.3
Pronunciation: / rɪˈtɝn əˈdrɛs / (AmE), / rɪˈtɜːn əˈdrɛs / (BrE)
the sender's address written on a letter or parcel, indicating where it should be sent back if delivery is unsuccessful

Examples

The return address on envelope
Please write your return address clearly on the envelope to ensure it can be returned if undelivered.
She forgot to include a return address on her letter, so it couldn't be returned.
The package was sent back to the return address after several failed delivery attempts.

Related Terms

  • synonym: sender address - the address where the letter or parcel originates
