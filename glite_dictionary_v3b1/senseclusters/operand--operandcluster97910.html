operand

noun
quantity acted on in an operation
Pronunciation: / ˈɑː.pəˌrænd / (AmE), / ˈɒp.ər.ænd / (BrE)
a quantity or value on which a mathematical operation is performed.

Examples

A operand value
In the equation 7 + 4, both 7 and 4 are operands for the addition operation.
The computer program processes each operand before performing the division.
When you multiply two numbers, each number is called an operand.

Related Terms

  • antonym: operator - a symbol or function indicating an operation to be performed
