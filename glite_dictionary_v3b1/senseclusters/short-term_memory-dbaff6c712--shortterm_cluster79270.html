short-term memory

noun
memory retained for a short duration
3.9 1.25/m
Pronunciation: / Àà É…îrt t…úrm Ààm…õm…ôri / (AmE), / Àà É…îÀêt t…úÀêm Ààm…õm…ôri / (BrE)
the ability to remember information for a brief period, typically seconds, after it has been perceived

Examples

Short-term memory loss
He repeated the words back using only his short-term memory.
People with amnesia often have difficulty forming new short-term memory.
Reading a phone number aloud tests your short-term memory.

Synonyms

  • STM
    6.1 0.50/m
    STM is the short label used in scientific papers and notes, while short-term memory is the full phrase used in general descriptions.
  • immediate memory
    4.1 0.02/m
    immediate memory is recalling information right after it is perceived in the first moment, while short-term memory keeps items active for several seconds.
