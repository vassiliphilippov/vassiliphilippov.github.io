subchannel

noun
separate data stream
7.0

a separate part of a channel's frequency or bandwidth used to carry an individual data stream in telecommunications or signal processing

Examples

Data subchannel
Each subchannel in the system can transmit a different type of data simultaneously.
Digital television often uses a subchannel to broadcast additional content alongside the main program.
Engineers assigned a unique subchannel to every user to prevent signal interference.
