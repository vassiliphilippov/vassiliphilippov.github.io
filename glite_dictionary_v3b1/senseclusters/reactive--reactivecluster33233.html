reactive

adjective
event-driven, asynchronous (computing)
6.1 0.20/m
Pronunciation: / ric86kt6av / (AmE), / ric86kt6av / (BrE)
(in computing) describing software, systems, or architectures that respond to events or continuous data streams in real time, typically using asynchronous, non-blocking processing so the system remains responsive under load.

Examples

Reactive system
The new messaging app uses a reactive architecture to display messages instantly as they arrive.
In class we learned that reactive systems handle data streams without blocking the main processing thread.
To improve performance, engineers made the service reactive so it scales with fluctuating traffic automatically.

Related Terms

  • synonym: asynchronous - not happening at the same time; operations that do not block waiting for other tasks.
  • synonym: event-driven - describing systems that react to events as they occur.
  • antonym: proactive - acting in advance to prevent problems rather than only responding to events; often contrasted with reactive.
