seeder

noun
file sharer online
5.8
Pronunciation: /ˈsiː.dər/ (AmE), /ˈsiː.də/ (BrE)
a person or computer that shares files for others to download, especially in peer-to-peer networks

Examples

File seeder
The more seeders a file has, the faster it can usually be downloaded.
After finishing the download, she became a seeder to help others get the file.
Some torrent sites display the number of seeders and leechers for each file.

Related Terms

  • antonym: leecher - a person or computer that downloads files without sharing them
  • synonym: uploader - a person or computer that sends files to a network for others to access
