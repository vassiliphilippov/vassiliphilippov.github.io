timeserver

noun
network device that provides accurate time
7.3 0.01/m

A computer system or network service that provides accurate time information (clock signals) to other devices, typically using protocols such as NTP or through GPS time sources.

Examples

Network timeserver
The admin configured the timeserver to lock network clocks against drift using GPS PPS signals.
Every computer on the campus synchronizes its clock nightly with the central timeserver over NTP.
If the timeserver fails, logs show events with inconsistent timestamps across distributed systems.

Related Terms

  • synonym: NTP server - a server that implements the Network Time Protocol to synchronize clocks on computer systems
  • synonym: time source - any source (e.g., GPS, atomic clock) used to obtain accurate time for synchronization
