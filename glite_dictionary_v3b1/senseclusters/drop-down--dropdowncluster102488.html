drop-down

adjective
UI control that reveals a downward list
3.9 1.33/m
Pronunciation: / drÉ‘p daÊŠn / (AmE), / drÉ’p daÊŠn / (BrE)
(computing) (of a user-interface control) showing a list of options that appears downward when activated; also used as a noun for the control itself.

Examples

A drop-down menu
When filling out the form, select your country from the drop-down menu to continue.
Click the small arrow to open the drop-down and choose your preferred language.
The website uses a drop-down for shipping options to keep the page layout simple.

Synonyms

  • drop-down menu
    3.2 0.77/m
    a drop-down menu shows a list of choices below a field for the user to pick one, while a drop-down names the control itself.

Related Terms

  • synonym: combo box - a GUI widget that combines a drop-down list with an editable text box
  • synonym: menu - a list of options or commands presented to the user in a user interface
