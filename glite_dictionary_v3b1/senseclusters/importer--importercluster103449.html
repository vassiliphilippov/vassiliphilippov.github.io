importer

noun
software module that imports data
4.2 0.08/m
Pronunciation: / …™mÀàp…îrt…ôr / (AmE), / …™mÀàp…îÀêt…ô / (BrE)
a software tool, module, or component that reads data in an external format and converts or loads it into a system, application, or database.

Examples

A data importer
The data pipeline uses a importer to load JSON feeds into the warehouse.
Our CMS includes an importer that converts CSV files into structured articles automatically.
Developers created several importers to migrate legacy XML records into the new database.

Synonyms

  • data converter
    5.1 0.00/m
    a data converter changes information between formats or codes, while an importer also reads external files and loads them into a system.

Related Terms

  • antonym: exporter - a tool or process that writes data out of a system to an external format or location
  • synonym: ingest - to take data into a system, often for processing or storage
