watch

noun
command-line monitoring utility
7.0
Pronunciation: /wɑːtʃ/ /; / /wɔːtʃ/ (AmE), /wɒtʃ/ (BrE)
(computing) a command-line utility or program that runs a specified command at regular intervals and displays the command's output, typically for monitoring purposes.

Examples

Watch command output
The admin used the watch command to display updated server stats every five seconds.
On Linux, she ran watch -n 2 df -h to see disk usage refresh automatically.
During the demo, he showed how watch helps detect sudden changes in log output.
