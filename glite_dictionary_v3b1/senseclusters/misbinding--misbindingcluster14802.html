misbinding

noun
incorrect association
Pronunciation: / ˌmɪsˈbaɪndɪŋ / (AmE), / ˌmɪsˈbaɪndɪŋ / (BrE)
an incorrect association between a name, variable, function, or resource and the value or object it should reference, especially in computing or software systems

Examples

Variable misbinding
The software error was traced back to a misbinding of the configuration variable.
Careful review of the code revealed a subtle misbinding that caused the function to fail.
A misbinding in the resource allocation system led to unexpected behavior during testing.

Related Terms

  • antonym: binding - the process of associating a name or variable with a value or object
  • synonym: misassignment - the act of assigning something incorrectly
