security director

noun
head of a company's security force
Pronunciation: / sɪˈkjʊrəti dəˈrɛktər / (AmE), / sɪˈkjʊərɪti dɪˈrɛktə / (BrE)
the head of a private security force responsible for overseeing the security operations of a company or organization

Examples

Security director oversees protection
As security director, she oversees the protection of all company assets and personnel.
Every major event required the approval of the security director before the venue could open to the public.
The security director reviewed the company's procedures after the recent break-in.

Related Terms

  • degree higher: chief security officer - senior executive responsible for an organization's entire security division
  • synonym: security manager - person responsible for managing security measures
  • degree lower: security officer - person responsible for enforcing security policy
