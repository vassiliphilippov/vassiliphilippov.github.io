containerize

verb
package software in a container
7.3
Pronunciation: /kən.ˈteɪ.nə.raɪz/ (AmE), /kən.ˈteɪ.nə.raɪz/ (BrE)
(in computing) to package an application, service, or its environment into a container image or runtime so it can run consistently across different systems.

Examples

To containerize an app
The team decided to containerize the microservices architecture to streamline deployment across cloud environments.
After they containerized the app, rolling updates became faster and resource use was easier to manage.
The startup began containerizing services during testing to ensure consistent behavior on developers' machines.
Tech teams containerise their apps so they run consistently across development and production environments.
By containerising the service, the engineer isolated dependencies and simplified deployment to the cloud.
After the update, they had containerised the database and rolled back when tests failed.

Related Terms

  • synonym: containerize - package and deploy software in containers (American spelling)
  • synonym: dockerize - to put an application into a Docker container (a specific container platform)
  • synonym: package - to bundle software and its dependencies together for distribution or deployment
