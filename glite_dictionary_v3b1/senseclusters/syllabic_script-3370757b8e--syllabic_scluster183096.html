syllabic script

noun
writing system with symbols for syllables
8.5 6.4
Pronunciation: /sə.ˈlæb.ɪk skrɪpt/ /; / /sɪ.ˈlæb.ɪk skrɪpt/ (AmE), /sɪ.ˈlæb.ɪk skrɪpt/ /; / /sə.ˈlæb.ɪk skrɪpt/ (BrE)
a writing system in which each character or symbol represents a syllable rather than an individual sound or letter

Examples

A syllabic script system
Cherokee is written in a syllabic script invented by Sequoyah in the early 19th century.
The Japanese writing system uses a syllabic script called hiragana for native words.
Unlike alphabetic systems, a syllabic script assigns a unique symbol to each possible syllable.

Related Terms

  • antonym: alphabetic script - writing system where each symbol represents a single sound (phoneme)
