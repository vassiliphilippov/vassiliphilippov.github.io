delayed blowback

noun
firearms operating system with delayed unlocking
8.9
Pronunciation: / dɪˈleɪd ˈbloʊˌbæk / (AmE), / dɪˈleɪd ˈbləʊˌbæk / (BrE)
A firearms operating system where the bolt overcomes initial resistance and is not fully locked when firing, delaying its opening.

Examples

Delayed blowback system
Firearms with delayed blowback tend to handle higher-pressure cartridges safely.
The new pistol uses a delayed blowback system to improve reliability and reduce recoil.
Unlike simple blowback actions, delayed blowback requires a more complex mechanism.

Related Terms

  • synonym: blowback - a firearms operating system using rearward force to operate the action
