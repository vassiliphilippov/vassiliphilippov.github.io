dex

noun
Android bytecode file
Pronunciation: / dɛks / (AmE), / dɛks / (BrE)
a compiled executable file format (Dalvik Executable) used by Android to store app bytecode

Examples

Android dex file
The developer analyzed the dex file to understand how the Android app was structured.
After compiling the project, a new dex file appeared in the build directory.
Security researchers often inspect the dex files inside APKs for vulnerabilities.

Related Terms

  • synonym: APK - the package file format used by Android to distribute and install apps
  • synonym: bytecode - a compiled code that can be executed by a virtual machine
