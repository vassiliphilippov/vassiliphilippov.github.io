untainted

adjective
safe from untrusted input (computing)
6.5 0.00/m

(in computing and software security) not originating from, or not affected by, untrusted or potentially dangerous input; considered safe for use because it has been validated, sanitized, or proven free of harmful content.

Examples

Untainted data
The security analyzer marked the user input untainted after thorough validation and encoding before use.
Before inserting into the page, the template engine rejected any string not proven untainted by the sanitizer.
Developers rely on static analysis to show that a variable remains untainted during program execution.
