set off

verb
to trigger or initiate something
5.9 2.6
Pronunciation: / sɛt ɔf / (AmE), / sɛt ɒf / (BrE)
to cause something to start, happen, or become active; to trigger or activate an event, reaction, or process

Examples

To set off an alarm
His careless remarks set off a heated argument among the team members.
The alarm was set off when someone opened the emergency exit door.
The discovery of oil set off a rush of investors to the region.
Dropping the glass set off the alarm and woke everyone up.
His careless remark set off a heated argument at the dinner table.
The announcement of sudden layoffs set off widespread protests in the city.
Accidentally dropping the device could set off a dangerous explosion.
Lightning can easily set off a forest fire if the conditions are dry.
The engineers carefully set off the demolition charges at noon.
