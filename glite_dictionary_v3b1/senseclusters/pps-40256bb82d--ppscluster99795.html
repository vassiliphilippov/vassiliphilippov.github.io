PPS

noun
network throughput unit
Pronunciation: / pi pi ɛs / (AmE), / piː piː ɛs / (BrE)
an abbreviation for packets per second, a measure of network throughput indicating how many data packets are transmitted each second

Examples

Network PPS rate
The router's performance was measured in PPS to assess its ability to handle heavy traffic.
Network engineers often use PPS to compare the efficiency of different switches.
A higher PPS value means the device can process more packets every second.

Related Terms

  • synonym: Mbps - megabits per second, a measure of data transfer speed
  • synonym: bandwidth - the maximum rate of data transfer across a network
