doctype

noun
markup document declaration
7.8
Pronunciation: / ˈdɑkˌtaɪp / (AmE), / ˈdɒkˌtaɪp / (BrE)
a declaration at the beginning of a markup document that specifies its type and version

Examples

HTML doctype declaration
Every HTML document should start with a doctype to ensure proper rendering in browsers.
If you forget the doctype, your web page might not display correctly.
The doctype tells the browser which version of HTML you are using.

Related Terms

  • synonym: declaration - a statement in code that defines something
  • synonym: metadata - data that provides information about other data
