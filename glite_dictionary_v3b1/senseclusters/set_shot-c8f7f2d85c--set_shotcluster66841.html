set shot

noun
stationary two-handed basketball shot
Pronunciation: / sɛt ʃɑt / (AmE), / sɛt ʃɒt / (BrE)
a basketball shot taken with both hands from a stationary position, without jumping

Examples

A set shot in basketball
He made a perfect set shot from beyond the three-point line.
In early basketball history, the set shot was far more common than the jump shot.
The coach taught the players how to use the set shot in certain game situations.

Related Terms

  • antonym: jump shot - a basketball shot taken while jumping
