bastard title

noun
page showing only book's title at front
Pronunciation: / ˈbæstɚd ˈtaɪtəl / (AmE), / ˈbɑːstəd ˈtaɪtəl / (BrE)
the page at the beginning of some books that displays only the title, usually prior to the main title page

Examples

The bastard title page
Collectors value books with the original bastard title pages intact.
Only the bastard title appeared on the first page; the author's name followed later.
This edition features a classic bastard title before the main text begins.

Related Terms

  • synonym: half title - a page preceding the title page with only the book's title
