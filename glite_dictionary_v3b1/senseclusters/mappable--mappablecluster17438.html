mappable

adjective
able to be mapped
6.1
Pronunciation: / ˈmæpəbəl / (AmE), / ˈmæpəbəl / (BrE)
able to be associated, assigned, or converted to another set, structure, or representation, especially in mathematics or computing

Examples

Mappable data formats
The data is mappable to different formats for use in various applications.
Only mappable fields will be transferred during the import process.
In this software, every user input must be mappable to a database column.

Related Terms

  • synonym: assignable - able to be assigned or allocated
  • synonym: convertible - able to be changed into a different form or use
  • antonym: fixed - not able to be changed or assigned
