dummy run

noun
practice or test session for preparation
5.9
Pronunciation: / ˈdʌmi rʌn / (AmE), / ˈdʌmi rʌn / (BrE)
an occasion when something is practiced or tested in order to find and correct problems before the real event or activity

Examples

A dummy run test
Before the competition, we did a dummy run to spot any problems in the setup.
The team scheduled a dummy run to ensure everything would work smoothly on the big day.
They always perform a dummy run before launching a new product.

Related Terms

  • synonym: dry run - a rehearsal or practice without real consequences
  • synonym: practice run - a trial performance or rehearsal to prepare for an event
