pass muster

verb
be acceptable; meet a required standard
9.8
Pronunciation: / pæs ˈmʌstɚ / (AmE), / pɑːs ˈmʌstə / (BrE)
to be judged as acceptable or satisfactory according to a set standard

Examples

To pass muster test
His essay didn't pass muster with the professor, so he had to rewrite it.
Only proposals that pass muster will be forwarded to the next stage.
The new safety equipment must pass muster before it can be used on site.
