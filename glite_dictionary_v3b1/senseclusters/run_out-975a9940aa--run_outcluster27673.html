run out

verb
exhaust the supply of something
1.9 28/m
Pronunciation: / rÊŒn aÊŠt / (AmE), / rÊŒn aÊŠt / (BrE)
to use up or exhaust something so that none remains; to have no more of something left

Examples

Run out of time
If we run out of milk, I'll go to the store and buy more.
The team ran out of energy in the final minutes of the game.
We must leave now before we run out of time.
The printer ran out of paper during my presentation.
They were worried they might run out of time to finish the project.
We need to hurry before the water runs out.
After several hours of hiking, I ran out and had to take a rest.
She runs out quickly after intense exercise.
The old battery will run out and stop powering your device.

Synonyms

  • use up
    2.7 4.16/m
    use up is when someone consumes all of something, while run out describes having none left.
  • exhaust
    3.7 4.43/m
    exhaust means using all of a resource completely, while run out focuses on there being nothing left.
  • deplete
    6.3 1.51/m
    deplete means to reduce a supply until little remains, while run out means there is no supply left.
  • run dry
    4.2 0.43/m
    run dry describes a source slowly becoming empty, while run out describes having no more of something left.
  • give out
    4.1 1.96/m
    give out can mean something stops working or is used up, while run out only means nothing remains.
  • run down
    3.2 0.59/m
    run down means to lose energy or power over time, while run out means to have no more of something left.
