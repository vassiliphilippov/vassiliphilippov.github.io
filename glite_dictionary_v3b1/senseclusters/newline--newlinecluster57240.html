newline

noun
line break character
Pronunciation: / ˈnuːˌlaɪn / (AmE), / ˈnjuːˌlaɪn / (BrE)
a character or symbol in computing that marks the end of a line and moves to the next line of text

Examples

A newline character
When you press Enter, a newline is added to the text file.
The program failed because it was missing a newline at the end of the last line.
Some operating systems use different codes to represent a newline in text files.
When you press Enter, a newline is added to the text file.
Some programming languages use a special symbol to represent a newline in code.
The document did not display correctly because the newline characters were missing.

Related Terms

  • synonym: carriage return - a control character that returns the cursor to the beginning of the line
  • synonym: line break - the point where a line of text ends and a new one begins
