resizable

adjective
can change size
Pronunciation: / riˈsaɪzəbəl / (AmE), / rɪˈsaɪzəbəl / (BrE)
able to be changed in size, especially by making larger or smaller as needed

Examples

Resizable image
The image is resizable, so you can make it fit your document perfectly.
Most modern software uses resizable windows to improve user experience.
He prefers a resizable table that can be extended for more guests.

Related Terms

  • synonym: adjustable - able to be changed or adapted to fit different conditions
  • antonym: fixed - not able to be changed or moved
