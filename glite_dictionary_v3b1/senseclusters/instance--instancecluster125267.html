instance

noun
cloud virtual machine
7.0
Pronunciation: /ˈɪn.stəns/ (AmE), /ˈɪn.stəns/ (BrE)
(in cloud computing) a running virtual machine or compute resource created from a machine image, used to run software or services.

Examples

Cloud instance
We launched a new instance of the application on the cloud to handle traffic spikes.
Each instance runs its own operating system and can be resized without affecting other services.
The admin terminated several idle instances overnight to reduce cloud costs and free resources.

Related Terms

  • synonym: server - a computer or program that provides services to other computers or clients
  • synonym: virtual machine - a software emulation of a physical computer that runs an operating system and applications
