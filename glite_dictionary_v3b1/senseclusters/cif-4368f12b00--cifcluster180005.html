CIF

noun
crystallography file format
8.4
Pronunciation: / siː aɪ ɛf / (AmE), / siː aɪ ɛf / (BrE)
a standard text file format used to store crystallographic data about the atomic structure of molecules and crystals

Examples

Crystal CIF file
The researcher saved the crystal structure as a CIF for further analysis.
Many scientific journals require data to be submitted in CIF format for crystallographic studies.
She converted the experimental results into a CIF to share with her colleagues.

Related Terms

  • synonym: PDB - a file format for storing three-dimensional structures of molecules, especially proteins
  • synonym: mmCIF - a macromolecular extension of the CIF file format for larger biological structures
