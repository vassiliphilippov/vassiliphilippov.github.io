initialization

noun
setting up a storage device or system to start
Pronunciation: / ɪˌnɪʃəlaɪˈzeɪʃən /; / ɪˌnɪʃələˈzeɪʃən / (AmE), / ɪˌnɪʃɪlaɪˈzeɪʃən /; / ɪˌnɪʃəlaɪˈzeɪʃən / (BrE)
the process of preparing a computer, program, or device so it is ready to use, including preparing a storage device (such as a hard disk) by formatting it and setting an initial state so it can be used by an operating system.

Examples

Initialization process
After initialization, the operating system can recognize and use the disk.
Before using the new hard drive, an initialization is required to format its sectors.
The initialization process ensures that all data storage locations are accessible.
The initialization of the new computer system was completed in under an hour.
Before using the device, proper initialization is required to ensure it works correctly.
A progress bar showed the initialization process as the software loaded.
The initialization of the new software ensures that all settings are correct before use.
After initialization, the device is ready to perform its tasks efficiently.
A quick initialization process can save users a lot of time when starting a program.
