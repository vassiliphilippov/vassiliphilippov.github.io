low-level formatting

noun
initial disk preparation in computing
6.6
Pronunciation: /loʊ ˈle.vəl ˈfɑːr.mæt.ɪŋ/ /; / /loʊ ˈle.vəl ˈfɔːr.mæt.ɪŋ/ (AmE), /ləʊ ˈle.vəl ˈfɔː.mæt.ɪŋ/ /; / /laʊ ˈle.vəl ˈfɔː.mæt.ɪŋ/ (BrE)
(computer science) the process of preparing a hard disk drive or similar storage device by creating sector and track structures so the operating system can access and store data.

Examples

Low-level formatting process
Before using the new hard drive, the technician performed low-level formatting to prepare it for data storage.
Data recovery can sometimes require low-level formatting to erase all previous traces on a drive.
Most modern users never need to worry about low-level formatting, as it comes done from the factory.

Related Terms

  • synonym: formatting - preparing a storage medium for use by an operating system
  • antonym: high-level formatting - formatting a storage device at the file system level, after physical structure is set
