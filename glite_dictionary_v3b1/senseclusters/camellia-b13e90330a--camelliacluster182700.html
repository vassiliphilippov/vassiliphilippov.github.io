Camellia

noun
symmetric block cipher algorithm
Pronunciation: / kəˈmiliə / (AmE), / kəˈmiːliə / (BrE)
(computing, cryptography) a symmetric-key block cipher algorithm used to encrypt data; standardized and widely adopted for secure communications.

Examples

Camellia encryption
The security team implemented Camellia to encrypt sensitive customer data at rest.
Some government systems prefer Camellia because it offers strong encryption with efficient performance.
The open-source library includes support for Camellia alongside AES and other ciphers.

Related Terms

  • synonym: block cipher - an encryption algorithm that operates on fixed-size blocks of data
  • synonym: symmetric-key cipher - an encryption method where the same key is used for encryption and decryption
