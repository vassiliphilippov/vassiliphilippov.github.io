exponential

noun
mathematical function with a variable as an exponent
4.2
Pronunciation: / ˌɛkspəˈnɛnʃəl / (AmE), / ˌɛkspəˈnɛnʃəl / (BrE)
a mathematical function in which an independent variable appears in the exponent, such as f(x) = a^x

Examples

An exponential function
An exponential differs from a linear function in its rate of change.
Scientists often use an exponential to model population growth.
The graph of an exponential rises rapidly as the value of x increases.

Related Terms

  • antonym: logarithm - the inverse mathematical function of an exponential
