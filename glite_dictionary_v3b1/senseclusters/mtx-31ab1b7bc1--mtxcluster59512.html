MTX

noun
matrix file format
Pronunciation: / ɛm ti ɛks / (AmE), / ɛm ti ɛks / (BrE)
a file format (with extension .mtx) used to store sparse matrices, especially in scientific computing and numerical analysis

Examples

MTX file format
The researcher saved the results as an MTX file for easy sharing with colleagues.
Many scientific software tools can read and write data in the MTX format.
Before analysis, she converted the dataset into an MTX file to handle sparse matrices efficiently.

Related Terms

  • antonym: CSV - a file format for storing tabular data as comma-separated values
  • synonym: Matrix Market - a standard for exchanging matrix data, often using the MTX format
