Dewey decimal system

noun
library classification system using numbers
4.9 0.02/m
Pronunciation: / ÀàduÀêi Ààd…õs…™m…ôl Àås…™st…ôm / (AmE), / ÀàdjuÀêi Ààd…õs…™m…ôl Àås…™st…ôm / (BrE)
a system used in libraries to organize books and materials by assigning them numbers according to subject, with each subject represented by a three-digit numeral and additional details after a decimal point

Examples

The Dewey decimal system
I learned how to look up books by number in the Dewey decimal system during library orientation.
Our local library uses the Dewey decimal system to organize its nonfiction collection.
The Dewey decimal system makes it easier to find books on similar topics.

Synonyms

  • Dewey decimal classification

  • decimal system of classification
    6.5 0.00/m
    decimal system of classification describes any library numbering using decimals, while Dewey decimal system is the specific named scheme widely used today.
