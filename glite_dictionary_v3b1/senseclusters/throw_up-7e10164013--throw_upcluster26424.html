throw up

verb
display or generate (a dialog or message)
3.7
Pronunciation: /θroʊ ʌp/ /; / /θroʊ əp/ (AmE), /θrəʊ ʌp/ /; / /θrəʊ əp/ (BrE)
to cause a window, message, dialog, or other interface element to appear on a screen; to display or generate (something) in a software context.

Examples

Throw up an error
When the server times out, the website will throw up a message asking you to try again.
After the invalid input, the program threw up multiple warnings and prevented you from continuing.
During testing, developers kept throwing up mock dialogs to reproduce the user's problem.
