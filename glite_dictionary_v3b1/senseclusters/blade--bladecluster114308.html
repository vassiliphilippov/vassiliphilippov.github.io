blade

noun
modular server unit
6.6
Pronunciation: / bleɪd / (AmE), / bleɪd / (BrE)
(in computing) a slim, modular server unit designed to fit into a shared chassis or enclosure for high-density data-centre deployments.

Examples

Blade server
The company installed a dozen new blade servers to host its cloud services in a compact rack.
Technicians slid the faulty blade out of the chassis and replaced it within minutes.
During the upgrade they consolidated several blades into a virtual cluster for better efficiency.

Related Terms

  • synonym: rack server - a server designed to be mounted in a standard equipment rack; contrasts with blades by packaging and density
  • synonym: server - a computer or device that provides services, data, or resources to other computers over a network
