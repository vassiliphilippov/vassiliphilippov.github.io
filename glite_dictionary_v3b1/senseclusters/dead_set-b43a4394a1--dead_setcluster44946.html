dead set

adjective
firmly determined or resolutely opposed
6.5 0.58/m
Pronunciation: / dÉ›d sÉ›t / (AmE), / dÉ›d sÉ›t / (BrE)
firmly determined or resolutely opposed to something; fixed in one's purpose or decision, especially when used with 'against' or 'on'.

Examples

Be dead set on success
He is dead set on becoming a doctor despite the challenges.
She was dead set against moving to another city.
They are dead set on finishing the project by next week.

Synonyms

  • dead set on
    4.8 0.23/m
    dead set on expresses firm personal determination to achieve something, while dead set describes strong resolve or opposition often with 'against' or 'on'.
  • resolute
    4.8 0.06/m
    resolute describes steady, morally strong determination, while dead set suggests a firm, often emotional decision or clear opposition.
  • adamant
    6.8 3.55/m
    adamant refuses to change and sounds absolute and formal, while dead set feels more colloquial and often shows firm opposition or resolve.
  • stubborn
    2.9 8.86/m
    stubborn holds to an idea despite reasons and often feels negative, while dead set shows firm determination or clear opposition.

Related Terms

  • synonym: adamant - refusing to be persuaded or to change one's mind
  • synonym: determined - having made a firm decision and being resolved not to change it
  • antonym: hesitant - uncertain or lacking determination
