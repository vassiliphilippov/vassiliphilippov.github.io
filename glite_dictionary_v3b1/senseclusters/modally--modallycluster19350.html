modally

adverb
by mode
Pronunciation: / ˈmoʊdəli / (AmE), / ˈməʊdəli / (BrE)
in a way that relates to the mode, or most frequent value, in a set of data or distribution

Examples

Modally distributed data
The data were modally distributed, with most responses falling into one category.
In this survey, participants modally selected option B as their favorite.
The results were modally clustered around a single answer, showing a clear preference.

Related Terms

  • antonym: mean - the average value in a data set
  • antonym: median - the middle value in a data set
  • synonym: mode - the value that appears most frequently in a data set
