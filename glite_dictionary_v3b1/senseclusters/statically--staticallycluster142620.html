statically

adverb
in a fixed way
Pronunciation: / ˈstætɪkli / (AmE), / ˈstætɪkli / (BrE)
in a way that is fixed or determined before a program runs, especially at compile or link time in computing.

Examples

Statically linked library
The configuration file was statically included in the application during the build process.
Variables declared statically remain in memory for the entire duration of the program.
Some libraries are statically linked to ensure all dependencies are present at launch.

Related Terms

  • antonym: dynamically - in a way that is determined or allocated while a program is running
