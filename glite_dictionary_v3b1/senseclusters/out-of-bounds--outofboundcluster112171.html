out-of-bounds

adjective
access beyond valid memory or array limits
7.8
Pronunciation: / cca8at 59v 8ba8andz / (AmE), / cca8at 59v 8ba8andz / (BrE)
(computing) describing an access or index that lies outside the valid numeric limits of an array, buffer, or memory region and may cause an error or corruption.

Examples

Out-of-bounds error
The program crashed after an out-of-bounds access to the array.
A unit test caught an out-of-bounds read that produced corrupt output during processing.
Static analysis tools warn when code may perform an out-of-bounds write to memory.

Related Terms

  • antonym: in-bounds - inside the allowed limits or valid range (opposite of out-of-bounds)
  • synonym: out-of-range - having a value outside the allowed or expected numeric range
