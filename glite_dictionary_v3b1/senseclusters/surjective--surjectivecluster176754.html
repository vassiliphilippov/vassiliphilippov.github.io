surjective

adjective
mapping onto set
9.9
Pronunciation: / sɚˈdʒɛktɪv / (AmE), / sɜːˈdʒɛktɪv / (BrE)
describing a function in mathematics where every element in the target set is mapped to by at least one element from the domain

Examples

Surjective function
A function is called surjective if every value in the codomain is assigned to at least once.
The teacher asked whether the mapping was surjective or not for the given sets.
In mathematics, a surjective function is also known as an onto function.

Related Terms

  • degree higher: bijective - describing a function that is both injective and surjective
  • antonym: injective - describing a function where each element of the target set is mapped to by at most one element of the domain
