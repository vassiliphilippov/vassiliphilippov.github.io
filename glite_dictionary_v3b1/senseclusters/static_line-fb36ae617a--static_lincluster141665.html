static line

noun
cord for automatic parachute deployment
Pronunciation: / ˈstætɪk laɪn / (AmE), / ˈstætɪk laɪn / (BrE)
a cord attached to both the aircraft and a parachute pack, used to automatically deploy the parachute when a jumper exits the aircraft

Examples

A static line cord
A malfunction in the static line can prevent the parachute from opening.
During training jumps, beginners often use a static line for added safety.
The static line ensures that the parachute deploys as soon as the jumper leaves the plane.
