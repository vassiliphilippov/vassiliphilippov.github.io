file server

noun
computer that controls access to shared network files
1.0
Pronunciation: / ˈfaɪl ˌsɝːvər / (AmE), / ˈfaɪl ˌsɜːvə / (BrE)
a computer on a network that manages and provides controlled access to shared files and resources for other computers

Examples

The file server crashed
Our office file server crashed last night, so we temporarily lost access to all shared documents.
The IT team upgraded the file server to improve network performance and security.
You need permission from the administrator to store files on the company's file server.

Related Terms

  • synonym: server - a computer or program that provides services to other computers in a network
