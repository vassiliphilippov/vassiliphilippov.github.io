swappable

adjective
able to be swapped
6.1
Pronunciation: / ˈswɑːpəbəl / (AmE), / ˈswɒpəbəl / (BrE)
able to be moved from main memory to secondary storage by an operating system.

Examples

Swappable memory page
The operating system marked the memory page as swappable to free up space.
Only swappable files can be moved to the swap partition during heavy usage.
If a process uses too much RAM, its swappable data may be written to disk.

Related Terms

  • antonym: fixed - not able to be moved or changed
  • synonym: removable - able to be taken out or detached
