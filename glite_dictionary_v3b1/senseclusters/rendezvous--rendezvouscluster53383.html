rendezvous

noun
server or mechanism for peer discovery
7.7
Pronunciation: /ˈrɑːn.deɪ.vuː/ /; / /ˈrɑːn.də.vuː/ (AmE), /ˈrɒn.deɪ.vuː/ /; / /ˈrɒn.də.vuː/ (BrE)
(computing, networking) a server or mechanism that helps two or more network endpoints discover each other and establish a connection, often by exchanging addresses or connection details.

Examples

Rendezvous server
The peer-to-peer app uses a rendezvous server to locate peers before starting the connection process.
Developers implemented a lightweight rendezvous service so devices behind firewalls could find each other.
During testing, the client failed because the central rendezvous had been misconfigured and returned wrong addresses.

Related Terms

  • synonym: discovery server - a service that helps clients find other services or peers on a network
  • synonym: signaling server - a server that exchanges session information and connection parameters between peers to establish a direct link
