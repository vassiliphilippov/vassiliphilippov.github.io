pin table

noun
British term for pinball machine
Pronunciation: / pɪn ˈteɪbəl / (AmE), / pɪn ˈteɪbl / (BrE)
A machine or table on which the game of pinball is played; chiefly used in British English for 'pinball machine'.

Examples

A pin table game
He spent hours trying to beat the high score on the pin table.
The old pub still has a pin table in the corner, attracting both locals and tourists.
When visiting England, I learned that a pin table is what they call a pinball machine.
