memory access

noun
operation of reading or writing computer memory
1.3
Pronunciation: / ˈmɛməri ˈæksɛs / (AmE), / ˈmɛməri ˈæksɛs / (BrE)
(computer science) the process of reading from or writing to a computer's memory.

Examples

Memory access error
Errors in memory access may cause a program to crash or behave unpredictably.
Modern processors use caches to optimize memory access times.
The speed of memory access can greatly affect a computer's overall performance.
