stubbing

noun
temporary code for testing
Pronunciation: / ˈstʌbɪŋ / (AmE), / ˈstʌbɪŋ / (BrE)
the practice in programming of replacing a real function or module with a simple temporary version for testing purposes

Examples

Stubbing a function
The developers used stubbing to test the login feature without connecting to the real database.
By applying stubbing, the team could focus on the user interface before the backend was ready.
Effective stubbing helps identify bugs early in the software development process.

Related Terms

  • antonym: integration testing - testing combined parts of a program to see if they work together
  • synonym: mocking - using simulated objects or functions in software testing
