contig

noun
continuous data block
Pronunciation: / ˈkɒntɪɡ / (AmE), / ˈkɒntɪɡ / (BrE)
a continuous block of storage, such as memory or disk sectors, used to hold a file or data in computing

Examples

Contig block
The software merged several fragments into a single contig to improve performance.
A large contig can speed up data retrieval from the hard drive.
Engineers analyzed each contig to ensure the file was stored efficiently.

Related Terms

  • antonym: fragment - a small or broken part of data or storage
  • synonym: segment - a part or section of something, especially in computing
