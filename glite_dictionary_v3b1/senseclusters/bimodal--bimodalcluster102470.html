bimodal

adjective
involving two data modalities
6.8
Pronunciation: /baɪ.ˈmoʊ.dəl/ /; / /baɪ.ˈməʊ.dəl/ (AmE), /baɪ.ˈməʊ.dəl/ (BrE)
(in data science and machine learning) involving or using two different data modalities or input types (for example, text and images) in a single model, dataset, or task.

Examples

Bimodal data model
Researchers trained a bimodal model to match captions with corresponding images for better retrieval.
The app uses a bimodal approach, combining voice commands and camera input for accessibility.
During the course, students built a bimodal dataset containing paired sentences and photos for training.

Related Terms

  • synonym: multimodal - involving multiple data modalities (often two or more, e.g., text and image)
  • antonym: unimodal - involving a single data modality (only one type of input)
