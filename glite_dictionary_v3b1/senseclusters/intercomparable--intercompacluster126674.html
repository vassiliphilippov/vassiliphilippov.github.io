intercomparable

adjective
able to be compared
7.7
Pronunciation: / ˌɪntərkəmˈpɛrəbəl / (AmE), / ˌɪntəkəmˈpærəbəl / (BrE)
able to be compared with each other in a meaningful or valid way

Examples

Intercomparable data sets
The data sets are intercomparable because they were collected using the same criteria.
For the results to be intercomparable, all experiments must follow identical procedures.
The two surveys are not intercomparable due to differences in their target populations.
