drop-down menu

noun
computer menu appearing below an item
Pronunciation: / ˈdrɑp ˌdaʊn ˈmɛnju / (AmE), / ˈdrɒp ˌdaʊn ˈmɛnjuː / (BrE)
A list of options that appears below a selected item on a computer screen, allowing the user to choose one.

Examples

The drop-down menu list
Click the 'File' drop-down menu to save your document.
Hovering over the account icon opens a drop-down menu with additional profile settings.
The website's navigation bar includes a drop-down menu for selecting languages.

Related Terms

  • synonym: menu - a list of commands or options in a computer interface
