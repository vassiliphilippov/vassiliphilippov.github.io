subpanel

noun
secondary interface section
4.8
Pronunciation: / ˈsʌbˌpænəl / (AmE), / ˈsʌbˌpænəl / (BrE)
a smaller or secondary section within a main panel, often in a user interface, that contains related controls or information

Examples

Settings subpanel
You can find advanced settings in the subpanel below the main options.
The help subpanel appears when you click the question mark icon.
Each subpanel organizes related tools for easier navigation in the software.

Related Terms

  • synonym: panel - a main section or area in a user interface containing controls or information
  • synonym: window - a separate viewing area in a user interface
