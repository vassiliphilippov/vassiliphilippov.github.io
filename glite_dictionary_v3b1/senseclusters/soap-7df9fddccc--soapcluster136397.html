SOAP

noun
XML-based web services protocol
6.3
Pronunciation: / ˌɛs oʊ eɪ ˈpi / (AmE), / ˌɛs əʊ eɪ ˈpi / (BrE)
An XML-based protocol for exchanging structured information in web services; acronym of Simple Object Access Protocol.

Examples

SOAP web protocol
The developer configured the API to accept requests via SOAP for legacy integration.
Many modern services prefer JSON over SOAP because REST APIs are simpler and faster.
She studied SOAP messages to debug the authentication problem between systems.
