SOAP

noun
XML-based web services protocol
6.3
Pronunciation: /soʊp/ (AmE), /səʊp/ (BrE)
An XML-based protocol for exchanging structured information in web services; acronym of Simple Object Access Protocol.

Examples

SOAP web protocol

Many modern services prefer JSON over SOAP because REST APIs are simpler and faster.
