(in mathematics) having a determinant of plus or minus one, especially referring to a matrix

Examples

Unimodular matrix
A unimodular matrix is important in integer programming and combinatorics.
If the transformation is unimodular, it preserves the volume in space.
The algorithm only works when the input matrix is unimodular.
A matrix is unimodular if its determinant is plus or minus one.
The pair (a, b) is unimodular if there exist x and y such that ax + by = 1.
In module theory, a unimodular tuple can generate the entire module as an ideal.
