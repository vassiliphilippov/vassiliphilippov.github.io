table

verb
arrange in tabular form
Pronunciation: / ˈteɪ.bəl / (AmE), / ˈteɪ.bəl / (BrE)
to arrange or record information in a table (an organized set of data in columns and rows)

Examples

To table the data
All the expenses were carefully tabled in the monthly report.
The researcher decided to table the survey results for easier analysis.
You should table the data so that stakeholders can interpret it at a glance.

Related Terms

  • synonym: tabulate - to arrange data systematically in a table
