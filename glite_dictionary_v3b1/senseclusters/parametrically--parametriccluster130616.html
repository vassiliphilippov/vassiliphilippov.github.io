parametrically

adverb
using parameters
Pronunciation: / ˌpærəˈmɛtrɪkli / (AmE), / ˌpærəˈmɛtrɪkli / (BrE)
in a way that involves or uses parameters, especially in mathematics or statistics

Examples

To analyze parametrically
The researchers analyzed the results parametrically to account for all relevant variables.
Some models can be solved parametrically rather than using numerical methods.
The experiment was designed parametrically to test the effect of each parameter separately.
The engineer defined the curve parametrically to make it easier to modify its shape.
In computer graphics, shapes are often described parametrically for greater flexibility.
The function was plotted parametrically to show how its coordinates change with the parameter.
