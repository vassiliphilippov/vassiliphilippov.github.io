Synonyms

  • central processing unit
    4.2 0.13/m
    central processing unit is the full technical name used in formal writing, while CPU is the common abbreviation used in speech and labels.
  • processor
    3.4 8.88/m
    processor is a short, general name for the chip that runs instructions, while CPU refers specifically to the central processing unit.
  • central processor
    3.7 0.07/m
    central processor is a less common phrase for the main computing chip, whereas CPU is the widely used abbreviation in technical contexts.
