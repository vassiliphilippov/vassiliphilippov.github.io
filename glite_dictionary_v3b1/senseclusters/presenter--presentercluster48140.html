presenter

noun
software component in MVP pattern
Pronunciation: / prɪˈzɛntər / (AmE), / prɪˈzɛntə / (BrE)
(in software design) a component in the Model–View–Presenter (MVP) pattern that mediates between the model and the view and contains presentation logic.

Examples

MVP presenter
In the MVP pattern, the presenter coordinates updates between the model and the view.
When testing, we mocked the presenter to verify it handled user actions correctly and updated the view.
The junior developer refactored the presenter to remove UI code and keep business logic in the model.

Related Terms

  • synonym: controller - component in the MVC pattern that handles user input and coordinates model and view; a related role in a different pattern.
  • synonym: view-model - component in the MVVM pattern that exposes data and commands for the view; a related design alternative.
