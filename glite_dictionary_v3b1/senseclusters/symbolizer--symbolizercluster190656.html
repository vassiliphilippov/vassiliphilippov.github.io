symbolizer

noun
GIS map-rendering component
7.8
Pronunciation: /ˈsɪm.bə.laɪ.zər/ (AmE), /ˈsɪm.bə.laɪ.zə/ (BrE)
(in GIS and digital cartography) a style specification or software component that defines how geographic features are drawn on a map, including shapes, colours, line widths, and labeling rules.

Examples

Map symbolizer settings
The cartographer adjusted the symbolizer to make rivers appear thicker and darker on the map.
In the mapping software, each layer uses a different symbolizer to show points, lines, or polygons clearly.
They exported the style file so the same symbolizers could be reused across multiple projects.

Related Terms

  • synonym: renderer - a program or component that draws or displays graphical content, including map features
  • synonym: style - a set of visual rules or specifications that determine the appearance of map features
