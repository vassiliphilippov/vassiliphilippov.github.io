(in web development) a pattern or expression (for example in CSS or JavaScript) used to identify one or more HTML elements so styles or scripts can be applied.

Examples

CSS selector
In CSS, a selector targets elements so the browser can apply the chosen styles.
Use a class-based selector when you want to style many elements the same way.
The JavaScript function document.querySelector accepts a CSS selector string to find the element.

Related Terms

  • synonym: CSS selector - the CSS syntax for writing selectors that match HTML elements for styling
  • synonym: querySelector - a JavaScript method that accepts a selector string and returns matching elements
