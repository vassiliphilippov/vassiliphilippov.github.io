swap

verb
move program part into memory (computing)
3.0 5.2
Pronunciation: /swɑːp/ (AmE), /swɒp/ (BrE)
(in computing) To move a program or a part of it into memory so it can be executed, typically by exchanging it with another part already in memory.

Examples

To swap memory
Older systems often had to swap large parts of programs to minimize memory usage.
The process scheduler may swap a task back into memory when it is ready to run.
When the operating system needs more RAM, it can swap inactive program segments into memory.
