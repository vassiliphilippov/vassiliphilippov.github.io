get going

verb
start to be active; begin
1.0
Pronunciation: / ɡɛt ˈɡoʊɪŋ / (AmE), / ɡɛt ˈɡəʊɪŋ / (BrE)
to start being active, begin doing something, or set out to go somewhere

Examples

To get going early
It took her a while to get going this morning after staying up late.
Let's get going on that project before the deadline approaches.
We should get going if we want to catch the early train.
It took a bit of motivation, but I finally got going on my project.
Let’s get going before the weather gets worse.
We need to get going or we’ll be late for the meeting.
