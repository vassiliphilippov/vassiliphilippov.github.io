linefeed

noun
computer control character
Pronunciation: / ˈlaɪnˌfiːd / (AmE), / ˈlaɪnˌfiːd / (BrE)
a special character or control code in computing that moves the cursor or printer to the next line, or marks the end of a line in text files

Examples

A linefeed character
Each time the printer receives a linefeed, it advances the paper to the next line.
Some text files use a linefeed character to separate lines, while others use a carriage return.
The programmer had to replace every linefeed in the document with a different character.
On Unix systems, each new line in a text file is marked by a linefeed character.
The programmer had to replace every linefeed with a carriage return for compatibility.
Some editors let you choose between linefeed and carriage return for line endings.

Related Terms

  • antonym: carriage return - a control character that moves the cursor to the beginning of the line
  • synonym: newline - a character or sequence that marks the end of a line of text
