abend

verb
to end abnormally
9.6
Pronunciation: / əˈbɛnd / (AmE), / əˈbɛnd / (BrE)
to stop a program or process unexpectedly due to an error, especially in computing contexts

Examples

To abend suddenly
If the application encounters a memory error, it may abend without warning.
The server will abend if the configuration file is missing or corrupted.
Developers worked late to fix the bug that caused the program to abend during startup.

Related Terms

  • synonym: crash - to stop functioning suddenly due to an error
  • synonym: fail - to stop working as intended
  • antonym: run - to operate or function as intended
