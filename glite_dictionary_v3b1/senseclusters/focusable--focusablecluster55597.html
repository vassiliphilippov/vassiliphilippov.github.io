focusable

adjective
can receive focus
3.9
Pronunciation: /ˈfoʊ.kə.bəl/ (AmE), /ˈfəʊ.kə.bəl/ (BrE)
able to receive input focus in a user interface, allowing selection or activation by keyboard or other navigation methods

Examples

Focusable button
The button is focusable, so users can select it using the Tab key.
If a field is not focusable, it cannot be accessed by keyboard navigation.
Web developers should ensure all important elements are focusable for accessibility.

Related Terms

  • antonym: disabled - not able to be selected or activated
  • synonym: interactive - allowing user interaction or response
  • synonym: selectable - able to be chosen or highlighted in a user interface
