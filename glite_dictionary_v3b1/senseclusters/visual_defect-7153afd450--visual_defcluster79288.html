visual defect

noun
flaw in appearance or rendering
4.2
Pronunciation: /ˈvɪ.ʒu.əl ˈdiː.fekt/ /; / /ˈvɪʒ.wəl ˈdiː.fekt/ (AmE), /ˈvɪz.juː.əl ˈdiː.fekt/ /; / /ˈvɪ.ʒuː.əl ˈdiː.fekt/ (BrE)
a flaw, error, or unintended problem in the visual appearance or rendering of a product, interface, image, or design (for example a UI bug, color mismatch, or pixelation).

Examples

UI visual defect
During testing, we logged a minor visual defect where the header gradient appeared pixelated on some devices.
The designer marked the brochure's visual defect and asked the printer to correct the color mismatch immediately.
After the update, several users reported a visual defect where thumbnails overlapped in the gallery view.

Related Terms

  • synonym: UI bug - a fault in a user interface that affects how elements are displayed or behave
  • synonym: cosmetic defect - a non-functional flaw that affects appearance rather than operation
  • synonym: visual bug - an unintended rendering or layout problem visible to users
