discriminant

noun
classification value
Pronunciation: / dɪˈskrɪmɪnənt / (AmE), / dɪˈskrɪmɪnənt / (BrE)
a value or function used to distinguish or classify items into different groups, especially in statistics or machine learning

Examples

Statistical discriminant
The machine learning model calculated a discriminant to separate the two types of flowers.
Each applicant received a discriminant score to help determine their eligibility for the program.
In statistics, a discriminant function can help classify data points into different categories.

Related Terms

  • synonym: classifier - a function or algorithm that assigns items to categories
  • synonym: predictor - a variable or function used to forecast outcomes
