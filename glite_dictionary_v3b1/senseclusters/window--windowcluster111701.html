window

noun
computer screen area displaying information
1.6
Pronunciation: / ˈwɪndoʊ / (AmE), / ˈwɪndəʊ / (BrE)
A rectangular area on a computer screen that displays information independently of other areas.

Examples

A computer window
If too many windows are open, it can make the computer run slowly.
She opened a new window to check her email without closing the document.
You can resize the window by dragging its edges with the mouse.

Related Terms

  • synonym: dialog box - A small window on a computer screen for communication with the user.
