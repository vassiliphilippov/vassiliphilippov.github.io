contrib

noun
software add-ons folder
9.3
Pronunciation: / kənˈtrɪb / (AmE), / kənˈtrɪb / (BrE)
a directory or section in a software project that contains contributed add-ons, plugins, or optional modules from users or third parties

Examples

Contrib folder
You can find extra themes in the contrib directory of the project repository.
Developers are encouraged to submit their modules to the contrib section for others to use.
The contrib folder contains useful scripts that are not part of the main software.

Related Terms

  • antonym: core - the main part of a software project, excluding optional or contributed modules
  • synonym: plugin - a software component that adds a specific feature to an existing program
