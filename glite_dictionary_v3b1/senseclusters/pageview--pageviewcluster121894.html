pageview

noun
web page visit
3.1
Pronunciation: / ˈpeɪdʒˌvjuː / (AmE), / ˈpeɪdʒˌvjuː / (BrE)
an instance of a single web page being loaded or viewed by a user in a browser

Examples

A pageview count
The website recorded over 10,000 pageviews during the first week of its launch.
Each time a user loads a new article, it counts as a pageview for the site.
Mark analyzed the pageviews to see which blog posts were most popular last month.

Related Terms

  • synonym: hit - a request for a file from a web server
  • degree higher: session - a period of user activity on a website
