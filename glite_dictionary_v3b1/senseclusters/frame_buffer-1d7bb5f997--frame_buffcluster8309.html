frame buffer

noun
memory storing display pixel data
5.4
Pronunciation: /freɪm ˈbʌf.ər/ (AmE), /freɪm ˈbʌf.ə/ (BrE)
(computing) a region of memory used to store pixel data for an image or display screen, with each position corresponding to a pixel's value.

Examples

The frame buffer memory
The graphics card uses a frame buffer to store the images before displaying them on the monitor.
Updating the frame buffer rapidly is essential for smooth animation in video games.
When the frame buffer gets corrupted, the screen can display random colors or patterns.

Related Terms

  • synonym: video memory - memory used for storing graphical data in computers or consoles
