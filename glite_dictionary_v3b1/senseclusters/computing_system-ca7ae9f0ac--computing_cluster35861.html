computing system

noun
group of computers with shared software and storage
Pronunciation: / kəmˈpjuːtɪŋ ˈsɪstəm / (AmE), / kəmˈpjuːtɪŋ ˈsɪstəm / (BrE)
A group of one or more computers, together with their software and shared storage, that operate as a unit.

Examples

A computing system unit
A modern computing system can include both on-premises servers and cloud resources.
Security protocols must protect every component of the computing system.
The company upgraded its computing system to handle larger data sets efficiently.
