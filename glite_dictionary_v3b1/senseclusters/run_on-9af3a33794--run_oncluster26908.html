run on

verb
be powered by or depend on
1.4
Pronunciation: / rʌn ɑn / (AmE), / rʌn ɒn / (BrE)
to be powered by a particular fuel or to rely on something as a source of energy or motivation

Examples

Cars run on petrol
Most modern cars run on unleaded petrol instead of diesel.
Some people seem to run on very little sleep and still function well.
This generator can run on both gasoline and propane.

Related Terms
