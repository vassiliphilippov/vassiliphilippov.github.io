reflection

noun
programming feature to inspect/modify code at runtime
7.9
Pronunciation: /rə.ˈflek.ʃən/ /; / /rɪ.ˈflek.ʃən/ (AmE), /rɪ.ˈflek.ʃən/ /; / /rə.ˈflek.ʃən/ (BrE)
(in programming) the ability of a program to examine and, in some languages, modify its own classes, methods, or metadata at runtime.

Examples

Code reflection
Java's reflection API allows developers to inspect classes and call methods dynamically at runtime.
Using reflection, the testing framework created mock objects for private methods during unit tests.
Overuse of reflection can make code slower and harder to understand and maintain.

Related Terms

  • synonym: introspection - in programming, inspecting a program's own structure or type information at runtime
  • synonym: metaprogramming - techniques for writing programs that generate, manipulate, or reason about other programs or themselves
