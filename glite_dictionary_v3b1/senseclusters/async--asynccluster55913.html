async

noun
asynchronous discussion
7.7
Pronunciation: / ˈeɪsɪŋk / (AmE), / ˈeɪsɪŋk / (BrE)
an asynchronous process, function, or discussion (especially in computing or online communication)

Examples

An async update
We decided to keep the project updates async so everyone could reply when convenient.
The team prefers async communication to accommodate different time zones.
Instead of a live meeting, they used an async to gather feedback over several days.
The developer created an async to handle data requests without blocking the user interface.
Each async in the program runs independently, improving overall performance.
You should check if the async has completed before accessing its result.

Related Terms

  • synonym: asynchronous - not happening at the same time
  • synonym: asynchronous function - a function that operates independently of the main program flow
  • antonym: real-time - happening instantly as events occur
  • antonym: synchronous operation - an operation that runs in sequence with the main program flow
