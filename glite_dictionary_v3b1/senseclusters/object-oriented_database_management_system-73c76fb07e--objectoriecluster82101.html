object-oriented database management system

noun
system managing object-oriented databases
6.7
Pronunciation: / ˌɑb.dʒɪkt ˈɔːr.i.ɛn.tɪd ˌdeɪ.təˈbeɪs ˌmæn.ɪdʒ.mənt ˈsɪs.təm / (AmE), / ˌɒb.dʒɪkt ˈɒː.ri.ɛn.tɪd ˌdeɪ.təˈbeɪs ˌmæn.ɪdʒ.mənt ˈsɪs.təm / (BrE)
A database management system designed to handle, store, and manage data as objects, as used in object-oriented programming.

Examples

An object-oriented database management system
An object-oriented database management system efficiently handles multimedia and hierarchical data structures.
Many companies prefer an object-oriented database management system for complex data modeling tasks.
The university migrated its student records to an object-oriented database management system for better scalability.
