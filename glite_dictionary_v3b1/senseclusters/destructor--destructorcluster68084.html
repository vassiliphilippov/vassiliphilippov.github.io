destructor

noun
object-destroying function
Pronunciation: / dɪˈstrʌktər / (AmE), / dɪˈstrʌktə / (BrE)
a special function in programming that is called to destroy an object or a device that destroys something

Examples

C++ destructor function
In C++, a destructor is used to release resources when an object goes out of scope.
The old machine acted as a destructor, breaking down waste materials for recycling.
Every class in the program should have a properly defined destructor to prevent memory leaks.

Related Terms

  • antonym: constructor - a special function in programming that creates and initializes an object
  • synonym: destroyer - someone or something that destroys
