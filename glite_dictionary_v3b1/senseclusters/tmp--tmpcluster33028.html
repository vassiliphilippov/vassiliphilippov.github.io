tmp

noun
temporary file directory
Pronunciation: / ti ɛm pi / (AmE), / ti ɛm pi / (BrE)
a system environment variable that indicates the directory used for storing temporary files on a computer.

Examples

Tmp variable path
If the tmp variable is not set correctly, some programs may not run as expected.
The script saves its temporary data in the directory specified by tmp.
Clearing the files in tmp can help free up disk space on your computer.

Related Terms

  • synonym: cache - a storage location for temporary data to speed up processes
  • synonym: temp - a common abbreviation for temporary, often used for temporary files or folders
