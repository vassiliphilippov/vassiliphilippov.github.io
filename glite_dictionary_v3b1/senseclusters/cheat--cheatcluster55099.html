cheat

noun
hidden game command or program
5.2
Pronunciation: / tʃit / (AmE), / tʃiːt / (BrE)
a hidden command, code, program, or file used to gain an unfair advantage in a video game or software application.

Examples

A game cheat code

Players often download cheats online that change character stats and make the game easier.
Using a cheat during competitive matches can result in bans from the server.
