homepage

noun
main or first page of a website
Pronunciation: / ˈhoʊmˌpeɪdʒ / (AmE), / ˈhəʊmˌpeɪdʒ / (BrE)
the main or first page of a website, or the default page displayed when a web browser starts

Examples

Website homepage
She redesigned their homepage to make it more user-friendly.
The company's homepage features the latest news and updates.
You can set your favorite site as your browser's homepage.
