bridle path

noun
path for riding or leading horses
7.9
Pronunciation: / ˈbraɪdəl ˌpæθ / (AmE), / ˈbraɪdəl ˌpɑːθ / (BrE)
a path or trail designed for riding or leading horses, but not for vehicles

Examples

A bridle path for horses
Cyclists are not allowed on the bridle path, which is reserved for horses.
The old estate still maintains a private bridle path for horseback riders.
We rode along the bridle path that winds through the forest.

Related Terms

  • synonym: trail - a path or track, especially through a wild or rural area
