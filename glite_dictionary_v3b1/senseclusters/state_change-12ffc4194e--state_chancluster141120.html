state change

noun
change from one physical state to another
4.1 0.01/m
Pronunciation: / steÉªt tÊƒeÉªndÊ’ / (AmE), / steÉªt tÊƒeÉªndÊ’ / (BrE)
a physical process in which a substance changes from one state (such as solid, liquid, or gas) to another without altering its chemical composition

Examples

A state change process
Boiling water involves a state change from liquid to gas.
Freezing is the state change when a liquid becomes a solid.
Melting ice is an example of a state change from solid to liquid.

Synonyms

  • phase transition
    5.5 0.35/m
    phase transition is a formal scientific term for changes between solid, liquid, or gas, while state change is the general physical process.
  • phase change
    4.1 0.13/m
    phase change is the common phrase for switching between solid, liquid, and gas, while state change names the same physical process.
  • physical change
    3.4 0.36/m

  • change of state
    4.3 0.03/m
