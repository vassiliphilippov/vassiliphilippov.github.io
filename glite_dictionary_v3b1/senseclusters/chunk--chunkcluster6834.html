chunk

noun
block of digital data
5.1
Pronunciation: /tʃʌŋk/ (AmE), /tʃʌŋk/ (BrE)
A block or piece of digital data (often fixed-size) used for storage, processing, transmission, or streaming.

Examples

A data chunk
The video was split into 4KB chunks to enable smooth buffering during playback.
Each database shard stores a different chunk of the dataset to balance load across servers.
When uploading, the client sends each file in several chunks so transfers can resume if interrupted.

Related Terms

  • synonym: block - a contiguous set of data or storage units treated as a single unit
  • synonym: segment - a portion of data or a stream divided for transmission or processing
