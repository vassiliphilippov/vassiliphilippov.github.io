repository

noun
digital store for source code (e.g., Git)
4.4
Pronunciation: /rɪ.ˈpɑː.zɪ.tɔːr.iː/ /; / /riː.ˈpɑː.zə.tɔːr.iː/ (AmE), /rɪ.ˈpɒz.ɪ.tə.riː/ /; / /rɪ.ˈpɒz.ɪ.triː/ (BrE)
(computing) a digital storage location, often version-controlled, for source code and related files and history, typically managed by systems such as Git.

Examples

A code repository
The team pushed the fix to the repository on the cloud, ready for review.
I cloned the repository to my laptop and began working on the bug fix.
Before releasing, she checked every branch in the repository for failing tests.
