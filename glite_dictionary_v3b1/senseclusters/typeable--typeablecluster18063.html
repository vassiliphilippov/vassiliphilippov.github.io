typeable

adjective
can be typed
Pronunciation: / ˈtaɪpəbəl / (AmE), / ˈtaɪpəbəl / (BrE)
able to be entered or produced using a keyboard or typewriter

Examples

Typeable characters
Please use only typeable characters when creating your username.
Some symbols are not typeable on a standard keyboard.
The form requires all answers to be typeable for easy processing.

Related Terms

  • synonym: printable - able to be printed or displayed in print
  • antonym: untypable - not able to be entered using a keyboard or typewriter
