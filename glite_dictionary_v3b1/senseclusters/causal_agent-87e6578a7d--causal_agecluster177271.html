causal agent

noun
entity that causes an effect or event
4.4
Pronunciation: / ˈkɔzəl ˈeɪdʒənt / (AmE), / ˈkɔːzəl ˈeɪdʒənt / (BrE)
an entity that produces an effect or is responsible for causing events or outcomes

Examples

A causal agent causes events
In philosophy, a causal agent is necessary to explain why events occur.
Scientists are searching for the causal agent behind the sudden fish deaths.
The virus is considered the causal agent of the disease.
