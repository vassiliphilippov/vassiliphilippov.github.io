canonical

adjective
preferred webpage version (SEO)
Pronunciation: / kəˈnænɪkəl / (AmE), / kəˈnɒnɪkəl / (BrE)
(in web publishing and SEO) designated as the preferred or authoritative version of a web page or URL for indexing and ranking by search engines.

Examples

Canonical URL
The canonical URL tells search engines which page version should be indexed.
When a site has duplicates, add a canonical link element to point to the preferred page.
Google may ignore conflicting signals if the canonical tag is implemented incorrectly on many pages.

Related Terms

  • synonym: authoritative - regarded as reliable and accepted as the correct source
  • synonym: preferred - chosen as the most suitable or primary option among alternatives
