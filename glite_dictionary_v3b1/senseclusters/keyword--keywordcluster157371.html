keyword

noun
reserved programming word
4.7
Pronunciation: /ˈkiː.wɜːrd/ (AmE), /ˈkiː.wɜːd/ (BrE)
a reserved word in a computer programming language that has a special meaning and cannot be used as an ordinary identifier

Examples

Programming keyword
In Python, 'def' is a keyword that you cannot use as a variable name.
Each programming language has its own set of keywords that are reserved for special functions.
If you try to use a keyword as a variable, the code will produce an error.
