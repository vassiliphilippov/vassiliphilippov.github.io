set up as the patsy

verb
to frame someone to take the blame
9.1 9.1
Pronunciation: / sɛt ʌp æz ðə ˈpætsi / (AmE), / sɛt ʌp æz ðə ˈpætsi / (BrE)
to make someone appear to be guilty or responsible for something they did not do, usually by deliberate deception

Examples

In the movie, an innocent man is set up as the patsy to divert attention from the real criminals.
The company executive was set up as the patsy for the financial scandal.
