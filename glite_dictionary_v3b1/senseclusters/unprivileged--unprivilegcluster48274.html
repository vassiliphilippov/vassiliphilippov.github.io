unprivileged

adjective
lacking special permissions
Pronunciation: / ʌnˈprɪvəlɪdʒd / (AmE), / ʌnˈprɪvɪlɪdʒd / (BrE)
having only limited permissions or rights, especially not having administrative or elevated access in computing systems

Examples

Unprivileged user account
The software should always run in an unprivileged mode to prevent security risks.
He was unable to install new programs because his account was unprivileged.
Access to sensitive files is blocked for unprivileged users on the system.

Related Terms

  • synonym: restricted - limited in access or ability
