promotion system

noun
platform algorithm that boosts content
3.7
Pronunciation: / pr59c8mo8a8359n c8s6ast59m / (AmE), / pr59c8m598a8359n c8s6ast59m / (BrE)
(on digital platforms) an algorithm or set of rules that selects, ranks, or boosts content so it is shown to more users or given greater visibility.

Examples

Platform promotion system
The platform's promotion system determines which posts gain visibility among millions of daily users.
Advertisers paid extra to appear higher because the site's promotion system favored sponsored content that gets clicks.
Journalists argued the promotion system amplified misinformation by prioritizing engagement over accuracy during the election.

Related Terms

  • synonym: ranking algorithm - an algorithm that orders content by predicted relevance or engagement
  • synonym: recommendation system - a system that suggests or ranks items (e.g., posts, videos) for users based on algorithms and user data
