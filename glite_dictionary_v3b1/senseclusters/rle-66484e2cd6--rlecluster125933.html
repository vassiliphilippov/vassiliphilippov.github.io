RLE

sym
data compression method
8.2
Pronunciation: / ɑr ɛl i / (AmE), / ɑː ɛl iː / (BrE)
an abbreviation for 'run-length encoding', a method of compressing data by representing repeated values as a single value and count

Examples

RLE compression method
The graphics file was compressed using RLE to save storage space.
Many simple image formats support RLE for efficient data storage.
When transmitting data, RLE can help reduce the amount of information sent.

Related Terms

  • synonym: compression - the process of reducing the size of data
  • antonym: encryption - the process of converting data into a code to prevent unauthorized access
