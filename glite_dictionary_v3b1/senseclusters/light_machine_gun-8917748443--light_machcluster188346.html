light machine gun

noun
portable automatic firearm for individual use
Pronunciation: / laɪt məˈʃin ɡʌn / (AmE), / laɪt məˈʃiːn ɡʌn / (BrE)
a type of machine gun designed to be operated by a single soldier, usually lighter and more portable than other machine guns

Examples

A light machine gun
During the battle reenactment, an actor demonstrated how a light machine gun operates.
He trained for hours to master the effective use of a light machine gun in combat situations.
The squad deployed a light machine gun to provide covering fire for the advancing troops.

Related Terms

  • degree higher: machine gun - fully automatic gun that can fire bullets rapidly
  • degree lower: submachine gun - a lightweight automatic gun designed to be fired from the hip or shoulder
