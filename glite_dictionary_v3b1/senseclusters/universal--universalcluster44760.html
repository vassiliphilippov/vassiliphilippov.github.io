universal

noun
a feature present in all languages
Pronunciation: / ˌjuːnɪˈvɜrsəl / (AmE), / ˌjuːnɪˈvɜːsəl / (BrE)
(Linguistics) A grammatical rule or linguistic feature that appears in all human languages.

Examples

A language universal
Some scholars debate whether recursion is an actual universal or merely common among languages.
The search for a true universal has been a central question in linguistic theory for decades.
Word order is not consistent enough across languages to be considered a universal.
