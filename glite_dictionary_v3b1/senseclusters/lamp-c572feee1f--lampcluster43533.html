LAMP

noun
web development software stack (Linux/Apache/MySQL/PHP)
3.8
Pronunciation: /læmp/ (AmE), /læmp/ (BrE)
An open-source web development software stack composed of Linux (operating system), Apache (web server), MySQL (database), and PHP (scripting language); used to run dynamic websites and applications.

Examples

LAMP software stack
The start-up deployed its blog on a LAMP stack hosted on a virtual private server.
Several small e-commerce sites still use LAMP because it is stable and well supported.
During the class, students set up a LAMP server with Apache, MySQL and PHP components.

Related Terms

  • synonym: software stack - a set of software components used together to provide a complete platform or service
  • synonym: web stack - the combination of server, database and application software used to run websites
