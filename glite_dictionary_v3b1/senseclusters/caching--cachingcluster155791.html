caching

noun
temporary data storage
Pronunciation: / ˈkæʃɪŋ / (AmE), / ˈkæʃɪŋ / (BrE)
the process of storing data temporarily so it can be accessed more quickly in the future

Examples

Web caching data
Modern web browsers use caching to help websites load faster for users.
The app's caching feature reduces the need to download the same files repeatedly.
Efficient caching can greatly improve the performance of a computer system.

Related Terms

  • antonym: retrieval - the process of getting data back from storage
  • synonym: storage - the act of keeping data or information for later use
