frame

noun
tabular data structure (rows and columns)
5.1
Pronunciation: /freɪm/ (AmE), /freɪm/ (BrE)
(in computing and data analysis) a two-dimensional tabular data structure with labeled rows and columns used to store, manipulate, and analyze datasets.

Examples

Data frame
The analyst loaded the survey into a frame to clean and explore all missing values quickly.
She converted the JSON logs into a frame for easier filtering and grouping operations.
Large datasets often require multiple frames to store related tables before merging them.

Related Terms

  • synonym: dataset - a collection of related data, often stored or represented as a frame or table
  • synonym: table - a set of data arranged in rows and columns; often used interchangeably with data frame in user contexts
