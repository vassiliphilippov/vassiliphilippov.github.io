brushable

adjective
selectable with brush tool
Pronunciation: / ˈbrʌʃəbəl / (AmE), / ˈbrʌʃəbəl / (BrE)
able to be selected, highlighted, or modified using a brush tool in digital graphics or data visualizations

Examples

Brushable area

In this visualization, each point is brushable so users can interactively explore the data.
