OBJ

noun
3D model file format
9.6
Pronunciation: / oʊ biː dʒeɪ / (AmE), / əʊ biː dʒeɪ / (BrE)
a plain-text 3D geometry file format used to store information about the shape and structure of 3D models

Examples

3D OBJ format
She exported the 3D model as an OBJ to share it with her team.
Many 3D design programs can open and save files in the OBJ format.
The artist downloaded a free OBJ file to use in his animation project.

Related Terms

  • synonym: FBX - a 3D file format for exchanging models between different software
  • synonym: STL - a common 3D model file format used for 3D printing
