difference

noun
list of changes between file versions
Pronunciation: / ˈdɪfərəns / (AmE), / ˈdɪfərəns / (BrE)
(in computing) a list or set of changes between two versions of a file or codebase, often produced as a 'diff' and used to create or apply a patch.

Examples

Difference file
The developer sent the difference file so the reviewer could apply the changes to the branch.
Before merging, they inspected the difference to ensure no sensitive data would be accidentally included.
The patch contained a small difference that fixed a security bug in the authentication module.

Related Terms

  • synonym: diff - a file or output showing the changes between two versions of text or code
  • synonym: patch - a set of changes applied to software to update or fix it, often created from a difference
