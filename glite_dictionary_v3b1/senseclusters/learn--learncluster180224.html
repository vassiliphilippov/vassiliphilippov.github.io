learn

verb
model improves by training on data
Pronunciation: / lɝn / (AmE), / lɜːn / (BrE)
(in computing, especially machine learning) (of a computer program or model) to improve performance, make predictions, or acquire patterns by adjusting internal parameters through training on data.

Examples

To learn from data
The neural network will learn useful features automatically when trained on large labeled datasets.
After several iterations, the model learned to identify fraudulent transactions with surprising accuracy.
We hope our recommendation system keeps learning from user feedback to improve suggestions over time.

Related Terms

  • synonym: adapt - to change behaviour or parameters to suit new data or conditions
  • synonym: train - to adjust a model's parameters using data so it performs a task
