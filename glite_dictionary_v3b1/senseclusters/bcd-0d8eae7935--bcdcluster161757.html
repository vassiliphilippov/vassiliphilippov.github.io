BCD

noun
Windows boot data store
9.3 0.02/m
Pronunciation: / bi si di / (AmE), / biÀê siÀê diÀê / (BrE)
a Windows system data store (Boot Configuration Data) that contains boot parameters and controls how the operating system starts

Examples

Windows BCD error
If the BCD is missing or corrupted, Windows may fail to boot properly.
Technicians often repair the BCD when troubleshooting startup issues on Windows computers.
You can use special commands to rebuild the BCD and restore your system.

Related Terms

  • synonym: bootloader - a program that loads the operating system when a computer starts
  • synonym: registry - a database in Windows that stores configuration settings
