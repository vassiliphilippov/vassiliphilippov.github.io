A line drawn on a map connecting points where a given variable has the same value, such as temperature or rainfall.

Examples

An isopleth line
By following each isopleth, scientists can identify regions with the same temperature.
Elevation changes are typically shown with isopleths marked at specific intervals.
The meteorologist drew an isopleth on the weather map to represent areas with equal rainfall.

Synonyms

  • isoline
    8.0 0.01/m
    an isoline is a broad map term for lines joining equal values like temperature or elevation, while isopleth names that same line.
  • isarithm
    9.9 0.00/m
    an isarithm is a technical map term stressing exact numerical equality of values, while isopleth is the general name for those lines.
  • isogram
    9.9 0.00/m
    an isogram is a less common map term for lines joining equal values, while isopleth is the more standard word used by mapmakers.
  • isometric line
    7.2 0.00/m
    an isometric line appears on maps or diagrams to join equal values but often appears in technical diagrams, while isopleth refers specifically to map lines.
