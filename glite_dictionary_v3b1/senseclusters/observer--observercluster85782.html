observer

noun
software component receiving updates
5.8 3.31/m
Pronunciation: / …ôbÀàz…úrv…ôr / (AmE), / …ôbÀàz…úÀêv…ô / (BrE)
(in software engineering) a component or object that subscribes to another object (the subject or observable) and is notified automatically when that object's state changes.

Examples

Software observer
In the Observer pattern, the observer registers with a subject to receive change notifications.
When the model updates, all registered observers refresh their views automatically in the user interface.
The developer implemented an observer that listens for configuration changes and reloads settings.

Synonyms

  • listener
    6.3 0.78/m
    listener waits for external events or network connections and responds when they occur, while observer registers to another object's state and gets notified on changes.

Related Terms

  • synonym: listener - a software object that listens for and responds to events or messages
  • synonym: subscriber - a program component that registers to receive events or messages from a publisher
