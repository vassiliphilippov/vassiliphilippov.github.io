slice

noun
virtual portion of a telecom network
Pronunciation: / slaɪs / (AmE), / slaɪs / (BrE)
(telecommunications) a virtualized, logically isolated portion of a physical network allocated for a particular service, customer, or quality-of-service requirement.

Examples

A network slice
The operator provisioned a separate slice of the 5G network to handle video conferencing traffic.
Enterprises can lease a dedicated slice to guarantee low latency for critical IoT applications.
Engineers used network management tools to monitor each tenant's slice for performance and security issues.

Related Terms

  • synonym: network partition - a division of a network's resources into separate segments for management or isolation
  • synonym: virtual network - a logically separate network created within shared physical infrastructure using virtualization techniques
