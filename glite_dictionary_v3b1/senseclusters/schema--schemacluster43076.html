schema

noun
cognitive framework organizing information
6.1
Pronunciation: / ˈskiːmə / (AmE), / ˈskiːmə / (BrE)
(In psychology) a cognitive framework or concept that helps organize and interpret information, which can be adjusted with new experiences.

Examples

Mental schema
Children develop a new schema when they learn that not all four-legged animals are dogs.
The teacher designed activities to expand the students’ existing schema about plants.
Your schema for social situations changes as you gain more life experience.

Related Terms

  • synonym: framework - a basic structure underlying a system or concept
