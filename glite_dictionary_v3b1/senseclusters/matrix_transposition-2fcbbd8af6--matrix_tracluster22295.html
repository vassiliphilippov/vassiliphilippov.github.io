matrix transposition

noun
interchanging rows and columns of a matrix
Pronunciation: / ˈmeɪ.trɪks træn.spəˈzɪʃ.ən / (AmE), / ˈmeɪ.trɪks træn.spəˈzɪʃ.ən / (BrE)
the act of interchanging the rows and columns of a matrix, resulting in a new matrix where the rows become columns and the columns become rows

Examples

Matrix transposition operation
A matrix transposition leaves diagonal elements unchanged in a square matrix.
Many programming libraries include a built-in function for matrix transposition.
To compute the inverse, you must perform a matrix transposition before further calculations.

  • synonym: transpose - to interchange the rows and columns of a matrix
