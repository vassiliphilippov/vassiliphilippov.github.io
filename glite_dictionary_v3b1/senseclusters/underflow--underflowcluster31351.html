underflow

noun
computing calculation error
8.8
Pronunciation: /ˈʌn.dər.floʊ/ (AmE), /ˈʌn.də.fləʊ/ (BrE)
a condition in computing where a calculation results in a value smaller than the smallest value the system can represent

Examples

Underflow error in calculation
The software detected an underflow when the result was too close to zero to be stored.
If you divide a very small number by a large one, you might cause an underflow in the calculation.
Programmers must check for underflow to prevent unexpected results in scientific computations.

Related Terms

  • antonym: overflow - a condition where a calculation exceeds the maximum value a system can represent
  • synonym: rounding error - an error caused by approximating a number to fit within a system's limits
