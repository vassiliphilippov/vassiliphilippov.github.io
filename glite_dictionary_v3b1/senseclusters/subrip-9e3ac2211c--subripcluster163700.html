SubRip

noun
subtitle file format
Pronunciation: / ˈsʌbˌrɪp / (AmE), / ˈsʌbˌrɪp / (BrE)
a digital file format that stores movie subtitles as plain text, commonly with the .srt extension

Examples

SubRip file format
You can download the SubRip file to add subtitles to your movie.
Most video players are compatible with SubRip subtitle files for easy viewing.
She edited the SubRip file to correct the timing of the subtitles.

  • synonym: subtitle - text displayed on screen translating or transcribing dialogue
