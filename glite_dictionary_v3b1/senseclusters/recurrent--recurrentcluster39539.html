recurrent

adjective
computing: with feedback over time
Pronunciation: / r6ac8k5dd0r59nt / (AmE), / r6ac8k8cr59nt / (BrE)
(in computing, especially machine learning) describing a model, layer, or network that uses feedback or cyclical connections so information can persist across time steps; often used of recurrent neural networks (RNNs).

Examples

Recurrent neural network
Researchers trained a recurrent neural network to analyze sequential weather data.
The translation model includes a recurrent layer to maintain context across sentences.
Using recurrent connections, the system can remember earlier inputs when processing long sequences.

Related Terms

  • synonym: recurrent neural network - a neural network architecture with feedback connections that retains information across time steps
  • synonym: sequence model - a model designed to process and make predictions about sequences of data
