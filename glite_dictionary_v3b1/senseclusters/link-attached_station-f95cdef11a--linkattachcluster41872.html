link-attached station

noun
terminal connected by a data link
Pronunciation: / lɪŋk əˈtætʃt ˈsteɪʃən / (AmE), / lɪŋk əˈtætʃt ˈsteɪʃən / (BrE)
a computer terminal that is connected to a computer system via a data link, rather than being directly connected

Examples

A link-attached station
A link-attached station allows users in remote locations to access central computing resources.
The network administrator monitored each link-attached station to ensure a stable connection.
When upgrading the system, they replaced all old terminals with modern link-attached stations.

Related Terms

  • synonym: terminal - a device through which a user interacts with a computer system
