to start or switch on a machine, device, computer, or software application; to boot or launch it.

Examples

To fire up a computer
Before the meeting, she decided to fire up her laptop to review the slides.
They fired up the test server at midnight to run the automated deployment scripts.
If the app crashes, try firing up the program again after clearing the cache.

Synonyms

  • start up
    3.0 1.68/m
    start up is a neutral phrase for beginning a machine or system, while fire up is more informal and often used for engines or apps.
  • boot
    4.5 2.38/m
    boot is the computer process of loading the operating system, while fire up is an informal way to start machines or software.
  • spin up
    5.2 0.15/m
    spin up is a technical computing term for initializing services or resources, while fire up is more informal and general for starting devices.
  • turn on
    1.0 23/m
    turn on makes a device begin working by a switch or button, while fire up sounds more informal and energetic.
  • switch on
    1.6 3.13/m
    switch on makes power flow when you press a switch, while fire up is informal and often means starting engines or software.
