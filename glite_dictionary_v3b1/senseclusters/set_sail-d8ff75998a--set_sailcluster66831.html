set sail

verb
begin a boat or ship journey
Pronunciation: / sɛt seɪl / (AmE), / sɛt seɪl / (BrE)
to begin a journey by boat or ship; to start sailing

Examples

To set sail at dawn
The ship will set sail at dawn if the weather permits.
They set sail from the harbor, eager for their next adventure.
We watched as the old fishing boat set sail into the distance.

Related Terms

  • antonym: arrive - to reach a destination
  • synonym: depart - to leave, especially to start a journey
