so far as

adverb
to the extent that; inasmuch as
Pronunciation: / soʊ fɑr æz / (AmE), / səʊ fɑːr æz / (BrE)
to the degree or extent that; inasmuch as.

Examples

So far as honesty goes
So far as his honesty is concerned, I have no doubts.
So far as we know, the project will continue next year.
You should only intervene so far as it is absolutely necessary.

  • synonym: insofar as - to the extent that; inasmuch as
