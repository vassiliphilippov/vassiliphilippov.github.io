evaluable

adjective
able to be evaluated
7.1
Pronunciation: / ɪˈvæljuəbl̩ / (AmE), / ɪˈvæljuəbl̩ / (BrE)
able to be calculated, measured, or given a value, especially in computing or logic

Examples

Evaluable expression
Before running the program, check that every variable is evaluable in the current context.
The formula is only evaluable if all required data is provided.
In some programming languages, not every expression is directly evaluable at runtime.

Related Terms

  • synonym: computable - able to be calculated by a computer or algorithm
  • synonym: measurable - able to be measured or quantified
  • antonym: unevaluable - not able to be evaluated or given a value
