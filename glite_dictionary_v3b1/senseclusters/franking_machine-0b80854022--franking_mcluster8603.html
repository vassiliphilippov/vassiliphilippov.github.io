franking machine

noun
machine for stamping and calculating postage
Pronunciation: / ˈfræŋkɪŋ məˈʃin / (AmE), / ˈfræŋkɪŋ məˈʃiːn / (BrE)
A machine used for automatically stamping letters or packages and calculating the postage charge.

Examples

The franking machine stamps letters
The franking machine calculates the exact postal charges for every parcel.
The company recently invested in a new franking machine to streamline its outgoing mail.
Using a franking machine saves time compared to manual stamping of letters.

Related Terms

  • synonym: postage meter - a machine that prints postage directly on mail
