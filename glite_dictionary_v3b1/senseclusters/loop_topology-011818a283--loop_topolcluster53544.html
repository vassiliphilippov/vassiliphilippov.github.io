loop topology

noun
network layout forming a closed loop
6.9
Pronunciation: /luːp tə.ˈpɔː.lə.dʒiː/ /; / /luːp tə.ˈpɑː.lə.dʒiː/ (AmE), /luːp tə.ˈpɒ.lə.dʒiː/ (BrE)
a network configuration in which each device is connected in a closed ring, so the last connects back to the first

Examples

A loop topology network
A loop topology can be problematic if one device fails, as it may disrupt the entire network.
Engineers debated whether to design the new office network with a loop topology or a star topology.
The company's old network used a loop topology that allowed data to circulate continuously around the system.

Related Terms

  • synonym: ring topology - network topology where each device is connected in a circular chain forming a ring
