spider

noun
web-crawling computer program
5.3
Pronunciation: / ˈspaɪdɚ / (AmE), / ˈspaɪdə / (BrE)
a computer program that automatically searches and indexes websites across the internet, typically for use by search engines

Examples

Web spider program
A search engine uses a spider to crawl the internet and index website content.
The spider identified hundreds of new pages overnight.
Web developers sometimes block a spider from accessing certain parts of their sites.

Related Terms

  • synonym: bot - an automated computer program performing tasks over the internet
  • synonym: web crawler - a program that systematically browses the internet for data collection
