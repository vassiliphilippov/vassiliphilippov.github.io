checkpoint

noun
saved state in computing
4.5
Pronunciation: /ˈtʃek.pɔɪnt/ (AmE), /ˈtʃek.pɔɪnt/ (BrE)
a saved state of a computer program, process, or machine-learning model that can be stored and later restored to resume work from that point.

Examples

Save a checkpoint
After each epoch, the experiment stored a checkpoint to resume training if interrupted.
The backup system creates a checkpoint every hour so we can restore the service quickly.
When the program crashed, loading the most recent checkpoint saved several hours of work.

Related Terms

  • synonym: backup - a copy of data kept to restore the original in case of loss or damage
  • synonym: snapshot - a saved state of a system or dataset at a particular moment in time
