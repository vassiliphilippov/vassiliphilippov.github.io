bus

noun
common electrical/data conductor
5.9 4.3
Pronunciation: /bʌs/ (AmE), /bʌs/ /; / /bʊs/ (BrE)
(technical) a system or conductor that connects different parts of an electronic or computer system to transmit signals, power, or data

Examples

Computer bus system
If the bus is overloaded, it can cause data transmission errors in the system.
Technicians traced the fault to a break in the main power bus.
The motherboard has a bus that connects the processor to memory and other components.
If a single cable in a bus topology fails, the entire network may be affected.
Many older local area networks used a bus to connect all computers and devices.
Network engineers debated whether to upgrade the old bus system to a more modern topology.

Related Terms

  • synonym: busbar - a thick strip of copper or aluminum in switchboards and electrical substations for distributing power
  • antonym: ring topology - network topology where nodes are connected in a closed loop
  • antonym: star topology - network topology where each node is connected to a central hub
  • antonym: tree topology - network topology with hierarchical connections
