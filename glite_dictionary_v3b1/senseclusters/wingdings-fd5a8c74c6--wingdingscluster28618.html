Wingdings

proper noun
symbol font
Pronunciation: / ˈwɪŋdɪŋz / (AmE), / ˈwɪŋdɪŋz / (BrE)
a font consisting of symbols and pictograms instead of standard letters or numbers

Examples

Wingdings font symbols
She changed the text to Wingdings to create a line of decorative symbols.
If you type in Wingdings, the letters appear as various icons and shapes.
The invitation used Wingdings to add fun pictograms next to each event detail.

Related Terms

  • synonym: Webdings - a font made up of symbols and icons for web use
  • synonym: font - a set of printed or displayed type of one style and size
