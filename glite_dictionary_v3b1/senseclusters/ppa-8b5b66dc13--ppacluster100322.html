PPA

noun
Ubuntu software repository
Pronunciation: / pi pi eɪ / (AmE), / piː piː eɪ / (BrE)
a user-hosted software repository for distributing packages, especially for Ubuntu Linux systems

Examples

Ubuntu PPA repository
To get the latest updates, you can add a PPA to your Ubuntu system.
Some software is only available through a PPA rather than the official repositories.
He removed the outdated PPA to avoid conflicts with newer packages.

Related Terms

  • antonym: official repository - the main software source maintained by the operating system developers
  • synonym: repository - a storage location for software packages
