1. ⌨️ equal-width letters ("constant-width font for code")

NOUN
Pronunciation: / ˈkɑn.stənt wɪdθ fɑnt / (AmE), / ˈkɒn.stənt wɪdθ fɒnt / (BrE)
a typeface in which each character occupies the same amount of horizontal space, as in a typewriter or computer terminal
font with equal-width characters

Examples

Many email clients allow you to switch to a constant-width font for reading plain text messages.
The report was printed using a constant-width font for easy tabulation of numbers.
When programming, it's best to use a constant-width font to align your code clearly.

Related Terms
