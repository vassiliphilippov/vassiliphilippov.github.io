NOUN
Pronunciation: / ˈmɑːnoʊˌspeɪst fɑnt / (AmE), / ˈmɒnəˌspeɪst fɒnt / (BrE)
a typeface in which each character occupies the same amount of horizontal space
typeface with equal-width characters

Examples

Many programmers prefer to write code in a monospaced font for better readability.
The old typewriter uses a monospaced font that looks very different from modern printers.
When creating tables, a monospaced font ensures the columns are perfectly aligned.
